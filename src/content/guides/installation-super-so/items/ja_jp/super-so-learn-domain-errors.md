これは FastComments がこのドメインであなたのアカウントの使用を許可するべきだと認識していないためです。

この場合、対処は簡単です。あなたのサイトを FastComments.com のアカウントに追加するだけで済みます。

[アカウントにサイトを追加するにはここを開いてください。](https://fastcomments.com/auth/my-account/configure-domains)
