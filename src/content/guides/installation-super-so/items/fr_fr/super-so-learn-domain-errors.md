Il se peut que vous receviez une erreur d'autorisation, comme la suivante :

<div class="screenshot white-bg">
    <div class="title">Configuration du domaine manquante</div>
    <img class="screenshot-image" src="/images/installation-guides/installation-guide-webflow-step-5.png" alt="Configuration du domaine manquante" />

Cela vient du fait que FastComments ne sait pas qu'il doit autoriser l'utilisation de votre compte sur ce domaine.

Dans ce cas, la correction est simple. Il vous suffit d'ajouter votre site à votre compte FastComments.com.

[Rendez-vous ici pour ajouter votre site à votre compte.](https://fastcomments.com/auth/my-account/configure-domains)
