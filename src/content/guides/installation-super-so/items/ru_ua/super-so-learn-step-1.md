Сначала нужно открыть редактор кода. Если вы хотите добавить FastComments на все страницы, просто выберите Code в левом нижнем углу:

<div class="screenshot white-bg">
    <div class="title">Открыть настройки кода</div>
    <img class="screenshot-image" src="/images/installation-guides/super-so-step-1-code-tab.png" alt="Открыть настройки кода" />
</div>

Если вы хотите добавить его на конкретную страницу, выберите `Edit Custom Code` в настройках этой страницы.

Теперь выберем вкладку `Body`. **Это важно!**. Установка фрагмента кода в `Head` **не работает**.

</div>
