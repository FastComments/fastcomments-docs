### 401 Σφάλματα μη εξουσιοδότησης

Εάν λαμβάνετε σφάλματα 401 κατά τη χρήση του επαληθευμένου API:

1. **Ελέγξτε το κλειδί API σας**: Βεβαιωθείτε ότι χρησιμοποιείτε το σωστό κλειδί API από τον πίνακα ελέγχου FastComments σας
2. **Επαληθεύστε το tenant ID**: Βεβαιωθείτε ότι το tenant ID αντιστοιχεί στον λογαριασμό σας
3. **Μορφή κλειδιού API**: Το κλειδί API πρέπει να οριστεί στον API client:

```swift
let defaultApi = DefaultAPI()
defaultApi.apiKey = "YOUR_API_KEY"
```

4. **Χρήση λάθους API**: Βεβαιωθείτε ότι χρησιμοποιείτε `DefaultAPI` (όχι `PublicAPI`) για εξουσιοδοτημένες κλήσεις

### Προβλήματα με διακριτικά SSO

Εάν τα διακριτικά SSO δεν λειτουργούν:

1. **Χρησιμοποιήστε ασφαλή λειτουργία για παραγωγή**: Χρησιμοποιείτε πάντα `FastCommentsSSO.createSecure()` με το κλειδί API σας για το περιβάλλον παραγωγής
2. **Μόνο στην πλευρά του διακομιστή**: Δημιουργείτε ασφαλή διακριτικά SSO στον διακομιστή σας, μην εκθέτετε ποτέ το κλειδί API στους πελάτες
3. **Ελέγξτε τα δεδομένα χρήστη**: Βεβαιωθείτε ότι όλα τα απαιτούμενα πεδία (id, email, username) παρέχονται
4. **Λήξη του διακριτικού**: Τα ασφαλή διακριτικά SSO περιλαμβάνουν χρονική σήμανση και μπορεί να λήξουν. Δημιουργήστε νέα διακριτικά όταν χρειάζεται.

### Σφάλματα SSL/TLS

Εάν αντιμετωπίσετε σφάλματα SSL/TLS:

1. Βεβαιωθείτε ότι το Info.plist της εφαρμογής σας επιτρέπει συνδέσεις HTTPS προς το fastcomments.com
2. Ελέγξτε ότι δεν χρησιμοποιείτε εξαιρέσεις του App Transport Security που μπορεί να μπλοκάρουν τη σύνδεση
