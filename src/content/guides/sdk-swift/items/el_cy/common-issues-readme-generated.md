### 401 Σφάλματα Μη Εξουσιοδοτημένου

Αν λαμβάνετε σφάλματα 401 κατά τη χρήση του εξουσιοδοτημένου API:

1. **Ελέγξτε το κλειδί API**: Βεβαιωθείτε ότι χρησιμοποιείτε το σωστό κλειδί API από τον πίνακα ελέγχου FastComments
2. **Επαληθεύστε το tenant ID**: Σιγουρευτείτε ότι το tenant ID ταιριάζει με τον λογαριασμό σας
3. **Μορφή κλειδιού API**: Το κλειδί API πρέπει να ορίζεται ως το header `x-api-key` στην κοινή ρύθμιση:

```swift
FastCommentsSwiftAPIConfiguration.shared.customHeaders["x-api-key"] = "YOUR_API_KEY"
```

4. **Χρήση λανθασμένου API**: Βεβαιωθείτε ότι χρησιμοποιείτε το `DefaultAPI` (και όχι το `PublicAPI`) για τις εξουσιοδοτημένες κλήσεις

### Προβλήματα με το Token SSO

Αν τα tokens SSO δεν λειτουργούν:

1. **Χρησιμοποιήστε ασφαλή λειτουργία για παραγωγή**: Πάντα χρησιμοποιείτε το `FastCommentsSSO.createSecure()` με το κλειδί API σας για παραγωγή
2. **Μόνο στο διακομιστή**: Δημιουργήστε ασφαλή tokens SSO στον διακομιστή σας, ποτέ μην εκθέτετε το κλειδί API σας στους πελάτες
3. **Ελέγξτε τα δεδομένα χρήστη**: Βεβαιωθείτε ότι παρέχονται όλα τα απαιτούμενα πεδία (id, email, username)
4. **Λήξη token**: Τα ασφαλή tokens SSO περιλαμβάνουν χρονική σήμανση και μπορεί να λήξουν. Δημιουργήστε νέα tokens όποτε χρειάζεται.

### Σφάλματα SSL/TLS

Αν αντιμετωπίζετε σφάλματα SSL/TLS:

1. Διασφαλίστε ότι το Info.plist της εφαρμογής σας επιτρέπει συνδέσεις HTTPS στο fastcomments.com
2. Ελέγξτε ότι δεν χρησιμοποιείτε εξαιρέσεις App Transport Security που μπορεί να μπλοκάρουν τη σύνδεση
