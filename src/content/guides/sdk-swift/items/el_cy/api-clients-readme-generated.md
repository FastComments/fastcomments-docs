Το FastComments SDK παρέχει τρεις πελάτες API:

### PublicAPI - Client-Safe Methods

Το `PublicAPI` περιέχει μεθόδους που είναι ασφαλείς για κλήση από κώδικα στην πλευρά του πελάτη (εφαρμογές iOS/macOS). Αυτές οι μέθοδοι:
- Δεν απαιτούν κλειδί API
- Μπορούν να χρησιμοποιήσουν διακριτικά SSO για έλεγχο ταυτότητας
- Έχουν περιορισμό ρυθμού ανά χρήστη/συσκευή
- Είναι κατάλληλες για εφαρμογές που απευθύνονται στον τελικό χρήστη

**Example use case**: Ανάκτηση και δημιουργία σχολίων στην εφαρμογή iOS σας

### DefaultAPI - Server-Side Methods

Το `DefaultAPI` περιέχει αυθεντικοποιημένες μεθόδους που απαιτούν κλειδί API. Αυτές οι μέθοδοι:
- Απαιτούν το κλειδί API του FastComments
- Θα πρέπει να κλήνονται ΜΟΝΟ από κώδικα στην πλευρά του διακομιστή
- Παρέχουν πλήρη πρόσβαση στα δεδομένα του FastComments
- Έχουν περιορισμό ρυθμού ανά ενοικιαστή

**Example use case**: Διαχειριστικές λειτουργίες, μαζική εξαγωγή δεδομένων, διαχείριση χρηστών

### ModerationAPI - Moderator Dashboard Methods

Το `ModerationAPI` παρέχει ένα εκτεταμένο σύνολο ζωντανών και γρήγορων API διαχείρισης. Κάθε μέθοδος `ModerationAPI` δέχεται μια παράμετρο `sso` και μπορεί να αυθεντικοποιηθεί μέσω SSO ή μέσω cookie συνεδρίας FastComments.com.

**Example use case**: Δημιουργία εμπειρίας διαχείρισης για τους συντονιστές της κοινότητάς σας

**IMPORTANT**: Ποτέ μην εκθέτετε το κλειδί API σας σε κώδικα στην πλευρά του πελάτη. Τα κλειδιά API πρέπει να χρησιμοποιούνται μόνο στην πλευρά του διακομιστή.
