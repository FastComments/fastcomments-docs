### Αναδημιουργία του Client

Για να αναδημιουργήσετε τον API client από την τελευταία προδιαγραφή OpenAPI:

1. Βεβαιωθείτε ότι ο FastComments server τρέχει τοπικά στο `http://localhost:3001`
2. Εκτελέστε το script ενημέρωσης:

```bash
./update.sh
```

Αυτό θα:
- Θα κατεβάσει την τελευταία προδιαγραφή OpenAPI
- Θα δημιουργήσει τον κώδικα του Swift client (με τεκμηρίωση API στο client/docs)
- Θα χτίσει το package για να επαληθεύσει ότι όλα λειτουργούν
