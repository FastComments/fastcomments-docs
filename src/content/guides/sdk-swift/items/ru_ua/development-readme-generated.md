### Перегенерация клиента

Чтобы перегенерировать клиент API из последней спецификации OpenAPI:

1. Убедитесь, что сервер FastComments запущен локально по адресу `http://localhost:3001`
2. Запустите скрипт обновления:

```bash
./update.sh
```

Это выполнит:
- Скачает последнюю спецификацию OpenAPI
- Сгенерирует Swift-клиент (с документацией API в client/docs)
- Соберёт пакет, чтобы проверить, что всё работает
