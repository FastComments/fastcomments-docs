### Поновно генерисање клијента

Да бисте поново генерисали API клијента из најновије OpenAPI спецификације:

1. Уверите се да вам је FastComments сервер покренут локално на `http://localhost:3001`
2. Покрените скрипту за ажурирање:

```bash
./update.sh
```

Ово ће:
- Преузети најновију OpenAPI спецификацију
- Генерисати Swift клијентски код (са API документацијом у client/docs)
- Изградити пакет да бисте проверили да све ради
