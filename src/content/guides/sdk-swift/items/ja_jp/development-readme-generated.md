### クライアントの再生成

最新の OpenAPI 仕様から API クライアントを再生成するには:

1. FastComments サーバーがローカルで `http://localhost:3001` で実行されていることを確認してください
2. 更新スクリプトを実行してください:

```bash
./update.sh
```

これにより:
- 最新の OpenAPI 仕様をダウンロードします
- Swift クライアントコードを生成します（API ドキュメントは client/docs に出力されます）
- パッケージをビルドして、すべてが動作することを確認します
