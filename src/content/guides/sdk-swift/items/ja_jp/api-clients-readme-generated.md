The FastComments SDK は 3 つの API クライアントを提供します:

### PublicAPI - クライアント安全なメソッド

`PublicAPI` には、クライアント側のコード（iOS/macOS アプリ）から安全に呼び出せるメソッドが含まれています。これらのメソッドは:
- API キーを必要としません
- 認証に SSO トークンを使用できます
- ユーザー／デバイスごとにレート制限があります
- エンドユーザー向けアプリケーションに適しています

**使用例**: iOS アプリでコメントを取得・作成する

### DefaultAPI - サーバー側メソッド

`DefaultAPI` には、API キーが必要な認証済みメソッドが含まれています。これらのメソッドは:
- FastComments API キーが必要です
- サーバー側のコードからのみ呼び出すべきです
- FastComments データへのフルアクセスが可能です
- テナントごとにレート制限があります

**使用例**: 管理操作、バルクデータエクスポート、ユーザー管理

### ModerationAPI - モデレーターダッシュボードメソッド

`ModerationAPI` は、豊富なライブ・高速モデレーション API を提供します。すべての `ModerationAPI` メソッドは `sso` パラメータを受け取り、SSO または FastComments.com のセッションクッキーで認証できます。

**使用例**: コミュニティのモデレーター向けモデレーション体験を構築する

**IMPORTANT**: クライアント側のコードに API キーを決して公開しないでください。API キーはサーバー側でのみ使用すべきです。
