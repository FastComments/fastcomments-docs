### Επαναδημιουργία του πελάτη

Για να επαναδημιουργήσετε τον πελάτη API από την πιο πρόσφατη προδιαγραφή OpenAPI:

1. Βεβαιωθείτε ότι έχετε τον διακομιστή FastComments να τρέχει τοπικά στο `http://localhost:3001`
2. Εκτελέστε το script ενημέρωσης:

```bash
./update.sh
```

Αυτό θα:
- Κατεβάσει την πιο πρόσφατη προδιαγραφή OpenAPI
- Γεννήσει τον κώδικα Swift client (με τεκμηρίωση API στο client/docs)
- Χτίσει το πακέτο για να επαληθεύσει ότι όλα λειτουργούν
