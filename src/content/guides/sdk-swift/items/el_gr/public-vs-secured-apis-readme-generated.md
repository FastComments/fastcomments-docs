Το FastComments SDK παρέχει δύο τύπους τελικών σημείων API:

### PublicAPI - Τελικά σημεία ασφαλή για κώδικα πελάτη

Το `PublicAPI` περιέχει τελικά σημεία που είναι ασφαλή για να κληθούν από κώδικα πλευράς πελάτη (εφαρμογές iOS/macOS). Αυτά τα τελικά σημεία:
- Δεν απαιτούν κλειδί API
- Μπορούν να χρησιμοποιήσουν tokens SSO για την αυθεντικοποίηση
- Υπόκεινται σε περιορισμό ρυθμού ανά χρήστη/συσκευή
- Είναι κατάλληλα για εφαρμογές που απευθύνονται σε τελικούς χρήστες

**Παράδειγμα χρήσης**: Ανάκτηση και δημιουργία σχολίων στην εφαρμογή iOS σας

### DefaultAPI - Τελικά σημεία για την πλευρά του διακομιστή

Το `DefaultAPI` περιέχει αυθεντικοποιημένα τελικά σημεία που απαιτούν κλειδί API. Αυτά τα τελικά σημεία:
- Απαιτούν το κλειδί API του FastComments
- Πρέπει να καλούνται ΜΟΝΟ από κώδικα που εκτελείται στον διακομιστή
- Παρέχουν πλήρη πρόσβαση στα δεδομένα του FastComments σας
- Υπόκεινται σε περιορισμό ρυθμού ανά tenant

**Παράδειγμα χρήσης**: Διοικητικές λειτουργίες, μαζική εξαγωγή δεδομένων, εργαλεία εποπτείας/διόρθωσης

**ΣΗΜΑΝΤΙΚΟ**: Μην αποκαλύπτετε ποτέ το κλειδί API στο κώδικα πλευράς πελάτη. Τα κλειδιά API πρέπει να χρησιμοποιούνται μόνο στην πλευρά του διακομιστή.
