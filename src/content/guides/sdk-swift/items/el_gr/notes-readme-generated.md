### Αναγνωριστικά μετάδοσης

Θα δείτε ότι πρέπει να περάσετε ένα `broadcastId` σε ορισμένες κλήσεις API. Όταν λαμβάνετε γεγονότα, θα λάβετε πίσω αυτό το ID, ώστε να γνωρίζετε να αγνοήσετε το γεγονός αν σκοπεύετε να εφαρμόσετε τις αλλαγές στον πελάτη με αισιόδοξο τρόπο (κάτι που πιθανότατα θα θέλετε να κάνετε καθώς προσφέρει την καλύτερη εμπειρία). Περάστε εδώ ένα UUID. Το ID θα πρέπει να είναι αρκετά μοναδικό ώστε να μην εμφανιστεί δύο φορές σε μια συνεδρία.

```swift
let broadcastId = UUID().uuidString
```
