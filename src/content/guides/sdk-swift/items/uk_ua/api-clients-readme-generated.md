The FastComments SDK надає три клієнти API:

### PublicAPI - Методи, безпечні для клієнтського коду

The `PublicAPI` містить методи, які безпечно викликати з клієнтського коду (додатки iOS/macOS). Ці методи:
- Не вимагають API-ключа
- Можуть використовувати SSO-токени для автентифікації
- Мають ліміти запитів на користувача/пристрій
- Підходять для додатків, орієнтованих на кінцевих користувачів

**Example use case**: Отримання та створення коментарів у вашому iOS-додатку

### DefaultAPI - Методи для серверної частини

The `DefaultAPI` містить автентифіковані методи, які вимагають API-ключа. Ці методи:
- Вимагають вашого API-ключа FastComments
- Слід ВИКЛЮЧНО викликати з серверного коду
- Надають повний доступ до ваших даних FastComments
- Мають ліміти запитів на орендаря

**Example use case**: Адміністративні операції, масовий експорт даних, керування користувачами

### ModerationAPI - Методи панелі модератора

The `ModerationAPI` містить методи, які забезпечують роботу панелі модератора. Ці методи охоплюють:
- **Модерація коментарів** - перелік, підрахунок, пошук, отримання логів та експорт коментарів
- **Дії модерації** - видалення/відновлення коментарів, позначення, встановлення статусу на рецензію/спам/підтвердження, керування голосами та повторне відкриття/закриття тем
- **Блокування** - заблокувати користувача від коментаря, скасувати блокування, отримати підсумки перед блокуванням, перевірити статус і налаштування блокування та прочитати кількість заблокованих користувачів
- **Бейджі та довіра** - присвоювати/знімати бейджі, перелік вручну призначених бейджів, отримувати/встановлювати коефіцієнт довіри користувача та читати внутрішній профіль користувача

Every `ModerationAPI` method accepts an `sso` parameter so moderators can be authenticated via SSO.

**Example use case**: Створення досвіду модерації для модераторів вашої спільноти

**IMPORTANT**: Ніколи не розкривайте ваш API-ключ у клієнтському коді. API-ключі слід використовувати лише на сервері.
