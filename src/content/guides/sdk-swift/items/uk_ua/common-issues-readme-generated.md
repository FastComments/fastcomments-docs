### 401 Неавторизовані помилки

Якщо ви отримуєте помилки 401 під час використання автентифікованого API:

1. **Перевірте ваш API‑ключ**: Переконайтеся, що ви використовуєте правильний API‑ключ з вашої панелі FastComments
2. **Перевірте ідентифікатор орендаря**: Переконайтеся, що ID орендаря відповідає вашому обліковому запису
3. **Формат API‑ключа**: API‑ключ має бути встановлений як заголовок `x-api-key` у спільній конфігурації:

```swift
FastCommentsSwiftAPIConfiguration.shared.customHeaders["x-api-key"] = "YOUR_API_KEY"
```

4. **Використання неправильного API**: Переконайтеся, що ви використовуєте `DefaultAPI` (а не `PublicAPI`) для автентифікованих викликів

### Проблеми з токеном SSO

Якщо токени SSO не працюють:

1. **Використовувати безпечний режим у продакшн**: Завжди використовуйте `FastCommentsSSO.createSecure()` з вашим API‑ключем у продакшн‑середовищі
2. **Тільки на сервері**: Генеруйте безпечні SSO‑токени на вашому сервері, ніколи не розкривайте ваш API‑ключ клієнтам
3. **Перевірте дані користувача**: Переконайтеся, що всі необхідні поля (id, email, username) надані
4. **Термін дії токену**: Безпечні SSO‑токени містять мітку часу і можуть закінчитися. Генеруйте нові токени за потреби.

### Помилки SSL/TLS

Якщо ви стикаєтеся з помилками SSL/TLS:

1. Переконайтеся, що ваш файл Info.plist у додатку дозволяє HTTPS‑з’єднання з fastcomments.com
2. Перевірте, що ви не використовуєте виключення App Transport Security, які можуть блокувати з’єднання
