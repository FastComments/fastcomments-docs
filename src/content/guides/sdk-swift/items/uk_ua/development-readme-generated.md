### Перегенерація клієнта

Щоб перегенерувати клієнта API з останньої специфікації OpenAPI:

1. Переконайтеся, що сервер FastComments запущено локально за адресою `http://localhost:3001`
2. Запустіть скрипт оновлення:

```bash
./update.sh
```

Це зробить:
- Завантажить останню специфікацію OpenAPI
- Згенерує Swift-клієнтський код (з документацією API в client/docs)
- Збере пакет, щоб перевірити, що все працює
