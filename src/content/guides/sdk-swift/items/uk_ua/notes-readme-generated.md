### Ідентифікатори Broadcast

Ви побачите, що у деяких викликах API потрібно передавати `broadcastId`. Коли ви отримуватимете події, ви отримаєте цей ID назад, тож знатимете, що слід ігнорувати подію, якщо плануєте оптимістично застосовувати зміни на клієнті (що, ймовірно, ви захочете зробити, оскільки це забезпечує найкращий досвід). Передайте тут UUID. ID має бути достатньо унікальним, щоб не повторюватися двічі в одній сесії.

```swift
let broadcastId = UUID().uuidString
```
