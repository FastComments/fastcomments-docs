The FastComments SDK надає два типи кінцевих точок API:

### PublicAPI - Кінцеві точки, безпечні для клієнтського коду

The `PublicAPI` contains endpoints that are safe to call from client-side code (iOS/macOS apps). These endpoints:
- Не вимагають API-ключа
- Можуть використовувати SSO-токени для автентифікації
- Мають ліміти швидкості запитів на кожного користувача/пристрій
- Підходять для застосунків, орієнтованих на кінцевих користувачів

**Приклад використання**: Отримання та створення коментарів у вашому iOS-додатку

### DefaultAPI - Кінцеві точки для серверної частини

The `DefaultAPI` contains authenticated endpoints that require an API key. These endpoints:
- Потребують вашого FastComments API-ключа
- Мають викликатися ТІЛЬКИ з серверного коду
- Надають повний доступ до ваших даних FastComments
- Мають обмеження швидкості запитів на рівні орендаря

**Приклад використання**: адміністративні операції, масовий експорт даних, інструменти модерування

**ВАЖЛИВО**: Ніколи не розголошуйте ваш API-ключ у клієнтському коді. API-ключі повинні використовуватися лише на стороні сервера.
