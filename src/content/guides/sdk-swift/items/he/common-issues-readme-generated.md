### שגיאות 401 לא מורשות

אם אתה מקבל שגיאות 401 כאשר אתה משתמש ב‑API המאומת:

1. **בדוק את מפתח ה‑API שלך**: וודא שאתה משתמש במפתח ה‑API הנכון מלוח הבקרה של FastComments
2. **ודא את מזהה השוכרת**: ודא שמזהה השוכרת תואם לחשבון שלך
3. **פורמט מפתח ה‑API**: מפתח ה‑API צריך להיות מוגדר ככותרת `x-api-key` בתצורה המשותפת:

```swift
FastCommentsSwiftAPIConfiguration.shared.customHeaders["x-api-key"] = "YOUR_API_KEY"
```

4. **שימוש ב‑API הלא נכון**: ודא שאתה משתמש ב‑`DefaultAPI` (ולא ב‑`PublicAPI`) עבור קריאות מאומתות

### בעיות אסימון SSO

אם אסימוני SSO אינם פועלים:

1. **השתמש במצב מאובטח עבור ייצור**: תמיד השתמש ב‑`FastCommentsSSO.createSecure()` עם מפתח ה‑API שלך עבור ייצור
2. **צד השרת בלבד**: צור אסימוני SSO מאובטחים על השרת שלך, לעולם אל תחשוף את מפתח ה‑API ללקוחות
3. **בדוק נתוני משתמש**: ודא שכל השדות הדרושים (id, email, username) סופקו
4. **תפוגת אסימון**: אסימוני SSO מאובטחים כוללים חותמת זמן ויכולים לפוג. צור אסימונים חדשים לפי הצורך.

### שגיאות SSL/TLS

אם אתה נתקל בשגיאות SSL/TLS:

1. וודא שקובץ Info.plist של האפליקציה מאפשר חיבורים HTTPS אל fastcomments.com
2. בדוק שאינך משתמש בחריגות של App Transport Security שעשויות לחסום את החיבור
