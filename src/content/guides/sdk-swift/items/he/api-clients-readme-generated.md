ה‑SDK של FastComments מספק שלושה לקוחות API:

### PublicAPI - שיטות בטוחות ללקוח

ה‑`PublicAPI` מכילה שיטות בטוחות לקריאה מקוד בצד הלקוח (אפליקציות iOS/macOS). שיטות אלו:
- אינן דורשות מפתח API
- יכולות להשתמש באסימוני SSO לאימות
- מוגבלות בקצב לכל משתמש/מכשיר
- מתאימות ליישומים המיועדים למשתמש הקצה

**דוגמה לשימוש**: קבלת והוספת תגובות באפליקציית iOS שלך

### DefaultAPI - שיטות בצד השרת

ה‑`DefaultAPI` מכילה שיטות מאומתות הדורשות מפתח API. שיטות אלו:
- דורשות את מפתח ה‑API של FastComments שלך
- צריך לקרוא להן רק מקוד בצד השרת
- מספקות גישה מלאה לנתוני FastComments שלך
- מוגבלות בקצב לכל שוכר

**דוגמה לשימוש**: פעולות ניהוליות, ייצוא נתונים בכמויות גדולות, ניהול משתמשים

### ModerationAPI - שיטות ללוח המחוונים של המפקח

ה‑`ModerationAPI` מספקת חבילה נרחבת של API לשיפור מודרציה בזמן אמת ומהירה. כל שיטת `ModerationAPI` מקבלת פרמטר `sso` ויכולה לאמת באמצעות SSO או קובץ עוגייה של FastComments.com.

**דוגמה לשימוש**: בניית חוויית מודרציה למפקחים בקהילה שלך

**חשוב**: לעולם אל תחשוף את מפתח ה‑API שלך בקוד בצד הלקוח. מפתחות API צריכים לשמש רק בצד השרת.
