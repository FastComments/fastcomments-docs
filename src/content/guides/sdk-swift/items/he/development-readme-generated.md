### יצירת מחדש של הלקוח

כדי ליצור מחדש את לקוח ה-API מהמפרט העדכני של OpenAPI:

1. ודא שהשרת של FastComments פועל מקומית ב- `http://localhost:3001`
2. הפעל את סקריפט העדכון:

```bash
./update.sh
```

זה יבצע את הפעולות הבאות:
- הורדת מפרט ה-OpenAPI העדכני ביותר
- יצירת קוד הלקוח ב-Swift (עם תיעוד ה-API ב-client/docs)
- בניית החבילה כדי לאמת שהכל פועל
