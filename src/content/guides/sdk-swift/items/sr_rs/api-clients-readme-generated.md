The FastComments SDK пружа три API клијента:

### PublicAPI - Методе безбедне за клијентски код

The `PublicAPI` садржи методе које је безбедно позивати из клијентског кода (iOS/macOS апликације). Ове методе:
- Не захтевају API key
- Могу користити SSO tokens за аутентификацију
- Имају ограничење брзине захтева по кориснику/уређају
- Погодне су за апликације намењене крајњим корисницима

**Example use case**: Преузимање и креирање коментара у вашој iOS апликацији

### DefaultAPI - Методе за серверску страну

The `DefaultAPI` садржи аутентификоване методе које захтевају API key. Ове методе:
- Захтевају ваш FastComments API key
- Требају БИТИ позиване ИСКЉУЧИВО са серверске стране
- Пружају пун приступ вашим FastComments подацима
- Имају ограничење брзине по tenant-у

**Example use case**: Административне операције, масован извоз података, управљање корисницима

### ModerationAPI - Методе за модераторску контролну таблу

The `ModerationAPI` садржи методе које погоне модераторску контролну таблу. Ове методе обухватају:
- **Comment moderation** - листање, бројање, претрага, преузимање логова, и извоз коментара
- **Moderation actions** - уклањање/враћање коментара, означавање, постављање статуса за преглед/спам/одобрење, управљање гласовима, и поновно отварање/затварање тема
- **Bans** - забрана корисника за коментар, поништавање забрана, преузимање сажетака пре забране, провера статуса забране и подешавања, и читање броја забрањених корисника
- **Badges & trust** - додељивање/уклањање значки, листање ручних значки, добијање/постављање корисниковог фактора поверења, и читање унутрашњег профила корисника

Every `ModerationAPI` method accepts an `sso` parameter so moderators can be authenticated via SSO.

**Example use case**: Прављење модераторског искуства за модераторе ваше заједнице

**IMPORTANT**: Никада не откривајте ваш API key у клијентском коду. API keys треба користити само на серверској страни.
