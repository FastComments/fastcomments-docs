FastComments SDK пружа три API клијента:

### PublicAPI - Метoде безбедне за клијент

`PublicAPI` садржи методе које је безбедно позивати из кода на страни клијента (iOS/macOS апликације). Ове методе:
- Не захтевају API кључ
- Могу користити SSO токене за аутентификацију
- Ограничене су по броју захтева по кориснику/уређају
- Погодне су за апликације усмерене према крајњим корисницима

**Пример употребе**: Прибављање и креирање коментара у вашој iOS апликацији

### DefaultAPI - Методе на страни сервера

`DefaultAPI` садржи аутентификоване методе које захтевају API кључ. Ове методе:
- Захтевају ваш FastComments API кључ
- Треба их позивати ИСКЉУЧИВО из кода на страни сервера
- Омогућавају пун приступ вашим FastComments подацима
- Ограничене су по броју захтева по тенанту

**Пример употребе**: Административне операције, масовни извоз података, управљање корисницима

### ModerationAPI - Методе табле модератора

`ModerationAPI` пружа обиман сет живих и брзих API‑ја за модерацију. Сваки метод `ModerationAPI` прихвата параметар `sso` и може се аутентификовати преко SSO или FastComments.com колачића сесије.

**Пример употребе**: Изградња искуства модерације за модераторе ваше заједнице

**ВАЖНО**: Никада не откривајте ваш API кључ у коду на страни клијента. API кључеви треба да се користе само на страни сервера.
