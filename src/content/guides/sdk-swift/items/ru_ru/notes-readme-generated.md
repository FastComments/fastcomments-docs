### Идентификаторы трансляций

Вы увидите, что в некоторых вызовах API нужно передавать `broadcastId`. Когда вы получаете события, вы получите этот идентификатор обратно, поэтому вы сможете игнорировать событие, если планируете оптимистично применить изменения на клиенте (что, вероятно, стоит сделать, поскольку это обеспечивает наилучший опыт). Передайте здесь UUID. Идентификатор должен быть достаточно уникальным, чтобы не повториться дважды в одной сессии.

```swift
let broadcastId = UUID().uuidString
```
