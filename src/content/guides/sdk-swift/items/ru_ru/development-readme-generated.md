### Перегенерация клиента

Чтобы заново сгенерировать клиент API из последней спецификации OpenAPI:

1. Убедитесь, что сервер FastComments запущен локально на `http://localhost:3001`
2. Выполните скрипт обновления:

```bash
./update.sh
```

Это выполнит:
- Загрузит последнюю спецификацию OpenAPI
- Сгенерирует код Swift-клиента (с документацией API в client/docs)
- Соберёт пакет, чтобы убедиться, что всё работает
