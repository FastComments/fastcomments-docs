### 401 Unauthorized Errors

Если вы получаете ошибки 401 при использовании аутентифицированного API:

1. **Проверьте ваш API key**: Убедитесь, что вы используете правильный API key из панели управления FastComments
2. **Проверьте tenant ID**: Убедитесь, что tenant ID соответствует вашей учетной записи
3. **API key format**: API key должен быть установлен на API client:

```swift
let defaultApi = DefaultAPI()
defaultApi.apiKey = "YOUR_API_KEY"
```

4. **Использование неправильного API**: Убедитесь, что вы используете `DefaultAPI` (а не `PublicAPI`) для аутентифицированных вызовов

### SSO Token Issues

Если SSO-токены не работают:

1. **Используйте secure режим для production**: Всегда используйте `FastCommentsSSO.createSecure()` с вашим API key в production
2. **Только на стороне сервера**: Генерируйте secure SSO-токены на вашем сервере, никогда не раскрывайте ваш API key клиентам
3. **Проверьте данные пользователя**: Убедитесь, что все обязательные поля (id, email, username) предоставлены
4. **Истечение токена**: Защищённые SSO-токены содержат метку времени и могут истечь. Генерируйте новые токены по мере необходимости.

### SSL/TLS Errors

Если вы сталкиваетесь с ошибками SSL/TLS:

1. Убедитесь, что Info.plist вашего приложения позволяет HTTPS-подключения к fastcomments.com
2. Проверьте, что вы не используете исключения App Transport Security, которые могут блокировать подключение
