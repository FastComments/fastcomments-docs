Собирает документы, группируя их (если указано `groupBy`), и применяя несколько операций.  
Поддерживаются различные операции (например, `sum`, `countDistinct`, `avg` и т.д.).

## Параметры

| Имя | Тип | Местоположение | Обязательно | Описание |
|------|------|----------|----------|-------------|
| tenantId | string | query | Yes |  |
| parentTenantId | string | query | No |  |
| includeStats | boolean | query | No |  |

## Ответ

Returns: [`AggregateResponse`](https://github.com/FastComments/fastcomments-swift/blob/main/client/FastCommentsSwift/Models/AggregateResponse.swift)
