FastComments SDK предоставя три API клиента:

### PublicAPI - Методи, безопасни за клиент

`PublicAPI` съдържа методи, които са безопасни за извикване от клиентски код (iOS/macOS приложения). Тези методи:
- Не изискват API ключ
- Могат да използват SSO токени за автентикация
- Имат ограничение на честотата за потребител/устройство
- Подходящи са за приложения, ориентирани към краен потребител

**Пример за използване**: Извличане и създаване на коментари във вашето iOS приложение

### DefaultAPI - Методи за сървърна страна

`DefaultAPI` съдържа автентицирани методи, които изискват API ключ. Тези методи:
- Изискват вашия FastComments API ключ
- Трябва да се извикват САМО от сървърен код
- Предоставят пълен достъп до вашите FastComments данни
- Имат ограничение на честотата за всеки наемател

**Пример за използване**: Административни операции, масов износ на данни, управление на потребители

### ModerationAPI - Методи за таблото на модератора

`ModerationAPI` предоставя обширен набор от живи и бързи API за модериране. Всеки метод на `ModerationAPI` приема параметър `sso` и може да се автентицира чрез SSO или с FastComments.com сесийна бисквитка.

**Пример за използване**: Създаване на опит за модериране за модераторите на вашата общност

**ВАЖНО**: Никога не разкривайте вашия API ключ в клиентски код. API ключовете трябва да се използват само от страна на сървъра.
