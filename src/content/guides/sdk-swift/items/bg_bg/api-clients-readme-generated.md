FastComments SDK предоставя три API клиента:

### PublicAPI - Методи, безопасни за клиентската страна

The `PublicAPI` contains methods that are safe to call from client-side code (iOS/macOS apps). These methods:
- Не изискват API ключ
- Могат да използват SSO токени за удостоверяване
- Имат ограничение на честотата на заявките на потребител/устройство
- Подходящи са за приложения, насочени към крайни потребители

**Пример за използване**: Извличане и създаване на коментари във вашето iOS приложение

### DefaultAPI - Сървърни методи

The `DefaultAPI` contains authenticated methods that require an API key. These methods:
- Изискват вашия FastComments API ключ
- Трябва да се извикват само от сървърен код
- Осигуряват пълен достъп до вашите данни във FastComments
- Имат ограничение на честотата за наемател

**Пример за използване**: Административни операции, експортиране на големи обеми данни, управление на потребители

### ModerationAPI - Методи за таблото на модератора

The `ModerationAPI` contains methods that power the moderator dashboard. These methods cover:
- **Модериране на коментари** - изброяване, броене, търсене, извличане на логове и експортиране на коментари
- **Действия по модерация** - премахване/възстановяване на коментари, маркиране (flag), задаване на статут преглед/спам/одобрение, управление на гласове и преотваряне/затваряне на нишки
- **Блокирания** - блокиране на потребител от коментар, отмяна на блокировки, извличане на предварителни обобщения преди блокиране, проверка на статуса на блокиране и предпочитания, и четене на брой блокирани потребители
- **Баджове и доверие** - присъждане/премахване на баджове, изброяване на ръчни баджове, получаване/задаване на фактор на доверие на потребител и четене на вътрешния профил на потребителя

Every `ModerationAPI` method accepts an `sso` parameter so moderators can be authenticated via SSO.

**Пример за използване**: Изграждане на модерационен интерфейс за модераторите на вашата общност

**ВАЖНО**: Никога не излагайте вашия API ключ в клиентски код. API ключовете следва да се използват само на сървъра.
