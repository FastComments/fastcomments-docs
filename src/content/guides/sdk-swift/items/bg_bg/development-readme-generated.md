### Прегенериране на клиента

За да прегенерирате API клиента от последната OpenAPI спецификация:

1. Уверете се, че FastComments сървърът работи локално на `http://localhost:3001`
2. Стартирайте скрипта за обновяване:

```bash
./update.sh
```

Това ще:
- Ще изтегли най-новата OpenAPI спецификация
- Ще генерира Swift клиентски код (с API документация в client/docs)
- Ще изгради пакета, за да провери, че всичко работи
