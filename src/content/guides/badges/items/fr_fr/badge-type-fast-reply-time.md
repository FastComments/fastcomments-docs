La progression du badge `Fast Reply Time` est acquise lorsqu'un utilisateur répond rapidement à un autre utilisateur. Cela peut être utilisé pour encourager les membres
de votre communauté à aider rapidement d'autres membres, entre autres. Seuls les commentaires approuvés sont pris en compte.

Pour que la progression vers ce badge soit prise en compte, la réponse doit être faite dans les 40 secondes suivant le message parent (FastComments est en direct, après tout !). Cela n'est pas configurable
aujourd'hui mais peut l'être via une demande au support.

Remarque : la suppression de commentaires ne retirera pas la progression vers ce badge.
