Das Abzeichen `Reply to Specific Page` kann so konfiguriert werden, dass es vergeben wird, wenn ein Benutzer auf einer bestimmten Seite kommentiert.

Zum Beispiel, wenn Sie eine
`FAQ`-Seite haben, möchten Sie Benutzer dafür belohnen, dass sie hier Fragen beantworten. Es basiert auf einer konfigurierbaren Schwelle, zum Beispiel möchten Sie
dieses Abzeichen erst vergeben, nachdem sie 10 Kommentare auf der genannten Seite hinterlassen haben.
