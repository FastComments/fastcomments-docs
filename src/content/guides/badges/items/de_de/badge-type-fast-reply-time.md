Der Fortschritt für das Abzeichen `Fast Reply Time` wird erzielt, wenn ein Benutzer schnell auf einen anderen Benutzer antwortet. Dies kann unter anderem dazu verwendet werden, Mitglieder
Ihrer Community dazu zu ermutigen, anderen Mitgliedern schnell zu helfen. Nur genehmigte Kommentare werden gezählt.

Um den Fortschritt für dieses Abzeichen zu zählen, muss die Antwort innerhalb von 40 Sekunden nach dem Elternkommentar erfolgen (FastComments ist schließlich live!). Dies ist derzeit nicht konfigurierbar
kann jedoch möglicherweise über eine Support-Anfrage aktiviert werden.

Hinweis: Durch das Entfernen von Kommentaren wird der Fortschritt für dieses Abzeichen nicht entfernt.
