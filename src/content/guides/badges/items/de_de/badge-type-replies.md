Das `Replies`-Abzeichen wird basierend auf der Anzahl der Antworten, die ein Benutzer erhält, verliehen. Dies dient dazu, die Interaktion zu fördern - wenn ein Benutzer Antworten auf seine Kommentare erhält, 
bringt er Nutzer dazu, Zeit auf Ihrer Website zu verbringen, daher sollten Sie ihn belohnen und dieses Verhalten fördern.

Hinweis: Das Löschen von Kommentaren entfernt dieses Abzeichen nicht.
