Τα Διακριτικά Χρηστών του FastComments ρυθμίζονται από διαχειριστές που έχουν την άδεια `Customize Data`.

Αυτό γίνεται μέσω του [Customize -> Badges](https://fastcomments.com/auth/my-account/configure-badges) στο ταμπλό διαχείρισης.

Όταν σε έναν χρήστη απονέμεται ένα διακριτικό, εμφανίζεται στο προφίλ του και στα σχόλιά του.

Κατά την προσθήκη ενός διακριτικού μπορούμε να ορίσουμε ένα `Display Label`, που είναι το όνομα που βλέπει ο χρήστης συσχετισμένο με το διακριτικό. Για παράδειγμα, αν προσθέσουμε ένα `Comment Count` διακριτικό μάλλον δεν θέλουμε να δείξουμε αυτό το τεχνικό όνομα γιατί είναι πολύ άχαρο. Θα μπορούσαμε να το ονομάσουμε `Super Member` ή κάτι παρόμοιο. Τα διακριτικά μπορούν επίσης να στοίβονται και να αντικαθιστούν το ένα το άλλο, όπως θα καλύψουμε αργότερα σε αυτό το έγγραφο.

Τα διακριτικά έχουν επίσης ρυθμιζόμενα όρια.

Τα διακριτικά μπορούν να δημιουργηθούν, και στη συνέχεια να απενεργοποιηθούν αποεπιλέγοντας το `Enabled`. Η απενεργοποίηση ενός διακριτικού σημαίνει ότι δεν θα απονέμεται πλέον αυτόματα και δεν θα εμφανίζεται στο μενού Award Manual Badge, αλλά οι χρήστες θα διατηρήσουν το διακριτικό.

### Τύποι Εμφάνισης Διακριτικών

Τα διακριτικά μπορούν να είναι εικόνες ή διακριτικά κειμένου, τα οποία υποστηρίζουν κάποιες βασικές μορφοποιήσεις (χρώμα κειμένου, χρώμα φόντου και χρώμα περιγράμματος). Μπορείτε επίσης να μορφοποιήσετε τα διακριτικά μέσω CSS.

Τα διακριτικά εικόνας μπορούν να είναι GIF για να εμφανίζουν κινούμενα διακριτικά.

### Συμβουλή - Μην Αφαιρείτε Διακριτικά!

Οι χρήστες αγαπούν τα διακριτικά. Τείνουν να τα νοιάζονται πολύ, ακόμη και αν είναι ένα λάθος που προσθέσατε κατά λάθος και θέλετε να αλλάξετε το εικονίδιο του διακριτικού.

Αν μάθαμε κάτι, είναι ότι είναι εξαιρετικά δύσκολο να αφαιρέσεις κάτι από τους χρήστες. Η αφαίρεση ενός διακριτικού επειδή εσείς ως ιδιοκτήτης του ιστότοπου δεν το συμπαθείτε πλέον, ή θέλετε να κάνετε αλλαγές, μπορεί να έχει ως αποτέλεσμα ένα πολύ οργισμένο πλήθος χρηστών που αιφνιδίως φεύγουν από τον ιστότοπό σας από απογοήτευση. Για αυτό το λόγο το `Delete` δεν ήταν καν επιλογή τους πρώτους μήνες που κυκλοφόρησε αυτή η λειτουργία - ωστόσο τελικά αναγκαστήκαμε να το προσθέσουμε. Αλλά παρακαλούμε, να χρησιμοποιείτε τη διαγραφή με προσοχή. Έχουμε δει πολλούς μακροχρόνιους, πολυετούς χρήστες να εκνευρίζονται πολύ και να εγκαταλείπουν τις κοινότητές τους επειδή οι διαχειριστές αποφάσισαν να διαγράψουν ένα διακριτικό.

Αν πρέπει οπωσδήποτε να σταματήσετε να χρησιμοποιείτε ένα διακριτικό, προτείνουμε απλώς να το απενεργοποιήσετε ώστε οι χρήστες να διατηρήσουν το διακριτικό τους.

### Επαναεπεξεργασία Διακριτικών

Όταν ένα διακριτικό προστεθεί ή αλλάξει, το σύστημα θα ελέγξει αναδρομικά όσους αλληλεπίδρασαν με τον ιστότοπό σας για να δει αν θα πρέπει να λάβουν το διακριτικό. Αυτό θα είναι ορατό στη σελίδα Διακριτικών στο ταμπλό διαχείρισης, καθώς θα εμφανιστεί ένας περιστρεφόμενος δείκτης (spinner) αντί για τον αριθμό των χρηστών που έχουν το διακριτικό. Αυτό συμβαίνει επειδή ο αριθμός των χρηστών προσδιορίζεται.

### Πώς να Δείτε Ποιος Έχει Ένα Διακριτικό

Στη λίστα Διακριτικών κάθε σύνδεσμος έχει την επιλογή `View Users` για να εμφανίσει τη λίστα των χρηστών που έχουν κερδίσει ή χειροκίνητα τους έχει απονεμηθεί ένα διακριτικό.
