Прогресс для значка `Fast Reply Time` начисляется, когда пользователь быстро отвечает другому пользователю. Это можно использовать, чтобы побудить участников
вашего сообщества быстро помогать другим участникам, среди прочего. В расчёт идут только одобренные комментарии.

Чтобы учитывался прогресс для этого значка, ответ должен быть дан в течение 40 секунд от родительского комментария (`FastComments is live, after all!`). Это в настоящее время не настраивается
но может быть изменено по запросу в службу поддержки.

Примечание: удаление комментариев не приведёт к удалению прогресса для этого значка.
