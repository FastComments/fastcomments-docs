Τα Σήματα Χρηστών του FastComments διαμορφώνονται από διαχειριστές με την άδεια `Customize Data`.

Αυτό γίνεται μέσω [Customize -> Badges](https://fastcomments.com/auth/my-account/configure-badges) στον πίνακα ελέγχου διαχειριστή σας.

Όταν ένας χρήστης λαμβάνει ένα σήμα, αυτό εμφανίζεται στο προφίλ του και στα σχόλιά του.

Κατά την προσθήκη ενός σήματος μπορούμε να ορίσουμε ένα `Display Label`, το οποίο είναι το όνομα που βλέπει ο χρήστης συνδεδεμένο με το σήμα. Για παράδειγμα, αν προσθέσουμε ένα σήμα `Comment Count`
πιθανότατα δεν θέλουμε να εμφανίσουμε αυτό το τεχνικό όνομα επειδή είναι πολύ φτωχό. Μπορούμε να το ονομάσουμε `Super Member` ή κάτι παρόμοιο. Τα σήματα μπορούν επίσης να στοίβονται και να αντικαθιστούν το ένα το άλλο, όπως θα καλύψουμε
αργότερα σε αυτό το έγγραφο.

Τα σήματα έχουν επίσης ρυθμιζόμενα κατώφλια.

Τα σήματα μπορούν να δημιουργηθούν, και στη συνέχεια να απενεργοποιηθούν ξετικάροντας το `Enabled`. Η απενεργοποίηση ενός σήματος σημαίνει ότι δεν θα απονέμεται πλέον αυτόματα, και δεν θα εμφανίζεται στο μενού Award Manual Badge, αλλά
οι χρήστες θα διατηρήσουν το σήμα.

### Τύποι εμφάνισης Σημάτων

Τα σήματα μπορούν να είναι εικόνες ή κείμενο, που υποστηρίζουν κάποια βασικά στυλ (χρώμα κειμένου, χρώμα φόντου και χρώμα περιγράμματος). Μπορείτε επίσης να μορφοποιήσετε τα σήματα μέσω CSS.

Τα σήματα εικόνας μπορούν να είναι εικόνες GIF για να δείχνουν κινούμενα σήματα.

### Συμβουλή - Μην αφαιρείτε τα σήματα!

Οι χρήστες αγαπούν τα σήματα. Τείνουν να τα εκτιμούν πολύ, ακόμη και αν πρόκειται για σφάλμα που προσθέσατε κατά λάθος, και θέλετε να αλλάξετε το εικονίδιο του σήματος.

Αν μάθαμε κάτι, είναι ότι είναι εξαιρετικά δύσκολο να αφαιρέσεις κάτι από τους χρήστες. Η αφαίρεση ενός σήματος επειδή εσείς ως ιδιοκτήτης του site δεν το συμπαθείτε πια, ή θέλετε να κάνετε αλλαγές, μπορεί να έχει ως αποτέλεσμα ένα πολύ εξαγριωμένο πλήθος χρηστών που εγκαταλείπουν ξαφνικά τον ιστότοπό σας από απογοήτευση. Για αυτόν τον λόγο
`Delete` δεν ήταν καν επιλογή για τους πρώτους μερικούς μήνες που κυκλοφόρησε αυτή η λειτουργία - ωστόσο τελικά αναγκαστήκαμε να το προσθέσουμε. Αλλά παρακαλώ, χρησιμοποιήστε τη διαγραφή με προσοχή. Έχουμε
δει πολλούς μακροχρόνιους, πολλών ετών, χρήστες να απογοητεύονται πολύ και να εγκαταλείπουν τις κοινότητές τους επειδή οι διαχειριστές αποφάσισαν να διαγράψουν ένα σήμα.

Αν πρέπει να σταματήσετε να χρησιμοποιείτε ένα σήμα, προτείνουμε απλώς να το απενεργοποιήσετε ώστε οι χρήστες να διατηρήσουν το σήμα.

### Επαναεπεξεργασία Σημάτων

Όταν ένα σήμα προστίθεται ή αλλάζει, το σύστημα θα ελέγξει αναδρομικά οποιονδήποτε έχει αλληλεπιδράσει με τον ιστότοπό σας για να δει αν πρέπει να λάβει το σήμα. Αυτό θα είναι ορατό στη σελίδα Badges στον πίνακα ελέγχου διαχειριστή, καθώς θα εμφανίζεται ένας δείκτης φόρτωσης αντί για τον αριθμό των χρηστών που έχουν το σήμα. Αυτό συμβαίνει επειδή ο αριθμός των χρηστών υπολογίζεται.

### Εμφάνιση ποιος έχει ένα σήμα

Στη λίστα Badges κάθε σύνδεσμος έχει μια επιλογή `View Users` για να εμφανίσει τη λίστα χρηστών που κέρδισαν ή τους απονεμήθηκε χειροκίνητα ένα σήμα.
