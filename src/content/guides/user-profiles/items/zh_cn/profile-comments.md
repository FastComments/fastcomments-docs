Profile Comments 允许用户直接在其他用户的个人资料页面上留下消息，创建个人留言墙或留言簿式的互动。

### 什么是个人资料评论？

个人资料评论与常规文章评论不同：
- 它们出现在用户的个人资料页面，而不是内容文章上
- 任何查看该资料的人都能看到它们（除非隐私设置限制）
- 它们是一种识别、感谢或与特定用户交流的方式
- 它们适用于所有 FastComments 社区

### 留下个人资料评论

要在某人的资料上发表评论：

1. 访问该用户的个人资料页面
2. 导航到“个人资料评论”选项卡
3. 使用评论框撰写你的消息
4. 提交评论

**注意：** 个人资料所有者的隐私设置决定你的评论是立即显示还是需要审批。

### 查看个人资料评论

**在你自己的资料上：**
1. 访问你的个人资料页面
2. 点击“个人资料评论”选项卡
3. 查看其他人在你的资料上留下的所有评论

**在其他人的资料上：**
如果用户没有启用“将个人资料评论设为私有”设置，则可以看到个人资料评论。

### 评论审批

根据个人资料所有者的设置（参见 [隐私设置](/guide-user-profiles.html#privacy-settings)），评论可能会：

1. **立即显示** - 根据信任等级或审批设置自动批准
2. **需要审批** - 暂存等待个人资料所有者审核

当你的评论需要审批时：
- 你会看到待处理状态指示
- 个人资料所有者会收到通知
- 一旦被批准，评论对其他人可见

### 在你的资料上管理个人资料评论

如果你已配置评论审批（手动批准全部或手动批准新评论），你可以管理收到的评论：

**通过电子邮件：**
你会收到带有批准/拒绝链接的电子邮件通知，用于待处理评论。点击链接即可直接从电子邮件中采取操作。

**从你的资料：**
1. 访问你资料的“个人资料评论”选项卡
2. 查看待处理评论
3. 批准或拒绝每条评论

**来自通知：**
从你的通知中心访问待处理评论以便快速审查。

### 回复个人资料评论

当有人在你的资料上发表评论时，你可以：
- 直接回复他们的评论（创建对话线程）
- 访问他们的资料并回帖
- 发送私信给他们（如果他们允许私信）

### 删除个人资料评论

**在你自己的资料上：**
你可以删除任何留在你资料上的评论：
1. 找到你想删除的评论
2. 点击删除/移除选项
3. 确认删除

**在别人的资料上：**
你可以删除你在他人资料上留下的评论，但不能删除他人的评论（除非你是该资料的所有者）。

### 个人资料评论通知

当发生以下情况时，你会收到通知：
- 有人在你的资料上留言
- 有人在你参与的个人资料评论线程中回复
- 你留下的个人资料评论被批准或收到回复

在你的 [账户设置](https://fastcomments.com/auth/my-account/edit-notifications) 中管理通知偏好。

### 隐私与个人资料评论

使用以下隐私设置控制个人资料评论的可见性：

**将个人资料评论设为私有：**
- 隐藏访问者的“个人资料评论”选项卡
- 只有你能看到留在你资料上的评论
- 阻止发布新评论

**评论审批模式：**
- 选择谁可以在无需审批的情况下发表评论
- 为受信任用户与新用户设置不同的审批规则
- 对所有评论要求手动审批

更多详情，请参见 [隐私设置](/guide-user-profiles.html#privacy-settings)。

### 个人资料评论的使用场景

个人资料评论非常适合于：

- **感谢有帮助的社区成员**
- **表彰有价值的贡献**
- **与活跃参与者建立关系**
- **留下推荐或背书**
- **创建社交留言簿体验**
- **社区认可与赞赏**

### 最佳实践

**在发表评论时：**
- 保持尊重和建设性
- 提供有价值的内容，而不仅仅是通用的“不错的个人资料”类信息
- 在适用时提及具体贡献或互动

**管理你的资料时：**
- 根据你的舒适度设置合适的审批设置
- 回复个人资料评论以鼓励社区参与
- 如果有人反复留下不当评论，请使用屏蔽功能

**为了社区健康：**
- 个人资料评论遵循与文章评论相同的社区指南
- 如果遇到不当的个人资料评论，请举报
- 版主可以对违反规则的个人资料评论采取行动
