FastComments User Profiles are highly customizable, allowing users to express their personality and build their online presence.

### Modification de votre profil

Pour personnaliser votre profil, rendez-vous sur votre page [Détails du compte](https://fastcomments.com/auth/my-account/edit-details). Vous pouvez y modifier tous les aspects de l'apparence et des informations de votre profil.

### Avatar du profil

Votre avatar est l'image de profil qui apparaît à côté de vos commentaires et sur votre page de profil.

**Pour changer votre avatar :**
1. Rendez-vous sur votre page de profil
2. Cliquez sur le lien "Modifier l'avatar" (visible uniquement sur votre propre profil)
3. Téléversez une nouvelle image

**Exigences pour l'avatar :**
- Taille recommandée : 200x200 pixels ou plus
- Formats pris en charge : JPG, PNG, GIF
- Les images carrées sont les plus adaptées

Remarque : les utilisateurs SSO peuvent voir leur avatar contrôlé par le système SSO de leur site.

### Arrière-plan de l'en-tête du profil

L'arrière-plan de l'en-tête est une grande image de bannière en haut de votre profil qui peut être personnalisée pour refléter votre style.

**Pour téléverser un arrière-plan d'en-tête :**
1. Rendez-vous sur votre page de profil
2. Cliquez sur le bouton "Téléverser l'en-tête"
3. Sélectionnez une image depuis votre appareil

**Exigences pour l'en-tête :**
- Taille recommandée : 1200x300 pixels
- Formats pris en charge : JPG, PNG
- Les images au format paysage/large sont les plus adaptées
- Les images sont redimensionnées automatiquement pour s'adapter

Si vous ne téléversez pas d'en-tête personnalisé, un arrière-plan par défaut sera affiché.

### Nom affiché

Votre nom affiché est la façon dont vous apparaîtrez auprès des autres utilisateurs sur FastComments. Contrairement aux noms d'utilisateur (qui doivent être uniques), les noms affichés :

- N'ont pas besoin d'être uniques
- Peuvent contenir des espaces et des caractères spéciaux
- Peuvent inclure des emojis

**Pour définir votre nom affiché :**
1. Allez sur [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Saisissez votre nom affiché préféré
3. Enregistrez les modifications

### Nom d'utilisateur

Votre nom d'utilisateur est un identifiant unique utilisé pour vous connecter à FastComments.

- Doit être unique parmi tous les utilisateurs de FastComments
- Utilisé dans l'URL de votre profil : `fastcomments.com/auth/user-profile/username`
- Ne peut pas contenir d'espaces ni de caractères spéciaux

**Remarque :** les utilisateurs SSO peuvent avoir des noms d'utilisateur non uniques car ils sont liés à un locataire/communauté spécifique.

### Biographie

Ajoutez une biographie personnelle pour raconter aux autres qui vous êtes, vos centres d'intérêt ou ce que vous faites.

**Pour ajouter une biographie :**
1. Allez sur [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Saisissez du texte dans le champ Biographie
3. Enregistrez les modifications

Votre biographie apparaît de manière visible sur votre page de profil et aide les autres à comprendre qui vous êtes.

### URL du site web

Ajoutez un lien vers votre site personnel, blog ou portfolio.

**Pour ajouter un site web :**
1. Allez sur [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Saisissez l'URL de votre site web dans le champ Site web
3. Enregistrez les modifications

Le lien vers votre site web sera affiché sur votre profil avec une icône cliquable.

### Liens vers les réseaux sociaux

Connectez vos profils de réseaux sociaux à votre profil FastComments. Les plateformes prises en charge incluent :

- **Facebook** - Lien vers votre profil Facebook
- **Instagram** - Lien vers votre compte Instagram
- **LinkedIn** - Lien vers votre profil LinkedIn
- **Liens personnalisés** - Ajoutez tout autre profil social ou web

**Pour ajouter des liens sociaux :**
1. Allez sur [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Saisissez les URL de vos comptes de réseaux sociaux
3. Enregistrez les modifications

Les liens sociaux apparaissent sur votre profil avec des icônes de plateformes reconnaissables, facilitant la connexion des membres de la communauté avec vous ailleurs.

### Pays

Vous pouvez facultativement définir votre pays, ce qui affiche une icône de drapeau à côté de votre nom.

**Pour définir votre pays :**
1. Allez sur [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Sélectionnez votre pays dans le menu déroulant
3. Enregistrez les modifications

Le drapeau du pays apparaît à côté de votre nom d'utilisateur sur votre profil.
