Les commentaires de profil permettent aux utilisateurs de laisser des messages directement sur la page de profil d'un autre utilisateur, créant un mur de commentaires personnel ou une interaction de type livre d'or.

### Que sont les commentaires de profil ?

Les commentaires de profil sont différents des commentaires d'article classiques :
- Ils apparaissent sur la page de profil d'un utilisateur, pas sur des articles de contenu
- Ils sont visibles par toute personne consultant le profil (sauf si les paramètres de confidentialité en restreignent l'accès)
- Ils constituent un moyen de reconnaître, remercier ou communiquer avec des utilisateurs spécifiques
- Ils fonctionnent dans toutes les communautés FastComments

### Laisser un commentaire sur un profil

Pour commenter le profil de quelqu'un :

1. Rendez-vous sur la page de profil de l'utilisateur
2. Allez dans l'onglet "Commentaires de profil"
3. Utilisez la zone de commentaire pour rédiger votre message
4. Envoyez le commentaire

**Remarque :** Les paramètres de confidentialité du propriétaire du profil déterminent si votre commentaire apparaît immédiatement ou s'il nécessite une approbation.

### Voir les commentaires de profil

**Sur votre propre profil :**
1. Rendez-vous sur votre page de profil
2. Cliquez sur l'onglet "Commentaires de profil"
3. Voyez tous les commentaires que d'autres ont laissés sur votre profil

**Sur d'autres profils :**
Les commentaires de profil sont visibles si l'utilisateur n'a pas activé le paramètre "Rendre les commentaires de profil privés".

### Approbation des commentaires

Selon les paramètres du propriétaire du profil (voir [Paramètres de confidentialité](/guides/user-profiles/privacy-settings)), les commentaires peuvent :

1. **Apparaître immédiatement** - Auto-approuvés en fonction du niveau de confiance ou des paramètres d'approbation
2. **Nécessiter une approbation** - Mis en attente pour que le propriétaire du profil les consulte

Lorsque votre commentaire nécessite une approbation :
- Vous verrez un indicateur d'état en attente
- Le propriétaire du profil reçoit une notification
- Une fois approuvé, le commentaire devient visible pour les autres

### Gérer les commentaires de profil sur votre profil

Si vous avez configuré l'approbation des commentaires (Approbation manuelle de tous ou Approbation manuelle des nouveaux), vous pouvez gérer les commentaires entrants :

**Par e-mail :**
Vous recevrez des notifications par e-mail avec des liens d'approbation/rejet pour les commentaires en attente. Cliquez sur le lien pour agir directement depuis votre e-mail.

**Depuis votre profil :**
1. Rendez-vous dans l'onglet "Commentaires de profil" de votre profil
2. Voyez les commentaires en attente
3. Approuvez ou rejetez chaque commentaire

**Depuis les notifications :**
Accédez aux commentaires en attente depuis votre centre de notifications pour une revue rapide.

### Répondre aux commentaires de profil

Lorsqu'une personne commente votre profil, vous pouvez :
- Répondre directement à son commentaire (créant un fil de conversation)
- Visiter son profil et laisser un commentaire en retour
- Lui envoyer un message direct (si elle autorise les messages privés)

### Supprimer des commentaires de profil

**Sur votre propre profil :**
Vous pouvez supprimer n'importe quel commentaire laissé sur votre profil :
1. Trouvez le commentaire que vous souhaitez supprimer
2. Cliquez sur l'option supprimer/retirer
3. Confirmez la suppression

**Sur d'autres profils :**
Vous pouvez supprimer vos propres commentaires laissés sur les profils d'autres personnes, mais vous ne pouvez pas supprimer les commentaires d'autrui (sauf si vous êtes le propriétaire du profil).

### Notifications de commentaires de profil

Vous recevrez des notifications lorsque :
- Quelqu'un laisse un commentaire sur votre profil
- Quelqu'un répond à un fil de commentaires de profil auquel vous participez
- Un commentaire de profil que vous avez laissé est approuvé ou reçoit une réponse

Gérez les préférences de notification dans vos [Paramètres du compte](https://fastcomments.com/auth/my-account/edit-notifications).

### Confidentialité et commentaires de profil

Contrôlez la visibilité des commentaires de profil avec ces paramètres de confidentialité :

**Rendre les commentaires de profil privés :**
- Masque l'onglet Commentaires de profil aux visiteurs
- Seul vous pouvez voir les commentaires laissés sur votre profil
- Empêche la publication de nouveaux commentaires

**Modes d'approbation des commentaires :**
- Choisissez qui peut commenter sans approbation
- Définissez des règles d'approbation différentes pour les utilisateurs de confiance et les nouveaux utilisateurs
- Exigez une approbation manuelle pour tous les commentaires

Pour plus de détails, voir [Paramètres de confidentialité](/guides/user-profiles/privacy-settings).

### Cas d'utilisation des commentaires de profil

Les commentaires de profil sont parfaits pour :

- **Remercier des membres utiles de la communauté**
- **Reconnaître des contributions précieuses**
- **Créer des liens avec des participants actifs**
- **Laisser des témoignages ou des recommandations**
- **Créer une expérience de livre d'or social**
- **Appréciation et reconnaissance de la communauté**

### Bonnes pratiques

**Pour laisser des commentaires :**
- Soyez respectueux et constructif
- Apportez de la valeur plutôt que des messages génériques du type « joli profil »
- Faites référence à des contributions ou interactions spécifiques lorsque c'est pertinent

**Pour gérer votre profil :**
- Définissez des paramètres d'approbation appropriés en fonction de votre niveau de confort
- Répondez aux commentaires de profil pour encourager l'engagement de la communauté
- Utilisez le blocage si quelqu'un laisse répété des commentaires inappropriés

**Pour la santé de la communauté :**
- Les commentaires de profil suivent les mêmes règles de la communauté que les commentaires d'article
- Signalez les commentaires de profil inappropriés si vous en rencontrez
- Les modérateurs peuvent agir sur les commentaires de profil qui enfreignent les règles
