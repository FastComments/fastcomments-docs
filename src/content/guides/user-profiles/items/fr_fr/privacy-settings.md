FastComments vous donne un contrôle granulaire sur la confidentialité de votre profil et sur qui peut interagir avec vous. Tous les paramètres de confidentialité peuvent être gérés depuis la section Préférences de confidentialité de la page de votre profil.

### Accéder aux paramètres de confidentialité

Pour gérer vos paramètres de confidentialité :
1. Rendez-vous sur votre page de profil
2. Cliquez sur l'onglet ou la section "Préférences de confidentialité"
3. Ajustez vos paramètres selon vos préférences
4. Les modifications sont enregistrées automatiquement

### Visibilité de l'activité

Contrôlez qui peut voir votre activité de commentaires à travers les communautés FastComments.

**Paramètre :** Rendre l'activité du profil privée

- **Lorsqu'il est désactivé (par défaut) :** Tout le monde peut voir vos commentaires récents et votre activité sur votre profil
- **Lorsqu'il est activé :** Seul vous pouvez voir votre historique de commentaires et votre fil d'activité

Ceci est utile si vous souhaitez participer aux discussions mais préférez garder votre historique de commentaires privé vis-à-vis des autres utilisateurs.

### Visibilité des commentaires sur le profil

Contrôlez si les autres peuvent voir et laisser des commentaires sur votre page de profil.

**Paramètre :** Rendre les commentaires du profil privés

- **Lorsqu'il est désactivé (par défaut) :** Tout le monde peut voir et laisser des commentaires sur votre profil
- **Lorsqu'il est activé :** Les commentaires du profil sont cachés aux autres ; seul vous pouvez voir les commentaires laissés sur votre profil

Utilisez ce paramètre si vous souhaitez limiter les interactions sur votre page de profil tout en permettant l'activité ailleurs.

### Messages privés

Contrôlez si d'autres utilisateurs peuvent vous envoyer des messages privés.

**Paramètre :** Désactiver les messages privés

- **Lorsqu'il est désactivé (par défaut) :** Les autres utilisateurs peuvent vous envoyer des messages privés
- **Lorsqu'il est activé :** L'option Message privé est cachée depuis votre profil ; les utilisateurs ne peuvent pas initier de conversation avec vous

Ceci est utile si vous préférez ne pas recevoir de messages privés ou si vous souhaitez limiter les conversations en tête-à-tête.

### Approbation des commentaires du profil

Contrôlez la manière dont les commentaires sur votre profil sont modérés avant d'apparaître publiquement.

**Paramètre :** Mode d'approbation des commentaires du profil

Vous avez quatre options pour gérer les commentaires sur le profil :

#### 1. Approuver automatiquement tout
Tous les commentaires publiés sur votre profil sont immédiatement visibles sans nécessiter d'approbation.

**Idéal pour :** Les utilisateurs qui veulent une section de commentaires ouverte et sans restriction.

#### 2. Approuver automatiquement les utilisateurs de confiance (par défaut)
Les commentaires des utilisateurs de confiance sont automatiquement approuvés, tandis que les commentaires des utilisateurs nouveaux ou non fiables nécessitent une approbation manuelle.

**Idéal pour :** La plupart des utilisateurs - équilibre ouverture et protection contre le spam ou le contenu indésirable.

**Ce qui rend un utilisateur "de confiance" :**
- Utilisateurs avec un bon karma
- Utilisateurs avec un historique de commentaires établi
- Utilisateurs qui n'ont pas été signalés pour spam ou violations

#### 3. Approuver manuellement tout
Tous les commentaires nécessitent votre approbation manuelle avant d'apparaître sur votre profil, quel que soit leur auteur.

**Idéal pour :** Les utilisateurs qui veulent un contrôle complet sur ce qui apparaît sur leur profil.

**Comment fonctionne l'approbation :**
- Les nouveaux commentaires sont mis en attente pour révision
- Vous recevez une notification lorsqu'une personne commente
- Vous pouvez approuver ou rejeter les commentaires par e-mail ou depuis votre profil

#### 4. Approuver manuellement les nouveaux
Seuls les commentaires des utilisateurs nouveaux ou non fiables nécessitent une approbation. Les commentaires des membres établis de la communauté sont approuvés automatiquement.

**Idéal pour :** Les utilisateurs qui veulent filtrer les nouveaux utilisateurs tout en permettant aux membres établis de la communauté de commenter librement.

### Gestion des commentaires en attente sur le profil

Lorsque des commentaires nécessitent une approbation, vous recevrez des notifications et pourrez les gérer de plusieurs façons :

1. **Via e-mail :** Approuvez ou rejetez directement depuis l'e-mail de notification
2. **Depuis votre profil :** Consultez les commentaires en attente dans l'onglet Commentaires du profil
3. **Centre de notifications :** Accédez aux commentaires en attente depuis vos notifications

### Bloquer des utilisateurs

En plus des paramètres de confidentialité du profil, vous pouvez bloquer des utilisateurs spécifiques pour qu'ils n'interagissent pas avec vous n'importe où sur FastComments :

- Les utilisateurs bloqués ne peuvent pas commenter sur votre profil
- Les utilisateurs bloqués ne peuvent pas vous envoyer de messages privés
- Les commentaires des utilisateurs bloqués sont cachés de votre vue
- Vous ne verrez pas l'activité de l'autre

Pour bloquer un utilisateur, rendez-vous sur son profil et cliquez sur l'option "Bloquer l'utilisateur".

### Bonnes pratiques en matière de confidentialité

Voici quelques configurations de confidentialité recommandées pour différents cas d'utilisation :

**Membre d'une communauté publique :**
- Visibilité de l'activité : Publique
- Commentaires du profil : Publics
- Messages privés : Activés
- Approbation des commentaires : Approuver automatiquement les utilisateurs de confiance

**Utilisateur privé :**
- Visibilité de l'activité : Privée
- Commentaires du profil : Privés
- Messages privés : Désactivés
- Approbation des commentaires : Approuver manuellement tout

**Modérateur / Personnalité publique :**
- Visibilité de l'activité : Publique
- Commentaires du profil : Publics
- Messages privés : Activés (ou désactivés si trop sollicités)
- Approbation des commentaires : Approuver automatiquement les utilisateurs de confiance ou Approuver manuellement les nouveaux
