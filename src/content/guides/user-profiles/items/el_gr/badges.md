Τα διακριτικά είναι οπτικά επιτεύγματα που εμφανίζονται στο Προφίλ χρήστη σας, αναγνωρίζοντας τις συνεισφορές και τη δραστηριότητά σας σε κοινότητες του FastComments.

### Τι είναι τα Διακριτικά;

Τα διακριτικά είναι αναγνωρίσεις που κερδίζονται για διάφορες ενέργειες όπως:
- Αριθμός σχολίων που αναρτήθηκαν
- Ψηφίσματα υπέρ που λήφθηκαν
- Απαντήσεις που δημιουργήθηκαν
- Ενέργειες συντονιστή
- Χρόνος ως μέλος της κοινότητας
- Ειδικά επιτεύγματα

Κάθε διακριτικό εμφανίζεται στο προφίλ σας και μπορεί προαιρετικά να εμφανίζεται δίπλα στο όνομά σας στα σχόλια.

### Προβολή των Διακριτικών σας

Τα διακριτικά που έχετε κερδίσει εμφανίζονται στη σελίδα προφίλ σας στην ενότητα κεφαλίδας. Υπάρχουν δύο κατηγορίες:

1. **Αποκτηθέντα Διακριτικά** - Διακριτικά που έχετε πλήρως κερδίσει και ξεκλειδώσει
2. **Σε Εξέλιξη** - Διακριτικά που εργάζεστε για να αποκτήσετε, με μπάρες προόδου που δείχνουν πόσο κοντά είστε

### Εμφάνιση Διακριτικών

Έχετε πλήρη έλεγχο σχετικά με το πώς εμφανίζονται τα διακριτικά:

**Στο Προφίλ σας:**
Όλα τα αποκτηθέντα διακριτικά εμφανίζονται στην κεφαλίδα του προφίλ σας για να τα βλέπουν οι άλλοι.

**Στα σχόλιά σας:**
Μπορείτε να επιλέξετε ποια διακριτικά εμφανίζονται δίπλα στο όνομά σας όταν σχολιάζετε:

1. Επισκεφτείτε τη σελίδα προφίλ σας
2. Κάντε κλικ στο "Διαχείριση Διακριτικών" ή ανοίξτε το παράθυρο διαχείρισης διακριτικών
3. Εναλλάξτε ποια διακριτικά θα εμφανίζονται στα σχόλιά σας
4. Αναδιατάξτε τα διακριτικά σύροντάς τα για να ορίσετε την προτεραιότητα εμφάνισης

Αυτό σας επιτρέπει να επιδεικνύετε τα πιο σημαντικά σας επιτεύγματα χωρίς να γεμίζει η υπογραφή του σχολίου σας.

### Τύποι Διακριτικών

Τα διακριτικά υπάρχουν σε διάφορους τύπους με βάση διαφορετικές κατηγορίες επιτευγμάτων. Για μια πλήρη επισκόπηση όλων των διαθέσιμων τύπων διακριτικών και του τρόπου απόκτησής τους, δείτε τον [Οδηγό Διακριτικών](/guides/badges).

Συνηθισμένες κατηγορίες διακριτικών περιλαμβάνουν:

- **Διακριτικά Συμμετοχής** - Βασίζονται στον αριθμό σχολίων, τις απαντήσεις και την αλληλεπίδραση
- **Διακριτικά Ποιότητας** - Βασίζονται σε ψήφους υπέρ και την αποδοχή της κοινότητας
- **Διακριτικά βάσει χρόνου** - Κατάσταση βετεράνου, δραστηριότητα νυχτερινού τύπου
- **Διακριτικά Συντονιστή** - Αναγνώριση για ενέργειες συντονισμού
- **Ειδικά Διακριτικά** - Χορηγούνται χειροκίνητα ή είναι ειδικά για την κοινότητα

### Διακριτικά Συγκεκριμένης Κοινότητας

Τα διακριτικά μπορούν να αποκτηθούν σε διάφορες κοινότητες του FastComments:

- **Παγκόσμια Διακριτικά** - Κερδίζονται σε όλη τη χρήση του FastComments
- **Διακριτικά Κοινότητας** - Κερδίζονται εντός συγκεκριμένων ιστότοπων/ενοικιαστών

Το προφίλ σας δείχνει ποια κοινότητα απένειμε κάθε διακριτικό, βοηθώντας τους επισκέπτες να κατανοήσουν τους τομείς εξειδίκευσης και συμμετοχής σας.

### Παρακολούθηση Προόδου Διακριτικών

Για διακριτικά που δεν έχετε κερδίσει πλήρως ακόμα, το προφίλ σας εμφανίζει:

- **Μπάρα Προόδου** - Οπτικός δείκτης πόσο κοντά είστε στο να κερδίσετε το διακριτικό
- **Απαιτήσεις** - Τι πρέπει να κάνετε για να ξεκλειδώσετε το διακριτικό
- **Πλαίσιο Κοινότητας** - Ποια κοινότητα σχετίζεται με το διακριτικό

Αυτό ενθαρρύνει τη συνεχή συμμετοχή και σας δίνει σαφείς στόχους προς επίτευξη.

### Προσαρμογή Εμφάνισης Διακριτικών

**Σειρά Διακριτικών:**
Μπορείτε να αναδιατάξετε τα διακριτικά για να ελέγξετε ποια εμφανίζονται πρώτα. Η σειρά που ορίζετε καθορίζει:
- Την προτεραιότητα στην εμφάνιση στο προφίλ
- Ποια διακριτικά εμφανίζονται σε καταστάσεις περιορισμένου χώρου
- Τις προσωπικές προτιμήσεις για την ανάδειξη επιτευγμάτων

**Ορατότητα:**
Ενεργοποιήστε ή απενεργοποιήστε μεμονωμένα διακριτικά για την εμφάνιση στα σχόλια ενώ τα διατηρείτε ορατά στο προφίλ σας.

### Χειροκίνητα Διακριτικά

Οι διαχειριστές κοινότητας μπορούν να απονείμουν ειδικά χειροκίνητα διακριτικά για να αναγνωρίσουν:
- Ειδικές συνεισφορές
- Νικητές διαγωνισμών
- VIP μέλη
- Προσαρμοσμένα επιτεύγματα

Αυτά τα διακριτικά εμφανίζονται μαζί με τα αυτόματα κερδισμένα διακριτικά στο προφίλ σας.

### Για περισσότερες πληροφορίες

Για πλήρεις λεπτομέρειες σχετικά με το σύστημα διακριτικών, συμπεριλαμβανομένων όλων των τύπων διακριτικών, των κριτηρίων απόκτησης και της διοικητικής διαμόρφωσης, δείτε τον πλήρη [Οδηγό Διακριτικών](/guides/badges).
