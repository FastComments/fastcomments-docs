Άμεσα Μηνύματα (DM) επιτρέπουν στους χρήστες του FastComments να έχουν ιδιωτικές, κατ’ ιδίαν συνομιλίες μεταξύ τους, ξεχωριστές από τα δημόσια σχόλια και τις αλληλεπιδράσεις στο προφίλ.

### Τι είναι τα Άμεσα Μηνύματα;

Τα Άμεσα Μηνύματα παρέχουν:
- **Ιδιωτικές συνομιλίες** μεταξύ δύο χρηστών
- **Ανταλλαγή μηνυμάτων σε πραγματικό χρόνο** με ειδοποιήσεις
- **Ιστορικό συνομιλιών** που διατηρείται μεταξύ των συνεδριών
- **Δείκτες διαδικτυακής κατάστασης** για να δείτε πότε κάποιος είναι διαθέσιμος
- **Παρακολούθηση μη αναγνωσμένων μηνυμάτων** για να μένετε ενημερωμένοι για νέα μηνύματα

### Ξεκινώντας ένα Άμεσο Μήνυμα

Υπάρχουν διάφοροι τρόποι για να ξεκινήσετε μια συνομιλία DM:

**Από το Προφίλ Χρήστη:**
1. Επισκεφτείτε τη σελίδα προφίλ του χρήστη
2. Κάντε κλικ στην καρτέλα "Άμεσα Μηνύματα" ή το κουμπί "Αποστολή Μηνύματος"
3. Πληκτρολογήστε το μήνυμά σας και στείλτε το

**Από ένα Σχόλιο:**
Ορισμένες υλοποιήσεις επιτρέπουν το κλικ στο όνομα ή το avatar ενός χρήστη για γρήγορη πρόσβαση στο προφίλ και τις επιλογές μηνυμάτων του.

**Άμεσος Σύνδεσμος:**
Εάν έχετε ένα ID συνομιλίας ή σύνδεσμο ανακατεύθυνσης DM, μπορείτε να έχετε πρόσβαση στις συνομιλίες απευθείας.

### Πρόσβαση στα Μηνύματά σας

Για να δείτε όλες τις συνομιλίες άμεσων μηνυμάτων σας:

1. Επισκεφτείτε τη σελίδα προφίλ σας
2. Κάντε κλικ στην καρτέλα "Άμεσα Μηνύματα"
3. Δείτε μια λίστα με όλες τις συνομιλίες σας

Κάθε συνομιλία εμφανίζει:
- Το avatar και το όνομα του άλλου συμμετέχοντα
- Την κατάσταση σύνδεσης/離σύνδεσης τους (πράσινος δείκτης όταν είναι online)
- Προεπισκόπηση του τελευταίου μηνύματος
- Αριθμό μη αναγνωσμένων μηνυμάτων (αν υπάρχουν)
- Χρονική σήμανση της τελευταίας δραστηριότητας

### Προβολή Συνομιλίας

Όταν ανοίγετε μια συνομιλία, θα δείτε:

- **Ολόκληρο το ιστορικό μηνυμάτων** - Όλα τα μηνύματα μεταξύ εσάς και του άλλου χρήστη
- **Ενημερώσεις σε πραγματικό χρόνο** - Τα νέα μηνύματα εμφανίζονται αμέσως μέσω WebSocket
- **Κατάσταση σύνδεσης** - Δείτε αν το άλλο άτομο είναι αυτή τη στιγμή online
- **Χρονικές σημάνσεις μηνυμάτων** - Πότε στάλθηκε κάθε μήνυμα
- **Σύνθεση μηνύματος** - Πλαίσιο κειμένου για να πληκτρολογήσετε και να στείλετε νέα μηνύματα

### Ειδοποιήσεις Μηνυμάτων

Θα λαμβάνετε ειδοποιήσεις για νέα άμεσα μηνύματα:

- **Ειδοποιήσεις εντός εφαρμογής** - Δείκτης μετρητή στο προφίλ σας
- **Ειδοποιήσεις μέσω email** - Ανάλογα με τις ρυθμίσεις ειδοποιήσεων σας
- **Ειδοποιήσεις σε πραγματικό χρόνο** - Άμεσες ειδοποιήσεις όταν είστε συνδεδεμένοι

Διαχειριστείτε τις προτιμήσεις ειδοποιήσεών σας στις [Ρυθμίσεις λογαριασμού](https://fastcomments.com/auth/my-account/edit-notifications).

### Απόρρητο και Αποκλεισμός

**Απενεργοποίηση Άμεσων Μηνυμάτων:**
Αν δεν θέλετε να λαμβάνετε DM, μπορείτε να τα απενεργοποιήσετε εντελώς:
1. Μεταβείτε στις Ρυθμίσεις Απορρήτου στο προφίλ σας
2. Ενεργοποιήστε το "Απενεργοποίηση Άμεσων Μηνυμάτων"
3. Η επιλογή DM θα αποκρυφτεί από το προφίλ σας

Για περισσότερες λεπτομέρειες, δείτε τις [Ρυθμίσεις απορρήτου](/guide-user-profiles.html#privacy-settings).

**Αποκλεισμός Χρηστών:**
Αν κάποιος σας παρενοχλεί μέσω DM:
1. Επισκεφτείτε το προφίλ του
2. Αποκλείστε το χρήστη
3. Δεν θα μπορεί πλέον να σας στέλνει μηνύματα

Οι αποκλεισμένοι χρήστες δεν μπορούν:
- Να σας στέλνουν άμεσα μηνύματα
- Να βλέπουν τη δραστηριότητά σας
- Να σχολιάζουν στο προφίλ σας
- Να αλληλεπιδρούν μαζί σας οπουδήποτε στο FastComments

### Διαχείριση Συνομιλιών

**Απόκρυψη Συνομιλιών:**
Μπορείτε να αποκρύψετε συνομιλίες που δεν θέλετε να βλέπετε πλέον στη λίστα συνομιλιών σας:
1. Ανοίξτε την καρτέλα Άμεσα Μηνύματα
2. Βρείτε τη συνομιλία που θέλετε να αποκρύψετε
3. Επιλέξτε την επιλογή απόκρυψης/αποθήκευσης

Οι κρυφές συνομιλίες δεν εμφανίζονται στη κύρια λίστα σας αλλά μπορείτε να τις προσπελάσετε αν το άλλο άτομο στείλει νέο μήνυμα.

**Σήμανση ως Αναγνωσμένο:**
Τα μηνύματα επισημαίνονται αυτόματα ως αναγνωσμένα όταν τα προβάλετε. Μπορείτε επίσης να επισημάνετε χειροκίνητα ολόκληρες συνομιλίες ως αναγνωσμένες:
1. Ανοίξτε την καρτέλα Άμεσα Μηνύματα
2. Επιλέξτε μια συνομιλία
3. Επιλέξτε "Σήμανση ως Αναγνωσμένο"

Αυτό διαγράφει τον μετρητή μη αναγνωσμένων για εκείνη τη συνομιλία.

### Καλές Πρακτικές

**Πότε να Χρησιμοποιείτε Άμεσα Μηνύματα:**
- Για να κάνετε δεύτερες ερωτήσεις ιδιωτικά
- Να ευχαριστήσετε κάποιον για χρήσιμες συμβουλές
- Να συζητήσετε θέματα εκτός θέματος χωρίς να γεμίζετε τα δημόσια σχόλια
- Να συντονιστείτε με άλλα μέλη της κοινότητας
- Να δώσετε ιδιωτικά σχόλια ή προτάσεις

**Εθιμοτυπία στα DM:**
- Να είστε σεβαστικοί και επαγγελματικοί
- Μην στέλνετε ανεπιθύμητα μηνύματα σε χρήστες
- Σεβαστείτε αν κάποιος δεν απαντά ή απενεργοποιεί τα DM
- Διατηρήστε τις συνομιλίες σχετικές και εποικοδομητικές
- Μην μοιράζεστε τα DM άλλων χωρίς άδεια

**Συμβουλές Ασφαλείας:**
- Μην μοιράζεστε προσωπικές πληροφορίες (τηλέφωνο, διεύθυνση κ.λπ.) εκτός αν εμπιστεύεστε το άτομο
- Αποκλείστε και αναφέρετε χρήστες που παρενοχλούν ή καταχρώνται τα DM
- Προσέχετε ύποπτους συνδέσμους ή αιτήματα
- Χρησιμοποιήστε τη λειτουργία αποκλεισμού αν κάποιος σας κάνει να νιώθετε άβολα

### Περιορισμοί και Σημειώσεις

**Ποιον Μπορείτε να Μηνυμάσετε:**
- Οποιονδήποτε χρήστη του FastComments που δεν έχει απενεργοποιήσει τα DM
- Χρήστες που δεν σας έχουν αποκλείσει
- Χρήστες σε όλες τις κοινότητες του FastComments

**Περιεχόμενο Μηνυμάτων:**
- Υποστηρίζονται μηνύματα κειμένου
- Τα μηνύματα ακολουθούν τις ίδιες πολιτικές περιεχομένου με τα σχόλια
- Ακατάλληλο περιεχόμενο μπορεί να αναφερθεί

**Πεδίο Συνομιλίας:**
- Οι συνομιλίες DM είναι ιδιωτικές μεταξύ δύο ατόμων
- Δεν υπάρχει ομαδική συνομιλία (προς το παρόν μόνο ένας προς έναν)
- Το ιστορικό συνομιλιών διατηρείται επ’ αόριστον

### Αντιμετώπιση Προβλημάτων

**Δεν Μπορώ να Στείλω Μήνυμα;**
Ο άλλος χρήστης μπορεί να έχει:
- Απενεργοποιήσει τα άμεσα μηνύματα στις ρυθμίσεις απορρήτου του
- Σας έχει αποκλείσει
- Διαγράψει τον λογαριασμό του

**Δεν Λαμβάνω Ειδοποιήσεις;**
Ελέγξτε τις ρυθμίσεις ειδοποιήσεων στη σελίδα [Ρυθμίσεις λογαριασμού](https://fastcomments.com/auth/my-account/edit-notifications) για να βεβαιωθείτε ότι οι ειδοποιήσεις DM είναι ενεργοποιημένες.

**Τα Μηνύματα Δεν Αποστέλλονται;**
- Ελέγξτε τη σύνδεση στο διαδίκτυο
- Ανανέώστε τη σελίδα και δοκιμάστε ξανά
- Βεβαιωθείτε ότι δεν έχετε αποκλειστεί
- Επικοινωνήστε με την υποστήριξη αν το πρόβλημα συνεχίζεται
