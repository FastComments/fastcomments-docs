Τα Άμεσα Μηνύματα (DM) επιτρέπουν στους χρήστες του FastComments να έχουν ιδιωτικές, ένα-προς-ένα συνομιλίες μεταξύ τους, ανεξάρτητες από δημόσια σχόλια και αλληλεπιδράσεις προφίλ.

### Τι είναι τα Άμεσα Μηνύματα;

Τα Άμεσα Μηνύματα παρέχουν:
- **Ιδιωτικές συνομιλίες** μεταξύ δύο χρηστών
- **Άμεση ανταλλαγή μηνυμάτων σε πραγματικό χρόνο** με ειδοποιήσεις
- **Ιστορικό συνομιλιών** που διατηρείται μεταξύ των συνεδριών
- **Ενδείξεις κατάστασης σύνδεσης** για να βλέπετε πότε κάποιος είναι διαθέσιμος
- **Παρακολούθηση μη αναγνωσμένων μηνυμάτων** για να ενημερώνεστε για νέα μηνύματα

### Ξεκινώντας ένα Άμεσο Μήνυμα

Υπάρχουν αρκετοί τρόποι για να ξεκινήσετε μια συνομιλία DM:

**Από ένα Προφίλ Χρήστη:**
1. Επισκεφθείτε τη σελίδα προφίλ του χρήστη
2. Κάντε κλικ στην καρτέλα "Άμεσα Μηνύματα" ή στο κουμπί "Αποστολή Μηνύματος"
3. Πληκτρολογήστε το μήνυμά σας και στείλτε το

**Από ένα Σχόλιο:**
Ορισμένες υλοποιήσεις επιτρέπουν το κλικ στο όνομα ή το avatar ενός χρήστη για γρήγορη πρόσβαση στο προφίλ και στις επιλογές μηνυμάτων του.

**Άμεσος Σύνδεσμος:**
Εάν έχετε ένα ID συνομιλίας ή έναν σύνδεσμο ανακατεύθυνσης DM, μπορείτε να αποκτήσετε πρόσβαση σε συνομιλίες απευθείας.

### Πρόσβαση στα Μηνύματά σας

Για να δείτε όλες τις συνομιλίες άμεσων μηνυμάτων σας:

1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στην καρτέλα "Άμεσα Μηνύματα"
3. Δείτε μια λίστα με όλες τις συνομιλίες σας

Κάθε συνομιλία εμφανίζει:
- Το avatar και το όνομα του άλλου συμμετέχοντα
- Την κατάσταση σύνδεσής τους (online/offline) (πράσινος δείκτης όταν είναι online)
- Μια προεπισκόπηση του τελευταίου μηνύματος
- Αριθμό μη αναγνωσμένων μηνυμάτων (αν υπάρχουν)
- Χρονική ένδειξη της τελευταίας δραστηριότητας

### Προβολή Συνομιλίας

Όταν ανοίγετε μια συνομιλία, θα δείτε:

- **Ολόκληρο το ιστορικό μηνυμάτων** - Όλα τα μηνύματα μεταξύ εσάς και του άλλου χρήστη
- **Ενημερώσεις σε πραγματικό χρόνο** - Νέα μηνύματα εμφανίζονται άμεσα μέσω WebSocket
- **Κατάσταση σύνδεσης** - Δείτε αν το άλλο άτομο είναι αυτή τη στιγμή online
- **Χρονικές ενδείξεις μηνυμάτων** - Πότε εστάλη κάθε μήνυμα
- **Σύνθεση μηνύματος** - Πεδίο κειμένου για να πληκτρολογήσετε και να στείλετε νέα μηνύματα

### Ειδοποιήσεις Μηνυμάτων

Θα λαμβάνετε ειδοποιήσεις για νέα άμεσα μηνύματα:

- **Ειδοποιήσεις εντός εφαρμογής** - Μετρητής σήματος στο προφίλ σας
- **Ειδοποιήσεις μέσω email** - Ανάλογα με τις ρυθμίσεις ειδοποιήσεων σας
- **Ειδοποιήσεις σε πραγματικό χρόνο** - Άμεσες ειδοποιήσεις όταν είστε συνδεδεμένοι

Διαχειριστείτε τις προτιμήσεις ειδοποιήσεων στις [Account Settings](https://fastcomments.com/auth/my-account/edit-notifications).

### Απόρρητο και Αποκλεισμός

**Απενεργοποίηση Άμεσων Μηνυμάτων:**
Αν δεν θέλετε να λαμβάνετε DMs, μπορείτε να τα απενεργοποιήσετε εντελώς:
1. Πηγαίνετε στις Ρυθμίσεις Απορρήτου στο προφίλ σας
2. Ενεργοποιήστε την επιλογή "Απενεργοποίηση Άμεσων Μηνυμάτων"
3. Η επιλογή DM θα αποκρυφτεί από το προφίλ σας

Για περισσότερες λεπτομέρειες, δείτε τις [Privacy Settings](/guide-user-profiles.html#privacy-settings).

**Αποκλεισμός Χρηστών:**
Αν κάποιος σας παρενοχλεί μέσω DM:
1. Επισκεφθείτε το προφίλ του
2. Αποκλείστε τον χρήστη
3. Δεν θα μπορεί πλέον να σας στέλνει μηνύματα

Οι αποκλεισμένοι χρήστες δεν μπορούν:
- Να σας στείλουν άμεσα μηνύματα
- Να βλέπουν τη δραστηριότητά σας
- Να σχολιάζουν στο προφίλ σας
- Να αλληλεπιδρούν μαζί σας οπουδήποτε στο FastComments

### Διαχείριση Συνομιλιών

**Απόκρυψη Συνομιλιών:**
Μπορείτε να αποκρύψετε συνομιλίες που δεν θέλετε πια να βλέπετε στη λίστα συνομιλιών σας:
1. Ανοίξτε την καρτέλα Άμεσα Μηνύματα
2. Βρείτε τη συνομιλία που θέλετε να αποκρύψετε
3. Επιλέξτε την επιλογή απόκρυψης/αρχείου

Οι κρυφές συνομιλίες δεν θα εμφανίζονται στην κύρια λίστα σας αλλά μπορείτε να τις προσπελάσετε αν το άλλο άτομο στείλει νέο μήνυμα.

**Σήμανση ως Αναγνωσμένο:**
Τα μηνύματα μαρκάρονται αυτόματα ως αναγνωσμένα όταν τα προβάλλετε. Μπορείτε επίσης να μαρκάρετε ολόκληρες συνομιλίες ως αναγνωσμένες χειροκίνητα:
1. Ανοίξτε την καρτέλα Άμεσα Μηνύματα
2. Επιλέξτε μια συνομιλία
3. Επιλέξτε "Σήμανση ως Αναγνωσμένο"

Αυτό διαγράφει τον μετρητή μη αναγνωσμένων για αυτή τη συνομιλία.

### Καλύτερες Πρακτικές

**Πότε να χρησιμοποιείτε τα Άμεσα Μηνύματα:**
- Για να κάνετε ιδιωτικές ερωτήσεις παρακολούθησης
- Να ευχαριστήσετε κάποιον για χρήσιμες συμβουλές
- Να συζητήσετε εκτός θέματος χωρίς να γεμίζετε τα δημόσια σχόλια
- Να συντονιστείτε με άλλα μέλη της κοινότητας
- Να δώσετε ιδιωτικά σχόλια ή προτάσεις

**Κανόνες συμπεριφοράς στα DM:**
- Να είστε σεβαστικοί και επαγγελματικοί
- Μην στέλνετε ανεπιθύμητα μηνύματα spam σε χρήστες
- Σεβαστείτε αν κάποιος δεν απαντά ή απενεργοποιεί τα DMs
- Διατηρήστε τις συζητήσεις σχετικές και εποικοδομητικές
- Μην κοινοποιείτε τα DMs άλλων χωρίς άδεια

**Συμβουλές Ασφαλείας:**
- Μην μοιράζεστε προσωπικές πληροφορίες (τηλέφωνο, διεύθυνση, κ.λπ.) εκτός αν εμπιστεύεστε το άτομο
- Αποκλείστε και αναφέρετε χρήστες που παρενοχλούν ή καταχρώνται τα DMs
- Να είστε προσεκτικοί με ύποπτους συνδέσμους ή αιτήματα
- Χρησιμοποιήστε τη λειτουργία αποκλεισμού αν κάποιος σας κάνει να νιώθετε άβολα

### Περιορισμοί και Σημειώσεις

**Σε ποιον μπορείτε να στείλετε μήνυμα:**
- Σε οποιονδήποτε χρήστη του FastComments που δεν έχει απενεργοποιήσει τα DMs
- Σε χρήστες που δεν σας έχουν αποκλείσει
- Σε χρήστες σε όλες τις κοινότητες του FastComments

**Περιεχόμενο Μηνυμάτων:**
- Υποστηρίζονται μηνύματα κειμένου
- Τα μηνύματα ακολουθούν τις ίδιες πολιτικές περιεχομένου με τα σχόλια
- Ακατάλληλο περιεχόμενο μπορεί να αναφερθεί

**Πεδίο Συνομιλίας:**
- Οι συνομιλίες DM είναι ιδιωτικές μεταξύ δύο ατόμων
- Δεν υπάρχει ομαδική ανταλλαγή μηνυμάτων (προς το παρόν μόνο ένα-προς-ένα)
- Το ιστορικό συνομιλιών διατηρείται επ’ αόριστον

### Αντιμετώπιση Προβλημάτων

**Δεν μπορείτε να στείλετε μήνυμα;**
Ο άλλος χρήστης μπορεί να έχει:
- Απενεργοποιήσει τα άμεσα μηνύματα στις ρυθμίσεις απορρήτου του
- Σας έχει αποκλείσει
- Διαγράψει τον λογαριασμό του

**Δεν λαμβάνετε ειδοποιήσεις;**
Ελέγξτε τις ρυθμίσεις ειδοποιήσεων στις [Account Settings](https://fastcomments.com/auth/my-account/edit-notifications) για να βεβαιωθείτε ότι οι ειδοποιήσεις DM είναι ενεργοποιημένες.

**Τα μηνύματα δεν αποστέλλονται;**
- Ελέγξτε τη σύνδεση στο διαδίκτυο
- Ανανεώστε τη σελίδα και προσπαθήστε ξανά
- Βεβαιωθείτε ότι δεν έχετε αποκλειστεί
- Επικοινωνήστε με την υποστήριξη αν το πρόβλημα συνεχίζεται
