Άμεσα Μηνύματα (DM) επιτρέπουν στους χρήστες του FastComments να έχουν ιδιωτικές, ένας-προς-έναν συνομιλίες μεταξύ τους, ξεχωριστές από τα δημόσια σχόλια και τις αλληλεπιδράσεις προφίλ.

### Τι είναι τα Άμεσα Μηνύματα;

Τα Άμεσα Μηνύματα προσφέρουν:
- **Ιδιωτικές συνομιλίες** μεταξύ δύο χρηστών
- **Άμεση ανταλλαγή μηνυμάτων** με ειδοποιήσεις
- **Ιστορικό συνομιλιών** που διατηρείται μεταξύ των συνεδριών
- **Ενδείκτες κατάστασης σύνδεσης** για να βλέπετε πότε κάποιος είναι διαθέσιμος
- **Παρακολούθηση μη αναγνωσμένων μηνυμάτων** για να μη χάσετε νέα μηνύματα

### Ξεκινώντας ένα Άμεσο Μήνυμα

Υπάρχουν διάφοροι τρόποι να ξεκινήσετε μια συνομιλία DM:

**Από προφίλ χρήστη:**
1. Επισκεφθείτε τη σελίδα προφίλ του χρήστη
2. Κάντε κλικ στην καρτέλα "Direct Messages" ή στο κουμπί "Send Message"
3. Πληκτρολογήστε το μήνυμά σας και στείλτε το

**Από ένα σχόλιο:**
Κάποιες υλοποιήσεις επιτρέπουν το κλικ στο όνομα ή το avatar ενός χρήστη για γρήγορη πρόσβαση στο προφίλ και τις επιλογές μηνυμάτων του.

**Άμεσος σύνδεσμος:**
Εάν έχετε ένα ID συνομιλίας ή σύνδεσμο ανακατεύθυνσης DM, μπορείτε να έχετε πρόσβαση στις συνομιλίες απευθείας.

### Πρόσβαση στα Μηνύματά σας

Για να δείτε όλες τις συνομιλίες σας με άμεσα μηνύματα:

1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στην καρτέλα "Direct Messages"
3. Δείτε τη λίστα με όλες τις συνομιλίες σας

Κάθε συνομιλία εμφανίζει:
- Το avatar και το όνομα του άλλου συμμετέχοντα
- Την κατάσταση σύνδεσης τους (πράσινος δείκτης όταν είναι συνδεδεμένοι)
- Προεπισκόπηση του τελευταίου μηνύματος
- Αριθμό μη αναγνωσμένων μηνυμάτων (εάν υπάρχουν)
- Χρονική σήμανση της τελευταίας δραστηριότητας

### Προβολή Συνομιλίας

Όταν ανοίγετε μια συνομιλία, θα δείτε:

- **Πλήρες ιστορικό μηνυμάτων** - Όλα τα μηνύματα μεταξύ εσάς και του άλλου χρήστη
- **Άμεσες ενημερώσεις** - Τα νέα μηνύματα εμφανίζονται άμεσα μέσω WebSocket
- **Κατάσταση σύνδεσης** - Δείτε εάν το άλλο άτομο είναι αυτή τη στιγμή συνδεδεμένο
- **Χρονικές σημάνσεις μηνυμάτων** - Πότε στάλθηκε κάθε μήνυμα
- **Σύνθεση μηνύματος** - Πλαίσιο κειμένου για να πληκτρολογήσετε και να στείλετε νέα μηνύματα

### Ειδοποιήσεις Μηνυμάτων

Θα λαμβάνετε ειδοποιήσεις για νέα άμεσα μηνύματα:

- **Ειδοποιήσεις εντός εφαρμογής** - Μετρητής στο προφίλ σας
- **Ειδοποιήσεις μέσω email** - Ανάλογα με τις ρυθμίσεις ειδοποιήσεων σας
- **Άμεσες ειδοποιήσεις** - Ειδοποιήσεις σε πραγματικό χρόνο όταν είστε συνδεδεμένοι

Διαχειριστείτε τις προτιμήσεις ειδοποιήσεων στις [Account Settings](https://fastcomments.com/auth/my-account/notification-settings).

### Απόρρητο και Αποκλεισμός

**Απενεργοποίηση Άμεσων Μηνυμάτων:**
Εάν δεν θέλετε να λαμβάνετε DM, μπορείτε να τα απενεργοποιήσετε πλήρως:
1. Μεταβείτε στις Ρυθμίσεις απορρήτου στο προφίλ σας
2. Ενεργοποιήστε την επιλογή "Disable Direct Messages"
3. Η επιλογή DM θα κρυφτεί από το προφίλ σας

Για περισσότερες λεπτομέρειες, δείτε τις [Privacy Settings](/guides/user-profiles/privacy-settings).

**Αποκλεισμός χρηστών:**
Εάν κάποιος σας παρενοχλεί μέσω DM:
1. Επισκεφθείτε το προφίλ του
2. Αποκλείστε τον χρήστη
3. Δεν θα μπορεί πλέον να σας στέλνει μηνύματα

Οι αποκλεισμένοι χρήστες δεν μπορούν:
- Να σας στέλνουν άμεσα μηνύματα
- Να βλέπουν τη δραστηριότητά σας
- Να σχολιάζουν στο προφίλ σας
- Να αλληλεπιδρούν μαζί σας οπουδήποτε στο FastComments

### Διαχείριση Συνομιλιών

**Απόκρυψη συνομιλιών:**
Μπορείτε να αποκρύψετε συνομιλίες που δεν θέλετε πλέον να εμφανίζονται στη λίστα συνομιλιών σας:
1. Ανοίξτε την καρτέλα Direct Messages
2. Βρείτε τη συνομιλία που θέλετε να αποκρύψετε
3. Επιλέξτε την επιλογή απόκρυψης/αρχειοθέτησης

Οι κρυφές συνομιλίες δεν θα εμφανίζονται στην κύρια λίστα σας αλλά μπορούν να ανακτηθούν αν το άλλο άτομο στείλει νέο μήνυμα.

**Σήμανση ως αναγνωσμένο:**
Τα μηνύματα σημειώνονται αυτόματα ως αναγνωσμένα όταν τα βλέπετε. Μπορείτε επίσης να επισημάνετε ολόκληρες συνομιλίες ως αναγνωσμένες χειροκίνητα:
1. Ανοίξτε την καρτέλα Direct Messages
2. Επιλέξτε μια συνομιλία
3. Επιλέξτε "Mark as Read"

Αυτό καθαρίζει τον μετρητή μη αναγνωσμένων για αυτή τη συνομιλία.

### Καλές Πρακτικές

**Πότε να χρησιμοποιείτε τα Άμεσα Μηνύματα:**
- Για να κάνετε απορροφητικές ερωτήσεις ιδιωτικά
- Για να ευχαριστήσετε κάποιον για χρήσιμη συμβουλή
- Για να συζητήσετε εκτός θέματος χωρίς να γεμίζετε τα δημόσια σχόλια
- Για συντονισμό με άλλα μέλη της κοινότητας
- Για να δώσετε ιδιωτικά σχόλια ή προτάσεις

**Εθιμοτυπία στα DM:**
- Να είστε σεβαστοί και επαγγελματίες
- Μην στέλνετε ανεπιθύμητα μηνύματα (spam)
- Σεβαστείτε αν κάποιος δεν απαντά ή απενεργοποιεί τα DM
- Κρατήστε τις συνομιλίες σχετικές και εποικοδομητικές
- Μην κοινοποιείτε τα DM άλλων χωρίς άδεια

**Συμβουλές ασφάλειας:**
- Μην μοιράζεστε προσωπικές πληροφορίες (τηλέφωνο, διεύθυνση κ.λπ.) εκτός αν εμπιστεύεστε το άτομο
- Αποκλείστε και αναφέρετε χρήστες που παρενοχλούν ή καταχρώνται τα DM
- Να είστε επιφυλακτικοί με ύποπτους συνδέσμους ή αιτήματα
- Χρησιμοποιήστε τη λειτουργία αποκλεισμού εάν κάποιος σας κάνει να αισθανθείτε άβολα

### Περιορισμοί και Σημειώσεις

**Σε ποιους μπορείτε να στέλνετε μηνύματα:**
- Σε οποιονδήποτε χρήστη του FastComments που δεν έχει απενεργοποιήσει τα DM
- Σε χρήστες που δεν σας έχουν αποκλείσει
- Σε χρήστες σε όλες τις κοινότητες του FastComments

**Περιεχόμενο μηνυμάτων:**
- Υποστηρίζονται μηνύματα κειμένου
- Τα μηνύματα ακολουθούν τις ίδιες πολιτικές περιεχομένου με τα σχόλια
- Ακατάλληλο περιεχόμενο μπορεί να αναφερθεί

**Εύρος συνομιλίας:**
- Οι συνομιλίες DM είναι ιδιωτικές μεταξύ δύο ατόμων
- Δεν υπάρχουν ομαδικά μηνύματα (προς το παρόν μόνο ένας-προς-έναν)
- Το ιστορικό συνομιλιών διατηρείται επ’ αόριστον

### Αντιμετώπιση Προβλημάτων

**Δεν μπορείτε να στείλετε μήνυμα;**
Ο άλλος χρήστης μπορεί να έχει:
- Απενεργοποιήσει τα άμεσα μηνύματα στις ρυθμίσεις απορρήτου του
- Σας έχει αποκλείσει
- Διαγράψει τον λογαριασμό του

**Δεν λαμβάνετε ειδοποιήσεις;**
Ελέγξτε τις ρυθμίσεις ειδοποιήσεων στις [Account Settings](https://fastcomments.com/auth/my-account/notification-settings) για να βεβαιωθείτε ότι οι ειδοποιήσεις DM είναι ενεργοποιημένες.

**Τα μηνύματα δεν αποστέλλονται;**
- Ελέγξτε τη σύνδεση στο Internet
- Ανανεώστε τη σελίδα και δοκιμάστε ξανά
- Βεβαιωθείτε ότι δεν έχετε αποκλειστεί
- Επικοινωνήστε με την υποστήριξη εάν το πρόβλημα συνεχίζεται
