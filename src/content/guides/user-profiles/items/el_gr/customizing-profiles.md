Τα προφίλ χρηστών του FastComments είναι ιδιαίτερα προσαρμόσιμα, επιτρέποντας στους χρήστες να εκφράσουν την προσωπικότητά τους και να χτίσουν την διαδικτυακή τους παρουσία.

### Επεξεργασία του Προφίλ σας

Για να προσαρμόσετε το προφίλ σας, επισκεφθείτε τη σελίδα [Λεπτομέρειες λογαριασμού](https://fastcomments.com/auth/my-account/edit-details). Εδώ μπορείτε να τροποποιήσετε όλες τις πτυχές της εμφάνισης και των πληροφοριών του προφίλ σας.

### Εικόνα Προφίλ

Η εικόνα προφίλ είναι η φωτογραφία που εμφανίζεται δίπλα στα σχόλιά σας και στη σελίδα του προφίλ σας.

**Για να αλλάξετε την εικόνα προφίλ σας:**
1. Μεταβείτε στη σελίδα του προφίλ σας
2. Κάντε κλικ στον σύνδεσμο "Επεξεργασία εικόνας προφίλ" (ορατό μόνο στο δικό σας προφίλ)
3. Ανεβάστε μια νέα εικόνα

**Απαιτήσεις εικόνας προφίλ:**
- Συνιστώμενο μέγεθος: 200x200 pixels ή μεγαλύτερο
- Υποστηριζόμενα φορμά: JPG, PNG, GIF
- Οι τετράγωνες εικόνες ταιριάζουν καλύτερα

Σημείωση: Οι χρήστες SSO μπορεί να έχουν την εικόνα προφίλ τους ελεγχόμενη από το σύστημα SSO του ιστότοπού τους.

### Φόντο Κεφαλίδας Προφίλ

Το φόντο της κεφαλίδας είναι μια μεγάλη εικόνα banner στο πάνω μέρος του προφίλ σας που μπορεί να προσαρμοστεί για να αντικατοπτρίζει το στυλ σας.

**Για να ανεβάσετε φόντο κεφαλίδας:**
1. Μεταβείτε στη σελίδα του προφίλ σας
2. Κάντε κλικ στο κουμπί "Μεταφόρτωση κεφαλίδας"
3. Επιλέξτε μια εικόνα από τη συσκευή σας

**Απαιτήσεις κεφαλίδας:**
- Συνιστώμενο μέγεθος: 1200x300 pixels
- Υποστηριζόμενα φορμά: JPG, PNG
- Οι οριζόντιες/ευρείες εικόνες ταιριάζουν καλύτερα
- Οι εικόνες αλλάζουν αυτόματα μέγεθος για να προσαρμοστούν

Αν δεν ανεβάσετε προσαρμοσμένη κεφαλίδα, θα εμφανιστεί ένα προεπιλεγμένο φόντο.

### Όνομα Εμφάνισης

Το όνομα εμφάνισης είναι ο τρόπος που εμφανίζεστε σε άλλους χρήστες στο FastComments. Σε αντίθεση με τα ονόματα χρήστη (τα οποία πρέπει να είναι μοναδικά), τα ονόματα εμφάνισης:

- Δεν χρειάζεται να είναι μοναδικά
- Μπορούν να περιέχουν κενά και ειδικούς χαρακτήρες
- Μπορούν να περιλαμβάνουν emoji

**Για να ορίσετε το όνομα εμφάνισης:**
1. Μεταβείτε στις [Λεπτομέρειες λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε το προτιμώμενο όνομα εμφάνισης
3. Αποθηκεύστε τις αλλαγές

### Όνομα χρήστη

Το όνομα χρήστη είναι ένας μοναδικός αναγνωριστικός που χρησιμοποιείται για να συνδεθείτε στο FastComments.

- Πρέπει να είναι μοναδικό σε όλους τους χρήστες του FastComments
- Χρησιμοποιείται στο URL του προφίλ σας: `fastcomments.com/auth/user-profile/username`
- Δεν μπορεί να περιέχει κενά ή ειδικούς χαρακτήρες

Σημείωση: Οι χρήστες SSO μπορούν να έχουν μη-μοναδικά ονόματα χρήστη καθώς συνδέονται με έναν συγκεκριμένο tenant/community.

### Βιογραφικό

Προσθέστε ένα προσωπικό βιογραφικό για να πείτε στους άλλους ποιοι είστε, τα ενδιαφέροντά σας ή τι κάνετε.

**Για να προσθέσετε βιογραφικό:**
1. Μεταβείτε στις [Λεπτομέρειες λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε κείμενο στο πεδίο Βιογραφικό
3. Αποθηκεύστε τις αλλαγές

Το βιογραφικό σας εμφανίζεται με έμφαση στη σελίδα του προφίλ σας και βοηθά τους άλλους να καταλάβουν ποιοι είστε.

### Διεύθυνση Ιστοτόπου

Προσθέστε ένα σύνδεσμο προς τον προσωπικό σας ιστότοπο, το blog ή το portfolio.

**Για να προσθέσετε ιστότοπο:**
1. Μεταβείτε στις [Λεπτομέρειες λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε το URL του ιστότοπού σας στο πεδίο Ιστότοπος
3. Αποθηκεύστε τις αλλαγές

Ο σύνδεσμος προς τον ιστότοπό σας θα εμφανίζεται στο προφίλ σας με ένα κλικαμπλ εικονίδιο.

### Σύνδεσμοι Κοινωνικών Δικτύων

Συνδέστε τα προφίλ κοινωνικών δικτύων σας με το προφίλ σας στο FastComments. Οι υποστηριζόμενες πλατφόρμες περιλαμβάνουν:

- **Facebook** - Σύνδεσμος στο προφίλ σας στο Facebook
- **Instagram** - Σύνδεσμος στο λογαριασμό σας στο Instagram
- **LinkedIn** - Σύνδεσμος στο προφίλ σας στο LinkedIn
- **Custom Links** - Προσθέστε οποιαδήποτε άλλη κοινωνική πλατφόρμα ή προφίλ ιστού

**Για να προσθέσετε συνδέσμους κοινωνικών δικτύων:**
1. Μεταβείτε στις [Λεπτομέρειες λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε τα URL για τους λογαριασμούς κοινωνικών δικτύων σας
3. Αποθηκεύστε τις αλλαγές

Οι σύνδεσμοι κοινωνικών δικτύων εμφανίζονται στο προφίλ σας με αναγνωρίσιμα εικονίδια πλατφόρμας, διευκολύνοντας τα μέλη της κοινότητας να συνδεθούν μαζί σας αλλού.

### Χώρα

Μπορείτε προαιρετικά να ορίσετε τη χώρα σας, η οποία εμφανίζει ένα εικονίδιο σημαίας δίπλα στο όνομά σας.

**Για να ορίσετε τη χώρα σας:**
1. Μεταβείτε στις [Λεπτομέρειες λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Επιλέξτε τη χώρα σας από το αναπτυσσόμενο μενού
3. Αποθηκεύστε τις αλλαγές

Η σημαία της χώρας εμφανίζεται δίπλα στο όνομα χρήστη σας στο προφίλ σας.

### Εφεδρικό Email

Μπορείτε να προσθέσετε ένα εφεδρικό email στον λογαριασμό σας για σκοπούς ανάκτησης. Εάν χάσετε την πρόσβαση στο κύριο email σας, το εφεδρικό email σας μπορεί να χρησιμοποιηθεί για σύνδεση ή ανάκτηση του ονόματος χρήστη σας.

**Για να προσθέσετε εφεδρικό email:**
1. Μεταβείτε στις [Λεπτομέρειες λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Κάντε κλικ στο **Ορισμός εφεδρικού email** κάτω από το πεδίο email
3. Εισάγετε τη διεύθυνση εφεδρικού email και υποβάλετε
4. Κάντε κλικ στον σύνδεσμο επαλήθευσης που αποστάλθηκε σε αυτή τη διεύθυνση

Το εφεδρικό email χρησιμοποιείται μόνο για ανάκτηση λογαριασμού και δεν θα λαμβάνει ειδοποιήσεις ή άλλα email. Μπορείτε να το αλλάξετε ή να το αφαιρέσετε οποιαδήποτε στιγμή.

Σημείωση: Το εφεδρικό email είναι διαθέσιμο μόνο για κανονικούς χρήστες, όχι για χρήστες SSO.
