FastComments User Profiles are highly customizable, allowing users to express their personality and build their online presence.

### Επεξεργασία του προφίλ σας

Για να προσαρμόσετε το προφίλ σας, επισκεφτείτε τη σελίδα [Στοιχεία λογαριασμού](https://fastcomments.com/auth/my-account/edit-details). Εδώ μπορείτε να τροποποιήσετε κάθε πτυχή της εμφάνισης και των πληροφοριών του προφίλ σας.

### Άβαταρ προφίλ

Το άβαταρ σας είναι η φωτογραφία προφίλ που εμφανίζεται δίπλα στα σχόλιά σας και στη σελίδα του προφίλ σας.

**Για να αλλάξετε το άβαταρ σας:**
1. Επισκεφτείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στο σύνδεσμο "Επεξεργασία άβαταρ" (ορατό μόνο στο δικό σας προφίλ)
3. Ανεβάστε μια νέα εικόνα

**Απαιτήσεις άβαταρ:**
- Συνιστώμενο μέγεθος: 200x200 pixels ή μεγαλύτερο
- Υποστηριζόμενα φορμά: JPG, PNG, GIF
- Οι τετράγωνες εικόνες λειτουργούν καλύτερα

Σημείωση: Οι χρήστες SSO μπορεί να έχουν το άβαταρ τους ελεγχόμενο από το σύστημα SSO του ιστότοπού τους.

### Φόντο κεφαλίδας προφίλ

Το φόντο της κεφαλίδας είναι μια μεγάλη εικόνα banner στην κορυφή του προφίλ σας που μπορεί να προσαρμοστεί για να αντικατοπτρίζει το στυλ σας.

**Για να ανεβάσετε φόντο κεφαλίδας:**
1. Επισκεφτείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στο κουμπί "Ανέβασμα κεφαλίδας"
3. Επιλέξτε μια εικόνα από τη συσκευή σας

**Απαιτήσεις κεφαλίδας:**
- Συνιστώμενο μέγεθος: 1200x300 pixels
- Υποστηριζόμενα φορμά: JPG, PNG
- Οι τοπίου/ευρείες εικόνες λειτουργούν καλύτερα
- Οι εικόνες αλλάζουν μέγεθος αυτόματα για να ταιριάξουν

Εάν δεν ανεβάσετε προσαρμοσμένη κεφαλίδα, θα εμφανιστεί ένα προεπιλεγμένο φόντο.

### Όνομα εμφάνισης

Το όνομα εμφάνισης είναι ο τρόπος που εμφανίζεστε σε άλλους χρήστες σε όλο το FastComments. Σε αντίθεση με τα ονόματα χρήστη (τα οποία πρέπει να είναι μοναδικά), τα ονόματα εμφάνισης:

- Δεν χρειάζεται να είναι μοναδικά
- Μπορούν να περιέχουν κενά και ειδικούς χαρακτήρες
- Μπορούν να περιλαμβάνουν emoji

**Για να ορίσετε το όνομα εμφάνισής σας:**
1. Μεταβείτε στα [Στοιχεία λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε το προτιμώμενο όνομα εμφάνισης
3. Αποθηκεύστε τις αλλαγές

### Όνομα χρήστη

Το όνομα χρήστη είναι ένας μοναδικός αναγνωριστής που χρησιμοποιείται για να συνδεθείτε στο FastComments.

- Πρέπει να είναι μοναδικό για όλους τους χρήστες του FastComments
- Χρησιμοποιείται στο URL του προφίλ σας: `fastcomments.com/auth/user-profile/username`
- Δεν μπορεί να περιέχει κενά ή ειδικούς χαρακτήρες

**Σημείωση:** Οι χρήστες SSO μπορούν να έχουν μη-μοναδικά ονόματα χρήστη καθώς συνδέονται με έναν συγκεκριμένο tenant/κοινότητα.

### Βιογραφικό

Προσθέστε ένα προσωπικό βιογραφικό για να πείτε στους άλλους για τον εαυτό σας, τα ενδιαφέροντά σας ή τι κάνετε.

**Για να προσθέσετε βιογραφικό:**
1. Μεταβείτε στα [Στοιχεία λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε κείμενο στο πεδίο Βιογραφικό
3. Αποθηκεύστε τις αλλαγές

Το βιογραφικό σας εμφανίζεται εμφανώς στη σελίδα του προφίλ σας και βοηθά τους άλλους να καταλάβουν ποιοι είστε.

### Διεύθυνση ιστότοπου

Προσθέστε έναν σύνδεσμο προς τον προσωπικό σας ιστότοπο, το blog ή το portfolio σας.

**Για να προσθέσετε έναν ιστότοπο:**
1. Μεταβείτε στα [Στοιχεία λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε τη διεύθυνση του ιστότοπού σας στο πεδίο Ιστότοπος
3. Αποθηκεύστε τις αλλαγές

Ο σύνδεσμος του ιστότοπού σας θα εμφανίζεται στο προφίλ σας με ένα εικονίδιο που μπορεί να κλικάρει.

### Σύνδεσμοι κοινωνικών μέσων

Συνδέστε τα προφίλ σας στα κοινωνικά μέσα με το προφίλ FastComments σας. Οι υποστηριζόμενες πλατφόρμες περιλαμβάνουν:

- **Facebook** - Σύνδεσμος στο προφίλ σας στο Facebook
- **Instagram** - Σύνδεσμος στο λογαριασμό σας στο Instagram
- **LinkedIn** - Σύνδεσμος στο προφίλ σας στο LinkedIn
- **Προσαρμοσμένοι σύνδεσμοι** - Προσθέστε οποιοδήποτε άλλο προφίλ κοινωνικού μέσου ή ιστότοπο

**Για να προσθέσετε συνδέσμους κοινωνικών μέσων:**
1. Μεταβείτε στα [Στοιχεία λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε τις διευθύνσεις URL των λογαριασμών σας στα κοινωνικά μέσα
3. Αποθηκεύστε τις αλλαγές

Οι σύνδεσμοι κοινωνικών μέσων εμφανίζονται στο προφίλ σας με αναγνωρίσιμα εικονίδια πλατφορμών, διευκολύνοντας τα μέλη της κοινότητας να συνδεθούν μαζί σας αλλού.
