Τα Προφίλ Χρηστή στο FastComments παρέχουν έναν αφιερωμένο χώρο για κάθε χρήστη ώστε να παρουσιάσει την ταυτότητα, τη δραστηριότητα και τις συνεισφορές του στην κοινότητά σας.

### Τι είναι ένα Προφίλ Χρήστη;

Ένα Προφίλ Χρήστη είναι μια προσωποποιημένη σελίδα για κάθε χρήστη του FastComments που εμφανίζει:

- **Προφίλ Header** - Μια προσαρμόσιμη εικόνα φόντου που προσωποποιεί το προφίλ
- **Άβαταρ** - Η εικόνα προφίλ του χρήστη με δείκτη κατάστασης σε σύνδεση/εκτός σύνδεσης
- **Πληροφορίες Εμφάνισης** - Όνομα χρήστη, εμφανιζόμενο όνομα και προαιρετική σημαία χώρας
- **Βιογραφία** - Μια προσωπική περιγραφή ή εισαγωγή
- **Σύνδεσμοι Κοινωνικών Δικτύων** - Συνδέσεις προς τα προφίλ κοινωνικών δικτύων και τον ιστότοπο του χρήστη
- **Διακριτικά** - Αποκτηθέντα επιτεύγματα και αναγνωρίσεις
- **Στατιστικά** - Karma χρήστη και συνολικός αριθμός σχολίων
- **Κοινότητες** - Σε ποιες ιστοσελίδες/τομείς είναι ενεργός ο χρήστης

### Πρόσβαση σε Προφίλ Χρηστών

Υπάρχουν πολλοί τρόποι για να αποκτήσετε πρόσβαση στο προφίλ ενός χρήστη:

1. **Κάντε κλικ σε ένα άβαταρ** - Στο widget σχολίων, κάντε κλικ στο άβαταρ οποιουδήποτε χρήστη για να δείτε το προφίλ του
2. **Κάντε κλικ σε ένα όνομα χρήστη** - Τα ονόματα χρηστών στα σχόλια είναι κλικαριστοί σύνδεσμοι προς προφίλ
3. **Άμεσο URL** - Επισκεφτείτε `https://fastcomments.com/auth/user-profile/[userId]`

### Προβολές Προφίλ

Όταν βλέπετε ένα προφίλ, θα δείτε διαφορετικές καρτέλες ανάλογα με το αν βλέπετε το δικό σας προφίλ ή το προφίλ κάποιου άλλου:

#### Το Προφίλ σας
- **Ειδοποιήσεις** - Οι ειδοποιήσεις και οι αναφορές σας
- **Πρόσφατη Δραστηριότητα** - Το ιστορικό σχολίων σας σε όλες τις κοινότητες
- **Σχόλια Προφίλ** - Τα σχόλια που άλλοι έχουν αφήσει στη σελίδα του προφίλ σας
- **Άμεσα Μηνύματα** - Ιδιωτικές συνομιλίες με άλλους χρήστες

#### Προφίλ άλλων χρηστών
- **Πρόσφατη Δραστηριότητα** - Το δημόσιο ιστορικό σχολίων τους (αν δεν είναι ρυθμισμένο ως ιδιωτικό)
- **Σχόλια Προφίλ** - Σχόλια στο προφίλ τους (αν δεν είναι ρυθμισμένο ως ιδιωτικό)
- **Άμεσα Μηνύματα** - Ξεκινήστε ή συνεχίστε μια ιδιωτική συνομιλία (αν επιτρέπουν άμεσα μηνύματα)

### Κατάσταση Σύνδεσης

Τα προφίλ χρηστών εμφανίζουν σε πραγματικό χρόνο την κατάσταση σύνδεσης:
- **Πράσινος δείκτης** - Ο χρήστης είναι αυτή τη στιγμή σε σύνδεση
- **Χωρίς δείκτη** - Ο χρήστης είναι εκτός σύνδεσης

Αυτό σας βοηθά να γνωρίζετε πότε κάποιος χρησιμοποιεί ενεργά την πλατφόρμα, κάτι που είναι ιδιαίτερα χρήσιμο για τα άμεσα μηνύματα.

### Τύποι Χρηστών

Το FastComments υποστηρίζει δύο τύπους χρηστών με προφίλ:

1. **Κανονικοί Χρήστες** - Χρήστες που εγγράφηκαν απευθείας στο FastComments
2. **Χρήστες SSO** - Χρήστες που αυθεντικοποιούνται μέσω της ενσωμάτωσης Single Sign-On του ιστότοπού σας

Και οι δύο τύποι χρηστών έχουν πρόσβαση στο πλήρες σύστημα προφίλ, αν και οι χρήστες SSO μπορεί να έχουν κάποιους περιορισμούς στην επεξεργασία ορισμένων πεδίων (όπως τα άβαταρ), ανάλογα με τη ρύθμιση του SSO σας.
