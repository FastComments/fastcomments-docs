Τα διακριτικά είναι οπτικά επιτεύγματα που εμφανίζονται στο Προφίλ Χρήστη σας, αναγνωρίζοντας τις συνεισφορές και τη δραστηριότητά σας στις κοινότητες του FastComments.

### Τι είναι τα Διακριτικά;

Τα διακριτικά είναι αναγνωρίσεις που αποκτώνται για διάφορες δραστηριότητες, όπως:
- Αριθμός σχολίων που έχετε δημοσιεύσει
- Ψήφοι υπέρ που λάβατε
- Απαντήσεις που δημιουργήσατε
- Ενέργειες διαχειριστή
- Χρόνος ως μέλος της κοινότητας
- Ειδικά επιτεύγματα

Κάθε διακριτικό εμφανίζεται στο προφίλ σας και μπορεί προαιρετικά να εμφανιστεί δίπλα στο όνομά σας στα σχόλια.

### Προβολή των Διακριτικών Σας

Τα διακριτικά που έχετε αποκτήσει εμφανίζονται στη σελίδα του προφίλ σας στην επικεφαλίδα. Υπάρχουν δύο κατηγορίες:

1. **Κατακτημένα Διακριτικά** - Διακριτικά που έχετε πλήρως κατακτήσει και ξεκλειδώσει
2. **Σε Εξέλιξη** - Διακριτικά προς κατάκτηση, με γραμμές προόδου που δείχνουν πόσο κοντά βρίσκεστε

### Εμφάνιση Διακριτικών

Έχετε πλήρη έλεγχο πάνω στο πώς εμφανίζονται τα διακριτικά:

**Στο Προφίλ Σας:**
Όλα τα κατακτημένα διακριτικά εμφανίζονται στην επικεφαλίδα του προφίλ σας για να τα βλέπουν οι άλλοι.

**Στα Σχόλιά Σας:**
Μπορείτε να επιλέξετε ποια διακριτικά εμφανίζονται δίπλα στο όνομά σας όταν σχολιάζετε:

1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ "Διαχείριση Διακριτικών" ή ανοίξτε το παράθυρο διαχείρισης διακριτικών
3. Ενεργοποιήστε/απενεργοποιήστε ποια διακριτικά πρέπει να εμφανίζονται στα σχόλιά σας
4. Αναδιατάξτε τα διακριτικά σύροντάς τα για να ορίσετε προτεραιότητα εμφάνισης

Αυτό σας επιτρέπει να προβάλλετε τα πιο σημαντικά σας επιτεύγματα χωρίς να γεμίζετε την υπογραφή του σχολίου σας.

### Τύποι Διακριτικών

Τα διακριτικά υπάρχουν σε διάφορους τύπους που βασίζονται σε διαφορετικές κατηγορίες επιτευγμάτων. Για πλήρη επισκόπηση όλων των διαθέσιμων τύπων διακριτικών και πώς να τα αποκτήσετε, δείτε τον [Οδηγό Διακριτικών](/guides/badges).

Κοινές κατηγορίες διακριτικών περιλαμβάνουν:

- **Διακριτικά Συμμετοχής** - Βασίζονται στον αριθμό σχολίων, τις απαντήσεις και την αλληλεπίδραση
- **Διακριτικά Ποιότητας** - Βασίζονται στις ψήφους υπέρ και την αποδοχή από την κοινότητα
- **Χρονικά Διακριτικά** - Π.χ. κατάσταση βετεράνου, δραστηριότητα τη νύχτα
- **Διακριτικά Συντονιστή** - Αναγνώριση για ενέργειες συντονισμού
- **Ειδικά Διακριτικά** - Χορηγούνται χειροκίνητα ή είναι ειδικά για μια κοινότητα

### Διακριτικά Ειδικά για Κοινότητα

Τα διακριτικά μπορούν να αποκτηθούν σε διάφορες κοινότητες του FastComments:

- **Παγκόσμια Διακριτικά** - Αποκτώνται σε όλη τη χρήση του FastComments
- **Διακριτικά Κοινότητας** - Αποκτώνται μέσα σε συγκεκριμένους ιστότοπους/tenants

Το προφίλ σας δείχνει ποια κοινότητα απένειμε κάθε διακριτικό, βοηθώντας τους επισκέπτες να καταλάβουν τους τομείς εξειδίκευσης και συμμετοχής σας.

### Παρακολούθηση Προόδου Διακριτικών

Για διακριτικά που δεν έχετε ακόμη πλήρως αποκτήσει, το προφίλ σας δείχνει:

- **Γραμμή Προόδου** - Οπτικός δείκτης του πόσο κοντά είστε στο να αποκτήσετε το διακριτικό
- **Απαιτήσεις** - Τι πρέπει να κάνετε για να ξεκλειδώσετε το διακριτικό
- **Κοινοτικό Πλαίσιο** - Με ποια κοινότητα σχετίζεται το διακριτικό

Αυτό ενθαρρύνει τη συνεχιζόμενη συμμετοχή και σας δίνει σαφείς στόχους προς επίτευξη.

### Προσαρμογή Εμφάνισης Διακριτικών

**Σειρά Διακριτικών:**
Μπορείτε να επαναταξινομήσετε τα διακριτικά για να ελέγξετε ποια εμφανίζονται πρώτα. Η σειρά που ορίζετε καθορίζει:
- Προτεραιότητα στην εμφάνιση στο προφίλ
- Ποια διακριτικά εμφανίζονται σε περιπτώσεις περιορισμένου χώρου
- Προσωπική προτίμηση για την ανάδειξη επιτευγμάτων

**Ορατότητα:**
Ενεργοποιήστε/απενεργοποιήστε μεμονωμένα διακριτικά για εμφάνιση στα σχόλια, ενώ παραμένουν ορατά στο προφίλ σας.

### Χειροκίνητα Διακριτικά

Οι διαχειριστές της κοινότητας μπορούν να απονείμουν ειδικά χειροκίνητα διακριτικά για να αναγνωρίσουν:
- Ειδικές συνεισφορές
- Νικητές διαγωνισμών
- VIP μέλη
- Προσαρμοσμένα επιτεύγματα

Αυτά τα διακριτικά εμφανίζονται μαζί με τα αυτομάτως αποκτηθέντα διακριτικά στο προφίλ σας.

### Για Περισσότερες Πληροφορίες

Για πλήρεις λεπτομέρειες σχετικά με το σύστημα διακριτικών, συμπεριλαμβανομένων όλων των τύπων διακριτικών, κριτηρίων απόκτησης και ρυθμίσεων για διαχειριστές, δείτε τον ολοκληρωμένο [Οδηγό Διακριτικών](/guides/badges).
