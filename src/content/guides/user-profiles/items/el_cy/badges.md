Τα διακριτικά είναι οπτικά επιτεύγματα που εμφανίζονται στο Προφίλ σας, αναγνωρίζοντας τις συμβολές και τη δραστηριότητά σας στις κοινότητες FastComments.

### Τι είναι τα Διακριτικά;

Τα διακριτικά είναι αναγνωρίσεις που κερδίζονται για διάφορες δραστηριότητες όπως:
- Αριθμός σχολίων που αναρτήθηκαν
- Λήψη upvotes
- Παραγόμενες απαντήσεις
- Ενέργειες συντονιστή
- Χρόνος ως μέλος της κοινότητας
- Ειδικά επιτεύγματα

Κάθε διακριτικό εμφανίζεται στο προφίλ σας και μπορεί προαιρετικά να εμφανίζεται δίπλα στο όνομά σας στα σχόλια.

### Προβολή των Διακριτικών σας

Τα διακριτικά που έχετε κερδίσει εμφανίζονται στη σελίδα του προφίλ σας στην ενότητα της επικεφαλίδας. Υπάρχουν δύο κατηγορίες:

1. **Κερδισμένα Διακριτικά** - Διακριτικά που έχετε πλήρως κερδίσει και ξεκλειδώσει
2. **Σε Εξέλιξη** - Διακριτικά προς κατάκτηση, με μπάρες προόδου που δείχνουν πόσο κοντά είστε

### Εμφάνιση Διακριτικών

Έχετε πλήρη έλεγχο για το πώς εμφανίζονται τα διακριτικά:

**Στο Προφίλ σας:**
Όλα τα κερδισμένα διακριτικά εμφανίζονται στην επικεφαλίδα του προφίλ σας για να τα βλέπουν οι άλλοι.

**Στα Σχόλιά σας:**
Μπορείτε να επιλέξετε ποια διακριτικά εμφανίζονται δίπλα στο όνομά σας όταν σχολιάζετε:

1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στο "Διαχείριση Διακριτικών" ή ανοίξτε το διάλογο διαχείρισης διακριτικών
3. Ενεργοποιήστε/απενεργοποιήστε ποια διακριτικά πρέπει να εμφανίζονται στα σχόλιά σας
4. Αναδιατάξτε τα διακριτικά σύροντάς τα για να ορίσετε την προτεραιότητα εμφάνισης

Αυτό σας επιτρέπει να προβάλετε τα πιο σημαντικά επιτεύγματά σας χωρίς να υπερφορτώνετε την υπογραφή του σχολίου σας.

### Τύποι Διακριτικών

Τα διακριτικά υπάρχουν σε διάφορους τύπους βασισμένους σε διαφορετικές κατηγορίες επιτευγμάτων. Για πλήρη επισκόπηση όλων των διαθέσιμων τύπων διακριτικών και του τρόπου απόκτησής τους, δείτε τον [Οδηγό Διακριτικών](/guide-badges.html).

Συνήθεις κατηγορίες διακριτικών περιλαμβάνουν:

- **Διακριτικά Συμμετοχής** - Βασισμένα στον αριθμό σχολίων, απαντήσεων και αλληλεπίδρασης
- **Διακριτικά Ποιότητας** - Βασισμένα σε upvotes και την αποδοχή από την κοινότητα
- **Χρονικά Διακριτικά** - Κατάσταση βετεράνου, νυχτερινή δραστηριότητα
- **Διακριτικά Συντονιστή** - Αναγνώριση για ενέργειες διαχείρισης
- **Ειδικά Διακριτικά** - Χορηγούνται χειροκίνητα ή αφορούν συγκεκριμένα της κοινότητας επιτεύγματα

### Κοινότητας-Ειδικά Διακριτικά

Τα διακριτικά μπορούν να κερδηθούν σε διαφορετικές κοινότητες FastComments:

- **Παγκόσμια Διακριτικά** - Κερδίζονται σε όλη τη χρήση του FastComments
- **Διακριτικά Κοινότητας** - Κερδίζονται εντός συγκεκριμένων ιστότοπων/ενοικιαστών

Το προφίλ σας δείχνει ποια κοινότητα απένειμε κάθε διακριτικό, βοηθώντας τους επισκέπτες να κατανοήσουν τους τομείς εξειδίκευσής σας και συμμετοχής.

### Παρακολούθηση Προόδου Διακριτικών

Για διακριτικά που δεν έχετε ακόμη κερδίσει πλήρως, το προφίλ σας εμφανίζει:

- **Μπάρα Προόδου** - Οπτικός δείκτης του πόσο κοντά είστε στο να κερδίσετε το διακριτικό
- **Απαιτήσεις** - Τι χρειάζεται να κάνετε για να ξεκλειδώσετε το διακριτικό
- **Πλαίσιο Κοινότητας** - Σε ποια κοινότητα σχετίζεται το διακριτικό

Αυτό ενθαρρύνει τη συνεχιζόμενη συμμετοχή και σας δίνει ξεκάθαρους στόχους προς επίτευξη.

### Προσαρμογή Εμφάνισης Διακριτικών

**Σειρά Διακριτικών:**
Μπορείτε να αναδιατάξετε τα διακριτικά για να ελέγξετε ποια εμφανίζονται πρώτα. Η σειρά που ορίζετε καθορίζει:
- Την προτεραιότητα στην εμφάνιση στο προφίλ
- Ποια διακριτικά εμφανίζονται σε περιπτώσεις περιορισμένου χώρου
- Τις προσωπικές προτιμήσεις για την ανάδειξη επιτευγμάτων

**Ορατότητα:**
Ενεργοποιήστε ή απενεργοποιήστε μεμονωμένα διακριτικά για εμφάνιση στα σχόλια, ενώ τα διατηρείτε ορατά στο προφίλ σας.

### Χειροκίνητα Διακριτικά

Οι διαχειριστές της κοινότητας μπορούν να απονείμουν ειδικά χειροκίνητα διακριτικά για να αναγνωρίσουν:
- Ειδικές συνεισφορές
- Νικητές διαγωνισμών
- VIP μέλη
- Προσαρμοσμένα επιτεύγματα

Αυτά τα διακριτικά εμφανίζονται μαζί με τα αυτόματα κερδισμένα διακριτικά στο προφίλ σας.

### Περισσότερες Πληροφορίες

Για πλήρεις λεπτομέρειες σχετικά με το σύστημα διακριτικών, συμπεριλαμβανομένων όλων των τύπων διακριτικών, των κριτηρίων απόκτησης και της διαχειριστικής διαμόρφωσης, δείτε τον ολοκληρωμένο [Οδηγό Διακριτικών](/guide-badges.html).
