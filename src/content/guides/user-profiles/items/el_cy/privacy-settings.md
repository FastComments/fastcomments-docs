FastComments σας δίνει λεπτομερή έλεγχο πάνω στην ιδιωτικότητα του προφίλ σας και στο ποιος μπορεί να αλληλεπιδρά μαζί σας. Όλες οι ρυθμίσεις ιδιωτικότητας μπορούν να διαχειριστούν από το τμήμα Προτιμήσεις Απορρήτου στη σελίδα του προφίλ σας.

### Πρόσβαση στις Ρυθμίσεις Απορρήτου

Για να διαχειριστείτε τις ρυθμίσεις απορρήτου σας:
1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στην καρτέλα ή το τμήμα "Προτιμήσεις Απορρήτου"
3. Προσαρμόστε τις ρυθμίσεις όπως επιθυμείτε
4. Οι αλλαγές αποθηκεύονται αυτόματα

### Ορατότητα Δραστηριότητας

Ελέγξτε ποιος μπορεί να βλέπει τη δραστηριότητά σας με σχόλια σε κοινότητες του FastComments.

**Ρύθμιση:** Κάντε τη Δραστηριότητα Προφίλ Ιδιωτική

- **Όταν απενεργοποιημένο (προεπιλογή):** Οποιοσδήποτε μπορεί να δει τα πρόσφατα σχόλια και τη δραστηριότητά σας στο προφίλ
- **Όταν ενεργοποιημένο:** Μόνο εσείς μπορείτε να δείτε το ιστορικό σχολίων και τη ροή δραστηριότητας

Αυτό είναι χρήσιμο αν θέλετε να συμμετέχετε σε συζητήσεις αλλά προτιμάτε να κρατήσετε το ιστορικό σχολίων σας ιδιωτικό από άλλους χρήστες.

### Ορατότητα Σχολίων Προφίλ

Ελέγξτε αν άλλοι μπορούν να βλέπουν και να αφήνουν σχόλια στη σελίδα του προφίλ σας.

**Ρύθμιση:** Κάντε τα Σχόλια Προφίλ Ιδιωτικά

- **Όταν απενεργοποιημένο (προεπιλογή):** Οποιοσδήποτε μπορεί να δει και να αφήσει σχόλια στο προφίλ σας
- **Όταν ενεργοποιημένο:** Τα σχόλια προφίλ είναι κρυμμένα από άλλους· μόνο εσείς μπορείτε να δείτε τα σχόλια που έχουν αφήσει στο προφίλ σας

Χρησιμοποιήστε αυτή τη ρύθμιση αν θέλετε να περιορίσετε την αλληλεπίδραση στη σελίδα του προφίλ σας ενώ επιτρέπετε δραστηριότητα αλλού.

### Άμεσα Μηνύματα

Ελέγξτε αν άλλοι χρήστες μπορούν να σας στέλνουν άμεσα μηνύματα.

**Ρύθμιση:** Απενεργοποίηση Άμεσων Μηνυμάτων

- **Όταν απενεργοποιημένο (προεπιλογή):** Άλλοι χρήστες μπορούν να σας στέλνουν άμεσα μηνύματα
- **Όταν ενεργοποιημένο:** Η επιλογή Άμεσο Μήνυμα είναι κρυμμένη από το προφίλ σας· οι χρήστες δεν μπορούν να ξεκινήσουν συνομιλίες μαζί σας

Αυτό είναι χρήσιμο αν προτιμάτε να μην λαμβάνετε ιδιωτικά μηνύματα ή θέλετε να περιορίσετε τις ατομικές συνομιλίες.

### Έγκριση Σχολίων Προφίλ

Ελέγξτε πώς τα σχόλια στο προφίλ σας μεσολαβούνται πριν εμφανιστούν δημόσια.

**Ρύθμιση:** Λειτουργία Έγκρισης Σχολίων Προφίλ

Έχετε τέσσερις επιλογές για τη διαχείριση των σχολίων προφίλ:

#### 1. Αυτόματη Έγκριση Όλων
Όλα τα σχόλια που δημοσιεύονται στο προφίλ σας είναι αμέσως ορατά χωρίς να απαιτείται έγκριση.

**Καλύτερο για:** Χρήστες που θέλουν μια ανοιχτή, χωρίς περιορισμούς ενότητα σχολίων στο προφίλ.

#### 2. Αυτόματη Έγκριση για Έμπιστους (Προεπιλογή)
Τα σχόλια από έμπιστους χρήστες εγκρίνονται αυτόματα, ενώ τα σχόλια από νέους ή μη έμπιστους χρήστες απαιτούν χειροκίνητη έγκριση.

**Καλύτερο για:** Τους περισσότερους χρήστες - εξισορροπεί την ανοιχτότητα με την προστασία από spam ή ανεπιθύμητο περιεχόμενο.

**Τι καθιστά έναν χρήστη "έμπιστο":**
- Χρήστες με καλό karma
- Χρήστες με καθιερωμένο ιστορικό σχολίων
- Χρήστες που δεν έχουν επισημανθεί για spam ή παραβιάσεις

#### 3. Χειροκίνητη Έγκριση Όλων
Όλα τα σχόλια απαιτούν τη χειροκίνητη έγκρισή σας πριν εμφανιστούν στο προφίλ σας, ανεξάρτητα από το ποιος τα δημοσίευσε.

**Καλύτερο για:** Χρήστες που θέλουν πλήρη έλεγχο σε ό,τι εμφανίζεται στο προφίλ τους.

**Πώς λειτουργεί η έγκριση:**
- Τα νέα σχόλια κρατούνται για έλεγχο
- Λαμβάνετε ειδοποίηση όταν κάποιος σχολιάζει
- Μπορείτε να εγκρίνετε ή να απορρίψετε σχόλια μέσω email ή από το προφίλ σας

#### 4. Χειροκίνητη Έγκριση Νέων
Μόνο τα σχόλια από νέους ή μη έμπιστους χρήστες απαιτούν έγκριση. Τα σχόλια των καθιερωμένων μελών της κοινότητας εγκρίνονται αυτόματα.

**Καλύτερο για:** Χρήστες που θέλουν να εξετάζουν νέους χρήστες ενώ επιτρέπουν στα καθιερωμένα μέλη να σχολιάζουν ελεύθερα.

### Διαχείριση Εκκρεμών Σχολίων Προφίλ

Όταν τα σχόλια απαιτούν έγκριση, θα λαμβάνετε ειδοποιήσεις και μπορείτε να τα διαχειριστείτε με διάφορους τρόπους:

1. **Μέσω Email:** Εγκρίνετε ή απορρίψτε απευθείας από το email ειδοποίησης
2. **Από το Προφίλ σας:** Δείτε τα εκκρεμή σχόλια στην καρτέλα Σχόλια Προφίλ
3. **Κέντρο Ειδοποιήσεων:** Πρόσβαση σε εκκρεμή σχόλια από τις ειδοποιήσεις σας

### Αποκλεισμός Χρηστών

Εκτός από τις ρυθμίσεις ιδιωτικότητας προφίλ, μπορείτε να αποκλείσετε συγκεκριμένους χρήστες από το να αλληλεπιδρούν μαζί σας οπουδήποτε στο FastComments:

- Οι αποκλεισμένοι χρήστες δεν μπορούν να σχολιάσουν στο προφίλ σας
- Οι αποκλεισμένοι χρήστες δεν μπορούν να σας στείλουν άμεσα μηνύματα
- Τα σχόλια των αποκλεισμένων χρηστών είναι κρυμμένα από την προβολή σας
- Δεν θα βλέπετε τη δραστηριότητα ο ένας του άλλου

Για να αποκλείσετε έναν χρήστη, επισκεφθείτε το προφίλ του και κάντε κλικ στην επιλογή "Αποκλεισμός χρήστη".

### Καλές Πρακτικές Ιδιωτικότητας

Εδώ είναι μερικές προτεινόμενες ρυθμίσεις ιδιωτικότητας για διαφορετικές περιπτώσεις χρήσης:

**Δημόσιο Μέλος Κοινότητας:**
- Ορατότητα Δραστηριότητας: Δημόσια
- Σχόλια Προφίλ: Δημόσια
- Άμεσα Μηνύματα: Ενεργοποιημένα
- Έγκριση Σχολίων: Αυτόματη Έγκριση για Έμπιστους

**Ιδιωτικός Χρήστης:**
- Ορατότητα Δραστηριότητας: Ιδιωτική
- Σχόλια Προφίλ: Ιδιωτικά
- Άμεσα Μηνύματα: Απενεργοποιημένα
- Έγκριση Σχολίων: Χειροκίνητη Έγκριση Όλων

**Διαχειριστής/Δημόσιο Πρόσωπο:**
- Ορατότητα Δραστηριότητας: Δημόσια
- Σχόλια Προφίλ: Δημόσια
- Άμεσα Μηνύματα: Ενεργοποιημένα (ή απενεργοποιημένα αν είναι υπερβολικά πολλά)
- Έγκριση Σχολίων: Αυτόματη Έγκριση για Έμπιστους ή Χειροκίνητη Έγκριση Νέων
