FastComments User Profiles are highly customizable, allowing users to express their personality and build their online presence.

### Επεξεργασία του Προφίλ σας

Για να προσαρμόσετε το προφίλ σας, επισκεφθείτε τη σελίδα [Λεπτομέρειες Λογαριασμού](https://fastcomments.com/auth/my-account/edit-details). Εδώ μπορείτε να τροποποιήσετε όλες τις πτυχές της εμφάνισης και των πληροφοριών του προφίλ σας.

### Άβαταρ Προφίλ

Το άβατάρ σας είναι η εικόνα προφίλ που εμφανίζεται δίπλα στα σχόλιά σας και στη σελίδα του προφίλ σας.

**Για να αλλάξετε το άβατάρ σας:**
1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στον σύνδεσμο "Επεξεργασία Άβαταρ" (ορατό μόνο στο δικό σας προφίλ)
3. Ανεβάστε μια νέα εικόνα

**Απαιτήσεις Άβαταρ:**
- Συνιστώμενο μέγεθος: 200x200 pixels ή μεγαλύτερο
- Υποστηριζόμενα φορμά: JPG, PNG, GIF
- Οι τετράγωνες εικόνες λειτουργούν καλύτερα

Σημείωση: Οι χρήστες SSO μπορεί να έχουν το άβατάρ τους υπό έλεγχο από το σύστημα SSO του ιστότοπού τους.

### Φόντο Επικεφαλίδας Προφίλ

Το φόντο της επικεφαλίδας είναι μια μεγάλη εικόνα-banner στην κορυφή του προφίλ σας που μπορεί να προσαρμοστεί για να αντικατοπτρίζει το στυλ σας.

**Για να ανεβάσετε φόντο επικεφαλίδας:**
1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στο κουμπί "Μεταφόρτωση Επικεφαλίδας"
3. Επιλέξτε μια εικόνα από τη συσκευή σας

**Απαιτήσεις Επικεφαλίδας:**
- Συνιστώμενο μέγεθος: 1200x300 pixels
- Υποστηριζόμενα φορμά: JPG, PNG
- Οι οριζόντιες/ευρείες εικόνες λειτουργούν καλύτερα
- Οι εικόνες προσαρμόζονται αυτόματα στο μέγεθος που απαιτείται

Αν δεν ανεβάσετε προσαρμοσμένη επικεφαλίδα, θα εμφανιστεί ένα προεπιλεγμένο φόντο.

### Όνομα Εμφάνισης

Το όνομα εμφάνισης είναι ο τρόπος που εμφανίζεστε σε άλλους χρήστες σε όλο το FastComments. Σε αντίθεση με τα ονόματα χρήστη (που πρέπει να είναι μοναδικά), τα ονόματα εμφάνισης:

- Δεν χρειάζεται να είναι μοναδικά
- Μπορούν να περιέχουν κενά και ειδικούς χαρακτήρες
- Μπορούν να περιλαμβάνουν emoji

**Για να ορίσετε το όνομα εμφάνισης:**
1. Μεταβείτε στις [Λεπτομέρειες Λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε το προτιμώμενο όνομα εμφάνισης
3. Αποθηκεύστε τις αλλαγές

### Όνομα Χρήστη

Το όνομα χρήστη σας είναι ένας μοναδικός αναγνωριστής που χρησιμοποιείται για να συνδεθείτε στο FastComments.

- Πρέπει να είναι μοναδικό σε όλους τους χρήστες του FastComments
- Χρησιμοποιείται στη διεύθυνση προφίλ σας: `fastcomments.com/auth/user-profile/username`
- Δεν μπορεί να περιέχει κενά ή ειδικούς χαρακτήρες

**Σημείωση:** Οι χρήστες SSO μπορούν να έχουν μη-μοναδικά ονόματα χρήστη καθώς συνδέονται με έναν συγκεκριμένο tenant/community.

### Βιογραφικό

Προσθέστε ένα προσωπικό βιογραφικό για να πείτε στους άλλους για εσάς, τα ενδιαφέροντά σας ή τι κάνετε.

**Για να προσθέσετε βιογραφικό:**
1. Μεταβείτε στις [Λεπτομέρειες Λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε κείμενο στο πεδίο Βιογραφικό
3. Αποθηκεύστε τις αλλαγές

Το βιογραφικό σας εμφανίζεται σε περίοπτη θέση στη σελίδα του προφίλ σας και βοηθά τους άλλους να καταλάβουν ποιοι είστε.

### Διεύθυνση Ιστοσελίδας

Προσθέστε έναν σύνδεσμο προς την προσωπική σας ιστοσελίδα, το blog ή το portfolio σας.

**Για να προσθέσετε ιστοσελίδα:**
1. Μεταβείτε στις [Λεπτομέρειες Λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε τη διεύθυνση της ιστοσελίδας σας στο πεδίο Ιστοσελίδα
3. Αποθηκεύστε τις αλλαγές

Ο σύνδεσμος της ιστοσελίδας σας θα εμφανίζεται στο προφίλ σας με ένα εικονίδιο που μπορείτε να κάνετε κλικ.

### Σύνδεσμοι Κοινωνικών Δικτύων

Συνδέστε τα προφίλ κοινωνικών δικτύων σας με το προφίλ FastComments. Οι υποστηριζόμενες πλατφόρμες περιλαμβάνουν:

- **Facebook** - Σύνδεσμος στο προφίλ σας στο Facebook
- **Instagram** - Σύνδεσμος στο λογαριασμό σας στο Instagram
- **LinkedIn** - Σύνδεσμος στο προφίλ σας στο LinkedIn
- **Προσαρμοσμένοι Σύνδεσμοι** - Προσθέστε οποιουσδήποτε άλλους συνδέσμους κοινωνικών δικτύων ή προφίλ στο web

**Για να προσθέσετε συνδέσμους κοινωνικών δικτύων:**
1. Μεταβείτε στις [Λεπτομέρειες Λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε τις διευθύνσεις URL για τους λογαριασμούς κοινωνικών δικτύων σας
3. Αποθηκεύστε τις αλλαγές

Οι σύνδεσμοι κοινωνικών δικτύων εμφανίζονται στο προφίλ σας με αναγνωρίσιμα εικονίδια πλατφόρμας, διευκολύνοντας τα μέλη της κοινότητας να συνδεθούν μαζί σας αλλού.

### Χώρα

Μπορείτε προαιρετικά να ορίσετε τη χώρα σας, η οποία εμφανίζει ένα εικονίδιο σημαίας δίπλα στο όνομά σας.

**Για να ορίσετε τη χώρα σας:**
1. Μεταβείτε στις [Λεπτομέρειες Λογαριασμού](https://fastcomments.com/auth/my-account/edit-details)
2. Επιλέξτε τη χώρα σας από το αναπτυσσόμενο μενού
3. Αποθηκεύστε τις αλλαγές

Η σημαία της χώρας εμφανίζεται δίπλα στο όνομα χρήστη σας στο προφίλ σας.
