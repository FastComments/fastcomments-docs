Τα Προφίλ Χρηστών του FastComments είναι ιδιαίτερα προσαρμόσιμα, επιτρέποντας στους χρήστες να εκφράσουν την προσωπικότητά τους και να χτίσουν την διαδικτυακή τους παρουσία.

### Επεξεργασία Προφίλ

Για να προσαρμόσετε το προφίλ σας, επισκεφθείτε τη σελίδα σας [Account Details](https://fastcomments.com/auth/my-account/edit-details). Εδώ μπορείτε να τροποποιήσετε όλες τις πτυχές της εμφάνισης και των πληροφοριών του προφίλ σας.

### Avatar Προφίλ

Το avatar σας είναι η εικόνα προφίλ που εμφανίζεται δίπλα στα σχόλιά σας και στη σελίδα του προφίλ σας.

**Για να αλλάξετε το avatar σας:**
1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στον σύνδεσμο "Επεξεργασία Avatar" (ορατός μόνο στο δικό σας προφίλ)
3. Μεταφορτώστε μια νέα εικόνα

**Απαιτήσεις Avatar:**
- Συνιστώμενο μέγεθος: 200x200 pixels ή μεγαλύτερο
- Υποστηριζόμενα φορμά: JPG, PNG, GIF
- Οι τετράγωνες εικόνες λειτουργούν καλύτερα

Σημείωση: Οι χρήστες SSO μπορεί να έχουν το avatar τους ελεγχόμενο από το σύστημα SSO του ιστότοπού τους.

### Επικεφαλίδα Προφίλ (Header Background)

Η επικεφαλίδα είναι μια μεγάλη εικόνα banner στην κορυφή του προφίλ σας που μπορεί να προσαρμοστεί για να αντικατοπτρίζει το στυλ σας.

**Για να ανεβάσετε μια επικεφαλίδα:**
1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στο κουμπί "Ανέβασμα Κεφαλίδας"
3. Επιλέξτε μια εικόνα από τη συσκευή σας

**Απαιτήσεις Επικεφαλίδας:**
- Συνιστώμενο μέγεθος: 1200x300 pixels
- Υποστηριζόμενα φορμά: JPG, PNG
- Οι εικόνες σε οριζόντια/ευρεία μορφή λειτουργούν καλύτερα
- Οι εικόνες προσαρμόζονται αυτόματα για να ταιριάξουν

Εάν δεν ανεβάσετε προσαρμοσμένη επικεφαλίδα, θα εμφανιστεί μια προεπιλεγμένη εικόνα φόντου.

### Εμφανιζόμενο Όνομα

Το εμφανιζόμενο όνομα είναι ο τρόπος με τον οποίο εμφανίζεστε σε άλλους χρήστες σε όλο το FastComments. Σε αντίθεση με τα usernames (που πρέπει να είναι μοναδικά), τα εμφανιζόμενα ονόματα:

- Δεν χρειάζεται να είναι μοναδικά
- Μπορούν να περιέχουν κενά και ειδικούς χαρακτήρες
- Μπορούν να περιλαμβάνουν emojis

**Για να ορίσετε το εμφανιζόμενο όνομα:**
1. Μεταβείτε στο [Account Details](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε το προτιμώμενο εμφανιζόμενο όνομα
3. Αποθηκεύστε τις αλλαγές

### Όνομα Χρήστη

Το όνομα χρήστη είναι ένας μοναδικός αναγνωριστής που χρησιμοποιείται για την είσοδο στο FastComments.

- Πρέπει να είναι μοναδικό μεταξύ όλων των χρηστών του FastComments
- Χρησιμοποιείται στο URL του προφίλ σας: `fastcomments.com/auth/user-profile/username`
- Δεν μπορεί να περιέχει κενά ή ειδικούς χαρακτήρες

**Σημείωση:** Οι χρήστες SSO μπορούν να έχουν μη μοναδικά ονόματα χρήστη επειδή συνδέονται με έναν συγκεκριμένο tenant/κοινότητα.

### Βιογραφία

Προσθέστε μια προσωπική βιογραφία για να πείτε στους άλλους για εσάς, τα ενδιαφέροντά σας ή τι κάνετε.

**Για να προσθέσετε βιογραφία:**
1. Μεταβείτε στο [Account Details](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε κείμενο στο πεδίο Βιογραφία
3. Αποθηκεύστε τις αλλαγές

Η βιογραφία σας εμφανίζεται με έμφαση στη σελίδα του προφίλ σας και βοηθά τους άλλους να καταλάβουν ποιοι είστε.

### Ιστοσελίδα (Website URL)

Προσθέστε έναν σύνδεσμο στην προσωπική σας ιστοσελίδα, το blog ή το portfolio σας.

**Για να προσθέσετε ιστοσελίδα:**
1. Μεταβείτε στο [Account Details](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε το URL της ιστοσελίδας σας στο πεδίο Website
3. Αποθηκεύστε τις αλλαγές

Ο σύνδεσμος της ιστοσελίδας σας θα εμφανίζεται στο προφίλ σας με ένα εικονίδιο που μπορεί να κλικαριστεί.

### Συνδέσεις Κοινωνικών Δικτύων

Συνδέστε τα προφίλ κοινωνικών δικτύων σας με το προφίλ σας στο FastComments. Οι υποστηριζόμενες πλατφόρμες περιλαμβάνουν:

- **Facebook** - Σύνδεσμος στο προφίλ σας στο Facebook
- **Instagram** - Σύνδεσμος στον λογαριασμό σας στο Instagram
- **LinkedIn** - Σύνδεσμος στο προφίλ σας στο LinkedIn
- **Προσαρμοσμένοι Σύνδεσμοι** - Προσθέστε οποιαδήποτε άλλη σελίδα κοινωνικού δικτύου ή προφίλ web

**Για να προσθέσετε συνδέσμους κοινωνικών δικτύων:**
1. Μεταβείτε στο [Account Details](https://fastcomments.com/auth/my-account/edit-details)
2. Εισάγετε τα URLs των λογαριασμών κοινωνικών δικτύων σας
3. Αποθηκεύστε τις αλλαγές

Οι σύνδεσμοι κοινωνικών δικτύων εμφανίζονται στο προφίλ σας με αναγνωρίσιμα εικονίδια πλατφορμών, καθιστώντας εύκολο για τα μέλη της κοινότητας να συνδεθούν μαζί σας αλλού.

### Χώρα

Μπορείτε προαιρετικά να ορίσετε τη χώρα σας, που εμφανίζει ένα εικονίδιο σημαίας δίπλα στο όνομά σας.

**Για να ορίσετε τη χώρα σας:**
1. Μεταβείτε στο [Account Details](https://fastcomments.com/auth/my-account/edit-details)
2. Επιλέξτε τη χώρα σας από το αναπτυσσόμενο μενού
3. Αποθηκεύστε τις αλλαγές

Η σημαία της χώρας εμφανίζεται δίπλα στο όνομα χρήστη σας στο προφίλ σας.

### Εφεδρικό Email (Backup Email)

Μπορείτε να προσθέσετε ένα εφεδρικό email στο λογαριασμό σας για σκοπούς ανάκτησης. Εάν χάσετε την πρόσβαση στο κύριο email σας, το εφεδρικό email
μπορεί να χρησιμοποιηθεί για είσοδο ή ανάκτηση του ονόματος χρήστη σας.

**Για να προσθέσετε εφεδρικό email:**
1. Μεταβείτε στο [Account Details](https://fastcomments.com/auth/my-account/edit-details)
2. Κάντε κλικ στο **Ορισμός Εφεδρικού Email** κάτω από το πεδίο email
3. Εισάγετε τη διεύθυνση εφεδρικού email και υποβάλετε
4. Κάντε κλικ στο σύνδεσμο επιβεβαίωσης που θα σταλεί σε αυτή τη διεύθυνση

Το εφεδρικό email χρησιμοποιείται μόνο για ανάκτηση λογαριασμού και δεν θα λαμβάνει ειδοποιήσεις ή άλλα email. Μπορείτε να το αλλάξετε ή να το αφαιρέσετε οποιαδήποτε στιγμή.

Σημείωση: Το εφεδρικό email είναι διαθέσιμο μόνο για κανονικούς χρήστες, όχι για χρήστες SSO.
