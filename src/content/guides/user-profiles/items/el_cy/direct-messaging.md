Direct Messaging (DM) επιτρέπει στους χρήστες του FastComments να έχουν ιδιωτικές, προσωπικές συνομιλίες ένα προς ένα μεταξύ τους, χωριστά από τα δημόσια σχόλια και τις αλληλεπιδράσεις προφίλ.

### Τι είναι τα Άμεσα Μηνύματα;

Τα Άμεσα Μηνύματα παρέχουν:
- **Ιδιωτικές συνομιλίες** μεταξύ δύο χρηστών
- **Ανταλλαγή μηνυμάτων σε πραγματικό χρόνο** με ειδοποιήσεις
- **Ιστορικό συνομιλιών** που διατηρείται μεταξύ των συνεδριών
- **Δείκτες κατάστασης σύνδεσης** για να βλέπετε πότε κάποιος είναι διαθέσιμος
- **Παρακολούθηση μη αναγνωσμένων μηνυμάτων** για να μην χάνετε νέα μηνύματα

### Ξεκινώντας ένα Άμεσο Μήνυμα

Υπάρχουν διάφοροι τρόποι για να ξεκινήσετε μια συνομιλία DM:

**Από το Προφίλ Χρήστη:**
1. Επισκεφτείτε τη σελίδα προφίλ του χρήστη
2. Κάντε κλικ στην καρτέλα "Άμεσα Μηνύματα" ή στο κουμπί "Αποστολή Μηνύματος"
3. Πληκτρολογήστε το μήνυμά σας και στείλτε το

**Από ένα Σχόλιο:**
Κάποιες υλοποιήσεις επιτρέπουν το κλικ στο όνομα ή το avatar ενός χρήστη για γρήγορη πρόσβαση στο προφίλ και τις επιλογές μηνυμάτων του.

**Άμεσος Σύνδεσμος:**
Αν έχετε ένα ID συνομιλίας ή σύνδεσμο ανακατεύθυνσης DM, μπορείτε να έχετε άμεση πρόσβαση στις συνομιλίες.

### Πρόσβαση στα Μηνύματά σας

Για να δείτε όλες τις συνομιλίες άμεσων μηνυμάτων σας:

1. Επισκεφτείτε τη σελίδα προφίλ σας
2. Κάντε κλικ στην καρτέλα "Άμεσα Μηνύματα"
3. Δείτε τη λίστα με όλες τις συνομιλίες σας

Κάθε συνομιλία εμφανίζει:
- Το avatar και το όνομα του άλλου συμμετέχοντα
- Την κατάσταση σύνδεσης τους (online/offline) (πράσινος δείκτης όταν είναι online)
- Προεπισκόπηση του τελευταίου μηνύματος
- Αριθμό μη αναγνωσμένων μηνυμάτων (αν υπάρχουν)
- Χρονική σήμανση της τελευταίας δραστηριότητας

### Προβολή Συνομιλίας

Όταν ανοίξετε μια συνομιλία, θα δείτε:

- **Ολόκληρο ιστορικό μηνυμάτων** - Όλα τα μηνύματα μεταξύ εσάς και του άλλου χρήστη
- **Ενημερώσεις σε πραγματικό χρόνο** - Τα νέα μηνύματα εμφανίζονται άμεσα μέσω WebSocket
- **Κατάσταση σύνδεσης** - Δείτε αν το άλλο άτομο είναι αυτή τη στιγμή online
- **Χρονικές σήμανσεις μηνυμάτων** - Πότε στάλθηκε κάθε μήνυμα
- **Σύνθεση μηνύματος** - Πλαίσιο κειμένου για να πληκτρολογήσετε και να στείλετε νέα μηνύματα

### Ειδοποιήσεις Μηνυμάτων

Θα λαμβάνετε ειδοποιήσεις για νέα άμεσα μηνύματα:

- **Ειδοποιήσεις εντός εφαρμογής** - Μετρητής εικονιδίου στο προφίλ σας
- **Ειδοποιήσεις μέσω email** - Ανάλογα με τις ρυθμίσεις ειδοποιήσεων σας
- **Άμεσες ειδοποιήσεις** - Αποστολή ειδοποιήσεων όταν είστε συνδεδεμένοι

Διαχειριστείτε τις προτιμήσεις ειδοποιήσεων στις [Ρυθμίσεις Λογαριασμού](https://fastcomments.com/auth/my-account/edit-notifications).

### Απόρρητο και Αποκλεισμός

**Απενεργοποίηση Άμεσων Μηνυμάτων:**
Αν δεν θέλετε να λαμβάνετε DM, μπορείτε να τα απενεργοποιήσετε εντελώς:
1. Μεταβείτε στις Ρυθμίσεις Απορρήτου στο προφίλ σας
2. Ενεργοποιήστε την επιλογή "Απενεργοποίηση Άμεσων Μηνυμάτων"
3. Η επιλογή DM θα κρυφτεί από το προφίλ σας

Για περισσότερες λεπτομέρειες, δείτε τις [Ρυθμίσεις Απορρήτου](/guide-user-profiles.html#privacy-settings).

**Αποκλεισμός Χρηστών:**
Αν κάποιος σας παρενοχλεί μέσω DM:
1. Επισκεφτείτε το προφίλ του
2. Αποκλείστε τον χρήστη
3. Δεν θα μπορεί πλέον να σας στέλνει μηνύματα

Οι αποκλεισμένοι χρήστες δεν μπορούν:
- Να σας στέλνουν άμεσα μηνύματα
- Να βλέπουν τη δραστηριότητά σας
- Να σχολιάζουν στο προφίλ σας
- Να αλληλεπιδρούν μαζί σας οπουδήποτε στο FastComments

### Διαχείριση Συνομιλιών

**Απόκρυψη Συνομιλιών:**
Μπορείτε να αποκρύψετε συνομιλίες που δεν θέλετε πια να βλέπετε στη λίστα συνομιλιών σας:
1. Ανοίξτε την καρτέλα Άμεσα Μηνύματα
2. Βρείτε τη συνομιλία που θέλετε να αποκρύψετε
3. Επιλέξτε την επιλογή απόκρυψης/αρχειοθέτησης

Οι κρυμμένες συνομιλίες δεν θα εμφανίζονται στη κύρια λίστα σας αλλά μπορείτε να τις προσπελάσετε αν το άλλο άτομο στείλει νέο μήνυμα.

**Σήμανση ως Αναγνωσμένο:**
Τα μηνύματα σημαίνονται αυτόματα ως αναγνωσμένα όταν τα προβάλλετε. Μπορείτε επίσης να επισημάνετε χειροκίνητα ολόκληρες συνομιλίες ως αναγνωσμένες:
1. Ανοίξτε την καρτέλα Άμεσα Μηνύματα
2. Επιλέξτε μια συνομιλία
3. Επιλέξτε "Σήμανση ως Αναγνωσμένο"

Αυτό θα καθαρίσει τον μετρητή μη αναγνωσμένων για εκείνη τη συνομιλία.

### Βέλτιστες Πρακτικές

**Πότε να Χρησιμοποιείτε τα Άμεσα Μηνύματα:**
- Για να κάνετε ερωτήσεις παρακολούθησης ιδιωτικά
- Για να ευχαριστήσετε κάποιον για χρήσιμες συμβουλές
- Να συζητήσετε εκτός θέματος χωρίς να γεμίζετε τα δημόσια σχόλια
- Να συντονιστείτε με άλλα μέλη της κοινότητας
- Να παρέχετε ιδιωτικά σχόλια ή προτάσεις

**Εθιμοτυπία DM:**
- Να είστε σεβαστικοί και επαγγελματικοί
- Μην στέλνετε ανεπιθύμητα μηνύματα σε χρήστες
- Σεβαστείτε αν κάποιος δεν απαντά ή απενεργοποιεί τα DM
- Κρατήστε τις συνομιλίες σχετικές και εποικοδομητικές
- Μην κοινοποιείτε τα DM άλλων χωρίς άδεια

**Συμβουλές Ασφαλείας:**
- Μην μοιράζεστε προσωπικά στοιχεία (τηλέφωνο, διεύθυνση, κ.λπ.) εκτός αν εμπιστεύεστε το άτομο
- Αποκλείστε και αναφέρετε χρήστες που παρενοχλούν ή κακοποιούν μέσω DM
- Να είστε προσεκτικοί με ύποπτους συνδέσμους ή αιτήματα
- Χρησιμοποιήστε τη λειτουργία αποκλεισμού αν κάποιος σας κάνει να αισθάνεστε άβολα

### Περιορισμοί και Σημειώσεις

**Σε ποιον μπορείτε να στείλετε μήνυμα:**
- Σε οποιονδήποτε χρήστη του FastComments που δεν έχει απενεργοποιήσει τα DM
- Σε χρήστες που δεν σας έχουν αποκλείσει
- Σε χρήστες σε όλες τις κοινότητες του FastComments

**Περιεχόμενο Μηνυμάτων:**
- Υποστηρίζονται μηνύματα κειμένου
- Τα μηνύματα ακολουθούν τις ίδιες πολιτικές περιεχομένου με τα σχόλια
- Ακατάλληλο περιεχόμενο μπορεί να αναφερθεί

**Εύρος Συνομιλιών:**
- Οι συνομιλίες DM είναι ιδιωτικές μεταξύ δύο ατόμων
- Δεν υπάρχει ομαδική συνομιλία (προς το παρόν μόνο ένα προς ένα)
- Το ιστορικό συνομιλιών διατηρείται επ' αόριστον

### Αντιμετώπιση Προβλημάτων

**Δεν μπορείτε να στείλετε μήνυμα;**
Ο άλλος χρήστης μπορεί να:
- Έχει απενεργοποιήσει τα άμεσα μηνύματα στις ρυθμίσεις απορρήτου του
- Σας έχει αποκλείσει
- Έχει διαγράψει τον λογαριασμό του

**Δεν λαμβάνετε ειδοποιήσεις;**
Ελέγξτε τις ρυθμίσεις ειδοποιήσεων στις [Ρυθμίσεις Λογαριασμού](https://fastcomments.com/auth/my-account/edit-notifications) για να βεβαιωθείτε ότι οι ειδοποιήσεις DM είναι ενεργοποιημένες.

**Τα μηνύματα δεν αποστέλλονται;**
- Ελέγξτε τη σύνδεση στο διαδίκτυο
- Ανανεώστε τη σελίδα και δοκιμάστε ξανά
- Βεβαιωθείτε ότι δεν έχετε αποκλειστεί
- Επικοινωνήστε με την υποστήριξη αν το πρόβλημα συνεχίζεται
