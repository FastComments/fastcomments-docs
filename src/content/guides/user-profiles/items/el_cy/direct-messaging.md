Άμεσα Μηνύματα (DM) επιτρέπουν στους χρήστες του FastComments να έχουν ιδιωτικές, one-on-one συζητήσεις μεταξύ τους, ξεχωριστές από τα δημόσια σχόλια και τις αλληλεπιδράσεις προφίλ.

### Τι είναι τα Άμεσα Μηνύματα;

Τα Άμεσα Μηνύματα παρέχουν:
- **Ιδιωτικές συνομιλίες** μεταξύ δύο χρηστών
- **Ανταλλαγή μηνυμάτων σε πραγματικό χρόνο** με ειδοποιήσεις
- **Ιστορικό συνομιλιών** που διατηρείται ανάμεσα στις συνεδρίες
- **Ένδειξη κατάστασης σύνδεσης** για να βλέπετε πότε κάποιος είναι διαθέσιμος
- **Παρακολούθηση μη αναγνωσμένων μηνυμάτων** για να μένετε ενημερωμένοι για νέα μηνύματα

### Ξεκινώντας ένα Άμεσο Μήνυμα

Υπάρχουν πολλοί τρόποι να ξεκινήσετε μια συνομιλία DM:

**Από το Προφίλ Χρήστη:**
1. Επισκεφθείτε τη σελίδα προφίλ του χρήστη
2. Κάντε κλικ στην καρτέλα "Direct Messages" ή στο κουμπί "Send Message"
3. Πληκτρολογήστε το μήνυμά σας και στείλτε το

**Από ένα Σχόλιο:**
Κάποιες υλοποιήσεις επιτρέπουν το κλικ στο όνομα ή το avatar ενός χρήστη για γρήγορη πρόσβαση στο προφίλ και τις επιλογές μηνυμάτων του.

**Άμεσος Σύνδεσμος:**
Εάν έχετε ένα ID συνομιλίας ή σύνδεσμο ανακατεύθυνσης DM, μπορείτε να έχετε πρόσβαση σε συνομιλίες απευθείας.

### Πρόσβαση στα Μηνύματά σας

Για να δείτε όλες τις συνομιλίες άμεσων μηνυμάτων σας:

1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στην καρτέλα "Direct Messages"
3. Δείτε μια λίστα με όλες τις συνομιλίες σας

Κάθε συνομιλία εμφανίζει:
- Το avatar και το όνομα του άλλου συμμετέχοντα
- Την κατάσταση σύνδεσής τους (πράσινος δείκτης όταν είναι online)
- Μια προεπισκόπηση του τελευταίου μηνύματος
- Πλήθος μη αναγνωσμένων μηνυμάτων (εάν υπάρχουν)
- Στίγμα χρόνου της τελευταίας δραστηριότητας

### Προβολή Συνομιλίας

Όταν ανοίγετε μια συνομιλία, θα δείτε:

- **Ολόκληρο το ιστορικό μηνυμάτων** - Όλα τα μηνύματα μεταξύ εσάς και του άλλου χρήστη
- **Ενημερώσεις σε πραγματικό χρόνο** - Νέα μηνύματα εμφανίζονται αμέσως μέσω WebSocket
- **Κατάσταση σύνδεσης** - Δείτε αν το άλλο άτομο είναι αυτή τη στιγμή online
- **Χρονικές ετικέτες μηνυμάτων** - Πότε στάλθηκε κάθε μήνυμα
- **Σύνθεση μηνύματος** - Πλαίσιο κειμένου για να γράψετε και να στείλετε νέα μηνύματα

### Ειδοποιήσεις Μηνυμάτων

Θα λαμβάνετε ειδοποιήσεις για νέα άμεσα μηνύματα:

- **Ειδοποιήσεις εντός εφαρμογής** - Μετρητής σήματος στο προφίλ σας
- **Ειδοποιήσεις μέσω email** - Ανάλογα με τις ρυθμίσεις ειδοποίησής σας
- **Ειδοποιήσεις σε πραγματικό χρόνο** - Άμεσες ειδοποιήσεις όταν είστε συνδεδεμένοι

Διαχειριστείτε τις προτιμήσεις ειδοποιήσεων στις [Ρυθμίσεις λογαριασμού](https://fastcomments.com/auth/my-account/edit-notifications).

### Απόρρητο και Αποκλεισμός

**Απενεργοποίηση Άμεσων Μηνυμάτων:**
Εάν δεν θέλετε να λαμβάνετε DMs, μπορείτε να τα απενεργοποιήσετε εντελώς:
1. Μεταβείτε στις Ρυθμίσεις Απορρήτου στο προφίλ σας
2. Ενεργοποιήστε "Disable Direct Messages"
3. Η επιλογή DM θα κρυφτεί από το προφίλ σας

Για περισσότερες λεπτομέρειες, δείτε τις [Ρυθμίσεις απορρήτου](/guide-user-profiles.html#privacy-settings).

**Αποκλεισμός χρηστών:**
Εάν κάποιος σας παρενοχλεί μέσω DM:
1. Επισκεφθείτε το προφίλ του
2. Αποκλείστε το χρήστη
3. Αυτός δεν θα μπορεί πλέον να σας στέλνει μηνύματα

Οι αποκλεισμένοι χρήστες δεν μπορούν:
- Να σας στείλουν άμεσα μηνύματα
- Να βλέπουν τη δραστηριότητά σας
- Να σχολιάζουν στο προφίλ σας
- Να αλληλεπιδρούν μαζί σας οπουδήποτε στο FastComments

### Διαχείριση Συνομιλιών

**Απόκρυψη Συνομιλιών:**
Μπορείτε να κρύψετε συνομιλίες που δεν θέλετε πλέον να βλέπετε στη λίστα συνομιλιών σας:
1. Ανοίξτε την καρτέλα Direct Messages
2. Βρείτε τη συνομιλία που θέλετε να αποκρύψετε
3. Επιλέξτε την επιλογή απόκρυψης/αρχειοθέτησης

Οι κρυμμένες συνομιλίες δεν θα εμφανίζονται στην κύρια λίστα σας αλλά μπορούν να προσπελαστούν εάν το άλλο άτομο στείλει νέο μήνυμα.

**Σήμανση ως Αναγνωσμένο:**
Τα μηνύματα σημειώνονται αυτόματα ως αναγνωσμένα όταν τα βλέπετε. Μπορείτε επίσης να σημειώσετε χειροκίνητα ολόκληρες συνομιλίες ως αναγνωσμένες:
1. Ανοίξτε την καρτέλα Direct Messages
2. Επιλέξτε μια συνομιλία
3. Επιλέξτε "Mark as Read"

Αυτό καθαρίζει τον μετρητή μη αναγνωσμένων για εκείνη τη συνομιλία.

### Καλύτερες Πρακτικές

**Πότε να Χρησιμοποιείτε τα Άμεσα Μηνύματα:**
- Για να κάνετε ιδιωτικές διευκρινιστικές ερωτήσεις
- Να ευχαριστήσετε κάποιον για χρήσιμες συμβουλές
- Να συζητήσετε θέματα εκτός θέματος χωρίς να γεμίζετε τα δημόσια σχόλια
- Να συντονιστείτε με άλλα μέλη της κοινότητας
- Να δώσετε ιδιωτική ανατροφοδότηση ή προτάσεις

**Εθιμοτυπία στα DM:**
- Να είστε σεβαστικοί και επαγγελματίες
- Μην στέλνετε ανεπιθύμητα μηνύματα σε χρήστες
- Σεβαστείτε αν κάποιος δεν απαντά ή απενεργοποιεί τα DMs
- Κρατήστε τις συνομιλίες σχετικές και εποικοδομητικές
- Μην μοιράζεστε τα DM άλλων χωρίς άδεια

**Συμβουλές Ασφαλείας:**
- Μην μοιράζεστε προσωπικές πληροφορίες (τηλέφωνο, διεύθυνση, κ.λπ.) εκτός αν εμπιστεύεστε το άτομο
- Αποκλείστε και αναφέρετε χρήστες που παρενοχλούν ή κακοποιούν μέσω DM
- Να είστε προσεκτικοί με ύποπτους συνδέσμους ή αιτήματα
- Χρησιμοποιήστε τη λειτουργία αποκλεισμού αν κάποιος σας κάνει να νιώθετε άβολα

### Περιορισμοί και Σημειώσεις

**Σε ποιους μπορείτε να στείλετε μήνυμα:**
- Σε οποιονδήποτε χρήστη του FastComments που δεν έχει απενεργοποιήσει τα DMs
- Σε χρήστες που δεν σας έχουν αποκλείσει
- Σε χρήστες σε όλες τις κοινότητες του FastComments

**Περιεχόμενο μηνύματος:**
- Υποστηρίζονται μηνύματα κειμένου
- Τα μηνύματα ακολουθούν τις ίδιες πολιτικές περιεχομένου με τα σχόλια
- Ακατάλληλο περιεχόμενο μπορεί να αναφερθεί

**Πεδίο συνομιλίας:**
- Οι συνομιλίες DM είναι ιδιωτικές μεταξύ δύο ατόμων
- Δεν υπάρχει ομαδική συνομιλία (επί του παρόντος μόνο ένα-προς-ένα)
- Το ιστορικό συνομιλιών διατηρείται επ' αόριστον

### Αντιμετώπιση προβλημάτων

**Δεν μπορείτε να στείλετε μήνυμα;**
Ο άλλος χρήστης μπορεί να έχει:
- Απενεργοποιήσει τα άμεσα μηνύματα στις ρυθμίσεις απορρήτου του
- Σας έχει αποκλείσει
- Διαγράψει το λογαριασμό του

**Δεν λαμβάνετε ειδοποιήσεις;**
Ελέγξτε τις ρυθμίσεις ειδοποιήσεων στις [Ρυθμίσεις λογαριασμού](https://fastcomments.com/auth/my-account/edit-notifications) για να βεβαιωθείτε ότι οι ειδοποιήσεις DM είναι ενεργοποιημένες.

**Τα μηνύματα δεν αποστέλλονται;**
- Ελέγξτε τη σύνδεσή σας στο διαδίκτυο
- Ανανέωση της σελίδας και δοκιμάστε ξανά
- Βεβαιωθείτε ότι δεν έχετε αποκλειστεί
- Επικοινωνήστε με την υποστήριξη εάν το πρόβλημα συνεχίζεται
