Άμεσα Μηνύματα (DM) επιτρέπουν στους χρήστες του FastComments να έχουν ιδιωτικές, ένας-προς-έναν συνομιλίες μεταξύ τους, ξεχωριστές από τα δημόσια σχόλια και τις αλληλεπιδράσεις προφίλ.

### Τι είναι τα Άμεσα Μηνύματα;

Τα Άμεσα Μηνύματα παρέχουν:
- **Ιδιωτικές συνομιλίες** μεταξύ δύο χρηστών
- **Μηνύματα σε πραγματικό χρόνο** με ειδοποιήσεις
- **Ιστορικό συνομιλίας** που διατηρείται μεταξύ των συνεδριών
- **Δείκτες κατάστασης σύνδεσης** για να βλέπετε πότε κάποιος είναι διαθέσιμος
- **Παρακολούθηση μη αναγνωσμένων μηνυμάτων** για να μην χάνετε νέα μηνύματα

### Ξεκινώντας ένα Άμεσο Μήνυμα

Υπάρχουν αρκετοί τρόποι για να ξεκινήσετε μια συνομιλία DM:

**Από το Προφίλ Χρήστη:**
1. Επισκεφθείτε τη σελίδα προφίλ του χρήστη
2. Κάντε κλικ στην καρτέλα "Direct Messages" ή στο κουμπί "Send Message"
3. Πληκτρολογήστε το μήνυμά σας και στείλτε το

**Από ένα Σχόλιο:**
Ορισμένες υλοποιήσεις επιτρέπουν το κλικ στο όνομα ή το avatar ενός χρήστη για γρήγορη πρόσβαση στο προφίλ και τις επιλογές μηνυμάτων του.

**Άμεσος Σύνδεσμος:**
Αν έχετε ένα ID συνομιλίας ή σύνδεσμο ανακατεύθυνσης DM, μπορείτε να έχετε πρόσβαση στις συνομιλίες απευθείας.

### Πρόσβαση στα Μηνύματά σας

Για να δείτε όλες τις συνομιλίες άμεσων μηνυμάτων σας:

1. Επισκεφθείτε τη σελίδα του προφίλ σας
2. Κάντε κλικ στην καρτέλα "Direct Messages"
3. Δείτε μια λίστα με όλες τις συνομιλίες σας

Κάθε συνομιλία εμφανίζει:
- Το avatar και το όνομα του άλλου συμμετέχοντα
- Την κατάσταση σύνδεσής τους (πράσινος δείκτης όταν είναι σε σύνδεση)
- Προεπισκόπηση του τελευταίου μηνύματος
- Αριθμό μη αναγνωσμένων μηνυμάτων (αν υπάρχουν)
- Χρονοσφραγίδα τελευταίας δραστηριότητας

### Προβολή Συνομιλίας

Όταν ανοίγετε μια συνομιλία, θα δείτε:

- **Πλήρες ιστορικό μηνυμάτων** - Όλα τα μηνύματα μεταξύ εσάς και του άλλου χρήστη
- **Ενημερώσεις σε πραγματικό χρόνο** - Νέα μηνύματα εμφανίζονται αμέσως μέσω WebSocket
- **Κατάσταση σύνδεσης** - Δείτε αν το άλλο άτομο είναι αυτή τη στιγμή σε σύνδεση
- **Χρονοσφραγίδες μηνυμάτων** - Πότε εστάλη κάθε μήνυμα
- **Σύνθεση μηνύματος** - Πλαίσιο κειμένου για να πληκτρολογήσετε και να στείλετε νέα μηνύματα

### Ειδοποιήσεις Μηνυμάτων

Θα λαμβάνετε ειδοποιήσεις για νέα άμεσα μηνύματα:

- **Ειδοποιήσεις εντός εφαρμογής** - Δείκτης μετρητή στο προφίλ σας
- **Ειδοποιήσεις μέσω ηλεκτρονικού ταχυδρομείου** - Ανάλογα με τις ρυθμίσεις ειδοποιήσεων σας
- **Άμεσες ειδοποιήσεις** - Άμεσες ειδοποιήσεις όταν είστε συνδεδεμένοι

Διαχειριστείτε τις προτιμήσεις ειδοποιήσεων στις [Account Settings](https://fastcomments.com/auth/my-account/notification-settings).

### Απόρρητο και Αποκλεισμός

**Απενεργοποίηση Άμεσων Μηνυμάτων:**
Αν δεν θέλετε να λαμβάνετε DM, μπορείτε να τα απενεργοποιήσετε εντελώς:
1. Πηγαίνετε στις Ρυθμίσεις Απορρήτου στο προφίλ σας
2. Ενεργοποιήστε την επιλογή "Disable Direct Messages"
3. Η επιλογή DM θα κρυφτεί από το προφίλ σας

Για περισσότερες λεπτομέρειες, δείτε τις [Privacy Settings](/guides/user-profiles/privacy-settings).

**Αποκλεισμός Χρηστών:**
Αν κάποιος σας παρενοχλεί μέσω DM:
1. Επισκεφθείτε το προφίλ του
2. Αποκλείστε τον χρήστη
3. Δεν θα μπορεί πλέον να σας στέλνει μηνύματα

Οι αποκλεισμένοι χρήστες δεν μπορούν να:
- Σας στείλουν άμεσα μηνύματα
- Δουν τη δραστηριότητά σας
- Σχολιάσουν στο προφίλ σας
- Αλληλεπιδράσουν μαζί σας οπουδήποτε στο FastComments

### Διαχείριση Συνομιλιών

**Απόκρυψη Συνομιλιών:**
Μπορείτε να αποκρύψετε συνομιλίες που δεν θέλετε πλέον να βλέπετε στη λίστα συνομιλιών σας:
1. Ανοίξτε την καρτέλα Direct Messages
2. Βρείτε τη συνομιλία που θέλετε να αποκρύψετε
3. Επιλέξτε την επιλογή απόκρυψης/αρχειοθέτησης

Οι κρυφές συνομιλίες δεν θα εμφανίζονται στην κύρια λίστα σας αλλά μπορούν να προσπελαστούν αν το άλλο άτομο στείλει νέο μήνυμα.

**Σήμανση ως Αναγνωσμένο:**
Τα μηνύματα επισημαίνονται αυτόματα ως αναγνωσμένα όταν τα προβάλλετε. Μπορείτε επίσης να επισημάνετε χειροκίνητα ολόκληρες συνομιλίες ως αναγνωσμένες:
1. Ανοίξτε την καρτέλα Direct Messages
2. Επιλέξτε μια συνομιλία
3. Επιλέξτε "Mark as Read"

Αυτό καθαρίζει τον μετρητή μη αναγνωσμένων για εκείνη τη συνομιλία.

### Καλές Πρακτικές

**Πότε να χρησιμοποιείτε τα Άμεσα Μηνύματα:**
- Για να κάνετε ερωτήσεις παρακολούθησης ιδιωτικά
- Για να ευχαριστήσετε κάποιον για χρήσιμες συμβουλές
- Για να συζητήσετε εκτός θέματος χωρίς να κάνουν ακαταστασία τα δημόσια σχόλια
- Για να συντονιστείτε με άλλα μέλη της κοινότητας
- Για να δώσετε ιδιωτικά σχόλια ή προτάσεις

**Εθιμοτυπία DM:**
- Να είστε σεβαστικοί και επαγγελματικοί
- Μην στέλνετε ανεπιθύμητα μηνύματα μαζικά
- Σεβαστείτε αν κάποιος δεν απαντά ή απενεργοποιεί τα DM
- Κρατήστε τις συνομιλίες σχετικές και εποικοδομητικές
- Μην κοινοποιείτε DMs άλλων χωρίς άδεια

**Συμβουλές Ασφαλείας:**
- Μην κοινοποιείτε προσωπικές πληροφορίες (τηλέφωνο, διεύθυνση, κ.λπ.) εκτός αν εμπιστεύεστε το άτομο
- Αποκλείστε και αναφέρετε χρήστες που παρενοχλούν ή κακοποιούν μέσω DM
- Να είστε προσεκτικοί με ύποπτους συνδέσμους ή αιτήματα
- Χρησιμοποιήστε τη λειτουργία αποκλεισμού αν κάποιος σας κάνει να νιώθετε άβολα

### Περιορισμοί και Σημειώσεις

**Σε ποιον μπορείτε να στείλετε μήνυμα:**
- Σε οποιονδήποτε χρήστη του FastComments που δεν έχει απενεργοποιήσει τα DM
- Σε χρήστες που δεν σας έχουν αποκλείσει
- Σε χρήστες σε όλες τις κοινότητες του FastComments

**Περιεχόμενο Μηνυμάτων:**
- Υποστηρίζονται μηνύματα κειμένου
- Τα μηνύματα ακολουθούν τις ίδιες πολιτικές περιεχομένου με τα σχόλια
- Ακατάλληλο περιεχόμενο μπορεί να αναφερθεί

**Εύρος Συνομιλίας:**
- Οι συνομιλίες DM είναι ιδιωτικές μεταξύ δύο ατόμων
- Δεν υπάρχει ομαδική αποστολή μηνυμάτων (επί του παρόντος μόνο ένας-προς-έναν)
- Το ιστορικό συνομιλίας διατηρείται επ' αόριστον

### Αντιμετώπιση Προβλημάτων

**Δεν μπορείτε να στείλετε μήνυμα;**
Ο άλλος χρήστης μπορεί να:
- Έχει απενεργοποιήσει τα άμεσα μηνύματα στις ρυθμίσεις απορρήτου του
- Σας έχει αποκλείσει
- Έχει διαγράψει τον λογαριασμό του

**Δεν λαμβάνετε ειδοποιήσεις;**
Ελέγξτε τις ρυθμίσεις ειδοποιήσεων στις [Account Settings](https://fastcomments.com/auth/my-account/notification-settings) για να βεβαιωθείτε ότι οι ειδοποιήσεις DM είναι ενεργοποιημένες.

**Τα μηνύματα δεν αποστέλλονται;**
- Ελέγξτε τη σύνδεση στο διαδίκτυο
- Ανανέωση της σελίδας και δοκιμάστε ξανά
- Βεβαιωθείτε ότι δεν έχετε αποκλειστεί
- Επικοινωνήστε με την υποστήριξη αν το πρόβλημα παραμένει
