Las Notas de Usuario te permiten mantener anotaciones privadas y personales sobre otros usuarios de FastComments. Estas notas son completamente privadas y solo visibles para ti.

### ¿Qué son las Notas de Usuario?

Las Notas de Usuario son:
- **Anotaciones privadas** que creas sobre otros usuarios
- **Visibles solo para ti**: el usuario sobre el que escribes no puede verlas
- **Persistentes**: guardadas y accesibles cada vez que visites el perfil de ese usuario
- **Útiles para recordar** contexto, interacciones o detalles importantes sobre los usuarios

### ¿Por qué usar las Notas de Usuario?

Las Notas de Usuario son útiles para:

**Moderadores:**
- Rastrear el historial de interacciones con usuarios específicos
- Anotar patrones de advertencia u observaciones de comportamiento
- Recordar el contexto para decisiones de moderación
- Documentar por qué se tomaron ciertas acciones

**Gestores de la comunidad:**
- Recordar colaboradores valiosos y su experiencia
- Anotar usuarios que podrían ser buenos candidatos para roles especiales
- Rastrear seguimientos necesarios con miembros de la comunidad

**Usuarios regulares:**
- Recordar cómo conoces a alguien en diferentes comunidades
- Anotar temas que has tratado con usuarios concretos
- Llevar un seguimiento de colaboradores útiles en tus áreas de interés
- Recordar detalles personales compartidos en conversaciones (dentro de límites apropiados)

### Crear una Nota de Usuario

Para añadir una nota sobre un usuario:

1. Visita la página de perfil del usuario
2. Busca la sección "Nota de usuario" o "Añadir nota"
3. Haz clic para crear una nueva nota
4. Escribe tu nota (hasta 1000 caracteres)
5. Guarda la nota

La nota se guardará y aparecerá cada vez que visites el perfil de ese usuario.

### Ver Notas de Usuario

Tus notas aparecen al ver el perfil del usuario asociado:

- **Ubicación:** Normalmente en una sección dedicada en la página de perfil
- **Visibilidad:** Solo visible para ti cuando hayas iniciado sesión
- **Contexto:** Aparece junto a la información pública del usuario

Esto facilita recordar el contexto importante sobre usuarios con los que interactúas regularmente.

### Editar Notas de Usuario

Para actualizar una nota existente:

1. Visita el perfil del usuario
2. Encuentra tu nota existente
3. Haz clic en "Editar" o haz clic dentro del campo de la nota
4. Realiza tus cambios
5. Guarda la nota actualizada

### Eliminar Notas de Usuario

Para eliminar una nota:

1. Visita el perfil del usuario
2. Encuentra tu nota
3. Haz clic en "Eliminar" o en la opción de eliminar
4. Confirma la eliminación

Una vez eliminada, la nota se elimina permanentemente y no puede recuperarse.

### Límite de caracteres

Las Notas de Usuario están limitadas a **1000 caracteres**. Esto fomenta:
- Notas concisas y enfocadas
- Puntos clave en lugar de narrativas largas
- Información de referencia rápida

Si necesitas más espacio, considera mantener documentación separada fuera de FastComments.

### Privacidad y seguridad

**Privacidad completa:**
- Las notas se almacenan con tu ID de usuario
- Solo tú puedes ver las notas que has creado
- El usuario sobre el que escribes no puede ver tus notas
- Otros usuarios no pueden ver tus notas
- Los administradores del sitio no pueden ver tus notas personales (se almacenan por separado de los registros de moderación)

**Nota sobre moderación:**
Las Notas de Usuario son diferentes de las notas de moderación o las señales:
- Usa Notas de Usuario para referencia personal
- Usa las herramientas de reporte de moderación para violaciones o problemas
- Las Notas de Usuario no desencadenan ninguna acción ni alertas

### Buenas prácticas

**Qué incluir:**
- Cómo conociste o conoces al usuario
- Temas sobre los que tiene conocimientos
- Interacciones previas útiles
- Contexto de conversaciones pasadas
- Áreas de experiencia o interés
- Recordatorios para hacer seguimiento

**Qué evitar:**
- No uses las notas para almacenar información personal sensible
- Evita contenido inflamatorio o inapropiado incluso en notas privadas
- No confíes en las notas para información crítica (guarda copias en otros sitios)
- Recuerda que las notas son solo para tu referencia, no registros oficiales

**Uso profesional:**
Para moderadores y gestores de la comunidad:
- Mantén las notas factuales y objetivas
- Incluye fechas para información sensible al tiempo
- Anota patrones en lugar de incidentes aislados
- Usa las notas para complementar (no reemplazar) los registros oficiales de moderación

### Casos de uso

**Ejemplo 1: Gestor de la comunidad**
> "Activo en discusiones de tecnología. Trabaja como desarrollador backend. Se ofreció a ayudar con la documentación. Hacer seguimiento sobre su contribución a la wiki."

**Ejemplo 2: Moderador**
> "Usuario servicial pero a veces entra en debates acalorados. Generalmente respetuoso. Respondió bien al recordatorio previo sobre civismo."

**Ejemplo 3: Usuario regular**
> "Conocido en hilos de gaming. Juega los mismos juegos que yo. Compartió un enlace a un tutorial útil. Con sede en la zona horaria del Reino Unido."

### Limitaciones

**Qué no hacen las Notas de Usuario:**
- No se sincronizan entre dispositivos (a menos que hayas iniciado sesión en la misma cuenta)
- No aparecen en comentarios u otros contextos, solo en perfiles
- No afectan el perfil del usuario ni su visibilidad de ninguna forma
- No son buscables (actualmente)
- No desencadenan notificaciones ni acciones

**Almacenamiento:**
- Las notas se almacenan indefinidamente a menos que las elimines
- Máximo 1000 caracteres por nota
- Una nota por usuario (actualiza la nota existente en lugar de crear varias)

### Detalles técnicos

Las Notas de Usuario:
- Se almacenan en la base de datos con tu ID de usuario como creador
- Están asociadas con el ID del usuario objetivo
- Cifradas en tránsito (HTTPS)
- Sujetas a las políticas estándar de retención de datos de FastComments
- No incluidas en exportaciones de datos públicos ni en respuestas de la API

### Solución de problemas

**La nota no se guarda:**
- Comprueba que hayas iniciado sesión
- Asegúrate de que la nota tenga menos de 1000 caracteres
- Intenta actualizar la página y añadirla de nuevo
- Revisa tu conexión a Internet

**La nota no aparece:**
- Asegúrate de haber iniciado sesión en la cuenta que creó la nota
- Actualiza la página de perfil del usuario
- Borra la caché del navegador si es necesario

**Nota eliminada accidentalmente:**
Desafortunadamente, las notas eliminadas no pueden recuperarse. Considera mantener copias de seguridad de información importante fuera de FastComments.
