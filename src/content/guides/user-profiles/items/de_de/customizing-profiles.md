FastComments-Benutzerprofile sind hochgradig anpassbar und ermöglichen es Nutzern, ihre Persönlichkeit auszudrücken und ihre Online-Präsenz aufzubauen.

### Ihr Profil bearbeiten

Um Ihr Profil anzupassen, besuchen Sie Ihre [Kontodetails](https://fastcomments.com/auth/my-account/edit-details)-Seite. Hier können Sie alle Aspekte des Erscheinungsbildes und der Informationen Ihres Profils ändern.

### Profil-Avatar

Ihr Avatar ist das Profilbild, das neben Ihren Kommentaren und auf Ihrer Profilseite angezeigt wird.

**So ändern Sie Ihren Avatar:**
1. Besuchen Sie Ihre Profilseite
2. Klicken Sie auf den Link "Avatar bearbeiten" (nur auf Ihrem eigenen Profil sichtbar)
3. Laden Sie ein neues Bild hoch

**Anforderungen an den Avatar:**
- Empfohlene Größe: 200x200 Pixel oder größer
- Unterstützte Formate: JPG, PNG, GIF
- Quadratische Bilder funktionieren am besten

Hinweis: SSO-Benutzer können ihren Avatar möglicherweise durch das SSO-System ihrer Seite gesteuert haben.

### Hintergrundbild des Profils (Header)

Der Header-Hintergrund ist ein großes Bannerbild oben auf Ihrem Profil, das angepasst werden kann, um Ihren Stil widerzuspiegeln.

**So laden Sie ein Header-Hintergrundbild hoch:**
1. Besuchen Sie Ihre Profilseite
2. Klicken Sie auf die Schaltfläche "Header hochladen"
3. Wählen Sie ein Bild von Ihrem Gerät aus

**Anforderungen an den Header:**
- Empfohlene Größe: 1200x300 Pixel
- Unterstützte Formate: JPG, PNG
- Bilder im Querformat/mit breiter Ausrichtung funktionieren am besten
- Bilder werden automatisch angepasst, damit sie passen

Wenn Sie keinen eigenen Header hochladen, wird ein Standardhintergrund angezeigt.

### Anzeigename

Ihr Anzeigename ist die Bezeichnung, unter der Sie anderen Nutzern auf FastComments erscheinen. Im Gegensatz zu Benutzernamen (die eindeutig sein müssen), gilt für Anzeigenamen:

- Sie müssen nicht eindeutig sein
- Sie können Leerzeichen und Sonderzeichen enthalten
- Sie können Emojis enthalten

**So legen Sie Ihren Anzeigenamen fest:**
1. Gehen Sie zu [Kontodetails](https://fastcomments.com/auth/my-account/edit-details)
2. Geben Sie Ihren gewünschten Anzeigenamen ein
3. Änderungen speichern

### Benutzername

Ihr Benutzername ist ein eindeutiger Identifikator, der zum Anmelden bei FastComments verwendet wird.

- Muss für alle FastComments-Benutzer eindeutig sein
- Verwendet in Ihrer Profil-URL: `fastcomments.com/auth/user-profile/username`
- Darf keine Leerzeichen oder Sonderzeichen enthalten

**Hinweis:** SSO-Benutzer können nicht eindeutige Benutzernamen haben, da diese an einen bestimmten Mandanten/Community gebunden sind.

### Bio

Fügen Sie eine persönliche Bio hinzu, um anderen etwas über sich, Ihre Interessen oder Ihre Tätigkeit zu erzählen.

**So fügen Sie eine Bio hinzu:**
1. Gehen Sie zu [Kontodetails](https://fastcomments.com/auth/my-account/edit-details)
2. Geben Sie Text in das Bio-Feld ein
3. Änderungen speichern

Ihre Bio erscheint prominent auf Ihrer Profilseite und hilft anderen zu verstehen, wer Sie sind.

### Website-URL

Fügen Sie einen Link zu Ihrer persönlichen Website, Ihrem Blog oder Portfolio hinzu.

**So fügen Sie eine Website hinzu:**
1. Gehen Sie zu [Kontodetails](https://fastcomments.com/auth/my-account/edit-details)
2. Geben Sie Ihre Website-URL in das Website-Feld ein
3. Änderungen speichern

Ihr Website-Link wird auf Ihrem Profil mit einem anklickbaren Symbol angezeigt.

### Social-Media-Links

Verbinden Sie Ihre Social-Media-Profile mit Ihrem FastComments-Profil. Unterstützte Plattformen sind unter anderem:

- **Facebook** - Link zu Ihrem Facebook-Profil
- **Instagram** - Link zu Ihrem Instagram-Konto
- **LinkedIn** - Link zu Ihrem LinkedIn-Profil
- **Benutzerdefinierte Links** - Fügen Sie beliebige andere Social-Media- oder Webprofile hinzu

**So fügen Sie Social Links hinzu:**
1. Gehen Sie zu [Kontodetails](https://fastcomments.com/auth/my-account/edit-details)
2. Geben Sie URLs für Ihre Social-Media-Konten ein
3. Änderungen speichern

Social-Media-Links erscheinen auf Ihrem Profil mit erkennbaren Plattform-Symbolen, sodass Community-Mitglieder sich leicht anderweitig mit Ihnen verbinden können.

### Land

Sie können optional Ihr Land festlegen, das ein Flaggen-Symbol neben Ihrem Namen anzeigt.

**So legen Sie Ihr Land fest:**
1. Gehen Sie zu [Kontodetails](https://fastcomments.com/auth/my-account/edit-details)
2. Wählen Sie Ihr Land aus dem Dropdown-Menü
3. Änderungen speichern

Die Länderflagge erscheint neben Ihrem Benutzernamen auf Ihrem Profil.

### Backup-E-Mail

Sie können Ihrer Kontoeinstellungen eine Backup-E-Mail-Adresse für Wiederherstellungszwecke hinzufügen. Wenn Sie den Zugriff auf Ihre primäre E-Mail verlieren, kann Ihre Backup-E-Mail
zum Anmelden oder zur Wiederherstellung Ihres Benutzernamens verwendet werden.

**So fügen Sie eine Backup-E-Mail hinzu:**
1. Gehen Sie zu [Kontodetails](https://fastcomments.com/auth/my-account/edit-details)
2. Klicken Sie unter dem E-Mail-Feld auf **Backup-E-Mail festlegen**
3. Geben Sie Ihre Backup-E-Mail-Adresse ein und senden Sie diese ab
4. Klicken Sie auf den Verifizierungslink, der an diese Adresse gesendet wird

Ihre Backup-E-Mail wird nur zur Kontowiederherstellung verwendet und erhält keine Benachrichtigungen oder andere E-Mails. Sie können diese jederzeit ändern oder entfernen.

Hinweis: Die Backup-E-Mail ist nur für reguläre Benutzer verfügbar, nicht für SSO-Benutzer.
