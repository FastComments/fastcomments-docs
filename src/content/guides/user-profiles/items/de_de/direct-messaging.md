Direktnachrichten (DM) ermöglichen FastComments-Benutzern private, eins-zu-eins-Unterhaltungen miteinander, getrennt von öffentlichen Kommentaren und Profilinteraktionen.

### Was sind Direktnachrichten?

Direktnachrichten bieten:
- **Private Unterhaltungen** zwischen zwei Benutzern
- **Echtzeitnachrichten** mit Benachrichtigungen
- **Konversationsverlauf**, der über Sitzungen hinweg erhalten bleibt
- **Online-Statusindikatoren**, um zu sehen, wann jemand verfügbar ist
- **Verfolgung ungelesener Nachrichten**, um neue Nachrichten im Blick zu behalten

### Starten einer Direktnachricht

Es gibt mehrere Möglichkeiten, ein DM-Gespräch zu starten:

**Vom Benutzerprofil:**
1. Besuchen Sie die Profilseite des Benutzers
2. Klicken Sie auf den Tab "Direktnachrichten" oder den Button "Nachricht senden"
3. Geben Sie Ihre Nachricht ein und senden Sie sie

**Von einem Kommentar:**
Einige Implementierungen erlauben das Klicken auf den Namen oder das Avatar eines Benutzers, um schnell auf dessen Profil und Messaging-Optionen zuzugreifen.

**Direkter Link:**
Wenn Sie eine Konversations-ID oder einen DM-Weiterleitungslink haben, können Sie direkt auf Konversationen zugreifen.

### Zugriff auf Ihre Nachrichten

Um alle Ihre Direktnachrichten-Konversationen anzuzeigen:

1. Besuchen Sie Ihre Profilseite
2. Klicken Sie auf den Tab "Direktnachrichten"
3. Sehen Sie eine Liste aller Ihrer Konversationen

Jede Konversation zeigt:
- Das Avatar und den Namen des anderen Teilnehmers
- Dessen Online-/Offline-Status (grüner Indikator, wenn online)
- Eine Vorschau der letzten Nachricht
- Anzahl ungelesener Nachrichten (falls vorhanden)
- Zeitstempel der letzten Aktivität

### Konversationsansicht

Wenn Sie eine Konversation öffnen, sehen Sie:

- **Vollständiger Nachrichtenverlauf** - Alle Nachrichten zwischen Ihnen und dem anderen Benutzer
- **Echtzeitaktualisierungen** - Neue Nachrichten erscheinen sofort über WebSocket
- **Online-Status** - Sehen Sie, ob die andere Person gerade online ist
- **Zeitstempel der Nachrichten** - Wann jede Nachricht gesendet wurde
- **Nachrichten verfassen** - Textfeld zum Eingeben und Senden neuer Nachrichten

### Nachrichten-Benachrichtigungen

Sie erhalten Benachrichtigungen für neue Direktnachrichten:

- **In-App-Benachrichtigungen** - Badge-Zähler in Ihrem Profil
- **E-Mail-Benachrichtigungen** - Je nach Ihren Benachrichtigungseinstellungen
- **Echtzeitwarnungen** - Sofortige Benachrichtigungen, wenn Sie angemeldet sind

Verwalten Sie Benachrichtigungseinstellungen in Ihren [Kontoeinstellungen](https://fastcomments.com/auth/my-account/notification-settings).

### Datenschutz und Blockieren

**Direktnachrichten deaktivieren:**
Wenn Sie keine DMs erhalten möchten, können Sie diese komplett deaktivieren:
1. Gehen Sie zu den Datenschutzeinstellungen in Ihrem Profil
2. Aktivieren Sie "Direktnachrichten deaktivieren"
3. Die DM-Option wird in Ihrem Profil ausgeblendet

Weitere Details finden Sie unter [Datenschutzeinstellungen](/guides/user-profiles/privacy-settings).

**Benutzer blockieren:**
Wenn Sie über DM belästigt werden:
1. Besuchen Sie deren Profil
2. Blockieren Sie den Benutzer
3. Dieser kann Ihnen dann keine Nachrichten mehr senden

Blockierte Benutzer können nicht:
- Ihnen Direktnachrichten senden
- Ihre Aktivitäten sehen
- Auf Ihrem Profil kommentieren
- Überall auf FastComments mit Ihnen interagieren

### Konversationen verwalten

**Konversationen ausblenden:**
Sie können Konversationen ausblenden, die Sie nicht mehr in Ihrer Konversationsliste sehen möchten:
1. Öffnen Sie Ihren Tab "Direktnachrichten"
2. Finden Sie die Konversation, die Sie ausblenden möchten
3. Wählen Sie die Option zum Ausblenden/Archivieren

Ausgeblendete Konversationen erscheinen nicht in Ihrer Hauptliste, können aber wieder aufgerufen werden, wenn die andere Person eine neue Nachricht sendet.

**Als gelesen markieren:**
Nachrichten werden automatisch als gelesen markiert, wenn Sie sie ansehen. Sie können ganze Konversationen auch manuell als gelesen markieren:
1. Öffnen Sie Ihren Tab "Direktnachrichten"
2. Wählen Sie eine Konversation aus
3. Wählen Sie "Als gelesen markieren"

Dies setzt den ungelesenen Zähler für diese Konversation zurück.

### Beste Vorgehensweisen

**Wann Direktnachrichten verwenden:**
- Folgefragen privat stellen
- Jemandem für hilfreiche Ratschläge danken
- Off-Topic-Themen besprechen, ohne öffentliche Kommentare zu überladen
- Mit anderen Community-Mitgliedern koordinieren
- Privates Feedback oder Vorschläge geben

**DM-Etikette:**
- Seien Sie respektvoll und professionell
- Versenden Sie keine Spam-Nachrichten an Nutzer
- Respektieren Sie es, wenn jemand nicht antwortet oder DMs deaktiviert hat
- Halten Sie Unterhaltungen relevant und konstruktiv
- Teilen Sie keine DMs anderer ohne Erlaubnis

**Sicherheitshinweise:**
- Teilen Sie keine persönlichen Informationen (Telefon, Adresse usw.), es sei denn, Sie vertrauen der Person
- Blockieren und melden Sie Benutzer, die per DM belästigen oder missbrauchen
- Seien Sie vorsichtig bei verdächtigen Links oder Anfragen
- Nutzen Sie die Blockierfunktion, wenn sich jemand unangenehm verhält

### Einschränkungen und Hinweise

**Wen Sie kontaktieren können:**
- Jeden FastComments-Benutzer, der DMs nicht deaktiviert hat
- Benutzer, die Sie nicht blockiert haben
- Benutzer aus allen FastComments-Communities

**Nachrichteninhalt:**
- Textnachrichten werden unterstützt
- Nachrichten unterliegen denselben Inhaltsrichtlinien wie Kommentare
- Unangemessene Inhalte können gemeldet werden

**Umfang der Konversation:**
- DM-Konversationen sind privat zwischen zwei Personen
- Keine Gruppen-Nachrichten (derzeit nur eins-zu-eins)
- Der Konversationsverlauf wird unbegrenzt gespeichert

### Fehlerbehebung

**Kann keine Nachricht senden?**
Der andere Benutzer könnte:
- Direktnachrichten in seinen Datenschutzeinstellungen deaktiviert haben
- Sie blockiert haben
- Sein Konto gelöscht haben

**Erhalte keine Benachrichtigungen?**
Prüfen Sie Ihre Benachrichtigungseinstellungen unter [Kontoeinstellungen](https://fastcomments.com/auth/my-account/notification-settings), um sicherzustellen, dass DM-Benachrichtigungen aktiviert sind.

**Nachrichten werden nicht gesendet?**
- Überprüfen Sie Ihre Internetverbindung
- Aktualisieren Sie die Seite und versuchen Sie es erneut
- Stellen Sie sicher, dass Sie nicht blockiert wurden
- Kontaktieren Sie den Support, wenn das Problem weiterhin besteht
