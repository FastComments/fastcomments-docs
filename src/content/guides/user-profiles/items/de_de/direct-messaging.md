Direktnachrichten (DM) ermöglichen FastComments-Benutzern private, eins-zu-eins Unterhaltungen miteinander, getrennt von öffentlichen Kommentaren und Profilinteraktionen.

### Was sind Direktnachrichten?

Direktnachrichten bieten:
- **Private Unterhaltungen** zwischen zwei Benutzern
- **Echtzeitnachrichten** mit Benachrichtigungen
- **Konversationsverlauf**, der über Sitzungen hinweg erhalten bleibt
- **Online-Status**-Anzeigen, um zu sehen, wann jemand verfügbar ist
- **Verfolgung ungelesener Nachrichten**, um neue Nachrichten nicht zu verpassen

### Starten einer Direktnachricht

Es gibt mehrere Möglichkeiten, eine DM-Konversation zu beginnen:

**Vom Benutzerprofil:**
1. Öffnen Sie die Profilseite des Benutzers
2. Klicken Sie auf die Registerkarte "Direktnachrichten" oder die Schaltfläche "Nachricht senden"
3. Geben Sie Ihre Nachricht ein und senden Sie sie

**Aus einem Kommentar:**
Bei einigen Implementierungen kann man auf den Namen oder das Avatar eines Benutzers klicken, um schnell auf dessen Profil und Messaging-Optionen zuzugreifen.

**Direkter Link:**
Wenn Sie eine Konversations-ID oder einen DM-Weiterleitungslink haben, können Sie direkt auf Konversationen zugreifen.

### Zugriff auf Ihre Nachrichten

Um alle Ihre Direktnachrichten-Konversationen anzuzeigen:

1. Öffnen Sie Ihre Profilseite
2. Klicken Sie auf die Registerkarte "Direktnachrichten"
3. Sehen Sie eine Liste aller Ihrer Konversationen

Jede Konversation zeigt:
- Avatar und Name des anderen Teilnehmers
- Ihren Online-/Offline-Status (grüner Indikator, wenn online)
- Eine Vorschau der letzten Nachricht
- Anzahl ungelesener Nachrichten (falls vorhanden)
- Zeitstempel der letzten Aktivität

### Gesprächsansicht

Wenn Sie eine Konversation öffnen, sehen Sie:

- **Vollständiger Nachrichtenverlauf** - Alle Nachrichten zwischen Ihnen und dem anderen Benutzer
- **Echtzeit-Aktualisierungen** - Neue Nachrichten erscheinen sofort über WebSocket
- **Online-Status** - Sehen Sie, ob die andere Person derzeit online ist
- **Zeitstempel der Nachrichten** - Wann jede Nachricht gesendet wurde
- **Nachrichtenverfassung** - Textfeld zum Schreiben und Senden neuer Nachrichten

### Nachrichtenbenachrichtigungen

Sie erhalten Benachrichtigungen für neue Direktnachrichten:

- **In-App-Benachrichtigungen** - Abzeichen-Zähler auf Ihrem Profil
- **E-Mail-Benachrichtigungen** - Abhängig von Ihren Benachrichtigungseinstellungen
- **Echtzeit-Benachrichtigungen** - Sofortige Benachrichtigungen, wenn Sie eingeloggt sind

Verwalten Sie Ihre Benachrichtigungseinstellungen in Ihren [Kontoeinstellungen](https://fastcomments.com/auth/my-account/edit-notifications).

### Datenschutz und Blockieren

**Deaktivieren von Direktnachrichten:**
Wenn Sie keine DMs erhalten möchten, können Sie sie vollständig deaktivieren:
1. Gehen Sie zu den Datenschutzeinstellungen in Ihrem Profil
2. Aktivieren Sie "Direktnachrichten deaktivieren"
3. Die DM-Option wird in Ihrem Profil ausgeblendet

Für weitere Details siehe [Datenschutzeinstellungen](/guide-user-profiles.html#privacy-settings).

**Benutzer blockieren:**
Wenn Sie per DM belästigt werden:
1. Besuchen Sie deren Profil
2. Blockieren Sie den Benutzer
3. Er/Sie kann Ihnen dann keine Nachrichten mehr senden

Blockierte Benutzer können nicht:
- Ihnen Direktnachrichten senden
- Ihre Aktivität sehen
- Ihr Profil kommentieren
- Mit Ihnen irgendwo auf FastComments interagieren

### Konversationen verwalten

**Konversationen ausblenden:**
Sie können Konversationen ausblenden, die Sie nicht mehr in Ihrer Konversationsliste sehen möchten:
1. Öffnen Sie Ihre Direktnachrichten-Registerkarte
2. Finden Sie die Konversation, die Sie ausblenden möchten
3. Wählen Sie die Option ausblenden/archivieren

Ausgeblendete Konversationen erscheinen nicht in Ihrer Hauptliste, können jedoch wieder aufgerufen werden, wenn die andere Person eine neue Nachricht sendet.

**Als gelesen markieren:**
Nachrichten werden automatisch als gelesen markiert, wenn Sie sie ansehen. Sie können auch ganze Konversationen manuell als gelesen markieren:
1. Öffnen Sie Ihre Direktnachrichten-Registerkarte
2. Wählen Sie eine Konversation aus
3. Wählen Sie "Als gelesen markieren"

Dies setzt den ungelesenen Zähler für diese Konversation zurück.

### Beste Vorgehensweisen

**Wann Direktnachrichten verwenden:**
- Nachfragen privat stellen
- Jemandem für hilfreiche Ratschläge danken
- Themen abseits der Diskussion besprechen, ohne öffentliche Kommentare zu belasten
- Koordination mit anderen Mitgliedern der Community
- Privates Feedback oder Vorschläge geben

**Etikette für Direktnachrichten:**
- Seien Sie respektvoll und professionell
- Spam-Nachrichten vermeiden
- Respektieren Sie es, wenn jemand nicht antwortet oder DMs deaktiviert
- Halten Sie Unterhaltungen relevant und konstruktiv
- Teilen Sie keine DMs anderer ohne Erlaubnis

**Sicherheitshinweise:**
- Teilen Sie keine persönlichen Informationen (Telefon, Adresse usw.), es sei denn, Sie vertrauen der Person
- Blockieren und melden Sie Benutzer, die in DMs belästigen oder missbrauchen
- Seien Sie vorsichtig bei verdächtigen Links oder Anfragen
- Verwenden Sie die Blockierfunktion, wenn sich jemand unangenehm verhält

### Einschränkungen und Hinweise

**Wen Sie anschreiben können:**
- Jeder FastComments-Benutzer, der DMs nicht deaktiviert hat
- Benutzer, die Sie nicht blockiert haben
- Benutzer in allen FastComments-Communities

**Nachrichteninhalt:**
- Textnachrichten werden unterstützt
- Nachrichten unterliegen denselben Inhaltsrichtlinien wie Kommentare
- Unangemessene Inhalte können gemeldet werden

**Umfang der Konversation:**
- DM-Konversationen sind privat zwischen zwei Personen
- Keine Gruppenunterhaltungen (derzeit nur eins-zu-eins)
- Der Konversationsverlauf wird unbegrenzt aufbewahrt

### Fehlerbehebung

**Kann keine Nachricht senden?**
Der andere Benutzer hat möglicherweise:
- Direktnachrichten in den Datenschutzeinstellungen deaktiviert
- Sie blockiert
- Das Konto gelöscht

**Erhalten Sie keine Benachrichtigungen?**
Überprüfen Sie Ihre Benachrichtigungseinstellungen in Ihren [Kontoeinstellungen](https://fastcomments.com/auth/my-account/edit-notifications), um sicherzustellen, dass DM-Benachrichtigungen aktiviert sind.

**Nachrichten werden nicht gesendet?**
- Überprüfen Sie Ihre Internetverbindung
- Aktualisieren Sie die Seite und versuchen Sie es erneut
- Stellen Sie sicher, dass Sie nicht blockiert wurden
- Kontaktieren Sie den Support, wenn das Problem weiterhin besteht
