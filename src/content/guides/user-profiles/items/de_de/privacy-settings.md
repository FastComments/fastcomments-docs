FastComments gibt Ihnen granulare Kontrolle über die Privatsphäre Ihres Profils und darüber, wer mit Ihnen interagieren kann. Alle Datenschutzeinstellungen können im Abschnitt "Datenschutzeinstellungen" Ihrer Profilseite verwaltet werden.

### Zugriff auf Datenschutzeinstellungen

So verwalten Sie Ihre Datenschutzeinstellungen:
1. Rufen Sie Ihre Profilseite auf
2. Klicken Sie auf die Registerkarte oder den Abschnitt "Datenschutzeinstellungen"
3. Passen Sie Ihre Einstellungen nach Wunsch an
4. Änderungen werden automatisch gespeichert

### Sichtbarkeit der Aktivitäten

Steuern Sie, wer Ihre Kommentaraktivitäten in den FastComments-Communities sehen kann.

**Einstellung:** Profilaktivität privat machen

- **Wenn deaktiviert (Standard):** Jeder kann Ihre jüngsten Kommentare und Aktivitäten auf Ihrem Profil sehen
- **Wenn aktiviert:** Nur Sie können Ihre Kommentarhistorie und Ihren Activity-Feed sehen

Das ist nützlich, wenn Sie an Diskussionen teilnehmen möchten, aber Ihre Kommentarhistorie vor anderen Nutzern verbergen wollen.

### Sichtbarkeit von Profilkommentaren

Steuern Sie, ob andere Nutzer Kommentare auf Ihrer Profilseite sehen und hinterlassen können.

**Einstellung:** Profilkommentare privat machen

- **Wenn deaktiviert (Standard):** Jeder kann Ihre Profilkommentare sehen und hinterlassen
- **Wenn aktiviert:** Profilkommentare sind für andere verborgen; nur Sie können Kommentare sehen, die auf Ihrem Profil hinterlassen wurden

Verwenden Sie diese Einstellung, wenn Sie die Interaktion auf Ihrer Profilseite begrenzen möchten, während an anderen Stellen weiterhin Aktivität möglich ist.

### Direktnachrichten

Steuern Sie, ob andere Nutzer Ihnen Direktnachrichten senden können.

**Einstellung:** Direktnachrichten deaktivieren

- **Wenn deaktiviert (Standard):** Andere Nutzer können Ihnen Direktnachrichten senden
- **Wenn aktiviert:** Die Option für Direktnachrichten wird in Ihrem Profil ausgeblendet; Nutzer können keine Unterhaltungen mit Ihnen initiieren

Das ist hilfreich, wenn Sie keine privaten Nachrichten erhalten möchten oder Einzelgespräche einschränken wollen.

### Genehmigung von Profilkommentaren

Steuern Sie, wie Kommentare auf Ihrem Profil moderiert werden, bevor sie öffentlich erscheinen.

**Einstellung:** Modus für Profilkommentargenehmigung

Sie haben vier Optionen zur Verwaltung von Profilkommentaren:

#### 1. Alle automatisch genehmigen
Alle auf Ihrem Profil veröffentlichten Kommentare sind sofort sichtbar, ohne dass eine Genehmigung erforderlich ist.

**Am besten für:** Nutzer, die einen offenen, uneingeschränkten Bereich für Profilkommentare möchten.

#### 2. Automatisch vertrauenswürdige genehmigen (Standard)
Kommentare von vertrauenswürdigen Nutzern werden automatisch genehmigt, während Kommentare von neuen oder nicht vertrauenswürdigen Nutzern eine manuelle Genehmigung erfordern.

**Am besten für:** Die meisten Nutzer - bietet ein Gleichgewicht zwischen Offenheit und Schutz vor Spam oder unerwünschten Inhalten.

**Was einen Benutzer "vertrauenswürdig" macht:**
- Benutzer mit gutem Karma
- Benutzer mit etablierter Kommentierhistorie
- Benutzer, die nicht wegen Spam oder Verstößen markiert wurden

#### 3. Alle manuell genehmigen
Alle Kommentare erfordern Ihre manuelle Genehmigung, bevor sie auf Ihrem Profil erscheinen, unabhängig davon, wer sie gepostet hat.

**Am besten für:** Nutzer, die vollständige Kontrolle darüber wollen, was auf ihrem Profil erscheint.

**Wie die Genehmigung funktioniert:**
- Neue Kommentare werden zur Überprüfung zurückgehalten
- Sie erhalten eine Benachrichtigung, wenn jemand kommentiert
- Sie können Kommentare per E-Mail oder von Ihrem Profil aus genehmigen oder ablehnen

#### 4. Neue manuell genehmigen
Nur Kommentare von neuen oder nicht vertrauenswürdigen Nutzern erfordern eine Genehmigung. Kommentare etablierter Community-Mitglieder werden automatisch genehmigt.

**Am besten für:** Nutzer, die neue Nutzer prüfen möchten, während etablierte Community-Mitglieder frei kommentieren können.

### Verwalten ausstehender Profilkommentare

Wenn Kommentare eine Genehmigung erfordern, erhalten Sie Benachrichtigungen und können diese auf mehrere Arten verwalten:

1. **Per E-Mail:** Genehmigen oder ablehnen direkt aus der Benachrichtigungs-E-Mail
2. **Von Ihrem Profil:** Ausstehende Kommentare in Ihrer Registerkarte Profilkommentare anzeigen
3. **Benachrichtigungszentrum:** Auf ausstehende Kommentare aus Ihren Benachrichtigungen zugreifen

### Benutzer blockieren

Zusätzlich zu den Datenschutzeinstellungen Ihres Profils können Sie bestimmte Nutzer daran hindern, irgendwo auf FastComments mit Ihnen zu interagieren:

- Blockierte Nutzer können nicht auf Ihrem Profil kommentieren
- Blockierte Nutzer können Ihnen keine Direktnachrichten senden
- Kommentare blockierter Nutzer sind vor Ihrer Ansicht verborgen
- Sie sehen die Aktivitäten des jeweils anderen nicht

Um einen Nutzer zu blockieren, rufen Sie sein Profil auf und klicken Sie auf die Option "Benutzer blockieren".

### Datenschutz-Empfehlungen

Hier sind einige empfohlene Datenschutzeinstellungen für verschiedene Anwendungsfälle:

**Öffentliches Community-Mitglied:**
- Sichtbarkeit der Aktivitäten: Öffentlich
- Profilkommentare: Öffentlich
- Direktnachrichten: Aktiviert
- Kommentargenehmigung: Automatisch vertrauenswürdige genehmigen

**Privater Nutzer:**
- Sichtbarkeit der Aktivitäten: Privat
- Profilkommentare: Privat
- Direktnachrichten: Deaktiviert
- Kommentargenehmigung: Alle manuell genehmigen

**Moderator/öffentliche Persönlichkeit:**
- Sichtbarkeit der Aktivitäten: Öffentlich
- Profilkommentare: Öffentlich
- Direktnachrichten: Aktiviert (oder deaktiviert, wenn überfordert)
- Kommentargenehmigung: Automatisch vertrauenswürdige genehmigen oder Neue manuell genehmigen
