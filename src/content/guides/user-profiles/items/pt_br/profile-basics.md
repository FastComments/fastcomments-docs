Perfis de Usuário no FastComments fornecem um espaço dedicado para cada usuário mostrar sua identidade, atividade e contribuições em sua comunidade.

### O que é um Perfil de Usuário?

Um Perfil de Usuário é uma página personalizada para cada usuário do FastComments que exibe:

- **Cabeçalho do Perfil** - Uma imagem de fundo personalizável que personaliza o perfil
- **Avatar** - A foto do perfil do usuário com indicador de status online/offline
- **Informações de Exibição** - Nome de usuário, nome exibido e opcional bandeira do país
- **Bio** - Uma descrição pessoal ou introdução
- **Links Sociais** - Conexões com os perfis de redes sociais e site do usuário
- **Distintivos** - Conquistas e reconhecimentos ganhos
- **Estatísticas** - Karma do usuário e contagem total de comentários
- **Comunidades** - Em quais sites/domínios o usuário está ativo

### Acessando Perfis de Usuário

Existem várias maneiras de acessar o perfil de um usuário:

1. **Clique em um avatar** - No widget de comentários, clique no avatar de qualquer usuário para ver o perfil
2. **Clique em um nome de usuário** - Nomes de usuário em comentários são links clicáveis para perfis
3. **URL direta** - Visite `https://fastcomments.com/auth/user-profile/[userId]`

### Visualizações do Perfil

Ao visualizar um perfil, você verá abas diferentes dependendo se está visualizando seu próprio perfil ou o de outra pessoa:

#### Seu Próprio Perfil
- **Notificações** - Suas notificações e menções
- **Atividade Recente** - Seu histórico de comentários em todas as comunidades
- **Comentários do Perfil** - Comentários que outros deixaram na sua página de perfil
- **Mensagens Diretas** - Conversas privadas com outros usuários

#### Perfis de Outros Usuários
- **Atividade Recente** - O histórico público de comentários deles (se não estiver definido como privado)
- **Comentários do Perfil** - Comentários no perfil deles (se não estiver definido como privado)
- **Mensagens Diretas** - Iniciar ou continuar uma conversa privada (se eles permitirem DMs)

### Status Online

Perfis de usuário exibem status online em tempo real:
- **Indicador verde** - O usuário está atualmente online
- **Sem indicador** - O usuário está offline

Isso ajuda você a saber quando alguém está usando ativamente a plataforma, o que é especialmente útil para mensagens diretas.

### Tipos de Usuários

O FastComments oferece suporte a dois tipos de usuários com perfis:

1. **Usuários regulares** - Usuários que se inscreveram diretamente no FastComments
2. **Usuários SSO** - Usuários que se autenticam através da integração Single Sign-On do seu site

Ambos os tipos de usuário têm acesso ao sistema completo de perfis, embora usuários SSO possam ter algumas restrições na edição de certos campos (como avatares) dependendo da configuração do seu SSO.
