User Profiles in FastComments provide a dedicated space for each user to showcase their identity, activity, and contributions across your community.

### 사용자 프로필이란?

사용자 프로필은 각 FastComments 사용자를 위한 개인화된 페이지로 다음을 표시합니다:

- **Profile Header** - 프로필을 개인화하는 커스터마이즈 가능한 배경 이미지
- **Avatar** - 온라인/오프라인 상태 표시가 있는 사용자의 프로필 사진
- **Display Information** - 사용자 이름, 표시 이름 및 선택적 국가 깃발
- **Bio** - 개인 설명 또는 소개
- **Social Links** - 사용자의 소셜 미디어 프로필 및 웹사이트 연결
- **Badges** - 획득한 업적 및 인정
- **Statistics** - 사용자 카르마 및 총 댓글 수
- **Communities** - 사용자가 활동하는 사이트/도메인

### 사용자 프로필에 접근하는 방법

사용자 프로필에 접근하는 방법은 여러 가지가 있습니다:

1. **아바타를 클릭** - 댓글 위젯에서 아무 사용자의 아바타를 클릭하여 프로필을 확인
2. **사용자명을 클릭** - 댓글의 사용자명은 프로필로 이동하는 클릭 가능한 링크
3. **직접 URL** - `https://fastcomments.com/auth/user-profile/[userId]` 방문

### 프로필 보기

프로필을 볼 때, 본인 프로필을 보는지 다른 사람의 프로필을 보는지에 따라 다른 탭이 표시됩니다:

#### 내 프로필
- **Notifications** - 본인의 알림 및 멘션
- **Recent Activity** - 모든 커뮤니티에서의 본인 댓글 기록
- **Profile Comments** - 다른 사람들이 본인 프로필 페이지에 남긴 댓글
- **Direct Messages** - 다른 사용자와의 비공개 대화

#### 다른 사용자의 프로필
- **Recent Activity** - 공개로 설정된 경우 그들의 공개 댓글 기록
- **Profile Comments** - 공개로 설정된 경우 프로필에 대한 댓글
- **Direct Messages** - 비공개 대화를 시작하거나 이어갈 수 있음(그들이 DM을 허용하는 경우)

### 온라인 상태

사용자 프로필은 실시간 온라인 상태를 표시합니다:
- **Green indicator** - 사용자가 현재 온라인 상태임
- **No indicator** - 사용자가 오프라인 상태임

이는 누군가가 플랫폼을 활발히 사용 중일 때를 알 수 있게 해주며, 특히 다이렉트 메시지에 유용합니다.

### 사용자 유형

FastComments는 프로필을 가진 두 가지 유형의 사용자를 지원합니다:

1. **Regular Users** - FastComments에 직접 가입한 사용자
2. **SSO Users** - 사이트의 Single Sign-On 통합을 통해 인증하는 사용자

두 사용자 유형 모두 전체 프로필 시스템에 접근할 수 있지만, SSO 사용자는 SSO 구성에 따라 특정 필드(예: 아바타) 편집에 일부 제한이 있을 수 있습니다.
