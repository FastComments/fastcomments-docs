Direct Messaging (DM) allows FastComments users to have private, one-on-one conversations with each other, separate from public comments and profile interactions.

### 다이렉트 메시지란?

다이렉트 메시지(DM)는 다음을 제공합니다:
- **두 사용자 간의 개인 대화**
- **알림이 포함된 실시간 메시징**
- **세션 간 지속되는 대화 기록**
- **사용 가능 여부를 표시하는 온라인 상태 표시**
- **새 메시지를 파악할 수 있는 읽지 않은 메시지 추적**

### 다이렉트 메시지 시작하기

DM 대화를 시작하는 방법은 여러 가지가 있습니다:

**사용자 프로필에서:**
1. 사용자의 프로필 페이지를 방문합니다
2. "다이렉트 메시지" 탭 또는 "메시지 보내기" 버튼을 클릭합니다
3. 메시지를 입력하고 전송합니다

**댓글에서:**
일부 구현에서는 사용자의 이름이나 아바타를 클릭하여 빠르게 프로필과 메시지 옵션에 접근할 수 있습니다.

**직접 링크:**
대화 ID나 DM 리디렉트 링크가 있는 경우 대화에 직접 접근할 수 있습니다.

### 메시지 접근하기

모든 다이렉트 메시지 대화를 보려면:

1. 프로필 페이지를 방문합니다
2. "다이렉트 메시지" 탭을 클릭합니다
3. 모든 대화 목록을 확인합니다

각 대화에는 다음이 표시됩니다:
- 상대 참가자의 아바타와 이름
- 온라인/오프라인 상태 (온라인일 때 녹색 표시기)
- 마지막 메시지 미리보기
- 읽지 않은 메시지 수 (있는 경우)
- 마지막 활동의 타임스탬프

### 대화 보기

대화를 열면 다음을 볼 수 있습니다:

- **전체 메시지 기록** - 당신과 다른 사용자 간의 모든 메시지
- **실시간 업데이트** - 새 메시지가 WebSocket을 통해 즉시 나타납니다
- **온라인 상태** - 상대가 현재 온라인인지 확인
- **메시지 타임스탬프** - 각 메시지가 전송된 시간
- **메시지 작성** - 새 메시지를 입력하고 전송할 텍스트 박스

### 메시지 알림

새 다이렉트 메시지에 대해 알림을 받습니다:

- **앱 내 알림** - 프로필에 표시되는 배지 카운터
- **이메일 알림** - 알림 설정에 따라
- **실시간 경고** - 로그인되어 있을 때 즉시 알림

알림 환경설정은 [계정 설정](https://fastcomments.com/auth/my-account/notification-settings)에서 관리하세요.

### 개인정보 보호 및 차단

**다이렉트 메시지 비활성화:**
DM을 받고 싶지 않은 경우 완전히 비활성화할 수 있습니다:
1. 프로필의 개인정보 보호 설정으로 이동합니다
2. "다이렉트 메시지 비활성화"를 활성화합니다
3. 프로필에서 DM 옵션이 숨겨집니다

자세한 내용은 [개인정보 보호 설정](/guides/user-profiles/privacy-settings)를 참조하세요.

**사용자 차단:**
누군가 DM으로 괴롭힌다면:
1. 해당 사용자의 프로필을 방문합니다
2. 사용자를 차단합니다
3. 더 이상 그 사용자가 당신에게 메시지를 보낼 수 없습니다

차단된 사용자는 다음을 할 수 없습니다:
- 당신에게 다이렉트 메시지를 보내기
- 당신의 활동을 보기
- 당신의 프로필에 댓글 달기
- FastComments 내에서 당신과 상호작용하기

### 대화 관리

**대화 숨기기:**
대화 목록에서 더 이상 보고 싶지 않은 대화를 숨길 수 있습니다:
1. 다이렉트 메시지 탭을 엽니다
2. 숨기려는 대화를 찾습니다
3. 숨기기/보관 옵션을 선택합니다

숨겨진 대화는 기본 목록에 나타나지 않지만 상대가 새 메시지를 보내면 접근할 수 있습니다.

**읽음으로 표시하기:**
메시지는 보았을 때 자동으로 읽음으로 표시됩니다. 전체 대화를 수동으로 읽음으로 표시할 수도 있습니다:
1. 다이렉트 메시지 탭을 엽니다
2. 대화를 선택합니다
3. "읽음으로 표시"를 선택합니다

이렇게 하면 해당 대화의 읽지 않은 카운터가 지워집니다.

### 모범 사례

**다이렉트 메시지를 사용해야 할 때:**
- 개인적으로 후속 질문을 할 때
- 유용한 조언에 대해 감사 인사를 전할 때
- 공개 댓글을 어지럽히지 않고 주제에서 벗어난 내용을 논의할 때
- 커뮤니티 구성원과 조율할 때
- 개인적인 피드백이나 제안을 제공할 때

**DM 예절:**
- 정중하고 전문적으로 행동하세요
- 원치 않는 메시지로 사용자를 스팸하지 마세요
- 누군가 응답하지 않거나 DM을 비활성화하면 이를 존중하세요
- 대화를 관련 있고 건설적으로 유지하세요
- 허가 없이 다른 사람의 DM을 공유하지 마세요

**안전 수칙:**
- 상대를 신뢰하지 않는 한 개인 정보(전화번호, 주소 등)를 공유하지 마세요
- DM으로 괴롭힘이나 학대를 당하면 차단 및 신고하세요
- 의심스러운 링크나 요청을 주의하세요
- 불편함을 느끼면 차단 기능을 사용하세요

### 제한 사항 및 참고

**누구에게 메시지를 보낼 수 있나요:**
- DM을 비활성화하지 않은 모든 FastComments 사용자
- 당신을 차단하지 않은 사용자
- 모든 FastComments 커뮤니티의 사용자

**메시지 내용:**
- 텍스트 메시지가 지원됩니다
- 메시지는 댓글과 동일한 콘텐츠 정책을 따릅니다
- 부적절한 콘텐츠는 신고할 수 있습니다

**대화 범위:**
- DM 대화는 두 사람 간의 비공개입니다
- 그룹 메시징은 지원되지 않습니다(현재 1:1만 가능)
- 대화 기록은 무기한 보존됩니다

### 문제 해결

**메시지를 보낼 수 없나요?**
상대 사용자가 다음을 했을 수 있습니다:
- 개인정보 보호 설정에서 다이렉트 메시지를 비활성화함
- 당신을 차단함
- 계정을 삭제함

**알림을 받지 못하나요?**
DM 알림이 활성화되어 있는지 확인하려면 [계정 설정](https://fastcomments.com/auth/my-account/notification-settings)에서 알림 설정을 확인하세요.

**메시지가 전송되지 않나요?**
- 인터넷 연결을 확인하세요
- 페이지를 새로고침하고 다시 시도하세요
- 차단되지 않았는지 확인하세요
- 문제가 지속되면 지원팀에 문의하세요
