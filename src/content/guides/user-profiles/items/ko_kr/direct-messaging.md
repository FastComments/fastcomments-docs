다이렉트 메시지(DM)는 FastComments 사용자가 공개 댓글 및 프로필 상호작용과 분리된 개인 1:1 대화를 나눌 수 있도록 합니다.

### 다이렉트 메시지란?

다이렉트 메시지는 다음을 제공합니다:
- **두 사용자 간의 개인 대화**
- **알림을 포함한 실시간 메시징**
- **세션 간 유지되는 대화 기록**
- **상대의 사용 가능 여부를 보여주는 온라인 상태 표시**
- **새 메시지를 파악할 수 있는 안 읽은 메시지 추적**

### 다이렉트 메시지 시작하기

DM 대화를 시작하는 방법은 여러 가지가 있습니다:

**사용자 프로필에서:**
1. 사용자의 프로필 페이지를 방문하세요
2. "Direct Messages" 탭 또는 "Send Message" 버튼을 클릭하세요
3. 메시지를 입력하고 전송하세요

**댓글에서:**
일부 구현에서는 사용자의 이름이나 아바타를 클릭하여 빠르게 프로필 및 메시지 옵션에 접근할 수 있습니다.

**직접 링크:**
대화 ID 또는 DM 리디렉션 링크가 있는 경우 대화에 직접 접근할 수 있습니다.

### 메시지에 접근하기

모든 다이렉트 메시지 대화를 보려면:

1. 프로필 페이지를 방문하세요
2. "Direct Messages" 탭을 클릭하세요
3. 모든 대화 목록을 확인하세요

각 대화에는 다음이 표시됩니다:
- 상대 참가자의 아바타와 이름
- 온라인/오프라인 상태(온라인일 때 녹색 표시)
- 마지막 메시지 미리보기
- 안 읽은 메시지 수(있는 경우)
- 마지막 활동의 타임스탬프

### 대화 보기

대화를 열면 다음을 볼 수 있습니다:

- **전체 메시지 기록** - 사용자와 상대방 간의 모든 메시지
- **실시간 업데이트** - 새 메시지는 WebSocket을 통해 즉시 표시됩니다
- **온라인 상태** - 상대가 현재 온라인인지 확인
- **메시지 타임스탬프** - 각 메시지가 전송된 시간
- **메시지 작성** - 새 메시지를 입력하고 전송할 수 있는 텍스트 상자

### 메시지 알림

새 다이렉트 메시지에 대한 알림을 받습니다:

- **앱 내 알림** - 프로필의 배지 카운터
- **이메일 알림** - 알림 설정에 따라
- **실시간 알림** - 로그인되어 있을 때 즉시 알림

알림 기본 설정은 [계정 설정](https://fastcomments.com/auth/my-account/edit-notifications)에서 관리하세요.

### 개인정보 및 차단

**다이렉트 메시지 비활성화:**
DM을 받고 싶지 않은 경우 완전히 비활성화할 수 있습니다:
1. 프로필의 개인정보 설정으로 이동
2. "Disable Direct Messages"를 활성화
3. 프로필에서 DM 옵션이 숨겨집니다

자세한 내용은 [개인정보 설정](/guide-user-profiles.html#privacy-settings)을 참조하세요.

**사용자 차단:**
누군가 DM으로 괴롭힘을 하는 경우:
1. 해당 사용자의 프로필을 방문
2. 사용자를 차단
3. 해당 사용자는 더 이상 메시지를 보낼 수 없습니다

차단된 사용자는 다음을 할 수 없습니다:
- 다이렉트 메시지 전송
- 귀하의 활동 보기
- 프로필에 댓글 달기
- FastComments의 어느 곳에서도 귀하와 상호작용하기

### 대화 관리

**대화 숨기기:**
대화 목록에서 더 이상 보고 싶지 않은 대화를 숨길 수 있습니다:
1. 다이렉트 메시지 탭을 엽니다
2. 숨기려는 대화를 찾습니다
3. 숨기기/보관 옵션을 선택합니다

숨긴 대화는 기본 목록에 표시되지 않지만 상대방이 새 메시지를 보내면 접근할 수 있습니다.

**읽음으로 표시:**
메시지는 열람 시 자동으로 읽음 처리됩니다. 전체 대화를 수동으로 읽음 처리할 수도 있습니다:
1. 다이렉트 메시지 탭을 엽니다
2. 대화를 선택합니다
3. "Mark as Read"를 선택합니다

이렇게 하면 해당 대화의 안 읽은 카운터가 해제됩니다.

### 모범 사례

**다이렉트 메시지를 사용해야 할 때:**
- 추가 질문을 비공개로 물어볼 때
- 도움을 준 사람에게 감사 인사를 전할 때
- 공개 댓글을 어수선하게 하지 않고 주제와 벗어난 사항을 논의할 때
- 커뮤니티의 다른 구성원과 조정할 때
- 비공개 피드백이나 제안을 제공할 때

**DM 에티켓:**
- 정중하고 전문적으로 행동하세요
- 원치 않는 메시지로 스팸을 보내지 마세요
- 누군가 응답하지 않거나 DM을 비활성화하면 이를 존중하세요
- 대화를 관련성 있고 건설적으로 유지하세요
- 허가 없이 다른 사람의 DM을 공유하지 마세요

**안전 팁:**
- 신뢰하지 않는 상대에게는 개인 정보(전화번호, 주소 등)를 공유하지 마세요
- 괴롭힘이나 악용을 당하면 사용자를 차단하고 신고하세요
- 의심스러운 링크나 요청에 주의하세요
- 불편함을 느끼면 차단 기능을 사용하세요

### 제한 사항 및 참고

**누구에게 메시지를 보낼 수 있나:**
- DM을 비활성화하지 않은 모든 FastComments 사용자
- 귀하를 차단하지 않은 사용자
- 모든 FastComments 커뮤니티의 사용자

**메시지 내용:**
- 텍스트 메시지를 지원합니다
- 메시지는 댓글과 동일한 콘텐츠 정책을 준수합니다
- 부적절한 콘텐츠는 신고할 수 있습니다

**대화 범위:**
- DM 대화는 두 사람 간의 비공개입니다
- 그룹 메시지는 없음(현재 1:1만 지원)
- 대화 기록은 무기한 보존됩니다

### 문제 해결

**메시지를 보낼 수 없나요?**
상대방이 다음을 했을 수 있습니다:
- 개인정보 설정에서 다이렉트 메시지를 비활성화함
- 귀하를 차단함
- 계정을 삭제함

**알림을 받지 못하나요?**
DM 알림이 활성화되어 있는지 [계정 설정](https://fastcomments.com/auth/my-account/edit-notifications)에서 확인하세요.

**메시지가 전송되지 않나요?**
- 인터넷 연결을 확인하세요
- 페이지를 새로 고치고 다시 시도하세요
- 차단되지 않았는지 확인하세요
- 문제가 지속되면 지원팀에 문의하세요
