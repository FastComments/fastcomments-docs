FastComments는 프로필의 개인정보와 누가 귀하와 상호작용할 수 있는지에 대해 세밀한 제어를 제공합니다. 모든 개인정보 설정은 프로필 페이지의 "개인정보 설정" 섹션에서 관리할 수 있습니다.

### 개인정보 설정에 접근하기

개인정보 설정을 관리하려면:
1. 프로필 페이지를 방문하세요
2. "개인정보 설정" 탭 또는 섹션을 클릭하세요
3. 원하는 대로 설정을 조정하세요
4. 변경 사항은 자동으로 저장됩니다

### 활동 공개 범위

FastComments 커뮤니티 전반에서 귀하의 댓글 활동을 누가 볼 수 있는지 제어합니다.

**설정:** 프로필 활동 비공개로 설정

- **비활성화된 경우 (기본값):** 누구나 프로필에서 최근 댓글 및 활동을 볼 수 있습니다
- **활성화된 경우:** 오직 본인만 댓글 내역 및 활동 피드를 볼 수 있습니다

토론에 참여하되 댓글 내역을 다른 사용자로부터 비공개로 유지하고 싶을 때 유용합니다.

### 프로필 댓글 공개 범위

다른 사용자가 귀하의 프로필 페이지에서 댓글을 보고 남길 수 있는지를 제어합니다.

**설정:** 프로필 댓글 비공개로 설정

- **비활성화된 경우 (기본값):** 누구나 귀하의 프로필을 보고 댓글을 남길 수 있습니다
- **활성화된 경우:** 프로필 댓글이 다른 사용자에게 숨겨지며; 오직 본인만 프로필에 남겨진 댓글을 볼 수 있습니다

프로필 페이지에서의 상호작용을 제한하면서 다른 곳에서는 활동을 허용하고 싶을 때 이 설정을 사용하세요.

### 다이렉트 메시지

다른 사용자가 귀하에게 다이렉트 메시지를 보낼 수 있는지를 제어합니다.

**설정:** 다이렉트 메시지 비활성화

- **비활성화된 경우 (기본값):** 다른 사용자가 귀하에게 다이렉트 메시지를 보낼 수 있습니다
- **활성화된 경우:** 프로필에서 다이렉트 메시지 옵션이 숨겨지며 사용자가 대화를 시작할 수 없습니다

비공개 메시지를 받기 원치 않거나 일대일 대화를 제한하고 싶을 때 유용합니다.

### 프로필 댓글 승인

프로필에 달린 댓글이 공개되기 전에 어떻게 검토되는지를 제어합니다.

**설정:** 프로필 댓글 승인 모드

프로필 댓글 관리를 위해 네 가지 옵션이 있습니다:

#### 1. 모든 자동 승인
프로필에 게시된 모든 댓글이 승인 없이 즉시 표시됩니다.

**추천 대상:** 열린 댓글 섹션을 원하는 사용자

#### 2. 신뢰된 사용자 자동 승인 (기본값)
신뢰된 사용자로부터 온 댓글은 자동으로 승인되며, 신규 사용자나 신뢰되지 않은 사용자의 댓글은 수동 승인이 필요합니다.

**추천 대상:** 대부분의 사용자 - 개방성과 스팸 또는 원치 않는 콘텐츠로부터의 보호를 균형 있게 제공합니다.

**사용자를 "신뢰된" 것으로 만드는 요소:**
- 좋은 카르마를 가진 사용자
- 확립된 댓글 기록이 있는 사용자
- 스팸 또는 규정 위반으로 신고된 이력이 없는 사용자

#### 3. 모든 수동 승인
누가 게시했는지에 관계없이 모든 댓글은 프로필에 표시되기 전에 귀하의 수동 승인이 필요합니다.

**추천 대상:** 프로필에 표시되는 내용을 완전히 통제하고자 하는 사용자

**승인 방식:**
- 새로운 댓글은 검토를 위해 보류됩니다
- 누군가 댓글을 남기면 알림을 받습니다
- 이메일 또는 프로필에서 댓글을 승인하거나 거부할 수 있습니다

#### 4. 신규 수동 승인
신규 사용자나 신뢰되지 않은 사용자로부터 온 댓글만 승인이 필요합니다. 커뮤니티 내에서 자리잡은 회원의 댓글은 자동 승인됩니다.

**추천 대상:** 신규 사용자를 선별하면서도 기존 회원이 자유롭게 댓글을 남기도록 허용하고자 하는 사용자

### 보류 중인 프로필 댓글 관리

댓글이 승인을 필요로 할 때, 알림을 받고 다음과 같은 방법으로 관리할 수 있습니다:

1. **이메일을 통해:** 알림 이메일에서 직접 승인하거나 거부
2. **프로필에서:** 프로필 댓글 탭에서 보류 중인 댓글 보기
3. **알림 센터:** 알림에서 보류 중인 댓글에 접근

### 사용자 차단

프로필 개인정보 설정 외에도 특정 사용자가 FastComments 어디에서든 귀하와 상호작용하지 못하도록 차단할 수 있습니다:

- 차단된 사용자는 귀하의 프로필에 댓글을 달 수 없습니다
- 차단된 사용자는 귀하에게 다이렉트 메시지를 보낼 수 없습니다
- 차단된 사용자의 댓글은 귀하의 보기에서 숨겨집니다
- 서로의 활동을 볼 수 없습니다

사용자를 차단하려면 해당 사용자의 프로필을 방문하여 "사용자 차단" 옵션을 클릭하세요.

### 개인정보 관련 권장 설정

다양한 사용 사례에 대한 권장 개인정보 구성은 다음과 같습니다:

**공개 커뮤니티 회원:**
- 활동 공개 범위: 공개
- 프로필 댓글: 공개
- 다이렉트 메시지: 활성화
- 댓글 승인: 신뢰된 사용자 자동 승인

**비공개 사용자:**
- 활동 공개 범위: 비공개
- 프로필 댓글: 비공개
- 다이렉트 메시지: 비활성화
- 댓글 승인: 모든 댓글 수동 승인

**모더레이터/공인:**
- 활동 공개 범위: 공개
- 프로필 댓글: 공개
- 다이렉트 메시지: 활성화 (혹은 과부하일 경우 비활성화)
- 댓글 승인: 신뢰된 사용자 자동 승인 또는 신규 수동 승인
