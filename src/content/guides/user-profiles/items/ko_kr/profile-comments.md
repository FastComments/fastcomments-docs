Profile Comments는 사용자가 다른 사용자의 프로필 페이지에 직접 메시지를 남길 수 있게 하여 개인적인 댓글 게시판 또는 방명록 스타일의 상호작용을 만듭니다.

### 프로필 댓글이란?

프로필 댓글은 일반 기사 댓글과 다릅니다:
- 콘텐츠 기사에 달리는 것이 아니라 사용자의 프로필 페이지에 표시됩니다
- (개인정보 설정이 제한하지 않는 한) 프로필을 보는 누구나 볼 수 있습니다
- 특정 사용자에게 감사를 표하거나 인사하거나 소통하는 방법입니다
- 모든 FastComments 커뮤니티에서 작동합니다

### 프로필에 댓글 남기기

다른 사람의 프로필에 댓글을 달려면:

1. 해당 사용자의 프로필 페이지를 방문합니다
2. "Profile Comments" 탭으로 이동합니다
3. 댓글 입력란에 메시지를 작성합니다
4. 댓글을 제출합니다

**참고:** 프로필 소유자의 개인정보 설정에 따라 댓글이 즉시 표시되거나 승인이 필요할 수 있습니다.

### 프로필 댓글 보기

**내 프로필에서:**
1. 내 프로필 페이지를 방문합니다
2. "Profile Comments" 탭을 클릭합니다
3. 다른 사람이 내 프로필에 남긴 모든 댓글을 확인합니다

**다른 사람의 프로필에서:**
사용자가 "프로필 댓글 비공개로 설정(Make Profile Comments Private)"을 활성화하지 않은 경우 프로필 댓글이 표시됩니다.

### 댓글 승인

프로필 소유자의 설정에 따라(자세한 내용은 [프라이버시 설정](/guides/user-profiles/privacy-settings) 참조) 댓글은 다음과 같이 처리될 수 있습니다:

1. **즉시 표시** - 신뢰 수준이나 승인 설정에 따라 자동 승인됩니다
2. **승인 필요** - 프로필 소유자가 검토할 때까지 보류됩니다

댓글이 승인을 필요로 할 때:
- 보류 상태 표시기를 보게 됩니다
- 프로필 소유자가 알림을 받습니다
- 승인되면 댓글이 다른 사용자에게 표시됩니다

### 내 프로필에서 프로필 댓글 관리

댓글 승인을 구성한 경우(모두 수동 승인 또는 신규만 수동 승인), 들어오는 댓글을 관리할 수 있습니다:

**이메일을 통해:**
보류 중인 댓글에 대해 승인/거부 링크가 포함된 이메일 알림을 받습니다. 이메일에서 링크를 클릭하여 직접 조치를 취하세요.

**프로필에서:**
1. 내 프로필의 "Profile Comments" 탭을 방문합니다
2. 보류 중인 댓글을 확인합니다
3. 각 댓글을 승인하거나 거부합니다

**알림에서:**
알림 센터에서 보류 중인 댓글에 빠르게 접근하여 검토할 수 있습니다.

### 프로필 댓글에 응답하기

누군가 내 프로필에 댓글을 남기면 다음을 할 수 있습니다:
- 댓글에 직접 답글을 달아 대화 스레드를 만듭니다
- 그들의 프로필을 방문하여 답글을 남깁니다
- (해당 사용자가 허용하는 경우) 개인 메시지를 보냅니다

### 프로필 댓글 삭제

**내 프로필에서:**
내 프로필에 남겨진 댓글은 누구든 삭제할 수 있습니다:
1. 삭제하려는 댓글을 찾습니다
2. 삭제/제거 옵션을 클릭합니다
3. 삭제를 확인합니다

**다른 사람의 프로필에서:**
다른 사람의 프로필에 남긴 내 댓글은 삭제할 수 있지만, 다른 사람의 댓글은 삭제할 수 없습니다(프로필 소유자인 경우 제외).

### 프로필 댓글 알림

다음과 같은 경우 알림을 받게 됩니다:
- 누군가 내 프로필에 댓글을 남긴 경우
- 참여 중인 프로필 댓글 스레드에 누군가 답글을 남긴 경우
- 내가 남긴 프로필 댓글이 승인되거나 응답된 경우

알림 환경설정은 [계정 설정](https://fastcomments.com/auth/my-account/notification-settings)에서 관리하세요.

### 개인정보 및 프로필 댓글

다음 개인정보 설정으로 프로필 댓글 가시성을 제어할 수 있습니다:

**프로필 댓글을 비공개로 설정하기(Make Profile Comments Private):**
- 방문자에게 Profile Comments 탭을 숨깁니다
- 오직 본인만 프로필에 남겨진 댓글을 볼 수 있습니다
- 새 댓글 게시를 방지합니다

**댓글 승인 모드:**
- 승인 없이 댓글을 달 수 있는 대상을 선택합니다
- 신뢰된 사용자와 신규 사용자에 대해 서로 다른 승인 규칙을 설정합니다
- 모든 댓글에 대해 수동 승인을 요구합니다

자세한 내용은 [프라이버시 설정](/guides/user-profiles/privacy-settings)을 참조하세요.

### 프로필 댓글의 사용 사례

프로필 댓글은 다음에 유용합니다:

- **도움이 되는 커뮤니티 구성원에게 감사 표시**
- **가치 있는 기여를 인정**
- **활동적인 참여자와의 친분 형성**
- **추천글 또는 지지 메시지 남기기**
- **소셜 방명록 경험 생성**
- **커뮤니티 감사 및 인정**

### 모범 사례

**댓글을 남길 때:**
- 정중하고 건설적으로 작성하세요
- 단순한 "프로필 멋지네요" 같은 일반적인 메시지보다 가치를 추가하세요
- 가능하면 특정 기여나 상호작용을 언급하세요

**프로필을 관리할 때:**
- 편안한 수준에 맞는 승인 설정을 적용하세요
- 커뮤니티 참여를 장려하기 위해 프로필 댓글에 답글을 달아주세요
- 누군가 반복적으로 부적절한 댓글을 남기면 차단 기능을 사용하세요

**커뮤니티 건강을 위해:**
- 프로필 댓글은 기사 댓글과 동일한 커뮤니티 가이드라인을 따릅니다
- 부적절한 프로필 댓글을 발견하면 신고하세요
- 규칙을 위반하는 프로필 댓글에 대해 모더레이터가 조치를 취할 수 있습니다
