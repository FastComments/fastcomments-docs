FastComments 사용자 프로필은 매우 사용자 정의 가능하며, 사용자는 이를 통해 자신의 개성을 표현하고 온라인 존재감을 구축할 수 있습니다.

### 프로필 편집

프로필을 사용자 지정하려면 [Account Details](https://fastcomments.com/auth/my-account/edit-details) 페이지를 방문하세요. 여기에서 프로필 외형과 정보의 모든 측면을 수정할 수 있습니다.

### 프로필 아바타

아바타는 댓글 옆과 프로필 페이지에 표시되는 프로필 사진입니다.

**아바타를 변경하려면:**
1. 프로필 페이지를 방문합니다
2. "Edit Avatar" 링크를 클릭합니다 (자신의 프로필에서만 보임)
3. 새 이미지를 업로드합니다

**아바타 요구사항:**
- 권장 크기: 200x200 픽셀 이상
- 지원 형식: JPG, PNG, GIF
- 정사각형 이미지가 가장 적합합니다

참고: SSO 사용자의 아바타는 사이트의 SSO 시스템에 의해 관리될 수 있습니다.

### 프로필 헤더 배경

헤더 배경은 프로필 상단에 표시되는 큰 배너 이미지로, 스타일을 반영하도록 사용자 지정할 수 있습니다.

**헤더 배경을 업로드하려면:**
1. 프로필 페이지를 방문합니다
2. "Upload Header" 버튼을 클릭합니다
3. 기기에서 이미지를 선택합니다

**헤더 요구사항:**
- 권장 크기: 1200x300 픽셀
- 지원 형식: JPG, PNG
- 가로형/와이드 이미지가 가장 적합합니다
- 이미지는 자동으로 크기가 조정됩니다

사용자 정의 헤더를 업로드하지 않으면 기본 배경이 표시됩니다.

### 표시 이름

표시 이름은 FastComments 전역에서 다른 사용자에게 보이는 이름입니다. 고유해야 하는 사용자 이름과 달리 표시 이름은:

- 고유할 필요가 없음
- 공백 및 특수 문자를 포함할 수 있음
- 이모지를 포함할 수 있음

**표시 이름을 설정하려면:**
1. [Account Details](https://fastcomments.com/auth/my-account/edit-details)로 이동합니다
2. 원하는 표시 이름을 입력합니다
3. 변경 사항을 저장합니다

### 사용자 이름

사용자 이름은 FastComments에 로그인하는 데 사용되는 고유 식별자입니다.

- FastComments의 모든 사용자 중에서 고유해야 함
- 프로필 URL에 사용됩니다: `fastcomments.com/auth/user-profile/username`
- 공백이나 특수 문자를 포함할 수 없음

참고: SSO 사용자는 특정 테넌트/커뮤니티에 연결되어 있으므로 사용자 이름이 고유하지 않아도 될 수 있습니다.

### 소개

자신에 대해, 관심사에 대해 또는 하는 일에 대해 다른 사람에게 알려주기 위해 개인 소개를 추가하세요.

**소개를 추가하려면:**
1. [Account Details](https://fastcomments.com/auth/my-account/edit-details)로 이동합니다
2. 소개 필드에 텍스트를 입력합니다
3. 변경 사항을 저장합니다

소개는 프로필 페이지에 눈에 띄게 표시되어 다른 사람들이 당신이 누구인지 이해하는 데 도움이 됩니다.

### 웹사이트 URL

개인 웹사이트, 블로그 또는 포트폴리오에 대한 링크를 추가하세요.

**웹사이트를 추가하려면:**
1. [Account Details](https://fastcomments.com/auth/my-account/edit-details)로 이동합니다
2. 웹사이트 필드에 웹사이트 URL을 입력합니다
3. 변경 사항을 저장합니다

웹사이트 링크는 클릭 가능한 아이콘과 함께 프로필에 표시됩니다.

### 소셜 미디어 링크

소셜 미디어 프로필을 FastComments 프로필에 연결하세요. 지원되는 플랫폼에는 다음이 포함됩니다:

- **Facebook** - Facebook 프로필 링크
- **Instagram** - Instagram 계정 링크
- **LinkedIn** - LinkedIn 프로필 링크
- **Custom Links** - 기타 소셜 미디어 또는 웹 프로필 추가

**소셜 링크를 추가하려면:**
1. [Account Details](https://fastcomments.com/auth/my-account/edit-details)로 이동합니다
2. 소셜 미디어 계정의 URL을 입력합니다
3. 변경 사항을 저장합니다

소셜 링크는 플랫폼을 식별할 수 있는 아이콘과 함께 프로필에 표시되어 커뮤니티 구성원이 다른 곳에서 쉽게 연결할 수 있습니다.

### 국가

선택적으로 국가를 설정할 수 있으며, 설정하면 이름 옆에 국기 아이콘이 표시됩니다.

**국가를 설정하려면:**
1. [Account Details](https://fastcomments.com/auth/my-account/edit-details)로 이동합니다
2. 드롭다운에서 국가를 선택합니다
3. 변경 사항을 저장합니다

국기 아이콘은 프로필의 사용자 이름 옆에 표시됩니다.

### 백업 이메일

계정 복구를 위해 백업 이메일을 추가할 수 있습니다. 기본 이메일에 접근할 수 없게 된 경우 백업 이메일을 사용하여 로그인하거나 사용자 이름을 복구할 수 있습니다.

**백업 이메일을 추가하려면:**
1. [Account Details](https://fastcomments.com/auth/my-account/edit-details)로 이동합니다
2. 이메일 필드 아래의 **Define a Backup Email**을 클릭합니다
3. 백업 이메일 주소를 입력하고 제출합니다
4. 해당 주소로 전송된 인증 링크를 클릭합니다

백업 이메일은 계정 복구에만 사용되며 알림이나 기타 이메일을 받지 않습니다. 언제든지 변경하거나 제거할 수 있습니다.

참고: 백업 이메일은 SSO 사용자가 아닌 일반 사용자에게만 제공됩니다.
