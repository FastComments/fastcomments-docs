The Activity Feed on User Profiles displays a user's comment history, notifications, and community participation across all FastComments-powered sites.

### What is the Activity Feed?

활동 피드는 다음의 연대기적 보기를 제공합니다:
- 사용자가 게시한 댓글
- 받은 답글
- 멘션 및 상호작용
- 다양한 커뮤니티에서의 활동
- 사용자 참여에 대한 실시간 업데이트

### Viewing Activity

**Your Own Activity:**
프로필을 볼 때 추가 탭이 표시됩니다:
1. **Notifications** - 답글, 멘션 및 상호작용에 대한 알림
2. **Recent Activity** - 모든 사이트에 걸친 전체 댓글 기록

**Other Users' Activity:**
다른 사용자의 프로필을 볼 때:
1. **Recent Activity** - 공개 댓글 기록(비공개로 설정하지 않은 경우)

활동 탭은 사용자가 FastComments 네트워크 전반에서 언제 어디서 활동하는지를 보여줍니다.

### Notifications Tab

알림 탭에는 다음이 표시됩니다:

**What You'll See:**
- 내 댓글에 대한 답글
- 사용자명 멘션
- 프로필 댓글 알림
- 다이렉트 메시지 알림
- 배지 획득 알림
- 내 콘텐츠에 대한 조치(운영/모더레이션)

**Notification Features:**
- **Unread counter** - 읽지 않은 알림 수 표시
- **Real-time updates** - 새로운 알림은 WebSocket을 통해 즉시 나타납니다
- **Clickable items** - 관련 댓글이나 대화로 이동하려면 알림을 클릭하세요
- **Read/unread status** - 어떤 알림을 확인했는지에 대한 시각적 표시

**Managing Notifications:**
- 알림은 사용자가 확인하면 읽음으로 표시됩니다
- 어떤 유형의 알림을 받을지 [알림 설정](https://fastcomments.com/auth/my-account/notification-settings)에서 구성하세요

자세한 알림 관리 방법은 [알림 가이드](/guides/notifications)를 참조하세요.

### Recent Activity Tab

최근 활동 탭에는 사용자가 FastComments 커뮤니티 전반에 게시한 모든 댓글이 표시됩니다.

**What's Displayed:**
- **Comment content** - 각 댓글의 전체 텍스트
- **Context** - 댓글이 달린 기사/페이지
- **Timestamp** - 댓글이 게시된 시간
- **Community** - 댓글이 속한 사이트 또는 도메인
- **Engagement** - 투표, 답글 및 상호작용 지표

**Activity Filtering:**
자신의 프로필에서는 활동을 필터링하여 표시할 수 있습니다:
- **All Activity** - 사용자가 게시한 모든 댓글
- **Replies to Me** - 내 댓글에 대한 답글만 표시

이 기능으로 직접 참여한 대화에 집중하기 쉬워집니다.

### Community Participation

프로필에는 사용자가 활동하는 커뮤니티도 표시됩니다:

- **Community List** - 사용자가 댓글을 남긴 사이트/도메인 목록
- **Activity Distribution** - 각 커뮤니티에서 사용자가 얼마나 활동하는지
- **Community Links** - 해당 사이트로 이동할 수 있는 클릭 가능한 링크

이를 통해 사용자가 주로 활동하는 장소와 관심 분야를 파악할 수 있습니다.

### Statistics

사용자 프로필에는 주요 통계가 표시됩니다:

**Karma:**
- 업보트와 커뮤니티 반응을 기반으로 한 평판 점수
- 댓글이 업보트를 받을 때 증가
- 댓글이 다운보트를 받을 때 감소
- 기여에 대한 전체적인 커뮤니티의 평가를 반영

**Comment Count:**
- 모든 커뮤니티에 걸쳐 게시한 댓글의 총 수
- 최상위 댓글과 답글 모두 포함
- 참여할 때 실시간으로 업데이트

이 통계는 사용자의 참여 수준과 커뮤니티 내 위상을 빠르게 보여줍니다.

### Privacy Controls

사용자는 개인 정보 설정을 통해 활동 공개 범위를 제어할 수 있습니다:

**Make Profile Activity Private:**
활성화하면:
- 최근 활동 탭은 본인만 볼 수 있습니다
- 다른 사용자는 프로필에서 댓글 기록을 볼 수 없습니다
- 공개 댓글은 원래의 맥락(기사 등)에서는 계속 표시됩니다
- 프로필에 표시되는 집계된 활동 보기만 숨겨집니다

자세한 내용은 [개인정보 보호 설정](/guides/user-profiles/privacy-settings)를 참조하세요.

### Real-Time Updates

활동 피드는 실시간 업데이트를 위해 WebSocket 기술을 사용합니다:

- **Instant notifications** - 새로운 답글과 멘션이 즉시 나타납니다
- **Live activity updates** - 최근 댓글이 게시와 동시에 추가됩니다
- **Online status** - 대화 상대가 활동 중인지 확인할 수 있습니다
- **No page refresh needed** - 페이지 새로고침 없이 모든 것이 자동으로 업데이트됩니다

### Activity Across Communities

FastComments 사용자 프로필의 강력한 기능 중 하나는 모든 커뮤니티에서의 활동을 확인할 수 있다는 점입니다:

**Benefits:**
- 사용자가 어느 분야에 지식이 있거나 활발한지 발견할 수 있습니다
- 다양한 사이트에서 공통 관심사를 가진 사용자를 찾을 수 있습니다
- 전체 FastComments 네트워크에서 평판을 쌓을 수 있습니다
- 커뮤니티 참여의 폭을 확인할 수 있습니다

**Examples:**
- 기술 포럼과 게임 사이트에서 활발히 활동하는 사용자
- 뉴스 블로그와 레시피 블로그에 모두 기여하는 사람
- 관련 주제 전반에 걸쳐 참여하는 커뮤니티 구성원

### Best Practices

**For Viewing Activity:**
- 사용자의 전문성과 관심사를 이해하려면 활동을 확인하세요
- 공통 기반을 찾기 위해 커뮤니티 참여를 살펴보세요
- 참여 수준을 확인하려면 카르마와 댓글 수를 검토하세요

**For Managing Your Activity:**
- 본인의 편안한 수준에 맞게 개인정보 설정을 구성하세요
- 활동이 표시될 수 있음을 인지하고 신중하게 참여하세요(비공개로 설정하지 않은 경우)
- 관련 대화에 집중하려면 필터를 사용하세요
- 대화에 계속 참여하려면 알림을 확인하세요

**For Community Building:**
- 활동 피드를 통해 활발한 참가자를 인식하세요
- 관심을 공유하는 사용자와 연결하세요
- 활동 기록을 사용해 가치 있는 커뮤니티 구성원을 식별하세요

### Troubleshooting

**Activity Not Showing:**
- 사용자가 "Make Profile Activity Private"를 활성화했는지 확인하세요
- 필요 시 로그인되어 있는지 확인하세요
- 활동이 오래된 것 같으면 페이지를 새로고침하세요

**Notifications Not Appearing:**
- 계정의 알림 설정을 확인하세요
- 알림이 이메일 제공업체에 의해 필터링되지 않는지 확인하세요
- WebSocket 연결(실시간 업데이트)을 확인하세요

**Wrong Activity Displayed:**
- 브라우저 캐시를 지우고 새로고침하세요
- 로그아웃했다가 다시 로그인하세요
- 지속적인 문제가 있으면 지원팀에 보고하세요
