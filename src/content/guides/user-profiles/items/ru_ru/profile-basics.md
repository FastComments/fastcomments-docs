Профили пользователей в FastComments предоставляют отдельное пространство для каждого пользователя, где он может демонстрировать свою личность, активность и вклад в ваше сообщество.

### Что такое профиль пользователя?

Профиль пользователя — это персонализированная страница для каждого пользователя FastComments, на которой отображается:

- **Заголовок профиля** - настраиваемое фоновое изображение, которое персонализирует профиль
- **Аватар** - фотография профиля пользователя с индикатором статуса онлайн/офлайн
- **Отображаемая информация** - имя пользователя, отображаемое имя и необязательный флаг страны
- **Биография** - личное описание или введение
- **Ссылки на соцсети** - ссылки на профили пользователя в социальных сетях и сайт
- **Значки** - полученные достижения и признание
- **Статистика** - карма пользователя и общее количество комментариев
- **Сообщества** - на каких сайтах/доменах пользователь активен

### Доступ к профилям пользователей

Существует несколько способов получить доступ к профилю пользователя:

1. **Клик по аватару** - в виджете комментариев нажмите на аватар любого пользователя, чтобы просмотреть его профиль
2. **Клик по имени пользователя** - имена пользователей в комментариях являются кликабельными ссылками на профиль
3. **Прямая ссылка** - перейдите по адресу `https://fastcomments.com/auth/user-profile/[userId]`

### Просмотр профиля

При просмотре профиля вы увидите разные вкладки в зависимости от того, просматриваете ли вы свой профиль или профиль другого пользователя:

#### Ваш собственный профиль
- **Уведомления** - ваши уведомления и упоминания
- **Последняя активность** - история ваших комментариев во всех сообществах
- **Комментарии на профиле** - комментарии, которые другие оставили на вашей странице профиля
- **Личные сообщения** - приватные переписки с другими пользователями

#### Профили других пользователей
- **Последняя активность** - их публичная история комментариев (если не установлена приватность)
- **Комментарии на профиле** - комментарии на их профиле (если не установлена приватность)
- **Личные сообщения** - начать или продолжить приватный разговор (если они разрешают личные сообщения)

### Статус онлайн

Профили пользователей отображают статус онлайн в реальном времени:
- **Зелёный индикатор** - пользователь сейчас в сети
- **Нет индикатора** - пользователь не в сети

Это помогает понять, когда кто-то активно использует платформу, что особенно полезно для личных сообщений.

### Типы пользователей

FastComments поддерживает два типа пользователей с профилями:

1. **Обычные пользователи** - пользователи, зарегистрировавшиеся напрямую в FastComments
2. **Пользователи SSO** - пользователи, которые проходят аутентификацию через интеграцию Single Sign-On вашего сайта

Оба типа пользователей имеют доступ ко всей системе профилей, хотя у пользователей SSO могут быть ограничения на редактирование определённых полей (например, аватаров) в зависимости от вашей конфигурации SSO.
