FastComments User Profiles sont hautement personnalisables, permettant aux utilisateurs d'exprimer leur personnalité et de développer leur présence en ligne.

### Editing Your Profile

Pour personnaliser votre profil, visitez votre page [Détails du compte](https://fastcomments.com/auth/my-account/edit-details). Vous pouvez y modifier tous les aspects de l'apparence et des informations de votre profil.

### Profile Avatar

Votre avatar est l'image de profil qui apparaît à côté de vos commentaires et sur votre page de profil.

**To change your avatar:**
1. Visitez votre page de profil
2. Cliquez sur le lien "Modifier l'avatar" (visible uniquement sur votre propre profil)
3. Téléversez une nouvelle image

**Avatar Requirements:**
- Taille recommandée : 200x200 pixels ou plus
- Formats pris en charge : JPG, PNG, GIF
- Les images carrées conviennent le mieux

Remarque : les utilisateurs SSO peuvent voir leur avatar contrôlé par le système SSO de leur site.

### Profile Header Background

Le fond d'en-tête est une grande image de bannière en haut de votre profil qui peut être personnalisée pour refléter votre style.

**To upload a header background:**
1. Visitez votre page de profil
2. Cliquez sur le bouton "Téléverser l'en-tête"
3. Sélectionnez une image depuis votre appareil

**Header Requirements:**
- Taille recommandée : 1200x300 pixels
- Formats pris en charge : JPG, PNG
- Les images en format paysage conviennent le mieux
- Les images sont redimensionnées automatiquement pour s'adapter

Si vous n'uploadez pas d'en-tête personnalisé, un fond par défaut sera affiché.

### Display Name

Votre nom d'affichage correspond à la façon dont vous apparaissez aux autres utilisateurs sur FastComments. Contrairement aux noms d'utilisateur (qui doivent être uniques), les noms d'affichage :

- N'ont pas besoin d'être uniques
- Peuvent contenir des espaces et des caractères spéciaux
- Peuvent inclure des émojis

**To set your display name:**
1. Allez à [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Entrez votre nom d'affichage préféré
3. Enregistrez les modifications

### Username

Votre nom d'utilisateur est un identifiant unique utilisé pour vous connecter à FastComments.

- Doit être unique parmi tous les utilisateurs de FastComments
- Utilisé dans l'URL de votre profil : `fastcomments.com/auth/user-profile/username`
- Ne peut pas contenir d'espaces ni de caractères spéciaux

**Note:** les utilisateurs SSO peuvent avoir des noms d'utilisateur non uniques puisqu'ils sont liés à un locataire/communauté spécifique.

### Bio

Ajoutez une biographie personnelle pour parler de vous, de vos intérêts ou de ce que vous faites.

**To add a bio:**
1. Allez à [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Saisissez du texte dans le champ Biographie
3. Enregistrez les modifications

Votre biographie s'affiche de manière visible sur votre page de profil et aide les autres à comprendre qui vous êtes.

### Website URL

Ajoutez un lien vers votre site personnel, votre blog ou votre portfolio.

**To add a website:**
1. Allez à [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Entrez l'URL de votre site dans le champ Website
3. Enregistrez les modifications

Le lien vers votre site web s'affichera sur votre profil avec une icône cliquable.

### Social Media Links

Connectez vos profils de réseaux sociaux à votre profil FastComments. Les plateformes prises en charge incluent :

- **Facebook** - Lien vers votre profil Facebook
- **Instagram** - Lien vers votre compte Instagram
- **LinkedIn** - Lien vers votre profil LinkedIn
- **Custom Links** - Ajoutez tout autre profil de réseau social ou site web

**To add social links:**
1. Allez à [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Entrez les URL de vos comptes de réseaux sociaux
3. Enregistrez les modifications

Les liens sociaux apparaissent sur votre profil avec des icônes reconnaissables des plateformes, ce qui permet aux membres de la communauté de se connecter avec vous ailleurs.

### Country

Vous pouvez éventuellement définir votre pays, ce qui affiche une icône de drapeau à côté de votre nom.

**To set your country:**
1. Allez à [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Sélectionnez votre pays dans le menu déroulant
3. Enregistrez les modifications

Le drapeau du pays apparaît à côté de votre nom d'utilisateur sur votre profil.

### Backup Email

Vous pouvez ajouter un courriel de secours à votre compte pour la récupération. Si vous perdez l'accès à votre courriel principal, votre courriel de secours peut être utilisé pour vous connecter ou récupérer votre nom d'utilisateur.

**To add a backup email:**
1. Allez à [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Cliquez sur **Définir un courriel de secours** sous le champ de courriel
3. Entrez votre adresse de courriel de secours et soumettez
4. Cliquez sur le lien de vérification envoyé à cette adresse

Votre courriel de secours n'est utilisé que pour la récupération du compte et ne recevra pas de notifications ni d'autres courriels. Vous pouvez le modifier ou le supprimer à tout moment.

Remarque : le courriel de secours n'est disponible que pour les utilisateurs réguliers, pas pour les utilisateurs SSO.
