Les profils utilisateur FastComments sont hautement personnalisables, permettant aux utilisateurs d'exprimer leur personnalité et de construire leur présence en ligne.

### Modifier votre profil

Pour personnaliser votre profil, rendez-vous sur votre page [Détails du compte](https://fastcomments.com/auth/my-account/edit-details). Ici, vous pouvez modifier tous les aspects de l'apparence et des informations de votre profil.

### Avatar du profil

Votre avatar est la photo de profil qui apparaît à côté de vos commentaires et sur votre page de profil.

**Pour changer votre avatar :**
1. Rendez-vous sur votre page de profil
2. Cliquez sur le lien "Modifier l'avatar" (visible seulement sur votre propre profil)
3. Téléversez une nouvelle image

**Exigences pour l'avatar :**
- Taille recommandée : 200x200 pixels ou plus
- Formats pris en charge : JPG, PNG, GIF
- Les images carrées conviennent le mieux

Remarque : les utilisateurs SSO peuvent avoir leur avatar contrôlé par le système SSO de leur site.

### Arrière-plan de l'en-tête du profil

L'arrière-plan de l'en-tête est une grande image de bannière en haut de votre profil qui peut être personnalisée pour refléter votre style.

**Pour téléverser un arrière-plan d'en-tête :**
1. Rendez-vous sur votre page de profil
2. Cliquez sur le bouton "Téléverser l'en-tête"
3. Sélectionnez une image depuis votre appareil

**Exigences pour l'en-tête :**
- Taille recommandée : 1200x300 pixels
- Formats pris en charge : JPG, PNG
- Les images au format paysage/large conviennent le mieux
- Les images sont automatiquement redimensionnées pour s'adapter

Si vous ne téléversez pas d'en-tête personnalisé, un arrière-plan par défaut sera affiché.

### Nom d'affichage

Votre nom d'affichage est la façon dont vous apparaissez aux autres utilisateurs sur FastComments. Contrairement aux noms d'utilisateur (qui doivent être uniques), les noms d'affichage :

- N'ont pas besoin d'être uniques
- Peuvent contenir des espaces et des caractères spéciaux
- Peuvent inclure des émojis

**Pour définir votre nom d'affichage :**
1. Allez dans [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Entrez le nom d'affichage souhaité
3. Enregistrez les modifications

### Nom d'utilisateur

Votre nom d'utilisateur est un identifiant unique utilisé pour vous connecter à FastComments.

- Doit être unique parmi tous les utilisateurs de FastComments
- Utilisé dans l'URL de votre profil : `fastcomments.com/auth/user-profile/username`
- Ne peut pas contenir d'espaces ni de caractères spéciaux

Remarque : les utilisateurs SSO peuvent avoir des noms d'utilisateur non uniques puisqu'ils sont liés à un locataire/une communauté spécifique.

### Bio

Ajoutez une bio personnelle pour parler de vous, de vos intérêts ou de ce que vous faites.

**Pour ajouter une bio :**
1. Allez dans [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Saisissez du texte dans le champ Bio
3. Enregistrez les modifications

Votre bio apparaît bien en évidence sur votre page de profil et aide les autres à comprendre qui vous êtes.

### URL du site Web

Ajoutez un lien vers votre site personnel, blog ou portfolio.

**Pour ajouter un site web :**
1. Allez dans [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Entrez l'URL de votre site dans le champ Site web
3. Enregistrez les modifications

Le lien vers votre site web s'affichera sur votre profil avec une icône cliquable.

### Liens de réseaux sociaux

Connectez vos profils de réseaux sociaux à votre profil FastComments. Les plateformes prises en charge incluent :

- **Facebook** - Lien vers votre profil Facebook
- **Instagram** - Lien vers votre compte Instagram
- **LinkedIn** - Lien vers votre profil LinkedIn
- **Liens personnalisés** - Ajoutez tout autre profil social ou web

**Pour ajouter des liens sociaux :**
1. Allez dans [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Entrez les URL de vos comptes de réseaux sociaux
3. Enregistrez les modifications

Les liens sociaux apparaissent sur votre profil avec des icônes reconnaissables des plateformes, facilitant la connexion des membres de la communauté avec vous ailleurs.

### Pays

Vous pouvez facultativement définir votre pays, ce qui affiche une icône de drapeau à côté de votre nom.

**Pour définir votre pays :**
1. Allez dans [Détails du compte](https://fastcomments.com/auth/my-account/edit-details)
2. Sélectionnez votre pays dans la liste déroulante
3. Enregistrez les modifications

Le drapeau du pays apparaît à côté de votre nom d'utilisateur sur votre profil.
