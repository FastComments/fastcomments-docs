Les Notes d'utilisateur vous permettent de conserver des annotations privées et personnelles à propos d'autres utilisateurs de FastComments. Ces notes sont entièrement privées et visibles seulement par vous.

### Qu'est-ce que les Notes d'utilisateur ?

Les Notes d'utilisateur sont :
- **Des annotations privées** que vous créez à propos d'autres utilisateurs
- **Visibles uniquement par vous** - l'utilisateur que vous décrivez ne peut pas les voir
- **Persistantes** - sauvegardées et accessibles chaque fois que vous consultez le profil de cet utilisateur
- **Utiles pour se rappeler** du contexte, des interactions ou des détails importants à propos des utilisateurs

### Pourquoi utiliser les Notes d'utilisateur ?

Les Notes d'utilisateur sont utiles pour :

**Modérateurs :**
- Suivre l'historique des interactions avec des utilisateurs spécifiques
- Noter des schémas d'avertissement ou des observations comportementales
- Se souvenir du contexte pour les décisions de modération
- Documenter pourquoi certaines actions ont été prises

**Responsables de communauté :**
- Se souvenir des contributeurs précieux et de leur expertise
- Noter les utilisateurs qui pourraient être de bons candidats pour des rôles spéciaux
- Suivre les relances nécessaires avec les membres de la communauté

**Utilisateurs réguliers :**
- Se rappeler comment vous connaissez quelqu'un à travers différentes communautés
- Noter les sujets que vous avez discutés avec des utilisateurs spécifiques
- Suivre les contributeurs utiles dans vos domaines d'intérêt
- Se souvenir des détails personnels partagés dans des conversations (dans des limites appropriées)

### Créer une Note d'utilisateur

Pour ajouter une note à propos d'un utilisateur :

1. Visitez la page de profil de l'utilisateur
2. Cherchez la section « Note d'utilisateur » ou « Ajouter une note »
3. Cliquez pour créer une nouvelle note
4. Tapez votre note (jusqu'à 1000 caractères)
5. Enregistrez la note

La note sera sauvegardée et apparaîtra chaque fois que vous consulterez le profil de cet utilisateur.

### Voir les Notes d'utilisateur

Vos notes apparaissent lorsque vous consultez le profil de l'utilisateur associé :

- **Emplacement :** Généralement dans une section dédiée sur la page de profil
- **Visibilité :** Visible uniquement par vous lorsque vous êtes connecté
- **Contexte :** Apparaît en même temps que les informations publiques de l'utilisateur

Cela facilite la mémorisation du contexte important à propos des utilisateurs avec lesquels vous interagissez régulièrement.

### Modifier les Notes d'utilisateur

Pour mettre à jour une note existante :

1. Visitez le profil de l'utilisateur
2. Trouvez votre note existante
3. Cliquez sur « Modifier » ou cliquez dans le champ de la note
4. Apportez vos modifications
5. Enregistrez la note mise à jour

### Supprimer les Notes d'utilisateur

Pour supprimer une note :

1. Visitez le profil de l'utilisateur
2. Trouvez votre note
3. Cliquez sur « Supprimer » ou l'option de suppression
4. Confirmez la suppression

Une fois supprimée, la note est définitivement effacée et ne peut pas être récupérée.

### Limite de caractères

Les Notes d'utilisateur sont limitées à **1000 caractères**. Cela encourage :
- Des notes concises et ciblées
- Des points clés plutôt que de longs récits
- Des informations de référence rapide

Si vous avez besoin de plus d'espace, envisagez de garder une documentation séparée en dehors de FastComments.

### Confidentialité et sécurité

**Confidentialité complète :**
- Les notes sont stockées avec votre ID d'utilisateur
- Seul vous pouvez voir les notes que vous avez créées
- L'utilisateur sur lequel vous écrivez ne peut pas voir vos notes
- Les autres utilisateurs ne peuvent pas voir vos notes
- Les administrateurs du site ne peuvent pas voir vos notes personnelles (elles sont stockées séparément des journaux de modération)

**Remarque concernant la modération :**
Les Notes d'utilisateur sont différentes des notes ou signalements de modération :
- Utilisez les Notes d'utilisateur pour votre référence personnelle
- Utilisez les outils de signalement/modération pour les violations ou problèmes
- Les Notes d'utilisateur ne déclenchent aucune action ni alerte

### Bonnes pratiques

**Ce qu'il faut inclure :**
- Comment vous avez rencontré ou connaissez l'utilisateur
- Les sujets sur lesquels il/elle est compétent(e)
- Les interactions utiles précédentes
- Le contexte des conversations passées
- Les domaines d'expertise ou d'intérêt
- Rappels pour des suivis

**Ce qu'il faut éviter :**
- N'utilisez pas les notes pour stocker des informations personnelles sensibles
- Évitez le contenu incendiaire ou inapproprié même dans des notes privées
- Ne comptez pas uniquement sur les notes pour des informations critiques (gardez des sauvegardes ailleurs)
- Rappelez-vous que les notes servent uniquement de référence personnelle, pas de documents officiels

**Usage professionnel :**
Pour les modérateurs et responsables de communauté :
- Gardez les notes factuelles et objectives
- Incluez des dates pour les informations sensibles dans le temps
- Notez des schémas plutôt que des incidents isolés
- Utilisez les notes pour compléter (et non remplacer) les journaux officiels de modération

### Cas d'utilisation

**Exemple 1 : Responsable de communauté**
> « Actif dans les discussions tech. Travaille comme développeur backend. A proposé d'aider à la documentation. Relancer au sujet de contribuer au wiki. »

**Exemple 2 : Modérateur**
> « Utilisateur utile mais qui parfois s'engage dans des débats houleux. Généralement respectueux. A bien réagi au rappel précédent sur la civilité. »

**Exemple 3 : Utilisateur régulier**
> « Rencontré dans des fils de discussion de jeux. Joue aux mêmes jeux que moi. A partagé un lien de tutoriel utile. Basé au fuseau horaire du Royaume-Uni. »

### Limitations

**Ce que les Notes d'utilisateur ne font pas :**
- Elles ne se synchronisent pas entre les appareils (sauf si vous êtes connecté au même compte)
- Elles n'apparaissent pas dans les commentaires ou autres contextes, seulement sur les profils
- Elles n'affectent en rien le profil de l'utilisateur ou sa visibilité
- Elles ne sont pas recherchables (actuellement)
- Elles ne déclenchent pas de notifications ni d'actions

**Stockage :**
- Les notes sont stockées indéfiniment sauf si vous les supprimez
- Maximum 1000 caractères par note
- Une note par utilisateur (mettez à jour la note existante plutôt que d'en créer plusieurs)

### Détails techniques

Les Notes d'utilisateur sont :
- Stockées dans la base de données avec votre ID d'utilisateur en tant que créateur
- Associées à l'ID de l'utilisateur cible
- Chiffrées en transit (HTTPS)
- Soumises aux politiques standard de rétention des données de FastComments
- Non incluses dans les exportations de données publiques ou les réponses API

### Dépannage

**La note ne s'enregistre pas :**
- Vérifiez que vous êtes connecté
- Assurez-vous que la note fait moins de 1000 caractères
- Essayez d'actualiser la page et de l'ajouter de nouveau
- Vérifiez votre connexion Internet

**La note n'apparaît pas :**
- Assurez-vous d'être connecté au compte qui a créé la note
- Actualisez la page de profil de l'utilisateur
- Videz le cache du navigateur si nécessaire

**Note supprimée par accident :**
Malheureusement, les notes supprimées ne peuvent pas être récupérées. Envisagez de conserver des sauvegardes des informations importantes en dehors de FastComments.
