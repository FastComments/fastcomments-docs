FastComments vous donne un contrôle granulaire sur la confidentialité de votre profil et sur qui peut interagir avec vous. Tous les paramètres de confidentialité peuvent être gérés depuis la section Préférences de confidentialité de votre page de profil.

### Accessing Privacy Settings

Pour gérer vos paramètres de confidentialité :
1. Visitez votre page de profil
2. Cliquez sur l'onglet ou la section « Préférences de confidentialité »
3. Ajustez vos paramètres comme vous le souhaitez
4. Les modifications sont enregistrées automatiquement

### Activity Visibility

Contrôlez qui peut voir votre activité de commentaires à travers les communautés FastComments.

**Paramètre :** Rendre l'activité du profil privée

- **Lorsque désactivé (par défaut) :** Toute personne peut voir vos commentaires récents et votre activité sur votre profil
- **Lorsque activé :** Seul vous pouvez voir votre historique de commentaires et votre fil d'activité

Ceci est utile si vous souhaitez participer aux discussions mais préférez garder l'historique de vos commentaires privé vis-à-vis des autres utilisateurs.

### Profile Comments Visibility

Contrôlez si les autres peuvent voir et laisser des commentaires sur votre page de profil.

**Paramètre :** Rendre les commentaires du profil privés

- **Lorsque désactivé (par défaut) :** Toute personne peut voir et laisser des commentaires sur votre profil
- **Lorsque activé :** Les commentaires du profil sont cachés aux autres ; seul vous pouvez voir les commentaires laissés sur votre profil

Utilisez ce paramètre si vous souhaitez limiter les interactions sur votre page de profil tout en permettant l'activité ailleurs.

### Direct Messaging

Contrôlez si d'autres utilisateurs peuvent vous envoyer des messages directs.

**Paramètre :** Désactiver les messages directs

- **Lorsque désactivé (par défaut) :** Les autres utilisateurs peuvent vous envoyer des messages directs
- **Lorsque activé :** L'option Messages directs est masquée de votre profil ; les utilisateurs ne peuvent pas initier de conversations avec vous

Ceci est utile si vous préférez ne pas recevoir de messages privés ou si vous souhaitez limiter les conversations en tête-à-tête.

### Profile Comment Approval

Contrôlez la façon dont les commentaires sur votre profil sont modérés avant d'apparaître publiquement.

**Paramètre :** Mode d'approbation des commentaires du profil

Vous avez quatre options pour gérer les commentaires de profil :

#### 1. Auto Approve All
Tous les commentaires publiés sur votre profil sont immédiatement visibles sans nécessiter d'approbation.

**Idéal pour :** Les utilisateurs qui veulent une section de commentaires ouverte et sans restriction.

#### 2. Auto Approve Trusted (Default)
Les commentaires des utilisateurs de confiance sont automatiquement approuvés, tandis que les commentaires des utilisateurs nouveaux ou non fiables nécessitent une approbation manuelle.

**Idéal pour :** La plupart des utilisateurs - équilibre entre ouverture et protection contre le spam ou le contenu indésirable.

**Ce qui rend un utilisateur « de confiance » :**
- Utilisateurs avec un bon karma
- Utilisateurs avec un historique de commentaires établi
- Utilisateurs qui n'ont pas été signalés pour spam ou violations

#### 3. Manual Approve All
Tous les commentaires nécessitent votre approbation manuelle avant d'apparaître sur votre profil, indépendamment de l'auteur.

**Idéal pour :** Les utilisateurs qui veulent un contrôle complet sur ce qui apparaît sur leur profil.

**Comment fonctionne l'approbation :**
- Les nouveaux commentaires sont retenus pour révision
- Vous recevez une notification lorsqu'une personne commente
- Vous pouvez approuver ou rejeter les commentaires via le courriel ou depuis votre profil

#### 4. Manual Approve New
Seuls les commentaires des utilisateurs nouveaux ou non fiables nécessitent une approbation. Les commentaires des membres établis de la communauté sont approuvés automatiquement.

**Idéal pour :** Les utilisateurs qui veulent filtrer les nouveaux utilisateurs tout en permettant aux membres établis de commenter librement.

### Managing Pending Profile Comments

Lorsque des commentaires nécessitent une approbation, vous recevrez des notifications et pourrez les gérer de plusieurs façons :

1. **Via le courriel :** Approuvez ou rejetez directement depuis le courriel de notification
2. **Depuis votre profil :** Voir les commentaires en attente dans l'onglet Commentaires de profil
3. **Centre de notifications :** Accéder aux commentaires en attente depuis vos notifications

### Blocking Users

En plus des paramètres de confidentialité du profil, vous pouvez bloquer des utilisateurs spécifiques pour les empêcher d'interagir avec vous n'importe où sur FastComments :

- Les utilisateurs bloqués ne peuvent pas commenter sur votre profil
- Les utilisateurs bloqués ne peuvent pas vous envoyer de messages directs
- Les commentaires des utilisateurs bloqués sont cachés de votre vue
- Vous ne verrez pas l'activité de l'un et l'autre

Pour bloquer un utilisateur, visitez son profil et cliquez sur l'option « Bloquer l'utilisateur ».

### Privacy Best Practices

Voici quelques configurations de confidentialité recommandées pour différents cas d'utilisation :

**Membre de communauté public :**
- Activity Visibility : Public
- Profile Comments : Public
- Direct Messages : Activés
- Comment Approval : Auto Approve Trusted

**Utilisateur privé :**
- Activity Visibility : Privé
- Profile Comments : Privé
- Direct Messages : Désactivés
- Comment Approval : Manual Approve All

**Modérateur / Personnalité publique :**
- Activity Visibility : Public
- Profile Comments : Public
- Direct Messages : Activés (ou désactivés si submergé)
- Comment Approval : Auto Approve Trusted ou Manual Approve New
