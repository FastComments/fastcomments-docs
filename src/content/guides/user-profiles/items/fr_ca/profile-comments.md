Les commentaires de profil permettent aux utilisateurs de laisser des messages directement sur la page de profil d’un autre utilisateur, créant un mur de commentaires personnel ou une interaction de style livre d’or.

### Qu’est-ce que les commentaires de profil ?

Les commentaires de profil diffèrent des commentaires d’articles réguliers :
- Ils apparaissent sur la page de profil d’un utilisateur, pas sur des articles de contenu
- Ils sont visibles par toute personne consultant le profil (sauf si les paramètres de confidentialité en restreignent l’accès)
- Ils constituent un moyen de reconnaître, remercier ou communiquer avec des utilisateurs spécifiques
- Ils fonctionnent dans toutes les communautés FastComments

### Laisser un commentaire de profil

Pour commenter le profil de quelqu’un :

1. Visitez la page de profil de l’utilisateur
2. Allez à l’onglet "Profile Comments"
3. Utilisez la zone de commentaire pour écrire votre message
4. Soumettez le commentaire

**Remarque :** Les paramètres de confidentialité du propriétaire du profil déterminent si votre commentaire apparaît immédiatement ou doit être approuvé.

### Voir les commentaires de profil

**Sur votre propre profil :**
1. Visitez votre page de profil
2. Cliquez sur l’onglet "Profile Comments"
3. Voyez tous les commentaires que d’autres ont laissés sur votre profil

**Sur d’autres profils :**
Les commentaires de profil sont visibles si l’utilisateur n’a pas activé le paramètre "Make Profile Comments Private".

### Approbation des commentaires

Selon les paramètres du propriétaire du profil (voir [Paramètres de confidentialité](/guide-user-profiles.html#privacy-settings)), les commentaires peuvent :

1. **Apparaître immédiatement** - Auto-approuvés en fonction du niveau de confiance ou des paramètres d’approbation
2. **Requérir une approbation** - Mis en attente pour que le propriétaire du profil les examine

Lorsque votre commentaire requiert une approbation :
- Vous verrez un indicateur d’état en attente
- Le propriétaire du profil reçoit une notification
- Une fois approuvé, le commentaire devient visible pour les autres

### Gérer les commentaires de profil sur votre profil

Si vous avez configuré l’approbation des commentaires (Approbation manuelle — Tous ou Approbation manuelle — Nouveaux), vous pouvez gérer les commentaires entrants :

**Par courriel :**
Vous recevrez des notifications par courriel avec des liens d’approbation/rejet pour les commentaires en attente. Cliquez sur le lien pour agir directement depuis votre courriel.

**Depuis votre profil :**
1. Visitez l’onglet "Profile Comments" de votre profil
2. Voyez les commentaires en attente
3. Approuvez ou rejettez chaque commentaire

**Depuis les notifications :**
Accédez aux commentaires en attente depuis votre centre de notifications pour un examen rapide.

### Répondre aux commentaires de profil

Lorsqu’une personne commente votre profil, vous pouvez :
- Répondre directement à son commentaire (crée un fil de conversation)
- Visiter son profil et laisser un commentaire en retour
- Lui envoyer un message direct (si elle autorise les messages directs)

### Supprimer des commentaires de profil

**Sur votre propre profil :**
Vous pouvez supprimer n’importe quel commentaire laissé sur votre profil :
1. Trouvez le commentaire que vous voulez supprimer
2. Cliquez sur l’option supprimer/retirer
3. Confirmez la suppression

**Sur d’autres profils :**
Vous pouvez supprimer vos propres commentaires laissés sur le profil d’autres personnes, mais vous ne pouvez pas supprimer les commentaires des autres (sauf si vous êtes le propriétaire du profil).

### Notifications de commentaires de profil

Vous recevrez des notifications lorsque :
- Quelqu’un laisse un commentaire sur votre profil
- Quelqu’un répond à un fil de commentaires de profil dont vous faites partie
- Un commentaire de profil que vous avez laissé est approuvé ou reçoit une réponse

Gérez les préférences de notification dans vos [Paramètres du compte](https://fastcomments.com/auth/my-account/edit-notifications).

### Confidentialité et commentaires de profil

Contrôlez la visibilité des commentaires de profil avec ces paramètres de confidentialité :

**Rendre les commentaires de profil privés :**
- Masque l’onglet Profile Comments aux visiteurs
- Seul vous pouvez voir les commentaires laissés sur votre profil
- Empêche la publication de nouveaux commentaires

**Modes d’approbation des commentaires :**
- Choisissez qui peut commenter sans approbation
- Définissez différentes règles d’approbation pour les utilisateurs de confiance vs les nouveaux utilisateurs
- Exigez une approbation manuelle pour tous les commentaires

Pour plus de détails, voir [Paramètres de confidentialité](/guide-user-profiles.html#privacy-settings).

### Cas d’utilisation des commentaires de profil

Les commentaires de profil sont idéaux pour :

- **Remercier des membres utiles de la communauté**
- **Reconnaître des contributions précieuses**
- **Créer des liens avec des participants actifs**
- **Laisser des témoignages ou des recommandations**
- **Créer une expérience de livre d’or social**
- **Appréciation et reconnaissance dans la communauté**

### Bonnes pratiques

**Pour laisser des commentaires :**
- Soyez respectueux et constructif
- Ajoutez de la valeur plutôt que de simples messages génériques comme « joli profil »
- Référez-vous à des contributions ou interactions spécifiques lorsque c’est pertinent

**Pour gérer votre profil :**
- Définissez des paramètres d’approbation appropriés selon votre niveau de confort
- Répondez aux commentaires de profil pour encourager l’engagement communautaire
- Utilisez le blocage si quelqu’un laisse des commentaires inappropriés de façon répétée

**Pour la santé de la communauté :**
- Les commentaires de profil suivent les mêmes lignes directrices communautaires que les commentaires d’articles
- Signalez les commentaires de profil inappropriés si vous en rencontrez
- Les modérateurs peuvent prendre des mesures sur les commentaires de profil qui violent les règles
