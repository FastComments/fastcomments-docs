Les commentaires de profil permettent aux utilisateurs de laisser des messages directement sur la page de profil d'un autre utilisateur, créant un mur de commentaires personnel ou une interaction de type livre d'or.

### Qu'est-ce que les commentaires de profil?

Les commentaires de profil sont différents des commentaires d'article habituels :
- Ils apparaissent sur la page de profil d'un utilisateur, pas sur des articles de contenu
- Ils sont visibles par toute personne consultant le profil (sauf si les paramètres de confidentialité en limitent l'accès)
- Ils permettent de reconnaître, remercier ou communiquer avec des utilisateurs spécifiques
- Ils fonctionnent dans toutes les communautés FastComments

### Laisser un commentaire de profil

Pour commenter le profil de quelqu'un :

1. Visitez la page de profil de l'utilisateur
2. Allez à l'onglet "Commentaires de profil"
3. Utilisez la zone de commentaire pour écrire votre message
4. Soumettez le commentaire

**Remarque :** Les paramètres de confidentialité du propriétaire du profil déterminent si votre commentaire apparaît immédiatement ou nécessite une approbation.

### Voir les commentaires de profil

**Sur votre propre profil :**
1. Visitez votre page de profil
2. Cliquez sur l'onglet "Commentaires de profil"
3. Voyez tous les commentaires que d'autres ont laissés sur votre profil

**Sur d'autres profils :**
Les commentaires de profil sont visibles si l'utilisateur n'a pas activé le paramètre "Rendre les commentaires de profil privés".

### Approbation des commentaires

Selon les paramètres du propriétaire du profil (voir [Privacy Settings](/guide-user-profiles.html#privacy-settings)), les commentaires peuvent :

1. **Apparaître immédiatement** - Auto-approuvé en fonction du niveau de confiance ou des paramètres d'approbation
2. **Nécessiter une approbation** - Mis en attente pour que le propriétaire du profil les examine

Lorsque votre commentaire nécessite une approbation :
- Vous verrez un indicateur d'état en attente
- Le propriétaire du profil reçoit une notification
- Une fois approuvé, le commentaire devient visible pour les autres

### Gérer les commentaires de profil sur votre profil

Si vous avez configuré l'approbation des commentaires (Approbation manuelle de tous ou Approbation manuelle des nouveaux), vous pouvez gérer les commentaires entrants :

**Par courriel :**
Vous recevrez des notifications par courriel avec des liens d'approuver/refuser pour les commentaires en attente. Cliquez sur le lien pour agir directement depuis votre courriel.

**Depuis votre profil :**
1. Visitez l'onglet "Commentaires de profil" de votre profil
2. Consultez les commentaires en attente
3. Approuvez ou rejetez chaque commentaire

**Depuis les notifications :**
Accédez aux commentaires en attente depuis votre centre de notifications pour une révision rapide.

### Répondre aux commentaires de profil

Lorsque quelqu'un commente votre profil, vous pouvez :
- Répondre directement à son commentaire (crée un fil de conversation)
- Visiter son profil et laisser un commentaire en retour
- Lui envoyer un message direct (s'il autorise les messages directs)

### Supprimer des commentaires de profil

**Sur votre propre profil :**
Vous pouvez supprimer n'importe quel commentaire laissé sur votre profil :
1. Trouvez le commentaire que vous souhaitez supprimer
2. Cliquez sur l'option supprimer/retirer
3. Confirmez la suppression

**Sur d'autres profils :**
Vous pouvez supprimer vos propres commentaires que vous avez laissés sur les profils d'autres personnes, mais vous ne pouvez pas supprimer les commentaires d'autres personnes (à moins que vous ne soyez le propriétaire du profil).

### Notifications de commentaires de profil

Vous recevrez des notifications lorsque :
- Quelqu'un laisse un commentaire sur votre profil
- Quelqu'un répond à un fil de commentaires de profil dont vous faites partie
- Un commentaire de profil que vous avez laissé est approuvé ou reçoit une réponse

Gérez les préférences de notification dans vos [Account Settings](https://fastcomments.com/auth/my-account/edit-notifications).

### Confidentialité et commentaires de profil

Contrôlez la visibilité des commentaires de profil avec ces paramètres de confidentialité :

**Rendre les commentaires de profil privés :**
- Masque l'onglet Commentaires de profil aux visiteurs
- Seul vous pouvez voir les commentaires laissés sur votre profil
- Empêche la publication de nouveaux commentaires

**Modes d'approbation des commentaires :**
- Choisissez qui peut commenter sans approbation
- Définissez des règles d'approbation différentes pour les utilisateurs de confiance vs. les nouveaux utilisateurs
- Exigez une approbation manuelle pour tous les commentaires

Pour plus de détails, consultez [Privacy Settings](/guide-user-profiles.html#privacy-settings).

### Cas d'utilisation des commentaires de profil

Les commentaires de profil sont excellents pour :

- **Remercier les membres utiles de la communauté**
- **Reconnaître des contributions de valeur**
- **Créer des liens avec les participants actifs**
- **Laisser des témoignages ou des recommandations**
- **Créer une expérience de livre d'or social**
- **Appréciation et reconnaissance communautaire**

### Meilleures pratiques

**Pour laisser des commentaires :**
- Soyez respectueux et constructif
- Ajoutez de la valeur plutôt que de simples messages génériques comme "beau profil"
- Faites référence à des contributions ou interactions spécifiques lorsque c'est pertinent

**Pour gérer votre profil :**
- Définissez des paramètres d'approbation appropriés selon votre niveau de confort
- Répondez aux commentaires de profil pour encourager l'engagement communautaire
- Utilisez le blocage si quelqu'un laisse des commentaires inappropriés de façon répétée

**Pour la santé de la communauté :**
- Les commentaires de profil suivent les mêmes directives de la communauté que les commentaires d'article
- Signalez les commentaires de profil inappropriés si vous en rencontrez
- Les modérateurs peuvent intervenir sur les commentaires de profil qui enfreignent les règles
