User Profiles in FastComments provide a dedicated space for each user to showcase their identity, activity, and contributions across your community.

### Что такое профиль пользователя?

Профиль пользователя — это персональная страница для каждого пользователя FastComments, которая отображает:

- **Шапка профиля** - Настраиваемое фоновое изображение, персонализирующее профиль
- **Аватар** - Фотография профиля пользователя с индикатором онлайн/оффлайн
- **Информация для отображения** - Имя пользователя, отображаемое имя и необязательный флаг страны
- **Биография** - Личное описание или представление
- **Ссылки на соцсети** - Ссылки на профили пользователя в социальных сетях и сайт
- **Значки** - Полученные достижения и признания
- **Статистика** - Карма пользователя и общее количество комментариев
- **Сообщества** - На каких сайтах/доменах пользователь активен

### Доступ к профилям пользователей

Существует несколько способов получить доступ к профилю пользователя:

1. **Клик по аватару** - В виджете комментариев кликните на аватар любого пользователя, чтобы просмотреть его профиль
2. **Клик по имени пользователя** - Имена пользователей в комментариях являются кликабельными ссылками на профиль
3. **Прямой URL** - Перейдите по `https://fastcomments.com/auth/user-profile/[userId]`

### Просмотр профиля

При просмотре профиля вы увидите разные вкладки в зависимости от того, просматриваете ли вы собственный профиль или чужой:

#### Ваш собственный профиль
- **Уведомления** - Ваши уведомления и упоминания
- **Последняя активность** - История ваших комментариев во всех сообществах
- **Комментарии в профиле** - Комментарии, которые другие оставили на вашей странице профиля
- **Личные сообщения** - Приватные переписки с другими пользователями

#### Профили других пользователей
- **Последняя активность** - Их публичная история комментариев (если не установлена приватность)
- **Комментарии в профиле** - Комментарии на их профиле (если не установлена приватность)
- **Личные сообщения** - Начать или продолжить приватный разговор (если они разрешают личные сообщения)

### Онлайн-статус

Профили пользователей отображают онлайн-статус в реальном времени:
- **Зелёный индикатор** - Пользователь сейчас онлайн
- **Отсутствие индикатора** - Пользователь офлайн

Это помогает понять, когда кто-то активно использует платформу, что особенно полезно для отправки личных сообщений.

### Типы пользователей

FastComments поддерживает два типа пользователей с профилями:

1. **Обычные пользователи** - Пользователи, которые зарегистрировались напрямую в FastComments
2. **Пользователи SSO** - Пользователи, которые аутентифицируются через интеграцию Single Sign-On вашего сайта

Оба типа пользователей имеют доступ ко всей системе профилей, хотя у пользователей SSO могут быть ограничения на редактирование некоторых полей (например, аватаров) в зависимости от конфигурации SSO.
