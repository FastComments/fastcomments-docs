使用者備註讓你可以針對其他 FastComments 使用者保留私人、個人的備註。這些備註完全私密，只有你能看到。

### 什麼是使用者備註？

使用者備註是：
- **私人註記** 你為其他使用者建立的
- **只有你可見** - 你所寫的使用者無法看到它們
- **持久存在** - 儲存並在你檢視該使用者的個人檔案時可存取
- **有助於記憶** 使用者的上下文、互動或重要細節

### 為什麼要使用使用者備註？

使用者備註對以下情況很有幫助：

**管理員：**
- 追蹤與特定使用者的互動歷史
- 記錄警告模式或行為觀察
- 記住作出管理決策的背景
- 記錄為何採取某些行動

**社群管理者：**
- 記住有價值的貢獻者及其專長
- 記錄可能適合擔任特殊職務的使用者
- 跟蹤需要與社群成員後續聯繫的事項

**一般使用者：**
- 記住你在不同社群中如何認識某人
- 記錄你與特定使用者討論過的主題
- 追蹤在你感興趣領域中有幫助的貢獻者
- 記住對話中分享的個人細節（在適當範圍內）

### 建立使用者備註

要新增有關某位使用者的備註：

1. 前往該使用者的個人檔案頁面
2. 尋找「使用者備註」或「新增備註」區塊
3. 點擊以建立新備註
4. 輸入你的備註（最多 1000 個字元）
5. 儲存備註

備註會被儲存，並在你檢視該使用者的個人檔案時顯示。

### 檢視使用者備註

你的備註會在檢視該使用者個人檔案時顯示：

- **位置：** 通常位於個人檔案頁的一個專屬區塊
- **可見性：** 只有在你登入時對你可見
- **上下文：** 與使用者的公開資訊一同顯示

這讓你在經常互動的使用者上更容易記住重要背景。

### 編輯使用者備註

要更新現有備註：

1. 前往該使用者的個人檔案
2. 找到你現有的備註
3. 點擊「編輯」或直接點入備註欄位
4. 做出變更
5. 儲存更新後的備註

### 刪除使用者備註

要移除備註：

1. 前往該使用者的個人檔案
2. 找到你的備註
3. 點擊「刪除」或移除選項
4. 確認刪除

一旦刪除，該備註即永久移除且無法復原。

### 字元限制

使用者備註限制為 **1000 個字元**。這鼓勵：
- 簡潔、集中的備註
- 記錄重點而不是冗長敘述
- 作為快速參考資訊

如果你需要更多空間，請考慮在 FastComments 之外另行保存文件。

### 隱私與安全

**完全私密：**
- 備註會與你的使用者 ID 一起儲存
- 只有你能看到你所建立的備註
- 你所寫的使用者無法看到你的備註
- 其他使用者無法看到你的備註
- 網站管理員無法看到你的個人備註（它們與管理記錄分開儲存）

**關於管理的說明：**
使用者備註不同於管理備註或檢舉：
- 將使用者備註用於個人參考
- 若要處理違規或問題，請使用管理報告工具
- 使用者備註不會觸發任何行動或警示

### 最佳實務

**應包含的內容：**
- 你如何認識或遇到該使用者
- 他們熟悉的主題
- 先前有幫助的互動
- 過去對話的背景
- 專長或興趣領域
- 後續聯繫的提醒

**應避免的內容：**
- 不要在備註中存放敏感個人資訊
- 即使是私人備註，也避免具煽動性或不當內容
- 不要依賴備註存放關鍵資訊（請在其他地方備份）
- 記住備註僅供你參考，並非正式紀錄

**專業用途：**
針對管理員和社群管理者：
- 保持備註事實性與客觀性
- 對於時間敏感資訊註明日期
- 記錄模式而非個別事件
- 使用備註來補充（而非取代）正式的管理記錄

### 使用案例

**範例 1：社群管理者**
> 「活躍於技術討論。擔任後端工程師。曾表示願意協助文件編寫。後續追蹤貢獻維基的事宜。」

**範例 2：管理員**
> 「是位有幫助的使用者，但有時會捲入激烈爭論。整體上態度尊重。對先前關於禮貌的提醒回應良好。」

**範例 3：一般使用者**
> 「在遊戲討論串認識。玩我相同的遊戲。分享過有用的教學連結。位於英國時區。」

### 限制

**使用者備註無法做到的事：**
- 不會在裝置之間同步（除非你使用相同帳號登入）
- 不會出現在留言或其他情境中，只會出現在個人檔案上
- 目前無法被搜尋
- 不會觸發通知或行動

**儲存：**
- 備註會無限期儲存，除非你刪除它們
- 每則備註最多 1000 個字元
- 每位使用者僅限一則備註（請更新現有備註而非建立多則）

### 技術細節

使用者備註為：
- 以你的使用者 ID 為建立者儲存在資料庫中
- 與目標使用者的 ID 相關聯
- 傳輸時加密（HTTPS）
- 受 FastComments 的標準資料保留政策約束
- 不包含在公開資料匯出或 API 回應中

### 疑難排解

**備註無法儲存：**
- 確認你已登入
- 確保備註少於 1000 個字元
- 嘗試重新整理頁面後再新增
- 檢查你的網路連線

**備註未顯示：**
- 確認你是以建立該備註的帳號登入
- 重新整理該使用者的個人檔案頁面
- 如有需要，清除瀏覽器快取

**意外刪除備註：**
很抱歉，已刪除的備註無法復原。建議在 FastComments 之外保留重要資訊的備份。
