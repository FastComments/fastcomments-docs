Tous les produits FastComments qui sont **intégrables dans votre application ou site web**, sont localisés.

Tous les **e-mails** sont entièrement localisés. La configuration régionale utilisée pour afficher les e-mails que nous envoyons est basée sur la dernière configuration régionale que l'utilisateur a utilisée lors de sa dernière connexion ou lorsqu'il a commenté avec le widget de commentaires.

Le **tableau de bord administrateur et utilisateur** n'est pas actuellement localisé, mais c'est prévu dans la feuille de route.
