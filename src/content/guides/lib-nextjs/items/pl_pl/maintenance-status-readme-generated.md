Te komponenty są wrapperami naszych podstawowych komponentów VanillaJS. Możemy automatycznie aktualizować te komponenty (naprawiać błędy, dodawać funkcje) bez publikowania tej biblioteki, więc choć może nie być ona publikowana przez jakiś czas, nie oznacza to, że FastComments nie jest aktywnie rozwijany! Zajrzyj śmiało na [nasz blog](https://blog.fastcomments.com/) po aktualizacje. Istotne zmiany API lub funkcje nigdy nie zostaną wprowadzone do leżącej u podstaw biblioteki core bez zwiększenia wersji w tej bibliotece.
