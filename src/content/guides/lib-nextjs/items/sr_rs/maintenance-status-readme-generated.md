Ови компоненти су омотачи око наших основних VanillaJS компоненти. Можемо аутоматски ажурирати ове компоненте (исправљање грешака, додавање функција) без објављивања ове библиотеке, тако да, иако можда неће бити објављена неко време, то не значи да FastComments није у активном развоју! Слободно погледајте [наш блог](https://blog.fastcomments.com/) за ажурирања.
