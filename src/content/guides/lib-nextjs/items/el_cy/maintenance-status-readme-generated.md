Αυτά τα components είναι περιτυλίγματα γύρω από τα βασικά components μας του VanillaJS. Μπορούμε να ενημερώνουμε αυτόματα αυτά τα components (διόρθωση σφαλμάτων, προσθήκη χαρακτηριστικών) χωρίς να δημοσιεύουμε αυτή τη βιβλιοθήκη, οπότε το ότι μπορεί να μην είναι δημοσιευμένη για κάποιο διάστημα δεν σημαίνει ότι το FastComments δεν βρίσκεται σε ενεργή ανάπτυξη! Μη διστάσετε να δείτε [το ιστολόγιό μας](https://blog.fastcomments.com/) για ενημερώσεις. Αλλαγές στο API που σπάνε τη συμβατότητα ή νέες λειτουργίες δεν θα μεταφερθούν στην υποκείμενη βασική βιβλιοθήκη χωρίς αύξηση έκδοσης σε αυτή τη βιβλιοθήκη.
