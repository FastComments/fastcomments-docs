---
| רכיב | תיאור |
| --- | --- |
| `FastComments` | ווידג'ט תגובות עם תגובות משנה, הצבעות ועוד |
| `FastCommentsCommentCount` | מציג את מספר התגובות בדף |
| `FastCommentsImageChat` | הערות על תמונות |
| `FastCommentsLiveChat` | ווידג'ט צ'אט בזמן אמת |
| `FastCommentsCollabChat` | תגובות שיתופיות בתוך הטקסט |
| `FastCommentsReviewsSummary` | סיכום ביקורות עם דירוג בכוכבים |
| `FastCommentsUserActivityFeed` | פיד פעילות משתמש |

כל הרכיבים מיוצאים משורש החבילה:

```tsx
import {
    FastComments,
    FastCommentsLiveChat,
    FastCommentsReviewsSummary,
} from 'fastcomments-nextjs';
```
