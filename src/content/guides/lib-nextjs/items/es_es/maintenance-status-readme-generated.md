Estos componentes son envoltorios alrededor de nuestros componentes core de VanillaJS. Podemos actualizar automáticamente estos componentes (corregir errores, añadir funciones) sin publicar esta biblioteca, así que, aunque no se publique por un tiempo, eso no significa que FastComments no esté en desarrollo activo. No dudes en consultar [nuestro blog](https://blog.fastcomments.com/) para actualizaciones. Los cambios incompatibles en la API o las nuevas funcionalidades nunca se incorporarán a la biblioteca core subyacente sin un aumento de versión en esta biblioteca.
