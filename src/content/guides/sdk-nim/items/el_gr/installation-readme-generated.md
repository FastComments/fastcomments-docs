### Περιεχόμενα Βιβλιοθήκης

Αυτή η βιβλιοθήκη περιέχει τον παραγόμενο πελάτη API και τα βοηθητικά εργαλεία SSO για να καταστήσει την εργασία με το API πιο εύκολη.

- [Τεκμηρίωση Βιβλιοθήκης Πελάτη API](https://github.com/FastComments/fastcomments-nim/blob/master/client/README.md)

### Δημόσια vs Ασφαλή API

Για τον πελάτη API, υπάρχουν τρία modules API, `api_default`, `api_public`, και `api_moderation`. Το `api_default` περιέχει μεθόδους που απαιτούν το API key σας, και το `api_public` περιέχει κλήσεις api που μπορούν να γίνουν απευθείας από έναν browser/κινητή συσκευή/κ.λπ. χωρίς έλεγχο ταυτότητας. Το module `api_moderation` περιέχει μεθόδους για τον πίνακα ελέγχου moderator.

Οι μέθοδοι του `api_moderation` καλύπτουν την καταγραφή, την καταμέτρηση, την αναζήτηση και την εξαγωγή σχολίων και των logs τους· ενέργειες moderation όπως αφαίρεση/επανόρθωση σχολίων, σήμανση, ρύθμιση κατάστασης για έλεγχο/spam/έγκριση, προσαρμογή ψήφων και επανανοίγω/κλείσιμο νήματος· bans (αποκλεισμός χρήστη από ένα σχόλιο, αναίρεση ban, προ-πληροφορίες ban, κατάσταση και προτιμήσεις ban, και μετρήσεις αποκλεισμένων χρηστών)· και badges & trust (απονομή/αφαίρεση badge, λίστα χειροκίνητων badge, λήψη/ρύθμιση του trust factor ενός χρήστη, και ανάκτηση του εσωτερικού προφίλ ενός χρήστη). Κάθε μέθοδος του `api_moderation` δέχεται ένα παράμετρο `sso` ώστε η κλήση να είναι αυθεντικοποιημένη ως SSO moderator.
