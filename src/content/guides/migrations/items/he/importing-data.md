בעוד שצוות התמיכה של FastComments יכול לסייע עם הגירות, רוב ההגירות ניתנות לביצוע ומעקב בקלות ללא כל התערבות מצד צוות התמיכה.

We natively support importing exports from the following providers:

- Commento
- Disqus
- Hyvor Talk
- Muut Comments
- IntenseDebate
- Just-Comments
- WordPress (via the plugin)

By navigating [כאן](https://fastcomments.com/auth/my-account/manage-data/import) we can upload the file containing the data to migrate.

[app-screenshot-start url='/auth/my-account/manage-data/import'; selector = '.account-block'; title='The Import Page Form' app-screenshot-end]

### ניטור ייבוא

FastComments משתמשת במערכת עיבוד עבודות לעיבוד ייבוא וייצוא. ברגע שהמערכת תתפוס את המשימה שלך, היא תעדכן מעת לעת את סטטוס המשימה בממשק הייבוא או הייצוא.

[app-screenshot-start url='/auth/my-account/manage-data/import?demo=true'; selector = '.content'; title='Import Job Status' app-screenshot-end]

שים לב שמצב הייבוא והייצוא ניתן לצפייה על ידי כל המנהלים בחשבון.

אם המשימה שלך נכשלת, היא לא תופעל מחדש אוטומטית. יהיה צורך לנסות את הייבוא שוב. אם ייבוא או ייצוא כלשהו נכשל, מנהלי המערכת שלנו יקבלו הודעה אוטומטית. אם נזהה בעיה, נפנה אליך כדי לראות אם נוכל לעזור.

### הרצת הייבוא מחדש

במהלך חלק מההגירות, יש צורך להפעיל את הייבוא מספר פעמים. לדוגמה, נהוג לבצע העברה ראשונית למטרות בדיקה, ואז להריץ את הייבוא שוב עם הנתונים העדכניים לפני המעבר הסופי.

ייבוא מחדש של אותו תוכן **לא ייצור כפילויות**.

### אבטחת נתונים ותפוגה

קבצי הייבוא אינם נגישים בבקשות חיצוניות בכל צורה, וקבצי הייבוא נמחקים ממערכתנו מיד עם סיום הייבוא.
