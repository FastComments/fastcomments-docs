Grâce à FastComments, plusieurs utilisateurs peuvent être ajoutés à votre tenant, et leurs permissions peuvent être configurées en fonction des rôles dans votre
organisation.

[app-screenshot-start url='/auth/my-account/users/new/edit'; selector = '.account-block'; title='Adding a User' app-screenshot-end]

Les utilisateurs peuvent être ajoutés via cette interface. Si vous avez de nombreux utilisateurs, nous pouvons les importer pour vous [via le support client](https://fastcomments.com/auth/my-account/help).

Pour ajouter des modérateurs, voir [la documentation correspondante](/guide-moderation#moderators-adding).

### Remarque concernant l'authentification

Vos utilisateurs peuvent chercher un endroit pour saisir leur mot de passe. FastComments utilise "magic links" pour la connexion - aucun mot de passe n'est donc requis.
