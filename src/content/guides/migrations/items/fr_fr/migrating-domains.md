FastComments fournit un moyen automatisé de migrer vos commentaires entre des domaines.

La migration de domaine nécessite simplement un domaine `from` et un domaine `to`.

This **déplace** les commentaires, il ne les copie pas. Si vous souhaitez copier des commentaires, contactez-nous.

[app-screenshot-start url='/auth/my-account/manage-data/migrate-domains?demo=true'; linkUrl='/auth/my-account/manage-data/migrate-domains'; selector = '.content'; title='Migrating Domains' app-screenshot-end]

Ceci est également utile, par exemple, si une partie de votre migration vers FastComments implique une migration depuis un autre fournisseur, donc vos données d'importation
de commentaires peuvent contenir des données qui doivent être migrées. Dans ce cas, vous pouvez exécuter l'importation, puis la migration de domaine.

### Suivi de l'avancement

L'outil de migration de domaine utilise le même système de traitement des tâches de FastComments que les autres outils de gestion des données.

Il peut y avoir un délai avant le démarrage de votre migration. C'est normal, car le système vérifie périodiquement la présence de nouvelles tâches à traiter.

Pendant l'exécution de la tâche, il affichera le nombre de commentaires trouvés à migrer et le nombre migré jusqu'à présent.
