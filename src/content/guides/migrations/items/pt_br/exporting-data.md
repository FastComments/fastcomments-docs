Todos os dados de comentários podem ser exportados da nossa plataforma em formato CSV.

Observe que as datas neste arquivo estarão no fuso horário UTC, no formato ISO.

Se você pretende buscar dados periodicamente do FastComments, sugerimos usar Webhooks.

#### WordPress

No caso de integrações com WordPress, o FastComments sincroniza automaticamente os dados de volta para a sua instalação do WordPress
periodicamente. Por causa disso, migrar do FastComments significa que você simplesmente precisa desativá-lo.
