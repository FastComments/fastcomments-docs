С FastComments можно добавить множество пользователей в ваш tenant, и их разрешения можно настроить на основе ролей в вашей
организации.

[app-screenshot-start url='/auth/my-account/users/new/edit'; selector = '.account-block'; title='Adding a User' app-screenshot-end]

Пользователей можно добавлять через этот интерфейс. Если у вас много пользователей, мы можем импортировать их для вас [через службу поддержки клиентов](https://fastcomments.com/auth/my-account/help).

Для добавления модераторов смотрите [соответствующую документацию](/guide-moderation#moderators-adding).

### Примечание об аутентификации

Ваши пользователи могут искать место для ввода пароля. FastComments использует "magic links" для входа - поэтому пароли не требуются.
