Ενώ το FastComments Support μπορεί να βοηθήσει με τις μεταφορές, οι περισσότερες μπορούν να εκτελεστούν και να παρακολουθηθούν εύκολα χωρίς οποιαδήποτε παρέμβαση
του προσωπικού υποστήριξης.

Υποστηρίζουμε εγγενώς την εισαγωγή αρχείων εξαγωγής από τους ακόλουθους παρόχους:

- Commento
- Disqus
- Hyvor Talk
- Muut Comments
- IntenseDebate
- Just-Comments
- WordPress (via the plugin)

By navigating [here](https://fastcomments.com/auth/my-account/manage-data/import) we can upload the file containing the data to migrate.

### Monitoring Imports

Το FastComments χρησιμοποιεί ένα σύστημα επεξεργασίας εργασιών για την επεξεργασία εισαγωγών και εξαγωγών. Μόλις το σύστημα αναλάβει την εργασία σας, θα
αναφέρει περιοδικά την κατάσταση της εργασίας στο περιβάλλον εισαγωγής ή εξαγωγής.

[app-screenshot-start url='/auth/my-account/manage-data/import?demo=true'; selector = '.content'; title='Import Job Status' app-screenshot-end]

Σημειώστε ότι οι καταστάσεις για Εισαγωγές και Εξαγωγές είναι ορατές από όλους τους διαχειριστές στον λογαριασμό.

Εάν η εργασία σας αποτύχει, δεν θα επανεκκινηθεί αυτόματα. Η εισαγωγή θα πρέπει να επιχειρηθεί ξανά. Εάν οποιαδήποτε εισαγωγή ή εξαγωγή αποτύχει,
οι διαχειριστές του συστήματός μας ειδοποιούνται αυτόματα. Εάν εντοπίσουμε κάποιο πρόβλημα, θα επικοινωνήσουμε μαζί σας για να δούμε αν μπορούμε να βοηθήσουμε.

### Re-Running The Import

Κατά τη διάρκεια κάποιων μεταφορών, είναι απαραίτητο να εκτελεστεί η εισαγωγή πολλές φορές. Για παράδειγμα, είναι συνηθισμένο να γίνει μια πρώτη
μεταφορά για δοκιμές και στη συνέχεια να εκτελεστεί ξανά η εισαγωγή με τα πιο πρόσφατα δεδομένα πριν την ενεργοποίηση.

Re-importing the same content **will not create duplicates**.

### Data Security and Expiration

Τα αρχεία εισαγωγής δεν είναι προσβάσιμα με κανένα τρόπο μέσω εξωτερικών αιτημάτων, και τα αρχεία εισαγωγής διαγράφονται από το σύστημά μας μόλις
η εισαγωγή ολοκληρωθεί.
