Όλα τα δεδομένα σχολίων μπορούν να εξαχθούν από την πλατφόρμα μας σε μορφή CSV.

Σημειώστε ότι οι ημερομηνίες σε αυτό το αρχείο θα είναι σε ζώνη ώρας UTC, σε μορφή ISO.

If you are looking to periodically pull data from FastComments, we suggest using Webhooks.

#### WordPress

Σε περίπτωση ενσωματώσεων με το WordPress, το FastComments συγχρονίζει αυτόματα τα δεδομένα πίσω στην εγκατάσταση του WordPress σας
περιοδικά. Γι' αυτό, η μετανάστευση από το FastComments σημαίνει απλώς ότι πρέπει να το απενεργοποιήσετε.
