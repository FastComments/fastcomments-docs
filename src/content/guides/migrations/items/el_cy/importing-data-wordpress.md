Το Πρόσθετό μας για WordPress (https://wordpress.org/plugins/fastcomments/) διαθέτει έναν ισχυρό μηχανισμό εισαγωγής μέσω διεπαφής χρήστη. Μετά την εγκατάσταση του προσθέτου,
θα σας καθοδηγήσει στη σύνδεση της εγκατάστασης WordPress με το FastComments και στην αντιγραφή των υπαρχόντων δεδομένων σχολίων σας.

**Αυτό γίνεται χωρίς να αντιγράφετε ή να κατεβάζετε τίποτα χειροκίνητα.**

Η διαδικασία μετανάστευσης θα σας εμφανίζεται μέσω της διεπαφής χρήστη κατά τη διάρκεια της μετανάστευσης. Οι περισσότερες μεταναστεύσεις διαρκούν μόνο λίγα λεπτά.

Ο μηχανισμός έχει σχεδιαστεί ώστε να μην επιβαρύνει υπερβολικά την εγκατάσταση WordPress σας κατά τη μετανάστευση.

### Cloudflare & Τείχη Προστασίας

Για να λειτουργήσει η αυτοματοποιημένη ρύθμιση του WordPress, πρέπει να πραγματοποιήσουμε κλήσεις στην εγκατάσταση WordPress σας.
Τείχη προστασίας όπως το Cloudflare μπορεί να μας αποκλείσουν και να προκαλέσουν αποτυχία της ενσωμάτωσης. Σε τέτοιες περιπτώσεις, [μπορούμε να σας παρέχουμε](https://fastcomments.com/auth/my-account/help) ένα σύνολο διευθύνσεων IP για να τις βάλετε στη λίστα επιτρεπόμενων για την ενσωμάτωση.

### Ιδιοκτησία Δεδομένων

Στην περίπτωση της μετανάστευσης μέσω του προσθέτου μας για το WordPress, οποιαδήποτε νέα ή ενημερωμένα δεδομένα σχολίων συγχρονίζονται αυτόματα πίσω στην εγκατάσταση WordPress σας στο παρασκήνιο. Αυτό σημαίνει ότι, ενώ τα σχόλια εξυπηρετούνται από το FastComments για να μειωθεί το φόρτο στην εγκατάσταση WordPress σας, **επίσης** τα αποθηκεύουμε στη βάση δεδομένων σας ως αντίγραφο ασφαλείας. Αυτό σημαίνει επίσης ότι, εάν επιθυμείτε να αποχωρήσετε από το FastComments, τα δεδομένα σας είναι ήδη μεταφερμένα και ενημερωμένα.
