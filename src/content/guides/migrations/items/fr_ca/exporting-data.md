Toutes les données de commentaires peuvent être exportées depuis notre plateforme au format CSV.

Notez que les dates dans ce fichier seront en fuseau horaire UTC, au format ISO.

Si vous souhaitez récupérer périodiquement des données de FastComments, nous vous suggérons d'utiliser les Webhooks.

#### WordPress

Dans le cas des intégrations WordPress, FastComments synchronise automatiquement les données vers votre installation WordPress
périodiquement. En conséquence, pour migrer depuis FastComments, il vous suffit de le désactiver.
