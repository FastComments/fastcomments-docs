Avec FastComments, plusieurs utilisateurs peuvent être ajoutés à votre locataire, et leurs autorisations peuvent être configurées en fonction des rôles au sein de votre organisation.

[app-screenshot-start url='/auth/my-account/users/new/edit'; selector = '.account-block'; title='Ajouter un utilisateur' app-screenshot-end]

Les utilisateurs peuvent être ajoutés via cette interface. Si vous avez plusieurs utilisateurs, nous pouvons les importer pour vous [via le support client](https://fastcomments.com/auth/my-account/help).

Pour ajouter des modérateurs, consultez [la documentation correspondante](/guide-moderation#moderators-adding).

### Remarque sur l'authentification

Vos utilisateurs peuvent chercher un endroit pour entrer leur mot de passe. FastComments utilise des « liens magiques » pour la connexion - donc aucun mot de passe n'est requis.
