Ενώ η Υποστήριξη του FastComments μπορεί να βοηθήσει με τις μεταφορές, οι περισσότερες μπορούν να εκτελεστούν και να παρακολουθηθούν εύκολα χωρίς παρέμβαση του προσωπικού υποστήριξης.

Υποστηρίζουμε εγγενώς την εισαγωγή εξαγωγών από τους ακόλουθους παρόχους:

- Commento
- Disqus
- Hyvor Talk
- Muut Comments
- IntenseDebate
- Just-Comments
- WordPress (via the plugin)

Πλοηγούμενοι [εδώ](https://fastcomments.com/auth/my-account/manage-data/import) μπορούμε να ανεβάσουμε το αρχείο που περιέχει τα δεδομένα προς μετανάστευση.

### Παρακολούθηση Εισαγωγών

Το FastComments χρησιμοποιεί ένα σύστημα επεξεργασίας εργασιών για την επεξεργασία εισαγωγών και εξαγωγών. Μόλις το σύστημα αναλάβει την εργασία σας, θα αναφέρει περιοδικά την κατάσταση της εργασίας στη διεπαφή εισαγωγής ή εξαγωγής.

[app-screenshot-start url='/auth/my-account/manage-data/import?demo=true'; selector = '.content'; title='Import Job Status' app-screenshot-end]

Σημειώστε ότι η κατάσταση των Εισαγωγών και Εξαγωγών μπορεί να προβληθεί από όλους τους διαχειριστές στον λογαριασμό.

Εάν η εργασία σας αποτύχει, δεν θα επανεκκινηθεί αυτόματα. Η εισαγωγή θα πρέπει να επιχειρηθεί ξανά. Εάν οποιαδήποτε εισαγωγή ή εξαγωγή αποτύχει, οι διαχειριστές του συστήματός μας ειδοποιούνται αυτόματα. Εάν εντοπίσουμε κάποιο πρόβλημα, θα επικοινωνήσουμε μαζί σας για να δούμε εάν μπορούμε να βοηθήσουμε.

### Επανεκτέλεση της Εισαγωγής

Κατά τη διάρκεια ορισμένων μεταφορών, είναι απαραίτητο να εκτελεστεί η εισαγωγή πολλές φορές. Για παράδειγμα, είναι συνηθισμένο να γίνεται μια πρώτη μεταφορά για δοκιμές και στη συνέχεια να εκτελείται ξανά η εισαγωγή με τα πιο πρόσφατα δεδομένα πριν την ενεργοποίηση.

Η επανεισαγωγή του ίδιου περιεχομένου **δεν θα δημιουργήσει διπλότυπα**.

### Ασφάλεια Δεδομένων και Λήξη

Τα αρχεία εισαγωγής δεν είναι προσβάσιμα μέσω εξωτερικών αιτήσεων με κανέναν τρόπο, και διαγράφονται από το σύστημά μας μόλις ολοκληρωθεί η εισαγωγή.
