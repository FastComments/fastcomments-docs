Το FastComments παρέχει έναν αυτοματοποιημένο τρόπο για να μεταφέρετε τα σχόλιά σας μεταξύ τομέων.

The domain migration simply requires a `from` and a `to` domain.

Αυτό **μετακινεί** τα σχόλια, δεν τα αντιγράφει. Αν επιθυμείτε να αντιγράψετε σχόλια, επικοινωνήστε μαζί μας.

[app-screenshot-start url='/auth/my-account/manage-data/migrate-domains?demo=true'; linkUrl='/auth/my-account/manage-data/migrate-domains'; selector = '.content'; title='Migrating Domains' app-screenshot-end]

Αυτό είναι επίσης χρήσιμο, για παράδειγμα, εάν μέρος της μετανάστευσής σας στο FastComments περιλαμβάνει μετανάστευση από άλλον πάροχο, οπότε τα δεδομένα εισαγωγής σχολίων σας
ενδέχεται να περιέχουν δεδομένα που χρειάζεται να μεταφερθούν. Σε αυτή την περίπτωση, μπορείτε να εκτελέσετε πρώτα την εισαγωγή και μετά τη μετανάστευση domain.

### Παρακολούθηση προόδου

Το εργαλείο Domain Migration χρησιμοποιεί το ίδιο σύστημα επεξεργασίας εργασιών του FastComments όπως και τα άλλα εργαλεία διαχείρισης δεδομένων.

Μπορεί να υπάρξει καθυστέρηση πριν ξεκινήσει η μετανάστευσή σας. Αυτό είναι φυσιολογικό, καθώς το σύστημα ελέγχει περιοδικά για νέες εργασίες προς επεξεργασία.

Καθώς η εργασία εκτελείται, θα εμφανίζει τον αριθμό των σχολίων που βρέθηκαν για μεταφορά και τον αριθμό που έχει μεταφερθεί μέχρι στιγμής.
