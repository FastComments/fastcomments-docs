Το [πρόσθετο WordPress](https://wordpress.org/plugins/fastcomments/) μας διαθέτει έναν ισχυρό μηχανισμό εισαγωγής βασισμένο στο UI. Μετά την εγκατάσταση του προσθέτου,
θα σας καθοδηγήσει στη σύνδεση της εγκατάστασής σας WordPress με το FastComments και στην αντιγραφή των υπαρχόντων δεδομένων σχολίων σας.

**Αυτό γίνεται χωρίς να χρειάζεται να αντιγράψετε ή να κατεβάσετε τίποτα χειροκίνητα.**

Η διαδικασία μετανάστευσης θα σας υποδειχθεί μέσω του UI κατά τη διάρκεια της μετανάστευσης. Οι περισσότερες μεταναστεύσεις διαρκούν μόνο λίγα λεπτά.

Ο μηχανισμός έχει σχεδιαστεί ώστε να μην επιβαρύνει υπερβολικά την εγκατάστασή σας WordPress κατά τη διάρκεια της μετανάστευσης.

### CloudFlare & Τείχη προστασίας

Για να λειτουργήσει η αυτοματοποιημένη ρύθμιση του WordPress, πρέπει να πραγματοποιήσουμε κλήσεις προς την εγκατάστασή σας WordPress.
Τα τείχη προστασίας όπως το Cloudflare μπορεί να μας αποκλείσουν και να προκαλέσουν αποτυχία της ενσωμάτωσης. Σε τέτοιες περιπτώσεις, [μπορούμε να σας παρέχουμε](https://fastcomments.com/auth/my-account/help) μια λίστα με διευθύνσεις IP για να τις προσθέσετε στη λίστα επιτρεπόμενων για την ενσωμάτωση.

### Ιδιοκτησία Δεδομένων

Στην περίπτωση της μετανάστευσής μας από WordPress, οποιαδήποτε νέα ή ενημερωμένα δεδομένα σχολίων συγχρονίζονται αυτόματα ξανά στην εγκατάστασή σας WordPress
στο παρασκήνιο. Αυτό σημαίνει ότι, ενώ τα σχόλια παρέχονται από το ίδιο το FastComments για να μειώσει το φόρτο στην ανάπτυξη WordPress σας,
εμείς **επίσης** τα αποθηκεύουμε στη βάση δεδομένων σας ως αντίγραφο ασφαλείας. Αυτό σημαίνει επίσης ότι, εάν επιθυμείτε να απομακρυνθείτε από το FastComments, τα δεδομένα σας είναι
ήδη μεταφερμένα και ενημερωμένα.
