Tutti i dati dei commenti possono essere esportati dalla nostra piattaforma in formato CSV.

Si noti che le date in questo file saranno nel fuso orario UTC, in formato ISO.

Se desideri prelevare periodicamente i dati da FastComments, suggeriamo di utilizzare i Webhooks.

#### WordPress

Nel caso di integrazioni con WordPress, FastComments sincronizza automaticamente i dati con la tua installazione di WordPress
periodicamente. Per questo motivo, migrare da FastComments significa che devi semplicemente disattivarlo.
