Alle Kommentardaten können von unserer Plattform im CSV-Format exportiert werden.

Beachten Sie, dass Datumsangaben in dieser Datei in der UTC-Zeitzone im ISO-Format vorliegen.

Wenn Sie Daten periodisch von FastComments abrufen möchten, empfehlen wir die Verwendung von Webhooks.

#### WordPress

Im Fall von WordPress-Integrationen synchronisiert FastComments die Daten automatisch zurück zu Ihrer WordPress-Installation
periodisch. Deshalb bedeutet eine Migration von FastComments, dass Sie es einfach deaktivieren müssen.
