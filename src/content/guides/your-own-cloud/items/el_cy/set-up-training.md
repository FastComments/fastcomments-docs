Παρέχουμε εκπαίδευση για προγραμματιστές και συντονιστές που αναμένετε να χρησιμοποιήσουν το σύστημα.

Αυτό γίνεται στο πλαίσιο δύο συνεδριών διάρκειας δύο ωρών η κάθε μία, οι οποίες μπορούν να προγραμματιστούν χωριστά. Η ιδέα
είναι ότι μετά την πρώτη συνεδρία θα έχετε χρόνο να διαμορφώσετε ερωτήσεις και θέματα που μπορεί
να θέλετε να εστιάσουμε στη δεύτερη συνεδρία.

Η εκπαίδευση για τους προγραμματιστές και για τους διαχειριστές/συντονιστές είναι ξεχωριστή, και μπορεί να αγοραστεί ξεχωριστά ή μαζί.
