Tendrá acceso completo al sistema de inquilinos con marca blanca que FastComments proporciona.

Puede gestionar sus inquilinos usando las mismas [instrucciones aquí](https://docs.fastcomments.com/guide-white-labeling.html#white-labeling-using-the-dashboard).

Esto también incluye una API para automatizar la gestión de los inquilinos y sus usuarios, incluidas las capacidades de SSO.
