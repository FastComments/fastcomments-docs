Laten we nu onze widget-code toevoegen.

Kopieer de code hieronder. Je wilt ervoor zorgen dat je bent aangemeld bij [fastcomments.com](https://fastcomments.com) 
en laad deze pagina opnieuw als dat niet het geval is, zodat de code vooraf wordt ingevuld met je accountgegevens, anders wordt de democode weergegeven.

Kopieer nu de code:

[inline-code-attrs-start title = 'Zyro Reacties Code'; type = 'html'; isFunctional = false; inline-code-attrs-end]
[inline-code-start]
<script src="https://cdn.fastcomments.com/js/embed-v2.min.js"></script>
<div id="fastcomments-widget"></div>
<script>
    FastCommentsUI(document.getElementById('fastcomments-widget'), {
        tenantId: "demo",
        pageTitle: window.parent.document.title,
        urlId: window.parent.location.href,
        url: window.parent.location.href
    });
</script>
[inline-code-end]

Ga nu terug naar onze sitebouwer en klik op `Enter code`:

### Let op!

Het is belangrijk dat je de bovenstaande code gebruikt en niet de codefragmenten uit andere documentatie, aangezien dit fragment speciaal voor Zyro is gemaakt.

Je zou nu iets moeten hebben dat eruitziet zoals het volgende en dat leeg lijkt. Dit is normaal. Beweeg je muis over het gebied
waar de widget zou moeten komen:
