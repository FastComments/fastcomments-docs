Laten we nu onze widgetcode toevoegen.

Kopieer de onderstaande code. Zorg dat je bent ingelogd op [fastcomments.com](https://fastcomments.com) en herlaad deze pagina indien je dat nog niet bent, zodat de code vooraf wordt ingevuld met je accountgegevens; anders wordt de democode weergegeven.

Kopieer nu de code:

[inline-code-attrs-start title = 'Zyro Reacties Code'; type = 'html'; isFunctional = false; inline-code-attrs-end]
[inline-code-start]
<script async src="https://cdn.fastcomments.com/js/embed-v2-async.min.js"></script>
<div id="fastcomments-widget"></div>
<script>
    window.fcConfigs = [{
        target: '#fastcomments-widget',
        tenantId: "demo",
        pageTitle: window.parent.document.title,
        urlId: window.parent.location.href,
        url: window.parent.location.href
    }];
</script>
[inline-code-end]

Ga nu terug naar onze sitebouwer en klik `Enter code`:

### Let op!

Het is belangrijk dat je de bovenstaande code gebruikt en niet de codefragmenten uit andere documentatie, omdat dit fragment specifiek voor Zyro is gemaakt.
