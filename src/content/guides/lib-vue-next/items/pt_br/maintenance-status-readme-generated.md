Estes componentes são wrappers em torno de nossos componentes principais VanillaJS. Podemos atualizar automaticamente esses componentes (corrigir bugs, adicionar recursos) sem publicar esta biblioteca, então, embora ela possa não ser publicada por um tempo, isso não significa que o FastComments não esteja em desenvolvimento ativo! Sinta-se à vontade para conferir [nosso blog](https://blog.fastcomments.com/) para atualizações. Alterações que quebram a API ou novos recursos nunca serão enviados para a biblioteca subjacente sem um aumento de versão nesta biblioteca.
