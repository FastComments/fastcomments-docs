Da biste videli svaki widget i flow koji rade lokalno protiv javnog `demo` tenanta, klonirajte repozitorijum i pokrenite:

```bash
npm install
npm run dev
```

Ulazna tačka prikaza nalazi se u `src/App.vue` i `index.html` — svaki widget/flow se renderuje u istoj aplikaciji koju možete kopirati u sopstveni Vue 3 projekat.
