Да бисте видели све виџете и токове који се покрећу локално против јавног `demo` тенанта, клонирајте репозиторијум и покрените:

```bash
npm install
npm run dev
```

Улазна тачка приказа се налази у `src/App.vue` и `index.html` — сваки виџет/ток се рендерује у истој апликацији коју можете копирати у свој Vue 3 пројекат.
