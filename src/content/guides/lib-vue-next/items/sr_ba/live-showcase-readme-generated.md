Da biste vidjeli svaki widget i flow koji se pokreću lokalno protiv javnog `demo` tenanta, klonirajte repozitorij i pokrenite:

```bash
npm install
npm run dev
```

Ulazna tačka prikaza se nalazi u `src/App.vue` i `index.html` — svaki widget/flow se renderuje u istoj aplikaciji koju možete kopirati u svoj Vue 3 projekat.
