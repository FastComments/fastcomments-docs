Estos componentes son envoltorios alrededor de nuestros componentes core de VanillaJS. Podemos actualizar automáticamente estos componentes (corregir errores, añadir funciones) sin publicar esta librería, así que aunque puede que no se publique durante un tiempo eso no significa que FastComments no esté en desarrollo activo. Siéntete libre de consultar [nuestro blog](https://blog.fastcomments.com/) para actualizaciones. Los cambios incompatibles en la API o nuevas funcionalidades nunca se enviarán a la librería core subyacente sin un incremento de versión en esta librería.
