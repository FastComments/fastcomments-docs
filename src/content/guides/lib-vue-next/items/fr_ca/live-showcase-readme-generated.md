Pour voir tous les widgets et flux s'exécuter localement contre le locataire public `demo`, clonez le dépôt et exécutez :

```bash
npm install
npm run dev
```

Le point d'entrée de la démonstration se trouve dans `src/App.vue` et `index.html` — chaque widget/flux est rendu dans la même application que vous pouvez copier dans votre propre projet Vue 3.
