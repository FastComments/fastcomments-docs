Για να δείτε κάθε widget και ροή να τρέχει τοπικά ενάντια στο δημόσιο tenant `demo`, κλωνοποιήστε το αποθετήριο και εκτελέστε:

```bash
npm install
npm run dev
```

Το σημείο εισόδου της παρουσίασης βρίσκεται σε `src/App.vue` και `index.html` — κάθε widget/ροή αποδίδεται στην ίδια εφαρμογή την οποία μπορείτε να αντιγράψετε στο δικό σας έργο Vue 3.
