Herkese açık `demo` kiracına karşı yerel olarak çalışan tüm widget'ları ve akışları görmek için depoyu klonlayın ve çalıştırın:

```bash
npm install
npm run dev
```

Örnek uygulamanın giriş noktası `src/App.vue` ve `index.html` içinde yer alır — her bir widget/akış aynı uygulamada render edilir; bunu kendi Vue 3 projenize kopyalayabilirsiniz.
