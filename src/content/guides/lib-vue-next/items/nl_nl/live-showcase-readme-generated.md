Om elke widget en flow lokaal te zien draaien tegen de publieke `demo` tenant, kloon de repo en voer:

```bash
npm install
npm run dev
```

Het toegangspunt van de showcase bevindt zich in `src/App.vue` en `index.html` — elke widget/flow wordt gerenderd in dezelfde app die je naar je eigen Vue 3-project kunt kopiëren.
