Aby zobaczyć wszystkie widżety i przepływy działające lokalnie w ramach publicznego tenanta `demo`, sklonuj repozytorium i uruchom:

```bash
npm install
npm run dev
```

Punkt wejścia pokazu znajduje się w `src/App.vue` i `index.html` — każdy widżet/przepływ jest renderowany w tej samej aplikacji, którą możesz skopiować do własnego projektu Vue 3.
