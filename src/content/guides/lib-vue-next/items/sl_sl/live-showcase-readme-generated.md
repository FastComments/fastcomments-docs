Če želite videti vse gradnike in tokove, ki tečejo lokalno proti javnemu najemniku `demo`, klonirajte repozitorij in zaženite:

```bash
npm install
npm run dev
```

Vstopna točka predstavitve se nahaja v `src/App.vue` in `index.html` — vsak gradnik/tok se upodablja v isti aplikaciji, ki jo lahko kopirate v svoj Vue 3 projekt.
