Da biste vidjeli sve widgete i tokove koji se pokreću lokalno protiv javnog `demo` tenanta, klonirajte repozitorij i pokrenite:

```bash
npm install
npm run dev
```

Ulazna točka prikaza nalazi se u `src/App.vue` i `index.html` — svaki widget/tok se renderira u istoj aplikaciji koju možete kopirati u vlastiti Vue 3 projekt.
