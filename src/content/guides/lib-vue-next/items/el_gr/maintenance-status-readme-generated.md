Αυτά τα components είναι wrappers γύρω από τα core VanillaJS components. Μπορούμε να ενημερώνουμε αυτόματα αυτά τα components (επιδιόρθωση σφαλμάτων, προσθήκη δυνατοτήτων) χωρίς να δημοσιεύσουμε αυτή τη βιβλιοθήκη, οπότε αν και μπορεί να μην δημοσιευτεί για κάποιο διάστημα, αυτό δεν σημαίνει ότι το FastComments δεν βρίσκεται υπό ενεργή ανάπτυξη! Μη διστάσετε να ελέγξετε [το ιστολόγιό μας](https://blog.fastcomments.com/) για ενημερώσεις. Σημαντικές αλλαγές που σπάνε το API ή νέες δυνατότητες δεν θα σταλούν ποτέ στην υποκείμενη core βιβλιοθήκη χωρίς αύξηση έκδοσης σε αυτή τη βιβλιοθήκη.
