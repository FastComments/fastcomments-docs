Για να δείτε κάθε widget και flow να τρέχει τοπικά απέναντι στον δημόσιο `demo` tenant, κλωνοποιήστε το αποθετήριο και τρέξτε:

```bash
npm install
npm run dev
```

Το σημείο εισόδου της επίδειξης βρίσκεται στο `src/App.vue` και `index.html` — κάθε widget/flow αποδίδεται στην ίδια εφαρμογή που μπορείτε να αντιγράψετε στο δικό σας έργο Vue 3.
