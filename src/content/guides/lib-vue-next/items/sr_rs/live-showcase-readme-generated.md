Да бисте видели све видгете и флоу-ове који се локално покрећу против јавног `demo` tenant-а, клонирајте репозиторијум и покрените:

```bash
npm install
npm run dev
```

Улазна тачка демонстрације налази се у `src/App.vue` и `index.html` — сваки видгет/флоу се рендерује у истој апликацији коју можете копирати у свој Vue 3 пројекат.
