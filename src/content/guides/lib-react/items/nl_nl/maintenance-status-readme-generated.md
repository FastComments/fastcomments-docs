Deze componenten zijn wrappers rond onze core VanillaJS components. We kunnen deze componenten automatisch bijwerken (bugs oplossen, functies toevoegen) zonder deze bibliotheek te publiceren, dus hoewel het even kan duren voordat deze gepubliceerd wordt, betekent dat niet dat FastComments niet actief in ontwikkeling is! Bekijk gerust [onze blog](https://blog.fastcomments.com/) voor updates. Brekende API-wijzigingen of functies zullen nooit naar de onderliggende core-bibliotheek worden uitgebracht zonder een versieverhoging in deze bibliotheek.
