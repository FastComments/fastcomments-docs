---
Om elke widget en flow lokaal te zien draaien tegen de openbare `demo` tenant, kloon de repo en voer het volgende uit:

```bash
cd examples/example-showcase
npm install
npm start
```

Elke widget/flow heeft zijn eigen view onder `examples/example-showcase/src/views/` die je rechtstreeks in je eigen React-app kunt kopiëren.
