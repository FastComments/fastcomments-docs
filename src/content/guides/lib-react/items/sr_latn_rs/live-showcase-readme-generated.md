Da biste videli sve widgete i tokove koji se izvršavaju lokalno protiv javnog `demo` tenanta, klonirajte repozitorijum i pokrenite:

```bash
cd examples/example-showcase
npm install
npm start
```

Svaki widget/tok ima sopstveni prikaz u `examples/example-showcase/src/views/` koji možete direktno kopirati u svoju React aplikaciju.
