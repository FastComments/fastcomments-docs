Estos componentes son envoltorios alrededor de nuestros componentes VanillaJS principales. Podemos actualizar automáticamente estos componentes (corregir errores, añadir funcionalidades) sin publicar esta biblioteca, así que aunque puede que no se publique durante un tiempo eso no significa que FastComments no esté en desarrollo activo! No dudes en consultar [nuestro blog](https://blog.fastcomments.com/) para actualizaciones. Los cambios incompatibles en la API o las nuevas funcionalidades nunca se enviarán a la biblioteca subyacente principal sin un aumento de versión en esta biblioteca.
