Da biste vidjeli sve widgete i tokove koji se pokreću lokalno protiv javnog `demo` tenanta, klonirajte repozitorij i pokrenite:

```bash
cd examples/example-showcase
npm install
npm start
```

Svaki widget/tok ima svoj prikaz u `examples/example-showcase/src/views/` koji možete izravno kopirati u svoju React aplikaciju.
