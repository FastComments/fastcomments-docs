Te komponenty są nakładkami na nasze podstawowe komponenty VanillaJS. Możemy automatycznie aktualizować te komponenty (naprawiać błędy, dodawać funkcje) bez publikowania tej biblioteki, więc choć może ona nie być opublikowana przez jakiś czas, nie oznacza to, że FastComments nie jest aktywnie rozwijany! Zachęcamy do odwiedzenia [naszego bloga](https://blog.fastcomments.com/) w celu uzyskania informacji o aktualizacjach. Zmiany łamiące API lub nowe funkcje nigdy nie zostaną wprowadzone do podstawowej biblioteki bez podniesienia wersji w tej bibliotece.
