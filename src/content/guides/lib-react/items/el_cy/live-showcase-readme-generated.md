Για να δείτε κάθε widget και flow να τρέχει τοπικά χρησιμοποιώντας τον δημόσιο tenant `demo`, κλωνοποιήστε το αποθετήριο και εκτελέστε:

```bash
cd examples/example-showcase
npm install
npm start
```

Κάθε widget/flow έχει τη δική του προβολή στο `examples/example-showcase/src/views/` την οποία μπορείτε να αντιγράψετε απευθείας στην δική σας εφαρμογή React.
