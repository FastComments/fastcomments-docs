---
Um alle Widgets und Flows lokal gegen den öffentlichen `demo`‑Mandanten zu sehen, klonen Sie das Repo und führen Sie aus:

```bash
cd examples/example-showcase
npm install
npm run dev
```

Jedes Widget/Flow hat seine eigene Ansicht unter `examples/example-showcase/src/views/`, die Sie direkt in Ihre eigene React‑App kopieren können.
