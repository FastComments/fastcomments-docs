Για να δείτε κάθε widget και flow να εκτελούνται τοπικά ενάντια στον δημόσιο tenant `demo`, κλωνοποιήστε το αποθετήριο και εκτελέστε:

```bash
cd examples/example-showcase
npm install
npm start
```

Κάθε widget/flow έχει τη δική του προβολή κάτω από `examples/example-showcase/src/views/` την οποία μπορείτε να αντιγράψετε απευθείας στην δική σας React εφαρμογή.
