Щоб побачити кожен віджет і потік, що працює локально проти публічного `demo` орендаря, клонувати репозиторій і запустити:

```bash
cd examples/example-showcase
npm install
npm run dev
```

Кожен віджет/потік має власний вигляд у `examples/example-showcase/src/views/`, який ви можете скопіювати безпосередньо у свою React‑аплікацію.
