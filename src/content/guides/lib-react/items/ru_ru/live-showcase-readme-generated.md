Чтобы увидеть каждый виджет и поток, работающие локально против публичного арендатора `demo`, клонируйте репозиторий и запустите:

```bash
cd examples/example-showcase
npm install
npm run dev
```

Каждый виджет/поток имеет собственный представление в `examples/example-showcase/src/views/`, которое вы можете скопировать напрямую в своё React‑приложение.
