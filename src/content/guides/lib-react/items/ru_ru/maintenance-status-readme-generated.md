Эти компоненты являются оболочками поверх наших основных компонентов VanillaJS. Мы можем автоматически обновлять эти компоненты (исправлять ошибки, добавлять функции) без публикации этой библиотеки, так что хотя она может некоторое время не публиковаться, это не означает, что FastComments не находится в активной разработке! Не стесняйтесь заглядывать в [наш блог](https://blog.fastcomments.com/) за обновлениями. Несовместимые изменения API или новые функции никогда не будут добавлены в базовую библиотеку без повышения версии этой библиотеки.
