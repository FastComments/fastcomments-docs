Para agregar FastComments a sitios WooCommerce, consulte [nuestra documentación del plugin de WordPress](/guide-installation-wordpress.html).

Este plugin agregará automáticamente comentarios a las páginas de productos.

El plugin de WordPress ha sido probado para funcionar junto con el plugin de WooCommerce. Si tiene problemas, por favor [háganoslo saber](https://fastcomments.com/auth/my-account/tickets).
