Για να προσθέσετε το FastComments σε ιστοσελίδες WooCommerce, ανατρέξτε στην [τεκμηρίωση του WordPress plugin μας](/guide-installation-wordpress.html).

Αυτό το plugin θα προσθέσει αυτόματα σχολιασμό στις σελίδες προϊόντων.

Το WordPress plugin έχει δοκιμαστεί να λειτουργεί μαζί με το WooCommerce plugin. Αν έχετε προβλήματα, παρακαλούμε [ενημερώστε μας](https://fastcomments.com/auth/my-account/tickets).
