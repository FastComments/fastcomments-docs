Pour ajouter FastComments aux sites WooCommerce, consultez [notre documentation du plugin WordPress](/guide-installation-wordpress.html).

Ce plugin ajoutera automatiquement les commentaires aux pages de produits.

Le plugin WordPress a été testé pour fonctionner avec le plugin WooCommerce. Si vous avez des problèmes, veuillez [nous le faire savoir](https://fastcomments.com/auth/my-account/tickets).
