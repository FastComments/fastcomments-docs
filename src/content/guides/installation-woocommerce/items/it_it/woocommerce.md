Per aggiungere FastComments ai siti WooCommerce, consulta [la nostra documentazione del plugin WordPress](/guide-installation-wordpress.html).

Questo plugin aggiungerà automaticamente i commenti alle pagine dei prodotti.

Il plugin WordPress è stato testato per funzionare insieme al plugin WooCommerce. Se hai problemi, per favore [facci sapere](https://fastcomments.com/auth/my-account/tickets).
