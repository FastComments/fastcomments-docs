Vous pouvez obtenir une erreur d'autorisation, comme la suivante :

<div class="screenshot white-bg">
    <div class="title">Configuration de domaine manquante</div>
    <img class="screenshot-image" src="/images/installation-guides/installation-guide-webflow-step-5.png" alt="Configuration de domaine manquante" />
</div>

Cela est dû au fait que FastComments ne sait pas qu'il doit autoriser l'utilisation de votre compte sur ce domaine.

Dans ce cas, la solution est simple. Vous devez simplement ajouter votre site à votre compte FastComments.com.

[Rendez-vous ici pour ajouter votre site à votre compte.](https://fastcomments.com/auth/my-account/configure-domains)
