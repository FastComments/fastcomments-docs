FastComments jest testowany w przeglądarkach, które stanowią przytłaczającą większość udziału w rynku.

This includes:

- Microsoft Edge
- FireFox
- Chrome
- Safari
- iOS Safari
- Chrome for Android
- Samsung Internet

Ze względów bezpieczeństwa i aby zapewnić jak najlepsze doświadczenie dla większości naszych klientów, nie
wspieramy przeglądarek, które nie otrzymują już aktualizacji zabezpieczeń lub mają bardzo niewielki ruch.
