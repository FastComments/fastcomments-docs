FastComments έχει δοκιμαστεί με τα προγράμματα περιήγησης που αποτελούν την συντριπτική πλειονότητα του μεριδίου αγοράς των περιηγητών.

Αυτό περιλαμβάνει:

- Microsoft Edge
- FireFox
- Chrome
- Safari
- iOS Safari
- Chrome for Android
- Samsung Internet

Για λόγους ασφάλειας και για να παρέχουμε την καλύτερη εμπειρία στην πλειονότητα των πελατών μας, δεν υποστηρίζουμε περιηγητές που δεν λαμβάνουν πλέον ενημερώσεις ασφαλείας ή έχουν πολύ μικρή επισκεψιμότητα.
