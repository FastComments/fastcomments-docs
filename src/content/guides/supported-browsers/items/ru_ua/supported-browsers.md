FastComments тестируется в браузерах, которые составляют подавляющее большинство доли рынка браузеров.

This includes:

- Microsoft Edge
- FireFox
- Chrome
- Safari
- iOS Safari
- Chrome for Android
- Samsung Internet

В целях безопасности и для обеспечения наилучшего опыта для большинства наших клиентов, мы не
поддерживаем браузеры, которые больше не получают обновления безопасности или имеют очень небольшой объём трафика.
