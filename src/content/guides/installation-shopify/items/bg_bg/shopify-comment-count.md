Блокът **FastComments - Comment Count** показва малък брой коментари за една страница. Използвайте го в списъци с публикации в блога, в карти на продукти или във всеки шаблон, който връзва към страница с коментари, така че посетителите да могат да видят колко активна е всяка нишка преди да кликнат.

### Add the block

1. Отворете редактора на темата в Shopify.
2. Отворете шаблона, където искате броячът да се показва. Например, шаблона **Blog** (списък с публикации) или секция за листинг на продукти.
3. Кликнете **Add block** в секцията, която рендерира всеки елемент.
4. Под **Apps** изберете **FastComments - Comment Count**.
5. Кликнете **Save**.

### Settings

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Замества кой FastComments tenant да чете броячът. Оставете празно, за да използвате автоматично конфигурирания tenant на магазина. | (blank) |
| Custom URL ID | Замества идентификатора на страницата, който броячът търси. Използвайте това, когато броячът е на различна страница от блока FastComments, който следи. | (auto-detected) |

### How the count matches the comment thread

Блокът Comment Count използва същата логика за автоматично откриване като блока **FastComments**:

- Шаблон за публикация в блога: `shopify-article-{article.id}`
- Шаблон за продукт: `shopify-product-{product.id}`
- Други шаблони: пътят на заявката

Ако зададете **Custom URL ID** на блока **FastComments** на дадена страница, задайте същия Custom URL ID и на блока Comment Count, така че те да сочат към същата нишка.

### Tips

- Броят за всеки елемент на страницата се извлича с една заявка, така че добавянето на блока към всеки елемент в дълъг списък не води до допълнителни заявки.
- Очакваното използване е един блок Comment Count за статия или продукт в списък; блокът може да бъде добавян толкова пъти, колкото ви е необходимо.
