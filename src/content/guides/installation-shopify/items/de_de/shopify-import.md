Wenn Sie bereits Kommentare zu Ihren Shopify-Blogbeiträgen haben und diese behalten möchten, importieren Sie sie in FastComments, bevor Sie die native Kommentarfunktion von Shopify deaktivieren.

Der Import muss ausgeführt werden, solange die nativen Shopify-Kommentare noch aktiv und sichtbar sind. Sobald Sie diese deaktivieren, kann FastComments sie nicht mehr lesen.

### So starten Sie einen Import

1. Stellen Sie sicher, dass die FastComments-App in Ihrem Shop installiert ist.
2. Lassen Sie die nativen Shopify-Kommentare vorerst **aktiviert**.
3. [Öffnen Sie ein Support-Ticket](https://fastcomments.com/auth/my-account/tickets) und teilen Sie uns mit, welche Blogs importiert werden sollen.

Der Import dauert weniger als einen Tag.

Sobald Ihre vorhandenen Kommentare in FastComments sind, fügen Sie den **FastComments**-Block in Ihre Blog-Beitragsvorlage ein und entfernen oder deaktivieren Sie die nativen Shopify-Kommentare. Siehe [Kommentare zu Blogbeiträgen und Produkten hinzufügen](/guide-installation-shopify.html#shopify-add-comments).

Wenn Sie die vorhandenen Kommentare nicht behalten müssen, können Sie diese Seite überspringen.
