Der **FastComments - Recent Discussions**-Block listet die Seiten in Ihrem Shop mit der zuletzt erfolgten Kommentaraktivität, jeweils ein Eintrag pro Seite. Verwenden Sie ihn, wenn Sie möchten, dass Besucher aktive Threads statt einzelner Kommentare entdecken.

### Block hinzufügen

1. Öffnen Sie den Shopify-Theme-Editor.
2. Öffnen Sie die Vorlage, in der Sie den Feed platzieren möchten, z. B. eine Seitenleiste, die Fußzeile, die Blog-Übersicht oder einen Abschnitt auf der Startseite.
3. Klicken Sie in dem Abschnitt, der ihn aufnehmen soll, auf **Block hinzufügen**.
4. Unter **Apps** wählen Sie **FastComments - Recent Discussions**.
5. Klicken Sie auf **Speichern**.

### Einstellungen

| Einstellung | Was es bewirkt | Standard |
|---|---|---|
| Tenant ID (optional) | Überschreibt, von welchem FastComments-Tenant der Feed liest. Leer lassen, um den automatisch konfigurierten Tenant des Shops zu verwenden. | (leer) |
| Number of comments | Wie viele kürzlich aktive Diskussionen angezeigt werden sollen. Bereich 1 bis 50. | 20 |

### Recent Discussions vs Recent Comments

Beide Blöcke nutzen dieselben Daten, zeigen jedoch unterschiedliche Ansichten:

- **Recent Comments**: a flat list of the latest individual comments. Best for "what was just said".
- **Recent Discussions**: one entry per page, ordered by most recent activity. Best for "which threads are alive right now".

Sie können beide im selben Shop verwenden. Zum Beispiel Recent Comments in der Blog-Seitenleiste und Recent Discussions auf der Startseite.
