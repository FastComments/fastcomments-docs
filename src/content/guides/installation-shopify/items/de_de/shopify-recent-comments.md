Der **FastComments - Recent Comments** Block zeigt die neuesten Kommentare aus Ihrem gesamten Shop. Verwenden Sie ihn, um aktuelle Aktivität in einer Seitenleiste, Fußzeile oder einem Bereich auf der Startseite anzuzeigen.

### Block hinzufügen

1. Öffnen Sie den Shopify-Theme-Editor.
2. Öffnen Sie die Vorlage, in der Sie den Feed haben möchten. Eine Seitenleisten-Sektion, die Startseite oder der Blog-Index eignen sich gut.
3. Klicken Sie in der Sektion, die ihn enthalten soll, auf **Block hinzufügen**.
4. Unter **Apps** wählen Sie **FastComments - Recent Comments** aus.
5. Klicken Sie auf **Speichern**.

### Einstellungen

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Überschreiben, von welchem FastComments-Tenant der Feed gelesen wird. Leer lassen, um den automatisch für den Shop konfigurierten Tenant zu verwenden. | (leer) |
| Number of comments | Wie viele aktuelle Kommentare angezeigt werden sollen. Bereich 1 bis 50. | 5 |

### Tipps

- Jeder Kommentar verlinkt auf die Seite, auf der er veröffentlicht wurde, sodass der Block gleichzeitig als Navigationshinweis dient, der zeigt, worüber gerade gesprochen wird.
- Der Feed gilt shopweit. Um die neuesten Kommentare für eine einzelne Seite anzuzeigen, verwenden Sie stattdessen den **FastComments**-Block auf dieser Seite.
