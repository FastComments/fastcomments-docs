С помощью FastComments мы можем легко добавить комментарии в реальном времени на любой сайт Shopify.

Инструкции ниже предназначены для блога, созданного на Shopify, но код встраивания также можно вручную добавить на любую страницу, отредактировав другие файлы темы. Служба поддержки FastComments также может помочь с этим.

Обратите внимание, что для этого руководства требуется учетная запись FastComments. Рекомендуется сначала зарегистрироваться, а затем вернуться сюда. [Вы можете создать учетную запись здесь.](https://fastcomments.com/auth/tenant-signup?packageId=flex)
