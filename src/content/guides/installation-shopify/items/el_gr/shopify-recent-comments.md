The **FastComments - Recent Comments** block εμφανίζει τα πιο πρόσφατα σχόλια σε ολόκληρο το κατάστημά σας. Χρησιμοποιήστε το για να δείξετε πρόσφατη δραστηριότητα σε μια πλαϊνή στήλη, στο υποσέλιδο ή σε μια ενότητα της αρχικής σελίδας.

### Προσθήκη του μπλοκ

1. Ανοίξτε τον επεξεργαστή θέματος του Shopify.
2. Ανοίξτε το template όπου θέλετε τη ροή. Μια ενότητα πλαϊνής στήλης, η αρχική σελίδα ή ο κατάλογος blog λειτουργούν καλά.
3. Κάντε κλικ στο **Add block** στην ενότητα που πρέπει να την περιέχει.
4. Στην κατηγορία **Apps**, επιλέξτε **FastComments - Recent Comments**.
5. Κάντε κλικ στο **Save**.

### Ρυθμίσεις

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Αντικαθιστά τον FastComments tenant από τον οποίο διαβάζει η ροή. Αφήστε κενό για να χρησιμοποιηθεί ο αυτόματα ρυθμισμένος tenant του καταστήματος. | (blank) |
| Number of comments | Πόσα πρόσφατα σχόλια να εμφανιστούν. Εύρος 1 έως 50. | 5 |

### Συμβουλές

- Κάθε σχόλιο συνδέεται με τη σελίδα στην οποία δημοσιεύτηκε, οπότε το μπλοκ λειτουργεί και ως υπόδειξη πλοήγησης "για το τι συζητά ο κόσμος αυτήν τη στιγμή".
- Η ροή είναι σε επίπεδο καταστήματος. Για να εμφανίσετε πρόσφατα σχόλια για μία μόνο σελίδα, χρησιμοποιήστε αντίστοιχα το **FastComments** μπλοκ σε εκείνη τη σελίδα.
