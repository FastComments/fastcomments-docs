Το μπλοκ **FastComments - Comment Count** εμφανίζει έναν μικρό μετρητή σχολίων για μια μεμονωμένη σελίδα. Χρησιμοποιήστε το σε λίστες δημοσιεύσεων ιστολογίου, κάρτες προϊόντων ή οποιοδήποτε πρότυπο που οδηγεί σε μια σελίδα με σχόλια, ώστε οι επισκέπτες να μπορούν να δουν πόσο ενεργό είναι κάθε νήμα πριν κάνουν κλικ.

### Προσθήκη του μπλοκ

1. Ανοίξτε τον επεξεργαστή θέματος Shopify.
2. Ανοίξτε το πρότυπο όπου θέλετε να εμφανίζεται ο μετρητής. Για παράδειγμα, το πρότυπο **Blog** (η λίστα δημοσιεύσεων) ή μια ενότητα λίστας προϊόντων.
3. Κάντε κλικ στο **Προσθήκη μπλοκ** στην ενότητα που αποδίδει κάθε στοιχείο.
4. Κάτω από **Εφαρμογές**, επιλέξτε **FastComments - Comment Count**.
5. Κάντε κλικ στο **Αποθήκευση**.

### Ρυθμίσεις

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (προαιρετικό) | Επικαλύπτει ποιος tenant του FastComments χρησιμοποιείται για τον μετρητή. Αφήστε κενό για να χρησιμοποιηθεί ο αυτόματα ρυθμισμένος tenant του καταστήματος. | (κενό) |
| Προσαρμοσμένο Αναγνωριστικό URL | Παρέχει προσαρμοσμένο αναγνωριστικό σελίδας που αναζητά ο μετρητής. Χρησιμοποιήστε το όταν ο μετρητής βρίσκεται σε διαφορετική σελίδα από το μπλοκ FastComments που παρακολουθεί. | (αναγνωρίζεται αυτόματα) |

### Πώς ο μετρητής ταιριάζει με το νήμα σχολίων

Το μπλοκ Comment Count χρησιμοποιεί την ίδια λογική αυτόματης ανίχνευσης με το μπλοκ **FastComments**:

- Πρότυπο δημοσίευσης ιστολογίου: `shopify-article-{article.id}`
- Πρότυπο προϊόντος: `shopify-product-{product.id}`
- Άλλα πρότυπα: the request path

Εάν ορίσετε ένα **Προσαρμοσμένο Αναγνωριστικό URL** στο μπλοκ **FastComments** σε μια σελίδα, ορίστε το ίδιο **Προσαρμοσμένο Αναγνωριστικό URL** και στο μπλοκ Comment Count ώστε να δείχνουν στο ίδιο νήμα.

### Συμβουλές

- Οι μετρήσεις για κάθε στοιχείο στη σελίδα ανακτώνται με ένα αίτημα, οπότε η προσθήκη του μπλοκ σε κάθε στοιχείο σε μια μεγάλη λίστα δεν προσθέτει επιπλέον κόστος δικτύου.
- Η προβλεπόμενη χρήση είναι ένα μπλοκ Comment Count ανά άρθρο ή προϊόν σε μια λίστα· το μπλοκ μπορεί να προστεθεί όσες φορές χρειάζεται.
