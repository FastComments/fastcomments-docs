Εάν έχετε ήδη σχόλια στις αναρτήσεις του Shopify ιστολογίου σας και θέλετε να τα κρατήσετε, εισαγάγετέ τα στο FastComments πριν απενεργοποιήσετε τα ενσωματωμένα σχόλια του Shopify.

Η εισαγωγή πρέπει να εκτελεστεί ενώ τα ενσωματωμένα σχόλια του Shopify είναι ακόμη ενεργά και ορατά. Μόλις τα απενεργοποιήσετε, το FastComments δεν θα μπορεί πλέον να τα διαβάσει.

### Πώς να ξεκινήσετε μια εισαγωγή

1. Βεβαιωθείτε ότι η εφαρμογή FastComments είναι εγκατεστημένη στο κατάστημά σας.
2. Αφήστε τα ενσωματωμένα σχόλια του Shopify **ενεργοποιημένα** προς το παρόν.
3. [Open a support ticket](https://fastcomments.com/auth/my-account/tickets) και πείτε μας ποιες ιστολογίες να εισαγάγουμε.

Η εισαγωγή ολοκληρώνεται σε λιγότερο από μια ημέρα.

Μόλις τα υπάρχοντα σχόλιά σας βρίσκονται στο FastComments, προσθέστε το μπλοκ **FastComments** στο πρότυπο ανάρτησης του ιστολογίου σας και αφαιρέστε ή απενεργοποιήστε τα ενσωματωμένα σχόλια του Shopify. Δείτε [Προσθήκη σχολίων σε αναρτήσεις ιστολογίου και προϊόντα](/guide-installation-shopify.html#shopify-add-comments).

Εάν δεν χρειάζεται να διατηρήσετε τα υπάρχοντα σχόλια, μπορείτε να παραλείψετε αυτή τη σελίδα.
