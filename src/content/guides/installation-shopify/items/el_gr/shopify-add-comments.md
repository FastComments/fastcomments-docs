The **FastComments** block είναι το κύριο widget σχολιασμών. Προσθέστε το σε πρότυπα δημοσιευμάτων ιστολογίου, πρότυπα προϊόντων ή οποιαδήποτε άλλη σελίδα όπου θέλετε ένα νήμα συζήτησης ή μια ζωντανή συνομιλία.

### Προσθήκη του μπλοκ

1. Ανοίξτε τον επεξεργαστή θέματος του Shopify (**Online Store > Themes > Customize**).
2. Επιλέξτε το πρότυπο στο οποίο θέλετε σχόλια: **Blog post**, **Product**, ή οποιοδήποτε άλλο πρότυπο σελίδας ή ενότητας.
3. Στην ενότητα όπου θέλετε να εμφανιστούν τα σχόλια, κάντε κλικ στο **Add block**.
4. Υπό **Apps**, επιλέξτε **FastComments**.
5. Κάντε κλικ στο **Save**.

Το μπλοκ εμφανίζεται αμέσως. Δεν χρειάζεται να εισαγάγετε Tenant ID· ο tenant του καταστήματός σας συνδέεται αυτόματα όταν εγκαθιστάτε την εφαρμογή.

### Ρυθμίσεις

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Επικαλύπτει τον tenant του FastComments με τον οποίο θα αποδίδεται το μπλοκ. Αφήστε κενό για να χρησιμοποιηθεί ο αυτόματα ρυθμισμένος tenant του καταστήματος. Βρείτε ένα χειροκίνητο tenant ID στο fastcomments.com/auth/my-account/api-secret. | (κενό) |
| SSO | Κάνει αυτόματη σύνδεση του επισκέπτη ως ο λογαριασμός πελάτη Shopify πριν σχολιάσει. Δείτε [Auto-Login Shopify Customers](/guide-installation-shopify.html#shopify-sso). | On |
| Commenting Style | **Threaded** για εμφάνιση απαντήσεων σε επίπεδα και ψήφων, ή **Streaming** για πραγματικό χρόνο ροής σχολίων. | Threaded |
| Custom URL ID | Υπερισχύει του αυτόματα εντοπιζόμενου αναγνωριστικού σελίδας. Χρησιμοποιήστε το όταν θέλετε δύο διευθύνσεις URL να μοιράζονται το ίδιο νήμα σχολίων. | (auto-detected) |

### Πώς επιλέγεται το αναγνωριστικό σελίδας

Κάθε νήμα σχολίων ταυτοποιείται με ένα URL ID. Το μπλοκ επιλέγει ένα αυτόματα:

- **Blog post template:** `shopify-article-{article.id}`, το οποίο παραμένει σταθερό παρά τις αλλαγές slug και τίτλου.
- **Product template:** `shopify-product-{product.id}`, το οποίο παραμένει σταθερό παρά τις αλλαγές slug και τίτλου.
- **Other templates:** το μονοπάτι του αιτήματος.

Εάν ορίσετε **Custom URL ID**, χρησιμοποιείται εκείνη η τιμή αντί αυτής. Χρησιμοποιήστε το ίδιο Custom URL ID σε πολλαπλά μπλοκ (για παράδειγμα, σε μια τοπικοποιημένη παραλλαγή μιας σελίδας προϊόντος) για να μοιράζεστε ένα νήμα σχολίων.

### Threaded vs Streaming

**Threaded** είναι η προεπιλογή. Οι επισκέπτες απαντούν ο ένας στον άλλο, ψηφίζουν και τα εργαλεία εποπτείας λειτουργούν όπως αναμένεται. Καλύτερο για δημοσιεύσεις ιστολογίου και αξιολογήσεις προϊόντων.

**Streaming** αφαιρεί τη δομή νήματος και εμφανίζει νέα σχόλια σε πραγματικό χρόνο καθώς δημοσιεύονται, σαν ροή συνομιλίας. Καλύτερο για λανσαρίσματα προϊόντων, ζωντανές εκδηλώσεις και σελίδες κοινοτήτων.

### Πολλαπλά μπλοκ στην ίδια σελίδα

Το μπλοκ μπορεί να προστεθεί περισσότερες από μία φορές στο ίδιο πρότυπο. Για παράδειγμα, μια Σύνοψη Αξιολογήσεων στην κορυφή μιας σελίδας προϊόντος και ένα μπλοκ FastComments στο κάτω μέρος. Τα μπλοκ μοιράζονται ένα URL ID, οπότε η σύνοψη αντικατοπτρίζει τα σχόλια από κάτω.

### Συμβουλές

- Το μπλοκ κρύβεται στην προεπισκόπηση του επεξεργαστή θέματος με μια κίτρινη ειδοποίηση αν δεν μπορεί να βρει tenant. Εάν αυτό εμφανιστεί στο ζωντανό σας κατάστημα, επανεγκαταστήστε την εφαρμογή FastComments.
- Για μια σελίδα προϊόντος, το μπλοκ FastComments λειτουργεί και ως widget αξιολογήσεων προϊόντος. Συνδυάστε το με **FastComments - Reviews Summary** για μια περίληψη βαθμολογίας με αστέρια στην κορυφή της σελίδας.
