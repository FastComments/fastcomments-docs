The **FastComments - Recent Discussions** block lists the pages on your store with the most recent comment activity, one entry per page. Use it when you want visitors to discover active threads rather than individual comments.

### Προσθήκη του μπλοκ

1. Ανοίξτε τον επεξεργαστή θέματος του Shopify.
2. Ανοίξτε το πρότυπο όπου θέλετε να τοποθετηθεί η ροή, όπως μια πλαϊνή στήλη, υποσέλιδο, ευρετήριο ιστολογίου ή ενότητα αρχικής σελίδας.
3. Κάντε κλικ στο **Add block** στην ενότητα που θα το φιλοξενήσει.
4. Στην ενότητα **Apps**, επιλέξτε **FastComments - Recent Discussions**.
5. Κάντε κλικ στο **Save**.

### Ρυθμίσεις

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Επιτρέπει να παρακάμψετε τον tenant του FastComments από τον οποίο διαβάζει η ροή. Αφήστε κενό για να χρησιμοποιηθεί ο αυτόματα ρυθμισμένος tenant του καταστήματος. | (blank) |
| Number of comments | Πόσες πρόσφατες συζητήσεις να εμφανίζονται. Εύρος 1 έως 50. | 20 |

### Recent Discussions vs Recent Comments

Και τα δύο μπλοκ αντλούν από τα ίδια δεδομένα, αλλά εμφανίζουν διαφορετικές προβολές:

- **Recent Comments**: μια επίπεδη λίστα με τα πιο πρόσφατα μεμονωμένα σχόλια. Κατάλληλο για «τι ειπώθηκε μόλις».
- **Recent Discussions**: μία εγγραφή ανά σελίδα, ταξινομημένη κατά τη πιο πρόσφατη δραστηριότητα. Κατάλληλο για «ποια νήματα είναι ενεργά τώρα».

Μπορείτε να χρησιμοποιήσετε και τα δύο στο ίδιο κατάστημα. Για παράδειγμα, Recent Comments στην πλαϊνή στήλη του ιστολογίου και Recent Discussions στην αρχική σελίδα.
