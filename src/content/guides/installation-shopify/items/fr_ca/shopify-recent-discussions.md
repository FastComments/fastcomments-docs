Le bloc **FastComments - Recent Discussions** répertorie les pages de votre boutique ayant l'activité de commentaires la plus récente, une entrée par page. Utilisez-le quand vous voulez que les visiteurs découvrent des fils de discussion actifs plutôt que des commentaires individuels.

### Add the block

1. Ouvrez l'éditeur de thème Shopify.
2. Ouvrez le modèle où vous voulez le flux, comme une barre latérale, le pied de page, l'index du blogue ou une section de la page d'accueil.
3. Cliquez sur **Add block** dans la section qui doit le contenir.
4. Sous **Apps**, sélectionnez **FastComments - Recent Discussions**.
5. Cliquez sur **Save**.

### Settings

| Setting | What it does | Default |
|---|---|---|
| ID du locataire (facultatif) | Remplace le locataire FastComments dont le flux lit les données. Laissez vide pour utiliser le locataire configuré automatiquement pour la boutique. | (vide) |
| Nombre de commentaires | Combien de discussions récentes à afficher. Plage de 1 à 50. | 20 |

### Recent Discussions vs Recent Comments

Both blocks pull from the same data, but they show different views:

- **Recent Comments**: a flat list of the latest individual comments. Best for "what was just said".
- **Recent Discussions**: one entry per page, ordered by most recent activity. Best for "which threads are alive right now".

You can use both on the same store. For example, Recent Comments in the blog sidebar and Recent Discussions on the home page.
