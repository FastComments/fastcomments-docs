Le bloc **FastComments - Recent Comments** affiche les commentaires les plus récents de l'ensemble de votre boutique. Utilisez-le pour montrer une activité récente dans une barre latérale, le pied de page ou une section de la page d'accueil.

### Ajouter le bloc

1. Ouvrez l'éditeur de thème Shopify.
2. Ouvrez le gabarit où vous souhaitez afficher le fil. Une section de barre latérale, la page d'accueil ou l'index du blogue conviennent bien.
3. Cliquez sur **Ajouter un bloc** dans la section qui doit le contenir.
4. Sous **Applications**, sélectionnez **FastComments - Recent Comments**.
5. Cliquez sur **Enregistrer**.

### Paramètres

| Paramètre | Fonction | Valeur par défaut |
|---|---|---|
| Tenant ID (optionnel) | Remplace le tenant FastComments à partir duquel le fil est lu. Laissez vide pour utiliser le tenant automatiquement configuré de la boutique. | (vide) |
| Nombre de commentaires | Nombre de commentaires récents à afficher. Plage : 1 à 50. | 5 |

### Conseils

- Chaque commentaire renvoie à la page où il a été publié, donc le bloc sert aussi d'indice de navigation « ce dont les gens parlent en ce moment ».
- Le fil couvre l'ensemble de la boutique. Pour afficher les commentaires récents d'une seule page, utilisez plutôt le bloc **FastComments** sur cette page.
