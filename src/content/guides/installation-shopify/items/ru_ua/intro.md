С помощью FastComments мы можем легко добавить комментарии в реальном времени на любой сайт Shopify.

Инструкции ниже предназначены для блога, созданного на Shopify, но код вставки также можно вручную добавить на любую страницу, отредактировав другие файлы темы. Поддержка FastComments также может помочь с этим.

Обратите внимание, что для этого руководства требуется аккаунт FastComments. Рекомендуется сначала зарегистрироваться, а затем вернуться сюда. [Вы можете создать аккаунт здесь.](https://fastcomments.com/auth/tenant-signup?packageId=flex)
