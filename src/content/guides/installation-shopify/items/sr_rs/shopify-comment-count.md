Блок **FastComments - Број коментара** приказује мали број коментара за једну страницу. Користите га у списковима блог постова, картицама производа, или било ком шаблону који води ка страници са коментарима, тако да посетиоци могу видети колико је свака нит активна пре него што кликну.

### Додајте блок

1. Отворите уређивач Shopify теме.
2. Отворите шаблон у ком желите да се број приказује. На пример, **Blog** шаблон (листa постова) или секција листе производа.
3. Кликните **Add block** у секцији која рендерује сваки елемент.
4. Под **Apps**, изаберите **FastComments - Број коментара**.
5. Кликните **Save**.

### Подешавања

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Пренадгледа са ког FastComments tenant-а бројач чита податке. Оставите празно да бисте користили tenant који је продавница аутоматски конфигурисала. | (празно) |
| Custom URL ID | Замењује идентификатор странице који бројач тражи. Користите ово када је бројач на другој страници од FastComments блока који прати. | (аутоматски детектовано) |

### Како се бројач подудара са нити коментара

The Comment Count block uses the same auto-detection logic as the **FastComments** block:

- Шаблон блог поста: `shopify-article-{article.id}`
- Шаблон производа: `shopify-product-{product.id}`
- Други шаблони: the request path

Ако на страници подесите **Custom URL ID** на **FastComments** блоку, подесите исти **Custom URL ID** на **FastComments - Број коментара** блоку тако да показују на исту нит.

### Савети

- Бројеви за сваки елемент на страници дохватају се у једном захтеву, тако да додавање блока на сваки елемент у дугачкој листи не захтева додатне мрежне захтеве.
- Очекиван начин коришћења је један **FastComments - Број коментара** блок по чланку или производу у листингу; блок можете додати онолико пута колико вам треба.
