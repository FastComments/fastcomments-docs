The **FastComments - Recent Discussions** блок наводи странице у вашој продавници са најновијом активношћу коментара, по један унос по страници. Користите га када желите да посетиоци открију активне теме уместо појединачних коментара.

### Додајте блок

1. Отворите уређивач тема у Shopify-у.
2. Отворите шаблон где желите фид, као што су бочни панел, подножје, индекс блога или одељак на почетној страници.
3. Кликните **Додај блок** у одељку који треба да га садржи.
4. Под **Апликације**, изаберите **FastComments - Recent Discussions**.
5. Кликните **Сачувај**.

### Подешавања

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Замењује који FastComments tenant фид чита. Оставите празно да бисте користили аутоматски конфигурисани tenant продавнице. | (празно) |
| Number of comments | Колико најновијих дискусија да се прикаже. Опсег 1 до 50. | 20 |

### Recent Discussions vs Recent Comments

Both blocks pull from the same data, but they show different views:

- **Recent Comments**: равна листа најновијих појединачних коментара. Најбоље за "оно што је управо речено".
- **Recent Discussions**: по један унос по страници, по редоследу најновије активности. Најбоље за "које теме су тренутно активне".

Можете користити оба на истој продавници. На пример, Recent Comments у бочном панелу блога и Recent Discussions на почетној страници.
