The **FastComments - Recent Comments** block muestra los comentarios más recientes de toda tu tienda. Úsalo para mostrar actividad reciente en una barra lateral, pie de página o sección de la página de inicio.

### Añadir el bloque

1. Abre el editor de temas de Shopify.
2. Abre la plantilla donde quieres el feed. Una sección de barra lateral, la página de inicio o el índice del blog funcionan bien.
3. Haz clic en **Add block** en la sección que debe contenerlo.
4. Bajo **Apps**, selecciona **FastComments - Recent Comments**.
5. Haz clic en **Save**.

### Configuración

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Anula de qué tenant de FastComments proviene el feed. Déjalo en blanco para usar el tenant configurado automáticamente para la tienda. | (vacío) |
| Number of comments | Cuántos comentarios recientes mostrar. Rango de 1 a 50. | 5 |

### Consejos

- Cada comentario enlaza a la página donde se publicó, por lo que el bloque también funciona como una pista de navegación de "sobre qué está hablando la gente ahora".
- El feed abarca toda la tienda. Para mostrar comentarios recientes de una sola página, usa el bloque **FastComments** en esa página en su lugar.
