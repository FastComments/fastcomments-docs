Si instalaste FastComments desde la Shopify App Store, el dominio de tu tienda se añade automáticamente a los dominios autorizados del tenant y no deberías ver un error de dominio. Esta página se aplica si seguiste la ruta de instalación manual, o si tu tienda se sirve en un dominio personalizado que no estaba registrado con Shopify en el momento en que se instaló la aplicación.

Puedes recibir un error de autorización como el siguiente:

<div class="screenshot white-bg">
    <div class="title">Falta configuración del dominio</div>
    <img class="screenshot-image" src="/images/installation-guides/installation-guide-webflow-step-5.png" alt="Falta configuración del dominio" />
</div>

Esto significa que FastComments no reconoce el dominio en el que se está cargando el widget como autorizado para tu tenant.

Para solucionarlo, añade el dominio a tu cuenta de FastComments: [Configurar dominios](https://fastcomments.com/auth/my-account/configure-domains).
