הבלוק **FastComments - Comment Count** מציג ספירת תגובות קטנה עבור דף בודד. השתמש בו ברשימות פוסטים בבלוג, בכרטיסי מוצרים, או בכל תבנית שמקשרת לדף עם תגובות, כדי שמבקרים יוכלו לראות כמה פעיל כל אשכול לפני שהם נכנסים אליו.

### הוספת הבלוק

1. פתח את עורך התבנית של Shopify.
2. פתח את התבנית שבה תרצה שהספירה תופיע. לדוגמה, תבנית **Blog** (רשימת הפוסטים) או מקטע רשימת מוצרים.
3. לחץ על **Add block** באזור שמציג כל פריט.
4. תחת **Apps**, בחר ב־**FastComments - Comment Count**.
5. לחץ על **Save**.

### הגדרות

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (אופציונלי) | מעקף את השוכר (tenant) של FastComments ממנו הספירה קוראת. השאר ריק כדי להשתמש בשוכר שהוגדר אוטומטית לחנות. | (ריק) |
| Custom URL ID | מעקף את מזהה הדף שהספירה מחפשת. השתמש בזה כאשר הספירה נמצאת בדף שונה מהבלוק של FastComments שהיא עוקבת אחריו. | (זוהה אוטומטית) |

### כיצד הספירה מתאמת לאשכול התגובות

בלוק ספירת התגובות משתמש באותה לוגיקת זיהוי אוטומטי כמו הבלוק **FastComments**:

- תבנית פוסט בבלוג: `shopify-article-{article.id}`
- תבנית מוצר: `shopify-product-{product.id}`
- תבניות אחרות: נתיב הבקשה

אם הגדרת **Custom URL ID** בבלוק **FastComments** בעמוד, הגדר את אותו Custom URL ID בבלוק ספירת התגובות כך שהם יפנו לאותו אשכול.

### טיפים

- הספירות לכל פריט בעמוד נמשכות בבקשה אחת, כך שהוספת הבלוק לכל פריט ברשימה ארוכה אינה מייצרת קריאות נוספות.
- שימוש צפוי הוא בלוק אחד של ספירת תגובות לכל מאמר או מוצר ברשימה; ניתן להוסיף את הבלוק כמה פעמים שצריך.
