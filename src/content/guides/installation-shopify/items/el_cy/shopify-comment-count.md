Το **FastComments - Comment Count** μπλοκ εμφανίζει μια μικρή ένδειξη αριθμού σχολίων για μία μόνο σελίδα. Χρησιμοποιήστε το σε λίστες αναρτήσεων ιστολογίου, κάρτες προϊόντων ή οποιοδήποτε πρότυπο που συνδέει σε μια σελίδα με σχόλια, ώστε οι επισκέπτες να βλέπουν πόσο ενεργό είναι κάθε νήμα πριν κάνουν κλικ.

### Προσθήκη του μπλοκ

1. Ανοίξτε τον επεξεργαστή θεμάτων του Shopify.
2. Ανοίξτε το πρότυπο όπου θέλετε να εμφανίζεται ο αριθμός. Για παράδειγμα, το πρότυπο **Blog** (η λίστα αναρτήσεων) ή μια ενότητα λίστας προϊόντων.
3. Κάντε κλικ στο **Add block** στην ενότητα που αποδίδει κάθε στοιχείο.
4. Κάτω από **Apps**, επιλέξτε **FastComments - Comment Count**.
5. Κάντε κλικ στο **Save**.

### Settings

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Παράκαμψη του Tenant του FastComments από τον οποίο θα διαβάζει ο αριθμός. Αφήστε κενό για να χρησιμοποιηθεί ο αυτόματα ρυθμισμένος tenant του καταστήματος. | (κενό) |
| Custom URL ID | Παράκαμψη του αναγνωριστικού σελίδας που αναζητεί ο αριθμός. Χρησιμοποιήστε το όταν ο αριθμός βρίσκεται σε διαφορετική σελίδα από το μπλοκ FastComments που παρακολουθεί. | (εντοπισμένο αυτόματα) |

### Πώς ο αριθμός αντιστοιχεί στο νήμα σχολίων

Το μπλοκ Comment Count χρησιμοποιεί την ίδια λογική αυτόματης ανίχνευσης όπως το μπλοκ **FastComments**:

- Πρότυπο ανάρτησης ιστολογίου: `shopify-article-{article.id}`
- Πρότυπο προϊόντος: `shopify-product-{product.id}`
- Άλλα πρότυπα: η διαδρομή του αιτήματος

Αν ρυθμίσετε ένα **Custom URL ID** στο μπλοκ **FastComments** σε μια σελίδα, ορίστε το ίδιο Custom URL ID και στο μπλοκ Comment Count ώστε να δείχνουν στο ίδιο νήμα.

### Συμβουλές

- Οι μετρήσεις για κάθε στοιχείο στη σελίδα ανακτώνται με ένα μόνο αίτημα, οπότε η προσθήκη του μπλοκ σε κάθε στοιχείο μιας μεγάλης λίστας δεν έχει επιπλέον κόστος round-trip.
- Η αναμενόμενη χρήση είναι ένα μπλοκ Comment Count ανά άρθρο ή προϊόν σε μια λίστα· μπορείτε να προσθέσετε το μπλοκ όσες φορές χρειάζεστε.
