Το μπλοκ **FastComments - Top Pages** εμφανίζει τις σελίδες στο κατάστημά σας με τη μεγαλύτερη δραστηριότητα σχολιασμού συνολικά. Χρησιμοποιήστε το για να αναδείξετε τις πιο πολυσυζητημένες δημοσιεύσεις ή προϊόντα σας.

### Add the block

1. Ανοίξτε τον επεξεργαστή θεμάτων του Shopify.
2. Ανοίξτε το πρότυπο όπου θέλετε τη λίστα, όπως μια πλαϊνή στήλη, το υποσέλιδο ή την ενότητα ευρετηρίου του ιστολογίου.
3. Κάντε κλικ στο **Προσθήκη μπλοκ** στην ενότητα που θα το φιλοξενεί.
4. Κάτω από **Εφαρμογές**, επιλέξτε **FastComments - Top Pages**.
5. Κάντε κλικ στο **Αποθήκευση**.

### Settings

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Παρακάμψτε ποιο tenant του FastComments θα χρησιμοποιεί η λίστα. Αφήστε κενό για να χρησιμοποιηθεί ο tenant που έχει ρυθμιστεί αυτόματα για το κατάστημα. | (κενό) |

### Top Pages vs Recent Discussions

- **Top Pages** ταξινομεί κατά συνολική αλληλεπίδραση, οπότε τα ίδια άρθρα τείνουν να παραμένουν κοντά στην κορυφή για αρκετό καιρό. Κατάλληλο για μια πλαϊνή στήλη «δημοφιλείς δημοσιεύσεις».
- **Recent Discussions** ταξινομεί με βάση την πιο πρόσφατη δραστηριότητα, οπότε η λίστα ανανεώνεται συχνά. Κατάλληλο για ένα πλαίσιο «τι είναι δημοφιλές τώρα».
