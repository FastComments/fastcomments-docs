Το **FastComments** block είναι το κύριο widget σχολιασμού. Προσθέστε το σε πρότυπα αναρτήσεων ιστολογίου, πρότυπα προϊόντων, ή σε οποιαδήποτε άλλη σελίδα όπου θέλετε ένα νήμα συζήτησης ή μια ζωντανή συνομιλία.

### Προσθήκη του block

1. Ανοίξτε τον επεξεργαστή θέματος του Shopify (**Online Store > Themes > Customize**).
2. Επιλέξτε το template στο οποίο θέλετε σχόλια: **Blog post**, **Product**, ή οποιοδήποτε άλλο template σελίδας ή ενότητας.
3. Στην ενότητα όπου θέλετε να εμφανίζονται τα σχόλια, κάντε κλικ στο **Add block**.
4. Κάτω από **Apps**, επιλέξτε **FastComments**.
5. Κάντε κλικ στο **Save**.

Το block εμφανίζεται αμέσως. Δεν υπάρχει Tenant ID για εισαγωγή· ο tenant του καταστήματός σας συνδέεται αυτόματα όταν εγκαταστήσετε την εφαρμογή.

### Ρυθμίσεις

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Αντικαθιστά σε ποιο tenant του FastComments αποδίδεται το block. Αφήστε κενό για να χρησιμοποιηθεί ο tenant που έχει διαμορφωθεί αυτόματα από το κατάστημα. Βρείτε ένα χειροκίνητο tenant ID στο fastcomments.com/auth/my-account/api-secret. | (blank) |
| SSO | Συνδέει αυτόματα τον επισκέπτη ως τον λογαριασμό πελάτη Shopify πριν σχολιάσει. Δείτε [Αυτόματη σύνδεση πελατών Shopify](/guide-installation-shopify.html#shopify-sso). | On |
| Commenting Style | **Threaded** για εμφωλευμένες απαντήσεις και ψήφους, ή **Streaming** για ροή συνομιλίας σε πραγματικό χρόνο. | Threaded |
| Custom URL ID | Αντικαθιστά τον αυτόματα ανιχνευμένο αναγνωριστικό σελίδας. Χρησιμοποιήστε το όταν θέλετε δύο URLs να μοιράζονται ένα νήμα σχολίων. | (auto-detected) |

### Πώς επιλέγεται ο αναγνωριστής της σελίδας

Κάθε νήμα σχολίων ταυτοποιείται από ένα URL ID. Το block επιλέγει ένα αυτόματα:

- **Blog post template:** `shopify-article-{article.id}`, το οποίο είναι σταθερό ανεξάρτητα από αλλαγές στο slug και τον τίτλο.
- **Product template:** `shopify-product-{product.id}`, το οποίο είναι σταθερό ανεξάρτητα από αλλαγές στο slug και τον τίτλο.
- **Other templates:** το μονοπάτι του αιτήματος.

Αν ορίσετε **Custom URL ID**, αυτή η τιμή χρησιμοποιείται αντί αυτής. Χρησιμοποιήστε το ίδιο Custom URL ID σε πολλαπλά blocks (για παράδειγμα, σε μια τοπικοποιημένη παραλλαγή μιας σελίδας προϊόντος) για να μοιράζεστε ένα νήμα σχολίων.

### Threaded vs Streaming

**Threaded** είναι η προεπιλογή. Οι επισκέπτες απαντούν ο ένας στον άλλο, ψηφίζουν, και τα εργαλεία moderation λειτουργούν όπως αναμένεται. Ιδανικό για αναρτήσεις ιστολογίου και κριτικές προϊόντων.

**Streaming** καταργεί τη δομή των νημάτων και εμφανίζει νέα σχόλια σε πραγματικό χρόνο καθώς δημοσιεύονται, σαν ροή συνομιλίας. Ιδανικό για λανσαρίσματα προϊόντων, ζωντανές εκδηλώσεις και σελίδες κοινότητας.

### Πολλαπλά blocks στην ίδια σελίδα

Το block μπορεί να προστεθεί περισσότερες από μία φορές στο ίδιο template. Για παράδειγμα, μια Reviews Summary στην κορυφή μιας σελίδας προϊόντος και ένα FastComments block στο κάτω μέρος. Τα blocks μοιράζονται ένα URL ID, έτσι η σύνοψη αντικατοπτρίζει τα σχόλια παρακάτω.

### Συμβουλές

- Το block κρύβεται στην προεπισκόπηση του theme editor με μια κίτρινη ειδοποίηση αν δεν μπορεί να βρει tenant. Αν εμφανιστεί αυτό στο live κατάστημά σας, επανεγκαταστήστε την εφαρμογή FastComments.
- Σε σελίδα προϊόντος, το FastComments block λειτουργεί ταυτόχρονα και ως το widget κριτικών προϊόντων σας. Συνδυάστε το με **FastComments - Reviews Summary** για μια σύνοψη αξιολόγησης με αστέρια στην κορυφή της σελίδας.
