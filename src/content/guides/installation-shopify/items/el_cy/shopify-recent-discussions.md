Το μπλοκ **FastComments - Recent Discussions** απαριθμεί τις σελίδες στο κατάστημά σας με την πιο πρόσφατη δραστηριότητα σχολίων, μία εγγραφή ανά σελίδα. Χρησιμοποιήστε το όταν θέλετε οι επισκέπτες να ανακαλύψουν ενεργά νήματα αντί για μεμονωμένα σχόλια.

### Προσθήκη του μπλοκ

1. Ανοίξτε τον επεξεργαστή θέματος Shopify.
2. Ανοίξτε το πρότυπο όπου θέλετε τη ροή, όπως μια πλαϊνή στήλη, υποσέλιδο, ευρετήριο ιστολογίου ή ενότητα αρχικής σελίδας.
3. Κάντε κλικ στο **Προσθήκη μπλοκ** στην ενότητα που θα το φιλοξενεί.
4. Κάτω από **Εφαρμογές**, επιλέξτε **FastComments - Recent Discussions**.
5. Κάντε κλικ στο **Αποθήκευση**.

### Ρυθμίσεις

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Παράκαμψη του tenant του FastComments από τον οποίο διαβάζει η ροή. Αφήστε κενό για να χρησιμοποιηθεί ο tenant που έχει διαμορφωθεί αυτόματα για το κατάστημα. | (κενό) |
| Number of comments | Πόσες πρόσφατες συζητήσεις να εμφανίζονται. Εύρος 1 έως 50. | 20 |

### Recent Discussions vs Recent Comments

Και τα δύο μπλοκ αντλούν από τα ίδια δεδομένα, αλλά δείχνουν διαφορετικές προβολές:

- **Recent Comments**: μια επίπεδη λίστα των τελευταίων μεμονωμένων σχολίων. Καλύτερο για «τι ειπώθηκε μόλις».
- **Recent Discussions**: μία εγγραφή ανά σελίδα, ταξινομημένη κατά πιο πρόσφατη δραστηριότητα. Καλύτερο για «ποια νήματα είναι ενεργά αυτή τη στιγμή».

Μπορείτε να χρησιμοποιήσετε και τα δύο στο ίδιο κατάστημα. Για παράδειγμα, το Recent Comments στην πλαϊνή στήλη του ιστολογίου και το Recent Discussions στην αρχική σελίδα.
