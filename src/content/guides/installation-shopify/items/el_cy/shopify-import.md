Εάν έχετε ήδη σχόλια στις δημοσιεύσεις του blog σας στο Shopify και θέλετε να τα διατηρήσετε, εισαγάγετέ τα στο FastComments πριν απενεργοποιήσετε τα εγγενή σχόλια του Shopify.

Η εισαγωγή πρέπει να εκτελεστεί ενώ τα εγγενή σχόλια του Shopify εξακολουθούν να είναι ενεργά και ορατά. Μόλις τα απενεργοποιήσετε, το FastComments δεν μπορεί πλέον να τα διαβάσει.

### Πώς να ξεκινήσετε μια εισαγωγή

1. Βεβαιωθείτε ότι η εφαρμογή FastComments είναι εγκατεστημένη στο κατάστημά σας.
2. Αφήστε τα εγγενή σχόλια του Shopify **ενεργοποιημένα** προς το παρόν.
3. [Ανοίξτε ένα αίτημα υποστήριξης](https://fastcomments.com/auth/my-account/tickets) και ενημερώστε μας ποια ιστολόγια να εισάγουμε.

Η εισαγωγή ολοκληρώνεται σε λιγότερο από μία ημέρα.

Μόλις τα υπάρχοντα σχόλιά σας βρίσκονται στο FastComments, προσθέστε το μπλοκ **FastComments** στο πρότυπο της δημοσίευσης του ιστολογίου σας και αφαιρέστε ή απενεργοποιήστε τα εγγενή σχόλια του Shopify. Δείτε [Προσθήκη σχολίων σε δημοσιεύσεις ιστολογίου και προϊόντα](/guide-installation-shopify.html#shopify-add-comments).

Αν δεν χρειάζεται να διατηρήσετε τα υπάρχοντα σχόλια, μπορείτε να παραλείψετε αυτή τη σελίδα.
