If you already have comments on your Shopify blog posts and want to keep them, import them into FastComments before you disable Shopify's native commenting.

The import has to run while Shopify's native comments are still active and visible. Once you disable them, FastComments can no longer read them.

### How to start an import

1. Make sure the FastComments app is installed on your store.
2. Leave Shopify's native commenting **enabled** for now.
3. [Open a support ticket](https://fastcomments.com/auth/my-account/tickets) and tell us which blogs to import.

The import runs in less than a day.

Once your existing comments are in FastComments, add the **FastComments** block to your blog post template and remove or disable Shopify's native comments. See [Add Comments to Blog Posts and Products](/guide-installation-shopify.html#shopify-add-comments).

If you don't need to keep existing comments, you can skip this page.
