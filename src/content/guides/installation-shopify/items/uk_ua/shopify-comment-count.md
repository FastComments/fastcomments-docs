The **FastComments - Comment Count** block відображає невеликий лічильник коментарів для однієї сторінки. Використовуйте його в списках постів блогу, картках товарів або в будь-якому шаблоні, що посилається на сторінку з коментарями, щоб відвідувачі бачили, наскільки активна кожна тема, перш ніж переходити.

### Add the block

1. Відкрийте редактор тем Shopify.
2. Відкрийте шаблон, де ви хочете, щоб лічильник з'явився. Наприклад, шаблон **Blog** (список постів) або секцію зі списком товарів.
3. Натисніть **Add block** у секції, яка виводить кожен елемент.
4. У розділі **Apps** оберіть **FastComments - Comment Count**.
5. Натисніть **Save**.

### Settings

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optional) | Перезаписує, з якого оренданта FastComments зчитується лічильник. Залиште порожнім, щоб використовувати оренданта, налаштованого автоматично для магазину. | (порожньо) |
| Custom URL ID | Перезаписує ідентифікатор сторінки, за яким шукається лічильник. Використовуйте це, коли лічильник знаходиться на іншій сторінці, ніж блок FastComments, який він відстежує. | (визначається автоматично) |

### How the count matches the comment thread

The Comment Count block uses the same auto-detection logic as the **FastComments** block:

- Blog post template: `shopify-article-{article.id}`
- Product template: `shopify-product-{product.id}`
- Other templates: the request path

Якщо ви встановили **Custom URL ID** у блоці **FastComments** на сторінці, встановіть той самий Custom URL ID у блоці Comment Count, щоб вони вказували на одну і ту ж нитку коментарів.

### Tips

- Лічильники для всіх елементів на сторінці отримуються одним запитом, тому додавання блоку до кожного елемента в довгому списку не додає додаткових кругових запитів.
- Очікуване використання — один блок Comment Count на статтю або товар у списку; блок можна додавати стільки разів, скільки потрібно.
