Sada ćemo urediti temu naše trgovine kako bismo dodali FastComments.

U lijevom panelu otvorite `Themes`:

<div class="screenshot white-bg">
    <div class="title">Otvorite Themes</div>
    <img class="screenshot-image" src="/images/installation-guides/shopify-step-2-2-open-themes.png" alt="Otvorite Themes" />
</div>

Under `Current theme`, select `Actions` and then `Edit code`:

<div class="screenshot white-bg">
    <div class="title">Uredi kod</div>
    <img class="screenshot-image" src="/images/installation-guides/shopify-step-2-3-edit-code.png" alt="Uredi kod" />
</div>

This will open up a code editor, with a file browser on the left and the code on the right.

All we need to do is copy a small piece of code that adds FastComments and paste it on the right line in the right file.

In the file browser on the left, scroll down and click `Sections`:

<div class="screenshot white-bg">
    <div class="title">Odaberite Sections</div>
    <img class="screenshot-image" src="/images/installation-guides/shopify-step-2-4-sections.png" alt="Odaberite Sections" />
</div>

Now we're going to scroll down and select `main-article.liquid`:

<div class="screenshot white-bg">
    <div class="title">Odaberite main-article</div>
    <img class="screenshot-image" src="/images/installation-guides/shopify-step-2-5-main-article.png" alt="Odaberite main-article" />
</div>

This will open the *predložak teme* used to render a single blog article.

You should now see something similar to the following, with `main-article.liquid` selected at the top:
