Si vous avez installé FastComments depuis le Shopify App Store, le domaine de votre boutique est automatiquement ajouté aux domaines autorisés du locataire et vous ne devriez pas rencontrer d'erreur de domaine. Cette page s'applique si vous êtes passé par l'installation manuelle, ou si votre vitrine est hébergée sur un domaine personnalisé qui n'était pas enregistré auprès de Shopify au moment de l'installation de l'application.

Vous pouvez recevoir une erreur d'autorisation ressemblant à ceci :

<div class="screenshot white-bg">
    <div class="title">Configuration du domaine manquante</div>
    <img class="screenshot-image" src="/images/installation-guides/installation-guide-webflow-step-5.png" alt="Configuration du domaine manquante" />
</div>

Cela signifie que FastComments ne reconnaît pas le domaine sur lequel le widget se charge comme étant autorisé pour votre locataire.

Pour le corriger, ajoutez le domaine à votre compte FastComments : [Configurer les domaines](https://fastcomments.com/auth/my-account/configure-domains).
