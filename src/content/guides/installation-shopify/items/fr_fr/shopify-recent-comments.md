Le **FastComments - Recent Comments** block affiche les commentaires les plus récents dans l’ensemble de votre boutique. Utilisez-le pour montrer une activité récente dans une barre latérale, le pied de page ou une section de la page d’accueil.

### Ajouter le bloc

1. Ouvrez l’éditeur de thème Shopify.
2. Ouvrez le gabarit où vous voulez le flux. Une section de barre latérale, la page d’accueil ou l’index du blog conviennent bien.
3. Cliquez sur **Ajouter un bloc** dans la section qui doit le contenir.
4. Sous **Applications**, sélectionnez **FastComments - Recent Comments**.
5. Cliquez sur **Enregistrer**.

### Paramètres

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optionnel) | Remplace le tenant FastComments que le flux lit. Laisser vide pour utiliser le tenant automatiquement configuré par la boutique. | (vide) |
| Number of comments | Combien de commentaires récents afficher. Plage de 1 à 50. | 5 |

### Conseils

- Chaque commentaire contient un lien vers la page sur laquelle il a été posté, donc le bloc sert aussi d’indication de navigation "ce dont les gens parlent en ce moment".
- Le flux couvre toute la boutique. Pour afficher les commentaires récents d’une seule page, utilisez plutôt le bloc **FastComments** sur cette page.
