Le bloc **FastComments - Recent Discussions** répertorie les pages de votre boutique ayant l'activité de commentaires la plus récente, une entrée par page. Utilisez-le lorsque vous voulez que les visiteurs découvrent des fils de discussion actifs plutôt que des commentaires individuels.

### Ajouter le bloc

1. Ouvrez l'éditeur de thème Shopify.
2. Ouvrez le modèle où vous souhaitez afficher le flux, par exemple une barre latérale, le pied de page, l'index du blog ou une section de la page d'accueil.
3. Cliquez sur **Ajouter un bloc** dans la section qui doit le contenir.
4. Sous **Applications**, sélectionnez **FastComments - Recent Discussions**.
5. Cliquez sur **Enregistrer**.

### Paramètres

| Setting | What it does | Default |
|---|---|---|
| Tenant ID (optionnel) | Remplace le tenant FastComments à partir duquel le flux lit. Laissez vide pour utiliser le tenant configuré automatiquement pour la boutique. | (vide) |
| Number of comments | Combien de discussions récentes à afficher. Plage : 1 à 50. | 20 |

### Recent Discussions vs Recent Comments

Les deux blocs tirent des mêmes données, mais affichent des vues différentes :

- **Recent Comments** : une liste plate des derniers commentaires individuels. Idéal pour « ce qui vient d'être dit ».
- **Recent Discussions** : une entrée par page, ordonnée par activité la plus récente. Idéal pour « quels fils sont actifs en ce moment ».

Vous pouvez utiliser les deux sur la même boutique. Par exemple, Recent Comments dans la barre latérale du blog et Recent Discussions sur la page d'accueil.
