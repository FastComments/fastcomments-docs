Les couleurs, les polices, les règles de modération, le HTML autorisé et les commentaires anonymes peuvent tous être configurés pour correspondre à l'identité visuelle de votre boutique.

Ouvrez l'application FastComments dans l'administration Shopify et cliquez sur la tuile **Customize** pour ouvrir l'écran de personnalisation du widget. Elle utilise un lien de connexion à usage unique, vous n'avez donc pas besoin d'une authentification séparée.

Pour la liste complète des options de personnalisation, consultez le [Guide de personnalisation et de configuration](/guide-customizations-and-configuration.html).
