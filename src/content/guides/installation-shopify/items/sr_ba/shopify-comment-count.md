The **FastComments - Број коментара** блок приказује мали број коментара за појединачну страницу. Користите га у листама блог постова, картицама производа или било ком шаблону који води до странице са коментарима, како би посјетиоци могли видјети колико је сваки тред активан прије кликања.

### Додајте блок

1. Отворите уређивач Shopify теме.
2. Отворите шаблон гдје желите да се број прикаже. На примјер, **Blog** шаблон (листа постова) или секција листе производа.
3. Кликните **Add block** у секцији која рендерује сваку ставку.
4. Под **Apps**, изаберите **FastComments - Comment Count**.
5. Кликните **Save**.

### Подешавања

| Поставка | Шта ради | Подразумевано |
|---|---|---|
| Tenant ID (опционo) | Превладава који FastComments tenant се користи за читање броја. Оставите празно да бисте користили tenant који је аутоматски конфигурисан за продавницу. | (празно) |
| Prilagođeni ID URL-а | Превладава идентификатор странице који се користи за проналажење броја. Користите ово када се број налази на другој страници од FastComments блока који прати. | (аутоматски откривено) |

### Како број одговара нити коментара

The Comment Count block uses the same auto-detection logic as the **FastComments** block:

- Blog post template: `shopify-article-{article.id}`
- Product template: `shopify-product-{product.id}`
- Other templates: the request path

Ако подесите **Custom URL ID** на **FastComments** блоку на некој страници, подесите исти Custom URL ID и на блоку Број коментара тако да оба показују на исту нит.

### Савјети

- Бројеви за све ставке на страници се дохватају у једном захтјеву, па додавање блока на сваку ставку у дугачкој листи не повећава број мрежних захтева.
- Очекује се употреба једног блока Број коментара по чланку или производу у листингу; блок се може додати онолико пута колико вам треба.
