The **FastComments - Recent Discussions** block lists the pages on your store with the most recent comment activity, one entry per page. Use it when you want visitors to discover active threads rather than individual comments.

### Додајте блок

1. Отворите уређивач теме у Shopify-ју.
2. Отворите шаблон у који желите да поставите фид, на пример бочну траку, футер, индекс блога или секцију почетне странице.
3. Кликните **Add block** у секцији која треба да га садржи.
4. У оквиру **Apps** изаберите **FastComments - Recent Discussions**.
5. Кликните **Save**.

### Подешавања

| Поставка | Шта ради | Подразумевано |
|---|---|---|
| Tenant ID (optional) | Override which FastComments tenant the feed reads from. Leave blank to use the store's automatically-configured tenant. | (празно) |
| Number of comments | Колико недавних дискусија приказати. Опсег 1 до 50. | 20 |

### Recent Discussions vs Recent Comments

Оба блока користе исте податке, али приказују различите приказе:

- **Recent Comments**: равна листа најновијих појединачних коментара. Најбоље за "оно што је управо речено".
- **Recent Discussions**: по један унос по страници, уређено по најновијој активности. Најбоље за "које теме су тренутно активне".

Можете користити оба на истој продавници. На пример, Recent Comments у бочној траци блога и Recent Discussions на почетној страници.
