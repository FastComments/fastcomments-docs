`@mentions` l'autocomplétion fonctionne en complétant automatiquement les noms d'utilisateur à partir de diverses sources.

Les utilisateurs affichés dans la liste d'autocomplétion sont déterminés par :

- Les utilisateurs qui ont commenté sur la même page ou le même fil.
- Les utilisateurs qui ont créé leurs comptes FastComments via le même site.
- Les modérateurs du site actuel.
- Les utilisateurs SSO appartenant au tenant actuel.
