Όταν γράφετε, ή επεξεργάζεστε, ένα σχόλιο, απλώς πληκτρολογήστε το σύμβολο `@`, και την αρχή του ονόματος του χρήστη.

    <img src="https://blog.fastcomments.com/images/fc-mentions.png" alt="Παράδειγμα αναφορών" title="Παράδειγμα αναφορών" />
</div>

Απλώς κάντε κλικ στο όνομα του χρήστη που θέλετε να επισημάνετε.

Επιπλέον, μπορείτε να χρησιμοποιήσετε τα πλήκτρα βέλους πάνω και κάτω για να περιηγηθείτε σε αυτή τη λίστα, και να πατήσετε enter για να επιλέξετε, ή escape για να φύγετε.

Γνωρίζουμε ότι ίσως θέλετε να χρησιμοποιήσετε το σύμβολο `@` χωρίς να επισημάνετε κάποιον χρήστη. FastComments θα το εντοπίσει και δεν θα παρεμβαίνει στην εμπειρία πληκτρολόγησής σας.

Σημειώστε ότι το αυτόματο συμπλήρωμα αρχικά προτιμά αντιστοιχίες με διάκριση πεζών-κεφαλαίων, και στη συνέχεια θα επιχειρήσει επίσης να αναζητήσει αποτελέσματα χωρίς διάκριση πεζών-κεφαλαίων.
