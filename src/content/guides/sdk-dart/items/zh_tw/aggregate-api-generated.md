Aggregates documents by grouping them (if groupBy is provided) and applying multiple operations.  
不同的操作（例如 sum、countDistinct、avg 等）受支援。

## Parameters

| 名稱 | 類型 | 位置 | 必填 | 說明 |
|------|------|----------|----------|-------------|
| tenantId | string | 查詢 | 是 |  |
| parentTenantId | string | 查詢 | 否 |  |
| includeStats | boolean | 查詢 | 否 |  |

## Response

回傳：`AggregateResponse`
