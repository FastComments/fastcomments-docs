Это происходит потому, что FastComments не знает, что ваш аккаунт должен быть разрешён для использования на этом домене.

В этом случае решение простое. Вам нужно просто добавить свой сайт в аккаунт на FastComments.com.

[Перейдите по этой ссылке, чтобы добавить свой сайт в аккаунт.](https://fastcomments.com/auth/my-account/configure-domains)

Теперь на ваш сайт должны быть добавлены быстрые, живые комментарии!
