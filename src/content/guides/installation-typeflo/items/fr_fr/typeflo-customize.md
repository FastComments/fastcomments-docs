FastComments est conçu pour être personnalisé afin de s'adapter au design de votre blog.

Si vous souhaitez ajouter des styles personnalisés ou ajuster la configuration, [Consultez notre documentation de personnalisation](/guide-customizations-and-configuration.html) pour savoir comment.

Pour les blogs Typeflo.io, vous pouvez ajouter des styles CSS personnalisés dans la même section Code personnalisé où vous avez ajouté le script FastComments.
