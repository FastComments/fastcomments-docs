O FastComments foi projetado para ser personalizado para combinar com o design do seu blog.

Se você quiser adicionar estilos personalizados ou ajustar a configuração, [Confira nossa Documentação de Personalização](/guide-customizations-and-configuration.html) para saber como.

Para blogs Typeflo.io, você pode adicionar estilos CSS personalizados na mesma seção Código Personalizado onde você adicionou o script do FastComments.
