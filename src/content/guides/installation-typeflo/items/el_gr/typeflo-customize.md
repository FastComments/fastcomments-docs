Το FastComments έχει σχεδιαστεί ώστε να μπορεί να προσαρμοστεί ώστε να ταιριάζει με το σχεδιασμό του ιστολογίου σας.

Εάν θέλετε να προσθέσετε προσαρμοσμένο στιλ ή να τροποποιήσετε τη ρύθμιση, [Δείτε την τεκμηρίωση προσαρμογής μας](/guide-customizations-and-configuration.html) για να μάθετε πώς.

Για ιστολόγια Typeflo.io, μπορείτε να προσθέσετε προσαρμοσμένα στυλ CSS στην ίδια ενότητα Προσαρμοσμένου Κώδικα όπου προσθέσατε το script του FastComments.
