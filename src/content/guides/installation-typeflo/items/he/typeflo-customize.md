FastComments תוכנן כך שניתן להתאים אותו לעיצוב הבלוג שלך.

אם ברצונך להוסיף עיצוב מותאם אישית או לשנות תצורה, [עיין בתיעוד ההתאמות והקונפיגורציה שלנו](/guide-customizations-and-configuration.html) כדי ללמוד כיצד.

עבור בלוגים של Typeflo.io, ניתן להוסיף סגנונות CSS מותאמים באותו אזור Custom Code שבו הוספת את סקריפט FastComments.
