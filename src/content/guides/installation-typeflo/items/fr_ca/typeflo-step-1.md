Pour ajouter FastComments à votre blogue Typeflo.io, accédez d'abord au tableau de bord de votre blogue et cliquez sur **Paramètres** dans la barre latérale gauche.

<div class="screenshot white-bg">
    <div class="title">Cliquez sur Paramètres</div>
    <img class="screenshot-image" src="/images/installation-guides/typeflo-step-1-click-settings.png" alt="Navigate to Settings in Typeflo.io" />
</div>

Ensuite, cliquez sur l'onglet **Code personnalisé** en haut de la page des paramètres.

<div class="screenshot white-bg">
    <div class="title">Cliquez sur l'onglet Code personnalisé</div>
    <img class="screenshot-image" src="/images/installation-guides/typeflo-step-2-click-custom-code-tab.png" alt="Click Custom Code Tab in Typeflo.io Settings" />
</div>

Vous verrez trois sections pour ajouter du code personnalisé : En-tête, Corps et Pied de page. Nous utiliserons la section Pied de page pour FastComments.
