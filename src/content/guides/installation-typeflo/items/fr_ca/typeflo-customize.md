FastComments est conçu pour être personnalisé afin d'harmoniser son apparence avec le design de votre blogue.

Si vous souhaitez ajouter des styles personnalisés ou ajuster la configuration, [Consultez notre documentation sur la personnalisation](/guide-customizations-and-configuration.html) pour savoir comment.

Pour les blogues Typeflo.io, vous pouvez ajouter des styles CSS personnalisés dans la même section Code personnalisé où vous avez ajouté le script FastComments.
