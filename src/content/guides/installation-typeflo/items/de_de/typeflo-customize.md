FastComments ist so konzipiert, dass es an das Design Ihres Blogs angepasst werden kann.

Wenn Sie benutzerdefinierte Styles hinzufügen oder die Konfiguration anpassen möchten, lesen Sie unsere Anpassungsdokumentation [Anpassungsdokumentation](/guide-customizations-and-configuration.html), um zu erfahren, wie.

Für Typeflo.io-Blogs können Sie benutzerdefinierte CSS-Stile im selben 'Custom Code'-Bereich hinzufügen, in dem Sie das FastComments-Skript hinzugefügt haben.
