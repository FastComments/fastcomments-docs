FastComments, blogunuzun tasarımına uyacak şekilde özelleştirilebilir.

Özel stil eklemek veya yapılandırmayı değiştirmek isterseniz, nasıl yapacağınızı öğrenmek için [Özelleştirme Belgelerimizi İnceleyin](/guide-customizations-and-configuration.html).

Typeflo.io blogları için, FastComments betiğini eklediğiniz aynı Özel Kod bölümüne özel CSS stilleri ekleyebilirsiniz.
