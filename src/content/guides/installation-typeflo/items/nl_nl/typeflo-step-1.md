Om FastComments aan je Typeflo.io-blog toe te voegen, ga eerst naar het dashboard van je blog en klik in de linkerzijbalk op **Instellingen**.

<div class="screenshot white-bg">
    <div class="title">Klik op Instellingen</div>
    <img class="screenshot-image" src="/images/installation-guides/typeflo-step-1-click-settings.png" alt="Navigeer naar Instellingen in Typeflo.io" />
</div>

Klik vervolgens op het tabblad **Aangepaste code** bovenaan de instellingenpagina.

<div class="screenshot white-bg">
    <div class="title">Klik op het tabblad Aangepaste code</div>
    <img class="screenshot-image" src="/images/installation-guides/typeflo-step-2-click-custom-code-tab.png" alt="Klik op het tabblad Aangepaste code in de instellingen van Typeflo.io" />
</div>

Je ziet drie secties om aangepaste code toe te voegen: Header, Body en Footer. We gebruiken de Footer-sectie voor FastComments.
