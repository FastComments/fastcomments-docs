Το FastComments έχει σχεδιαστεί ώστε να μπορεί να προσαρμοστεί για να ταιριάζει με τον σχεδιασμό του ιστολογίου σας.

Αν θέλετε να προσθέσετε προσαρμοσμένο στυλ, ή να τροποποιήσετε τη διαμόρφωση, [Δείτε την Τεκμηρίωση Προσαρμογών μας](/guide-customizations-and-configuration.html) για να μάθετε πώς.

Για τα ιστολόγια Typeflo.io, μπορείτε να προσθέσετε προσαρμοσμένα στυλ CSS στην ίδια ενότητα Προσαρμοσμένου Κώδικα όπου προσθέσατε το script του FastComments.
