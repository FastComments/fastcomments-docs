## Google Chrome
### Pour activer les cookies dans Google Chrome (PC) :

1. Sélectionnez l'icône du menu Chrome
2. Sélectionnez Paramètres.
3. En bas de la page, sélectionnez Afficher les paramètres avancés.
4. Dans la section "Confidentialité", sélectionnez Paramètres de contenu.
5. Sélectionnez Autoriser le stockage des données locales (recommandé).
6. Sélectionnez Terminé.

### Pour activer les cookies dans Google Chrome (Mac) :

1. Ouvrez les préférences de Chrome, cliquez sur Paramètres, puis Afficher les paramètres avancés.
2. Sous Confidentialité, cliquez sur Paramètres de contenu.
3. Assurez-vous que "Bloquer les cookies tiers et les données de site" n'est pas coché
4. Si votre navigateur n'est pas répertorié ci-dessus, veuillez consulter les pages d'aide de votre navigateur.

#### Pour activer les cookies dans Google Chrome (Android) :

1. Sur votre appareil Android, ouvrez l'application Chrome.
2. En haut à droite, appuyez sur Plus puis sur Paramètres.
3. Appuyez sur Paramètres du site puis sur Cookies.
4. À côté de "Cookies", activez le paramètre.
5. Pour autoriser les cookies tiers, cochez la case à côté de "Autoriser les cookies tiers."

## Safari
### Pour activer les cookies dans Safari (Mac) :

1. Allez dans le menu déroulant Safari.
2. Sélectionnez Préférences.
3. Cliquez sur Confidentialité dans le panneau supérieur.
4. Sous 'Bloquer les cookies', sélectionnez l'option 'Jamais.'
5. Pour une sécurité accrue, une fois que vous avez fini d'utiliser le site, veuillez remettre le paramètre de confidentialité sur Toujours.
### Pour activer les cookies dans Safari (iPhone/iPad iOS 11) :

1. Ouvrez vos Paramètres.
2. Faites défiler vers le bas et sélectionnez Safari.
3. Sous Confidentialité et sécurité, désactivez "Empêcher le suivi intersite" et "Bloquer tous les cookies".


## Safari iOS
### Pour activer les cookies dans Safari sur iOS :

1. Ouvrez vos Paramètres.
2. Faites défiler vers le bas et sélectionnez Safari.
3. Sous Confidentialité et sécurité, sélectionnez "Bloquer les cookies".
4. Sélectionnez "Toujours autoriser" ou "Autoriser depuis les sites que je visite".

## Mozilla Firefox
### Pour activer les cookies dans Mozilla Firefox (PC) :

1. Cliquez sur Outils > Options.
2. Cliquez sur Confidentialité dans le panneau supérieur.
3. Cochez la case intitulée 'Accepter les cookies des sites.'
4. Cochez la case intitulée 'Accepter les cookies tiers.'
5. Cliquez sur OK.
