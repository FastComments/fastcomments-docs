## Google Chrome
### Чтобы включить файлы cookie в Google Chrome (ПК):

1. Выберите значок меню Chrome
2. Выберите Настройки.
3. Внизу страницы выберите Показать дополнительные настройки.
4. В разделе «Конфиденциальность» выберите Настройки контента.
5. Выберите Разрешить сохранение локальных данных (рекомендуется).
6. Выберите Готово.

### Чтобы включить файлы cookie в Google Chrome (Mac):

1. Откройте настройки Chrome, нажмите Настройки, затем Показать дополнительные настройки.
2. В разделе Конфиденциальность нажмите Настройки контента.
3. Убедитесь, что флажок «Блокировать сторонние файлы cookie и данные сайтов» не установлен
4. Если вашего браузера нет в списке выше, обратитесь к справочным страницам вашего браузера.

#### Чтобы включить файлы cookie в Google Chrome (Android):

1. На устройстве Android откройте приложение Chrome.
2. В правом верхнем углу нажмите «Ещё», затем «Настройки».
3. Нажмите «Настройки сайта», затем «Файлы cookie».
4. Рядом с «Файлы cookie» включите настройку.
5. Чтобы разрешить сторонние файлы cookie, установите флажок рядом с «Разрешить сторонние файлы cookie».

## Safari
### Чтобы включить файлы cookie в Safari (Mac):

1. Перейдите в выпадающее меню Safari.
2. Выберите Настройки.
3. Нажмите «Конфиденциальность» в верхней панели.
4. В разделе «Блокировать файлы cookie» выберите опцию «Никогда».
5. Для повышения безопасности после завершения работы с сайтом измените настройку конфиденциальности обратно на «Всегда».
### Чтобы включить файлы cookie в Safari (iPhone/iPad iOS 11):

1. Откройте Настройки.
2. Прокрутите вниз и выберите Safari.
3. В разделе «Конфиденциальность и безопасность» отключите «Предотвращать отслеживание между сайтами» и «Блокировать все файлы cookie».


## Safari iOS
### Чтобы включить файлы cookie в Safari на iOS:

1. Откройте Настройки.
2. Прокрутите вниз и выберите Safari.
3. В разделе «Конфиденциальность и безопасность» выберите «Блокировать файлы cookie».
4. Выберите «Всегда разрешать» или «Разрешать с посещаемых сайтов».

## Mozilla Firefox
### Чтобы включить файлы cookie в Mozilla Firefox (ПК):

1. Нажмите Инструменты > Параметры.
2. Нажмите «Конфиденциальность» в верхней панели.
3. Установите флажок «Принимать файлы cookie с сайтов».
4. Установите флажок «Принимать сторонние файлы cookie».
5. Нажмите ОК.
