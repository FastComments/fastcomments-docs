## Google Chrome
### So aktivieren Sie Cookies in Google Chrome (PC):

1. Wählen Sie das Chrome-Menüsymbol
2. Wählen Sie Einstellungen.
3. Wählen Sie unten auf der Seite Erweiterte Einstellungen anzeigen.
4. Wählen Sie im Abschnitt "Datenschutz" Inhaltseinstellungen.
5. Wählen Sie Lokale Daten setzen zulassen (empfohlen).
6. Wählen Sie Fertig.

### So aktivieren Sie Cookies in Google Chrome (Mac):

1. Öffnen Sie die Chrome-Einstellungen, klicken Sie auf Einstellungen und dann auf Erweiterte Einstellungen anzeigen.
2. Klicken Sie unter Datenschutz auf Inhaltseinstellungen.
3. Stellen Sie sicher, dass "Cookies und Website-Daten von Drittanbietern blockieren" nicht aktiviert ist
4. Wenn Ihr Browser oben nicht aufgeführt ist, lesen Sie bitte die Hilfeseiten Ihres Browsers.

#### So aktivieren Sie Cookies in Google Chrome (Android):

1. Öffnen Sie auf Ihrem Android-Gerät die Chrome-App.
2. Tippen Sie oben rechts auf Mehr und dann auf Einstellungen.
3. Tippen Sie auf Website-Einstellungen und dann auf Cookies.
4. Aktivieren Sie die Einstellung neben "Cookies".
5. Um Cookies von Drittanbietern zuzulassen, aktivieren Sie das Kontrollkästchen neben "Cookies von Drittanbietern zulassen."

## Safari
### So aktivieren Sie Cookies in Safari (Mac):

1. Gehen Sie zum Safari-Dropdown-Menü.
2. Wählen Sie Einstellungen.
3. Klicken Sie im oberen Bereich auf Datenschutz.
4. Wählen Sie unter 'Cookies blockieren' die Option 'Nie.'
5. Für erhöhte Sicherheit ändern Sie nach der Nutzung der Website die Datenschutzeinstellung wieder auf Immer.
### So aktivieren Sie Cookies in Safari (iPhone/iPad iOS 11):

1. Öffnen Sie Ihre Einstellungen.
2. Scrollen Sie nach unten und wählen Sie Safari.
3. Deaktivieren Sie unter Datenschutz & Sicherheit "Cross-Site-Tracking verhindern" und "Alle Cookies blockieren".


## Safari iOS
### So aktivieren Sie Cookies in Safari auf iOS:

1. Öffnen Sie Ihre Einstellungen.
2. Scrollen Sie nach unten und wählen Sie Safari.
3. Wählen Sie unter Datenschutz & Sicherheit "Cookies blockieren".
4. Wählen Sie "Immer erlauben" oder "Von Websites erlauben, die ich besuche".

## Mozilla Firefox
### So aktivieren Sie Cookies in Mozilla Firefox (PC):

1. Klicken Sie auf Extras > Optionen.
2. Klicken Sie im oberen Bereich auf Datenschutz.
3. Aktivieren Sie das Kontrollkästchen 'Cookies von Websites akzeptieren.'
4. Aktivieren Sie das Kontrollkästchen 'Cookies von Drittanbietern akzeptieren.'
5. Klicken Sie auf OK.
