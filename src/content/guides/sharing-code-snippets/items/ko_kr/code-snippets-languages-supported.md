코드 스니펫을 공유할 때, FastComments는 코드 스니펫이 어떤
프로그래밍 언어인지 자동으로 판단하여 자동으로 서식을 적용하고 구문 하이라이팅을 적용하려고 최선을 다합니다.

FastComments의 구문 하이라이팅은 현재 39개의 일반적인 언어를 지원합니다:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

지원하지 않는 언어로 된 코드 스니펫을 공유하면 코드로는 여전히 렌더링되지만 - 구문 하이라이팅이나 재포맷팅은 적용되지 않습니다.
