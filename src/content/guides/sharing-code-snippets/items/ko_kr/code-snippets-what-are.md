FastComments에서는 코드 스니펫을 단순히 붙여넣기만 하면 자동으로 감지되고 형식화됩니다.

댓글을 작성할 때, 코드 스니펫은 `<code>` 블록으로 감싸지며, 형식화는
댓글을 제출한 후에 이루어집니다.

많은 언어에서 코드 스니펫을 붙여넣을 때 `<code>` 블록으로의 감싸기는 자동으로 이루어집니다.

또는 텍스트 일부를 강조 표시한 다음 도구 모음에서 `Code` 버튼을 클릭할 수 있습니다.
