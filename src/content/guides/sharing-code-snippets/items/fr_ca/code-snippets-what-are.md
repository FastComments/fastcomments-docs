Les extraits de code (ou « snippets ») sont de petits morceaux de code qui peuvent être partagés dans les commentaires.

Avec FastComments, il suffit de coller un extrait de code pour qu'il soit automatiquement détecté et mis en forme.

Lors de la rédaction d'un commentaire, l'extrait de code sera enveloppé dans des blocs `<code>`, et la mise en forme se produit
après l'envoi de votre commentaire.

L'encapsulation dans des blocs `<code>` s'effectue automatiquement lorsque vous collez un extrait de code, pour de nombreux langages.

Sinon, vous pouvez sélectionner un passage de texte et cliquer sur le bouton `Code` dans la barre d'outils.
