Lorsque vous partagez des extraits de code, FastComments fait de son mieux pour déterminer automatiquement dans quel langage se trouve l'extrait afin de le formater automatiquement et d'appliquer la coloration syntaxique.

FastComments syntax highlighting currently supports 39 common languages:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

Lorsque vous partagez un extrait de code dans un langage que nous ne prenons pas en charge, il sera toujours rendu comme du code - mais aucune coloration syntaxique ni aucun reformatage ne sera appliqué.
