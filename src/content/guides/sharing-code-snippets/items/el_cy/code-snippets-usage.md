Για να χρησιμοποιήσετε την υποστήριξη αποσπασμάτων κώδικα, απλά επικολλήστε οποιοδήποτε κομμάτι κώδικα που είναι με εσοχή δύο ή τεσσάρων κενών και περιλαμβάνει τους χαρακτήρες `{` ή `(`.

Εναλλακτικά, επικολλήστε το απόσπασμα κώδικα σας, επιλέξτε το, και κάντε κλικ στην επιλογή `Code` στη γραμμή εργαλείων
