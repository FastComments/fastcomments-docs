Όταν μοιράζεστε αποσπάσματα κώδικα, το FastComments κάνει το καλύτερο δυνατό για να προσδιορίσει αυτόματα σε ποια γλώσσα προγραμματισμού ανήκει το απόσπασμα κώδικα, ώστε να το μορφοποιήσει και να εφαρμόσει αυτόματα επισήμανση σύνταξης.

Η επισήμανση σύνταξης του FastComments υποστηρίζει επί του παρόντος 39 κοινές γλώσσες:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

Όταν μοιράζεστε ένα απόσπασμα κώδικα σε μια γλώσσα που δεν υποστηρίζουμε, θα εξακολουθεί να αποδίδεται ως κώδικας - αλλά δεν θα εφαρμοστεί επισήμανση σύνταξης ή επαναμορφοποίηση.
