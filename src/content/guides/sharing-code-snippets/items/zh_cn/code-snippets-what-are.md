在 FastComments 中，简单粘贴一段代码片段应该会被自动检测并格式化。

在发表评论时，代码片段会被包裹在 `<code>` 块中，格式化会在
您提交评论后发生。

在许多语言中，粘贴代码片段时，包裹到 `<code>` 块中是自动完成的。

或者，您可以高亮选中一段文本，然后在工具栏中单击 `Code` 按钮。
