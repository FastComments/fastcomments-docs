在分享代码片段时，FastComments 会尽力自动判断代码片段所使用的编程语言，以自动格式化并应用语法高亮。

FastComments 的语法高亮当前支持 39 种常见语言：

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

When sharing a code snippet in a language we don't support, it will still be rendered like code -
but no syntax highlighting or reformatting will be applied.
