Når du deler kodeudsnit, gør FastComments sit bedste for automatisk at bestemme, hvilket programmeringssprog kodeudsnittet er i for automatisk at formatere og anvende syntaksfremhævning.

FastComments' syntaksfremhævning understøtter i øjeblikket 39 almindelige sprog:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

Når du deler et kodeudsnit i et sprog, vi ikke understøtter, vil det stadig blive gengivet som kode - men der vil ikke blive anvendt syntaksfremhævning eller reformattering.
