Когда вы делитесь фрагментами кода, FastComments делает все возможное, чтобы автоматически определить, на каком языке написан фрагмент кода, чтобы автоматически форматировать его и применить подсветку синтаксиса.

В настоящее время подсветка синтаксиса FastComments поддерживает 39 распространенных языков:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

Если вы поделитесь фрагментом кода на языке, который мы не поддерживаем, он по-прежнему будет отображаться как код — но подсветка синтаксиса и переразметка применяться не будут.
