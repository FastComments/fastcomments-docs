Фрагменты кода — это небольшие куски кода, отсюда название "snippets" (фрагменты), которыми можно делиться при комментировании.

С FastComments достаточно просто вставить фрагмент кода — он автоматически распознается и отформатируется.

При комментировании фрагмент кода будет обернут в блоки `<code>`, а форматирование происходит
после отправки комментария.

Обертывание в блоки `<code>` происходит автоматически при вставке фрагмента кода во многих языках.

В качестве альтернативы вы можете выделить фрагмент текста и нажать кнопку `Code` на панели инструментов.
