Um die Code-Snippet-Unterstützung zu verwenden, fügen Sie einfach einen beliebigen Codeabschnitt ein, der um zwei oder vier Leerzeichen eingerückt ist und die Zeichen `{` oder `(` enthält.

Alternativ fügen Sie Ihren Codeausschnitt ein, markieren ihn und klicken in der Symbolleiste auf die Option `Code`.
