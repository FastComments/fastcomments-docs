Когато споделяте фрагменти от код, FastComments се стреми автоматично да определи на кой програмен език е кодът, за да го форматира и да приложи синтактично оцветяване.

FastComments в момента поддържа синтактично оцветяване за 39 често използвани програмни езици:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

Когато споделите фрагмент от код на език, който не поддържаме, той все пак ще бъде изобразен като код — но няма да се приложи синтактично оцветяване или преформатиране.
