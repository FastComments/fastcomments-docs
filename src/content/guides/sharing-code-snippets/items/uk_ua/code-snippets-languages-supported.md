Коли ви ділитеся фрагментами коду, FastComments робить усе можливе, щоб автоматично визначити, якою
мовою програмування є фрагмент коду, щоб автоматично відформатувати його та застосувати підсвічування синтаксису.

Наразі підсвічування синтаксису FastComments підтримує 39 поширених мов:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

Коли ви ділитеся фрагментом коду мовою, яку ми не підтримуємо, він все одно відобразиться як код -
але підсвічування синтаксису або переформатування застосовано не буде.
