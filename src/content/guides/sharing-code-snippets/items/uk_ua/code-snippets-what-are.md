У FastComments просте вставлення фрагмента коду має автоматично розпізнаватися й форматуватися.

При коментуванні фрагмент коду буде обгорнуто в блоки `<code>`, а форматування відбувається
після відправлення вашого коментаря.

Обгортання у блоки `<code>` відбувається автоматично при вставленні фрагмента коду для багатьох мов.

Альтернативно, ви можете виділити частину тексту та натиснути кнопку `Code` на панелі інструментів.
