Када делите исјечке кода, FastComments чини све да аутоматски утврди у којем програмском језику је исјечак кода како би га аутоматски форматирао и применио истакнување синтаксе.

FastComments истакнување синтаксе тренутно подржава 39 уобичајених језика:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

When sharing a code snippet in a language we don't support, it will still be rendered like code - but no syntax highlighting or reformatting will be applied.
