コードスニペットを共有する際、FastComments はコードがどのプログラミング言語で書かれているかを自動的に判別し、適切なフォーマットと構文強調表示を適用するよう最善を尽くします。

FastComments の構文ハイライトは現在、39 の一般的な言語をサポートしています:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

サポートしていない言語のコードスニペットを共有した場合でも、コードとしては表示されますが、構文強調表示や再フォーマットは適用されません。
