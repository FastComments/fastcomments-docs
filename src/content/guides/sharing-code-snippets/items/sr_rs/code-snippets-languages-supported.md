Када делите исечке кода, FastComments чини све да аутоматски утврди који
програмски језик је у питању за исечак кода како би их аутоматски форматирао и применио истакњавање синтаксе.

FastComments тренутно за истакњавање синтаксе подржава 39 уобичајених језика:

- .properties
- Apache
- config
- Bash
- C
- C#
- C++
- CSS
- CoffeeScript
- Diff
- Go
- HTML, XML
- HTTP
- JSON
- Java
- JavaScript
- Kotlin
- Less
- Lua
- Makefile
- Markdown
- Nginx
- config
- Objective-C
- PHP
- PHP Template
- Perl
- Plain text
- Python
- Python REPL
- R
- Ruby
- Rust
- SCSS
- SQL
- Shell
- Session
- Swift
- TOML and INI
- TypeScript
- Visual Basic .NET
- YAML

Када делите исечак кода у језику који не подржавамо, он ће и даље бити приказан као код -
али неће бити примењено истакњавање синтаксе или реформатирање.
