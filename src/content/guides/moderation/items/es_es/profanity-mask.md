Por defecto, FastComments incluye una lista negra de aproximadamente ~450 frases y palabras.

Si se encuentra texto de la lista negra en un comentario nuevo, ese texto se reemplazará por asteriscos (***).

La página para personalizar la lista negra se encuentra en Moderar Comments -> Edit Moderation Settings -> Edit Word Blacklist, o accediendo directamente <a href="https://fastcomments.com/auth/my-account/moderate-comments/blacklist" target="_blank">aquí</a>.

[app-screenshot-start url='/auth/my-account/moderate-comments/blacklist?blurList=true'; linkUrl='/auth/my-account/moderate-comments/blacklist'; selector = '.account-block'; title='Comment Word Blacklist' app-screenshot-end]

Además, puedes desactivar la detección de lenguaje ofensivo en la página <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">Editar configuración de moderación</a>.
