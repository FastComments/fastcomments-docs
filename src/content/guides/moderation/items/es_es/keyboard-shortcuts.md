La página de Moderación de Comentarios admite atajos de teclado para navegar entre comentarios, realizar acciones y paginar, todo sin usar el ratón.

Presiona **?** en cualquier momento en la página de moderación para abrir un diálogo de ayuda que enumera todos los atajos disponibles.

## Navegación

| Key | Acción |
|-----|--------|
| `j` | Enfocar el siguiente comentario |
| `k` | Enfocar el comentario anterior |
| `x` or `Enter` | Alternar selección del comentario enfocado |
| `Shift+j` | Seleccionar comentario enfocado y mover al siguiente |
| `Shift+k` | Seleccionar comentario enfocado y mover al anterior |

El comentario enfocado se resalta con un contorno azul. Esto es separado de "seleccionado" (marcado).

## Acciones sobre el comentario enfocado

| Key | Acción |
|-----|--------|
| `a` | Alternar aprobar / desaprobar |
| `s` | Alternar spam / no spam |
| `d` | Eliminar |
| `u` | Restaurar |
| `r` | Alternar revisado / no revisado |
| `e` | Editar |
| `b` | Bloquear / gestionar bloqueo |
| `p` | Alternar fijar / quitar fijación |
| `f` | Quitar marca |
| `i` | Ver todos los comentarios de la IP |
| `t` | Establecer factor de confianza |
| `m` | Gestionar distintivos |
| `l` | Ver registros |
| `c` | Alternar contexto |

Estos atajos solo se aplican cuando un comentario está enfocado. Si no hay ningún comentario enfocado, presiona `j` o `k` primero.

## Atajos globales

| Key | Acción |
|-----|--------|
| `Ctrl+a` | Seleccionar todos los comentarios |
| `Ctrl+z` | Deshacer la última acción |
| `?` | Mostrar ayuda de atajos de teclado |
| `g` | Alternar superposición de guías de atajos |
| `/` | Enfocar el campo de búsqueda |

## Paginación

| Key | Acción |
|-----|--------|
| `[` | Página anterior |
| `]` | Página siguiente |

## Guías de atajos

Presiona `g` para alternar el modo guías. Cuando está activo, aparecen pequeñas insignias en los botones de acción del comentario enfocado que muestran el atajo de teclado de cada botón. Esto es útil mientras aprendes los atajos. Presiona `g` otra vez para ocultarlas.

## Supresión de atajos

Los atajos se suprimen automáticamente cuando:

- El cursor está en el cuadro de búsqueda, en un campo de texto o en un campo editable. Presiona `Escape` para quitar el foco del campo y volver a los atajos.
- Hay un diálogo abierto. Cierra el diálogo primero.
