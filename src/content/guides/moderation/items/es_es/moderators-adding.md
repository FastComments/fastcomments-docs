Para los administradores, en la parte superior de la página de Moderación de Comentarios hay un botón Añadir Moderadores.

[app-screenshot-start url='/auth/my-account/moderate-comments?filter=&text-search=&page=1&count=3&demo=true'; linkUrl='/auth/my-account/moderate-comments'; selector = '.moderation-settings-options'; title='Comment Moderation Setting Buttons' app-screenshot-end]

Si ya tienes moderadores, este botón dirá "Editar Moderadores".

Echemos un vistazo a la página "Añadir un moderador".

[app-screenshot-start url='/auth/my-account/moderate-comments/moderator/new'; selector = '.account-block'; title='The Add a Moderator Page' app-screenshot-end]

Para añadir un moderador, todo lo que se necesita es un nombre y un correo electrónico.

Si el correo electrónico está asociado con una cuenta de FastComments existente, se les enviará una invitación por correo electrónico para unirse a tu cuenta como moderador.

Si el correo electrónico proporcionado no está asociado con una cuenta de FastComments existente, se creará una nueva cuenta para esa persona.

Se enviará un enlace de invitación al moderador, que puede iniciar sesión automáticamente. En el futuro, si desean iniciar sesión, pueden simplemente visitar la
<a href="https://fastcomments.com/auth/login" target="_blank">página de inicio de sesión</a> e introducir el nombre/correo electrónico que proporcionaste anteriormente. Esto les enviará
un enlace para iniciar sesión.

A menos que cierren la sesión, permanecerán conectados durante treinta días.
