Pritisnite **?** kadarkoli na strani za moderiranje, da odprete pogovorno okno s seznamom vseh razpoložljivih bližnjic.

## Navigacija

| Key | Action |
|-----|--------|
| `j` | Osredotoči naslednji komentar |
| `k` | Osredotoči prejšnji komentar |
| `x` or `Enter` | Preklopi izbiro osredotočenega komentarja |
| `Shift+j` | Izberi osredotočeni komentar & premakni na naslednjega |
| `Shift+k` | Izberi osredotočeni komentar & premakni na prejšnjega |

Osredotočeni komentar je poudarjen z modrim robom. To je ločeno od "izbranih" (označenih) komentarjev.

## Dejanja na osredotočenem komentarju

| Key | Action |
|-----|--------|
| `a` | Preklopi odobri / razveljavi odobritev |
| `s` | Preklopi spam / ne-spam |
| `d` | Izbriši |
| `u` | Razveljavi izbris |
| `r` | Preklopi pregledano / nepregledano |
| `e` | Uredi |
| `b` | Prepovej / upravljaj prepoved |
| `p` | Preklopi pripni / odstrani pripenjanje |
| `f` | Odstrani zastavico |
| `i` | Prikaži vse komentarje z IP |
| `t` | Nastavi faktor zaupanja |
| `m` | Upravljaj značke |
| `l` | Ogled dnevnikov |
| `c` | Preklopi kontekst |

Te bližnjice veljajo samo, ko je komentar osredotočen. Če ni osredotočen noben komentar, najprej pritisnite `j` ali `k`.

## Splošne bližnjice

| Key | Action |
|-----|--------|
| `Ctrl+a` | Izberi vse komentarje |
| `Ctrl+z` | Razveljavi zadnje dejanje |
| `?` | Prikaži pomoč z bližnjicami na tipkovnici |
| `g` | Preklopi prekrivni prikaz vodnikov bližnic |
| `/` | Osredotoči iskalno polje |

## Stranjevanje

| Key | Action |
|-----|--------|
| `[` | Prejšnja stran |
| `]` | Naslednja stran |

## Vodniki bližnjic

Pritisnite `g` za preklop načina vodnikov. Ko je aktiven, se na gumbih z dejanji osredotočenega komentarja pojavijo majhne značke, ki prikazujejo bližnjico na tipkovnici za vsak gumb. To je koristno pri učenju bližnjic. Pritisnite `g` znova, da jih skrijete.

## Onemogočanje bližnic

Bližnjice se samodejno onemogočijo, ko:

- Kazalec je v iskalnem polju, besedilnem vnosu ali polju z možnostjo urejanja vsebine. Pritisnite `Escape`, da odvzamete fokus polju in se vrnete k bližnjicam.
- Odprto je pogovorno okno. Najprej ga zaprite.
