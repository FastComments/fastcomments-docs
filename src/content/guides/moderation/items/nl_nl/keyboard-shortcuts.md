De pagina Reactiebeheer ondersteunt sneltoetsen om tussen reacties te navigeren, acties uit te voeren en te pagineren — alles zonder de muis te gebruiken.

Druk op **?** op elk moment op de moderatiepagina om een helpdialoog te openen met alle beschikbare sneltoetsen.

## Navigatie

| Toets | Actie |
|-----|--------|
| `j` | Focus op de volgende reactie |
| `k` | Focus op de vorige reactie |
| `x` of `Enter` | Selectie wisselen van de gefocuste reactie |
| `Shift+j` | Selecteer de gefocuste reactie en ga naar de volgende |
| `Shift+k` | Selecteer de gefocuste reactie en ga naar de vorige |

De gefocuste reactie wordt gemarkeerd met een blauwe rand. Dit staat los van "geselecteerd" (aangevinkte) reacties.

## Acties op de gefocuste reactie

| Toets | Actie |
|-----|--------|
| `a` | Wissel goedkeuren / niet-goedkeuren |
| `s` | Wissel spam / geen spam |
| `d` | Verwijderen |
| `u` | Herstellen |
| `r` | Wissel beoordeeld / niet beoordeeld |
| `e` | Bewerken |
| `b` | Verbannen / verbanning beheren |
| `p` | Wissel vastzetten / loszetten |
| `f` | Vlag verwijderen |
| `i` | Bekijk alle reacties van dit IP-adres |
| `t` | Stel vertrouwensfactor in |
| `m` | Beheer badges |
| `l` | Bekijk logs |
| `c` | Wissel context |

Deze sneltoetsen gelden alleen wanneer een reactie gefocust is. Als er geen reactie gefocust is, druk dan eerst op `j` of `k`.

## Algemene sneltoetsen

| Toets | Actie |
|-----|--------|
| `Ctrl+a` | Selecteer alle reacties |
| `Ctrl+z` | Ongedaan maken van laatste actie |
| `?` | Toon help met sneltoetsen |
| `g` | Schakel overlay met sneltoetswijzers in/uit |
| `/` | Focus op zoekveld |

## Paginering

| Toets | Actie |
|-----|--------|
| `[` | Vorige pagina |
| `]` | Volgende pagina |

## Sneltoetswijzers

Druk op `g` om de wijzermodus in/uit te schakelen. Als deze actief is verschijnen er kleine badges op de actieknoppen van de gefocuste reactie met de sneltoets voor elke knop. Dit is handig bij het leren van de sneltoetsen. Druk nogmaals op `g` om ze te verbergen.

## Onderdrukking van sneltoetsen

Sneltoetsen worden automatisch onderdrukt wanneer:

- De cursor staat in het zoekvak, een tekstveld of een content-editable veld. Druk op `Escape` om de focus uit het veld te halen en terug te keren naar de sneltoetsen.
- Er is een dialoog geopend. Sluit eerst de dialoog.
