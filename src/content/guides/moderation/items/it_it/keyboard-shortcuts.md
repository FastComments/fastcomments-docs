La pagina di moderazione dei commenti supporta scorciatoie da tastiera per navigare tra i commenti, eseguire azioni e paginare - tutto senza usare il mouse.

Premi **?** in qualsiasi momento sulla pagina di moderazione per aprire una finestra di aiuto che elenca tutte le scorciatoie disponibili.

## Navigazione

| Key | Action |
|-----|--------|
| `j` | Metti a fuoco il commento successivo |
| `k` | Metti a fuoco il commento precedente |
| `x` or `Enter` | Attiva/disattiva selezione del commento focalizzato |
| `Shift+j` | Seleziona il commento focalizzato e passa al successivo |
| `Shift+k` | Seleziona il commento focalizzato e torna al precedente |

Il commento focalizzato è evidenziato con un contorno blu. Questo è distinto dai commenti "selezionati" (spuntati).

## Azioni sul commento focalizzato

| Key | Action |
|-----|--------|
| `a` | Alterna approvazione / revoca approvazione |
| `s` | Segna / rimuovi come spam |
| `d` | Elimina |
| `u` | Ripristina |
| `r` | Alterna revisionato / non revisionato |
| `e` | Modifica |
| `b` | Banna / gestisci il ban |
| `p` | Alterna fissaggio / rimuovi fissaggio |
| `f` | Rimuovi segnalazione |
| `i` | Visualizza tutti i commenti dall'IP |
| `t` | Imposta fattore di fiducia |
| `m` | Gestisci badge |
| `l` | Visualizza i log |
| `c` | Alterna contesto |

Queste scorciatoie si applicano solo quando un commento è focalizzato. Se nessun commento è focalizzato, premi prima `j` o `k`.

## Scorciatoie globali

| Key | Action |
|-----|--------|
| `Ctrl+a` | Seleziona tutti i commenti |
| `Ctrl+z` | Annulla l'ultima azione |
| `?` | Mostra l'aiuto delle scorciatoie da tastiera |
| `g` | Attiva/disattiva la sovrapposizione delle guide delle scorciatoie |
| `/` | Metti a fuoco il campo di ricerca |

## Paginazione

| Key | Action |
|-----|--------|
| `[` | Pagina precedente |
| `]` | Pagina successiva |

## Guide delle scorciatoie

Premi `g` per attivare/disattivare la modalità guide. Quando attiva, piccoli badge compaiono sui pulsanti di azione del commento focalizzato mostrando la scorciatoia da tastiera per ogni pulsante. Questo è utile per apprendere le scorciatoie. Premi di nuovo `g` per nasconderle.

## Soppressione delle scorciatoie

Le scorciatoie vengono automaticamente disattivate quando:

- Il cursore si trova nella casella di ricerca, in un campo di testo o in un campo content-editable. Premi `Escape` per togliere il focus dal campo e tornare alle scorciatoie.
- È aperta una finestra di dialogo. Chiudi prima la finestra di dialogo.
