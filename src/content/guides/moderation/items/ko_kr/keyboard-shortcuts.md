댓글 모더레이션 페이지는 댓글 간 이동, 작업 수행 및 페이지 전환을 위한 키보드 단축키를 지원합니다 - 모두 마우스를 사용하지 않고도 가능합니다.

모더레이션 페이지에서 언제든 **?** 를 눌러 사용 가능한 모든 단축키가 나열된 도움말 대화상자를 엽니다.

## 탐색

| 키 | 동작 |
|-----|--------|
| `j` | 다음 댓글에 포커스 |
| `k` | 이전 댓글에 포커스 |
| `x` or `Enter` | 포커스된 댓글 선택/선택 해제 |
| `Shift+j` | 포커스된 댓글 선택 및 다음으로 이동 |
| `Shift+k` | 포커스된 댓글 선택 및 이전으로 이동 |

포커스된 댓글은 파란색 외곽선으로 강조 표시됩니다. 이는 "선택됨"(체크된) 댓글과는 별개입니다.

## 포커스된 댓글에서의 동작

| 키 | 동작 |
|-----|--------|
| `a` | 승인 / 승인 취소 토글 |
| `s` | 스팸 / 스팸 아님 토글 |
| `d` | 삭제 |
| `u` | 삭제 취소 |
| `r` | 검토됨 / 검토 안됨 토글 |
| `e` | 편집 |
| `b` | 차단 / 차단 관리 |
| `p` | 고정 / 고정 해제 토글 |
| `f` | 플래그 해제 |
| `i` | 해당 IP의 모든 댓글 보기 |
| `t` | 신뢰도 설정 |
| `m` | 뱃지 관리 |
| `l` | 로그 보기 |
| `c` | 컨텍스트 토글 |

이 단축키는 댓글에 포커스가 있을 때만 작동합니다. 댓글에 포커스가 없으면 먼저 `j` 또는 `k` 를 누르세요.

## 전역 단축키

| 키 | 동작 |
|-----|--------|
| `Ctrl+a` | 모든 댓글 선택 |
| `Ctrl+z` | 마지막 작업 실행 취소 |
| `?` | 키보드 단축키 도움말 표시 |
| `g` | 단축키 가이드 오버레이 토글 |
| `/` | 검색 입력에 포커스 |

## 페이지네이션

| 키 | 동작 |
|-----|--------|
| `[` | 이전 페이지 |
| `]` | 다음 페이지 |

## 단축키 가이드

가이더 모드를 토글하려면 `g` 를 누르세요. 활성화되면 포커스된 댓글의 동작 버튼에 각 버튼의 키보드 단축키를 표시하는 작은 배지가 나타납니다. 단축키를 익힐 때 유용합니다. 다시 `g` 를 눌러 숨깁니다.

## 입력 억제

다음 상황에서는 단축키가 자동으로 비활성화됩니다:

- 커서가 검색 상자, 텍스트 입력 필드 또는 콘텐츠 편집 가능 필드에 있을 때. 필드 포커스를 해제하고 단축키로 돌아가려면 `Escape` 를 누르세요.
- 대화상자가 열려 있을 때. 먼저 대화상자를 닫으세요.
