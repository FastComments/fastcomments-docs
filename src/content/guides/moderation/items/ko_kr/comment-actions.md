FastComments에서는 각 댓글에 대해 관리자와 중재자가 취할 수 있는 여러 작업이 있습니다. 즉:

- 해당 댓글 **삭제**
- 해당 댓글 **삭제** + 사용자 **차단** (영구 또는 섀도우(Shadow) 차단, 자세한 내용은 나중에)
- 해당 댓글 **편집**
- 해당 댓글의 **투표 수 조정**
- 해당 댓글을 **검토됨** 또는 **미검토**로 표시
- 해당 댓글을 **승인** 또는 **미승인**으로 표시
- 해당 댓글을 **스팸** 또는 **스팸 아님**으로 표시
- 동일한 IP 주소에서 작성된 모든 댓글 보기 *<sup>1</sup>.
- 댓글의 플래그 수 초기화

예를 들어, 댓글을 보관하되 숨기려면 단순히 **미승인**으로 표시하면 됩니다.

또한 관리자와 중재자는 댓글을 고정할 수 있지만, 이는 댓글 스레드에서만 가능합니다.

*<sup>1</sup> 이 기능을 위해 원시 IP 주소를 저장하지 않습니다.
