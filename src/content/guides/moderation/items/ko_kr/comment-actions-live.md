모더레이션 페이지에서 수행되는 모든 작업은 **실시간**입니다.

누군가 댓글 스레드를 보고 있는 상태에서 이전에 승인되지 않았던 댓글을 승인하면, 해당 댓글이 자동으로 그들에게 표시됩니다.

편집, 삭제, 스팸 표시도 마찬가지입니다.

### 실시간 새 댓글 알림

모더레이션 페이지는 현재 필터와 일치하는 새 댓글이 도착하면 알림을 제공합니다. 댓글 목록 상단에 몇 개의 새 댓글이 대기 중인지 표시하는 배너가 나타납니다. 배너를 클릭하면 전체 페이지 새로고침 없이 목록에 인라인으로 로드됩니다.

이 기능은 모든 필터에서 작동합니다 — 예를 들어 스팸 댓글만 보고 있다면 새 스팸 댓글에 대해서만 알림을 받습니다. 특정 구문을 포함하는 댓글을 검색하고 있다면, 일치하는 댓글에 대해서만 알림이 트리거됩니다.

연결이 일시적으로 끊기면, 재연결될 때 시스템이 놓친 모든 댓글을 자동으로 가져옵니다.
