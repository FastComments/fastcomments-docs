Die Kommentar-Moderationsseite unterstützt Tastenkürzel zum Navigieren zwischen Kommentaren, zum Ausführen von Aktionen und zum Blättern - alles ohne Verwendung der Maus.

Drücken Sie **?** jederzeit auf der Moderationsseite, um einen Hilfedialog zu öffnen, der alle verfügbaren Tastenkürzel auflistet.

## Navigation

| Key | Action |
|-----|--------|
| `j` | Nächsten Kommentar fokussieren |
| `k` | Vorherigen Kommentar fokussieren |
| `x` or `Enter` | Auswahl des fokussierten Kommentars umschalten |
| `Shift+j` | Fokussierten Kommentar auswählen & zum nächsten wechseln |
| `Shift+k` | Fokussierten Kommentar auswählen & zum vorherigen wechseln |

Der fokussierte Kommentar wird mit einer blauen Umrandung hervorgehoben. Das ist getrennt von "ausgewählten" (markierten) Kommentaren.

## Aktionen am fokussierten Kommentar

| Key | Action |
|-----|--------|
| `a` | Genehmigen / Genehmigung aufheben umschalten |
| `s` | Als Spam / Kein Spam umschalten |
| `d` | Löschen |
| `u` | Wiederherstellen |
| `r` | Als überprüft / nicht überprüft umschalten |
| `e` | Bearbeiten |
| `b` | Sperren / Sperre verwalten |
| `p` | Anpinnen / Lösen umschalten |
| `f` | Markierung entfernen |
| `i` | Alle Kommentare von dieser IP anzeigen |
| `t` | Vertrauensfaktor setzen |
| `m` | Abzeichen verwalten |
| `l` | Protokolle anzeigen |
| `c` | Kontext umschalten |

Diese Tastenkürzel gelten nur, wenn ein Kommentar fokussiert ist. Wenn kein Kommentar fokussiert ist, drücken Sie zuerst `j` oder `k`.

## Globale Tastenkürzel

| Key | Action |
|-----|--------|
| `Ctrl+a` | Alle Kommentare auswählen |
| `Ctrl+z` | Letzte Aktion rückgängig machen |
| `?` | Hilfe zu Tastenkürzeln anzeigen |
| `g` | Overlay mit Shortcut-Hinweisen umschalten |
| `/` | Suchfeld fokussieren |

## Paginierung

| Key | Action |
|-----|--------|
| `[` | Vorherige Seite |
| `]` | Nächste Seite |

## Shortcut-Hinweise

Drücken Sie `g`, um den Guiders-Modus umzuschalten. Wenn aktiv, erscheinen kleine Abzeichen auf den Aktionsschaltflächen des fokussierten Kommentars, die das Tastenkürzel für jede Schaltfläche anzeigen. Das ist nützlich beim Erlernen der Tastenkürzel. Drücken Sie `g` erneut, um sie auszublenden.

## Eingabeunterdrückung

Tastenkürzel werden automatisch unterdrückt, wenn:

- Der Cursor sich im Suchfeld, in einem Texteingabefeld oder in einem content-editable-Feld befindet. Drücken Sie `Escape`, um das Feld zu entfokussieren und zu den Tastenkürzeln zurückzukehren.
- Ein Dialog geöffnet ist. Schließen Sie zuerst den Dialog.
