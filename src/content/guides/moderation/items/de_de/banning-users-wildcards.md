Es ist möglich, Benutzer, die bestimmte E-Mail-Anbieter verwenden, mithilfe von Platzhaltern zu sperren.

Zum Beispiel, wenn Sie feststellen, dass alle Kommentare von **@bademail.com** Spam sind, können Sie einfach
diesen gesamten E-Mail-Anbieter sperren, indem Sie "*@bademail.com" in das E-Mail-Eingabefeld eingeben, wenn Sie einen gesperrten Benutzer hinzufügen.

Beachten Sie das "*" vor dem @ in der E-Mail.
