Beim Moderieren gibt es manchmal eine Reihe von Aktionen, die wir auf einmal durchführen möchten. Zum Beispiel finden Sie vielleicht eine
Reihe von Kommentaren, die Sie löschen, als Spam markieren, die Freigabe aufheben oder einfach als überprüft kennzeichnen möchten. Anstatt auf "Löschen" und "Bestätigen"
bei jedem Kommentar zu klicken, gibt es einen besseren Weg.

Next to each comment is a checkbox, which we can click to start a bulk action. From there we can pick a bulk action and confirm it.

Sie sehen den Fortschritt, während Ihre Massenaktionen ausgeführt werden, und Sie können mit ctrl+z jede Massenaktion rückgängig machen. Massenaktionen können außerdem
angehalten, gestoppt oder fortgesetzt werden.

### Auswahl über sichtbare Kommentare hinaus

Wenn Sie das Kontrollkästchen "Alles auswählen" aktivieren, wählt es standardmäßig nur die auf der Seite sichtbaren Kommentare aus, bietet Ihnen jedoch
eine Option an, alle Kommentare auf dem Server auszuwählen, die Ihren Filtern entsprechen. Also selbst wenn Sie genehmigen, löschen usw.
Millionen von Kommentaren können Sie mit wenigen Klicks verarbeiten.
