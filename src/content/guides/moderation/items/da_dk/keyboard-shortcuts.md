Kommentarmoderationssiden understøtter tastaturgenveje til navigation mellem kommentarer, udførelse af handlinger og paginering - alt sammen uden brug af musen.

Tryk på **?** når som helst på moderationssiden for at åbne en hjælpedialog, der viser alle tilgængelige genvejstaster.

## Navigation

| Key | Action |
|-----|--------|
| `j` | Flyt fokus til næste kommentar |
| `k` | Flyt fokus til forrige kommentar |
| `x` or `Enter` | Skift markering på den fokuserede kommentar |
| `Shift+j` | Vælg den fokuserede kommentar og gå til næste |
| `Shift+k` | Vælg den fokuserede kommentar og gå til forrige |

Den fokuserede kommentar fremhæves med en blå kant. Dette er adskilt fra "valgte" (afkrydsede) kommentarer.

## Handlinger for den fokuserede kommentar

| Key | Action |
|-----|--------|
| `a` | Skift godkendt / ikke godkendt |
| `s` | Skift spam / ikke spam |
| `d` | Slet |
| `u` | Gendan |
| `r` | Skift gennemgået / ikke gennemgået |
| `e` | Rediger |
| `b` | Bloker / administrer blokering |
| `p` | Skift fastgør / fjern fastgørelse |
| `f` | Fjern flag |
| `i` | Vis alle kommentarer fra IP |
| `t` | Sæt tillidsfaktor |
| `m` | Administrer badges |
| `l` | Vis logfiler |
| `c` | Skift kontekst |

Disse genveje gælder kun, når en kommentar har fokus. Hvis ingen kommentar har fokus, tryk først på `j` eller `k`.

## Globale genveje

| Key | Action |
|-----|--------|
| `Ctrl+a` | Vælg alle kommentarer |
| `Ctrl+z` | Fortryd sidste handling |
| `?` | Vis hjælp til tastaturgenveje |
| `g` | Skift overlayet for genvejsguidere |
| `/` | Sæt fokus i søgefeltet |

## Paginering

| Key | Action |
|-----|--------|
| `[` | Forrige side |
| `]` | Næste side |

## Genvejsguidere

Tryk på `g` for at skifte genvejsguider-tilstand. Når den er aktiv, vises små badges på den fokuserede kommentars handlingsknapper, som viser tastaturgenvejen for hver knap. Dette er nyttigt, mens man lærer genvejene. Tryk på `g` igen for at skjule dem.

## Undertrykkelse af genveje

Genveje undertrykkes automatisk når:

- Markøren er i søgefeltet, et tekstinput eller et content-editable-felt. Tryk på `Escape` for at fjerne fokus fra feltet og vende tilbage til genvejene.
- En dialog er åben. Luk dialogen først.
