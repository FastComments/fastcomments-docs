FastComments автоматически отслеживает подробные события для каждого комментария, чтобы обеспечить прозрачность решений модерации и действий системы. Эти журналы помогают понять, почему комментарий был одобрен, помечен как спам или у него был изменён статус.

Вы можете просмотреть журналы комментариев для отдельных комментариев на панели Moderate Comments, выбрав конкретный комментарий.

## Comment Log Events

Каждый комментарий ведёт журнал событий, происходящих в течение его жизненного цикла. Ниже перечислены типы отслеживаемых событий:

### Anonymization Events
- **Anonymized** - Содержимое комментария было очищено и пользователь помечен как удалённый

### Approval Events
- **ApprovedDueToPastComment** - Комментарий одобрен, потому что у пользователя ранее были одобренные комментарии
- **ApprovedIsAdmin** - Комментарий одобрен, потому что пользователь является администратором
- **NotApprovedRequiresApproval** - Комментарий требует ручного одобрения

### Spam Detection Events
- **IsSpam** - Комментарий помечен как спам механизмом обнаружения
- **IsSpamDueToBadWords** - Комментарий помечен как спам из-за фильтра нецензурной лексики
- **IsSpamFromLLM** - Комментарий помечен как спам AI/LLM движком
- **IsSpamRepeatComment** - Комментарий помечен как спам за повторяемость
- **NotSpamIsOnlyImage** - Комментарий не помечен как спам, потому что содержит только изображения
- **NotSpamIsOnlyReacts** - Комментарий не помечен как спам, потому что содержит только реакции
- **NotSpamNoLinkOrMention** - Комментарий не помечен как спам из‑за отсутствия подозрительных ссылок или упоминаний
- **NotSpamPerfectTrustFactor** - Комментарий не помечен как спам благодаря высокому уровню доверия пользователя
- **NotSpamTooShort** - Комментарий не помечен как спам, потому что он слишком короткий для анализа
- **NotSpamSkipped** - Проверка на спам была пропущена
- **NotSpamFromEngine** - Детектор определил, что комментарий не является спамом

### Bad Words/Profanity Events
- **BadWordsCheckFailed** - При проверке фильтра нецензурной лексики произошла ошибка
- **BadWordsFoundBadPhrase** - Фильтр нецензурной лексики обнаружил неподобающую фразу
- **BadWordsFoundBadWord** - Фильтр нецензурной лексики обнаружил неподобающее слово
- **BadWordsNoDefinitionForLocale** - Нет определений для фильтра нецензурной лексики для языка комментария

### User Verification Events
- **CommentMustBeVerifiedToApproveNotInVerifiedSession** - Комментарий требует верификации, но пользователь не в верифицированной сессии
- **CommentMustBeVerifiedToApproveNotVerifiedYet** - Комментарий требует верификации, но пользователь ещё не верифицирован
- **InVerifiedSession** - Пользователь, публикующий комментарий, находится в верифицированной сессии
- **SentVerificationEmailNoSession** - Письмо для верификации отправлено непроверенному пользователю
- **SentWelcomeEmail** - Новому пользователю отправлено приветственное письмо

### Trust and Security Events
- **TrustFactorChanged** - Коэффициент доверия пользователя был изменён
- **SpamFilterDisabledBecauseAdmin** - Фильтр спама был отключён для администратора
- **TenantSpamFilterDisabled** - Фильтр спама отключён для всего тенанта
- **RepeatCommentCheckIgnored** - Проверка повторных комментариев была пропущена
- **UserIsAdmin** - Пользователь идентифицирован как администратор
- **UserIsAdminParentTenant** - Пользователь идентифицирован как администратор родительского тенанта
- **UserIsAdminViaSSO** - Пользователь идентифицирован как администратор через SSO
- **UserIsMod** - Пользователь идентифицирован как модератор

### Comment Status Changes
- **ExpireStatusChanged** - Статус истечения срока комментария был изменён
- **ReviewStatusChanged** - Статус проверки комментария был изменён
- **SpamStatusChanged** - Статус спама комментария был обновлён
- **ApproveStatusChanged** - Статус одобрения комментария был изменён
- **TextChanged** - Текст комментария был отредактирован
- **VotesChanged** - Число голосов за комментарий было обновлено
- **Flagged** - Комментарий был помечен пользователями
- **UnFlagged** - Метки с комментария были сняты

### Moderation Actions
- **Pinned** - Комментарий был закреплён модератором
- **UnPinned** - Комментарий был откреплён модератором
- **RestoredFromAnonymized** - Комментарий восстановлен из состояния анонимизации

### Notification Events
- **CreatedNotifications** - Для комментария были созданы уведомления
- **NotificationCreateFailure** - Не удалось создать уведомления
- **BadgeAwarded** - Пользователю был присвоен значок за комментарий

### Publishing Events
- **PublishedLive** - Комментарий опубликован для живых подписчиков

### Integration Events
- **WebhookSynced** - Комментарий синхронизирован через webhook

### Spam Rule Events
- **SpamRuleMatch** - Комментарий соответствовал пользовательскому правилу спама

## Accessing Comment Logs

Журналы комментариев генерируются автоматически и хранятся вместе с каждым комментарием. Они предоставляют ценные сведения для:

- Понимания решений модерации
- Отладки проблем с одобрением/определением спама
- Отслеживания шаблонов поведения пользователей
- Аудита действий системы

Эти журналы помогают поддерживать прозрачность процесса модерации и способствуют тонкой настройке поведения вашей системы комментариев.
