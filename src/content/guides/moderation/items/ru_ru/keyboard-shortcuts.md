Страница модерации комментариев поддерживает сочетания клавиш для навигации между комментариями, выполнения действий и постраничной навигации — всё без использования мыши.

Нажмите **?** в любой момент на странице модерации, чтобы открыть диалоговое окно помощи со списком всех доступных сочетаний клавиш.

## Навигация

| Key | Action |
|-----|--------|
| `j` | Установить фокус на следующий комментарий |
| `k` | Установить фокус на предыдущий комментарий |
| `x` or `Enter` | Переключить выбор у сфокусированного комментария |
| `Shift+j` | Выбрать сфокусированный комментарий и перейти к следующему |
| `Shift+k` | Выбрать сфокусированный комментарий и перейти к предыдущему |

Сфокусированный комментарий выделяется синей рамкой. Это отличается от «выбранных» (отмеченных) комментариев.

## Действия над сфокусированным комментарием

| Key | Action |
|-----|--------|
| `a` | Одобрить / снять одобрение |
| `s` | Отметить как спам / снять отметку спам |
| `d` | Удалить |
| `u` | Восстановить |
| `r` | Отметить как просмотренное / снять отметку |
| `e` | Редактировать |
| `b` | Заблокировать / управление баном |
| `p` | Закрепить / открепить |
| `f` | Снять флаг |
| `i` | Просмотреть все комментарии с IP-адреса |
| `t` | Установить уровень доверия |
| `m` | Управление бейджами |
| `l` | Просмотреть логи |
| `c` | Показать/скрыть контекст |

Эти сочетания применимы только когда комментарий сфокусирован. Если ни один комментарий не сфокусирован, сначала нажмите `j` или `k`.

## Глобальные сочетания

| Key | Action |
|-----|--------|
| `Ctrl+a` | Выбрать все комментарии |
| `Ctrl+z` | Отменить последнее действие |
| `?` | Показать справку по сочетаниям клавиш |
| `g` | Переключить наложение подсказок сочетаний клавиш |
| `/` | Установить фокус в поле поиска |

## Пагинация

| Key | Action |
|-----|--------|
| `[` | Предыдущая страница |
| `]` | Следующая страница |

## Подсказки сочетаний

Нажмите `g`, чтобы переключить режим подсказок. Когда он активен, на кнопках действий у сфокусированного комментария появляются небольшие бейджи, показывающие соответствующее сочетание клавиш для каждой кнопки. Это полезно при изучении сочетаний клавиш. Нажмите `g` ещё раз, чтобы скрыть их.

## Подавление ввода

Сочетания клавиш автоматически подавляются когда:

- Курсор находится в поле поиска, текстовом вводе или в редактируемом контентом элементе. Нажмите `Escape`, чтобы убрать фокус с поля и вернуться к сочетаниям клавиш.
- Открыто диалоговое окно. Сначала закройте диалог.
