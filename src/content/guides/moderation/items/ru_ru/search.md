Комментарии можно искать с помощью следующего примера синтаксиса:

- Нечёткий поиск по словам: `cats love`
- Точное совпадение фразы: `I love cats.`
- Точное полное совпадение комментария: `exact="I love cats."`
  - Соответствует только комментариям, весь текст которых точно равен этому значению (с учётом регистра), а не комментариям, которые лишь содержат его.
- По заголовку страницы: `page:"Page Title"`
  - Поддерживает автозаполнение.
- По URL страницы: `page:"https://example.com/some-page"`
  - Поддерживает автозаполнение.
- По сайту/домену: `site:mysite.com` или `domain:othersite.com`
- По пользователю: `user:"Bob"`
  - Поддерживает автозаполнение.

Вы можете поделиться результатами поиска с другими модераторами или администраторами, поделившись URL страницы со страницы модерации. Поле поиска
значение будет включено в URL в вашем браузере после того, как вы нажмёте "Go".
