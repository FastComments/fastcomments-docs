Для администраторов, в верхней части страницы Comment Moderation находится кнопка Add Moderators.

[app-screenshot-start url='/auth/my-account/moderate-comments?filter=&text-search=&page=1&count=3&demo=true'; linkUrl='/auth/my-account/moderate-comments'; selector = '.moderation-settings-options'; title='Comment Moderation Setting Buttons' app-screenshot-end]

Если у вас уже есть модераторы, на этой кнопке будет написано "Edit Moderators".

Давайте посмотрим страницу "Add a Moderator".

[app-screenshot-start url='/auth/my-account/moderate-comments/moderator/new'; selector = '.account-block'; title='The Add a Moderator Page' app-screenshot-end]

Чтобы добавить модератора, достаточно указать имя и адрес электронной почты.

Если адрес электронной почты связан с существующей учетной записью FastComments, этому пользователю будет отправлено приглашение по электронной почте присоединиться к вашей учетной записи в качестве модератора.

Если указанный адрес электронной почты не связан с существующей учетной записью FastComments, для него будет создана новая учетная запись.

Модератору будет отправлена ссылка-приглашение, которая может автоматически выполнить вход. В будущем, если они захотят войти в систему, они могут просто посетить
<a href="https://fastcomments.com/auth/login" target="_blank">страницу входа</a> и ввести имя/электронную почту, которые вы указали ранее. Это отправит им
ссылку для входа.

Если они не выйдут из системы, вход будет сохраняться в течение тридцати дней.
