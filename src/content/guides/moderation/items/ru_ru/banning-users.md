Существует два способа запретить пользователям оставлять комментарии на вашем сайте с помощью FastComments.

Первый — если вы уже знаете их адрес электронной почты, вы можете ввести его на странице <a href="/auth/my-account/moderate-comments/banned-users" target="_blank">заблокированных пользователей</a>.

[app-screenshot-start url='/auth/my-account/moderate-comments/banned-users'; selector = '.content .account-block'; title='The Banned Users Page' app-screenshot-end]

Эту страницу можно открыть через Модерация комментариев -> Заблокированные пользователи

Когда мы блокируем пользователя, можно выбрать тип — либо Постоянный, либо Постоянный теневой бан:

[app-screenshot-start url='/auth/my-account/moderate-comments/banned-users/new'; selector = '.content .account-block'; title='Banning a User' app-screenshot-end]

Второй способ заблокировать пользователя — нажать кнопку бана, расположенную у каждого комментария на странице модерации комментариев.

Когда мы нажимаем кнопку бана, откроются параметры, где можно указать тип и длительность блокировки.

### Теневые баны

Теневой бан — это тип блокировки, при которой пользователю кажется, что его комментарий или голос был успешно сохранён, хотя на самом деле это не так. Это может быть желательным в некоторых ситуациях.

### Блокировка по IP-адресу

Если арендатор не решит отказаться, FastComments поддерживает блокировку по IP, сохраняя хешированную версию IP-адреса комментатора.
