Страницата за модериране на коментари поддържа клавишни комбинации за прелистване между коментари, изпълнение на действия и навигация между страници — всичко това без използване на мишката.

Натиснете **?** по всяко време в страницата за модериране, за да отворите диалог за помощ, който показва всички налични клавишни комбинации.

## Navigation

| Key | Action |
|-----|--------|
| `j` | Фокусирай следващия коментар |
| `k` | Фокусирай предишния коментар |
| `x` or `Enter` | Превключи селекцията за фокусирания коментар |
| `Shift+j` | Избери фокусирания коментар и премести към следващия |
| `Shift+k` | Избери фокусирания коментар и премести към предишния |

Фокусираният коментар е маркиран с син контур. Това е отделно от "selected" (позначени) коментари.

## Actions on Focused Comment

| Key | Action |
|-----|--------|
| `a` | Превключи одобрение / премахни одобрение |
| `s` | Превключи спам / не е спам |
| `d` | Изтрий |
| `u` | Възстанови |
| `r` | Превключи прегледан / непрегледан |
| `e` | Редактирай |
| `b` | Блокирай / управлявай блокирането |
| `p` | Закрепи / Разкрепи |
| `f` | Премахни флага |
| `i` | Прегледай всички коментари от IP адрес |
| `t` | Задай фактор на доверие |
| `m` | Управлявай значки |
| `l` | Прегледай логовете |
| `c` | Превключи контекст |

Тези клавишни комбинации важат само когато коментарът е фокусиран. Ако няма фокусиран коментар, натиснете първо `j` или `k`.

## Global Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+a` | Избери всички коментари |
| `Ctrl+z` | Отмени последното действие |
| `?` | Покажи помощ за клавишни комбинации |
| `g` | Превключи наслагването с указатели за клавишни комбинации |
| `/` | Фокусирай полето за търсене |

## Pagination

| Key | Action |
|-----|--------|
| `[` | Предишна страница |
| `]` | Следваща страница |

## Shortcut Guiders

Натиснете `g`, за да превключите режим на указатели. Когато е активен, върху бутоните за действие на фокусирания коментар се появяват малки значки, показващи клавишната комбинация за всеки бутон. Това е полезно при изучаване на клавишните комбинации. Натиснете отново `g`, за да ги скриете.

## Input Suppression

Клавишните комбинации автоматично се потискат когато:

- Курсорът е в полето за търсене, текстово поле или поле с възможност за редактиране на съдържание. Натиснете `Escape`, за да премахнете фокуса от полето и да се върнете към клавишните комбинации.
- Отворен е диалог. Затворете диалога първо.
