With FastComments, there are a number of actions that Admins and Moderators can take on each comment, namely:

- **Διαγραφή** αυτού του σχολίου
- **Διαγραφή** αυτού του σχολίου + **Αποκλεισμός του χρήστη** (Μόνιμος ή Σκιώδης, περισσότερες λεπτομέρειες αργότερα)
- **Επεξεργασία** αυτού του σχολίου
- **Προσαρμογή των ψήφων** αυτού του σχολίου
- Σημείωση αυτού του σχολίου ως **Ελεγμένο** ή **Μη Ελεγμένο**
- Σημείωση αυτού του σχολίου ως **Εγκεκριμένο** ή **Μη Εγκεκριμένο**
- Σημείωση αυτού του σχολίου ως **Ανεπιθύμητο** ή **Μη Ανεπιθύμητο**
- Προβολή όλων των σχολίων από την ίδια διεύθυνση IP *<sup>1</sup>.
- Επαναφορά του αριθμού σημαιώσεων ενός σχολίου.

For example, to keep a comment around but hide it, we can simply mark it as **Not Approved**.

Additionally, Admins and Moderators can pin comments, but this can only be done from the comment thread itself.

*<sup>1</sup> Δεν αποθηκεύουμε ακατέργαστες διευθύνσεις IP για αυτή τη δυνατότητα.
