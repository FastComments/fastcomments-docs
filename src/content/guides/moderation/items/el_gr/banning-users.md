Υπάρχουν δύο τρόποι για να απαγορεύσετε σε χρήστες να σχολιάζουν στον ιστότοπό σας με το FastComments.

Ο πρώτος είναι, εάν γνωρίζετε ήδη το email τους, να το εισάγετε στη <a href="https://fastcomments.com/auth/my-account/moderate-comments/banned-users" target="_blank">σελίδα αποκλεισμένων χρηστών</a>.

[app-screenshot-start url='/auth/my-account/moderate-comments/banned-users'; selector = '.content .account-block'; title='The Banned Users Page' app-screenshot-end]

Αυτή η σελίδα είναι προσβάσιμη μέσω Διαχείριση Σχολίων -> Αποκλεισμένοι Χρήστες

Όταν προχωράμε στον αποκλεισμό ενός χρήστη, μπορούμε να επιλέξουμε έναν τύπο, είτε Μόνιμος είτε Μόνιμος Σκιώδης Αποκλεισμός:

[app-screenshot-start url='/auth/my-account/moderate-comments/banned-users/new'; selector = '.content .account-block'; title='Banning a User' app-screenshot-end]

Ο δεύτερος τρόπος για να αποκλείσετε έναν χρήστη είναι κάνοντας κλικ στο κουμπί αποκλεισμού που βρίσκεται σε κάθε σχόλιο στη σελίδα Διαχείρισης Σχολίων.

Όταν κάνετε κλικ στο κουμπί αποκλεισμού, θα εμφανιστούν κάποιες επιλογές, όπου μπορούμε να καθορίσουμε τον τύπο και τη διάρκεια του αποκλεισμού.

### Σκιώδεις Αποκλεισμοί

Ένας σκιώδης αποκλεισμός είναι ένας τύπος αποκλεισμού που κάνει να φαίνεται ότι το σχόλιο ή η ψήφος του χρήστη αποθηκεύτηκε επιτυχώς, ενώ στην πραγματικότητα δεν αποθηκεύτηκε. Αυτό μπορεί να θεωρηθεί επιθυμητό σε ορισμένες περιπτώσεις.

### Αποκλεισμός μέσω Διεύθυνσης IP

Εκτός εάν ένας tenant επιθυμεί να εξαιρεθεί, το FastComments υποστηρίζει αποκλεισμούς μέσω IP αποθηκεύοντας μια κατακερματισμένη έκδοση της διεύθυνσης IP του σχολιαστή.
