Όταν κάνετε εποπτεία, μερικές φορές υπάρχει μια σειρά ενεργειών που θέλουμε να εκτελέσουμε ταυτόχρονα. Για παράδειγμα μπορεί να βρείτε μια
σειρά σχολίων που θέλετε να διαγράψετε, να επισημάνετε ως ανεπιθύμητα (spam), να αναιρέσετε την έγκριση, ή απλά να επισημάνετε ως ελεγμένα. Αντί να κάνετε κλικ στο "διαγραφή" και στο "επιβεβαίωση"
σε κάθε σχόλιο, υπάρχει καλύτερος τρόπος.

Δίπλα σε κάθε σχόλιο υπάρχει ένα πλαίσιο επιλογής (checkbox), το οποίο μπορούμε να κάνουμε κλικ για να ξεκινήσουμε μια μαζική ενέργεια. Από εκεί μπορούμε να επιλέξουμε μια μαζική ενέργεια και να την επιβεβαιώσουμε.

Θα βλέπετε την πρόοδο καθώς εκτελούνται οι μαζικές ενέργειες, και μπορείτε να πατήσετε ctrl+z για να αναιρέσετε οποιαδήποτε μαζική ενέργεια. Οι μαζικές ενέργειες μπορούν επίσης
να διακοπούν προσωρινά, να σταματήσουν, ή να συνεχιστούν.

### Επιλογή πέρα από τα ορατά σχόλια

Εάν επιλέξετε το πλαίσιο "Επιλογή όλων" θα επιλέξει κατά προεπιλογή μόνο τα ορατά σχόλια στη σελίδα, αλλά θα
σας παρουσιαστεί μια επιλογή για να επιλέξετε όλα τα σχόλια στον διακομιστή που ταιριάζουν με τα φίλτρα σας. Έτσι, ακόμη και αν χρειάζεται να εγκρίνετε, να διαγράψετε κ.λπ.
εκατομμύρια σχόλια, μπορείτε να το κάνετε με λίγα κλικ.
