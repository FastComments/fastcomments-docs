Από προεπιλογή, το FastComments περιλαμβάνει εκπαιδεύσιμη ανίχνευση ανεπιθύμητης αλληλογραφίας.

Καθώς μετριάζετε σχόλια και τα επισημαίνετε ως **Ανεπιθύμητη**, ή επισημαίνετε σχόλια που ανιχνεύονται αυτόματα ως **Ανεπιθύμητη** ως **Όχι Ανεπιθύμητη**, το σύστημα ανίχνευσης ανεπιθύμητης αλληλογραφίας θα μάθει από αυτές τις ενέργειες για να καθορίζει πιο με ακρίβεια τι θεωρείτε ανεπιθύμητο.

Τα σχόλια που έχουν επισημανθεί ως **Ανεπιθύμητη** δεν θα εγκρίνονται αυτόματα, οπότε δεν θα εμφανίζονται μέχρι να επισημανθούν ρητά ως **Όχι Ανεπιθύμητη**.

Η ανίχνευση ανεπιθύμητης αλληλογραφίας μπορεί να απενεργοποιηθεί μέσω της σελίδας Ρυθμίσεων Ελέγχου Σχολίων.

### Διαφορετικοί ανιχνευτές ανεπιθύμητης αλληλογραφίας

Το FastComments υποστηρίζει τρεις τρόπους ανίχνευσης ανεπιθύμητης αλληλογραφίας:

1. Ένας παραδοσιακός ταξινομητής Naïve-Bayes που εκπαιδεύεται συνεχώς, ο οποίος είναι κοινός σε όλους τους FastComments.com tenants.
2. Ένας παραδοσιακός ταξινομητής Naïve-Bayes που εκπαιδεύεται συνεχώς, ο οποίος είναι **απομονωμένος** στον tenant σας.
3. Χρήση ChatGPT 4.

Όλοι έχουν πρόσβαση στους κοινόχρηστους και στους απομονωμένους ταξινομητές Naïve-Bayes.

Η επιλογή ChatGPT 4 μπορεί να επιλεγεί στη σελίδα Ρυθμίσεων Ελέγχου Σχολίων εάν βρίσκεστε σε Flex billing, καθώς χρεώνεται με βάση τα χρησιμοποιημένα tokens.

### Παράγοντας Εμπιστοσύνης

Το FastComments προσαρμόζει το φίλτρο ανεπιθύμητης αλληλογραφίας για έναν χρήστη με βάση το πόσο τον εμπιστεύονται για τον συγκεκριμένο ιστότοπο.

Για παράδειγμα, εάν οι διαχειριστές έχουν καρφιτσώσει πολλά από τα σχόλιά τους, τότε πιθανότατα είναι πολύ αξιόπιστος χρήστης. Ή, εάν είναι μέλος του ιστότοπου για μεγάλο χρονικό διάστημα και έχουν πολλά σχόλια, τότε ο παράγοντας εμπιστοσύνης τους μπορεί επίσης να είναι υψηλός.

### SSO

Τα σχόλια που δημοσιεύονται από χρήστες SSO μπορεί να θεωρηθούν ανεπιθύμητα και θα ελεγχθούν ως τέτοια. Η εξαίρεση είναι εάν ο χρήστης SSO έχει το ίδιο email με έναν tenant user που έχει ένα ή περισσότερα από τα ακόλουθα δικαιώματα:

- Account Owner
- Super Admin
- Comment Moderator Admin

Χρήστες SSO με αυτά τα δικαιώματα δεν θα ελέγχονται για ανεπιθύμητη αλληλογραφία.

### Επαναλαμβανόμενα Μηνύματα

Το FastComments θα εντοπίσει και θα αποτρέψει επαναλαμβανόμενα μηνύματα. Επίσης θα εντοπίσει επαναλαμβανόμενα μηνύματα που είναι πολύ παρόμοια για να βοηθήσει στην πρόληψη ανεπιθύμητης αλληλογραφίας. Αυτό δεν μπορεί να απενεργοποιηθεί καθώς αποτρέπει την κακόβουλη χρήση της πλατφόρμας μας. Εάν έχετε υψηλό παράγοντα εμπιστοσύνης, αυτό λαμβάνεται υπόψη όταν γίνεται η αποτροπή επαναλαμβανόμενων μηνυμάτων.
