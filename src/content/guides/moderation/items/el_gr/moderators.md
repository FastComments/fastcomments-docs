Οι συντονιστές είναι ειδικοί χρήστες που έχουν πρόσβαση μόνο στη σελίδα Διαχείρισης Σχολίων του λογαριασμού FastComments σας.

Δεν μπορούν να προσθέτουν ή να αφαιρούν άλλους συντονιστές ή να επεξεργάζονται τις ρυθμίσεις διαχείρισης. Έχουν ωστόσο πλήρη έλεγχο στα δεδομένα των σχολίων
που εμφανίζονται στη σελίδα Διαχείρισης Σχολίων και μπορούν να πραγματοποιήσουν όλες τις ενέργειες που μπορεί να κάνει ένας διαχειριστής σε μεμονωμένο σχόλιο. Μπορούν επίσης
να εκτελέσουν μαζικές ενέργειες.

Δεν έχουν πρόσβαση στα εργαλεία διαχείρισης δεδομένων (εισαγωγή/εξαγωγή/μεταφορά σχολίων).
