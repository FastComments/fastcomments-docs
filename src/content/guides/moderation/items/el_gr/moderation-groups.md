Οι συντονιστές μπορούν να τοποθετηθούν σε ομάδες για να εποπτεύουν διαφορετικές σελίδες ή κατηγορίες περιεχομένου.

Όταν ένας Συντονιστής ανήκει σε μία ή περισσότερες ομάδες, θα βλέπει μόνο σχόλια από αυτές τις ομάδες στη σελίδα Διαχείρισης Σχολίων.

Για παράδειγμα, ας υποθέσουμε ότι λειτουργούμε έναν ιστότοπο που εμφανίζει βίντεο κατά κατηγορία. Ίσως θέλουμε να έχουμε διαφορετικούς συντονιστές για βίντεο με Γάτες, Σκύλους και Παπαγάλους, οπότε [ας προσθέσουμε αυτές τις ομάδες](https://fastcomments.com/auth/my-account/moderate-comments/moderation-groups).

[app-screenshot-start url='/auth/my-account/moderate-comments/moderation-groups?demo=true'; linkUrl='/auth/my-account/moderate-comments/moderation-groups'; selector = '.content'; title='The Moderation Groups Page' app-screenshot-end]

Όταν προσθέτουμε έναν συντονιστή, τώρα έχουμε την επιλογή να επιλέξουμε μία ή περισσότερες ομάδες στις οποίες θα ανήκει ο συντονιστής:

[app-screenshot-start url='/auth/my-account/moderate-comments/moderator/new?demo=true'; linkUrl='/auth/my-account/moderate-comments/moderator/new'; selector = '.account-block'; title='Adding A Moderator and Selecting a Group' app-screenshot-end]

Τέλος, τα σχόλια πρέπει να συνδεθούν με μία ή περισσότερες ομάδες ώστε οι κατάλληλοι συντονιστές να τα βλέπουν.

Αυτό μπορεί να ρυθμιστεί [προσθέτοντας μερικές ομάδες](https://fastcomments.com/auth/my-account/moderate-comments/moderation-groups) και στη συνέχεια καθορίζοντας τα αντίστοιχα `Moderation Group` ids στο widget σχολίων,
[όπως περιγράφεται εδώ](/guide-customizations-and-configuration.html#moderation-group-ids).
