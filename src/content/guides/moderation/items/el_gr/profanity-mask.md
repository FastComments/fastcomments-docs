Από προεπιλογή, το FastComments περιλαμβάνει μια λίστα αποκλεισμένων λέξεων με περίπου 450 φράσεις και λέξεις.

Εάν κείμενο από τη λίστα αποκλεισμένων λέξεων βρεθεί σε ένα νέο σχόλιο, το κείμενο αυτό θα αντικατασταθεί με αστερίσκους (***).

Η σελίδα για να προσαρμόσετε τη λίστα αποκλεισμού μπορεί να βρεθεί μέσω Moderate Comments -> Edit Moderation Settings -> Edit Word Blacklist, ή
πηγαίνοντας απευθείας <a href="https://fastcomments.com/auth/my-account/moderate-comments/blacklist" target="_blank">εδώ</a>.

[app-screenshot-start url='/auth/my-account/moderate-comments/blacklist?blurList=true'; linkUrl='/auth/my-account/moderate-comments/blacklist'; selector = '.account-block'; title='Comment Word Blacklist' app-screenshot-end]

Επιπλέον, μπορείτε να απενεργοποιήσετε την Ανίχνευση Βωμολοχίας στη σελίδα <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">Επεξεργασία ρυθμίσεων μετριασμού</a>.
