Η σελίδα Διαχείρισης Σχολίων υποστηρίζει συντομεύσεις πληκτρολογίου για πλοήγηση μεταξύ σχολίων, εκτέλεση ενεργειών και σελιδοποίηση - όλα χωρίς χρήση του ποντικιού.

Πατήστε **?** οποιαδήποτε στιγμή στη σελίδα διαχείρισης σχολίων για να ανοίξετε ένα παράθυρο βοήθειας που απαριθμεί όλες τις διαθέσιμες συντομεύσεις.

## Πλοήγηση

| Key | Action |
|-----|--------|
| `j` | Εστίαση στο επόμενο σχόλιο |
| `k` | Εστίαση στο προηγούμενο σχόλιο |
| `x` or `Enter` | Εναλλαγή επιλογής στο εστιασμένο σχόλιο |
| `Shift+j` | Επιλογή του εστιασμένου σχολίου & μετάβαση στο επόμενο |
| `Shift+k` | Επιλογή του εστιασμένου σχολίου & μετάβαση στο προηγούμενο |

Το εστιασμένο σχόλιο επισημαίνεται με μπλε περίγραμμα. Αυτό είναι ξεχωριστό από τα "επιλεγμένα" (επισημασμένα) σχόλια.

## Ενέργειες στο Εστιασμένο Σχόλιο

| Key | Action |
|-----|--------|
| `a` | Εναλλαγή έγκρισης / μη έγκρισης |
| `s` | Εναλλαγή spam / όχι spam |
| `d` | Διαγραφή |
| `u` | Αναίρεση διαγραφής |
| `r` | Εναλλαγή ελεγμένου / μη ελεγμένου |
| `e` | Επεξεργασία |
| `b` | Αποκλεισμός / διαχείριση αποκλεισμού |
| `p` | Εναλλαγή καρφίτσας / αφαίρεση καρφίτσας |
| `f` | Αφαίρεση επισήμανσης |
| `i` | Προβολή όλων των σχολίων από την IP |
| `t` | Ορισμός συντελεστή εμπιστοσύνης |
| `m` | Διαχείριση διακριτικών |
| `l` | Προβολή αρχείων καταγραφής |
| `c` | Εναλλαγή συμφραζομένων |

Αυτές οι συντομεύσεις ισχύουν μόνο όταν ένα σχόλιο είναι εστιασμένο. Εάν δεν είναι εστιασμένο κανένα σχόλιο, πατήστε πρώτα `j` ή `k`.

## Παγκόσμιες Συντομεύσεις

| Key | Action |
|-----|--------|
| `Ctrl+a` | Επιλογή όλων των σχολίων |
| `Ctrl+z` | Αναίρεση τελευταίας ενέργειας |
| `?` | Εμφάνιση βοήθειας συντομεύσεων πληκτρολογίου |
| `g` | Εναλλαγή επικάλυψης οδηγών συντομεύσεων |
| `/` | Εστίαση στο πεδίο αναζήτησης |

## Σελιδοποίηση

| Key | Action |
|-----|--------|
| `[` | Προηγούμενη σελίδα |
| `]` | Επόμενη σελίδα |

## Οδηγοί Συντομεύσεων

Πατήστε `g` για να ενεργοποιήσετε/απενεργοποιήσετε τη λειτουργία οδηγών. Όταν είναι ενεργή, μικρά διακριτικά εμφανίζονται στα κουμπιά ενεργειών του εστιασμένου σχολίου, δείχνοντας τη συντόμευση πληκτρολογίου για κάθε κουμπί. Αυτό είναι χρήσιμο κατά την εκμάθηση των συντομεύσεων. Πατήστε ξανά `g` για να τα αποκρύψετε.

## Καταστολή Εισόδων

Οι συντομεύσεις απενεργοποιούνται αυτόματα όταν:

- Ο κέρσορας βρίσκεται στο πλαίσιο αναζήτησης, σε ένα πεδίο εισαγωγής κειμένου ή σε ένα content-editable πεδίο. Πατήστε `Escape` για να αποεστιάσετε το πεδίο και να επιστρέψετε στις συντομεύσεις.
- Είναι ανοιχτό ένα παράθυρο διαλόγου. Κλείστε πρώτα το παράθυρο διαλόγου.
