Για τους διαχειριστές, στην κορυφή της σελίδας Διαχείρισης Σχολίων υπάρχει ένα κουμπί 'Προσθήκη Διαχειριστών'.

[app-screenshot-start url='/auth/my-account/moderate-comments?filter=&text-search=&page=1&count=3&demo=true'; linkUrl='/auth/my-account/moderate-comments'; selector = '.moderation-settings-options'; title='Comment Moderation Setting Buttons' app-screenshot-end]

Αν έχετε ήδη διαχειριστές, αυτό το κουμπί θα λέει «Επεξεργασία Διαχειριστών».

Ας ρίξουμε μια ματιά στη σελίδα «Προσθήκη Διαχειριστή».

[app-screenshot-start url='/auth/my-account/moderate-comments/moderator/new'; selector = '.account-block'; title='The Add a Moderator Page' app-screenshot-end]

Για να προσθέσετε έναν διαχειριστή, αρκούν ένα όνομα και μια διεύθυνση email.

Αν η διεύθυνση email είναι συνδεδεμένη με υπάρχον λογαριασμό FastComments, θα τους σταλεί πρόσκληση με email να ενταχθούν στον λογαριασμό σας ως διαχειριστές.

Αν η δοθείσα διεύθυνση email δεν συνδέεται με υπάρχον λογαριασμό FastComments, θα δημιουργηθεί νέος λογαριασμός για αυτούς.

Θα σταλεί στον διαχειριστή ένας σύνδεσμος πρόσκλησης, ο οποίος μπορεί να τους συνδέσει αυτόματα. Στο μέλλον, αν θέλουν να συνδεθούν, μπορούν απλώς να επισκεφθούν τη
<a href="https://fastcomments.com/auth/login" target="_blank">σελίδα σύνδεσης</a> και να εισάγουν το όνομα/τη διεύθυνση email που παρείχατε νωρίτερα. Αυτό θα τους στείλει ένα σύνδεσμο για να συνδεθούν.

Εκτός αν αποσυνδεθούν, θα παραμείνουν συνδεδεμένοι για τριάντα ημέρες.
