Stranica za moderaciju komentara podržava prečace na tastaturi za navigaciju između komentara, izvođenje radnji i paginaciju — sve bez korišćenja miša.

Pritisnite **?** u bilo kom trenutku na stranici za moderaciju da otvorite dijalog pomoći koji prikazuje sve dostupne prečace.

## Navigacija

| Key | Action |
|-----|--------|
| `j` | Fokus na sledeći komentar |
| `k` | Fokus na prethodni komentar |
| `x` or `Enter` | Prebaci selekciju na fokusirani komentar |
| `Shift+j` | Selektuj fokusirani komentar i pomeri se na sledeći |
| `Shift+k` | Selektuj fokusirani komentar i pomeri se na prethodni |

Fokusirani komentar je istaknut plavom ivicom. Ovo je odvojeno od "selected" (čekiranih) komentara.

## Radnje nad fokusiranim komentarom

| Key | Action |
|-----|--------|
| `a` | Prebaci odobri / opozovi odobrenje |
| `s` | Prebaci spam / nije spam |
| `d` | Obriši |
| `u` | Poništi brisanje |
| `r` | Prebaci pregledano / nije pregledano |
| `e` | Uredi |
| `b` | Zabrani / upravljaj zabranom |
| `p` | Prebaci zakači / otkači |
| `f` | Ukloni zastavicu |
| `i` | Pogledaj sve komentare sa IP adrese |
| `t` | Postavi faktor poverenja |
| `m` | Upravljaj bedževima |
| `l` | Pogledaj zapise |
| `c` | Prebaci kontekst |

Ovi prečaci važe samo kada je komentar fokusiran. Ako nijedan komentar nije fokusiran, prvo pritisnite `j` ili `k`.

## Globalni prečaci

| Key | Action |
|-----|--------|
| `Ctrl+a` | Selektuj sve komentare |
| `Ctrl+z` | Poništi poslednju radnju |
| `?` | Prikaži pomoć za prečace na tastaturi |
| `g` | Prebaci overlay sa vodičima prečaca |
| `/` | Fokusiraj polje za pretragu |

## Paginacija

| Key | Action |
|-----|--------|
| `[` | Prethodna stranica |
| `]` | Sledeća stranica |

## Vodiči prečaca

Pritisnite `g` da uključite režim vodiča. Kada je aktivan, male značke se pojave na dugmadima radnji fokusiranog komentara i prikazuju prečac na tastaturi za svako dugme. Ovo je korisno dok učite prečace. Pritisnite `g` ponovo da ih sakrijete.

## Onemogućavanje prečaca pri unosu

Prečaci se automatski onemogućavaju kada:

- Kursor je u polju za pretragu, u tekstualnom inputu ili u polju koje može da se uređuje (content-editable). Pritisnite `Escape` da uklonite fokus sa polja i vratite se na prečace.
- Dijalog je otvoren. Prvo zatvorite dijalog.
