Для администраторов в верхней части страницы Модерации комментариев находится кнопка Добавить модераторов.

[app-screenshot-start url='/auth/my-account/moderate-comments?filter=&text-search=&page=1&count=3&demo=true'; linkUrl='/auth/my-account/moderate-comments'; selector = '.moderation-settings-options'; title='Comment Moderation Setting Buttons' app-screenshot-end]

Если у вас уже есть модераторы, эта кнопка будет называться "Редактировать модераторов".

Давайте посмотрим на страницу "Добавить модератора".

[app-screenshot-start url='/auth/my-account/moderate-comments/moderator/new'; selector = '.account-block'; title='The Add a Moderator Page' app-screenshot-end]

Чтобы добавить модератора, нужно указать только имя и адрес электронной почты.

Если указанный адрес электронной почты связан с существующей учетной записью FastComments, этому пользователю будет отправлено приглашение по электронной почте присоединиться к вашей учетной записи в качестве модератора.

Если указанный адрес электронной почты не связан с существующей учетной записью FastComments, для них будет создана новая учетная запись.

Модератор получит ссылку-приглашение, с помощью которой он сможет автоматически войти в систему. В будущем, если им потребуется войти, они могут просто посетить
<a href="https://fastcomments.com/auth/login" target="_blank">страницу входа</a> и ввести имя/электронную почту, которые вы указали ранее. Это отправит им
ссылку для входа.

Если они не выйдут из системы, они будут оставаться авторизованными в течение тридцати дней.
