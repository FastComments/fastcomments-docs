По умолчанию FastComments содержит чёрный список примерно из ~450 фраз и слов.

Если в новом комментарии обнаруживается текст из чёрного списка, этот текст будет заменён звёздочками (***).

Страницу для настройки чёрного списка можно найти через Модерирование комментариев -> Редактировать настройки модерации -> Редактировать чёрный список слов, или перейдя напрямую <a href="https://fastcomments.com/auth/my-account/moderate-comments/blacklist" target="_blank">здесь</a>.

Кроме того, вы можете отключить обнаружение нецензурной лексики на странице <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">Редактировать настройки модерации</a>.
