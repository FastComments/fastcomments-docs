Комментарии можно искать с помощью следующего примера синтаксиса:

- Нечеткий поиск по словам: `cats love`
- Точное совпадение фразы: `I love cats.`
- Точное полное совпадение комментария: `exact="I love cats."`
  - Соответствует только комментариям, текст которых полностью совпадает с этим значением (с учетом регистра), а не комментариям, которые просто содержат эту фразу.
- По заголовку страницы: `page:"Page Title"`
  - Поддерживает автозаполнение.
- По URL страницы: `page:"https://example.com/some-page"`
  - Поддерживает автозаполнение.
- По сайту/домену: `site:mysite.com` или `domain:othersite.com`
- По пользователю: `user:"Bob"`
  - Поддерживает автозаполнение.

Вы можете делиться результатами поиска с другими модераторами или администраторами, поделившись URL страницы со страницы модерации. Значение поля поиска будет включено в URL в вашем браузере после того, как вы нажмёте «Перейти».
