Поиск комментариев можно выполнить с помощью следующего примера синтаксиса:

- Нечёткий поиск по словам: `cats love`
- Поиск точной фразы: `I love cats.`
- По заголовку страницы: `page:"Page Title"`
  - Поддерживается автозаполнение.
- По URL страницы: `page:"https://example.com/some-page"`
  - Поддерживается автозаполнение.
- По сайту/домену: `site:mysite.com` или `domain:othersite.com`
- По пользователю: `user:"Bob"`
  - Поддерживается автозаполнение.

Вы можете поделиться результатами поиска с другими модераторами или администраторами, отправив URL-адрес страницы со страницы модерации. Поле поиска
значение будет включено в URL в вашем браузере после нажатия "Перейти".
