Есть два способа заблокировать пользователей от комментирования на вашем сайте с помощью FastComments.

Первый — если вы уже знаете их адрес электронной почты, вы можете ввести его на странице <a href="https://fastcomments.com/auth/my-account/moderate-comments/banned-users" target="_blank">Заблокированные пользователи</a>.

[app-screenshot-start url='/auth/my-account/moderate-comments/banned-users'; selector = '.content .account-block'; title='The Banned Users Page' app-screenshot-end]

К этой странице можно перейти через Модерация комментариев -> Заблокированные пользователи

Когда мы блокируем пользователя, можно выбрать тип, либо Постоянный, либо Постоянный теневой бан:

[app-screenshot-start url='/auth/my-account/moderate-comments/banned-users/new'; selector = '.content .account-block'; title='Banning a User' app-screenshot-end]

Второй способ заблокировать пользователя — нажать кнопку бана, которая размещена у каждого комментария на странице Модерация комментариев.

При нажатии кнопки бана появятся параметры, где можно указать тип и длительность блокировки.

### Теневые баны

Теневой бан — это тип блокировки, при котором кажется, что комментарий или голос пользователя были успешно сохранены, тогда как на самом деле это не так. В некоторых ситуациях это может быть желательным.

### Блокировка по IP-адресу

Если тенант не пожелает отказаться, FastComments поддерживает блокировку по IP, сохраняя хешированную версию IP-адреса комментатора.
