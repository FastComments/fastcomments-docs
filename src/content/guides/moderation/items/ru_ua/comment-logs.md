FastComments автоматически отслеживает подробные события для каждого комментария, чтобы обеспечить прозрачность решений модерации и действий системы. Эти журналы помогают понять, почему комментарий был одобрен, помечен как спам или изменил свой статус.

Вы можете просматривать журналы комментариев для отдельных комментариев на панели Moderate Comments, выбрав конкретный комментарий.

## Comment Log Events

Каждый комментарий ведёт журнал событий, происходящих в течение его жизненного цикла. Ниже перечислены типы отслеживаемых событий:

### Anonymization Events
- **Anonymized** - Содержимое комментария было очищено, а пользователь помечен как удалённый

### Approval Events
- **ApprovedDueToPastComment** - Комментарий одобрен, поскольку у пользователя ранее были одобренные комментарии
- **ApprovedIsAdmin** - Комментарий одобрен, поскольку пользователь является администратором
- **NotApprovedRequiresApproval** - Комментарию требуется ручное одобрение

### Spam Detection Events
- **IsSpam** - Комментарий помечен как спам движком обнаружения
- **IsSpamDueToBadWords** - Комментарий помечен как спам из‑за фильтра нецензурной лексики
- **IsSpamFromLLM** - Комментарий помечен как спам AI/LLM‑движком
- **IsSpamRepeatComment** - Комментарий помечен как спам за повторяемость
- **NotSpamIsOnlyImage** - Комментарий не помечен как спам, потому что содержит только изображения
- **NotSpamIsOnlyReacts** - Комментарий не помечен как спам, потому что содержит только реакции
- **NotSpamNoLinkOrMention** - Комментарий не помечен как спам из‑за отсутствия подозрительных ссылок или упоминаний
- **NotSpamPerfectTrustFactor** - Комментарий не помечен как спам из‑за высокого доверия пользователя
- **NotSpamTooShort** - Комментарий не помечен как спам, потому что он слишком короткий для анализа
- **NotSpamSkipped** - Проверка на спам была пропущена
- **NotSpamFromEngine** - Комментарий определён движком обнаружения как не спам

### Bad Words/Profanity Events
- **BadWordsCheckFailed** - Проверка фильтра нецензурной лексики завершилась с ошибкой
- **BadWordsFoundBadPhrase** - Фильтр нецензурной лексики обнаружил неподобающую фразу
- **BadWordsFoundBadWord** - Фильтр нецензурной лексики обнаружил неподобающее слово
- **BadWordsNoDefinitionForLocale** - Для языка комментария отсутствуют определения для фильтра нецензурной лексики

### User Verification Events
- **CommentMustBeVerifiedToApproveNotInVerifiedSession** - Комментарию требуется верификация, но пользователь не верифицирован в сессии
- **CommentMustBeVerifiedToApproveNotVerifiedYet** - Комментарию требуется верификация, но пользователь ещё не верифицирован
- **InVerifiedSession** - Пользователь, публикующий комментарий, находится в верифицированной сессии
- **SentVerificationEmailNoSession** - Письмо для верификации отправлено непроверенному пользователю
- **SentWelcomeEmail** - Приветственное письмо отправлено новому пользователю

### Trust and Security Events
- **TrustFactorChanged** - Показатель доверия пользователя был изменён
- **SpamFilterDisabledBecauseAdmin** - Фильтрация спама обойдена для администратора
- **TenantSpamFilterDisabled** - Фильтрация спама отключена для всего арендатора
- **RepeatCommentCheckIgnored** - Проверка на повтор комментария была пропущена
- **UserIsAdmin** - Пользователь идентифицирован как администратор
- **UserIsAdminParentTenant** - Пользователь идентифицирован как администратор родительского арендатора
- **UserIsAdminViaSSO** - Пользователь идентифицирован как администратор через SSO
- **UserIsMod** - Пользователь идентифицирован как модератор

### Comment Status Changes
- **ExpireStatusChanged** - Статус сроков действия комментария был изменён
- **ReviewStatusChanged** - Статус проверки комментария был изменён
- **SpamStatusChanged** - Статус спама комментария был обновлён
- **ApproveStatusChanged** - Статус одобрения комментария был изменён
- **TextChanged** - Текст комментария был отредактирован
- **VotesChanged** - Количество голосов за комментарий было обновлено
- **Flagged** - Комментарий был помечен пользователями
- **UnFlagged** - Метки флагов комментария были удалены

### Moderation Actions
- **Pinned** - Комментарий был закреплён модератором
- **UnPinned** - Комментарий был откреплён модератором
- **RestoredFromAnonymized** - Комментарий был восстановлен из анонимизированного состояния

### Notification Events
- **CreatedNotifications** - Для комментария были созданы уведомления
- **NotificationCreateFailure** - Не удалось создать уведомления
- **BadgeAwarded** - Пользователю была присвоена награда за комментарий

### Publishing Events
- **PublishedLive** - Комментарий был опубликован для живых подписчиков

### Integration Events
- **WebhookSynced** - Комментарий был синхронизирован через webhook

### Spam Rule Events
- **SpamRuleMatch** - Комментарий совпал с пользовательским правилом спама

## Accessing Comment Logs

Журналы комментариев автоматически создаются и хранятся вместе с каждым комментарием. Они дают ценные сведения для:

- Понимания решений модерации
- Поиска и устранения проблем с одобрением/спамом
- Отслеживания закономерностей поведения пользователей
- Аудита действий системы

Эти журналы помогают поддерживать прозрачность процесса модерации и способствуют тонкой настройке поведения вашей системы комментариев.
