The Comment Moderation page supports keyboard shortcuts for navigating between comments, performing actions, and paginating - all without using the mouse.

Press **?** at any time on the moderation page to open a help dialog listing all available shortcuts.

## Навигация

| Key | Action |
|-----|--------|
| `j` | Focus next comment |
| `k` | Focus previous comment |
| `x` or `Enter` | Toggle select on focused comment |
| `Shift+j` | Select focused comment & move to next |
| `Shift+k` | Select focused comment & move to previous |

Сфокусированный комментарий подсвечивается синей рамкой. Это отличается от «выбранных» (отмеченных) комментариев.

## Действия над сфокусированным комментарием

| Key | Action |
|-----|--------|
| `a` | Toggle approve / unapprove |
| `s` | Toggle spam / not spam |
| `d` | Delete |
| `u` | Undelete |
| `r` | Toggle reviewed / not reviewed |
| `e` | Edit |
| `b` | Ban / manage ban |
| `p` | Toggle pin / unpin |
| `f` | Unflag |
| `i` | View all comments from IP |
| `t` | Set trust factor |
| `m` | Manage badges |
| `l` | View logs |
| `c` | Toggle context |

Эти сочетания клавиш работают только когда комментарий сфокусирован. Если ни один комментарий не сфокусирован, сначала нажмите `j` или `k`.

## Глобальные сочетания клавиш

| Key | Action |
|-----|--------|
| `Ctrl+a` | Select all comments |
| `Ctrl+z` | Undo last action |
| `?` | Show keyboard shortcuts help |
| `g` | Toggle shortcut guiders overlay |
| `/` | Focus search input |

## Навигация по страницам

| Key | Action |
|-----|--------|
| `[` | Previous page |
| `]` | Next page |

## Подсказки сочетаний клавиш

Нажмите `g`, чтобы переключить режим подсказок. Когда он активирован, на кнопках действий сфокусированного комментария появляются небольшие бейджи с соответствующими сочетаниями клавиш. Это полезно при изучении сочетаний. Нажмите `g` снова, чтобы скрыть их.

## Отключение сочетаний клавиш при вводе

Сочетания клавиш автоматически отключаются в следующих случаях:

- Курсор находится в строке поиска, текстовом поле или в content-editable поле. Нажмите `Escape` чтобы снять фокус с поля и вернуться к сочетаниям клавиш.
- Открыт диалог. Сначала закройте диалог.
