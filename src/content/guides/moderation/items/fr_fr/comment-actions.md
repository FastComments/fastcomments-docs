Avec FastComments, il y a un certain nombre d'actions que les Administrateurs et les Modérateurs peuvent effectuer sur chaque commentaire, à savoir :

- **Supprimer** ce commentaire
- **Supprimer** ce commentaire + **Bannir l'utilisateur** (permanent ou shadow, plus de détails ci-dessous)
- **Modifier** ce commentaire
- **Ajuster les votes** de ce commentaire
- Marquer ce commentaire **Révisé** ou **Non révisé**
- Marquer ce commentaire **Approuvé** ou **Non approuvé**
- Marquer ce commentaire comme **Spam** ou **Non spam**
- Voir tous les commentaires provenant de la même adresse IP *<sup>1</sup>.
- Réinitialiser le compteur de signalements d'un commentaire.

Par exemple, pour conserver un commentaire mais le masquer, nous pouvons simplement le marquer comme **Non approuvé**.

De plus, les Administrateurs et les Modérateurs peuvent épingler des commentaires, mais cela ne peut être fait que depuis le fil de commentaires lui-même.

Toutes les actions ci-dessus sont également disponibles via des raccourcis clavier. Voir la section [Raccourcis clavier](#keyboard-shortcuts) pour plus de détails.

*<sup>1</sup> Nous ne stockons pas les adresses IP brutes pour cette fonctionnalité.
