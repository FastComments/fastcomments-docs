Par défaut, FastComments est fourni avec une liste noire d'environ ~450 mots et expressions.

Si un texte présent dans la liste noire est trouvé dans un nouveau commentaire, ce texte sera remplacé par des astérisques (***).

La page pour personnaliser la liste noire se trouve via Modérer les commentaires -> Modifier les paramètres de modération -> Modifier la liste noire de mots, ou en y allant directement <a href="https://fastcomments.com/auth/my-account/moderate-comments/blacklist" target="_blank">ici</a>.

De plus, vous pouvez désactiver la détection des injures sur la page <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">Modifier les paramètres de modération</a>.
