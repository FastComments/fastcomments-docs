All actions taken on the moderation page are **live**.

If someone is viewing the comment thread, and you approve a previously unapproved comment, it will automatically appear for them.

The same goes for editing, deleting, and marking as spam.

### Live New Comment Notifications

The moderation page will also alert you when new comments arrive that match your current filters. A banner appears at the top of the
comment list showing how many new comments are waiting. Clicking the banner loads them inline without a full page refresh.

This works with all filters — if you're viewing only spam comments, you'll only be notified about new spam. If you're searching for
comments containing a specific phrase, only matching comments trigger the notification.

If your connection is temporarily interrupted, the system automatically catches up on any comments you missed when it reconnects.
