Para administradores, no topo da página de Moderação de Comentários há um botão Adicionar Moderadores.

[app-screenshot-start url='/auth/my-account/moderate-comments?filter=&text-search=&page=1&count=3&demo=true'; linkUrl='/auth/my-account/moderate-comments'; selector = '.moderation-settings-options'; title='Comment Moderation Setting Buttons' app-screenshot-end]

Se você já tiver moderadores, este botão dirá "Editar Moderadores".

Vamos dar uma olhada na página "Adicionar um Moderador".

[app-screenshot-start url='/auth/my-account/moderate-comments/moderator/new'; selector = '.account-block'; title='The Add a Moderator Page' app-screenshot-end]

Para adicionar um moderador, tudo o que é necessário é um nome e um e-mail.

Se o e-mail estiver associado a uma conta FastComments existente, a pessoa será convidada por e-mail para entrar na sua conta como moderador.

Se o e-mail fornecido não estiver associado a uma conta FastComments existente, uma nova será criada para essa pessoa.

Um link de convite será enviado ao moderador, que pode efetuar o login automaticamente. No futuro, se desejar fazer login, essa pessoa poderá simplesmente visitar a
<a href="https://fastcomments.com/auth/login" target="_blank">página de login</a> e inserir o nome/e-mail que você forneceu anteriormente. Isso enviará a essa pessoa
um link para efetuar o login.
