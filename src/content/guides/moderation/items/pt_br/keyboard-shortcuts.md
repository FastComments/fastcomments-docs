A página de Moderação de Comentários suporta atalhos de teclado para navegar entre comentários, executar ações e paginar - tudo sem usar o mouse.

Pressione **?** a qualquer momento na página de moderação para abrir um diálogo de ajuda listando todos os atalhos disponíveis.

## Navegação

| Tecla | Ação |
|-----|--------|
| `j` | Focar próximo comentário |
| `k` | Focar comentário anterior |
| `x` or `Enter` | Alternar seleção no comentário em foco |
| `Shift+j` | Selecionar comentário em foco e mover para o próximo |
| `Shift+k` | Selecionar comentário em foco e mover para o anterior |

O comentário em foco é destacado com um contorno azul. Isso é diferente dos comentários "selecionados" (marcados).

## Ações no Comentário em Foco

| Tecla | Ação |
|-----|--------|
| `a` | Alternar aprovar / desaprovar |
| `s` | Alternar spam / não spam |
| `d` | Excluir |
| `u` | Restaurar exclusão |
| `r` | Alternar revisado / não revisado |
| `e` | Editar |
| `b` | Banir / gerenciar banimento |
| `p` | Alternar fixar / desfazer fixação |
| `f` | Remover sinalização |
| `i` | Ver todos os comentários do IP |
| `t` | Definir fator de confiança |
| `m` | Gerenciar badges |
| `l` | Ver logs |
| `c` | Alternar contexto |

Esses atalhos só se aplicam quando um comentário está em foco. Se nenhum comentário estiver em foco, pressione `j` ou `k` primeiro.

## Atalhos Globais

| Tecla | Ação |
|-----|--------|
| `Ctrl+a` | Selecionar todos os comentários |
| `Ctrl+z` | Desfazer última ação |
| `?` | Mostrar ajuda de atalhos de teclado |
| `g` | Alternar sobreposição de guias de atalho |
| `/` | Focar o campo de pesquisa |

## Paginação

| Tecla | Ação |
|-----|--------|
| `[` | Página anterior |
| `]` | Próxima página |

## Guias de Atalho

Pressione `g` para alternar o modo de guias. Quando ativo, pequenas etiquetas aparecem nos botões de ação do comentário em foco mostrando o atalho de teclado para cada botão. Isso é útil ao aprender os atalhos. Pressione `g` novamente para ocultá-los.

## Supressão de Atalhos

Os atalhos são automaticamente suprimidos quando:

- O cursor está na caixa de busca, em um campo de texto ou em um campo editável de conteúdo. Pressione `Escape` para remover o foco do campo e voltar a usar os atalhos.
- Um diálogo está aberto. Feche o diálogo primeiro.
