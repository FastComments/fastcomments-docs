Strona moderacji komentarzy obsługuje skróty klawiaturowe do nawigacji między komentarzami, wykonywania akcji i przechodzenia między stronami - wszystko bez użycia myszy.

Naciśnij **?** w dowolnym momencie na stronie moderacji, aby otworzyć okno pomocy z listą wszystkich dostępnych skrótów.

## Nawigacja

| Klawisz | Akcja |
|-----|--------|
| `j` | Przejdź do następnego komentarza |
| `k` | Przejdź do poprzedniego komentarza |
| `x` lub `Enter` | Przełącz zaznaczenie skupionego komentarza |
| `Shift+j` | Zaznacz skupiony komentarz i przejdź do następnego |
| `Shift+k` | Zaznacz skupiony komentarz i przejdź do poprzedniego |

Skupiony komentarz jest wyróżniony niebieską obwódką. Jest to odrębne od komentarzy „zaznaczonych” (checked).

## Akcje na skupionym komentarzu

| Klawisz | Akcja |
|-----|--------|
| `a` | Przełącz zatwierdź / cofnij zatwierdzenie |
| `s` | Przełącz spam / nie spam |
| `d` | Usuń |
| `u` | Przywróć usunięcie |
| `r` | Przełącz sprawdzony / niesprawdzony |
| `e` | Edytuj |
| `b` | Zablokuj / zarządzaj blokadą |
| `p` | Przełącz przypnij / od przypnij |
| `f` | Usuń flagę |
| `i` | Pokaż wszystkie komentarze z tego IP |
| `t` | Ustaw wskaźnik zaufania |
| `m` | Zarządzaj odznakami |
| `l` | Pokaż logi |
| `c` | Przełącz kontekst |

Te skróty działają tylko, gdy komentarz jest skupiony. Jeśli żaden komentarz nie jest skupiony, najpierw naciśnij `j` lub `k`.

## Globalne skróty

| Klawisz | Akcja |
|-----|--------|
| `Ctrl+a` | Zaznacz wszystkie komentarze |
| `Ctrl+z` | Cofnij ostatnią akcję |
| `?` | Pokaż pomoc dotyczącą skrótów klawiaturowych |
| `g` | Przełącz nakładkę z podpowiedziami skrótów |
| `/` | Ustaw fokus w polu wyszukiwania |

## Stronicowanie

| Klawisz | Akcja |
|-----|--------|
| `[` | Poprzednia strona |
| `]` | Następna strona |

## Podpowiedzi skrótów

Naciśnij `g`, aby przełączyć tryb podpowiedzi. Gdy jest aktywny, małe znaczniki pojawiają się na przyciskach akcji skupionego komentarza, pokazując skrót klawiaturowy dla każdego przycisku. Jest to przydatne podczas nauki skrótów. Naciśnij ponownie `g`, aby je ukryć.

## Tłumienie skrótów

Skróty są automatycznie tłumione, gdy:

- Kursor znajduje się w polu wyszukiwania, polu tekstowym lub w polu content-editable. Naciśnij `Escape`, aby usunąć fokus z pola i powrócić do skrótów.
- Otwarte jest okno dialogowe. Najpierw zamknij dialog.
