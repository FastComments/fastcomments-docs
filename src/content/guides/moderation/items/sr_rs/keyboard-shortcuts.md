Страница за модерацију коментара подржава пречице на тастатури за навигацију између коментара, извођење радњи и пагинацију — све без коришћења миша.

Притисните **?** у било ком тренутку на страници за модерацију да бисте отворили помоћни дијалог са списком свих доступних пречица.

## Навигација

| Key | Action |
|-----|--------|
| `j` | Помери фокус на следећи коментар |
| `k` | Помери фокус на претходни коментар |
| `x` or `Enter` | Пребаци избор на фокусираном коментару |
| `Shift+j` | Одабери фокусиран коментар и помери се на следећи |
| `Shift+k` | Одабери фокусиран коментар и помери се на претходни |

Фокусиран коментар је истакнут плавом ивицом. Ово је одвојено од „изабраних“ (означених) коментара.

## Радње на фокусираном коментару

| Key | Action |
|-----|--------|
| `a` | Пребаци одобри / поништи одобрење |
| `s` | Пребаци спам / није спам |
| `d` | Избриши |
| `u` | Опозови брисање |
| `r` | Пребаци прегледано / није прегледано |
| `e` | Уреди |
| `b` | Забрани / управљај забраном |
| `p` | Пребаци закачено / одкачено |
| `f` | Уклони ознаку |
| `i` | Прикажи све коментаре са IP |
| `t` | Постави фактор поверења |
| `m` | Управљај значкама |
| `l` | Погледај записе |
| `c` | Пребаци контекст |

Ове пречице важе само када је коментар фокусиран. Ако ниједан коментар није фокусиран, прво притисните `j` или `k`.

## Глобалне пречице

| Key | Action |
|-----|--------|
| `Ctrl+a` | Одабери све коментаре |
| `Ctrl+z` | Опозови последњу радњу |
| `?` | Прикажи помоћ за пречице на тастатури |
| `g` | Пребаци прекривач водича пречица |
| `/` | Фокусирај поље за претрагу |

## Пагинација

| Key | Action |
|-----|--------|
| `[` | Претходна страница |
| `]` | Следећа страница |

## Водичи за пречице

Притисните `g` да бисте пребацили режим водича. Када је активан, на дугмадима за радње фокусираног коментара појављују се мале значке које показују пречицу на тастатури за свако дугме. Ово је корисно док учите пречице. Притисните `g` поново да их сакријете.

## Искључивање пречица

Пречице се аутоматски онемогућавају када:

- Курсор је у пољу за претрагу, текстуалном уносу или у content-editable пољу. Притисните `Escape` да уклоните фокус из поља и вратите пречице.
- Дијалог је отворен — прво га затворите.
