Avec FastComments, un commentaire Approuvé est un commentaire **visible**. Pour masquer un commentaire, vous devez le désapprouver.

Les commentaires peuvent être approuvés automatiquement, ou les modérateurs peuvent approuver manuellement chaque commentaire. Il est aussi possible d'exiger seulement l'approbation du premier commentaire d'un utilisateur — auquel cas les commentaires suivants sont approuvés automatiquement et ne nécessitent pas de modération.

FastComments possède le concept de Vérifié vs Non vérifié. Les commentaires Vérifiés sont soit publiés dans une session vérifiée par courriel (l'utilisateur est entièrement connecté, ou utilise SSO) soit ont été publiés dans un état Non vérifié puis vérifiés manuellement par courriel par la suite.

La notion de Vérifié peut être complètement masquée, si souhaité, via des règles de personnalisation.

Exiger l'approbation manuelle des commentaires non vérifiés peut aider contre le spam, car les bots valident rarement leurs commentaires par courriel. Dans ce cas vous voudrez activer l'approbation automatique des commentaires mais activer `Only Auto Approve Verified Comments`.

Tout cela peut être configuré par les administrateurs depuis les [Paramètres de modération](https://fastcomments.com/auth/my-account/moderate-comments/settings).
