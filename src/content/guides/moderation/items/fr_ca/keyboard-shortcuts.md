La page de modération des commentaires prend en charge des raccourcis clavier pour naviguer entre les commentaires, effectuer des actions et paginer — le tout sans utiliser la souris.

Appuyez sur **?** à tout moment sur la page de modération pour ouvrir une boîte d'aide répertoriant tous les raccourcis disponibles.

## Navigation

| Key | Action |
|-----|--------|
| `j` | Focus next comment |
| `k` | Focus previous comment |
| `x` or `Enter` | Toggle select on focused comment |
| `Shift+j` | Select focused comment & move to next |
| `Shift+k` | Select focused comment & move to previous |

Le commentaire focalisé est mis en évidence par un contour bleu. Ceci est distinct des commentaires "sélectionnés" (cochés).

## Actions on Focused Comment

| Key | Action |
|-----|--------|
| `a` | Basculer approuver / désapprouver |
| `s` | Basculer spam / non-spam |
| `d` | Supprimer |
| `u` | Restaurer |
| `r` | Basculer révisé / non révisé |
| `e` | Modifier |
| `b` | Bannir / gérer le bannissement |
| `p` | Épingler / désépingler |
| `f` | Retirer le signalement |
| `i` | Afficher tous les commentaires provenant de l'IP |
| `t` | Définir le facteur de confiance |
| `m` | Gérer les badges |
| `l` | Afficher les journaux |
| `c` | Basculer le contexte |

Ces raccourcis ne s'appliquent que lorsqu'un commentaire est focalisé. Si aucun commentaire n'est focalisé, appuyez d'abord sur `j` ou `k`.

## Global Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+a` | Sélectionner tous les commentaires |
| `Ctrl+z` | Annuler la dernière action |
| `?` | Afficher l'aide des raccourcis clavier |
| `g` | Basculer la superposition des guides de raccourcis |
| `/` | Placer le curseur dans le champ de recherche |

## Pagination

| Key | Action |
|-----|--------|
| `[` | Page précédente |
| `]` | Page suivante |

## Shortcut Guiders

Appuyez sur `g` pour basculer le mode des guides. Lorsqu'il est actif, de petits badges apparaissent sur les boutons d'action du commentaire focalisé, indiquant le raccourci clavier pour chaque bouton. Cela est utile pour apprendre les raccourcis. Appuyez de nouveau sur `g` pour les masquer.

## Input Suppression

Les raccourcis sont automatiquement désactivés lorsque :

- Le curseur se trouve dans la boîte de recherche, un champ de texte ou un champ éditable (contenteditable). Appuyez sur `Escape` pour retirer le focus du champ et revenir aux raccourcis.
- Une boîte de dialogue est ouverte. Fermez d'abord la boîte de dialogue.
