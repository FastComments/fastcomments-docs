---
Avec FastComments, il est possible pour les modérateurs de partager des liens vers le tableau de bord des modérateurs qui mènent directement à une certaine page avec certains filtres.

Par exemple, le cas d'utilisation suivant :

- Le modérateur A accède au tableau de bord de modération des commentaires
- Le modérateur A filtre par Comments Needing Review
- Le modérateur A recherche un commentaire contenant le texte "Mon nom d'entreprise"
- Le modérateur A trouve quelque chose d'intéressant à la deuxième page des résultats de recherche de commentaires.
- Le modérateur A peut **partager le lien** de cette page directement avec le modérateur B.
- Le modérateur B peut voir ce que le modérateur A a fait.
