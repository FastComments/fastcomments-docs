Avec FastComments, un certain nombre d’actions peuvent être effectuées par les administrateurs et modérateurs sur chaque commentaire, à savoir :

- **Supprimer** ce commentaire
- **Supprimer** ce commentaire + **Bannir l’utilisateur** (Permanent ou Shadow, plus de détails plus bas)
- **Modifier** ce commentaire
- **Ajuster les votes** de ce commentaire
- Marquer ce commentaire **Révisé** ou **Non révisé**
- Marquer ce commentaire **Approuvé** ou **Non approuvé**
- Marquer ce commentaire comme **pourriel** ou **non pourriel**
- Afficher tous les commentaires provenant de la même adresse IP *<sup>1</sup>.
- Réinitialiser le nombre de signalements d’un commentaire.

Par exemple, pour conserver un commentaire mais le masquer, nous pouvons simplement le marquer comme **Non approuvé**.

De plus, les administrateurs et modérateurs peuvent épingler des commentaires, mais cela ne peut être fait qu’à partir du fil de discussion du commentaire lui‑même.

Toutes les actions ci‑dessus sont également disponibles via des raccourcis clavier. Voir la section [Raccourcis clavier](#keyboard-shortcuts) pour les détails.

*<sup>1</sup> Nous ne stockons pas les adresses IP brutes pour cette fonctionnalité.
