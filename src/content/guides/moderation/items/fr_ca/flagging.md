Avec FastComments, les utilisateurs peuvent signaler des commentaires, ce qui augmente un compteur interne associé à ce commentaire.

Les modérateurs peuvent voir les commentaires signalés et le nombre de signalements correspondant à un commentaire via la page de modération des commentaires en utilisant le filtre Signalés.

### Masquage automatique des commentaires signalés

Les commentaires peuvent être masqués selon le nombre de fois où ils ont été signalés. Ce seuil peut être configuré dans la page [Paramètres de modération des commentaires](https://fastcomments.com/auth/my-account/moderate-comments/settings).

Voir le paramètre "Désapprouver automatiquement les commentaires signalés".

Par exemple, avec un seuil de 5, le commentaire sera désapprouvé et masqué après avoir été signalé par cinq utilisateurs.

### Retirer les signalements des commentaires

Depuis la page de modération des commentaires, les modérateurs peuvent retirer les signalements des commentaires, ce qui réinitialise le nombre de signalements associé à ce commentaire.

Si le commentaire avait été masqué en raison du nombre de signalements, alors le retrait du signalement fera réapparaître le commentaire.

La réapprobation d'un commentaire qui avait été désapprouvé (masqué) en raison de signalements réinitialisera également le nombre de signalements associé à ce commentaire.
