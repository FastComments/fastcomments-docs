Il est possible d'interdire des utilisateurs provenant de certains fournisseurs de courriel en utilisant des caractères génériques.

Par exemple, si vous constatez que tous les commentaires provenant de **@bademail.com** sont du pourriel, vous pouvez simplement bannir
tout ce fournisseur de courriel en entrant "*@bademail.com" dans le champ d'adresse courriel lors de l'ajout d'un utilisateur banni.

Remarquez le "*" avant le @ dans l'adresse courriel.
