Par défaut, la page de modération affiche tous les commentaires, mais nous pouvons filtrer les commentaires de plusieurs façons :

- Nécessitent une révision
- Nécessitent une approbation
- Signalés comme spam
- Signalés comme non-spam
- Signalés
- Déjà révisés
- Approuvés
- Provenant d'utilisateurs bannis

## Révisés vs Approuvés

Avec FastComments, seuls les commentaires approuvés pourront être affichés. Cela ne signifie pas que vous devez approuver manuellement chaque commentaire.
Par défaut, les commentaires sont approuvés automatiquement. Cela peut être configuré <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">ici</a>.

Donc, Approuvé signifie que le commentaire peut être consulté par les autres utilisateurs de votre site.

Révisé est simplement un outil pour les administrateurs et les modérateurs, et n'a aucun impact sur la visibilité d'un commentaire.

Le drapeau Révisé facilite la recherche des commentaires que vous avez révisés.
