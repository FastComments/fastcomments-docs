דף ניהול התגובות תומך בקיצורי מקלדת לניווט בין תגובות, לביצוע פעולות ולעימוד - הכל ללא שימוש בעכבר.

לחצו על **?** בכל רגע בדף הניהול כדי לפתוח חלון עזרה המפרט את כל הקיצורים הזמינים.

## Navigation

| Key | Action |
|-----|--------|
| `j` | העבר פוקוס לתגובה הבאה |
| `k` | העבר פוקוס לתגובה הקודמת |
| `x` or `Enter` | החלף סימון בתגובה הממוקדת |
| `Shift+j` | סמן את התגובה הממוקדת ועבור לבאה |
| `Shift+k` | סמן את התגובה הממוקדת ועבור לקודמת |

התגובה הממוקדת מוארת בקו מתאר כחול. זה נפרד מ"selected" (תגובות שסומנו).

## Actions on Focused Comment

| Key | Action |
|-----|--------|
| `a` | החלף מאושר / לא מאושר |
| `s` | החלף בין ספאם / לא ספאם |
| `d` | מחק |
| `u` | בטל מחיקה |
| `r` | החלף בין נבדק / לא נבדק |
| `e` | ערוך |
| `b` | חסום / נהל חסימה |
| `p` | החלף נעיצה / הסר נעיצה |
| `f` | הסר דגל |
| `i` | הצג את כל התגובות מאותה כתובת IP |
| `t` | הגדר פקטור אמון |
| `m` | נהל תגיות |
| `l` | הצג יומנים |
| `c` | הצג/הסתר הקשר |

קיצורי הדרך האלה תקפים רק כאשר תגובה ממוקדת. אם אין תגובה ממוקדת, לחצו `j` או `k` קודם.

## Global Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+a` | בחר את כל התגובות |
| `Ctrl+z` | בטל את הפעולה האחרונה |
| `?` | הצג עזרה לקיצורי מקלדת |
| `g` | הצג/הסתר שכבת הכוונה לקיצורי דרך |
| `/` | העבר פוקוס לשדה החיפוש |

## Pagination

| Key | Action |
|-----|--------|
| `[` | עמוד קודם |
| `]` | עמוד הבא |

## Shortcut Guiders

לחצו על `g` כדי להפעיל/לבטל מצב המחוונים. כאשר פעיל, סמלונים קטנים מופיעים על כפתורי הפעולה של התגובה הממוקדת ומציגים את קיצור המקלדת לכל כפתור. זה שימושי בזמן לימוד הקיצורים. לחצו שוב על `g` כדי להסתירם.

## Input Suppression

קיצורי הדרך מדוכאים אוטומטית כאשר:

- הסמן נמצא בתיבת החיפוש, בשדה טקסט או בשדה שניתן לעריכה בתוכן. לחצו על `Escape` כדי לאבד מיקוד מהשדה ולחזור לשימוש בקיצורים.
- חלון דיאלוג פתוח. יש לסגור את החלון קודם.
