Stranica za moderiranje komentara podržava prečice na tastaturi za navigaciju između komentara, izvršavanje radnji i paginaciju - sve bez upotrebe miša.

Pritisnite **?** u bilo kojem trenutku na stranici za moderiranje da otvorite dijalog pomoći koji prikazuje sve dostupne prečice.

## Navigacija

| Key | Action |
|-----|--------|
| `j` | Focus next comment |
| `k` | Focus previous comment |
| `x` or `Enter` | Toggle select on focused comment |
| `Shift+j` | Select focused comment & move to next |
| `Shift+k` | Select focused comment & move to previous |

Fokusirani komentar je istaknut plavim obrubom. Ovo je odvojeno od "selected" (checked) komentara.

## Actions on Focused Comment

| Key | Action |
|-----|--------|
| `a` | Prebaci odobri / poništi odobrenje |
| `s` | Prebaci spam / nije spam |
| `d` | Obriši |
| `u` | Vrati obrisano |
| `r` | Prebaci pregledano / nije pregledano |
| `e` | Uredi |
| `b` | Ban / upravljaj zabranom |
| `p` | Prebaci zakači / otkači |
| `f` | Ukloni oznaku |
| `i` | Prikaži sve komentare sa IP |
| `t` | Postavi faktor povjerenja |
| `m` | Upravljaj značkama |
| `l` | Pregledaj zapise |
| `c` | Prebaci kontekst |

Ove prečice važe samo kada je komentar fokusiran. Ako nijedan komentar nije fokusiran, prvo pritisnite `j` ili `k`.

## Global Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+a` | Select all comments |
| `Ctrl+z` | Undo last action |
| `?` | Show keyboard shortcuts help |
| `g` | Toggle shortcut guiders overlay |
| `/` | Focus search input |

## Pagination

| Key | Action |
|-----|--------|
| `[` | Previous page |
| `]` | Next page |

## Shortcut Guiders

Pritisnite `g` za prebacivanje načina rada vodiča. Kada je aktivno, male značke se pojavljuju na dugmadima za radnje fokusiranog komentara pokazujući prečicu na tastaturi za svako dugme. Ovo je korisno dok učite prečice. Pritisnite `g` ponovo da ih sakrijete.

## Input Suppression

Prečice se automatski onemogućavaju kada:

- Kursor se nalazi u polju za pretragu, tekstualnom unosu ili polju sa sadržajem koji se može uređivati. Pritisnite `Escape` da uklonite fokus sa polja i vratite prečice.
- Otvoren je dijalog. Prvo zatvorite dijalog.
