Οι συντονιστές είναι ειδικοί χρήστες που έχουν μόνο πρόσβαση στη Σελίδα Διαχείρισης Σχολίων του λογαριασμού σας στο FastComments.

Δεν μπορούν να προσθέσουν ή να αφαιρέσουν άλλους συντονιστές ούτε να επεξεργαστούν τις ρυθμίσεις εποπτείας. Ωστόσο, έχουν πλήρη έλεγχο των δεδομένων σχολίων
που εμφανίζονται στη Σελίδα Διαχείρισης Σχολίων και μπορούν να πραγματοποιήσουν όλες τις ενέργειες που μπορεί να εκτελέσει ένας διαχειριστής σε μεμονωμένο σχόλιο. Μπορούν επίσης να
εκτελέσουν μαζικές ενέργειες.

Δεν έχουν πρόσβαση στα εργαλεία διαχείρισης δεδομένων (εισαγωγή/εξαγωγή/μετανάστευση σχολίων).
