Για διαχειριστές, στην κορυφή της σελίδας Comment Moderation υπάρχει ένα κουμπί Add Moderators.

[app-screenshot-start url='/auth/my-account/moderate-comments?filter=&text-search=&page=1&count=3&demo=true'; linkUrl='/auth/my-account/moderate-comments'; selector = '.moderation-settings-options'; title='Comment Moderation Setting Buttons' app-screenshot-end]

Εάν έχετε ήδη διαχειριστές, αυτό το κουμπί θα λέει "Edit Moderators".

Ας ρίξουμε μια ματιά στη σελίδα "Add a Moderator".

[app-screenshot-start url='/auth/my-account/moderate-comments/moderator/new'; selector = '.account-block'; title='The Add a Moderator Page' app-screenshot-end]

Για να προσθέσετε έναν διαχειριστή, το μόνο που χρειάζεται είναι ένα όνομα και μια διεύθυνση email.

Εάν η διεύθυνση email συσχετίζεται με υπάρχον λογαριασμό FastComments, θα τους σταλεί πρόσκληση μέσω email για να ενταχθούν στον λογαριασμό σας ως διαχειριστές.

Εάν η δοθείσα διεύθυνση email δεν συσχετίζεται με υπάρχον λογαριασμό FastComments, θα δημιουργηθεί ένας νέος λογαριασμός για αυτούς.

Θα σταλεί στον διαχειριστή ένας σύνδεσμος πρόσκλησης, ο οποίος μπορεί να τους συνδέσει αυτόματα. Στο μέλλον, αν επιθυμούν να συνδεθούν, μπορούν απλά να επισκεφθούν τη
<a href="https://fastcomments.com/auth/login" target="_blank">σελίδα σύνδεσης</a> και να εισάγουν το όνομα/τη διεύθυνση email που δώσατε νωρίτερα. Αυτό θα τους στείλει
έναν σύνδεσμο για να συνδεθούν.

Εάν δεν αποσυνδεθούν, θα παραμείνουν συνδεδεμένοι για τριάντα ημέρες.
