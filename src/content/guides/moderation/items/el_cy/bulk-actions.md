Όταν κάνετε διαχείριση σχολίων, μερικές φορές υπάρχει μια σειρά ενεργειών που θέλουμε να εκτελέσουμε όλες μαζί. Για παράδειγμα, ίσως βρείτε μια σειρά σχολίων που θέλετε να διαγράψετε, να σημειώσετε ως ανεπιθύμητα (spam), να ανακαλέσετε την έγκριση τους, ή απλώς να τα επισημάνετε ως ελεγμένα. Αντί να κάνετε κλικ στο "διαγραφή" και στην "επιβεβαίωση" για κάθε σχόλιο, υπάρχει ένας καλύτερος τρόπος.

Δίπλα σε κάθε σχόλιο υπάρχει ένα πλαίσιο επιλογής, το οποίο μπορούμε να κάνουμε κλικ για να ξεκινήσουμε μια μαζική ενέργεια. Από εκεί μπορούμε να επιλέξουμε μια μαζική ενέργεια και να την επιβεβαιώσουμε.

Θα βλέπετε την πρόοδο καθώς εκτελούνται οι μαζικές ενέργειές σας, και μπορείτε να πατήσετε ctrl+z για να αναιρέσετε οποιαδήποτε μαζική ενέργεια. Οι μαζικές ενέργειες μπορούν επίσης να τεθούν σε παύση, να σταματήσουν ή να συνεχιστούν.

### Επιλογή πέραν των ορατών σχολίων

Αν επιλέξετε το πλαίσιο "Επιλογή όλων", εξ ορισμού θα επιλέξει μόνο τα ορατά σχόλια στη σελίδα, αλλά θα σας παρουσιαστεί μια επιλογή να επιλέξετε όλα τα σχόλια στον εξυπηρετητή που ταιριάζουν με τα φίλτρα σας. Έτσι, ακόμη και αν χρειάζεται να εγκρίνετε, διαγράψετε κ.λπ. εκατομμύρια σχόλια, μπορείτε να το κάνετε με μερικά κλικ.
