Εξ ορισμού, το FastComments περιλαμβάνει μια λίστα απαγορευμένων λέξεων με ~450 φράσεις και λέξεις.

Αν κείμενο από τη λίστα απαγορευμένων λέξεων βρεθεί σε ένα νέο σχόλιο, το κείμενο αυτό θα αντικατασταθεί με αστερίσκους (***).

Η σελίδα για να προσαρμόσετε τη λίστα απαγορευμένων λέξεων βρίσκεται μέσω Διαχείριση Σχολίων -> Επεξεργασία Ρυθμίσεων Εποπτείας -> Επεξεργασία Λίστας Απαγορευμένων Λέξεων, ή
πηγαίνοντας απευθείας <a href="https://fastcomments.com/auth/my-account/moderate-comments/blacklist" target="_blank">εδώ</a>.

[app-screenshot-start url='/auth/my-account/moderate-comments/blacklist?blurList=true'; linkUrl='/auth/my-account/moderate-comments/blacklist'; selector = '.account-block'; title='Λίστα Απαγορευμένων Λέξεων Σχολίων' app-screenshot-end]

Επιπλέον, μπορείτε να απενεργοποιήσετε την Ανίχνευση Βωμολοχίας στη σελίδα <a href="https://fastcomments.com/auth/my-account/moderate-comments/settings" target="_blank">Επεξεργασία Ρυθμίσεων Εποπτείας</a>.
