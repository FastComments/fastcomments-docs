Η σελίδα Εποπτείας Σχολίων υποστηρίζει συντομεύσεις πληκτρολογίου για πλοήγηση μεταξύ σχολίων, εκτέλεση ενεργειών και σελιδοποίηση - όλα χωρίς χρήση του ποντικιού.

Πατήστε **?** οποιαδήποτε στιγμή στη σελίδα εποπτείας για να ανοίξετε ένα παράθυρο βοήθειας που εμφανίζει όλες τις διαθέσιμες συντομεύσεις.

## Πλοήγηση

| Πλήκτρο | Ενέργεια |
|-----|--------|
| `j` | Εστίαση στο επόμενο σχόλιο |
| `k` | Εστίαση στο προηγούμενο σχόλιο |
| `x` or `Enter` | Εναλλαγή επιλογής στο εστιασμένο σχόλιο |
| `Shift+j` | Επιλογή του εστιασμένου σχολίου & μετακίνηση στο επόμενο |
| `Shift+k` | Επιλογή του εστιασμένου σχολίου & μετακίνηση στο προηγούμενο |

Το εστιασμένο σχόλιο επισημαίνεται με μπλε περίγραμμα. Αυτό είναι ξεχωριστό από τα "επιλεγμένα" (επισημασμένα) σχόλια.

## Ενέργειες στο Εστιασμένο Σχόλιο

| Πλήκτρο | Ενέργεια |
|-----|--------|
| `a` | Εναλλαγή έγκρισης / μη έγκρισης |
| `s` | Εναλλαγή spam / όχι spam |
| `d` | Διαγραφή |
| `u` | Επαναφορά |
| `r` | Εναλλαγή 'ελεγμένο' / 'μη ελεγμένο' |
| `e` | Επεξεργασία |
| `b` | Αποκλεισμός / διαχείριση αποκλεισμού |
| `p` | Εναλλαγή καρφίτσας / αφαίρεση καρφίτσας |
| `f` | Αποσήμανση |
| `i` | Προβολή όλων των σχολίων από τη διεύθυνση IP |
| `t` | Ορισμός παράγοντα εμπιστοσύνης |
| `m` | Διαχείριση διακριτικών |
| `l` | Προβολή αρχείων καταγραφής |
| `c` | Εναλλαγή συμφραζομένων |

Αυτές οι συντομεύσεις ισχύουν μόνο όταν ένα σχόλιο είναι εστιασμένο. Εάν δεν υπάρχει εστιασμένο σχόλιο, πατήστε πρώτα `j` ή `k`.

## Γενικές συντομεύσεις

| Πλήκτρο | Ενέργεια |
|-----|--------|
| `Ctrl+a` | Επιλογή όλων των σχολίων |
| `Ctrl+z` | Αναίρεση τελευταίας ενέργειας |
| `?` | Εμφάνιση βοήθειας για τις συντομεύσεις πληκτρολογίου |
| `g` | Εναλλαγή επικάλυψης οδηγών συντομεύσεων |
| `/` | Εστίαση στο πεδίο αναζήτησης |

## Σελιδοποίηση

| Πλήκτρο | Ενέργεια |
|-----|--------|
| `[` | Προηγούμενη σελίδα |
| `]` | Επόμενη σελίδα |

## Οδηγοί Συντομεύσεων

Πατήστε `g` για να εναλλάξετε τη λειτουργία οδηγών. Όταν είναι ενεργή, μικρά σήματα εμφανίζονται στα κουμπιά ενεργειών του εστιασμένου σχολίου που δείχνουν τη συντόμευση πληκτρολογίου για κάθε κουμπί. Αυτό είναι χρήσιμο κατά την εκμάθηση των συντομεύσεων. Πατήστε ξανά `g` για να τα αποκρύψετε.

## Καταστολή Εισόδου

Οι συντομεύσεις καταστέλλονται αυτόματα όταν:

- Ο κέρσορας βρίσκεται στο πλαίσιο αναζήτησης, σε ένα πεδίο κειμένου, ή σε ένα στοιχείο content-editable. Πατήστε `Escape` για να χάσει το πεδίο την εστίασή του και να επιστρέψετε στις συντομεύσεις.
- Ένας διάλογος είναι ανοιχτός. Κλείστε πρώτα τον διάλογο.
