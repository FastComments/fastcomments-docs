Οι Συντονιστές μπορούν να τοποθετηθούν σε ομάδες για να επιβλέπουν διαφορετικές σελίδες ή κατηγορίες περιεχομένου.

Όταν ένας Συντονιστής ανήκει σε μία ή περισσότερες ομάδες, θα βλέπει μόνο σχόλια από αυτές τις ομάδες στη σελίδα Διαχείρισης Σχολίων.

Για παράδειγμα, ας πούμε ότι τρέχουμε έναν ιστότοπο που εμφανίζει βίντεο κατά κατηγορία. Ίσως θέλουμε να έχουμε διαφορετικούς συντονιστές για βίντεο με Γάτες, Σκύλους και Παπαγάλους, οπότε [ας προσθέσουμε αυτές τις ομάδες](https://fastcomments.com/auth/my-account/moderate-comments/moderation-groups).

[app-screenshot-start url='/auth/my-account/moderate-comments/moderation-groups?demo=true'; linkUrl='/auth/my-account/moderate-comments/moderation-groups'; selector = '.content'; title='The Moderation Groups Page' app-screenshot-end]

Όταν προσθέτουμε έναν συντονιστή, έχουμε πλέον την επιλογή να επιλέξουμε μία ή περισσότερες ομάδες στις οποίες θα ανήκει ο συντονιστής:

[app-screenshot-start url='/auth/my-account/moderate-comments/moderator/new?demo=true'; linkUrl='/auth/my-account/moderate-comments/moderator/new'; selector = '.account-block'; title='Adding A Moderator and Selecting a Group' app-screenshot-end]

Τέλος, τα σχόλια πρέπει να συσχετιστούν με μία ή περισσότερες ομάδες ώστε οι σωστοί συντονιστές να τα βλέπουν.

Αυτό μπορεί να ρυθμιστεί με το [προσθέτοντας κάποιες ομάδες](https://fastcomments.com/auth/my-account/moderate-comments/moderation-groups) και στη συνέχεια προσδιορίζοντας τα αντίστοιχα `Moderation Group` ids στο widget σχολίων,
[όπως υποδεικνύεται εδώ](/guide-customizations-and-configuration.html#moderation-group-ids).
