在 FastComments 中，管理員和版主可以對每則評論採取多種操作，具體包括：

- **刪除** 該評論
- **刪除** 該評論 + **封鎖用戶**（永久或隱形，詳情稍後）
- **編輯** 該評論
- **調整該評論的票數**
- 將該評論標記為 **已審核** 或 **未審核**
- 將該評論標記為 **已核准** 或 **未核准**
- 將該評論標記為 **垃圾** 或 **非垃圾**
- 檢視來自相同 IP address 的所有評論 *<sup>1</sup>.
- 重設該評論的檢舉計數。

例如，若想保留一則評論但將其隱藏，我們可以簡單地將其標記為 **未核准**。

此外，管理員和版主可以將評論置頂，但只能在該評論的留言串中執行。

*<sup>1</sup> 我們不會為此功能儲存原始 ip addresses。
