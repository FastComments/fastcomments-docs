一部分的審核操作可以直接在評論串中執行，不需要前往「評論審核」頁面。

當您已登入時，點選評論右上角的編輯按鈕。作為審核人員，您應該會看到以下選項：

- **置頂** 該評論
- **刪除** 該評論
- **刪除** 該評論 + **封鎖使用者**（永久或隱形，詳情稍後）
- **編輯** 該評論
- 將該評論標記為 **已批准**（顯示）或 **未批准**（隱藏）
- 將該評論標記為 **垃圾** 或 **非垃圾**

### 關閉評論串

版主與管理員如果已登入，可以在評論區上方的三點選單中選擇 `Close Thread` 來鎖定或關閉評論串。之後任何時間都可以選擇 `Re-Open Thread` 來重新開放評論功能。

關閉評論串會阻止新增評論，但仍允許投票，且使用者仍可刪除自己的評論（若他們想刪除）。

關閉或重新開啟評論串會立即影響所有正在檢視該串的使用者。

您也可以透過為該頁面建立專門的自訂規則，將討論串標記為唯讀，這樣同時會移除投票與刪除選項。

### 即時更新

所有這些操作會即時更新其他使用者的評論串，無需他們重新載入頁面。然而，像是隱藏評論或標記為垃圾的審核動作，不會將該評論從**版主的**畫面上移除，以便他們在需要時可以快速還原該操作。為了表示該評論被隱藏，它會相較於其他評論以不同方式標示（標示顏色取決於移除的原因）。

例如，給定使用者 `A (commenter)`, `B (Moderator 1)`, 和 `C (Moderator 2)`。

...以及下列情景：

1. `User B (Moderator 1)` 隱藏了一則評論。
2. 對於 `User A (commenter)`，該評論會立即被隱藏。
3. 對於 `User C (Moderator 2)`，該評論會立即被隱藏。
4. 對於執行變更的使用者 `User B (Moderator 1)`，該評論仍會保留在他們的畫面上，但會以已移除的方式標示。他們可以選擇還原該操作，若還原則其他使用者會再次即時看到更新。
