Сторінка модерації коментарів підтримує клавіатурні скорочення для переміщення між коментарями, виконання дій, і пагінації - усе без використання миші.

Натисніть **?** в будь-який час на сторінці модерації, щоб відкрити діалог довідки зі списком усіх доступних скорочень.

## Навігація

| Key | Action |
|-----|--------|
| `j` | Перемістити фокус на наступний коментар |
| `k` | Перемістити фокус на попередній коментар |
| `x` or `Enter` | Перемкнути вибір для сфокусованого коментаря |
| `Shift+j` | Вибрати сфокусований коментар & перейти до наступного |
| `Shift+k` | Вибрати сфокусований коментар & перейти до попереднього |

Сфокусований коментар підсвічується синьою обводкою. Це відрізняється від "вибраних" (позначених) коментарів.

## Дії над сфокусованим коментарем

| Key | Action |
|-----|--------|
| `a` | Затвердити / скасувати затвердження |
| `s` | Позначити як спам / зняти позначку спаму |
| `d` | Видалити |
| `u` | Відновити |
| `r` | Позначити як переглянутий / зняти позначку перегляду |
| `e` | Редагувати |
| `b` | Заблокувати / керувати блокуванням |
| `p` | Закріпити / відкріпити |
| `f` | Зняти позначку |
| `i` | Переглянути всі коментарі з цієї IP-адреси |
| `t` | Встановити рівень довіри |
| `m` | Керувати бейджами |
| `l` | Переглянути журнали |
| `c` | Перемкнути контекст |

Ці скорочення діють лише коли коментар сфокусований. Якщо жоден коментар не сфокусований, спочатку натисніть `j` або `k`.

## Глобальні скорочення

| Key | Action |
|-----|--------|
| `Ctrl+a` | Вибрати всі коментарі |
| `Ctrl+z` | Скасувати останню дію |
| `?` | Показати довідку по клавіатурних скороченнях |
| `g` | Перемкнути накладку підказок скорочень |
| `/` | Перемістити фокус у поле пошуку |

## Пагінація

| Key | Action |
|-----|--------|
| `[` | Попередня сторінка |
| `]` | Наступна сторінка |

## Підказки скорочень

Натисніть `g`, щоб переключити режим підказок. Коли він активний, на кнопках дій сфокусованого коментаря з’являються невеликі бейджі, що показують клавіатурне скорочення для кожної кнопки. Це корисно під час вивчення скорочень. Натисніть `g` ще раз, щоб приховати їх.

## Підподавлення скорочень

Скорочення автоматично вимикаються коли:

- Курсор знаходиться у полі пошуку, текстовому полі або у content-editable полі. Натисніть `Escape`, щоб зняти фокус з поля та повернутися до скорочень.
- Відкрито діалог. Спочатку закрийте діалог.
