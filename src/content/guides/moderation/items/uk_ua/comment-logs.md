FastComments автоматично відстежує детальні події для кожного коментаря, щоб забезпечити прозорість у прийнятті рішень модерації та діях системи. Ці журнали допомагають зрозуміти, чому коментар було схвалено, позначено як спам або змінено його статус.

Ви можете переглянути журнали коментарів для окремих коментарів на панелі Moderate Comments, вибравши конкретний коментар.

## Comment Log Events

Кожен коментар зберігає журнал подій, які відбуваються протягом його життєвого циклу. Нижче наведено типи подій, що відстежуються:

### Anonymization Events
- **Anonymized** - Вміст коментаря був очищений і користувача позначено як видаленого

### Approval Events
- **ApprovedDueToPastComment** - Коментар схвалено, оскільки у користувача раніше були схвалені коментарі
- **ApprovedIsAdmin** - Коментар схвалено, оскільки користувач є адміністратором
- **NotApprovedRequiresApproval** - Коментар вимагає ручного схвалення

### Spam Detection Events
- **IsSpam** - Коментар позначено як спам детектором
- **IsSpamDueToBadWords** - Коментар позначено як спам через фільтр ненормативної лексики
- **IsSpamFromLLM** - Коментар позначено як спам AI/LLM-движком
- **IsSpamRepeatComment** - Коментар позначено як спам через повторюваність
- **NotSpamIsOnlyImage** - Коментар не позначено як спам, оскільки містить лише зображення
- **NotSpamIsOnlyReacts** - Коментар не позначено як спам, оскільки містить лише реакції
- **NotSpamNoLinkOrMention** - Коментар не позначено як спам через відсутність підозрілих посилань або згадок
- **NotSpamPerfectTrustFactor** - Коментар не позначено як спам через високий рівень довіри користувача
- **NotSpamTooShort** - Коментар не позначено як спам, оскільки він занадто короткий для аналізу
- **NotSpamSkipped** - Перевірка на спам була пропущена
- **NotSpamFromEngine** - Коментар визначено як не-спам детектором

### Bad Words/Profanity Events
- **BadWordsCheckFailed** - Перевірка фільтру ненормативної лексики завершилася помилкою
- **BadWordsFoundBadPhrase** - Фільтр ненормативної лексики виявив недоречну фразу
- **BadWordsFoundBadWord** - Фільтр ненормативної лексики виявив недоречне слово
- **BadWordsNoDefinitionForLocale** - Для мови коментаря відсутні визначення ненормативної лексики

### User Verification Events
- **CommentMustBeVerifiedToApproveNotInVerifiedSession** - Коментар вимагає верифікації, але користувач не верифікований у поточній сесії
- **CommentMustBeVerifiedToApproveNotVerifiedYet** - Коментар вимагає верифікації, але користувач ще не верифікований
- **InVerifiedSession** - Користувач, який залишає коментар, знаходиться у верифікованій сесії
- **SentVerificationEmailNoSession** - Лист для верифікації надіслано неперевіреному користувачу
- **SentWelcomeEmail** - Лист привітання надіслано новому користувачу

### Trust and Security Events
- **TrustFactorChanged** - Коефіцієнт довіри користувача було змінено
- **SpamFilterDisabledBecauseAdmin** - Фільтрація спаму була вимкнена для адміністратора
- **TenantSpamFilterDisabled** - Фільтрація спаму вимкнена для всього тенанта
- **RepeatCommentCheckIgnored** - Перевірка повторних коментарів була ігнорована
- **UserIsAdmin** - Користувача ідентифіковано як адміністратора
- **UserIsAdminParentTenant** - Користувача ідентифіковано як адміністратора батьківського тенанта
- **UserIsAdminViaSSO** - Користувача ідентифіковано як адміністратора через SSO
- **UserIsMod** - Користувача ідентифіковано як модератора

### Comment Status Changes
- **ExpireStatusChanged** - Статус строку дії коментаря було змінено
- **ReviewStatusChanged** - Статус перевірки коментаря було змінено
- **SpamStatusChanged** - Статус спаму коментаря було оновлено
- **ApproveStatusChanged** - Статус схвалення коментаря було змінено
- **TextChanged** - Текст коментаря було відредаговано
- **VotesChanged** - Кількість голосів за коментар було оновлено
- **Flagged** - Коментар було позначено користувачами
- **UnFlagged** - Позначки коментаря було знято

### Moderation Actions
- **Pinned** - Коментар було закріплено модератором
- **UnPinned** - Закріплення коментаря було знято модератором
- **RestoredFromAnonymized** - Коментар відновлено з анонімізованого стану

### Notification Events
- **CreatedNotifications** - Для коментаря створено сповіщення
- **NotificationCreateFailure** - Не вдалося створити сповіщення
- **BadgeAwarded** - Користувачу нараховано бейдж за коментар

### Publishing Events
- **PublishedLive** - Коментар опубліковано для активних підписників

### Integration Events
- **WebhookSynced** - Коментар синхронізовано через webhook

### Spam Rule Events
- **SpamRuleMatch** - Коментар відповідав користувацькому правилу спаму

## Accessing Comment Logs

Журнали коментарів генеруються автоматично й зберігаються разом з кожним коментарем. Вони надають цінну інформацію для:

- Розуміння рішень модерації
- Відлагодження проблем зі схваленням або спамом
- Відстеження шаблонів поведінки користувачів
- Аудиту дій системи

Ці журнали допомагають підтримувати прозорість у процесі модерації та сприяють тонкому налаштуванню поведінки вашої системи коментарів.
