Stranica za moderiranje komentara podržava tipkovničke prečace za navigaciju između komentara, izvođenje radnji i paginaciju - sve bez korištenja miša.

Pritisnite **?** u bilo kojem trenutku na stranici za moderiranje da otvorite dijalog pomoći koji prikazuje sve dostupne prečace.

## Navigacija

| Key | Action |
|-----|--------|
| `j` | Fokusiraj sljedeći komentar |
| `k` | Fokusiraj prethodni komentar |
| `x` or `Enter` | Prebaci odabir fokusiranog komentara |
| `Shift+j` | Odaberi fokusirani komentar i prijeđi na sljedeći |
| `Shift+k` | Odaberi fokusirani komentar i prijeđi na prethodni |

Fokusirani komentar je istaknut plavim obrubom. Ovo je odvojeno od "odabranih" (označenih) komentara.

## Radnje na fokusiranom komentaru

| Key | Action |
|-----|--------|
| `a` | Prebaci odobri / poništi odobrenje |
| `s` | Prebaci spam / nije spam |
| `d` | Izbriši |
| `u` | Poništi brisanje |
| `r` | Prebaci pregledano / nije pregledano |
| `e` | Uredi |
| `b` | Zabrani / upravljaj zabranom |
| `p` | Prikvači / otkači |
| `f` | Ukloni zastavicu |
| `i` | Prikaži sve komentare s IP adrese |
| `t` | Postavi faktor povjerenja |
| `m` | Upravljaj značkama |
| `l` | Prikaži zapise |
| `c` | Prebaci kontekst |

Ovi prečaci vrijede samo kada je komentar fokusiran. Ako nijedan komentar nije fokusiran, prvo pritisnite `j` ili `k`.

## Globalni prečaci

| Key | Action |
|-----|--------|
| `Ctrl+a` | Odaberi sve komentare |
| `Ctrl+z` | Poništi posljednju radnju |
| `?` | Prikaži pomoć za tipkovničke prečace |
| `g` | Prikaži/sakrij vodiče prečaca |
| `/` | Fokusiraj polje za pretraživanje |

## Paginacija

| Key | Action |
|-----|--------|
| `[` | Prethodna stranica |
| `]` | Sljedeća stranica |

## Vodiči za prečace

Pritisnite `g` za prebacivanje načina vodiča. Kada je aktivan, male značke pojavljuju se na gumbima radnji fokusiranog komentara i prikazuju tipkovnički prečac za svaki gumb. Ovo je korisno pri učenju prečaca. Pritisnite `g` ponovno da ih sakrijete.

## Onemogućavanje prečaca pri unosu

Prečaci se automatski onemogućuju kada:

- Kursor je u polju za pretraživanje, tekstualnom unosu ili u polju s uređivim sadržajem. Pritisnite `Escape` da biste uklonili fokus s polja i vratili se na prečace.
- Otvoren je dijalog. Prvo zatvorite dijalog.
