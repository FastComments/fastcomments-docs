Généralement, cela se produit si vous utilisez votre propre domaine au lieu du domaine par défaut "thrivecart.com".

Cela s'explique par le fait que FastComments ne sait pas qu'il doit autoriser l'utilisation de votre compte sur ce domaine.

Dans ce cas, la solution est simple. Vous devez simplement ajouter votre site à votre compte FastComments.com.

[Allez ici pour ajouter votre site à votre compte.](https://fastcomments.com/auth/my-account/configure-domains)
