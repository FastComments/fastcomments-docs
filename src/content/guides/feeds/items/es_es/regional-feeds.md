Los feeds regionales te permiten dirigir contenido a usuarios según su ubicación geográfica. Esto se logra usando tags que especifican información regional, permitiéndote crear experiencias de contenido específicas por ubicación.

### Segmentación por país

Para implementar feeds regionales, usa tags geográficos al crear y obtener publicaciones. El enfoque recomendado es usar tags que sigan una convención de nomenclatura geográfica consistente.

Para la segmentación a nivel de país, podrías usar tags en el formato `country:countrycode`. Por ejemplo: `country:usa`.
