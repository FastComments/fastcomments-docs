Региональные ленты позволяют направлять контент пользователям на основе их географического местоположения. Это достигается с помощью тегов, указывающих региональную информацию, что позволяет создавать контентный опыт, специфичный для местоположения.

### Таргетинг по странам

Для реализации региональных лент используйте географические теги при создании и получении публикаций. Рекомендуемый подход — использовать теги, следующие согласованной географической конвенции именования.

Для таргетинга на уровне страны вы можете использовать теги в формате `country:countrycode`. Например: `country:usa`.
