Τα περιφερειακά feeds σας επιτρέπουν να στοχεύετε περιεχόμενο σε χρήστες βασισμένο στη γεωγραφική τους τοποθεσία. Αυτό επιτυγχάνεται χρησιμοποιώντας tags που καθορίζουν περιφερειακές πληροφορίες, επιτρέποντάς σας να δημιουργείτε εμπειρίες περιεχομένου συγκεκριμένες για κάθε τοποθεσία.

### Στόχευση βάσει χώρας

Για να υλοποιήσετε περιφερειακά feeds, χρησιμοποιήστε γεωγραφικά tags κατά τη δημιουργία και ανάκτηση δημοσιεύσεων. Η συνιστώμενη προσέγγιση είναι να χρησιμοποιείτε tags που ακολουθούν μια συνεπή γεωγραφική σύμβαση ονοματοδοσίας.

Για στόχευση σε επίπεδο χώρας, μπορείτε να χρησιμοποιήσετε tags με τη μορφή `country:countrycode`. Για παράδειγμα: `country:usa`.
