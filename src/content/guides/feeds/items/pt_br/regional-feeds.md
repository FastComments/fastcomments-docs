Os feeds regionais permitem direcionar conteúdo aos usuários com base em sua localização geográfica. Isso é alcançado usando tags que especificam informações regionais, permitindo criar experiências de conteúdo específicas por localização.

### Segmentação por país

Para implementar feeds regionais, use tags geográficas ao criar e buscar publicações. A abordagem recomendada é usar tags que sigam uma convenção de nomenclatura geográfica consistente.

Para segmentação em nível de país, você pode usar tags no formato `country:countrycode`. Por exemplo: `country:usa`.
