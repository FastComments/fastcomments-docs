Las notificaciones se pueden cancelar de varias maneras. Si su cuenta es gestionada por FastComments.com y no a través de SSO, puede configurar sus
notificaciones a través de la [Página de configuración de notificaciones](https://fastcomments.com/auth/my-account/edit-notifications).

Aquí puede configurar las notificaciones de respuestas a comentarios y menciones, las notificaciones horarias de nuevos comentarios y cualquier notificación en general. Si es administrador del sitio o
moderador, también tendrá la opción de configurar la frecuencia del correo electrónico de resumen aquí.

Además, cada correo electrónico contiene un enlace para darse de baja. Para las notificaciones de respuestas a comentarios, el correo contiene un enlace para **darse de baja de las notificaciones para ese comentario**.

Las notificaciones también tendrán un enlace para cancelar la suscripción para desactivarlas, así como las cabeceras de correo electrónico apropiadas.

Los enlaces para cancelar la suscripción en los correos electrónicos pueden eliminarse a petición para un tenant determinado. Además, el enlace se puede personalizar para integraciones SSO o whitelabel.
