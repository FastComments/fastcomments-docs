Se puede optar por no recibir notificaciones de varias maneras. Si su cuenta está gestionada por FastComments.com y no mediante SSO, puede configurar sus
notificaciones a través de la [página de Configuración de notificaciones](https://fastcomments.com/auth/my-account/edit-notifications).

Aquí puede configurar las notificaciones de respuestas a comentarios y menciones, la frecuencia de notificaciones de suscripción (cada minuto, cada hora o diariamente), y cualquier notificación en general. Si usted es administrador del sitio o
moderador, también tendrá la opción de configurar aquí la frecuencia del correo electrónico Digest.

Además, cada correo electrónico contiene un enlace para darse de baja. Para las notificaciones de respuesta a comentarios, el correo contiene un enlace para **darse de baja de las notificaciones para ese comentario**.

Las notificaciones también incluirán un enlace para darse de baja para desactivarlas, así como las cabeceras de correo electrónico apropiadas.

Los enlaces para darse de baja en los correos pueden eliminarse bajo solicitud para un tenant determinado. Además, el enlace puede personalizarse para integraciones SSO o de marca blanca.
