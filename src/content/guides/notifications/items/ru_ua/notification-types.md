Существует несколько типов уведомлений и каналов, через которые FastComments отправляет уведомления:

- Немедленно по электронной почте.
- Немедленно через приложение.
  - Они отображаются под значком колокольчика уведомлений.
- Сводные по электронной почте.
  - Они собираются в сводки каждый час. Если в этот период для пользователя нечего отправлять, мы не отправляем письмо.
- По расписанию по электронной почте.
  - Примеры включают Community Digest Summary, которую можно отправлять ежедневно, еженедельно или ежемесячно (или отключить).
