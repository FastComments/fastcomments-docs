Von Benachrichtigungen kann auf mehrere Arten abgemeldet werden. Wenn Ihr Konto von FastComments.com verwaltet wird und nicht über SSO, können Sie Ihre
Benachrichtigungen über die [Benachrichtigungseinstellungen](https://fastcomments.com/auth/my-account/edit-notifications).

Hier können Sie Benachrichtigungen für Kommentarantworten und Erwähnungen, die Häufigkeit der Benachrichtigungen (jede Minute, stündlich oder täglich), die Häufigkeit von Abonnement-Benachrichtigungen (jede Minute, stündlich oder täglich) und allgemein jede Benachrichtigung konfigurieren. Wenn Sie ein Site-Administrator oder
Moderator sind, haben Sie außerdem eine separate Einstellung **Admin-Benachrichtigungsfrequenz**, die steuert, wie oft Sie E-Mails über neue Kommentare auf Ihrer Website erhalten (standardmäßig: stündlich), sowie die Möglichkeit, die Häufigkeit der Digest-E-Mails zu konfigurieren.

Außerdem enthält jede E-Mail einen Abmeldelink. Bei Benachrichtigungen über Kommentarantworten enthält die E-Mail einen Link, um **Benachrichtigungen für diesen Kommentar abzumelden**.

Benachrichtigungen enthalten außerdem einen Abmeldelink zum Deaktivieren sowie die entsprechenden E-Mail-Header.

Abmeldelinks in E-Mails können auf Anfrage für einen bestimmten Mandanten entfernt werden. Zusätzlich kann der Link für SSO- oder Whitelabel-Integrationen angepasst werden.
