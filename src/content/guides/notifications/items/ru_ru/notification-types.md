Существует несколько типов уведомлений и каналов, через которые FastComments отправляет уведомления:

- Мгновенные по электронной почте.
- Мгновенные через приложение.
  - Они отображаются под иконкой колокольчика уведомлений.
- Пакетные по электронной почте.
  - Они собираются в пакеты на ежечасной основе. Если в этот период для пользователя нет ничего для отправки, мы не отправляем письмо.
- По расписанию по электронной почте.
  - Например, Сводка сообщества, которую можно отправлять ежедневно, еженедельно или ежемесячно (или отключить).
