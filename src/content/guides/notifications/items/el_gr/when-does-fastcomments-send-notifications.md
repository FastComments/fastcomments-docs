Στην FastComments, γνωρίζουμε ότι λαμβάνετε ήδη αρκετές ειδοποιήσεις. Για αυτό, λαμβάνουμε κάποια μέτρα για να περιορίσουμε τις ειδοποιήσεις που λαμβάνουν οι χρήστες ενώ εξακολουθούμε
να τους κρατάμε σε επαφή με τις κοινότητές τους. Θέλουμε επίσης να κρατάμε τους διαχειριστές και τους συντονιστές ενημερωμένους και να τους ειδοποιούμε όταν απαιτείται να ληφθούν μέτρα.

#### Θα στείλουμε ειδοποιήσεις για τα παρακάτω συμβάντα προς διαχειριστές και συντονιστές:

- Σύνοψη κοινοτικής περίληψης (συχνότητα ρυθμιζόμενη).
- Αιτήματα βοήθειας κοινότητας και υπενθυμίσεις.
- Νέα σχόλια.

#### Για σχολιαστές:

- Όταν κάποιος απαντά στο σχόλιό σας (μέσω ηλεκτρονικού ταχυδρομείου).
- Όταν σας αναφέρουν (ειδοποίηση εντός εφαρμογής και μέσω ηλεκτρονικού ταχυδρομείου).
- Όταν κάποιος απαντά στο ίδιο νήμα (ειδοποίηση εντός εφαρμογής και μέσω ηλεκτρονικού ταχυδρομείου).
- Όταν κάποιος απαντά σε ένα υπο-σχόλιο στο ίδιο νήμα (ειδοποίηση εντός εφαρμογής και μέσω ηλεκτρονικού ταχυδρομείου).
- Όταν κάποιος απαντά σε μια σελίδα στην οποία έχετε εγγραφεί (ειδοποίηση εντός εφαρμογής και μέσω ηλεκτρονικού ταχυδρομείου, συχνότητα ρυθμιζόμενη ανά εγγραφή: κάθε λεπτό, ανά ώρα ή καθημερινά).
- Όταν ένας χρήστης σχολιάζει για πρώτη φορά (αλλά όχι με SSO).
- Όταν ένας χρήστης αφήνει ένα σχόλιο σε μια συνεδρία που δεν είναι επαληθευμένη (αλλά όχι με SSO).
  - Δεν στέλνουμε πολλαπλά μηνύματα επαλήθευσης σε αυτή την περίπτωση. Μόνο το πρώτο, το οποίο θα επαληθεύσει όλη τη δραστηριότητα στην ίδια συνεδρία.

#### Για όλους τους χρήστες:

- Όταν ανιχνεύεται σύνδεση από νέα IP address, αποστέλλεται ένα email ειδοποίησης ασφαλείας με την περίπου τοποθεσία και την IP address. Αυτό δεν ισχύει για την πολύ πρώτη σύνδεση του χρήστη.

#### ...και τελικά μόνο για διαχειριστές:

- Όταν ολοκληρώνονται οι ενσωματώσεις.
- Όταν ολοκληρώνονται οι μετεγκαταστάσεις.
- Όταν ολοκληρώνονται εισαγωγές ή εξαγωγές.
- Όταν υπάρχουν ζητήματα χρέωσης.
- Υπενθυμίσεις λήξης δοκιμαστικής περιόδου.

Ορισμένες ειδοποιήσεις ομαδοποιούνται για να αποτρέπεται η μαζική αποστολή ειδοποιήσεων προς τους χρήστες. Μάθετε περισσότερα σε αυτή την ενότητα `Notification Types`.
