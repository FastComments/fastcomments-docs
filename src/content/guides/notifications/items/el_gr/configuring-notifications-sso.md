Για το SSO υπάρχουν οι ακόλουθες ρυθμίσεις που πρέπει να ληφθούν υπόψη για τις ειδοποιήσεις:

- Εάν ο χρήστης έχει επιλέξει να λαμβάνει ειδοποιήσεις.
  - Αυτό γίνεται ρυθμίζοντας τη σημαία `optedInNotifications` σε `true` ή `false` στο αντικείμενο `SSOUser`.
  - Αυτό μπορεί να οριστεί μέσω του API.
  - Επίσης, αν περάσετε μια τιμή για αυτή τη σημαία στο payload, θα ενημερωθεί αυτόματα όταν ο χρήστης φορτώσει ένα νήμα σχολίων.
- Εάν ο χρήστης έχει επιλέξει να λαμβάνει ειδοποιήσεις **συνδρομής**.
  - Αυτό γίνεται ρυθμίζοντας τη σημαία `optedInSubscriptionNotifications` σε `true` ή `false` στο αντικείμενο `SSOUser`.
  - Αυτό μπορεί να οριστεί μέσω του API.
  - Επίσης, αν περάσετε μια τιμή για αυτή τη σημαία στο payload, θα ενημερωθεί αυτόματα όταν ο χρήστης φορτώσει ένα νήμα σχολίων.
- Ορισμός του email τους.
  - Εάν δεν υπάρχει, δεν μπορούμε να στείλουμε ειδοποιήσεις μέσω email.
- Εάν θα απενεργοποιηθούν οι σύνδεσμοι διαγραφής (unsubscribe) στα email.
  - Αυτό γίνεται μέσω της σημαίας `disableUnsubscribeLinks` στο αντικείμενο `Tenant`.
  - Αυτό μπορεί να οριστεί μέσω του API.
- Εάν θα χρησιμοποιηθεί προσαρμοσμένος σύνδεσμος διαγραφής (unsubscribe).
  - Αυτό γίνεται μέσω της ιδιότητας `footerUnsubscribeURL` στο αντικείμενο `DomainConfig`.
  - Αυτό μπορεί να οριστεί μέσω του API.
  - Μπορεί επίσης να θέλετε να εξετάσετε τον ορισμό των σχετικών headers διαγραφής μέσω της `emailHeaders` στο ίδιο αντικείμενο.
