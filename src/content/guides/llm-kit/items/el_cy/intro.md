## LLM Kit

Το FastComments LLM Kit βοηθά βοηθούς κωδικοποίησης AI όπως οι Claude, ChatGPT και Cursor να κατανοήσουν και να ενσωματωθούν με το FastComments, παρέχοντάς τους πρόσβαση στην πλήρη τεκμηρίωσή μας μέσω ενός απλού API αναζήτησης.

Προσθέτοντας ένα απόσπασμα στο αρχείο AGENT.md ή CLAUDE.md του έργου σας, επιτρέπετε στους βοηθούς AI να αναζητούν γρήγορα και να ανακτούν τη σχετική τεκμηρίωση του FastComments, καθιστώντας την υλοποίηση ταχύτερη και πιο ακριβή.
