## LLM Kit

FastComments LLM Kit помогает помощникам по программированию на базе ИИ, таким как Claude, ChatGPT и Cursor, понимать и интегрироваться с FastComments, предоставляя им доступ ко всей нашей документации через простой API поиска.

Добавив фрагмент в файл AGENT.md или CLAUDE.md вашего проекта, вы позволяете помощникам ИИ быстро искать и получать соответствующую документацию FastComments, что делает внедрение быстрее и точнее.
