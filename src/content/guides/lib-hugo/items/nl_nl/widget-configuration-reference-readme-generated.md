Voor de volledige lijst met widgetopties, zie de [FastComments customisatiedocumentatie](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Twee widgets gebruiken een verplichte optie die geen standaardwaarde heeft:

- `fastcomments-collab-chat` en `fastcomments-image-chat` vereisen `target`, een CSS-selector voor het element waaraan het moet worden gekoppeld.
- `fastcomments-user-activity-feed` vereist `userId`.
