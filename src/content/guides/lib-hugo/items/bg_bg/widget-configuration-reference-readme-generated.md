За пълен списък с опции за джаджите, вижте [документацията за персонализиране на FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Две джаджи изискват задължителна опция, която няма стойност по подразбиране:

- `fastcomments-collab-chat` и `fastcomments-image-chat` изискват `target`, CSS селектор за елемента, към който да се прикрепят.
- `fastcomments-user-activity-feed` изисква `userId`.
