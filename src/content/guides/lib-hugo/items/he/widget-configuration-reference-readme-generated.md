לרשימה המלאה של אפשרויות הווידג'ט, ראה את [תיעוד ההתאמות וההגדרות של FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). שני ווידג'טים דורשים אופציה חובה שאין לה ברירת מחדל:

- `fastcomments-collab-chat` ו-`fastcomments-image-chat` דורשים `target`, סלקטור CSS של האלמנט שאליו יש לצרף אותם.
- `fastcomments-user-activity-feed` דורש `userId`.
