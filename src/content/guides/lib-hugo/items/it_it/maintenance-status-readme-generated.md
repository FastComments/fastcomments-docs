Questi componenti sono dei wrapper leggeri intorno ai nostri widget core VanillaJS. Possiamo aggiornare quei widget (correggere bug, aggiungere funzionalità) senza pubblicare questo repository, quindi una cronologia di release silenziosa non significa che FastComments sia inattivo. Controlla il [nostro blog](https://blog.fastcomments.com/) per aggiornamenti. Le modifiche non retrocompatibili ai widget sottostanti vengono rilasciate insieme a un aumento di versione qui.
