Za celoten seznam možnosti widgetov si oglejte [dokumentacijo FastComments o prilagajanju in konfiguraciji](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Dva widgeta zahtevata obvezno možnost, ki nima privzete vrednosti:

- `fastcomments-collab-chat` in `fastcomments-image-chat` zahtevata `target`, CSS selektor elementa, na katerega se pripne.
- `fastcomments-user-activity-feed` zahteva `userId`.
