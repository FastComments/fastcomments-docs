For den fulde liste over widgetindstillinger, se [FastComments-tilpasningsdokumentation](https://docs.fastcomments.com/guide-customizations-and-configuration.html). To af widgets kræver en påkrævet indstilling, som ikke har en standardværdi:

- `fastcomments-collab-chat` og `fastcomments-image-chat` kræver `target`, en CSS-selector for det element, der skal tilknyttes.
- `fastcomments-user-activity-feed` kræver `userId`.
