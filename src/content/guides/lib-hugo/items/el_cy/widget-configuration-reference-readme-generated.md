Για την πλήρη λίστα των επιλογών των widget, δείτε την [τεκμηρίωση προσαρμογής του FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Δύο widgets απαιτούν μια υποχρεωτική επιλογή που δεν έχει προεπιλεγμένη τιμή:

- `fastcomments-collab-chat` και `fastcomments-image-chat` απαιτούν το `target`, έναν CSS επιλογέα για το στοιχείο στο οποίο θα προσαρτηθούν.
- `fastcomments-user-activity-feed` απαιτεί το `userId`.
