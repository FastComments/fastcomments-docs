Эти компоненты являются тонкими оболочками вокруг наших основных виджетов VanillaJS. Мы можем обновлять эти виджеты (исправлять ошибки, добавлять новые возможности) без публикации этого репозитория, поэтому тихая история релизов не означает, что FastComments неактивен. Следите за обновлениями в [нашем блоге](https://blog.fastcomments.com/). Ломающие изменения в базовых виджетах сопровождаются повышением версии в этом репозитории.
