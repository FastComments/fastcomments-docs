Ces composants sont de légères surcouches autour de nos widgets VanillaJS principaux. Nous pouvons mettre à jour ces widgets (corriger des bugs, ajouter des fonctionnalités) sans publier ce dépôt, donc un historique de versions discret ne signifie pas que FastComments est inactif. Consultez [notre blog](https://blog.fastcomments.com/) pour les mises à jour. Les changements incompatibles apportés aux widgets sous-jacents sont publiés ici accompagnés d'une mise à jour de version.
