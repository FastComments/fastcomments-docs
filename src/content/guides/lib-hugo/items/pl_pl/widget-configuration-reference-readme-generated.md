Aby uzyskać pełną listę opcji widgetów, zobacz [dokumentacja dostosowywania FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Dwa widgety wymagają opcji obowiązkowej, która nie ma wartości domyślnej:

- `fastcomments-collab-chat` i `fastcomments-image-chat` wymagają `target`, selektora CSS elementu, do którego mają zostać dołączone.
- `fastcomments-user-activity-feed` wymaga `userId`.
