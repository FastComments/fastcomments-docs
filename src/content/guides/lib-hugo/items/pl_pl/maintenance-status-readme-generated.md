Te komponenty są cienkimi powłokami wokół naszych podstawowych widżetów VanillaJS. Możemy aktualizować te widżety (naprawiać błędy, dodawać funkcje) bez publikowania tego repozytorium, więc cicha historia wydań nie oznacza, że FastComments jest nieaktywny. Sprawdź [nasz blog](https://blog.fastcomments.com/) po aktualizacje. Zmiany łamiące kompatybilność w podstawowych widżetach są wydawane wraz ze zwiększeniem numeru wersji tutaj.
