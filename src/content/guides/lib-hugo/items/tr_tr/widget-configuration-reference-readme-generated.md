Widget seçeneklerinin tam listesi için [FastComments özelleştirme belgelerine](https://docs.fastcomments.com/guide-customizations-and-configuration.html) bakın. İki widget, varsayılanı olmayan zorunlu bir seçenek gerektirir:

- `fastcomments-collab-chat` ve `fastcomments-image-chat` için `target` gereklidir; bu, iliştirilecek öğe için bir CSS seçicisidir.
- `fastcomments-user-activity-feed` için `userId` gereklidir.
