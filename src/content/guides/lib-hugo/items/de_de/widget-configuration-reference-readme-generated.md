Für die vollständige Liste der Widget-Optionen siehe die [FastComments-Anpassungsdokumentation](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Zwei Widgets benötigen eine erforderliche Option, die keinen Standardwert hat:

- `fastcomments-collab-chat` und `fastcomments-image-chat` benötigen `target`, einen CSS-Selektor für das Element, an das sie angehängt werden sollen.
- `fastcomments-user-activity-feed` benötigt `userId`.
