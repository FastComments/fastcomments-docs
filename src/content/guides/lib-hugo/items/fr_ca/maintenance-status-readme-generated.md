Ces composants sont de fines couches autour de nos widgets VanillaJS principaux. Nous pouvons mettre à jour ces widgets (corriger des bogues, ajouter des fonctionnalités) sans publier ce dépôt, donc un historique des versions silencieux ne signifie pas que FastComments est inactif. Consultez [notre blog](https://blog.fastcomments.com/) pour les mises à jour. Les changements incompatibles apportés aux widgets sous-jacents sont publiés ici accompagnés d'une augmentation de version.
