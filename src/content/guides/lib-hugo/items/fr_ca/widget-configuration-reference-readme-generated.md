Pour la liste complète des options de widget, voir la [documentation de personnalisation de FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Deux widgets nécessitent une option requise qui n'a pas de valeur par défaut :

- `fastcomments-collab-chat` et `fastcomments-image-chat` requièrent `target`, un sélecteur CSS pour l'élément auquel s'attacher.
- `fastcomments-user-activity-feed` requiert `userId`.
