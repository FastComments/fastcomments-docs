Полный список опций виджетов см. в [документации по настройке FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Два виджета принимают обязательный параметр, у которого нет значения по умолчанию:

- `fastcomments-collab-chat` и `fastcomments-image-chat` требуют `target`, CSS‑селектор элемента, к которому привязываются.
- `fastcomments-user-activity-feed` требует `userId`.
