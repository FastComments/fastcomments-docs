Para a lista completa de opções dos widgets, veja a [documentação de personalização do FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Dois widgets exigem uma opção obrigatória que não tem valor padrão:

- `fastcomments-collab-chat` e `fastcomments-image-chat` exigem `target`, um seletor CSS para o elemento ao qual anexar.
- `fastcomments-user-activity-feed` exige `userId`.
