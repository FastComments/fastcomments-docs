Esses componentes são camadas de abstração leves em torno dos nossos widgets centrais VanillaJS. Podemos atualizar esses widgets (corrigir bugs, adicionar recursos) sem publicar este repositório, então um histórico de lançamentos discreto não significa que o FastComments está inativo. Confira [nosso blog](https://blog.fastcomments.com/) para atualizações. Alterações que quebram a compatibilidade nos widgets subjacentes são lançadas com um aumento de versão aqui.
