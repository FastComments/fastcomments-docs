За комплетан списак опција видгета, погледајте [FastComments документацију о прилагођавању](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Два видгета захтијевају обавезну опцију која нема подразумевану вриједност:

- `fastcomments-collab-chat` и `fastcomments-image-chat` захтијевају `target`, CSS селектор за елемент на који ће се прикачити.
- `fastcomments-user-activity-feed` захтијева `userId`.
