Повний список параметрів віджетів можна знайти в [документації з налаштування та конфігурації FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Два віджети вимагають обов'язкової опції, яка не має значення за замовчуванням:

- `fastcomments-collab-chat` та `fastcomments-image-chat` вимагають `target` — CSS-селектор елемента, до якого слід приєднатися.
- `fastcomments-user-activity-feed` вимагає `userId`.
