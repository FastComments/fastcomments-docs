Για την πλήρη λίστα των επιλογών widget, δείτε την [τεκμηρίωση προσαρμογής του FastComments](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Δύο widget απαιτούν μια υποχρεωτική επιλογή που δεν έχει προεπιλογή:

- `fastcomments-collab-chat` και `fastcomments-image-chat` απαιτούν `target`, έναν επιλεκτή CSS για το στοιχείο στο οποίο θα προσαρτηθούν.
- `fastcomments-user-activity-feed` απαιτεί `userId`.
