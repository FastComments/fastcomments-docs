ウィジェットオプションの完全な一覧については、[FastComments のカスタマイズと構成に関するドキュメント](https://docs.fastcomments.com/guide-customizations-and-configuration.html) を参照してください。2つのウィジェットにはデフォルト値のない必須オプションがあります:

- `fastcomments-collab-chat` と `fastcomments-image-chat` は `target` が必要で、これはアタッチする要素のための CSS セレクタです。
- `fastcomments-user-activity-feed` は `userId` が必要です。
