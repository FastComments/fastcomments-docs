Za kompletan spisak opcija za widgete, pogledajte [FastComments dokumentaciju o prilagođavanju](https://docs.fastcomments.com/guide-customizations-and-configuration.html). Dva widgeta zahtevaju obaveznu opciju koja nema podrazumevanu vrednost:

- `fastcomments-collab-chat` i `fastcomments-image-chat` zahtevaju `target`, CSS selektor za element na koji treba da se prikači.
- `fastcomments-user-activity-feed` zahteva `userId`.
