Bağımlılıkları `npm install` (veya `pnpm install` veya `yarn`) ile yükledikten sonra, bir geliştirme sunucusu başlatın:

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

`src/lib` içindeki her şey kütüphanenin bir parçasıdır, `src/routes` içindeki her şey gösterimdir.
