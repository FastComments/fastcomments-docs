Essayez chaque widget en ligne sur <https://fastcomments.com/commenting-system-for-svelte>.

Un ensemble de widgets Svelte et de wrappers pour FastComments. `src/routes` contient des exemples exécutables - lancez simplement :

    npm install
    npm run dev

Ensuite, vous pouvez voir tous les widgets et les modifier - par exemple pour y renseigner votre propre tenant id ou essayer une configuration différente.

TypeScript a été utilisé afin que vous puissiez inspecter les objets de configuration de chaque widget pour voir quelles options ils acceptent.
