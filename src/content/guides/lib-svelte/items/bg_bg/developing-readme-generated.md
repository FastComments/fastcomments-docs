След като инсталирате зависимостите с `npm install` (или `pnpm install` или `yarn`), стартирайте сървър за разработка:

```bash
npm run dev

# или стартирайте сървъра и отворете приложението в нов раздел на браузъра
npm run dev -- --open
```

Всичко в `src/lib` е част от библиотеката, всичко в `src/routes` е демонстрационното приложение.
