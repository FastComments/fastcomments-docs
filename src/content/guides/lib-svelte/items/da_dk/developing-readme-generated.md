Når du har installeret afhængigheder med `npm install` (eller `pnpm install` eller `yarn`), start en udviklingsserver:

```bash
npm run dev

# eller start serveren og åbn appen i en ny browserfane
npm run dev -- --open
```

Alt inde i `src/lib` er en del af biblioteket, alt inde i `src/routes` er demoen.
