Prøv hver widget live på <https://fastcomments.com/commenting-system-for-svelte>.

Et sæt Svelte-widgets og wrappers til FastComments. `src/routes` indeholder kørbare eksempler - kør blot:

    npm install
    npm run dev

Så kan du se alle widgets og redigere dem - for eksempel indsætte din egen tenant id eller prøve forskellige konfigurationer.

TypeScript er blevet brugt, så du kan inspicere konfigurationsobjekterne for hver widget for at se, hvilke muligheder de tilbyder.
