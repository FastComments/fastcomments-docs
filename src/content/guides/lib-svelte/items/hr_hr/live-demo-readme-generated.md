Isprobajte svaki widget uživo na <https://fastcomments.com/commenting-system-for-svelte>.

Skup Svelte widgeta i omotača za FastComments. `src/routes` sadrži primjere koje možete pokrenuti - jednostavno pokrenite:

    npm install
    npm run dev

Tada možete vidjeti sve widgete i uređivati ih - na primjer da upišete svoj vlastiti tenant id ili isprobate različitu konfiguraciju.

TypeScript je korišten tako da možete pregledati konfiguracijske objekte za svaki widget i vidjeti koje opcije prihvaćaju.
