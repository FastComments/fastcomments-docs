Nakon što instalirate ovisnosti pomoću `npm install` (ili `pnpm install` ili `yarn`), pokrenite razvojni poslužitelj:

```bash
npm run dev

# ili pokrenite poslužitelj i otvorite aplikaciju u novoj kartici preglednika
npm run dev -- --open
```

Sve unutar `src/lib` je dio biblioteke, sve unutar `src/routes` je prikaz.
