Te komponenty są opakowaniami wokół naszych bazowych komponentów VanillaJS. Możemy automatycznie aktualizować te komponenty (poprawiać błędy, dodawać funkcje) bez publikowania tej biblioteki, więc choć może ona nie być opublikowana przez pewien czas, nie oznacza to, że FastComments nie jest aktywnie rozwijany! Zapraszamy do odwiedzenia [naszego bloga](https://blog.fastcomments.com/) po aktualizacje. Zmiany łamiące zgodność API lub nowe funkcje nigdy nie zostaną wprowadzone do bazowej biblioteki bez podniesienia wersji tej biblioteki.
