Po zainstalowaniu zależności za pomocą `npm install` (lub `pnpm install` lub `yarn`), uruchom serwer deweloperski:

```bash
npm run dev

# lub uruchom serwer i otwórz aplikację w nowej karcie przeglądarki
npm run dev -- --open
```

Wszystko w `src/lib` jest częścią biblioteki, wszystko w `src/routes` jest pokazem.
