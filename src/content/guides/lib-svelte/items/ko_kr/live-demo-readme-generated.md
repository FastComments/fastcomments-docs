모든 위젯을 <https://fastcomments.com/commenting-system-for-svelte>에서 직접 사용해 보세요.

FastComments용 Svelte 위젯 및 래퍼 모음입니다. `src/routes`에는 실행 가능한 예제가 포함되어 있습니다 - 단순히 다음을 실행하세요:

    npm install
    npm run dev

그런 다음 모든 위젯을 보고 수정할 수 있습니다 - 예를 들어 자신의 tenant id를 입력하거나 다른 구성을 시도해 볼 수 있습니다.

TypeScript가 사용되어 각 위젯의 config 객체를 검사하여 어떤 옵션을 사용하는지 확인할 수 있습니다.
