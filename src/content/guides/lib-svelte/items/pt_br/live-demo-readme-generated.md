Experimente todos os widgets ao vivo em <https://fastcomments.com/commenting-system-for-svelte>.

Um conjunto de widgets e wrappers Svelte para o FastComments. `src/routes` contém exemplos executáveis - basta executar:

    npm install
    npm run dev

Então você pode ver todos os widgets e editá-los - por exemplo, para inserir seu próprio tenant id ou testar diferentes configurações.

TypeScript foi usado para que você possa inspecionar os objetos de configuração de cada widget e ver quais opções eles aceitam.
