Αφού εγκαταστήσετε τις εξαρτήσεις με `npm install` (ή `pnpm install` ή `yarn`), ξεκινήστε έναν διακομιστή ανάπτυξης:

```bash
npm run dev

# ή ξεκινήστε τον διακομιστή και ανοίξτε την εφαρμογή σε νέα καρτέλα του περιηγητή
npm run dev -- --open
```

Όλα όσα βρίσκονται μέσα στο `src/lib` είναι μέρος της βιβλιοθήκης, όλα όσα βρίσκονται μέσα στο `src/routes` είναι η επίδειξη.
