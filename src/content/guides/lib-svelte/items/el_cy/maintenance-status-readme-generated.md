Αυτά τα components είναι wrappers γύρω από τα βασικά μας components σε VanillaJS. Μπορούμε να ενημερώνουμε αυτόματα αυτά τα components (να διορθώνουμε σφάλματα, να προσθέτουμε δυνατότητες) χωρίς να δημοσιεύουμε αυτή τη βιβλιοθήκη, οπότε το ότι μπορεί να μην έχει δημοσιευτεί για λίγο δεν σημαίνει ότι το FastComments δεν βρίσκεται σε ενεργό ανάπτυξη! Μη διστάσετε να ελέγξετε [το blog μας](https://blog.fastcomments.com/) για ενημερώσεις. Οι ριζικές αλλαγές στο API ή οι νέες δυνατότητες δεν θα ενσωματωθούν ποτέ στην υποκείμενη βασική βιβλιοθήκη χωρίς αύξηση έκδοσης σε αυτή τη βιβλιοθήκη.
