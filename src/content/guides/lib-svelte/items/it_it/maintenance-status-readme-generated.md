Questi componenti sono dei wrapper attorno ai nostri componenti core VanillaJS. Possiamo aggiornare automaticamente questi componenti (correggere bug, aggiungere funzionalità) senza pubblicare questa libreria, quindi anche se potrebbe non essere pubblicata per un po' questo non significa che FastComments non sia in sviluppo attivo! Non esitare a consultare [il nostro blog](https://blog.fastcomments.com/) per aggiornamenti. Modifiche incompatibili all'API o nuove funzionalità non verranno mai integrate nella libreria core sottostante senza un aumento della versione in questa libreria.
