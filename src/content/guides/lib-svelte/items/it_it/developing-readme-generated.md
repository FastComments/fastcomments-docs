Una volta installate le dipendenze con `npm install` (o `pnpm install` o `yarn`), avvia un server di sviluppo:

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

Tutto ciò che si trova in `src/lib` fa parte della libreria, tutto ciò che si trova in `src/routes` è la vetrina.
