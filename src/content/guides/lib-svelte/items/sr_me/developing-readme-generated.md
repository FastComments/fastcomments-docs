Када инсталирате зависности помоћу `npm install` (или `pnpm install` или `yarn`), покрените развојни сервер:

```bash
npm run dev

# или покрените сервер и отворите апликацију у новој картици прегледача
npm run dev -- --open
```

Све што се налази унутар `src/lib` је део библиотеке, а све што се налази унутар `src/routes` је демонстрација.
