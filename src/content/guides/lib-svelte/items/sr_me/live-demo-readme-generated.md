Пробајте сваки видџет уживо на <https://fastcomments.com/commenting-system-for-svelte>.

Скуп Svelte видџета и омотача за FastComments. `src/routes` садржи примјере које можете покренути - једноставно покрените:

    npm install
    npm run dev

Онда можете видјети све видџете и уредити их - на примјер да унесете свој tenant id или пробате различите конфигурације.

Коришћен је TypeScript тако да можете прегледати config објекте за сваки видџет и видјети које опције прихватају.
