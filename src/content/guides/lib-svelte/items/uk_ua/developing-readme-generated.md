---
Після встановлення залежностей за допомогою `npm install` (або `pnpm install` чи `yarn`), запустіть сервер розробки:

```bash
npm run dev

# або запустіть сервер і відкрийте додаток у новій вкладці браузера
npm run dev -- --open
```

Усе, що знаходиться в `src/lib`, є частиною бібліотеки, усе, що знаходиться в `src/routes`, — демонстрація.
