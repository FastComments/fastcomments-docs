Спробуйте кожен віджет онлайн на <https://fastcomments.com/commenting-system-for-svelte>.

A set of Svelte widgets and wrappers for FastComments. `src/routes` містить запускні приклади - просто виконайте:

    npm install
    npm run dev

Тоді ви зможете побачити всі віджети та редагувати їх - наприклад, щоб вказати власний tenant id або спробувати іншу конфігурацію.

TypeScript використано, щоб ви могли переглянути об'єкти конфігурації для кожного віджета та побачити, які параметри вони приймають.
