Эти компоненты являются обёртками над нашими основными компонентами VanillaJS. Мы можем автоматически обновлять эти компоненты (исправлять ошибки, добавлять функции) без публикации этой библиотеки, поэтому то, что она в течение некоторого времени может не публиковаться, не означает, что FastComments не находится в активной разработке! Не стесняйтесь заглядывать в [наш блог](https://blog.fastcomments.com/) для получения обновлений. Несовместимые изменения API или новые возможности никогда не будут внедрены в базовую библиотеку ядра без повышения версии в этой библиотеке.
