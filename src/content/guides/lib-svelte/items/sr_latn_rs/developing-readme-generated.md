---
Kada instalirate zavisnosti pomoću `npm install` (ili `pnpm install` ili `yarn`), pokrenite razvojni server:

```bash
npm run dev

# ili pokrenite server i otvorite aplikaciju u novoj kartici pregledača
npm run dev -- --open
```

Sve što se nalazi u `src/lib` je deo biblioteke, a sve što se nalazi u `src/routes` je demo.
