すべてのウィジェットをライブで試すには <https://fastcomments.com/commenting-system-for-svelte> をご覧ください。

FastComments 用の Svelte ウィジェットとラッパーのセットです。`src/routes` には実行可能なサンプルが含まれています - 次のコマンドを実行してください:

    npm install
    npm run dev

その後、すべてのウィジェットを確認して編集できます - 例えば自分の tenant id を入れたり、異なる設定を試したりできます。

TypeScript が使用されているので、各ウィジェットの config オブジェクトを調べて、受け取るオプションを確認できます。
