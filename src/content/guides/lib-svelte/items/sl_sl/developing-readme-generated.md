---
Ko namestite odvisnosti z `npm install` (ali `pnpm install` ali `yarn`), zaženite razvojni strežnik:

```bash
npm run dev

# ali zaženite strežnik in odprite aplikacijo v novem zavihku brskalnika
npm run dev -- --open
```

Vse, kar je v `src/lib`, je del knjižnice, vse, kar je v `src/routes`, je predstavitvena aplikacija.
