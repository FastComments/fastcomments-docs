Probeer elke widget live op <https://fastcomments.com/commenting-system-for-svelte>.

Een set Svelte-widgets en wrappers voor FastComments. `src/routes` bevat uitvoerbare voorbeelden - voer eenvoudig uit:

    npm install
    npm run dev

Then you can see all the widgets and edit them - for example to put in your own tenant id or try different configuration.

Er is TypeScript gebruikt zodat je de config-objecten voor elke widget kunt inspecteren om te zien welke opties ze ondersteunen.
