Sobald du die Abhängigkeiten mit `npm install` (oder `pnpm install` oder `yarn`) installiert hast, starte einen Entwicklungsserver:

```bash
npm run dev

# oder starte den Server und öffne die App in einem neuen Browser-Tab
npm run dev -- --open
```

Alles in `src/lib` ist Teil der Bibliothek, alles in `src/routes` ist die Beispielanwendung.
