נסו כל הווידג'ט בשידור חי ב־<https://fastcomments.com/commenting-system-for-svelte>.

מערך של ווידג'טים ועטיפות עבור FastComments. `src/routes` מכיל דוגמאות להרצה - פשוט הריצו:

    npm install
    npm run dev

אז תוכלו לראות את כל הווידג'טים ולערוך אותם - למשל כדי להכניס את ה-tenant id שלכם או לנסות קונפיגורציות שונות.

TypeScript שימש כדי שתוכלו לבחון את אובייקטי התצורה של כל ווידג'ט ולראות אילו אפשרויות הם מקבלים.
