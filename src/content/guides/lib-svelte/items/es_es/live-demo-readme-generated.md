Prueba cada widget en vivo en <https://fastcomments.com/commenting-system-for-svelte>.

Un conjunto de widgets y envoltorios de Svelte para FastComments. `src/routes` contiene ejemplos ejecutables - simplemente ejecuta:

    npm install
    npm run dev

Entonces puedes ver todos los widgets y editarlos - por ejemplo para poner tu propio tenant id o probar diferentes configuraciones.

Se ha utilizado TypeScript para que puedas inspeccionar los objetos de configuración de cada widget y ver qué opciones aceptan.
