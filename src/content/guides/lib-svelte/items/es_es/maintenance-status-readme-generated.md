Estos componentes son envoltorios alrededor de nuestros componentes core de VanillaJS. Podemos actualizar automáticamente estos componentes (corregir errores, añadir funciones) sin publicar esta biblioteca, así que, aunque puede que no se publique por un tiempo, ¡eso no significa que FastComments no esté en desarrollo activo! Siéntete libre de consultar [nuestro blog](https://blog.fastcomments.com/) para ver actualizaciones. Los cambios incompatibles de la API o las nuevas funciones nunca se enviarán a la biblioteca core subyacente sin un aumento de versión en esta biblioteca.
