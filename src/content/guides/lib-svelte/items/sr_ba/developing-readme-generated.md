---
Када сте инсталирали зависности помоћу `npm install` (или `pnpm install` или `yarn`), покрените развојни сервер:

```bash
npm run dev

# или покрените сервер и отворите апликацију у новој картици прегледача
npm run dev -- --open
```

Све унутар `src/lib` је део библиотеке, све унутар `src/routes` је демонстрација.
