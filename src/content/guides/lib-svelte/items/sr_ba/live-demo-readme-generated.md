Isprobajte svaki widget uživo na <https://fastcomments.com/commenting-system-for-svelte>.

Skup Svelte widgeta i omotača za FastComments. `src/routes` sadrži primjere koje možete pokrenuti - jednostavno pokrenite:

    npm install
    npm run dev

Nakon toga možete vidjeti sve widgete i uređivati ih - na primjer da unesete vlastiti tenant id ili isprobate drugačiju konfiguraciju.

Korišten je TypeScript kako biste mogli pregledati config objekte za svaki widget i vidjeti koje opcije podržavaju.
