Αυτά τα components είναι περιτυλίγματα γύρω από τα βασικά μας components VanillaJS. Μπορούμε να ενημερώνουμε αυτόματα αυτά τα components (διόρθωση σφαλμάτων, προσθήκη λειτουργιών) χωρίς να δημοσιεύουμε αυτή τη βιβλιοθήκη, οπότε παρόλο που μπορεί να μην δημοσιευτεί για κάποιο διάστημα, αυτό δεν σημαίνει ότι το FastComments δεν βρίσκεται σε ενεργή ανάπτυξη! Μη διστάσετε να ελέγξετε [το ιστολόγιό μας](https://blog.fastcomments.com/) για ενημερώσεις. Αλλαγές που σπάνε το API ή νέες δυνατότητες δεν θα προωθηθούν ποτέ στην υποκείμενη βασική βιβλιοθήκη χωρίς αύξηση της έκδοσης σε αυτή τη βιβλιοθήκη.
