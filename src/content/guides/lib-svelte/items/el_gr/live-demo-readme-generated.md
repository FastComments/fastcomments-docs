Δοκιμάστε κάθε widget ζωντανά στο <https://fastcomments.com/commenting-system-for-svelte>.

Μια συλλογή από Svelte widgets και περιτυλίγματα (wrappers) για το FastComments. Το `src/routes` περιέχει εκτελέσιμα παραδείγματα - απλά εκτελέστε:

    npm install
    npm run dev

Στη συνέχεια μπορείτε να δείτε όλα τα widgets και να τα επεξεργαστείτε - για παράδειγμα για να βάλετε το δικό σας tenant id ή να δοκιμάσετε διαφορετική διαμόρφωση.

Έχει χρησιμοποιηθεί TypeScript ώστε να μπορείτε να εξετάσετε τα config αντικείμενα για κάθε widget για να δείτε ποιες επιλογές δέχονται.
