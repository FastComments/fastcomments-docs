Αυτά τα components είναι wrappers γύρω από τα core VanillaJS components. Μπορούμε να ενημερώνουμε αυτόματα αυτά τα components (διόρθωση σφαλμάτων, προσθήκη δυνατοτήτων) χωρίς να δημοσιεύσουμε αυτή τη βιβλιοθήκη, οπότε, παρόλο που μπορεί να μην δημοσιευτεί για κάποιο διάστημα, αυτό δεν σημαίνει ότι το FastComments δεν βρίσκεται σε ενεργή ανάπτυξη! Μη διστάσετε να επισκεφθείτε [το blog μας](https://blog.fastcomments.com/) για ενημερώσεις. Αλλαγές που σπάνε το API ή νέες λειτουργίες δεν θα προωθηθούν ποτέ στην υποκείμενη βασική βιβλιοθήκη χωρίς αύξηση έκδοσης σε αυτή τη βιβλιοθήκη.
