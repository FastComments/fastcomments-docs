Te komponenty są opakowaniami wokół naszych podstawowych komponentów VanillaJS. Możemy automatycznie aktualizować te komponenty (naprawiać błędy, dodawać funkcje) bez publikowania tej biblioteki, więc choć może nie być ona publikowana przez pewien czas, nie oznacza to, że FastComments nie jest aktywnie rozwijany! Zachęcamy do sprawdzania [naszego bloga](https://blog.fastcomments.com/) w poszukiwaniu aktualizacji. Zmiany łamiące API lub nowe funkcje nigdy nie zostaną wprowadzone do bazowej biblioteki rdzeniowej bez podniesienia wersji tej biblioteki.
