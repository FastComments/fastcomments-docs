Deze componenten zijn wrappers rond onze core VanillaJS-componenten. We kunnen deze componenten automatisch bijwerken (bugs oplossen, functies toevoegen) zonder deze bibliotheek te publiceren, dus hoewel het misschien een tijdje niet gepubliceerd wordt, betekent dat niet dat FastComments niet actief in ontwikkeling is! Bekijk gerust [onze blog](https://blog.fastcomments.com/) voor updates. Compatibiliteitsbrekende API-wijzigingen of -functies zullen nooit naar de onderliggende core-bibliotheek worden geleverd zonder een versieverhoging in deze bibliotheek.
