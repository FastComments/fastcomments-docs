Estes componentes são wrappers em torno dos nossos componentes principais do VanillaJS. Podemos atualizar automaticamente esses componentes (corrigir bugs, adicionar recursos) sem publicar esta biblioteca, então, embora ela possa não ser publicada por um tempo, isso não significa que o FastComments não esteja em desenvolvimento ativo! Fique à vontade para conferir [nosso blog](https://blog.fastcomments.com/) para atualizações. Mudanças ou recursos que quebrem a API nunca serão entregues à biblioteca core subjacente sem um aumento de versão nesta biblioteca.
