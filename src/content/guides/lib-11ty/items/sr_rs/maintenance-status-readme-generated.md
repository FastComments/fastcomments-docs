Ове компоненте су омотачи око наших основних VanillaJS компонената. Можемо аутоматски ажурирати ове компоненте (исправљати грешке, додавати функције) без објављивања ове библиотеке, тако да иако можда неће бити објављена неко време, то не значи да FastComments није у активној развоју! Слободно погледајте [наш блог](https://blog.fastcomments.com/) за ажурирања.
