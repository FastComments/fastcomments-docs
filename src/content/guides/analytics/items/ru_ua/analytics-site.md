Site Analytics, или просто `Analytics` в панели управления, предоставляет обзор того, как ваше сообщество использует FastComments на всех ваших доменах.

FastComments предоставляет некоторые уникальные функции, которые многие другие платформы не предлагают, например **живые** отчёты о пользователях онлайн на каждой странице и сортировку страниц по количеству онлайн-пользователей. Для этого просто посетите [Analytics Page](https://fastcomments.com/auth/my-account/analytics) и нажмите `Sort by users online` в разделе `Top Pages`.

Как общее количество `Users Online`, так и метрики `Top Pages` работают в реальном времени и отображаются без задержки.

`Top Pages` по умолчанию сортирует по количеству комментариев на каждой странице.

Наконец, предоставляется разбивка общих метрик по вашему арендатору, по дням, во времени для:

- Page Loads
  - Это количество раз, когда пользователь открыл страницу, содержащую один или несколько виджетов FastComments. Если страница содержит несколько виджетов, это число будет увеличено на количество виджетов на этой странице. Если у вас SPA, то каждый раз, когда приложение открывает новую ветку комментариев, это число будет увеличиваться. Это также применимо к библиотеке React Native.
  - Эта метрика также используется для целей выставления счетов в планах Flex.
- Comments Left
  - Это включает все комментарии, независимо от статуса верификации или одобрения, или являются ли они спамом или нет.
- Votes Left
  - Это количество оставленных голосов. Будут учитываться только верифицированные голоса, если не включено анонимное голосование.
- Accounts Created
  - Эта метрика относится к случаям, когда добавляется новый SSO-пользователь или когда комментатор впервые комментирует с FastComments, используя ваш сайт.

Эти метрики работают почти в реальном времени с задержкой до одной минуты.
