Όλες οι μετρήσεις για τις οποίες μπορεί να περιοριστείτε ή να χρεωθείτε αναφέρονται στη σελίδα [Billing Analytics](https://fastcomments.com/auth/my-account/analytics/billing).

Μερικά παραδείγματα μετρήσεων που παρακολουθούνται:

- Φορτώσεις σελίδων (για τον τρέχοντα μήνα)
- Πιστώσεις API (για τον τρέχοντα μήνα)
- Νέα αποθηκευμένα σχόλια (για τον τρέχοντα μήνα)
- Χρήστες ενοικιαστή
- Χρήστες SSO (κανονικοί χρήστες SSO χωρίς δικαιώματα διαχειριστή/συντονιστή)
- Διαχειριστές SSO (χρήστες SSO με σημαίες διαχειριστή `isAccountOwner` ή `isAdminAdmin` - χρεώνονται ξεχωριστά από τους κανονικούς χρήστες SSO)
- Συντονιστές SSO (χρήστες SSO με σημαία `isCommentModeratorAdmin` - χρεώνονται ξεχωριστά από τους κανονικούς χρήστες SSO)
- Συντονιστές
- Τομείς

Για τα πλάνα `Creators` και `Pro`, αυτές οι μετρήσεις χρησιμεύουν απλώς ως οδηγός για να γνωρίζετε αν είστε κοντά ή έχετε φτάσει τα όρια του λογαριασμού σας.

Για το πλάνο `Flex`, αυτή η σελίδα θα εμφανίσει ανάλυση κόστους της τρέχουσας χρήσης για το **επόμενο** τιμολόγιο. Αυτό διευκολύνει την κατανόηση του πώς θα χρεωθείτε πριν συμβεί. Μπορείτε να δείτε προηγούμενα τιμολόγια στο [Billing History](https://fastcomments.com/auth/my-account/billing-history).

Οι μετρήσεις `Page Loads`, `API Credits` και `New Comments Stored` μπορεί να είναι μέχρι ένα λεπτό παρωχημένες. Οι άλλες μετρήσεις είναι σε πραγματικό χρόνο.

Αυτές οι πληροφορίες αποθηκεύονται σε μηνιαία βάση για ένα έτος, ή μέχρι να διαγράψετε τον ενοικιαστή σας, για ιστορικούς σκοπούς, αλλά η διεπαφή χρήστη θα εμφανίζει μόνο την τρέχουσα περίοδο.
