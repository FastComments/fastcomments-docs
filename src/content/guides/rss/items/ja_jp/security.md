FastComments の RSS フィードは、一意で推測不能な UUIDs を使用して保護されています。ただし、RSS フィードを有効にすると、その URL を知っている人は誰でもコメントにアクセスできます。

フィードはパスワード保護されていないため、URL は機密情報として扱い、信頼できる RSS リーダーやサービスとだけ共有してください。

Best practices:

- **URL を非公開に保つ**: RSS フィードの URL を公開したり、権限のないユーザーが見つけられる場所で共有したりしないでください。
- **信頼できる RSS リーダーを使う**: コメントデータにアクセスされるため、フィードは信頼できる RSS リーダーサービスにのみ追加してください。
- **アクセスを監視する**: フィードの URL が漏洩した疑いがある場合は、フィードを無効にして再度有効化することで新しい URL を生成できます。
- **不要なときは無効化する**: RSS フィードを使用していない場合は、ダッシュボードで無効にしてアクセスを防いでください。

RSS フィードには、コメントのテキストと HTML コンテンツ、コメント投稿者の名前、コメント日時、コメントが行われたページへのリンク、および投票数が含まれます。

フィードには、コメント投稿者のメールアドレス、IP アドレス、その他の機密性の高いユーザー情報は**含まれていません**。
