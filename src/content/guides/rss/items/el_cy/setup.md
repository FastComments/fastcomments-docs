Μεταβείτε στον πίνακα ελέγχου του λογαριασμού σας και ανοίξτε τη [σελίδα RSS](https://fastcomments.com/auth/my-account/rss).

[app-screenshot-start url='/auth/my-account/rss'; selector = '.content'; title='RSS Feed Configuration' app-screenshot-end]

Στη σελίδα RSS Feed, επιλέξτε το πλαίσιο ελέγχου "Enable RSS Feed Access" για να δημιουργηθεί το μοναδικό σας URL ροής RSS.

Όταν ενεργοποιήσετε τη ροή, θα εμφανιστεί μια ειδοποίηση ασφάλειας που εξηγεί ότι οποιοσδήποτε έχει τον σύνδεσμο μπορεί να αποκτήσει πρόσβαση στα σχόλιά σας.

Μόλις ενεργοποιηθεί, θα εμφανιστεί το μοναδικό σας URL ροής RSS. Κάντε κλικ στο κουμπί "Copy" για να αντιγράψετε το URL στο πρόχειρό σας.

Ανοίξτε την εφαρμογή ανάγνωσης RSS και προσθέστε το αντιγραμμένο URL ως νέα συνδρομή ροής. Τα ακριβή βήματα διαφέρουν ανάλογα με τον RSS reader, αλλά συνήθως θα αναζητήσετε ένα κουμπί "Add Feed" ή "Subscribe", θα επικολλήσετε το URL ροής RSS και θα δώσετε ένα όνομα στη ροή.

Μπορείτε επίσης να κάνετε κλικ στο κουμπί "View Feed" στη σελίδα RSS Feed για να δείτε τη ροή απευθείας στον περιηγητή σας.

Για να απενεργοποιήσετε την πρόσβαση στη ροή RSS, απλά αποεπιλέξτε το πλαίσιο "Enable RSS Feed Access". Αυτό θα εμποδίσει την πρόσβαση στη ροή, αν και το URL θα παραμείνει το ίδιο αν το ενεργοποιήσετε ξανά αργότερα.
