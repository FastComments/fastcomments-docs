FastComments 的 RSS 提要使用唯一且不可猜测的 UUID 进行保护。但是，一旦启用 RSS 提要，任何拥有该 URL 的人都可以访问你的评论。

该提要未受密码保护，因此该 URL 应视为敏感信息，仅与受信任的 RSS 阅读器或服务共享。

最佳实践：

- **保持 URL 私密**: 不要公开分享你的 RSS 提要 URL，或在未经授权的用户可能发现的地方发布。
- **使用受信任的 RSS 阅读器**: 仅将你的提要添加到你信任的 RSS 阅读器服务，因为它们将能够访问你的评论数据。
- **监控访问**: 如果你怀疑提要 URL 已被泄露，可以禁用并重新启用该提要以生成新的 URL。
- **不需要时禁用**: 如果你不再使用 RSS 提要，请在仪表板中将其禁用以防止任何访问。

RSS 提要包含评论文本和 HTML 内容、评论者姓名、评论日期、评论所在页面的链接以及投票计数。

该提要**不**包含评论者的电子邮件地址、IP 地址或其他敏感用户信息。
