Le bloc HTML de Webnode est ce que nous allons utiliser pour intégrer le widget de commentaires. Depuis le 26 juin 2023, ce bloc est une fonctionnalité payante de Webnode : il n’est pas disponible dans le forfait gratuit, et les projets créés après cette date nécessitent un forfait premium pour l’utiliser. Si vous ne voyez pas l’option `HTML` dans l’éditeur, passez à un forfait Webnode supérieur et elle apparaîtra.
