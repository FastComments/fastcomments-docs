Maintenant, insérons un bloc HTML dans cette section, où le fragment de code sera placé. Sélectionnez `HTML` :

> **Note :** À partir du 26 juin 2023, le bloc HTML de Webnode est une fonctionnalité payante. Il n'est pas disponible sur le plan gratuit, et pour les projets créés après cette date il nécessite un forfait premium. Si vous ne voyez pas l'option `HTML`, passez à un plan Webnode supérieur pour l'activer.
