Vous pouvez obtenir une erreur d'autorisation, comme la suivante :

<div class="screenshot white-bg">
    <div class="title">Configuration de domaine manquante</div>
    <img class="screenshot-image" src="/images/installation-guides/installation-guide-webflow-step-5.png" alt="Configuration de domaine manquante" />
</div>

C'est parce que FastComments ne sait pas qu'il est censé autoriser l'utilisation de votre compte sur ce domaine.

Dans ce cas, la solution est simple. Il vous suffit d'ajouter votre site à votre compte FastComments.com.

[Allez ici pour ajouter votre site à votre compte.](https://fastcomments.com/auth/my-account/configure-domains)
