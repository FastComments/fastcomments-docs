С помощью FastComments мы можем легко добавить комментарии в реальном времени на любой сайт BigCommerce.

Инструкции ниже предназначены для блога на BigCommerce, но они также подойдут для любой части BigCommerce.

Обратите внимание, что для этого руководства требуется учётная запись FastComments. Рекомендуется сначала зарегистрироваться, а затем вернуться сюда. [Вы можете создать учётную запись здесь.](https://fastcomments.com/auth/tenant-signup?packageId=flex)
