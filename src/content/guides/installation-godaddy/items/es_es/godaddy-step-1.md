Para añadir FastComments a nuestro sitio de Godaddy vamos a añadir un área para colocar el código del widget de FastComments.com.

**¡Nota!** Asegúrate de haber añadido una sección de blog y al menos una entrada de blog en tu sitio antes de continuar.

Vamos a añadir un área de código HTML moviendo el ratón por debajo de la entrada del blog en el editor del sitio y haciendo clic en `Add Section` como sigue:

<div class="screenshot white-bg">
    <div class="title">Añadir sección</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-section.png" alt="Add Section" />
</div>

Ahora busca `HTML`. Una vez que aparezca el cuadro "Insertar código personalizado en la página", haz clic en `Add`.

<div class="screenshot white-bg">
    <div class="title">Añadir cuadro HTML</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-html.png" alt="Add HTML Box" />
</div>

Ahora deberías ver un área de código vacía en el editor.

<div class="screenshot white-bg">
    <div class="title">Área de código vacía</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-section-added.png" alt="Empty Code Area" />
</div>

Haz clic en el área de código vacía que has añadido. En la esquina inferior derecha ahora verás un campo de entrada `Custom Code`.

<div class="screenshot white-bg">
    <div class="title">Área de entrada de código personalizado</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-code-area.png" alt="Custom Code Input Area" />
</div>

¡Eso es todo para el primer paso! En el paso dos añadiremos nuestro código personalizado específico de Godaddy.
