Um FastComments zu unserer Godaddy-Seite hinzuzufügen, fügen wir einen Bereich hinzu, in den der FastComments.com-Widget-Code eingefügt wird.

**Hinweis!** Stellen Sie sicher, dass Sie einen Blog-Bereich und mindestens einen Blogbeitrag zu Ihrer Seite hinzugefügt haben, bevor Sie fortfahren.

Fügen wir einen HTML-Codebereich hinzu, indem wir im Seiteneditor die Maus unter den Blogbeitrag bewegen und auf `Add Section` klicken, wie folgt:

<div class="screenshot white-bg">
    <div class="title">Abschnitt hinzufügen</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-section.png" alt="Abschnitt hinzufügen" />
</div>

Suchen Sie nun nach `HTML`. Sobald das Feld "Embed custom code on the page" angezeigt wird, klicken Sie auf `Add`.

<div class="screenshot white-bg">
    <div class="title">HTML-Box hinzufügen</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-html.png" alt="HTML-Box hinzufügen" />
</div>

Sie sollten nun einen leeren Codebereich im Editor sehen.

<div class="screenshot white-bg">
    <div class="title">Leerer Codebereich</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-section-added.png" alt="Leerer Codebereich" />
</div>

Klicken Sie auf den hinzugefügten leeren Codebereich. Unten rechts sehen Sie nun ein `Custom Code`-Eingabefeld.

<div class="screenshot white-bg">
    <div class="title">Eingabebereich für benutzerdefinierten Code</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-code-area.png" alt="Eingabebereich für benutzerdefinierten Code" />
</div>

Das war Schritt eins! In Schritt zwei fügen wir unseren Godaddy-spezifischen Code hinzu.
