Για να προσθέσουμε τα FastComments στην ιστοσελίδα μας στο Godaddy, θα προσθέσουμε μια περιοχή για να τοποθετήσουμε τον κώδικα του widget από το FastComments.com.

**Σημείωση!** Βεβαιωθείτε ότι έχετε προσθέσει μια ενότητα blog και τουλάχιστον μια δημοσίευση blog στην ιστοσελίδα σας πριν συνεχίσετε.

Ας προσθέσουμε μια περιοχή κώδικα HTML μετακινώντας το ποντίκι μας κάτω από τη δημοσίευση του blog στον επεξεργαστή της σελίδας και κάνοντας κλικ στο `Add Section` ως εξής:

<div class="screenshot white-bg">
    <div class="title">Προσθήκη Ενότητας</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-section.png" alt="Add Section" />
</div>

Τώρα αναζητήστε `HTML`. Μόλις εμφανιστεί το πλαίσιο "Embed custom code on the page", κάντε κλικ στο `Add`.

<div class="screenshot white-bg">
    <div class="title">Προσθήκη Πλαισίου HTML</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-html.png" alt="Add HTML Box" />
</div>

Τώρα θα πρέπει να βλέπετε μια κενή περιοχή κώδικα στον επεξεργαστή.

<div class="screenshot white-bg">
    <div class="title">Κενή Περιοχή Κώδικα</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-section-added.png" alt="Empty Code Area" />
</div>

Κάντε κλικ στην κενή περιοχή κώδικα που προσθέσατε. Στο κάτω δεξιά μέρος θα δείτε τώρα ένα πεδίο εισόδου `Custom Code`.

<div class="screenshot white-bg">
    <div class="title">Πεδίο Εισόδου Προσαρμοσμένου Κώδικα</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-code-area.png" alt="Custom Code Input Area" />
</div>

Αυτό ήταν για το πρώτο βήμα! Στο δεύτερο βήμα θα προσθέσουμε τον προσαρμοσμένο κώδικα ειδικά για το Godaddy.
