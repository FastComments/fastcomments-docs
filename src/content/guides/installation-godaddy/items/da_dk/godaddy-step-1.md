For at tilføje FastComments til vores Godaddy-site vil vi tilføje et område til at placere FastComments.com-widget-koden.

**Bemærk!** Sørg for, at du har føjet en blogsektion og mindst et blogindlæg til dit site, før du fortsætter.

Lad os tilføje et HTML-kodeområde ved at flytte musen under blogindlægget i site-editoren, og klikke på `Add Section` som følger:

<div class="screenshot white-bg">
    <div class="title">Add Section</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-section.png" alt="Add Section" />
</div>

Søg nu efter `HTML`. Når boksen "Embed custom code on the page" vises, klik på `Add`.

<div class="screenshot white-bg">
    <div class="title">Add HTML Box</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-html.png" alt="Add HTML Box" />
</div>

Du skulle nu kunne se et tomt kodeområde i editoren.

<div class="screenshot white-bg">
    <div class="title">Empty Code Area</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-section-added.png" alt="Empty Code Area" />
</div>

Klik på det tomme kodeområde, du har tilføjet. Nederst til højre vil du nu se et `Custom Code` inputfelt.

<div class="screenshot white-bg">
    <div class="title">Custom Code Input Area</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-code-area.png" alt="Custom Code Input Area" />
</div>

Det var det for trin et! I trin to vil vi tilføje vores Godaddy-specifikke kode.
