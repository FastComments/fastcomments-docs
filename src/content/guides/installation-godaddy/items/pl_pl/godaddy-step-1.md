Aby dodać FastComments do naszej strony Godaddy, dodamy obszar na kod widżetu FastComments.com.

**Uwaga!** Upewnij się, że dodałeś sekcję bloga i przynajmniej jeden wpis na blogu na swojej stronie przed kontynuowaniem.

Dodajmy obszar kodu HTML, przesuwając mysz pod wpis na blogu w edytorze strony i klikając `Add Section` w następujący sposób:

<div class="screenshot white-bg">
    <div class="title">Dodaj sekcję</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-section.png" alt="Add Section" />
</div>

Teraz wyszukaj `HTML`. Gdy pojawi się pole "Embed custom code on the page", kliknij `Add`.

<div class="screenshot white-bg">
    <div class="title">Dodaj pole HTML</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-html.png" alt="Add HTML Box" />
</div>

Powinieneś teraz zobaczyć pusty obszar kodu w edytorze.

<div class="screenshot white-bg">
    <div class="title">Pusty obszar kodu</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-section-added.png" alt="Empty Code Area" />
</div>

Kliknij na dodany pusty obszar kodu. W prawym dolnym rogu zobaczysz teraz pole `Custom Code`.

<div class="screenshot white-bg">
    <div class="title">Pole wejściowe Custom Code</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-code-area.png" alt="Custom Code Input Area" />
</div>

To wszystko w kroku pierwszym! W kroku drugim dodamy nasz niestandardowy kod specyficzny dla Godaddy.
