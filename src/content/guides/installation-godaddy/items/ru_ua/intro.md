С помощью FastComments мы можем легко добавить возможность живого комментирования на любой сайт, созданный с помощью Godaddy.

Обратите внимание, что для этого руководства требуется аккаунт FastComments. Рекомендуется сначала зарегистрироваться, а затем вернуться сюда. [Вы можете создать аккаунт здесь.](https://fastcomments.com/auth/tenant-signup?packageId=flex)

Мы покажем, как добавить FastComments в блог, созданный с помощью Godaddy, но те же общие инструкции применимы к любому сайту.
