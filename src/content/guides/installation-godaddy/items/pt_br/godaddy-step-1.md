Para adicionar o FastComments ao nosso site Godaddy, vamos adicionar uma área para colocar o código do widget do FastComments.com.

**Observação!** Certifique-se de ter adicionado uma seção de blog e pelo menos uma postagem ao seu site antes de continuar.

Vamos adicionar uma área de código HTML movendo o mouse abaixo da postagem do blog no editor do site e clicando em `Add Section` como segue:

<div class="screenshot white-bg">
    <div class="title">Adicionar Seção</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-section.png" alt="Adicionar Seção" />
</div>

Agora pesquise por `HTML`. Quando a caixa "Incorporar código personalizado na página" aparecer, clique em `Add`.

<div class="screenshot white-bg">
    <div class="title">Adicionar Caixa HTML</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-html.png" alt="Adicionar Caixa HTML" />
</div>

Você agora deve ver uma área de código vazia no editor.

<div class="screenshot white-bg">
    <div class="title">Área de Código Vazia</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-section-added.png" alt="Área de Código Vazia" />
</div>

Clique na área de código vazia que você adicionou. No canto inferior direito você verá agora um campo de entrada `Custom Code`.

<div class="screenshot white-bg">
    <div class="title">Área de Entrada de Código Personalizado</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-code-area.png" alt="Área de Entrada de Código Personalizado" />
</div>

Isso é tudo para a etapa um! Na etapa dois vamos adicionar nosso código específico do Godaddy.
