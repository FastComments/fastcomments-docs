Pour ajouter FastComments à notre site Godaddy, nous allons ajouter une zone pour insérer le code du widget FastComments.com.

**Remarque !** Assurez-vous d'avoir ajouté une section blog et au moins un article à votre site avant de continuer.

Ajoutons une zone de code HTML en déplaçant notre souris sous l'article du blog dans l'éditeur du site, et en cliquant sur `Add Section` comme suit :

<div class="screenshot white-bg">
    <div class="title">Ajouter une section</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-section.png" alt="Ajouter une section" />
</div>

Maintenant, recherchez `HTML`. Une fois que la boîte "Embed custom code on the page" apparaît, cliquez sur `Add`.

<div class="screenshot white-bg">
    <div class="title">Ajouter une zone HTML</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-html.png" alt="Ajouter une zone HTML" />
</div>

Vous devriez maintenant voir une zone de code vide dans l'éditeur.

<div class="screenshot white-bg">
    <div class="title">Zone de code vide</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-section-added.png" alt="Zone de code vide" />
</div>

Cliquez sur la zone de code vide que vous avez ajoutée. En bas à droite, vous verrez maintenant un champ de saisie `Custom Code`.

<div class="screenshot white-bg">
    <div class="title">Zone de saisie de code personnalisé</div>
    <img class="screenshot-image" src="/images/installation-guides/godaddy-step-1-add-code-area.png" alt="Zone de saisie de code personnalisé" />
</div>

C'est tout pour la première étape ! Dans la deuxième étape, nous ajouterons notre code personnalisé spécifique à Godaddy.
