Die Verwendung von FastComments Geplante Kommentare ist einfach. Zunachst mochten Sie [die Kommentare hier importieren](https://fastcomments.com/auth/my-account/manage-data/import-scheduled).

Diese Seite kann auch uber `Manage Data->Schedule Comments` aufgerufen werden.

### Manuell abgespielte Kommentare

*Fur manuell* abgespielte Kommentare (Sie mussen manuell auf Play drucken) haben Sie die Moglichkeit, die Wiedergabe von Kommentaren zu starten. Dies spielt die Kommentare auf jeder Seite ab, die Sie in der CSV-
Datei definiert haben, mit den Verzogerungen zwischen jedem Kommentar basierend auf der von Ihnen angegebenen Verzogerung.

Dies ist nutzlich, wenn Sie ein geplantes Live-Webinar haben, das zu einem bestimmten Zeitpunkt beginnt. Wenn das Webinar beginnt, drucken Sie einfach auf Play
im Dashboard.

### Automatisch abgespielte Kommentare

*Fur automatisch* abgespielte Kommentare werden die Kommentare bei jedem Seitenladevorgang fur jeden Benutzer abgespielt.

Dies ist nutzlich fur Szenarien, in denen Videos oder andere Inhalte bei jedem Seitenladevorgang von Anfang an beginnen.

### Dynamisch wachsende Wiedergabe

Jedes Mal, wenn das Autoplay-Skript fur einen Benutzer ausgefuhrt wird - beim Laden der Seite - gibt es immer noch die
Moglichkeit fur andere zu kommentieren.

Wenn Leute Kommentare hinterlassen, werden ihre Kommentare **automatisch zum Wiedergabe-
Skript hinzugefugt** mit demselben Offset vom Seitenladen, sodass die Konversation ohne
manuelle Arbeit weiter wachst.

Sie konnen zusatzlich die geposteten Kommentare **moderieren**, um auszuwahlen, welche Kommentare Sie
jedes Mal anzeigen mochten, wenn das Autoplay-Skript ausgefuhrt wird.

Die Seite `Moderate Comments` zeigt auch einen Zeitstempel wie `AutoPlay 1hr 2m 30s` neben jedem
Kommentar anstelle des Datums an.

Dies ist nur fur automatische Wiedergabe verfugbar, nicht fur manuell geplante Wiedergabe.

### Konfiguration

Jeder Kommentar wird **live** gepostet. Sie mochten vielleicht [das sofortige Anzeigen von Live-Kommentaren aktivieren](/guide-customizations-and-configuration.html#show-live-right-away).

Sie konnen mehr uber das Importformat im Abschnitt Importformat dieser Dokumentation erfahren.
