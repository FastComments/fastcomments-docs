### פרטי הפורמט

קובץ ה-CSV של תגובות מתוזמנות תומך בעמודות הבאות:

- ID
- Parent ID
- URL ID
- URL
- Name
- Avatar
- Comment
- Hours
- Minutes
- Seconds

העמודות הבאות הן **נדרשות**:

- ID
- URL ID
- Name
- Comment
- Hours
- Minutes
- Seconds

תצטרך את עמודת Parent ID אם אתה רוצה לתמוך בתשובות מקוננות אוטומטיות.

### איך הפורמט עובד

פורמט הייבוא עובד כך:

1. אתה מגדיר שורה ב-CSV לכל תגובה שאתה רוצה לפרסם.
2. התגובה מפורסמת אחרי ההשהיה שצוינה (שעות + דקות + שניות).
   1. עבור ייבוא מתוזמן ידנית, ההשהיות הן יחסיות למתי שאתה לוחץ על "play" בממשק המשתמש, אחרי שהייבוא מסתיים - **לא כשהייבוא מתחיל**.
   2. עבור ייבוא מתוזמן אוטומטית, ההשהיה היא מרגע טעינת הדף.
3. אתה חייב להגדיר ID. אתה יכול פשוט להשתמש במזהים עולים כמו 1, 2, 3, 4, 5.
4. אם אתה רוצה להשתמש בתשובות מקוננות, פשוט הגדר את ערך העמודה `Parent ID` ל-`ID` של תגובה אחרת.
5. השדה `Comment` תומך באותו תחביר ש-FastComments תומך בווידג'ט התגובות שלו לעיצוב טקסט והוספת תמונות.
6. השדה `Avatar`, אם משתמשים בו, חייב להיות תמונה נגישה לציבור. היא תועתק ותוגש מה-CDN שלנו.
7. אתה יכול למחוק את כל התגובות אחרי ההפעלה, או אם ההפעלה נעצרת.
8. המחיקה קורית בזמן אמת, כך שאתה יכול לעשות שימוש חוזר באותו ייבוא מתוזמן שוב ושוב.
