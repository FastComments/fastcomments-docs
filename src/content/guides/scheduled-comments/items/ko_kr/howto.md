FastComments 예약 댓글 사용은 간단합니다. 먼저 [여기에서 댓글을 가져오기](https://fastcomments.com/auth/my-account/manage-data/import-scheduled)하세요.

이 페이지는 `Manage Data->Schedule Comments`를 통해서도 접근할 수 있습니다.

### 수동 재생 댓글

*수동으로* 재생되는 댓글(수동으로 Play를 눌러야 함)의 경우, 댓글 재생을 시작할 수 있는 옵션이 있습니다. 이렇게 하면 CSV 파일에서 정의한 각 페이지에서
지정한 지연 시간에 따라 각 댓글 사이의 지연으로 댓글이 재생됩니다.

이것은 특정 시간에 시작하는 예정된 라이브 웨비나가 있을 때 유용합니다. 웨비나가 시작되면 대시보드에서 Play를 누르기만 하면 됩니다.

### 자동 재생 댓글

*자동으로* 재생되는 댓글의 경우, 각 사용자의 페이지 로드 시마다 댓글이 재생됩니다.

이것은 비디오나 기타 콘텐츠가 각 로드마다 처음부터 시작하는 시나리오에 유용합니다.

### 동적으로 성장하는 재생

자동 재생 스크립트가 사용자에 대해 실행될 때마다 - 페이지 로드 시 - 다른 사람들이 댓글을 달
기회가 여전히 있습니다.

사람들이 댓글을 남기면 해당 댓글은 페이지 로드로부터 동일한 오프셋에서 **재생 스크립트에 자동으로 추가**되므로
수동 작업 없이 대화가 계속 성장합니다.

또한 자동 재생 스크립트가 실행될 때마다 표시할 댓글을 선택하기 위해 게시된 댓글을
**관리**할 수 있습니다.

`Moderate Comments` 페이지에는 날짜 대신 각 댓글 옆에 `AutoPlay 1hr 2m 30s`와 같은
타임스탬프도 표시됩니다.

이것은 자동 재생에서만 사용 가능하며, 수동으로 예약된 재생에서는 사용할 수 없습니다.

### 구성

모든 댓글은 **실시간으로** 게시됩니다. [실시간 댓글을 즉시 표시하기 설정](/guide-customizations-and-configuration.html#show-live-right-away)을 고려해 볼 수 있습니다.

이 문서의 가져오기 형식 섹션에서 가져오기 형식에 대해 알아볼 수 있습니다.
