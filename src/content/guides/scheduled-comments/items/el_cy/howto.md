Η χρηση των FastComments Προγραμματισμενων Σχολιων ειναι απλη. Πρωτα, θα θελετε να [εισαγετε τα σχολια εδω](https://fastcomments.com/auth/my-account/manage-data/import-scheduled).

Αυτη η σελιδα μπορει επισης να προσπελαστει μεσω `Manage Data->Schedule Comments`.

### Χειροκινητα αναπαραγομενα σχολια

*Για χειροκινητα* αναπαραγομενα σχολια (πρεπει να πατησετε χειροκινητα το Play), εχετε την επιλογη να ξεκινησετε την αναπαραγωγη σχολιων. Αυτο θα αναπαραγει τα σχολια σε καθε σελιδα που εχετε οριστει στο CSV
αρχειο, με τις καθυστερησεις μεταξυ καθε σχολιου βασισμενες στην καθυστερηση που καθοριστε.

Αυτο ειναι χρησιμο οταν εχετε ενα προγραμματισμενο ζωντανο webinar που ξεκινα σε μια συγκεκριμενη ωρα. Οταν το webinar ξεκινησει, απλα πατηστε Play
στον πινακα ελεγχου.

### Αυτοματα αναπαραγομενα σχολια

*Για αυτοματα* αναπαραγομενα σχολια, τα σχολια αναπαραγονται σε καθε φορτωση σελιδας για καθε χρηστη.

Αυτο ειναι χρησιμο για σεναρια οπου βιντεο η αλλο περιεχομενο ξεκινα απο την αρχη με καθε φορτωση.

### Δυναμικα αυξανομενη αναπαραγωγη

Καθε φορα που το script αυτοματης αναπαραγωγης εκτελειται για εναν χρηστη - κατα τη φορτωση της σελιδας - υπαρχει ακομα
ευκαιρια για αλλους να σχολιασουν.

Καθως οι ανθρωποι αφηνουν σχολια, τα σχολια τους **προστιθενται αυτοματα στο script
αναπαραγωγης** στην ιδια μετατοπιση απο τη φορτωση της σελιδας, ετσι η συζητηση συνεχιζει να αναπτυσσεται χωρις
χειροκινητη εργασια.

Μπορειτε επιπλεον να **συντονισετε** τα δημοσιευμενα σχολια, για να επιλεξετε ποια σχολια θελετε να εμφανιζονται
καθε φορα που εκτελειται το script αυτοματης αναπαραγωγης.

Η σελιδα `Moderate Comments` θα εμφανιζει επισης μια χρονικη σημανση οπως `AutoPlay 1hr 2m 30s` διπλα σε καθε
σχολιο αντι για την ημερομηνια.

Αυτο ειναι διαθεσιμο μονο για αυτοματη αναπαραγωγη, οχι για χειροκινητα προγραμματισμενη αναπαραγωγη.

### Διαμορφωση

Καθε σχολιο θα δημοσιευτει **ζωντανα**. Ισως θελετε να εξετασετε το [ενεργοποιηση εμφανισης ζωντανων σχολιων αμεσως](/guide-customizations-and-configuration.html#show-live-right-away).

Μπορειτε να μαθετε για τη μορφη εισαγωγης στην ενοτητα Μορφη Εισαγωγης αυτης της τεκμηριωσης.
