### Παραδειγμα μορφης

Το CSV προγραμματισμενων σχολιων μοιαζει ετσι:

|ID |Parent ID|URL ID       |URL|Name         |Avatar|Comment                              | Hours | Minutes | Seconds |
|---|---------|-------------|---|-------------|------|-------------------------------------|-------|---------|---------|
|1  |         |scheduledtest|   |Test Person 1|      |Hello!                               | 0     | 0       | 3       |
|2  |1        |scheduledtest|   |Test Person 2|      |How are you?                         | 0     | 0       | 10      |
|3  |2        |scheduledtest|   |Test Person 3|      |Leave them alone                     | 0     | 1       | 30      |
|4  |         |scheduledtest|   |Test Person 4|      |Does anyone have a cute cat to share?| 1     | 10      | 0       |


### Λεπτομερειες μορφης

Το αρχειο CSV προγραμματισμενων σχολιων υποστηριζει τις ακολουθες στηλες:

- ID
- Parent ID
- URL ID
- URL
- Name
- Avatar
- Comment
- Hours
- Minutes
- Seconds

Οι ακολουθες στηλες ειναι **απαραιτητες**:

- ID
- URL ID
- Name
- Comment
- Hours
- Minutes
- Seconds

Θα χρειαστειτε τη στηλη Parent ID αν θελετε να υποστηριξετε αυτοματοποιημενες ενθετες απαντησεις.

### Πως λειτουργει η μορφη

Η μορφη εισαγωγης λειτουργει ως εξης:

1. Οριζετε μια γραμμη στο CSV για καθε σχολιο που θελετε να δημοσιευτει.
2. Το σχολιο δημοσιευεται μετα την καθορισμενη καθυστερηση (ωρες + λεπτα + δευτερολεπτα).
   1. Για χειροκινητα προγραμματισμενες εισαγωγες, οι καθυστερησεις ειναι σχετικες με το ποτε πατατε "play" στη διεπαφη χρηστη, αφου ολοκληρωθει η εισαγωγη - **οχι οταν ξεκινα η εισαγωγη**.
   2. Για αυτοματα προγραμματισμενες εισαγωγες, η καθυστερηση ειναι απο τη στιγμη της φορτωσης της σελιδας.
3. Πρεπει να οριστε ενα ID. Μπορειτε απλα να χρησιμοποιησετε αυξανομενους αναγνωριστες οπως 1, 2, 3, 4, 5.
4. Αν θελετε να χρησιμοποιησετε ενθετες απαντησεις, απλα οριστε την τιμη της στηλης `Parent ID` στο `ID` ενος αλλου σχολιου.
5. Το πεδιο `Comment` υποστηριζει την ιδια συνταξη που υποστηριζει το FastComments στο widget σχολιων του για μορφοποιηση κειμενου και προσθηκη εικονων.
6. Το πεδιο `Avatar`, αν χρησιμοποιηθει, πρεπει να ειναι μια δημοσια προσβασιμη εικονα. Θα αντιγραφει και θα εξυπηρετηθει απο το CDN μας.
7. Μπορειτε να διαγραψετε ολα τα σχολια μετα την αναπαραγωγη, η αν η αναπαραγωγη σταματησει.
8. Η διαγραφη συμβαινει ζωντανα, ωστε μπορειτε να επαναχρησιμοποιησετε την ιδια προγραμματισμενη εισαγωγη ξανα και ξανα.
