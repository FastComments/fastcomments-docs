Запланированные комментарии - это функция FastComments для воспроизведения комментариев в течение определенного периода времени, создавая впечатление, что
настоящие пользователи оставляют комментарии. Например, вы можете иметь один комментарий, опубликованный через несколько секунд, а затем через несколько
секунд еще один комментарий в качестве ответа, и так далее. Каждый комментарий может иметь собственное имя пользователя, аватар, по желанию.

Это может быть полезно для вебинаров или других мероприятий, где может быть желательно начать разговоры автоматизированным способом.
