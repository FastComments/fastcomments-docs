Использование FastComments Запланированных комментариев простое. Сначала вы захотите [импортировать комментарии здесь](https://fastcomments.com/auth/my-account/manage-data/import-scheduled).

Эту страницу также можно открыть через `Manage Data->Schedule Comments`.

### Комментарии с ручным воспроизведением

*Для комментариев* с ручным воспроизведением (вам нужно вручную нажать Play), у вас есть возможность начать воспроизведение комментариев. Это воспроизведет комментарии на каждой странице, которую вы определили в CSV-
файле, с задержками между каждым комментарием на основе указанной вами задержки.

Это полезно, когда у вас запланирован прямой вебинар, который начинается в определенное время. Когда вебинар начнется, просто нажмите Play
на панели управления.

### Автоматическое воспроизведение комментариев

*Для автоматически* воспроизводимых комментариев, комментарии воспроизводятся при каждой загрузке страницы для каждого пользователя.

Это полезно для сценариев, где видео или другой контент начинается сначала при каждой загрузке.

### Динамически растущее воспроизведение

Каждый раз, когда скрипт автовоспроизведения запускается для пользователя - при загрузке страницы - все еще есть
возможность для других комментировать.

Когда люди оставляют комментарии, их комментарии **автоматически добавляются в скрипт
воспроизведения** с тем же смещением от загрузки страницы, поэтому разговор продолжает расти без
ручной работы.

Вы также можете **модерировать** опубликованные комментарии, чтобы выбрать какие комментарии вы хотите показывать
каждый раз, когда запускается скрипт автовоспроизведения.

Страница `Moderate Comments` также будет показывать временную метку вроде `AutoPlay 1hr 2m 30s` рядом с каждым
комментарием вместо даты.

Это доступно только для автоматического воспроизведения, не для воспроизведения с ручным расписанием.

### Конфигурация

Каждый комментарий будет опубликован **в реальном времени**. Вы можете рассмотреть [включение показа комментариев в реальном времени сразу](/guide-customizations-and-configuration.html#show-live-right-away).

Вы можете узнать о формате импорта в разделе Формат импорта этой документации.
