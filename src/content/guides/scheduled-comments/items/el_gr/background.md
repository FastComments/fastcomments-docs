Τα Προγραμματισμένα Σχόλια είναι μια λειτουργία του FastComments για αναπαραγωγή σχολίων σε μια χρονική περίοδο, κάνοντας να φαίνεται σαν
πραγματικοί χρήστες να αφήνουν σχόλια. Για παράδειγμα, θα μπορούσατε να έχετε ένα σχόλιο να δημοσιεύεται μετά από μερικά δευτερόλεπτα, και μετά μερικά
δευτερόλεπτα αργότερα ένα άλλο σχόλιο ως απάντηση, και ούτω καθεξής. Κάθε σχόλιο μπορεί να έχει το δικό του όνομα χρήστη, avatar, όπως επιθυμείται.

Αυτό μπορεί να είναι χρήσιμο για webinars ή άλλες εκδηλώσεις όπου μπορεί να είναι επιθυμητό να ξεκινήσουν συζητήσεις με αυτοματοποιημένο τρόπο.
