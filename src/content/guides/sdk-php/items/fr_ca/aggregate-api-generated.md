Agrège les documents en les regroupant (si groupBy est fourni) et en appliquant plusieurs opérations.  
Différentes opérations (p. ex. sum, countDistinct, avg, etc.) sont prises en charge.

## Paramètres

| Nom | Type | Emplacement | Obligatoire | Description |
|------|------|----------|----------|-------------|
| tenantId | string | query | Oui |  |
| parentTenantId | string | query | Non |  |
| includeStats | boolean | query | Non |  |

## Réponse

Renvoie : [`AggregateResponse`](https://github.com/FastComments/fastcomments-php/blob/main/lib/Model/AggregateResponse.php)
