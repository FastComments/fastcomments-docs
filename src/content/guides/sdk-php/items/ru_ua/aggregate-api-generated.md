Aggregates документы, группируя их (если предоставлен `groupBy`) и применяя несколько операций. Поддерживаются различные операции (например, sum, countDistinct, avg и т.д.).

## Параметры

| Имя | Тип | Location | Обязательно | Описание |
|------|------|----------|--------------|----------|
| tenantId | string | query | Да |  |
| parentTenantId | string | query | Нет |  |
| includeStats | boolean | query | Нет |  |

## Ответ

Returns: [`AggregateResponse`](https://github.com/FastComments/fastcomments-php/blob/main/lib/Model/AggregateResponse.php)
