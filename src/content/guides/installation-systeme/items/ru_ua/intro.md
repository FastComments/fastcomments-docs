С помощью FastComments мы можем легко добавить живые комментарии в любой блог на Systeme.io.

Инструкции ниже предназначены для блога, созданного на Systeme.io, но код встраивания также можно вручную добавить на любую страницу, редактируя другие файлы темы. Служба поддержки FastComments также может помочь с этим.

Обратите внимание, что для этого руководства требуется аккаунт FastComments. Рекомендуется сначала зарегистрироваться, а затем вернуться сюда. [Вы можете создать аккаунт здесь.](https://fastcomments.com/auth/tenant-signup?packageId=flex)
