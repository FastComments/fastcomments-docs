С FastComments мы можем легко добавить комментарии в реальном времени в любой блог на Systeme.io.

Инструкции ниже предназначены для блога, созданного на Systeme.io, но встраиваемый код также можно вручную добавить на любую страницу, отредактировав другие файлы темы. Служба поддержки FastComments также может помочь с этим.

Обратите внимание, что для этого руководства требуется аккаунт FastComments. Рекомендуется сначала зарегистрироваться, а затем вернуться сюда. [Вы можете создать аккаунт здесь.](https://fastcomments.com/auth/tenant-signup?packageId=flex)
