#### Відкрийте Developer Keys у Canvas

Увійдіть до Canvas як адміністратор. Перейдіть до **Admin** (у лівій бічній панелі) > виберіть свій обліковий запис > **Developer Keys**.

#### Створіть LTI Developer Key

Натисніть **+ Developer Key** та оберіть **LTI Key**.

У формі конфігурації:

1. Встановіть **Method** у **Enter URL**.
2. Вставте **Configuration URL**, який ви скопіювали з FastComments, у поле URL.
3. Canvas автоматично завантажить конфігурацію LTI.
4. Дайте ключу ім'я (наприклад, "FastComments").
5. Натисніть **Save**.

#### Увімкніть Developer Key

Після збереження новий ключ з'явиться в таблиці Developer Keys з **State**, встановленим у **OFF**. Натисніть перемикач, щоб встановити **ON**.

#### Скопіюйте Client ID

У таблиці Developer Keys у стовпці Details показано числовий **Client ID** (наприклад `17000000000042`). Скопіюйте це число - воно знадобиться вам на наступному кроці.
