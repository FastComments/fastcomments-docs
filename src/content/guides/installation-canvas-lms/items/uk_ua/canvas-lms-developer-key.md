#### Відкрийте Ключі розробника в Canvas

Увійдіть до Canvas як адміністратор. Перейдіть до **Admin** (у лівій бічній панелі) > оберіть свій обліковий запис > **Developer Keys**.

#### Create an LTI Developer Key

Клацніть **+ Developer Key** і оберіть **LTI Key**.

У формі конфігурації:

1. У полі **Redirect URIs** (ліворуч) вставте **Launch URL** зі сторінки налаштування FastComments.
2. Праворуч встановіть **Method** на **Enter URL**.
3. Вставте **Configuration URL**, який ви скопіювали з FastComments, у поле **JSON URL**.
4. Canvas автоматично завантажить конфігурацію LTI.
5. Дайте ключу ім'я (наприклад "FastComments").
6. Натисніть **Save**.

#### Enable the Developer Key

Після збереження новий ключ з’явиться в таблиці Developer Keys зі станом **State**, встановленим на **OFF**. Клацніть перемикач, щоб встановити його на **ON**. Canvas може попросити підтвердження — натисніть **Allow**, щоб увімкнути ключ.

#### Copy the Client ID

У таблиці Developer Keys у стовпці Details відображається числовий **Client ID** (наприклад `17000000000042`). Скопіюйте це число — воно знадобиться на наступному кроці.
