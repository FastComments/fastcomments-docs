#### Перейдіть до налаштувань Canvas LTI

Увійдіть у свій обліковий запис FastComments та перейдіть до <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Мій обліковий запис > Конфігурація Canvas LTI</a>.

Сторінка налаштування використовує покроковий майстер, який проведе вас через кожен етап. Ви почнете з **Крок 1: Створення**.

#### Створення нової конфігурації LTI

Заповніть наступні поля:

- **Configuration Name** - необов’язкова мітка для ідентифікації цієї конфігурації (корисно, якщо ви підключаєте кілька екземплярів Canvas).
- **Platform URL** - URL вашого екземпляру Canvas (наприклад, `https://yourschool.instructure.com`). Це поле обов’язкове.

#### Виберіть розміщення

Під полем Platform URL ви можете обрати, де FastComments з’являтиметься у ваших курсах Canvas. Обидва варіанти увімкнені за замовчуванням:

- **Assignment View** - коментарі автоматично відображаються під завданнями.
- **Rich Content Editor Button** - викладачі можуть вбудовувати FastComments на Сторінки, Тести та Оголошення за допомогою кнопки на панелі інструментів Редактора багатого вмісту.

Ви можете увімкнути один або обидва варіанти. Має бути увімкнено щонайменше одне розміщення.

Натисніть **Створити конфігурацію**.

#### Скопіюйте URL конфігурації

Після створення конфігурації майстер переходить до **Крок 2: Підключення**. На верхній частині сторінки велике поле показує **Configuration URL** з кнопкою **Копіювати**.

Скопіюйте цей URL. Він знадобиться вам на наступному кроці для створення Developer Key у Canvas.

Сторінка також показує пронумеровані інструкції щодо дій у Canvas, а також поля для введення **Client ID** та необов’язкового **Deployment ID**, коли ви їх отримаєте.
