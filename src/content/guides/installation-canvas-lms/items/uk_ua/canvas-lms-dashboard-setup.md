#### Перейдіть до налаштувань LTI

Перейдіть до <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">ваших налаштувань LTI FastComments</a>.

Сторінка налаштування використовує покроковий майстер, який проведе вас через кожен крок. Ви почнете з **Крок 1: Створити**.

#### Створення нової конфігурації LTI

Заповніть наступні поля:

- **Configuration Name** - необов’язкова мітка для ідентифікації цієї конфігурації (корисно, якщо ви підключаєте кілька екземплярів Canvas).
- **Platform URL** - URL вашого екземпляра Canvas (наприклад `https://yourschool.instructure.com`). Це поле є обов’язковим.

#### Вибір розміщення

Під полем Platform URL ви можете обрати, де FastComments з’являтиметься у ваших курсах Canvas. Обидва варіанти розміщення увімкнені за замовчуванням:

- **Assignment View** - коментарі з’являються автоматично під завданнями.
- **Rich Content Editor Button** - викладачі можуть вбудовувати FastComments у Pages, Quizzes та Announcements за допомогою кнопки на панелі інструментів Rich Content Editor.

Ви можете ввімкнути один або обидва варіанти. Повинно бути увімкнено щонайменше одне розміщення.

Натисніть **Create Configuration**.

#### Скопіюйте URL конфігурації

Після створення конфігурації майстер перейде до **Крок 2: Підключити**. У верхній частині сторінки буде помітно відображено **Configuration URL** з кнопкою **Copy**.

Скопіюйте цей URL. Він знадобиться вам на наступному кроці для створення Developer Key у Canvas.

Сторінка також показує пронумеровані інструкції щодо дій у Canvas, а також поля для введення **Client ID** та необов’язкового **Deployment ID**, коли вони у вас будуть.
