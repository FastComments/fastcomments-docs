#### 在 FastComments 中輸入 Client ID

回到 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">您的 FastComments LTI 設定</a>。設定精靈應該停在 **Step 2: Connect**。

將從 Canvas 複製的 **Client ID** 貼到 **Client ID** 欄位。如您的 LMS 提供 **Deployment ID**，可選擇輸入。

點擊 **Save & Continue**。

#### 啟用整合

設定精靈會進入 **Step 3: Go Live**。畫面會顯示您設定的摘要（名稱、平台 URL、Client ID 和 Deployment ID）。

檢查各項細節，然後點擊 **Enable Integration** 以啟動 LTI 連線。

啟用後，設定精靈會顯示 **Management View**，您可以在此編輯設定、查看所有 LTI URL，或新增其他部署。

#### 在 Canvas 中安裝外部應用程式

在 Canvas 中，前往 **Admin** > 選擇您的帳戶 > **Settings** > **Apps** 分頁。

點擊 **+ App** 並設定：

1. 將 **Configuration Type** 設為 **By Client ID**。
2. 從 Developer Keys 表格貼上 **Client ID**。
3. 點擊 **Submit**。
4. 在提示時確認安裝。

FastComments 現已安裝於帳戶層級，並可供所有課程使用。
