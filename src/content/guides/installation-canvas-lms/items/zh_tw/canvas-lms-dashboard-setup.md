#### 前往 Canvas LTI 設定

登入您的 FastComments 帳戶並前往 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">我的帳戶 > Canvas LTI 設定</a>。

設定頁面使用逐步精靈引導您完成每個步驟。您將從 **步驟 1：建立** 開始。

#### 建立新的 LTI 設定

填寫以下欄位：

- **設定名稱 (Configuration Name)** - 一個可選的標籤用以識別此設定（若您連接多個 Canvas 實例時會很有用）。
- **平台 URL (Platform URL)** - 您的 Canvas 實例 URL（例如 `https://yourschool.instructure.com`）。此欄位為必填。

#### 選擇放置位置

在 **平台 URL (Platform URL)** 下方，您可以選擇 FastComments 在 Canvas 課程中的顯示位置。兩種放置位置預設皆已啟用：

- **作業檢視 (Assignment View)** - 評論會自動顯示在作業下方。
- **豐富內容編輯器按鈕 (Rich Content Editor Button)** - 教師可以使用豐富內容編輯器工具列的按鈕，在頁面、測驗與公告中嵌入 FastComments。

您可以啟用其中一個或兩個。至少必須啟用一個放置位置。

點選 **建立設定 (Create Configuration)**。

#### 複製設定 URL

建立設定後，精靈會進入 **步驟 2：連線**。頁面頂端會顯示顯眼的 **設定 URL (Configuration URL)**，並附有 **複製** 按鈕。

請複製此 URL。您在下一步建立 Canvas 的 Developer Key 時會需要它。

該頁面還會顯示在 Canvas 中應執行步驟的編號說明，以及在取得後輸入 **Client ID** 和可選 **Deployment ID** 的欄位。
