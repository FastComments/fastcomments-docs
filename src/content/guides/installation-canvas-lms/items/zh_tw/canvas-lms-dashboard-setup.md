#### 前往 LTI 設定

前往 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">您的 FastComments LTI 設定</a>。

設定頁面使用導引精靈逐步引導。您會從 **步驟 1：建立** 開始。

#### 建立新的 LTI 設定

填寫以下欄位：

- **Configuration Name** - 一個可選的標籤，用以識別這個設定（如果您連接多個 Canvas 實例會很有用）。
- **Platform URL** - 您的 Canvas 實例網址（例如 `https://yourschool.instructure.com`）。此欄位為必填。

#### 選擇放置位置

在 Platform URL 下方，您可以選擇 FastComments 在 Canvas 課程中的顯示位置。預設會啟用兩個放置位置：

- **Assignment View** - 評論會自動顯示在作業下方。
- **Rich Content Editor Button** - 授課者可以使用富內容編輯器工具列按鈕，將 FastComments 嵌入頁面、測驗及公告。

您可以啟用其中一個或兩個。至少需要啟用一個放置位置。

點選 **建立設定**。

#### 複製設定 URL

建立設定後，精靈會進入 **步驟 2：連線**。頁面頂端會顯示一個明顯的 **設定 URL**，並附有 **複製** 按鈕。

複製此 URL。您在下一步於 Canvas 建立開發者金鑰時會需要用到。

頁面還會顯示在 Canvas 中需執行步驟的編號說明，以及欄位讓您輸入 **Client ID** 和可選的 **Deployment ID**（在取得後）。
