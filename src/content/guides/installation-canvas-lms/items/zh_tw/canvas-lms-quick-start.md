1. 登入 FastComments 並前往 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">我的帳戶 > Canvas LTI 設定</a>。
2. 勾選 **啟用 LTI** 並按 **新增**。複製顯示的 **設定 URL**。
3. 在 Canvas 中，前往 **管理員 > 開發人員金鑰 > + 開發人員金鑰 > LTI 金鑰**。將 **方法** 設為 "輸入 URL"，並貼上設定 URL。儲存金鑰並將其 **狀態** 設為 **開啟**。
4. 從 Canvas 的開發人員金鑰表格中複製 **Client ID** 編號。將其貼到 FastComments 的 **Client ID** 欄位，然後按 **儲存**。
5. 在您的 Canvas 課程中，前往 **設定 > 導覽**，找到 **FastComments**，並啟用它。評論會以課程導覽項目的形式顯示。
