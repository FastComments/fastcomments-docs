#### Open Developer Keys in Canvas

登入 Canvas 並以管理員身分。前往 **Admin**（在左側邊欄）> 選取您的帳戶 > **Developer Keys**。

#### Create an LTI Developer Key

點擊 **+ Developer Key** 並選擇 **LTI Key**。

在設定表單：

1. 將 **Method** 設為 **Enter URL**。
2. 將您從 FastComments 複製的 **Configuration URL** 貼到 URL 欄位。
3. Canvas 會自動載入 LTI 設定。
4. 為金鑰命名（例如「FastComments」）。
5. 點擊 **Save**。

#### Enable the Developer Key

儲存後，新金鑰會出現在 Developer Keys 表格中，其 **State** 顯示為 **OFF**。點擊切換將其設為 **ON**。

#### Copy the Client ID

Developer Keys 表格在 Details 欄位會顯示一個數字形式的 **Client ID**（例如 `17000000000042`）。複製此數字 - 下一步會用到。
