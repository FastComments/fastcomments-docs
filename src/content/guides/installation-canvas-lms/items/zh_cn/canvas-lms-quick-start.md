1. 登录 FastComments 并转到 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>.
2. 选中 **Enable LTI** 并点击 **Add**。复制所示的 **Configuration URL**。
3. 在 Canvas 中，转到 **Admin > Developer Keys > + Developer Key > LTI Key**。将 **Method** 设置为 "Enter URL" 并粘贴 Configuration URL。保存密钥并将其 **State** 设置为 **ON**。
4. 从 Canvas 的 Developer Keys 表中复制 **Client ID** 编号。将其粘贴到 FastComments 中的 **Client ID** 字段并点击 **Save**。
5. 在您的 Canvas 课程中，转到 **Settings > Navigation**，找到 **FastComments**，并启用它。评论将作为课程导航项出现。
