#### 在 Canvas 中打开开发者密钥

以管理员身份登录 Canvas。转到左侧边栏的 **管理员** > 选择您的帐户 > **开发者密钥**。

#### 创建 LTI 开发者密钥

点击 **+ 开发者密钥** 并选择 **LTI 密钥**。

在配置表单中：

1. 将 **方法** 设置为 **输入 URL**。
2. 将您从 FastComments 复制的 **配置 URL** 粘贴到 URL 字段中。
3. Canvas 将自动加载 LTI 配置。
4. 为该密钥命名（例如 “FastComments”）。
5. 点击 **保存**。

#### 启用开发者密钥

保存后，新密钥会出现在 **开发者密钥** 表中，其 **状态** 显示为 **OFF**。点击切换按钮将其设置为 **ON**。Canvas 可能会提示您确认 — 点击 **Allow** 以启用该密钥。

#### 复制 Client ID

**开发者密钥** 表在“详情”列显示数字 **Client ID**（例如 `17000000000042`）。复制此数字 - 在下一步中您将需要它。
