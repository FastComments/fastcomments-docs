#### 在 Canvas 中打开 开发者密钥

以管理员身份登录 Canvas。转到左侧边栏的 **管理**（**Admin**）> 选择你的账户 > **开发者密钥**（**Developer Keys**）。

#### 创建 LTI 开发者密钥

点击 **+ 开发者密钥** 并选择 **LTI 密钥**。

在配置表单中：

1. 在 **重定向 URI**（左侧）字段中，粘贴来自 FastComments 设置页面的 **启动 URL**。
2. 在右侧，将 **方法** 设置为 **输入 URL**。
3. 将你从 FastComments 复制的 **配置 URL** 粘贴到 **JSON URL** 字段中。
4. Canvas 会自动加载 LTI 配置。
5. 为密钥命名（例如 “FastComments”）。
6. 点击 **保存**。

#### 启用 开发者密钥

保存后，新密钥会出现在开发者密钥表中，其 **状态** 显示为 **关闭**（OFF）。点击切换按钮将其设置为 **开启**（ON）。Canvas 可能会提示你确认 — 点击 **允许**（Allow）以启用该密钥。

#### 复制 客户端 ID

开发者密钥表在详情列中显示一个数字 **客户端 ID**（例如 `17000000000042`）。复制此编号——下一步你将需要它。
