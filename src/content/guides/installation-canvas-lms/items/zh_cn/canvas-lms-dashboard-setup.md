#### 导航到 Canvas LTI Config

登录到您的 FastComments 帐户并转到 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>。

设置页面使用引导向导，逐步引导您完成每一步。您将从 **Step 1: Create** 开始。

#### 创建新的 LTI Configuration

填写以下字段：

- **Configuration Name** - 一个可选标签，用于标识此配置（如果连接多个 Canvas 实例时很有用）。
- **Platform URL** - 您的 Canvas 实例 URL（例如 `https://yourschool.instructure.com`）。此字段为必填项。

点击 **Create Configuration**。

#### 复制 Configuration URL

创建配置后，向导会进入 **Step 2: Connect**。页面顶部会明显显示一个 **Configuration URL**，并带有一个 **Copy** 按钮。

复制此 URL。您将在下一步创建 Canvas 开发者密钥时需要它。

该页面还显示了在 Canvas 中应执行操作的编号说明，以及在获得后输入 **Client ID** 和可选 **Deployment ID** 的字段。
