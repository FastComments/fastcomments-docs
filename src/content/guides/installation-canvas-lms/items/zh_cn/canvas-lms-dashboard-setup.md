#### 导航到 Canvas LTI 配置

登录到您的 FastComments 帐户并转到 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">我的账户 > Canvas LTI 配置</a>。

设置页面使用向导引导您完成每一步。您将从 **第 1 步：创建** 开始。

#### 创建新的 LTI 配置

填写以下字段：

- **Configuration Name** - 一个可选标签，用于识别此配置（如果连接多个 Canvas 实例则很有用）。
- **Platform URL** - 您的 Canvas 实例的 URL（例如 `https://yourschool.instructure.com`）。此字段为必填项。

#### 选择放置位置

在平台 URL 下方，您可以选择 FastComments 在 Canvas 课程中出现的位置。两个放置位置默认均为启用：

- **Assignment View** - 评论会自动出现在作业下方。
- **Rich Content Editor Button** - 教师可以使用富文本编辑器工具栏按钮在页面、测验和公告中嵌入 FastComments。

您可以启用其中一个或两个。必须至少启用一个放置位置。

点击 **创建配置**。

#### 复制配置 URL

创建配置后，向导会进入 **第 2 步：连接**。页面顶部会显著显示 **配置 URL**，并带有一个 **复制** 按钮。

复制此 URL。在下一步创建 Canvas 开发者密钥时需要它。

页面还会显示在 Canvas 中应执行操作的编号说明，以及在获取后用于输入 **Client ID** 和可选 **Deployment ID** 的字段。
