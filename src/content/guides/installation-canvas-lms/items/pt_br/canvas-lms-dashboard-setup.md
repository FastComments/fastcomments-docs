#### Navegue até a Configuração LTI do Canvas

Faça login na sua conta FastComments e vá para <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Minha Conta > Configuração LTI do Canvas</a>.

A página de configuração utiliza um assistente guiado que o orienta em cada etapa. Você começará em **Passo 1: Criar**.

#### Criar uma Nova Configuração LTI

Preencha os seguintes campos:

- **Configuration Name** - um rótulo opcional para identificar esta configuração (útil se você conectar várias instâncias do Canvas).
- **Platform URL** - a URL da sua instância do Canvas (por exemplo `https://yourschool.instructure.com`). Este campo é obrigatório.

#### Escolha os locais

Abaixo do campo Platform URL, você pode escolher onde o FastComments aparece em seus cursos do Canvas. Ambos os locais são habilitados por padrão:

- **Assignment View** - os comentários aparecem automaticamente abaixo das tarefas.
- **Rich Content Editor Button** - os instrutores podem incorporar o FastComments em Páginas, Questionários e Anúncios usando o botão da barra de ferramentas no Editor de Conteúdo Rico.

Você pode habilitar um ou ambos. Pelo menos um local deve estar habilitado.

Clique em **Criar Configuração**.

#### Copie a URL de Configuração

Após criar a configuração, o assistente avança para **Passo 2: Conectar**. Uma **URL de Configuração** destacada é exibida no topo da página com um botão **Copiar**.

Copie essa URL. Você precisará dela na próxima etapa para criar uma Developer Key no Canvas.

A página também mostra instruções numeradas sobre o que fazer no Canvas, além de campos para inserir o **Client ID** e o opcional **Deployment ID** assim que os tiver.
