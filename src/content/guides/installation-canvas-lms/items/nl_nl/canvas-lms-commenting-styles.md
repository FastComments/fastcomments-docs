FastComments ondersteunt drie commentaarstijlen. U kunt dit configureren op de <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Canvas LTI Config</a>-pagina.

#### Comments

The standard FastComments widget appears on the page. Users can post threaded comments, reply, vote, and use all standard features.

    <div class="title">Comments Widget op een cursuspagina</div>
    <img class="screenshot-image" src="/images/installation-guides/installation-guide-moodle-course-comments.png" alt="FastComments Comments op Canvas-cursus" />
</div>

#### Collab Chat

With Collab Chat, users select text on the page to start inline discussions. A top bar shows online users and the number of active discussions. This is ideal for close reading, peer review, and study groups.

<div class="screenshot white-bg">
    <div class="title">Collab Chat met tekstmarkering</div>
    <img class="screenshot-image" src="/images/installation-guides/installation-guide-moodle-collab-chat.png" alt="FastComments Collab Chat op Canvas" />
</div>

#### Collab Chat + Comments

Both modes active together. The comment widget appears below the page content, and users can also select text to start inline discussions.
