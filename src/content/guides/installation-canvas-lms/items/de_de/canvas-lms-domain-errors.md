Das liegt daran, dass FastComments nicht weiß, dass Ihr Konto auf dieser Domain verwendet werden darf.

In diesem Fall ist die Lösung einfach. Sie müssen lediglich die Domain Ihrer Canvas-Instanz zu Ihrem FastComments-Konto hinzufügen.

<a href="https://fastcomments.com/auth/my-account/configure-domains" target="_blank">Hier können Sie Ihre Website zu Ihrem Konto hinzufügen.</a>

Fügen Sie die Domain Ihrer Canvas-Instanz hinzu (z. B. `yourschool.instructure.com`) und die Kommentare sollten korrekt geladen werden.
