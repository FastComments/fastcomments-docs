#### Navigieren Sie zur Canvas LTI-Konfiguration

Melden Sie sich bei Ihrem FastComments-Konto an und gehen Sie zu <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Mein Konto > Canvas LTI-Konfiguration</a>.

Die Einrichtungsseite verwendet einen geführten Assistenten, der Sie durch jeden Schritt führt. Sie beginnen bei **Schritt 1: Erstellen**.

#### Neue LTI-Konfiguration erstellen

Füllen Sie die folgenden Felder aus:

- **Konfigurationsname** - ein optionales Label, um diese Konfiguration zu identifizieren (nützlich, wenn Sie mehrere Canvas-Instanzen verbinden).
- **Plattform-URL** - die URL Ihrer Canvas-Instanz (z. B. `https://yourschool.instructure.com`). Dieses Feld ist erforderlich.

#### Platzierungen auswählen

Unterhalb der Plattform-URL können Sie auswählen, wo FastComments in Ihren Canvas-Kursen angezeigt wird. Beide Platzierungen sind standardmäßig aktiviert:

- **Aufgabenansicht** - Kommentare erscheinen automatisch unter Aufgaben.
- **Schaltfläche des Rich Content Editors** - Lehrende können FastComments in Seiten, Tests und Ankündigungen über die Schaltfläche in der Symbolleiste des Rich Content Editors einbetten.

Sie können eine oder beide aktivieren. Mindestens eine Platzierung muss aktiviert sein.

Klicken Sie auf **Konfiguration erstellen**.

#### Kopieren Sie die Konfigurations-URL

Nachdem Sie die Konfiguration erstellt haben, springt der Assistent zu **Schritt 2: Verbinden**. Oben auf der Seite wird eine prominente **Konfigurations-URL** mit einer **Kopieren**-Schaltfläche angezeigt.

Kopieren Sie diese URL. Sie benötigen sie im nächsten Schritt, um einen Entwicklerschlüssel in Canvas zu erstellen.

Die Seite zeigt außerdem nummerierte Anweisungen, was in Canvas zu tun ist, sowie Felder zum Eingeben der **Client ID** und der optionalen **Deployment ID**, sobald Sie diese haben.
