#### Navigieren Sie zur LTI-Konfiguration

Gehen Sie zu <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Ihre FastComments LTI-Konfiguration</a>.

Die Einrichtungsseite verwendet einen geführten Assistenten, der Sie durch jeden Schritt führt. Sie beginnen bei **Schritt 1: Erstellen**.

#### Erstellen einer neuen LTI-Konfiguration

Füllen Sie die folgenden Felder aus:

- **Configuration Name** - ein optionales Label zur Identifizierung dieser Konfiguration (nützlich, wenn Sie mehrere Canvas-Instanzen verbinden).
- **Platform URL** - die URL Ihrer Canvas-Instanz (z. B. `https://yourschool.instructure.com`). Dieses Feld ist erforderlich.

#### Platzierungen auswählen

Unter der Platform URL können Sie auswählen, wo FastComments in Ihren Canvas-Kursen angezeigt wird. Beide Platzierungen sind standardmäßig aktiviert:

- **Assignment View** - Kommentare erscheinen automatisch unter Aufgaben.
- **Rich Content Editor Button** - Lehrkräfte können FastComments in Seiten, Quizzen und Ankündigungen über die Schaltfläche in der Symbolleiste des Rich Content Editor einbetten.

Sie können eine oder beide aktivieren. Mindestens eine Platzierung muss aktiviert sein.

Klicken Sie auf **Konfiguration erstellen**.

#### Kopieren der Konfigurations-URL

Nachdem Sie die Konfiguration erstellt haben, wechselt der Assistent zu **Schritt 2: Verbinden**. Oben auf der Seite wird eine gut sichtbare **Konfigurations-URL** mit einer **Kopieren**-Schaltfläche angezeigt.

Kopieren Sie diese URL. Sie benötigen sie im nächsten Schritt, um einen Developer Key in Canvas zu erstellen.

Die Seite zeigt außerdem nummerierte Anweisungen, was in Canvas zu tun ist, sowie Felder zum Eingeben der **Client ID** und optionalen **Deployment ID**, sobald Sie diese haben.
