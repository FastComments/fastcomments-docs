#### Geben Sie die Client ID in FastComments ein

Gehen Sie in FastComments zurück zu <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>.

Suchen Sie Ihre Konfiguration und fügen Sie die **Client ID**, die Sie aus Canvas kopiert haben, in das Feld **Client ID** ein.

Falls noch nicht geschehen, geben Sie Ihre **Platform URL** (die URL Ihrer Canvas-Instanz, z. B. `https://yourschool.instructure.com`) ein.

Klicken Sie auf **Save**.

#### Installieren Sie die externe App in Canvas

Gehen Sie in Canvas zu **Admin** > wählen Sie Ihr Konto aus > **Settings** > Reiter **Apps**.

Klicken Sie auf **+ App** und konfigurieren Sie:

1. Stellen Sie **Configuration Type** auf **By Client ID**.
2. Fügen Sie die **Client ID** aus der Tabelle 'Developer Keys' ein.
3. Klicken Sie auf **Submit**.
4. Bestätigen Sie die Installation, wenn Sie dazu aufgefordert werden.

FastComments ist jetzt auf Kontoebene installiert und allen Kursen verfügbar.
