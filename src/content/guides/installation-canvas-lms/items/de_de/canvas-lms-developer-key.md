#### Entwicklerschlüssel in Canvas öffnen

Melden Sie sich bei Canvas als Administrator an. Gehen Sie zu **Admin** (in der linken Seitenleiste) > wählen Sie Ihr Konto aus > **Entwicklerschlüssel**.

#### Einen LTI-Entwicklerschlüssel erstellen

Klicken Sie auf **+ Entwicklerschlüssel** und wählen Sie **LTI-Schlüssel**.

Im Konfigurationsformular:

1. Setzen Sie **Methode** auf **URL eingeben**.
2. Fügen Sie die **Konfigurations-URL** ein, die Sie von FastComments kopiert haben, in das URL-Feld.
3. Canvas lädt die LTI-Konfiguration automatisch.
4. Geben Sie dem Schlüssel einen Namen (z. B. "FastComments").
5. Klicken Sie auf **Speichern**.

#### Den Entwicklerschlüssel aktivieren

Nach dem Speichern erscheint der neue Schlüssel in der Tabelle Entwicklerschlüssel mit dem **Status** auf **OFF** gesetzt. Klicken Sie auf den Schalter, um ihn auf **ON** zu setzen. Canvas kann Sie zur Bestätigung auffordern — klicken Sie auf **Allow**, um den Schlüssel zu aktivieren.

#### Die Client-ID kopieren

In der Tabelle Entwicklerschlüssel wird in der Spalte Details eine numerische **Client ID** angezeigt (z. B. `17000000000042`). Kopieren Sie diese Nummer - Sie benötigen sie im nächsten Schritt.
