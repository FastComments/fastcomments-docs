#### Entwickler-Keys in Canvas öffnen

Melden Sie sich in Canvas als Administrator an. Gehen Sie zu **Admin** (in der linken Seitenleiste) > wählen Sie Ihr Konto aus > **Developer Keys**.

#### Einen LTI Developer Key erstellen

Klicken Sie auf **+ Developer Key** und wählen Sie **LTI Key**.

Im Konfigurationsformular:

1. Setzen Sie **Method** auf **Enter URL**.
2. Fügen Sie die **Configuration URL**, die Sie aus FastComments kopiert haben, in das URL-Feld ein.
3. Canvas lädt die LTI-Konfiguration automatisch.
4. Geben Sie dem Key einen Namen (z. B. "FastComments").
5. Klicken Sie auf **Save**.

#### Den Developer Key aktivieren

Nach dem Speichern erscheint der neue Key in der Developer Keys-Tabelle mit dem **State** auf **OFF**. Klicken Sie auf den Schalter, um ihn auf **ON** zu setzen.

#### Die Client ID kopieren

Die Developer Keys-Tabelle zeigt in der Details-Spalte eine numerische **Client ID** (z. B. `17000000000042`). Kopieren Sie diese Zahl - Sie benötigen sie im nächsten Schritt.
