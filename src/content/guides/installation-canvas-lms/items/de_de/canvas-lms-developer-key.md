#### Entwickler-Schlüssel in Canvas öffnen

Melden Sie sich bei Canvas als Administrator an. Gehen Sie zu **Admin** (in der linken Seitenleiste) > wählen Sie Ihr Konto > **Entwickler-Schlüssel**.

#### Einen LTI-Entwickler-Schlüssel erstellen

Klicken Sie auf **+ Entwickler-Schlüssel** und wählen Sie **LTI-Schlüssel**.

Im Konfigurationsformular:

1. Fügen Sie im Feld **Redirect URIs** (linke Seite) die **Launch-URL** von der FastComments-Einrichtungsseite ein.
2. Stellen Sie rechts **Methode** auf **URL eingeben**.
3. Fügen Sie die **Konfigurations-URL**, die Sie von FastComments kopiert haben, in das Feld **JSON URL** ein.
4. Canvas lädt die LTI-Konfiguration automatisch.
5. Geben Sie dem Schlüssel einen Namen (z. B. „FastComments“).
6. Klicken Sie auf **Speichern**.

#### Den Entwickler-Schlüssel aktivieren

Nach dem Speichern erscheint der neue Schlüssel in der Tabelle der Entwickler-Schlüssel mit seinem **Status** auf **OFF** gesetzt. Klicken Sie auf den Schalter, um ihn auf **ON** zu setzen. Canvas kann Sie zur Bestätigung auffordern — klicken Sie auf **Zulassen**, um den Schlüssel zu aktivieren.

#### Client-ID kopieren

Die Tabelle der Entwickler-Schlüssel zeigt eine numerische **Client-ID** in der Spalte Details (z. B. `17000000000042`). Kopieren Sie diese Nummer - Sie benötigen sie im nächsten Schritt.
