#### LTI 設定に移動

次のページにアクセスしてください: <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">あなたの FastComments LTI 設定</a>。

セットアップページは、各ステップを案内するウィザードを使用します。**Step 1: Create** から開始します。

#### 新しい LTI 設定を作成

次の項目を入力してください:

- **Configuration Name** - この設定を識別するための任意のラベル（複数の Canvas インスタンスを接続する場合に便利です）。
- **Platform URL** - あなたの Canvas インスタンスの URL（例: `https://yourschool.instructure.com`）。この項目は必須です。

#### 配置を選択

Platform URL の下で、FastComments を Canvas コースのどこに表示するかを選択できます。両方の配置はデフォルトで有効になっています:

- **Assignment View** - コメントが課題の下に自動的に表示されます。
- **Rich Content Editor Button** - 教員はリッチコンテンツエディタのツールバーのボタンを使って、Pages、Quizzes、Announcements に FastComments を埋め込むことができます。

いずれか一方、または両方を有効にできます。少なくとも一つの配置を有効にする必要があります。

**Create Configuration** をクリックします。

#### 構成 URL をコピー

構成を作成すると、ウィザードは **Step 2: Connect** に進みます。ページ上部に目立つ **Configuration URL** が表示され、**Copy** ボタンがあります。

この URL をコピーしてください。次の手順で Canvas に Developer Key を作成する際に必要になります。

ページには、Canvas での操作手順（番号つき）と、取得後に入力する **Client ID** および任意の **Deployment ID** のフィールドも表示されます。
