#### Canvas LTI 設定に移動

FastComments アカウントにログインし、<a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">マイアカウント > Canvas LTI 設定</a> に移動します。

セットアップページはガイド付きウィザードを使って各ステップを案内します。**ステップ 1: 作成** から始まります。

#### 新しい LTI 設定を作成

以下の項目を入力します:

- **Configuration Name** - この構成を識別するための任意のラベル（複数の Canvas インスタンスを接続する場合に便利）。
- **Platform URL** - あなたの Canvas インスタンスの URL（例: `https://yourschool.instructure.com`）。この項目は必須です。

#### 配置を選択

Platform URL の下で、FastComments を Canvas コースのどこに表示するかを選択できます。どちらの配置もデフォルトで有効になっています:

- **Assignment View** - コメントは課題の下に自動で表示されます。
- **Rich Content Editor Button** - 講師はリッチコンテンツエディタのツールバーボタンを使って、ページ、クイズ、アナウンスメントに FastComments を埋め込むことができます。

片方または両方を有効にできます。少なくとも一つの配置を有効にする必要があります。

**Create Configuration** をクリックします。

#### Configuration URL をコピー

設定を作成すると、ウィザードは **ステップ 2: 接続** に進みます。ページ上部には目立つ **Configuration URL** が表示され、**コピー** ボタンがあります。

この URL をコピーしてください。次の手順で Canvas に Developer Key を作成する際に必要になります。

このページには、Canvas 側で行う手順の番号付きの説明と、取得後に **Client ID** および任意の **Deployment ID** を入力するためのフィールドも表示されます。
