1. FastCommentsにログインし、<a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">マイアカウント > Canvas LTI 設定</a>に移動します。
2. **LTI を有効にする**にチェックを入れ、**追加**をクリックします。表示された**構成 URL**をコピーします。
3. Canvasで、**管理 > 開発者キー > + 開発者キー > LTI キー**に移動します。**方法**を "URL を入力" に設定し、構成 URL を貼り付けます。キーを保存し、**状態**を**ON**に設定します。
4. Canvasの開発者キー表から**Client ID**の番号をコピーします。FastCommentsの**Client ID**フィールドに貼り付け、**保存**をクリックします。
5. Canvasコースで**設定 > ナビゲーション**に移動し、**FastComments**を見つけて有効にします。コメントはコースのナビゲーション項目として表示されます。
