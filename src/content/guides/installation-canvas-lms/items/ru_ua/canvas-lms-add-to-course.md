#### Включение FastComments в навигации курса

В вашем курсе Canvas перейдите в **Настройки** > вкладку **Навигация**.

Найдите **FastComments** в списке скрытых элементов внизу. Перетащите его в список видимых элементов (или нажмите на меню с тремя точками и выберите **Включить**).

Нажмите **Сохранить**. FastComments теперь появится в боковой панели навигации курса.

#### Использование FastComments в курсе

Когда студенты нажмут **FastComments** в навигации курса, LTI-запуск произойдёт автоматически:

- Студенты входят через свою учётную запись Canvas (имя, адрес электронной почты и аватар синхронизируются).
- Для каждого курса создаётся своя ветка комментариев.
- Комментарии отображаются в реальном времени и обновляются автоматически.

#### Добавление в отдельные модули (необязательно)

Вы также можете добавить FastComments в отдельные модули как элемент типа **Внешний инструмент**:

1. В модуле нажмите **+**, чтобы добавить элемент.
2. Выберите **Внешний инструмент**.
3. Выберите **FastComments** из списка.
4. Нажмите **Добавить элемент**.

Это создаёт для каждого модуля отдельную ветку обсуждения.
