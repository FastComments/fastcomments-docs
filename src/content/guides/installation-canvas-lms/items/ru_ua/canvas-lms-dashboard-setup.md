#### Перейдите к конфигурации Canvas LTI

Войдите в свою учетную запись FastComments и перейдите на <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Мой аккаунт > Конфигурация Canvas LTI</a>.

Страница настройки использует пошаговый мастер, который проводит вас через каждый этап. Вы начнете с **Шаг 1: Создать**.

#### Создание новой конфигурации LTI

Заполните следующие поля:

- **Название конфигурации** - необязательная метка для идентификации этой конфигурации (полезно, если вы подключаете несколько инстансов Canvas).
- **Platform URL** - URL вашей инстанции Canvas (например, `https://yourschool.instructure.com`). Это поле обязательно для заполнения.

Нажмите **Создать конфигурацию**.

#### Скопируйте URL конфигурации

После создания конфигурации мастер переходит к **Шаг 2: Подключить**. В верхней части страницы отображается заметный **Configuration URL** с кнопкой **Копировать**.

Скопируйте этот URL. Он понадобится вам на следующем шаге для создания ключа разработчика в Canvas.

Страница также показывает пронумерованные инструкции по действиям в Canvas, а также поля для ввода **ID клиента** и (необязательно) **ID развертывания**, когда они у вас появятся.
