#### Перейдите в конфигурацию Canvas LTI

Войдите в свою учетную запись FastComments и перейдите в <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Мой аккаунт > Конфигурация Canvas LTI</a>.

Страница настройки использует пошаговый мастер, который проводит вас через каждый шаг. Вы начнёте с **Шаг 1: Создать**.

#### Создайте новую LTI-конфигурацию

Заполните следующие поля:

- **Имя конфигурации** - необязательная метка для идентификации этой конфигурации (полезно, если вы подключаете несколько инстансов Canvas).
- **URL платформы** - URL вашей инстанции Canvas (например, `https://yourschool.instructure.com`). Это поле обязательно.

#### Выберите размещения

Под полем URL платформы вы можете выбрать, где FastComments будет отображаться в ваших курсах Canvas. Оба варианта размещения включены по умолчанию:

- **Просмотр задания** - комментарии автоматически отображаются под заданиями.
- **Кнопка Rich Content Editor** - преподаватели могут встраивать FastComments на Страницы, Викторины и Объявления, используя кнопку на панели инструментов Rich Content Editor.

Вы можете включить один или оба варианта. Должен быть включён хотя бы один вариант размещения.

Нажмите **Создать конфигурацию**.

#### Скопируйте URL конфигурации

После создания конфигурации мастер перейдёт к **Шаг 2: Подключение**. В верхней части страницы будет заметно отображён **URL конфигурации** с кнопкой **Копировать**.

Скопируйте этот URL. Он потребуется вам на следующем шаге для создания Developer Key в Canvas.

На странице также показаны пронумерованные инструкции для действий в Canvas, а также поля для ввода **Client ID** и необязательного **Deployment ID**, когда они у вас появятся.
