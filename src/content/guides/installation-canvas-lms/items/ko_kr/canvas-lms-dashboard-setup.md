#### Canvas LTI 구성으로 이동

FastComments 계정에 로그인한 후 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">내 계정 > Canvas LTI 구성</a>로 이동하세요.

설정 페이지는 단계별 안내 마법사를 사용하여 각 단계를 안내합니다. **1단계: 생성**에서 시작합니다.

#### 새 LTI 구성 생성

다음 필드를 작성하세요:

- **Configuration Name** - 이 구성을 식별하기 위한 선택적 레이블(여러 Canvas 인스턴스를 연결하는 경우 유용).
- **Platform URL** - 귀하의 Canvas 인스턴스 URL(예: `https://yourschool.instructure.com`). 이 필드는 필수입니다.

#### 배치 선택

Platform URL 아래에서 FastComments가 Canvas 과정 내 어디에 표시될지 선택할 수 있습니다. 기본적으로 두 가지 배치가 모두 활성화되어 있습니다:

- **과제 보기** - 댓글이 과제 아래에 자동으로 표시됩니다.
- **리치 콘텐츠 편집기 버튼** - 강사는 리치 콘텐츠 편집기 툴바의 버튼을 사용하여 페이지, 퀴즈 및 공지사항에 FastComments를 삽입할 수 있습니다.

하나 또는 둘 다 활성화할 수 있습니다. 최소한 하나의 배치는 활성화되어야 합니다.

**구성 생성**을 클릭하세요.

#### 구성 URL 복사

구성을 생성하면 마법사가 **2단계: 연결**로 이동합니다. 페이지 상단에 눈에 띄는 **구성 URL**이 **복사** 버튼과 함께 표시됩니다.

이 URL을 복사하세요. 다음 단계에서 Canvas에서 Developer Key를 생성할 때 필요합니다.

이 페이지에는 Canvas에서 해야 할 작업에 대한 번호 매겨진 지침과, Client ID 및 선택적 Deployment ID를 얻은 후 입력할 수 있는 필드도 표시됩니다.
