#### Canvas LTI 구성으로 이동

FastComments 계정에 로그인한 후 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">내 계정 > Canvas LTI 구성</a>로 이동하세요.

설정 페이지에는 각 단계를 안내하는 가이드 마법사가 있습니다. **1단계: 만들기**에서 시작합니다.

#### 새 LTI 구성 생성

다음 필드를 입력하세요:

- **구성 이름** - 이 구성을 식별하기 위한 선택적 레이블입니다 (여러 Canvas 인스턴스를 연결하는 경우 유용합니다).
- **플랫폼 URL** - 귀하의 Canvas 인스턴스 URL(예: `https://yourschool.instructure.com`). 이 필드는 필수입니다.

**구성 생성**을 클릭하세요.

#### 구성 URL 복사

구성을 생성하면 마법사가 **2단계: 연결**로 진행됩니다. 페이지 상단에는 **구성 URL**이 눈에 띄게 표시되며 **복사** 버튼이 있습니다.

이 URL을 복사하세요. 다음 단계에서 Canvas에 Developer Key를 생성할 때 필요합니다.

페이지에는 Canvas에서 수행할 작업에 대한 번호 매겨진 지침과, 획득한 후 입력할 **Client ID** 및 선택적 **Deployment ID** 필드도 표시됩니다.
