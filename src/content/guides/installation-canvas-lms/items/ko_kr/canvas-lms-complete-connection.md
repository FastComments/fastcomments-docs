#### FastComments에 Client ID 입력

FastComments에서 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>로 돌아가십시오. 마법사는 **Step 2: Connect** 단계에 있어야 합니다.

Canvas에서 복사한 **Client ID**를 **Client ID** 필드에 붙여넣으세요. LMS에서 제공하는 경우 **Deployment ID**를 선택적으로 입력하세요.

**Save & Continue**를 클릭하세요.

#### 통합 활성화

마법사가 **Step 3: Go Live** 단계로 진행됩니다. 설정 요약(이름, 플랫폼 URL, Client ID, 및 deployment ID)이 표시됩니다.

세부 정보를 검토한 후 **Enable Integration**을 클릭하여 LTI 연결을 활성화하십시오.

활성화 후, 마법사는 구성 편집, 모든 LTI URL 보기 또는 추가 배포 추가가 가능한 **Management View**를 표시합니다.

#### Canvas에 외부 앱 설치

Canvas에서 **Admin** > 계정 선택 > **Settings** > **Apps** 탭으로 이동합니다.

**+ App**을 클릭하고 다음을 구성합니다:

1. **Configuration Type**을 **By Client ID**로 설정합니다.
2. Developer Keys 표에서 **Client ID**를 붙여넣습니다.
3. **Submit**을 클릭합니다.
4. 프롬프트가 표시되면 설치를 확인합니다.

이제 FastComments가 계정 수준에 설치되어 모든 코스에서 사용할 수 있게 됩니다.
