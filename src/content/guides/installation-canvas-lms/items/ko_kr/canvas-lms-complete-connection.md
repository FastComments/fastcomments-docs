#### FastComments에 클라이언트 ID 입력

FastComments에서 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">내 계정 > Canvas LTI 구성</a>로 돌아가세요.

구성 항목을 찾아 Canvas에서 복사한 **Client ID**를 **Client ID** 필드에 붙여넣으세요.

아직 입력하지 않았다면, **Platform URL**(귀하의 Canvas 인스턴스 URL, 예: `https://yourschool.instructure.com`)을 입력하세요.

Click **Save**.

#### Canvas에 외부 앱 설치

Canvas에서 **Admin** > 계정을 선택 > **Settings** > **Apps** 탭으로 이동하세요.

Click **+ App** and configure:

1. **Configuration Type**을 **By Client ID**로 설정하세요.
2. Developer Keys 테이블에서 복사한 **Client ID**를 붙여넣으세요.
3. Click **Submit**.
4. 확인 메시지가 표시되면 설치를 확인하세요.

이제 FastComments가 계정 수준에 설치되어 모든 강좌에서 사용 가능합니다.
