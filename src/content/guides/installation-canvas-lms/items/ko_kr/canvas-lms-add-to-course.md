#### 강좌 탐색에서 FastComments 활성화

Canvas 강좌에서 **설정** > **탐색** 탭으로 이동합니다.

페이지 하단의 숨김 항목 목록에서 **FastComments**를 찾습니다. 가시 항목 목록으로 끌어오거나(또는 점 3개 메뉴를 클릭하고 **활성화**를 선택)합니다.

**저장**을 클릭합니다. 이제 FastComments가 강좌 내비게이션 사이드바에 표시됩니다.

#### 강좌에서 FastComments 사용하기

학생이 강좌 내비게이션에서 **FastComments**를 클릭하면 LTI가 자동으로 실행됩니다:

- 학생들은 Canvas 계정으로 자동 로그인됩니다(이름, 이메일, 아바타가 동기화됩니다).
- 각 강좌마다 고유한 댓글 스레드가 생성됩니다.
- 댓글은 실시간으로 업데이트됩니다.

#### 개별 모듈에 추가하기(선택 사항)

FastComments를 개별 모듈에 외부 도구 항목으로 추가할 수도 있습니다:

1. 모듈에서 항목을 추가하려면 **+**를 클릭합니다.
2. **외부 도구**를 선택합니다.
3. 목록에서 **FastComments**를 선택합니다.
4. **항목 추가**를 클릭합니다.

이렇게 하면 각 모듈마다 별도의 토론 스레드가 생성됩니다.
