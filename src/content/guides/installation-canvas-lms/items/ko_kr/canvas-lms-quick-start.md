1. FastComments에 로그인한 다음 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">내 계정 > Canvas LTI 구성</a>으로 이동합니다.
2. **구성 이름**과 **플랫폼 URL**(예: `https://yourschool.instructure.com`)을 입력한 다음 **구성 생성**을 클릭합니다. 마법사가 2단계로 진행되며 **구성 URL**을 표시합니다.
3. Canvas에서 **관리자 > 개발자 키 > + 개발자 키 > LTI 키**로 이동합니다. **Method**를 "URL 입력"으로 설정하고 구성 URL을 붙여넣습니다. 키를 저장하고 **State**를 **ON**으로 설정합니다.
4. Canvas의 Developer Keys 표에서 **Client ID** 번호를 복사합니다. FastComments로 돌아가 **Client ID** 필드에 붙여넣고 **저장 및 계속**을 클릭합니다.
5. 구성 요약을 검토한 후 **통합 활성화**를 클릭하여 적용합니다.
6. Canvas 강좌에서 **설정 > 탐색**으로 이동하여 **FastComments**를 찾아 활성화합니다. 댓글은 강좌 탐색 항목으로 표시됩니다.
