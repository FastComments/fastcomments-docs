1. FastComments에 로그인한 다음 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>로 이동합니다.
2. **Enable LTI**를 선택하고 **Add**를 클릭합니다. 표시된 **Configuration URL**을 복사합니다.
3. Canvas에서 **Admin > Developer Keys > + Developer Key > LTI Key**로 이동합니다. **Method**를 "Enter URL"로 설정하고 Configuration URL을 붙여넣습니다. 키를 저장한 다음 **State**를 **ON**으로 설정합니다.
4. Canvas의 Developer Keys 표에서 **Client ID** 번호를 복사합니다. FastComments의 **Client ID** 필드에 붙여넣고 **Save**를 클릭합니다.
5. Canvas 강좌에서 **Settings > Navigation**으로 가서 **FastComments**를 찾아 활성화합니다. 댓글은 강좌 내 탐색 항목으로 표시됩니다.
