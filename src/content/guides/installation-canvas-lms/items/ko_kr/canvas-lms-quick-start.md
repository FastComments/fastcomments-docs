1. FastComments에 로그인한 후 <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">내 계정 > Canvas LTI 구성</a>으로 이동하세요.
2. **Configuration Name**과 **Platform URL**을 입력하세요 (예: `https://yourschool.instructure.com`). 활성화할 **Placements**(Assignment View 및/또는 Editor Button — 둘 다 기본적으로 켜져 있음)를 선택하세요. **Create Configuration**을 클릭하세요. 마법사는 단계 2로 진행되며 **Configuration URL**을 표시합니다.
3. Canvas에서 **Admin > Developer Keys > + Developer Key > LTI Key**로 이동하세요. **Method**를 "Enter URL"로 설정하고 Configuration URL을 붙여넣으세요. 키를 저장한 다음 **State**를 **ON**으로 설정하고 안내 메시지가 표시되면 "Allow"를 클릭하세요.
4. Canvas의 Developer Keys 표에서 **Client ID** 번호를 복사하세요. FastComments로 돌아가서 **Client ID** 필드에 붙여넣고 **Save & Continue**를 클릭하세요.
5. 구성 요약을 검토한 후 **Enable Integration**을 클릭하여 라이브로 전환하세요.
6. Canvas에서 외부 앱을 설치하세요 (**Admin > Settings > Apps > + App > By Client ID**). 댓글은 과제 아래에 자동으로 표시되며, 강사는 Rich Content Editor 툴바 버튼을 통해 Pages, Quizzes 및 Announcements에 FastComments를 삽입할 수 있습니다.
