#### Navigate to Canvas LTI Config

Впишете се във вашия FastComments акаунт и отидете на <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Моят акаунт > Canvas LTI Config</a>.

Страницата за настройка използва ръководен съветник, който ви води през всяка стъпка. Ще започнете от **Стъпка 1: Създаване**.

#### Create a New LTI Configuration

Попълнете следните полета:

- **Configuration Name** - незадължителен етикет за идентифициране на тази конфигурация (полезно, ако свържете няколко инстанции на Canvas).
- **Platform URL** - URL на вашата инстанция на Canvas (например `https://yourschool.instructure.com`). Това поле е задължително.

Щракнете върху **Създаване на конфигурация**.

#### Copy the Configuration URL

След като създадете конфигурацията, съветникът преминава към **Стъпка 2: Свързване**. В горната част на страницата се показва ясно видим **URL на конфигурацията** с бутон **Копирай**.

Копирайте този URL. Ще ви трябва в следващата стъпка, за да създадете Developer Key в Canvas.

Страницата също показва номерирани инструкции за това какво да направите в Canvas, както и полета за въвеждане на **Client ID** и по избор **Deployment ID**, след като ги получите.
