FastComments υποστηρίζει τρεις μορφές σχολιασμού. Μπορείτε να το διαμορφώσετε στη σελίδα <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Διαμόρφωση LTI του Canvas</a>.

#### Σχόλια

Το προεπιλεγμένο widget του FastComments εμφανίζεται στη σελίδα. Οι χρήστες μπορούν να δημοσιεύουν νηματώδη σχόλια, να απαντούν, να ψηφίζουν και να χρησιμοποιούν όλες τις τυπικές λειτουργίες.

#### Collab Chat

Με το Collab Chat, οι χρήστες επιλέγουν κείμενο στη σελίδα για να ξεκινήσουν ενσωματωμένες (inline) συζητήσεις. Μια κορυφαία μπάρα δείχνει τους χρήστες που είναι online και τον αριθμό των ενεργών συζητήσεων. Αυτό είναι ιδανικό για προσεκτική ανάγνωση, αξιολόγηση ομότιμων και ομάδες μελέτης.

#### Collab Chat + Σχόλια

Και οι δύο λειτουργίες ενεργές ταυτόχρονα. Το widget σχολίων εμφανίζεται κάτω από το περιεχόμενο της σελίδας, και οι χρήστες μπορούν επίσης να επιλέξουν κείμενο για να ξεκινήσουν ενσωματωμένες συζητήσεις.
