#### Πώς Εμφανίζονται τα Σχόλια στα Μαθήματά σας

Μόλις η ολοκλήρωση LTI ενεργοποιηθεί και η Εξωτερική Εφαρμογή εγκατασταθεί, το FastComments λειτουργεί αυτόματα βάσει των τοποθετήσεων που διαμορφώσατε:

#### Προβολή Εργασίας

Εάν η τοποθέτηση **Προβολή Εργασίας** είναι ενεργοποιημένη, τα σχόλια εμφανίζονται αυτόματα κάτω από κάθε εργασία στο μάθημα. Οι φοιτητές και οι διδάσκοντες βλέπουν μια διακλαδωμένη ενότητα σχολίων όταν προβάλλουν μια εργασία — δεν απαιτείται επιπλέον ρύθμιση ανά εργασία.

Κάθε εργασία έχει το δικό της ξεχωριστό νήμα σχολίων.

#### Κουμπί Επεξεργαστή Πλούσιου Περιεχομένου

Εάν η τοποθέτηση **Κουμπί Επεξεργαστή** είναι ενεργοποιημένη, οι διδάσκοντες μπορούν να ενσωματώσουν το FastComments σε οποιοδήποτε περιεχόμενο χρησιμοποιεί τον Επεξεργαστή Πλούσιου Περιεχομένου:

1. Επεξεργαστείτε μια **Σελίδα**, ένα **Διαγώνισμα**, ή μια **Ανακοίνωση**.
2. Στη γραμμή εργαλείων του Επεξεργαστή Πλούσιου Περιεχομένου, κάντε κλικ στο κουμπί **FastComments**.
3. Το FastComments ενσωματώνεται αυτόματα στο περιεχόμενο.
4. Αποθηκεύστε τη σελίδα.

Όταν οι φοιτητές προβάλλουν τη σελίδα, το ενσωματωμένο widget του FastComments φορτώνει με ένα νήμα σχολίων μοναδικό για αυτή τη σελίδα.

#### Αυτόματο SSO

Σε αμφότερες τις τοποθετήσεις, οι φοιτητές συνδέονται αυτόματα μέσω του λογαριασμού τους στο Canvas. Τα ονόματα, τα emails και οι εικόνες προφίλ συγχρονίζονται μέσω της εκκίνησης LTI — δεν χρειάζεται ξεχωριστή σύνδεση.

#### Κλείδωμα Δημόσιας Πρόσβασης (Συνιστάται)

Εξ ορισμού, τα δεδομένα σχολίων του FastComments μπορούν να διαβαστούν δημόσια. Οποιοσδήποτε μπορεί να μαντέψει το URL ενός νήματος ή το API endpoint και να δει τα σχόλιά του, ακόμη και εκτός Canvas. Για συζητήσεις μαθήματος, σχεδόν σίγουρα θα θέλετε να περιορίσετε την προβολή μόνο σε εγγεγραμμένους φοιτητές.

Ανοίξτε τη σελίδα <a href="https://fastcomments.com/auth/my-account/customize-widget" target="_blank">προσαρμογής του widget</a> σας και δημιουργήστε έναν κανόνα με ενεργοποιημένο το **Απαιτείται SSO για Προβολή Σχολίων**, στη συνέχεια ορίστε το επίπεδο ασφάλειας σε **Ασφαλές SSO** ώστε τα νήματα να μπορούν να φορτώνονται μόνο μέσω της υπογεγραμμένης εκκίνησης LTI.

Δείτε [Προστασία Νημάτων Σχολίων με Ενιαία Σύνδεση](/guide-customizations-and-configuration.html#sso-require-to-view-comments) για τον πλήρη οδηγό, συμπεριλαμβανομένου του πώς να περιορίσετε τον κανόνα σε ένα μόνο domain ή σελίδα.
