#### Εισαγάγετε το Client ID στο FastComments

Επιστρέψτε στο <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a> στο FastComments.

Βρείτε τη ρύθμισή σας και επικολλήστε το **Client ID** που αντιγράψατε από το Canvas στο πεδίο **Client ID**.

Εάν δεν το έχετε ήδη κάνει, εισάγετε το **Platform URL** σας (το URL του Canvas σας, π.χ. `https://yourschool.instructure.com`).

Κάντε κλικ στο **Save**.

#### Install the External App in Canvas

Στο Canvas, πηγαίνετε στο **Admin** > επιλέξτε τον λογαριασμό σας > **Settings** > καρτέλα **Apps**.

Κάντε κλικ στο **+ App** και ρυθμίστε:

1. Ορίστε το **Configuration Type** σε **By Client ID**.
2. Επικολλήστε το **Client ID** από τον πίνακα Developer Keys.
3. Κάντε κλικ στο **Submit**.
4. Επιβεβαιώστε την εγκατάσταση όταν σας ζητηθεί.

Το FastComments είναι πλέον εγκατεστημένο σε επίπεδο λογαριασμού και διαθέσιμο σε όλα τα μαθήματα.
