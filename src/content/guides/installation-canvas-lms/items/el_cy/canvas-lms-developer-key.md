#### Open Developer Keys in Canvas

Συνδεθείτε στο Canvas ως διαχειριστής. Μεταβείτε στην ενότητα **Διαχειριστής** (στην αριστερή πλαϊνή γραμμή) > επιλέξτε τον λογαριασμό σας > **Κλειδιά προγραμματιστή**.

#### Create an LTI Developer Key

Κάντε κλικ στο **+ Κλειδί προγραμματιστή** και επιλέξτε **Κλειδί LTI**.

Στη φόρμα ρυθμίσεων:

1. Ορίστε το πεδίο **Μέθοδος** σε **Εισαγωγή URL**.
2. Επικολλήστε το **Configuration URL** που αντιγράψατε από το FastComments στο πεδίο URL.
3. Το Canvas θα φορτώσει αυτόματα τη διαμόρφωση LTI.
4. Δώστε στο κλειδί ένα όνομα (π.χ. "FastComments").
5. Κάντε κλικ στο **Αποθήκευση**.

#### Enable the Developer Key

Μετά την αποθήκευση, το νέο κλειδί θα εμφανιστεί στον πίνακα Κλειδιών προγραμματιστή με την **Κατάσταση** ορισμένη σε **OFF**. Κάντε κλικ στο διακόπτη για να την αλλάξετε σε **ON**. Το Canvas μπορεί να σας ζητήσει επιβεβαίωση — κάντε κλικ στο **Allow** για να ενεργοποιήσετε το κλειδί.

#### Copy the Client ID

Ο πίνακας Κλειδιών προγραμματιστή δείχνει έναν αριθμητικό **Client ID** στη στήλη Λεπτομέρειες (π.χ. `17000000000042`). Αντιγράψτε αυτόν τον αριθμό — θα τον χρειαστείτε στο επόμενο βήμα.
