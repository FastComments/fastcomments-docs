#### Άνοιγμα Κλειδιών Προγραμματιστή στο Canvas

Συνδεθείτε στο Canvas ως διαχειριστής. Μεταβείτε στο **Admin** (στην αριστερή πλαϊνή στήλη) > επιλέξτε τον λογαριασμό σας > **Developer Keys**.

#### Δημιουργία Κλειδιού Προγραμματιστή LTI

Κάντε κλικ στο **+ Developer Key** και επιλέξτε **LTI Key**.

Στη φόρμα ρύθμισης:

1. Στο πεδίο **Redirect URIs** (αριστερά), επικολλήστε το **Launch URL** από τη σελίδα ρύθμισης του FastComments.
2. Στα δεξιά, ορίστε το **Method** σε **Enter URL**.
3. Επικολλήστε το **Configuration URL** που αντιγράψατε από το FastComments στο πεδίο **JSON URL**.
4. Το Canvas θα φορτώσει αυτόματα τη ρύθμιση LTI.
5. Δώστε στο κλειδί ένα όνομα (π.χ. "FastComments").
6. Κάντε κλικ στο **Save**.

#### Ενεργοποίηση του Κλειδιού Προγραμματιστή

Μετά την αποθήκευση, το νέο κλειδί θα εμφανιστεί στον πίνακα Developer Keys με την **State** ορισμένη σε **OFF**. Κάντε κλικ στον διακόπτη για να το ορίσετε σε **ON**. Το Canvas μπορεί να σας ζητήσει επιβεβαίωση — κάντε κλικ στο **Allow** για να ενεργοποιηθεί το κλειδί.

#### Αντιγραφή του Client ID

Ο πίνακας Developer Keys εμφανίζει έναν αριθμητικό **Client ID** στη στήλη Details (π.χ. `17000000000042`). Αντιγράψτε αυτόν τον αριθμό — θα τον χρειαστείτε στο επόμενο βήμα.
