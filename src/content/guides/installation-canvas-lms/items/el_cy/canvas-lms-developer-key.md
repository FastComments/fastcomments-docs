#### Άνοιγμα των Developer Keys στο Canvas

Συνδεθείτε στο Canvas ως διαχειριστής. Μεταβείτε στο **Admin** (στην αριστερή πλευρική στήλη) > επιλέξτε το λογαριασμό σας > **Developer Keys**.

#### Δημιουργία ενός LTI Developer Key

Κάντε κλικ στο **+ Developer Key** και επιλέξτε **LTI Key**.

Στη φόρμα διαμόρφωσης:

1. Ορίστε το **Method** σε **Enter URL**.
2. Επικολλήστε το **Configuration URL** που αντιγράψατε από το FastComments στο πεδίο URL.
3. Το Canvas θα φορτώσει αυτόματα τη διαμόρφωση LTI.
4. Δώστε ένα όνομα στο κλειδί (π.χ. "FastComments").
5. Κάντε κλικ στο **Save**.

#### Ενεργοποίηση του Developer Key

Μετά την αποθήκευση, το νέο κλειδί θα εμφανιστεί στον πίνακα Developer Keys με την **State** ρυθμισμένη σε **OFF**. Κάντε κλικ στον διακόπτη για να τη θέσετε σε **ON**.

#### Αντιγραφή του Client ID

Ο πίνακας Developer Keys εμφανίζει έναν αριθμητικό **Client ID** στη στήλη Details (π.χ. `17000000000042`). Αντιγράψτε αυτόν τον αριθμό - θα τον χρειαστείτε στο επόμενο βήμα.
