Cela se produit parce que FastComments ne sait pas qu'il doit autoriser l'utilisation de votre compte sur ce domaine.

Dans ce cas, la solution est simple. Vous devez simplement ajouter le domaine de votre instance Canvas à votre compte FastComments.

<a href="https://fastcomments.com/auth/my-account/configure-domains" target="_blank">Allez ici pour ajouter votre site à votre compte.</a>

Ajoutez le domaine de votre Canvas (par ex. `yourschool.instructure.com`) et les commentaires devraient se charger correctement.
