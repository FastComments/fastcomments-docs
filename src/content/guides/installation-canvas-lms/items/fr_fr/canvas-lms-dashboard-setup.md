#### Accédez à la configuration LTI de Canvas

Connectez-vous à votre compte FastComments et allez sur <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Mon compte > Configuration LTI Canvas</a>.

La page de configuration utilise un assistant guidé qui vous accompagne à chaque étape. Vous commencerez à **Étape 1 : Créer**.

#### Créer une nouvelle configuration LTI

Remplissez les champs suivants:

- **Configuration Name** - une étiquette optionnelle pour identifier cette configuration (utile si vous connectez plusieurs instances de Canvas).
- **Platform URL** - l'URL de votre instance Canvas (par ex. `https://yourschool.instructure.com`). Ce champ est requis.

Cliquez sur **Créer la configuration**.

#### Copier l'URL de configuration

Après avoir créé la configuration, l'assistant passe à **Étape 2 : Connecter**. Une **URL de configuration** bien visible s'affiche en haut de la page avec un bouton **Copier**.

Copiez cette URL. Vous en aurez besoin à l'étape suivante pour créer une Developer Key dans Canvas.

La page affiche également des instructions numérotées indiquant quoi faire dans Canvas, ainsi que des champs pour saisir le **Client ID** et, facultativement, le **Deployment ID** une fois que vous les aurez.
