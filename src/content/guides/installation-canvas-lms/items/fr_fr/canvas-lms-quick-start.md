1. Accédez à <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">votre configuration LTI FastComments</a>.
2. Saisissez un **Configuration Name** et votre **Platform URL** (par ex. `https://yourschool.instructure.com`). Choisissez quels **Emplacements** activer (Assignment View and/or Editor Button — les deux sont activés par défaut). Cliquez sur **Create Configuration**. L'assistant passe à l'Étape 2 et affiche votre **Configuration URL**.
3. Dans Canvas, allez à **Admin > Developer Keys > + Developer Key > LTI Key**. Définissez **Method** sur "Enter URL" et collez la Configuration URL. Enregistrez la clé, puis réglez son **State** sur **ON** et cliquez sur **Allow** lorsque vous y êtes invité.
4. Copiez le numéro **Client ID** depuis la table Developer Keys dans Canvas. De retour sur FastComments, collez-le dans le champ **Client ID** et cliquez sur **Save & Continue**.
5. Vérifiez le résumé de la configuration et cliquez sur **Enable Integration** pour mettre en ligne.
6. Installez l'application externe dans Canvas (**Admin > Settings > Apps > + App > By Client ID**). Les commentaires apparaîtront automatiquement sous les devoirs, et les enseignants peuvent intégrer FastComments dans Pages, Quizzes et Announcements via le bouton de la barre d'outils Rich Content Editor.
