1. Connectez-vous à FastComments et allez à <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>.
2. Cochez **Enable LTI** et cliquez sur **Add**. Copiez la **Configuration URL** affichée.
3. Dans Canvas, allez à **Admin > Developer Keys > + Developer Key > LTI Key**. Définissez **Method** sur "Enter URL" et collez la Configuration URL. Enregistrez la clé et réglez son **State** sur **ON**.
4. Copiez le numéro **Client ID** depuis le tableau Developer Keys dans Canvas. Collez-le dans le champ **Client ID** de FastComments et cliquez sur **Save**.
5. Dans votre cours Canvas, allez à **Settings > Navigation**, trouvez **FastComments** et activez-le. Les commentaires apparaîtront comme un élément de navigation du cours.
