#### Ouvrir les clés développeur dans Canvas

Connectez-vous à Canvas en tant qu'administrateur. Allez dans **Admin** (dans la barre latérale gauche) > sélectionnez votre compte > **Developer Keys**.

#### Créer une clé développeur LTI

Cliquez sur **+ Developer Key** et sélectionnez **LTI Key**.

Dans le formulaire de configuration :

1. Définissez **Method** sur **Enter URL**.
2. Collez l'**Configuration URL** que vous avez copiée depuis FastComments dans le champ URL.
3. Canvas chargera automatiquement la configuration LTI.
4. Donnez un nom à la clé (par exemple "FastComments").
5. Cliquez sur **Save**.

#### Activer la clé développeur

Après l'enregistrement, la nouvelle clé apparaîtra dans le tableau Developer Keys avec son **State** réglé sur **OFF**. Cliquez sur le toggle pour le passer sur **ON**.

#### Copier le Client ID

Le tableau Developer Keys affiche un **Client ID** numérique dans la colonne Details (par exemple `17000000000042`). Copiez ce numéro - vous en aurez besoin à l'étape suivante.
