#### Ouvrir Developer Keys dans Canvas

Connectez-vous à Canvas en tant qu'administrateur. Allez dans **Admin** (dans la barre latérale gauche) > sélectionnez votre compte > **Developer Keys**.

#### Créer un LTI Developer Key

Cliquez sur **+ Developer Key** et sélectionnez **LTI Key**.

Dans le formulaire de configuration :

1. Réglez **Method** sur **Enter URL**.
2. Collez le **Configuration URL** que vous avez copié depuis FastComments dans le champ URL.
3. Canvas chargera automatiquement la configuration LTI.
4. Donnez un nom à la clé (par ex. « FastComments »).
5. Cliquez sur **Save**.

#### Activer la Developer Key

Après l'enregistrement, la nouvelle clé apparaîtra dans le tableau Developer Keys avec son **State** réglé sur **OFF**. Cliquez sur le commutateur pour le passer à **ON**. Canvas peut vous demander de confirmer — cliquez sur **Allow** pour activer la clé.

#### Copier le Client ID

Le tableau Developer Keys affiche un **Client ID** numérique dans la colonne Details (par ex. `17000000000042`). Copiez ce numéro — vous en aurez besoin à l'étape suivante.
