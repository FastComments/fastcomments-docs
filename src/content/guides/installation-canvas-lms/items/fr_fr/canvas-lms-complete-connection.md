#### Entrez le Client ID dans FastComments

Retournez à <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Mon compte > Configuration LTI Canvas</a> dans FastComments.

Trouvez votre configuration et collez le **Client ID** que vous avez copié depuis Canvas dans le champ **Client ID**.

Si ce n'est pas déjà fait, saisissez votre **Platform URL** (l'URL de votre instance Canvas, par ex. `https://yourschool.instructure.com`).

Cliquez sur **Enregistrer**.

#### Installer l'application externe dans Canvas

Dans Canvas, allez dans **Administration** > sélectionnez votre compte > **Paramètres** > onglet **Applications**.

Cliquez sur **+ Application** et configurez :

1. Définissez **Type de configuration** sur **Par ID client**.
2. Collez le **Client ID** depuis le tableau Clés développeur.
3. Cliquez sur **Soumettre**.
4. Confirmez l'installation lorsqu'on vous le demande.

FastComments est maintenant installé au niveau du compte et disponible pour tous les cours.
