#### Otwórz Developer Keys w Canvas

Zaloguj się do Canvas jako administrator. Przejdź do **Admin** (w lewym pasku bocznym) > wybierz swoje konto > **Developer Keys**.

#### Utwórz LTI Developer Key

Kliknij **+ Developer Key** i wybierz **LTI Key**.

W formularzu konfiguracji:

1. Ustaw **Method** na **Enter URL**.
2. Wklej **Configuration URL**, który skopiowałeś z FastComments, w pole URL.
3. Canvas automatycznie załaduje konfigurację LTI.
4. Nadaj kluczowi nazwę (np. "FastComments").
5. Kliknij **Save**.

#### Włącz Developer Key

Po zapisaniu nowy klucz pojawi się w tabeli Developer Keys z **State** ustawionym na **OFF**. Kliknij przełącznik, aby ustawić go na **ON**.

#### Skopiuj Client ID

W tabeli Developer Keys w kolumnie Details wyświetlany jest numeryczny **Client ID** (np. `17000000000042`). Skopiuj tę liczbę — będzie potrzebna w następnym kroku.
