#### Open Developer Keys in Canvas

Zaloguj się do Canvas jako administrator. Przejdź do **Admin** (w lewym pasku bocznym) > wybierz swoje konto > **Developer Keys**.

#### Create an LTI Developer Key

Kliknij **+ Developer Key** i wybierz **LTI Key**.

W formularzu konfiguracji:

1. Ustaw **Method** na **Enter URL**.
2. Wklej **Configuration URL**, które skopiowałeś ze FastComments, do pola URL.
3. Canvas automatycznie załaduje konfigurację LTI.
4. Nadaj kluczowi nazwę (np. "FastComments").
5. Kliknij **Save**.

#### Enable the Developer Key

Po zapisaniu nowy klucz pojawi się w tabeli Developer Keys z **State** ustawionym na **OFF**. Kliknij przełącznik, aby ustawić go na **ON**. Canvas może poprosić o potwierdzenie — kliknij **Allow**, aby włączyć klucz.

#### Copy the Client ID

W tabeli Developer Keys w kolumnie Details widoczny jest numeryczny **Client ID** (np. `17000000000042`). Skopiuj tę liczbę — będziesz jej potrzebować w następnym kroku.
