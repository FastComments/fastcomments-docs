1. Zaloguj się do FastComments i przejdź do <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>.
2. Zaznacz **Enable LTI** i kliknij **Add**. Skopiuj wyświetlony **Configuration URL**.
3. W Canvas przejdź do **Admin > Developer Keys > + Developer Key > LTI Key**. Ustaw **Method** na "Enter URL" i wklej Configuration URL. Zapisz klucz i ustaw jego **State** na **ON**.
4. Skopiuj numer **Client ID** z tabeli Developer Keys w Canvas. Wklej go w pole **Client ID** w FastComments i kliknij **Save**.
5. W swoim kursie Canvas przejdź do **Settings > Navigation**, znajdź **FastComments** i włącz go. Komentarze pojawią się jako element nawigacji kursu.
