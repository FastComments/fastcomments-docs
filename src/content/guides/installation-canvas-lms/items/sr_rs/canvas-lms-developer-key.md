#### Отворите Developer Keys у Canvas

Пријавите се у Canvas као администратор. Идите на **Admin** (у левој бочној траци) > изаберите свој налог > **Developer Keys**.

#### Креирајте LTI Developer Key

Кликните на **+ Developer Key** и изаберите **LTI Key**.

У формулару за конфигурацију:

1. Подесите **Method** на **Enter URL**.
2. Налепите **Configuration URL** који сте копирали са FastComments у поље URL.
3. Canvas ће аутоматски учитати LTI конфигурацију.
4. Дајте кључу име (нпр. "FastComments").
5. Кликните **Save**.

#### Омогућите Developer Key

Након чувања, нови кључ ће се појавити у табели Developer Keys са својим **State** постављеним на **OFF**. Кликните прекидач да бисте га поставили на **ON**.

#### Копирајте Client ID

У табели Developer Keys у колони Details се приказује нумерички **Client ID** (нпр. `17000000000042`). Копирајте овај број - биће вам потребан у следећем кораку.
