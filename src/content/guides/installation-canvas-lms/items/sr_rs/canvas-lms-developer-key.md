#### Отворите Developer Keys у Canvas

Пријавите се у Canvas као администратор. Идите на **Admin** (у левом бочном менију) > изаберите свој налог > **Developer Keys**.

#### Креирајте LTI Developer Key

Кликните **+ Developer Key** и изаберите **LTI Key**.

У формулару за конфигурацију:

1. Подесите **Method** на **Enter URL**.
2. Налепите **Configuration URL** који сте копирали са FastComments у поље за URL.
3. Canvas ће аутоматски учитати LTI конфигурацију.
4. Доделите кључу име (нпр. "FastComments").
5. Кликните **Save**.

#### Омогућите Developer Key

Након чувања, нови кључ ће се појавити у табели Developer Keys са **State** подешеним на **OFF**. Кликните на прекидач да бисте га подесили на **ON**. Canvas можда затражи потврду — кликните **Allow** да омогућите кључ.

#### Копирајте Client ID

У табели Developer Keys у колони Details приказан је нумерички **Client ID** (нпр. `17000000000042`). Копирајте овај број - биће вам потребан у следећем кораку.
