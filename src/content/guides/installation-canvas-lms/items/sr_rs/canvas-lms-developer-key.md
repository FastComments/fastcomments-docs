#### Отворите Developer Keys у Canvas

Пријавите се у Canvas као администратор. Идите на **Admin** (у левој бочној траци) > изаберите свој налог > **Developer Keys**.

#### Креирајте LTI Developer Key

Кликните **+ Developer Key** и изаберите **LTI Key**.

У формулару за конфигурацију:

1. У поље **Redirect URIs** (лево), налепите **Launch URL** са странице за подешавање FastComments.
2. На десној страни, поставите **Method** на **Enter URL**.
3. Налепите **Configuration URL** који сте копирали из FastComments у поље **JSON URL**.
4. Canvas ће аутоматски учитати LTI конфигурацију.
5. Дајте кључу име (нпр. "FastComments").
6. Кликните **Save**.

#### Омогућите Developer Key

Након чувања, нови кључ ће се појавити у табели Developer Keys са **State** подешеним на **OFF**. Кликните прекидач да га поставите на **ON**. Canvas може тражити потврду — кликните **Allow** да бисте омогућили кључ.

#### Копирајте Client ID

У табли Developer Keys у колони Details се приказује нумерички **Client ID** (нпр. `17000000000042`). Копирајте тај број — биће вам потребан у следећем кораку.
