#### Μεταβείτε στις ρυθμίσεις Canvas LTI

Συνδεθείτε στον λογαριασμό σας στο FastComments και μεταβείτε στο <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Λογαριασμός μου > Διαμόρφωση Canvas LTI</a>.

Η σελίδα ρύθμισης χρησιμοποιεί έναν καθοδηγούμενο οδηγό που σας καθοδηγεί σε κάθε βήμα. Θα ξεκινήσετε από το **Βήμα 1: Δημιουργία**.

#### Δημιουργία νέας διαμόρφωσης LTI

Συμπληρώστε τα ακόλουθα πεδία:

- **Όνομα Διαμόρφωσης** - μια προαιρετική ετικέτα για να αναγνωρίζετε αυτή τη διαμόρφωση (χρήσιμο αν συνδέσετε πολλαπλές περιπτώσεις Canvas).
- **URL Πλατφόρμας** - το URL της περίπτωσης Canvas σας (π.χ. `https://yourschool.instructure.com`). Αυτό το πεδίο είναι υποχρεωτικό.

#### Επιλέξτε Θέσεις εμφάνισης

Κάτω από το πεδίο URL Πλατφόρμας, μπορείτε να επιλέξετε πού εμφανίζεται το FastComments στα μαθήματά σας στο Canvas. Και οι δύο θέσεις είναι ενεργοποιημένες από προεπιλογή:

- **Προβολή Εργασιών** - τα σχόλια εμφανίζονται αυτόματα κάτω από τις εργασίες.
- **Κουμπί Πλούσιου Επεξεργαστή Περιεχομένου** - οι εκπαιδευτές μπορούν να ενσωματώσουν το FastComments σε Σελίδες, Τεστ και Ανακοινώσεις χρησιμοποιώντας το κουμπί της γραμμής εργαλείων στον Πλούσιο Επεξεργαστή Περιεχομένου.

Μπορείτε να ενεργοποιήσετε το ένα ή και τα δύο. Τουλάχιστον μία θέση πρέπει να είναι ενεργοποιημένη.

Κάντε κλικ στο **Δημιουργία Διαμόρφωσης**.

#### Αντιγράψτε το URL Διαμόρφωσης

Μετά τη δημιουργία της διαμόρφωσης, ο οδηγός προχωρά στο **Βήμα 2: Σύνδεση**. Ένα εμφανές **URL Διαμόρφωσης** εμφανίζεται στην κορυφή της σελίδας με ένα κουμπί **Αντιγραφή**.

Αντιγράψτε αυτό το URL. Θα το χρειαστείτε στο επόμενο βήμα για να δημιουργήσετε ένα Developer Key στο Canvas.

Η σελίδα εμφανίζει επίσης αριθμημένες οδηγίες για το τι πρέπει να κάνετε στο Canvas, καθώς και πεδία για να εισάγετε το **Client ID** και προαιρετικά το **Deployment ID** μόλις τα έχετε.
