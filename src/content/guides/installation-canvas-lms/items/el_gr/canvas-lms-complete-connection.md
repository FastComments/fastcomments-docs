#### Εισάγετε το Client ID στο FastComments

Επιστρέψτε στο <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Ο Λογαριασμός μου &gt; Canvas LTI Config</a> στο FastComments.

Βρείτε τη διαμόρφωσή σας και επικολλήστε το **Client ID** που αντιγράψατε από το Canvas στο πεδίο **Client ID**.

Αν δεν το έχετε ήδη κάνει, εισάγετε το **Platform URL** σας (το URL του Canvas instance σας, π.χ. `https://yourschool.instructure.com`).

Κάντε κλικ στο **Αποθήκευση**.

#### Εγκατάσταση της Εξωτερικής Εφαρμογής στο Canvas

Στο Canvas, πηγαίνετε σε **Διαχείριση** > επιλέξτε τον λογαριασμό σας > **Ρυθμίσεις** > καρτέλα **Εφαρμογές**.

Κάντε κλικ στο **+ App** και διαμορφώστε:

1. Ορίστε τον **Τύπο διαμόρφωσης** σε **By Client ID**.
2. Επικολλήστε το **Client ID** από τον πίνακα Developer Keys.
3. Κάντε κλικ στο **Υποβολή**.
4. Επιβεβαιώστε την εγκατάσταση όταν σας ζητηθεί.

Το FastComments είναι πλέον εγκατεστημένο σε επίπεδο λογαριασμού και διαθέσιμο σε όλα τα μαθήματα.
