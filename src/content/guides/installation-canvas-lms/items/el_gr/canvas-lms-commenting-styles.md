Το FastComments υποστηρίζει τρεις τρόπους σχολιασμού. Μπορείτε να το διαμορφώσετε στη σελίδα <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Διαμόρφωση Canvas LTI</a>.

#### Σχόλια

Το τυπικό widget του FastComments εμφανίζεται στη σελίδα. Οι χρήστες μπορούν να δημοσιεύουν νηματοποιημένα σχόλια, να απαντούν, να ψηφίζουν και να χρησιμοποιούν όλες τις τυπικές λειτουργίες.

#### Collab Chat

Με το Collab Chat, οι χρήστες επιλέγουν κείμενο στη σελίδα για να ξεκινήσουν ενσωματωμένες συζητήσεις. Μια επάνω γραμμή εμφανίζει τους συνδεδεμένους χρήστες και τον αριθμό των ενεργών συζητήσεων. Αυτό είναι ιδανικό για προσεκτική ανάγνωση, κριτική από ομότιμους και ομάδες μελέτης.

#### Collab Chat + Σχόλια

Και οι δύο λειτουργίες ενεργές ταυτόχρονα. Το widget σχολίων εμφανίζεται κάτω από το περιεχόμενο της σελίδας, και οι χρήστες μπορούν επίσης να επιλέξουν κείμενο για να ξεκινήσουν ενσωματωμένες συζητήσεις.
