1. היכנס ל-FastComments ועבור אל <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>.
2. סמן את **Enable LTI** ולחץ על **Add**. העתק את **Configuration URL** שמוצג.
3. ב-Canvas, עבור אל **Admin > Developer Keys > + Developer Key > LTI Key**. הגדר את **Method** ל-"Enter URL" והדבק את ה-Configuration URL. שמור את המפתח והגדר את **State** ל-**ON**.
4. העתק את מספר ה-**Client ID** מטבלת Developer Keys ב-Canvas. הדבק אותו בשדה **Client ID** ב-FastComments ולחץ על **Save**.
5. בקורס שלך ב-Canvas, עבור אל **Settings > Navigation**, מצא את **FastComments**, והפעל אותו. תגובות יופיעו כאייטם בתפריט הניווט של הקורס.
