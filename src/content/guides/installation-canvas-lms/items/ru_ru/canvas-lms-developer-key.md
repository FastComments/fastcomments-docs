#### Откройте Developer Keys в Canvas

Войдите в Canvas как администратор. Перейдите в **Admin** (в левом боковом меню) > выберите свою учетную запись > **Developer Keys**.

#### Создайте LTI Developer Key

Нажмите **+ Developer Key** и выберите **LTI Key**.

В форме конфигурации:

1. Установите **Method** в значение **Enter URL**.
2. Вставьте **Configuration URL**, скопированный вами из FastComments, в поле URL.
3. Canvas автоматически загрузит конфигурацию LTI.
4. Дайте ключу имя (например, «FastComments»).
5. Нажмите **Save**.

#### Включите Developer Key

После сохранения новый ключ появится в таблице **Developer Keys**, и в поле **State** будет установлено значение **OFF**. Нажмите переключатель, чтобы перевести его в **ON**.

#### Скопируйте Client ID

В таблице **Developer Keys** в колонке Details отображается числовой **Client ID** (например, `17000000000042`). Скопируйте это число — оно понадобится вам на следующем шаге.
