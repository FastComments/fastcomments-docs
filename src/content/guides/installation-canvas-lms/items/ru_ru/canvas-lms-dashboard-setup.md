#### Перейдите к настройке Canvas LTI

Войдите в свой аккаунт FastComments и перейдите на <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Мой аккаунт &gt; Настройка Canvas LTI</a>.

Страница настройки использует пошаговый мастер, который проведет вас через каждый этап. Вы начнёте с **Шаг 1: Создать**.

#### Создание новой конфигурации LTI

Заполните следующие поля:

- **Configuration Name** - необязательный ярлык для идентификации этой конфигурации (полезно, если вы подключаете несколько экземпляров Canvas).
- **Platform URL** - URL вашего экземпляра Canvas (например, `https://yourschool.instructure.com`). Это поле обязательно.

Нажмите **Создать конфигурацию**.

#### Скопируйте URL конфигурации

После создания конфигурации мастер перейдет к **Шаг 2: Подключение**. В верхней части страницы будет заметно отображаться **URL конфигурации** с кнопкой **Копировать**.

Скопируйте этот URL. Он понадобится вам на следующем шаге для создания ключа разработчика (Developer Key) в Canvas.

Страница также содержит пронумерованные инструкции по действиям в Canvas, а также поля для ввода **Client ID** и, при необходимости, **Deployment ID**, когда они у вас появятся.
