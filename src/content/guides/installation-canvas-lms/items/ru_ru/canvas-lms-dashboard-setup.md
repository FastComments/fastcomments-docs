#### Navigate to Canvas LTI Config

Войдите в свой аккаунт FastComments и перейдите на <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">My Account > Canvas LTI Config</a>.

Страница настройки использует пошаговый мастер, который проведет вас через каждый этап. Вы начнёте с **Step 1: Create**.

#### Create a New LTI Configuration

Заполните следующие поля:

- **Configuration Name** - необязательная метка для идентификации этой конфигурации (полезно, если вы подключаете несколько экземпляров Canvas).
- **Platform URL** - URL вашего экземпляра Canvas (например, `https://yourschool.instructure.com`). Это поле обязательно.

#### Choose Placements

Под полем **Platform URL** вы можете выбрать, где FastComments будет отображаться в ваших курсах Canvas. Оба варианта размещения включены по умолчанию:

- **Assignment View** - комментарии автоматически появляются под заданиями.
- **Rich Content Editor Button** - преподаватели могут встроить FastComments на Pages, Quizzes и Announcements с помощью кнопки на панели инструментов в Rich Content Editor.

Вы можете включить один или оба варианта. Должен быть включен как минимум один вариант размещения.

Нажмите **Create Configuration**.

#### Copy the Configuration URL

После создания конфигурации мастер перейдёт к **Step 2: Connect**. В верхней части страницы будет выделен **Configuration URL** с кнопкой **Copy**.

Скопируйте этот URL. Он понадобится вам на следующем шаге для создания Developer Key в Canvas.

Страница также показывает пронумерованные инструкции о том, что делать в Canvas, а также поля для ввода **Client ID** и (при необходимости) **Deployment ID**, как только они у вас появятся.
