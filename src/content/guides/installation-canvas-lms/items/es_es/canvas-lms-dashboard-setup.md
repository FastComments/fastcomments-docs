#### Navegar a la Configuración LTI

Ve a <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">tu Configuración LTI de FastComments</a>.

La página de configuración utiliza un asistente guiado que te guía a través de cada paso. Comenzarás en **Paso 1: Crear**.

#### Crear una Nueva Configuración LTI

Rellena los siguientes campos:

- **Configuration Name** - una etiqueta opcional para identificar esta configuración (útil si conectas varias instancias de Canvas).
- **Platform URL** - la URL de tu instancia de Canvas (p. ej. `https://yourschool.instructure.com`). Este campo es obligatorio.

#### Elegir Ubicaciones

Debajo de la URL de la plataforma, puedes elegir dónde aparece FastComments en tus cursos de Canvas. Ambas ubicaciones están habilitadas por defecto:

- **Assignment View** - los comentarios aparecen automáticamente debajo de las tareas.
- **Rich Content Editor Button** - los instructores pueden insertar FastComments en Páginas, Cuestionarios y Anuncios usando el botón de la barra de herramientas en el Editor de Contenido Enriquecido.

Puedes habilitar una o ambas. Debe habilitarse al menos una ubicación.

Haz clic en **Crear Configuración**.

#### Copiar la URL de Configuración

Después de crear la configuración, el asistente avanza a **Paso 2: Conectar**. Una **URL de Configuración** destacada se muestra en la parte superior de la página con un botón **Copiar**.

Copia esta URL. La necesitarás en el siguiente paso para crear una Developer Key en Canvas.

La página también muestra instrucciones numeradas sobre qué hacer en Canvas, además de campos para ingresar el **Client ID** y el **Deployment ID** opcional una vez que los tengas.
