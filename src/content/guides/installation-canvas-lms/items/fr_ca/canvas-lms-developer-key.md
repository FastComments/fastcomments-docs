#### Ouvrir les clés de développeur dans Canvas

Connectez-vous à Canvas en tant qu'administrateur. Allez dans **Admin** (dans la barre latérale gauche) > sélectionnez votre compte > **Clés de développeur**.

#### Créer une clé de développeur LTI

Cliquez sur **+ Clé de développeur** et sélectionnez **Clé LTI**.

Dans le formulaire de configuration :

1. Réglez **Méthode** sur **Entrer l’URL**.
2. Collez l'**URL de configuration** que vous avez copiée depuis FastComments dans le champ URL.
3. Canvas chargera automatiquement la configuration LTI.
4. Donnez un nom à la clé (p. ex. "FastComments").
5. Cliquez sur **Enregistrer**.

#### Activer la clé de développeur

Après l'enregistrement, la nouvelle clé apparaîtra dans le tableau des clés de développeur avec son **État** réglé sur **OFF**. Cliquez sur le commutateur pour le régler sur **ON**.

#### Copier l'ID client

Le tableau des clés de développeur affiche un **ID client** numérique dans la colonne Détails (p. ex. `17000000000042`). Copiez ce numéro — vous en aurez besoin à l'étape suivante.
