#### Ouvrir les clés développeur dans Canvas

Connectez-vous à Canvas en tant qu'administrateur. Allez dans **Administration** (dans la barre latérale gauche) > sélectionnez votre compte > **Clés développeur**.

#### Créer une clé développeur LTI

Cliquez sur **+ Clé développeur** et sélectionnez **Clé LTI**.

Dans le formulaire de configuration :

1. Définissez **Méthode** sur **Entrer l'URL**.
2. Collez l'**URL de configuration** que vous avez copiée depuis FastComments dans le champ URL.
3. Canvas chargera automatiquement la configuration LTI.
4. Donnez un nom à la clé (p. ex. "FastComments").
5. Cliquez sur **Enregistrer**.

#### Activer la clé développeur

Après l'enregistrement, la nouvelle clé apparaîtra dans le tableau des **Clés développeur** avec son **État** réglé sur **DÉSACTIVÉ**. Cliquez sur le commutateur pour le passer sur **ACTIVÉ**. Canvas peut vous demander de confirmer — cliquez sur **Autoriser** pour activer la clé.

#### Copier l'ID client

Le tableau des **Clés développeur** affiche un **ID client** numérique dans la colonne Détails (p. ex. `17000000000042`). Copiez ce nombre - vous en aurez besoin à l'étape suivante.
