#### Saisissez le Client ID dans FastComments

Retournez à <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">votre configuration LTI FastComments</a>. L'assistant devrait être à **Étape 2 : Connexion**.

Collez le **Client ID** que vous avez copié de Canvas dans le champ **Client ID**. Vous pouvez aussi saisir le **Deployment ID** si votre LMS en fournit un.

Cliquez sur **Enregistrer et continuer**.

#### Activer l'intégration

L'assistant passe à **Étape 3 : Mise en service**. Un résumé de votre configuration est affiché (nom, URL de la plateforme, Client ID et Deployment ID).

Vérifiez les détails, puis cliquez sur **Activer l’intégration** pour activer la connexion LTI.

Après l'activation, l'assistant affiche la **Vue de gestion** où vous pouvez modifier votre configuration, afficher toutes les URL LTI ou ajouter des déploiements supplémentaires.

#### Installer l'application externe dans Canvas

Dans Canvas, allez à **Admin** > sélectionnez votre compte > **Settings** > onglet **Apps**.

Cliquez sur **+ App** et configurez :

1. Réglez **Configuration Type** sur **By Client ID**.
2. Collez le **Client ID** depuis la table Developer Keys.
3. Cliquez sur **Submit**.
4. Confirmez l'installation lorsqu'on vous le demande.

FastComments est maintenant installé au niveau du compte et disponible pour tous les cours.
