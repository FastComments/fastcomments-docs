FastComments prend en charge trois styles de commentaires. Vous pouvez configurer cela dans la <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Configuration LTI de Canvas</a> page.

#### Commentaires

Le widget FastComments standard apparaît sur la page. Les utilisateurs peuvent publier des commentaires en fils, répondre, voter et utiliser toutes les fonctionnalités standard.

#### Collab Chat

Avec Collab Chat, les utilisateurs sélectionnent du texte sur la page pour démarrer des discussions en ligne. Une barre supérieure affiche les utilisateurs en ligne et le nombre de discussions actives. C'est idéal pour la lecture attentive, la revue par les pairs et les groupes d'étude.

#### Collab Chat + Commentaires

Les deux modes actifs simultanément. Le widget de commentaires apparaît sous le contenu de la page, et les utilisateurs peuvent également sélectionner du texte pour démarrer des discussions en ligne.
