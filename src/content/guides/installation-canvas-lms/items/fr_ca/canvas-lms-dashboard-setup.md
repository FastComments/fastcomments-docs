#### Accéder à la configuration LTI

Accédez à <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">votre configuration LTI FastComments</a>.

La page de configuration utilise un assistant guidé qui vous accompagne à chaque étape. Vous commencerez à **Étape 1 : Création**.

#### Créer une nouvelle configuration LTI

Remplissez les champs suivants :

- **Nom de la configuration** - un libellé optionnel pour identifier cette configuration (utile si vous connectez plusieurs instances Canvas).
- **URL de la plateforme** - l'URL de votre instance Canvas (p. ex. `https://yourschool.instructure.com`). Ce champ est requis.

#### Choisir les emplacements

Sous l'URL de la plateforme, vous pouvez choisir où FastComments apparaît dans vos cours Canvas. Les deux emplacements sont activés par défaut :

- **Assignment View** - les commentaires apparaissent automatiquement sous les devoirs.
- **Rich Content Editor Button** - les enseignants peuvent intégrer FastComments dans Pages, Quiz et Annonces en utilisant le bouton de la barre d'outils de l'éditeur de contenu enrichi.

Vous pouvez activer un ou les deux. Au moins un emplacement doit être activé.

Cliquez sur **Créer la configuration**.

#### Copier l'URL de configuration

Après avoir créé la configuration, l'assistant passe à **Étape 2 : Connexion**. Une **Configuration URL** bien en évidence est affichée en haut de la page avec un bouton **Copier**.

Copiez cette URL. Vous en aurez besoin à l'étape suivante pour créer une Developer Key dans Canvas.

La page affiche aussi des instructions numérotées pour ce qu'il faut faire dans Canvas, ainsi que des champs pour entrer le **Client ID** et le **Deployment ID** optionnel une fois que vous les aurez.
