#### Naviguer vers la configuration LTI Canvas

Connectez-vous à votre compte FastComments et allez à <a href="https://fastcomments.com/auth/my-account/canvas-lti-config" target="_blank">Mon compte > Configuration LTI Canvas</a>.

La page d'installation utilise un assistant guidé qui vous guide à travers chaque étape. Vous commencerez à **Étape 1 : Créer**.

#### Créer une nouvelle configuration LTI

Remplissez les champs suivants :

- **Configuration Name** - un libellé facultatif pour identifier cette configuration (utile si vous connectez plusieurs instances Canvas).
- **Platform URL** - l'URL de votre instance Canvas (p. ex. `https://yourschool.instructure.com`). Ce champ est requis.

#### Choisir les emplacements

Sous le **Platform URL**, vous pouvez choisir où FastComments apparaît dans vos cours Canvas. Les deux emplacements sont activés par défaut :

- **Assignment View** - les commentaires apparaissent automatiquement sous les devoirs.
- **Rich Content Editor Button** - les instructeurs peuvent intégrer FastComments dans les Pages, les Quiz et les Annonces en utilisant le bouton de la barre d'outils de l'éditeur de contenu enrichi.

Vous pouvez en activer un ou les deux. Au moins un emplacement doit être activé.

Cliquez sur **Créer la configuration**.

#### Copier l'URL de configuration

Après avoir créé la configuration, l'assistant passe à **Étape 2 : Connecter**. Une **Configuration URL** bien visible s'affiche en haut de la page avec un bouton **Copier**.

Copiez cette URL. Vous en aurez besoin à l'étape suivante pour créer une Developer Key dans Canvas.

La page affiche également des instructions numérotées expliquant quoi faire dans Canvas, ainsi que des champs pour entrer le **Client ID** et le **Deployment ID** optionnel une fois que vous les aurez.
