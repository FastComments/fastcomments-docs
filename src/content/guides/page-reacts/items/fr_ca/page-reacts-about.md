Le widget Page Reacts affiche un ensemble d'icônes de réaction personnalisable sur le widget de commentaires.

Les utilisateurs peuvent cliquer sur les images pour réagir, et il mémorise les réactions qu'ils ont sélectionnées pour chaque page. S'ils sont connectés à fastcomments.com, cela est lié à leur
compte.

Si vous cliquez sur `Run` dans l'exemple ci-dessous, vous pouvez le voir en action.
