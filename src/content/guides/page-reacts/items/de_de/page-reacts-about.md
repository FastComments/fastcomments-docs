Das Page Reacts-Widget zeigt eine anpassbare Auswahl von Reaktionssymbolen im Kommentar-Widget an.

Benutzer können auf die Bilder klicken, um zu reagieren, und es merkt sich die Reaktionen, die sie pro Seite ausgewählt haben. Wenn sie bei fastcomments.com angemeldet sind, ist dies mit ihrem
Konto verknüpft.

Wenn Sie im folgenden Beispiel auf `Run` klicken, können Sie es in Aktion sehen.
