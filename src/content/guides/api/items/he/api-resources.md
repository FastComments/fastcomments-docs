### שימוש במשאבים

יש לציין כי אחזור נתונים מה-API נספר כשימוש בחשבון שלך.

כל משאב יציין מה השימוש הזה בסעיף שלו.

חלק מהמשאבים יקרים יותר לשרת מאחרים. לכל נקודת קצה יש עלות קבועה של קרדיטים לכל קריאת API. עבור חלק מנקודות הקצה, מספר הקרדיטים
משתנה בהתאם לאפשרויות ולגדלי התגובות.

ניתן לבדוק את השימוש ב-API בדף [ניתוח חיוב](https://fastcomments.com/auth/my-account/analytics/billing) והוא מתעדכן כל מספר דקות.

#### הערה!

אנו מציעים לקרוא קודם את תיעוד הדפים, כדי לסייע בהגבלת הבלבול בעת קביעת אילו ערכים להעביר עבור `urlId` ב-API של Comment.
