### Χρήση Πόρων

Πρέπει να σημειωθεί ότι η ανάκτηση δεδομένων από το API μετράται ως χρήση στον λογαριασμό σας.

Κάθε πόρος θα αναφέρει ποια είναι αυτή η χρήση στη δική του ενότητα.

Ορισμένοι πόροι κοστίζουν περισσότερο στην εξυπηρέτηση από άλλους. Κάθε endpoint έχει ένα καθορισμένο κόστος credits ανά κλήση API. Για ορισμένα endpoints, ο αριθμός των credits
ποικίλλει ανάλογα με τις επιλογές και τα μεγέθη των απαντήσεων.

Η χρήση του API μπορεί να ελεγχθεί στη σελίδα [Billing Analytics](https://fastcomments.com/auth/my-account/analytics/billing) και ενημερώνεται κάθε λίγα λεπτά.

#### Σημείωση!

Προτείνουμε να διαβάσετε πρώτα την τεκμηρίωση των Pages, για να αποφύγετε τη σύγχυση όταν καθορίζετε ποιες τιμές να περάσετε για το `urlId` στο Comment API.
