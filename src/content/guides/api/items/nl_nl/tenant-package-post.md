[api-resource-header-start name = 'TenantPackage'; route = 'POST /api/v1/tenant-packages'; creditsCost = 1; api-resource-header-end]

Deze route biedt de mogelijkheid om een enkele `TenantPackage` toe te voegen.

Het aanmaken van een `TenantPackage` kent de volgende beperkingen:

- De volgende parameters zijn verplicht:
    - `name`
    - `tenantId`
    - `monthlyCostUSD` - Can be null.
    - `yearlyCostUSD` - Can be null.
    - `maxMonthlyPageLoads`
    - `maxMonthlyAPICredits`
    - `maxMonthlyComments`
    - `maxConcurrentUsers`
    - `maxTenantUsers`
    - `maxSSOUsers`
    - `maxModerators`
    - `maxDomains`
    - `hasDebranding`
    - `forWhoText`
    - `featureTaglines`
    - `hasFlexPricing` - If true, then all `flex*` parameters are required.
- De `name` mag niet langer zijn dan `50 characters`.
- Elk `forWhoText` item mag niet langer zijn dan `200 characters`.
- Elk `featureTaglines` item mag niet langer zijn dan `100 characters`.
- De `TenantPackage` moet "kleiner" zijn dan de parent tenant. Bijvoorbeeld, alle `max*` parameters moeten lagere waarden hebben dan de parent tenant. 
- Een white-labeled tenant mag **maximaal vijf pakketten** hebben.
- Alleen tenants met toegang tot white labeling mogen een `TenantPackage` aanmaken.
- Je mag geen pakketten toevoegen aan je eigen tenant. :)

We kunnen een `TenantPackage` als volgt aanmaken:

  --url 'https://fastcomments.com/api/v1/tenant-packages?tenantId=demo&API_KEY=DEMO_API_SECRET' \

  --data '{
  "name": "Default Package",
  "tenantId": "some-child-tenant-id",
  "monthlyCostUSD": null,
  "yearlyCostUSD": null,
  "maxMonthlyPageLoads": 50000,

  "hasWhiteLabeling": false,

  "forWhoText": "For Everyone",
