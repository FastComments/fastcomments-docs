Deze endpoint stelt je in staat een toewijzing van een gebruikersbadge te verwijderen.

Voorbeeldverzoek:

[inline-code-attrs-start title = 'Voorbeeld DELETE-aanvraag'; type = 'bash'; isFunctional = true; inline-code-attrs-end]
[inline-code-start]
curl -X DELETE "https://fastcomments.com/api/v1/user-badges/badge123?tenantId=demo&API_KEY=DEMO_API_SECRET"
[inline-code-end]

Voorbeeldantwoord:

[inline-code-attrs-start title = 'Antwoord'; type = 'json'; isFunctional = false; inline-code-attrs-end]
[inline-code-start]
{
  "status": "success"
}
[inline-code-end]
