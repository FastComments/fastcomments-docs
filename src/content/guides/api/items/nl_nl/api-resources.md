### Gebruik van resources

Het ophalen van gegevens van de API wordt als gebruik op uw account gerekend.

Elke resource vermeldt wat dat gebruik is in zijn eigen sectie.

Sommige resources kosten meer om te leveren dan andere. Elk endpoint heeft een vaste kosten in credits per API-aanroep. Voor sommige endpoints varieert het aantal credits
op basis van de opties en de grootte van de respons.

Het API-gebruik is te controleren op de [Factureringsanalyse](https://fastcomments.com/auth/my-account/analytics/billing)-pagina en wordt elk paar minuten bijgewerkt.

We raden aan eerst de Pages-documentatie te lezen, om verwarring te beperken bij het bepalen welke waarden je moet doorgeven voor `urlId` in de Comment API.
