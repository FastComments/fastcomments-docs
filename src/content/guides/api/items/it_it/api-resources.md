### Utilizzo delle risorse

Si noti che il recupero dei dati dall'API viene conteggiato come utilizzo sul tuo account.

Ogni risorsa indicherà quale sia tale utilizzo nella propria sezione.

Alcune risorse costano più di altre da servire. Ogni endpoint ha un costo fisso di crediti per chiamata API. Per alcuni endpoint, il numero di crediti
varia in base alle opzioni e alle dimensioni delle risposte.

L'utilizzo dell'API può essere controllato nella pagina [Analisi della fatturazione](https://fastcomments.com/auth/my-account/analytics/billing) ed è aggiornato ogni pochi minuti.

#### Nota!

Suggeriamo di leggere prima la documentazione Pages, per limitare la confusione quando si determina quali valori passare per `urlId` nella Comment API.
