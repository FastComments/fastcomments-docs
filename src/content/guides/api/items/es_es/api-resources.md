### Uso de Recursos

Debe tenerse en cuenta que obtener datos de la API se cuenta como uso en su cuenta.

Cada recurso listará cuál es ese uso en su propia sección.

Algunos recursos cuestan más de servir que otros. Cada endpoint tiene un costo fijo de créditos por llamada de API. Para algunos endpoints, el número de créditos
varía según las opciones y tamaños de respuesta.

El uso de la API se puede verificar en la página de [Análisis de Facturación](https://fastcomments.com/auth/my-account/analytics/billing) y se actualiza cada pocos minutos.

#### ¡Nota!

Sugerimos leer primero la documentación de Páginas, para ayudar a limitar la confusión al determinar qué valores pasar para `urlId` en la API de Comentarios.
