此端点允许您创建新的用户徽章分配。

示例请求：

[inline-code-attrs-start title = 'POST 请求示例'; type = 'bash'; isFunctional = true; inline-code-attrs-end]
[inline-code-start]
curl -X POST "https://fastcomments.com/api/v1/user-badges?tenantId=demo&API_KEY=DEMO_API_SECRET" \
-H "Content-Type: application/json" \
-d '{
  "userId": "user456",
  "badgeId": "badgeDef789",
  "displayedOnComments": true
}'
[inline-code-end]

请求体必须包含以下参数：

- `userId` (required) - 要分配徽章的用户 ID
- `badgeId` (required) - 要分配的徽章 ID
- `displayedOnComments` (optional) - 徽章是否应显示在用户的评论上（默认为 true）

重要说明：
1. 徽章必须存在于您的租户徽章目录中并已启用
2. 您只能将徽章分配给属于您的租户的用户或在您的网站上发表评论的用户
