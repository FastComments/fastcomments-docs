### Коришћење ресурса

Треба напоменути да се дохватање података са API-ја рачуна као коришћење на вашем налогу.

Сваки ресурс ће у свом одељку навести шта тачно подразумева то коришћење.

Неки ресурси су скупљи за послуживање од других. Сваки endpoint има фиксни трошак у кредитима по позиву API-ја. За неке endpoint-e, број кредита
варира у зависности од опција и величине одговора.

API коришћење можете провјерити на страници [Аналитика наплате](https://fastcomments.com/auth/my-account/analytics/billing) и ажурира се сваких неколико минута.

#### Напомена!

Препоручујемо да прво прочитате документацију за Pages, како бисте смањили конфузију при одређивању које вредности проследити за `urlId` у Comment API.
