요금은 댓글 요금제를 기준으로 하며, 즉 평점 및 리뷰는 패키지에 따라 댓글과 동일하게 청구됩니다 - depending on your package.

You are billed (on Flex) or limited (on fixed plans) based on the following:

- 계정에 있는 리뷰 수 (댓글 요금제와 동일).
- 요약 위젯 사용:
  - 집계가 캐시된 경우, 각 질문 + 하위 질문마다 페이지 로드당 API 크레딧 1개를 사용하며, 캐시되지 않은 경우 그 두 배를 사용합니다.
  - 따라서 메인 질문 하나와 하위 질문 세 개가 있는 경우, 대부분의 경우 페이지 로드당 4크레딧이 청구되며, 결과를 다시 계산해야 할 때는 8크레딧이 청구됩니다.
- API 사용 (각 API 호출에 대한 크레딧 사용량은 해당 문서에 포함되어 있음).

결과가 API 또는 요약 위젯을 통해 집계될 때에도 청구됩니다.
