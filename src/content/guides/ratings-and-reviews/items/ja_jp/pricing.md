価格はコメントの料金に基づいており、つまり、ご利用プランに応じて、評価およびレビューはコメントと同様に課金されます。

以下に基づいて（Flexでは課金され、固定プランでは制限されます）:

- アカウント内のレビュー数（Comment pricing と同じ）。
- Summary ウィジェットの使用:
  - ページ読み込みごとに、集約がキャッシュされている場合は各質問とサブ質問ごとに API クレジットを1つ使用し、キャッシュされていない場合はその2倍を使用します。
  - したがって、メインの質問が1つとサブ質問が3つある場合、ほとんどの場合は各ページ読み込みで4クレジットが課金され、結果の再計算が必要なときは8クレジットが課金されます。
- API の使用（各 API 呼び出しに対するクレジット使用量はそれぞれのドキュメントに記載されています）。

結果が API または Summary ウィジェット経由で集約される場合にも課金されます。
