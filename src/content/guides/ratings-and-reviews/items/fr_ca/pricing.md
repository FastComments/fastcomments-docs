La tarification est basée sur la tarification des commentaires, c’est-à-dire que vous êtes facturé pour les évaluations et les avis de la même manière que pour les commentaires - selon votre forfait.

Vous êtes facturé (sur Flex) ou soumis à des limites (sur les forfaits fixes) selon les éléments suivants :

- Le nombre d'avis que vous avez dans votre compte (même tarification que pour les commentaires).
- Utilisation du widget de résumé :
  - Chaque chargement de page consomme 1 crédit API pour chaque question + sous-question si l'agrégation est mise en cache, et le double si ce n'est pas le cas.
  - Ainsi, si vous avez une question principale et trois sous-questions, la plupart du temps vous êtes facturé 4 crédits par chargement de page, et parfois, lorsque les résultats doivent être recalculés, vous êtes facturé 8 crédits.
- Utilisation de l'API (l'utilisation des crédits pour chaque appel d'API est incluse dans la documentation respective).

Lorsque les résultats sont agrégés, que ce soit via l'API ou via les widgets de résumé, vous êtes également facturé.
