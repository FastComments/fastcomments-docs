Компоненты Ratings & Reviews, являющиеся частью модуля FastComments Feedback, имеют следующие заслуживающие внимания возможности:

- Два разных макета для вопросов — вертикальный и бок-о-бок (последний показан внизу этой страницы).
- Виджет сводки, который можно разместить на вашем сайте.
- Показывать пользователям агрегированную разбивку результатов после их ответа, отображая распределение всех ответов для этой страницы.
- *Скоро в январе 2024 года*: резюме и теги отзывов на основе ИИ.
- Агрегации по общим показателям и по времени в панели отчетности.
- API для интеграции для сохранения, получения и агрегации обратной связи.
- Возможность настраивать вопросы и подвопросы для каждого домена или каждой страницы без изменения кода.
