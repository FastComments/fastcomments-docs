Компоненти Ratings & Reviews, які є частиною модуля FastComments Feedback, мають такі особливі можливості:

- Два різні макети для запитань — вертикальний та макет поруч один з одним (останній показано внизу цієї сторінки).
- Віджет зведення, який можна розмістити на вашому сайті.
- Показувати користувачам агрегований розподіл результатів після відповіді, відображаючи розподіл усіх відповідей для цієї сторінки.
- *Незабаром у січні 2024 року*: узагальнення відгуків і теги з використанням AI.
- Загальні та часові агрегації в панелі звітності.
- API для інтеграції — для збереження, отримання та агрегації відгуків.
- Можливість налаштовувати питання та підпитання для кожного домену або сторінки без змін у коді.
