Ціноутворення базується на моделі для коментарів, тобто за рейтинги та відгуки вам виставляється рахунок так само, як і за коментарі — залежно від вашого пакета.

Вам виставляється рахунок (на Flex) або застосовуються обмеження (на фіксованих планах) на основі наступного:

- Кількість відгуків у вашому акаунті (так само, як ціноутворення для Comment).
- Використання зведеного віджета:
  - Кожне завантаження сторінки використовує 1 API-кредит для кожного питання + підпитання, якщо агрегація кешована, і вдвічі більше, якщо ні.
  - Тому, якщо у вас є головне питання та три підпитання, більшість часу списується 4 кредити за кожне завантаження сторінки, а іноді, коли результати потрібно перерахувати, списується 8 кредитів.
- Використання API (витрати кредитів для кожного виклику API включені в відповідні документи).

Коли результати агрегуються через API або через зведені віджети, вам також виставляється рахунок.
