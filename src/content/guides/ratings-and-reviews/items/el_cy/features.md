Τα στοιχεία Ratings & Reviews, που αποτελούν μέρος του FastComments Feedback module, έχουν τα ακόλουθα χαρακτηριστικά που αξίζει να επισημανθούν:

- Δύο διαφορετικές διατάξεις για ερωτήσεις - μία κάθετη και μία διάταξη δίπλα-δίπλα (η τελευταία εμφανίζεται στο κάτω μέρος αυτής της σελίδας).
- Ένα widget περίληψης που μπορείτε να τοποθετήσετε στον ιστότοπό σας.
- Εμφάνιση μιας συγκεντρωτικής ανάλυσης αποτελεσμάτων στους χρήστες μετά την απάντησή τους, δείχνοντας την κατανομή όλων των απαντήσεων για εκείνη τη σελίδα.
- *Πρόκειται να κυκλοφορήσει τον Ιανουάριο 2024*: Συνοψίσεις κριτικών και ετικέτες βασισμένες σε τεχνητή νοημοσύνη.
- Συνολικές και κατά χρονική περίοδο συγκεντρωτικές αναλύσεις στον πίνακα αναφορών.
- Ένα API για ενσωμάτωση, για αποθήκευση, ανάκτηση και συγκέντρωση ανατροφοδότησης.
- Η δυνατότητα προσαρμογής ερωτήσεων και υποερωτήσεων ανά domain ή ανά σελίδα χωρίς αλλαγή του κώδικα.
