Τα στοιχεία Ratings & Reviews, που αποτελούν μέρος του FastComments Feedback module, έχουν τα ακόλουθα χαρακτηριστικά που αξίζει να τονιστούν:

- Δύο διαφορετικές διατάξεις για τις ερωτήσεις - μία κάθετη και μία δίπλα-δίπλα (η δεύτερη εμφανίζεται στο κάτω μέρος αυτής της σελίδας).
- Ένα widget σύνοψης που μπορείτε να τοποθετήσετε στον ιστότοπό σας.
- *Έρχεται σύντομα τον Ιανουάριο του 2024*: Συνόψεις κριτικών και ετικέτες βασισμένες σε AI.
- Συγκεντρωτικά και χρονικά αθροίσματα στον πίνακα αναφορών.
- Ένα API για ενσωμάτωση που επιτρέπει την αποθήκευση, ανάκτηση και συγκέντρωση της ανατροφοδότησης.
- Δυνατότητα προσαρμογής ερωτήσεων και υποερωτήσεων ανά domain ή ανά σελίδα χωρίς αλλαγή κώδικα.
