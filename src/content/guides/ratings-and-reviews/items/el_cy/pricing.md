Η τιμολόγηση βασίζεται στην τιμολόγηση σχολίων, δηλαδή, σας χρεώνονται οι αξιολογήσεις & κριτικές με τον ίδιο τρόπο όπως τα σχόλια - ανάλογα με το πακέτο σας.

Σας χρεώνεται (στο Flex) ή περιορίζεστε (σε σταθερά πακέτα) βάσει των παρακάτω:

- Ο αριθμός των κριτικών που έχετε στο λογαριασμό σας (όπως και στην τιμολόγηση σχολίων).
- Χρήση του widget περίληψης:
  - Κάθε φόρτωση σελίδας χρησιμοποιεί 1 πίστωση API για κάθε ερώτηση + υποερώτηση αν η συγκέντρωση είναι αποθηκευμένη στην προσωρινή μνήμη, και το διπλάσιο αν όχι.
  - Έτσι, αν έχετε μια κύρια ερώτηση και τρεις υποερωτήσεις, τις περισσότερες φορές σας χρεώνονται 4 πιστώσεις ανά φόρτωση σελίδας, και μερικές φορές, όταν τα αποτελέσματα χρειάζεται να επανυπολογιστούν, σας χρεώνονται 8 πιστώσεις.
- Χρήση του API (η χρήση πιστώσεων για κάθε κλήση API περιλαμβάνεται στην αντίστοιχη τεκμηρίωση).

Όταν τα αποτελέσματα συγκεντρώνονται είτε μέσω του API είτε μέσω των widget περίληψης, επίσης χρεώνεστε.
