A precificação é baseada na precificação de comentários, isto é, você é cobrado por classificações e avaliações da mesma forma que por comentários - dependendo do seu pacote.

Você é cobrado (no Flex) ou limitado (em planos fixos) com base no seguinte:

- O número de avaliações que você tem na sua conta (mesmo que a precificação de comentários).
- Uso do widget de resumo:
  - Cada carregamento de página consome 1 crédito de API por cada pergunta + subpergunta se a agregação estiver em cache, e o dobro disso se não estiver.
  - Então, se você tiver uma pergunta principal e três subperguntas, na maioria das vezes você é cobrado 4 créditos a cada carregamento de página, e às vezes, quando os resultados precisam ser recalculados, você é cobrado 8 créditos.
- Uso da API (uso de créditos para cada chamada de API incluído na respectiva documentação).

Quando os resultados são agregados via a API ou via os widgets de resumo, você também é cobrado.
