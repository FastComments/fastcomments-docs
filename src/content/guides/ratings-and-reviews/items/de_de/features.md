Die Ratings- & Reviews-Komponenten, Teil des FastComments Feedback-Moduls, haben die folgenden hervorzuhebenden Funktionen:

- Zwei verschiedene Layouts für Fragen - ein vertikales und ein nebeneinander angeordnetes Layout (letzteres unten auf dieser Seite gezeigt).
- Ein Zusammenfassungs-Widget, das Sie auf Ihrer Website platzieren können.
- Den Nutzern nach ihrer Antwort eine Aufschlüsselung der aggregierten Ergebnisse anzeigen, die die Verteilung aller Antworten für diese Seite darstellt.
- *Kommt bald im Januar 2024*: KI-basierte Zusammenfassungen und Tags für Bewertungen.
- Gesamt- und zeitliche Aggregationen im Reporting-Dashboard.
- Eine API zur Integration zum Speichern, Abrufen und Aggregieren von Feedback.
- Die Möglichkeit, Fragen und Unterfragen pro Domain oder pro Seite anzupassen, ohne den Code zu ändern.
