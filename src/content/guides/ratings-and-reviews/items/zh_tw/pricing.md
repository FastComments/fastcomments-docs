價格是基於留言（comment）的計費，也就是說，評分與評論的計費方式與留言相同 - 視您的方案而定。

您會依據以下項目收費（在 Flex 方案）或受到限制（在固定方案）：

- 帳戶內的評論數量（與 Comment 計費相同）。
- Summary widget 使用情形：
  - 若聚合結果已快取，每次頁面載入對每個問題 + 子問題會使用 1 個 API 點數；若未快取則為兩倍。
  - 因此，如果您有一個主問題和三個子問題，大多數情況下每次頁面載入會收取 4 個點數，而在結果需要重新計算時會收取 8 個點數。
- API 使用（各 API 呼叫的點數使用詳見各自文件）。

當結果透過 API 或 summary widget 聚合時，也會產生收費。
