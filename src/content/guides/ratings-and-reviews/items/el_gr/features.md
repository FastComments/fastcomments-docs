Τα στοιχεία Ratings & Reviews, που αποτελούν μέρος της μονάδας FastComments Feedback, έχουν τα ακόλουθα χαρακτηριστικά που αξίζει να επισημανθούν:

- Δύο διαφορετικές διατάξεις για τις ερωτήσεις - μία κάθετη και μία δίπλα-δίπλα διάταξη (η τελευταία εμφανίζεται στο κάτω μέρος αυτής της σελίδας).
- Ένα widget περίληψης που μπορείτε να τοποθετήσετε στον ιστότοπό σας.
- Εμφάνιση συγκεντρωτικής ανάλυσης αποτελεσμάτων στους χρήστες μετά την απάντησή τους, προβάλλοντας την κατανομή όλων των απαντήσεων για εκείνη τη σελίδα.
- *Έρχεται σύντομα τον Ιανουάριο του 2024*: Συνόψεις κριτικών και ετικέτες βασισμένες σε AI.
- Συγκεντρωτικά συνολικά και χρονικά στοιχεία στον πίνακα αναφορών.
- Ένα API για ενσωμάτωση που επιτρέπει την αποθήκευση, ανάκτηση και συγκέντρωση της ανατροφοδότησης.
- Δυνατότητα προσαρμογής ερωτήσεων και υποερωτήσεων ανά domain ή ανά σελίδα χωρίς αλλαγή του κώδικα.
