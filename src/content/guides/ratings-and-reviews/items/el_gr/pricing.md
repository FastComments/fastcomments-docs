Η τιμολόγηση βασίζεται στην τιμολόγηση σχολίων, δηλαδή, χρεώνεστε για τις αξιολογήσεις και τις κριτικές με τον ίδιο τρόπο όπως για τα σχόλια - ανάλογα με το πακέτο σας.

You are billed (on Flex) or limited (on fixed plans) based on the following:

- Ο αριθμός των κριτικών που έχετε στον λογαριασμό σας (ίδιος με την τιμολόγηση σχολίων).
- Summary widget usage:
  - Κάθε φόρτωση σελίδας χρησιμοποιεί 1 API credit για κάθε ερώτηση + υποερώτηση αν η συγκέντρωση έχει αποθηκευτεί στην κρυφή μνήμη, και το διπλάσιο αν όχι.
  - Έτσι, αν έχετε μια κύρια ερώτηση και τρεις υποερωτήσεις, τις περισσότερες φορές σας χρεώνουν 4 credits ανά φόρτωση σελίδας, και μερικές φορές, όταν τα αποτελέσματα χρειαστεί να επανυπολογιστούν, σας χρεώνουν 8 credits.
- API usage (credits usage for each API call included in the respective docs).

Όταν τα αποτελέσματα συγκεντρώνονται είτε μέσω του API είτε μέσω των summary widgets, επίσης χρεώνεστε.
