La tarification est basée sur la tarification des commentaires, c'est-à-dire que vous êtes facturé pour les évaluations et avis de la même manière que pour les commentaires - en fonction de votre forfait.

Vous êtes facturé (sur Flex) ou limité (sur les forfaits fixes) en fonction de ce qui suit :

- Le nombre d'avis que vous avez dans votre compte (identique à la tarification des commentaires).
- Utilisation du widget récapitulatif :
  - Chaque chargement de page utilise 1 crédit API par question + sous-question si l'agrégation est mise en cache, et le double si ce n'est pas le cas.
  - Donc, si vous avez une question principale et trois sous-questions, la plupart du temps vous êtes facturé 4 crédits par chargement de page, et parfois, lorsque les résultats doivent être recalculés, vous êtes facturé 8 crédits.
- Utilisation de l'API (l'utilisation des crédits pour chaque appel d'API est incluse dans la documentation respective).

Lorsque les résultats sont agrégés via l'API ou via les widgets récapitulatifs, vous êtes également facturé.
