Компоненты Ratings & Reviews, являющиеся частью модуля Feedback FastComments, имеют следующие особенности, которые стоит выделить:

- Две разные компоновки вопросов — вертикальная и бок о бок (вторая показана внизу этой страницы).
- Виджет-резюме, который вы можете разместить на своём сайте.
- Показывает пользователям агрегированную разбивку результатов после ответа, отображая распределение всех ответов для этой страницы.
- *Скоро, в январе 2024 года*: резюме отзывов и теги на основе ИИ.
- Общие агрегаты и агрегаты по времени в панели отчётов.
- API для интеграции: сохранение, получение и агрегирование обратной связи.
- Возможность настраивать вопросы и подвопросы для каждого домена или страницы без изменения кода.
