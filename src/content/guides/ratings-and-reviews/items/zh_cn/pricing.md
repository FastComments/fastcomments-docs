定价基于评论定价，也就是说，评分和评论的计费方式与评论相同 - 取决于您的套餐。

在以下方面，您会被计费（在 Flex 计划上）或受到限制（在固定套餐上）：

- 您账户中的评价数量（与评论定价相同）。
- 汇总组件使用情况：
  - 如果聚合已缓存，则每次页面加载对每个问题及子问题消耗 1 个 API 积分；如果未缓存，则消耗两倍。
  - 因此，如果您有一个主问题和三个子问题，大多数情况下每次页面加载会计费 4 个积分；当结果需要重新计算时，可能会计费 8 个积分。
- API 使用（每次 API 调用的积分消耗详见各自文档）。

当结果通过 API 或汇总组件进行聚合时，也会产生计费。
