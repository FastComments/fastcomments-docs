### התקנת תלויות

```bash
sudo apt install libcpprest-dev libboost-all-dev
```

### בנייה מהמקור

```bash
mkdir build
cd build
cmake ..
make
```

### התקנה

```bash
sudo make install
```

### תוכן הספרייה

הספרייה הזו מכילה את לקוח ה-API שנוצר ואת כלי ה-SSO כדי להקל על עבודה עם ה-API.

- [תיעוד ספריית לקוח ה-API](https://github.com/FastComments/fastcomments-cpp/blob/master/client/README.md)

### ממשקי API ציבוריים מול מאובטחים

ל-API client יש שלוש מחלקות, `DefaultApi`, `PublicApi`, ו-`ModerationApi`. ה-`DefaultApi` מכילה שיטות שדורשות את מפתח ה-API שלך, ו-`PublicApi` מכילה
שיטות שניתן לקרוא מהדפדפן/טלפון נייד/וכו' ישירות ללא אימות. ה-`ModerationApi` מכילה שיטות שמפעילות את לוח הבקרה של המנחים - רישום,
ספירה, חיפוש, ייצוא ושליפת יומנים עבור תגובות, פעולות מתיחה (הסרה/שחזור, סימון, קביעת סטטוס סקירה/ספאם/אישור, התאמת הצבעות, פתיחה מחדש/סגירת שרשורים),
חסימות (חסימה מתוך תגובה, ביטול חסימות, סיכומי טרום-חסימה, מצב חסימה והעדפות, ספירות משתמשים חסומים), ותגים ואמון (הענקת/הסרת תגיות, תגיות ידניות, קבלת/הגדרת גורם אמון, פרופיל פנימי של משתמש). כל שיטה של `ModerationApi` מקבלת פרמטר `sso` כך שהקריאה מתבצעת בשם מנחה שאומת דרך SSO.
