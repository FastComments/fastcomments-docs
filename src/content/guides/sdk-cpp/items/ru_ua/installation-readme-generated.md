### Установка зависимостей

```bash
sudo apt install libcpprest-dev libboost-all-dev
```

### Сборка из исходников

```bash
mkdir build
cd build
cmake ..
make
```

### Установка

```bash
sudo make install
```

### Содержимое библиотеки

Эта библиотека содержит сгенерированный API-клиент и утилиты SSO, упрощающие работу с API.

- [Документация библиотеки API-клиента](https://github.com/FastComments/fastcomments-cpp/blob/master/client/README.md)

### Публичные и защищённые API

Для API-клиента есть три класса, `DefaultApi`, `PublicApi`, и `ModerationApi`. `DefaultApi` содержит методы, которые требуют вашего API-ключа, и `PublicApi` содержит
методы, которые можно вызывать напрямую из браузера/мобильного устройства/и т.д. без аутентификации. `ModerationApi` содержит методы, которые обеспечивают работу панели модератора - перечисление,
подсчёт, поиск, экспорт и получение логов для комментариев, действий модерации (удаление/восстановление, пометка/флаг, установка статуса на проверку/спам/одобрение, корректировка голосов, повторное открытие/закрытие веток),
баны (бан из комментария, отмена банов, предварительные сводки по банам, статус и настройки бана, подсчёт забаненных пользователей), и бейджи и доверие (назначение/удаление бейджей, ручные бейджи, получение/установка фактора доверия, внутренний профиль пользователя). Каждый метод `ModerationApi` принимает параметр `sso` так, чтобы вызов выполнялся от имени модератора, прошедшего аутентификацию через SSO.
