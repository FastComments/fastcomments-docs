### Αναγνωριστικά Broadcast

Θα δείτε ότι πρέπει να περάσετε ένα `broadcastId` σε κάποιες κλήσεις API. Όταν λάβετε συμβάντα, θα λάβετε πίσω αυτό το ID, οπότε θα ξέρετε να αγνοήσετε το συμβάν αν σκοπεύετε να εφαρμόσετε αλλαγές στον πελάτη με αισιοδοξία (κάτι που πιθανότατα θα θέλετε να κάνετε αφού προσφέρει την καλύτερη εμπειρία). Περνάτε εδώ ένα UUID. Το ID θα πρέπει να είναι αρκετά μοναδικό ώστε να μην εμφανιστεί δύο φορές σε μια συνεδρία προγράμματος περιήγησης.
