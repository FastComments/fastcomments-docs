Obecnie online oglądający stronę: osoby, których sesja websocket jest w tej chwili subskrybowana na tę stronę.
Zwraca anonCount + totalCount (subskrybenci w całym pokoju, w tym anonimowi oglądający, których nie wyliczamy).

## Parametry

| Name | Type | Required | Description |
|------|------|----------|-------------|
| tenantId | string | Yes |  |
| urlId | string | Yes |  |
| afterName | string | No |  |
| afterUserId | string | No |  |
