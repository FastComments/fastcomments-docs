### Установка зависимостей

```bash
sudo apt install libcpprest-dev libboost-all-dev
```

### Сборка из исходников

```bash
mkdir build
cd build
cmake ..
make
```

### Установка

```bash
sudo make install
```

### Содержимое библиотеки

Эта библиотека содержит сгенерированный клиент API и утилиты SSO, упрощающие работу с API.

- [Документация библиотеки клиента API](https://github.com/FastComments/fastcomments-cpp/blob/master/client/README.md)

### Публичные и защищённые API

Для клиентской библиотеки API существуют три класса, `DefaultApi`, `PublicApi`, и `ModerationApi`. `DefaultApi` содержит методы, требующие ваш API-ключ, а `PublicApi` содержит
методы, которые можно выполнять напрямую из браузера/мобильного устройства/и т.п. без аутентификации. `ModerationApi` содержит методы, обеспечивающие работу панели модератора — перечисление,
подсчёт, поиск, экспорт и получение журналов для комментариев, действий модерации (удаление/восстановление, пометка, установление статуса проверки/спама/одобрения, корректировка голосов, переоткрытие/закрытие веток),
баны (запрет комментирования, отмена банов, предварительные сводки перед баном, статус и настройки бана, количество забаненных пользователей), и значки и доверие (удостоение/удаление значков, ручные значки, получить/установить коэффициент доверия,
внутренний профиль пользователя). Каждый метод `ModerationApi` принимает параметр `sso`, поэтому вызов выполняется от имени модератора, аутентифицированного через SSO.
