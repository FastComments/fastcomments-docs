### Встановлення залежностей

```bash
sudo apt install libcpprest-dev libboost-all-dev
```

### Збирання з вихідного коду

```bash
mkdir build
cd build
cmake ..
make
```

### Встановлення

```bash
sudo make install
```

### Вміст бібліотеки

Ця бібліотека містить згенерований клієнт API та утиліти SSO, щоб полегшити роботу з API.

- [API Client Library Docs](https://github.com/FastComments/fastcomments-cpp/blob/master/client/README.md)

### Публічні та захищені API

Для клієнта API існують три класи: `DefaultApi`, `PublicApi` та `ModerationApi`. У `DefaultApi` містяться методи, які потребують вашого API-ключа, а `PublicApi` містить
методи, які можна викликати прямо з браузера/мобільного пристрою тощо без автентифікації. У `ModerationApi` містяться методи, які живлять панель модератора - перелік,
підрахунки, пошук, експорт та витяг логів для коментарів, дії модерації (видалення/відновлення, позначення, встановлення статусу на перевірку/спам/схвалення, коригування голосів, знову відкрити/закрити треди),
блокування (заборона у коментарі, скасування блокувань, підсумки перед блокуванням, статус і налаштування блокування, кількість заблокованих користувачів), та бейджі й довіра (нагородження/зняття бейджів, ручні бейджі, отримання/встановлення коефіцієнту довіри,
внутрішній профіль користувача). Кожен метод `ModerationApi` приймає параметр `sso`, тому виклик виконується від імені модератора, аутентифікованого через SSO.
