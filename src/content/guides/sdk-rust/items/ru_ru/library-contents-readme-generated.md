The FastComments Rust SDK состоит из нескольких модулей:

- **Client Module** - API-клиент для FastComments REST API
  - Полные определения типов для всех моделей API
  - Три API-клиента, покрывающие все методы FastComments:
    - `default_api` (**DefaultApi**) - методы, аутентифицируемые с помощью API-ключа, предназначенные для использования на стороне сервера
    - `public_api` (**PublicApi**) - публичные методы без API-ключа, безопасные для вызова из браузеров и мобильных приложений
    - `moderation_api` (**ModerationApi**) - методы, поддерживающие панель модератора, включая модерацию комментариев (список, подсчёт, поиск, логи, экспорт), действия модерации (удаление/восстановление, флаг, установка статуса на рассмотрение/спам/одобрение, голоса, повторное открытие/закрытие ветки), баны (забанить по комментарию, отмена, предварительные сводки по банам, статус/настройки бана, количество забаненных пользователей) и бейджи и доверие (выдача/удаление бейджей, ручные бейджи, получение/установка фактора доверия, внутренний профиль пользователя). Каждый метод Moderation принимает параметр `sso`, поэтому вызов может быть выполнен от имени модератора, аутентифицированного через SSO.
  - Полная поддержка async/await с tokio
  - См. [client/README.md](https://github.com/FastComments/fastcomments-rust/blob/main/client/README.md) для подробной документации по API

- **SSO Module** - Утилиты Single Sign-On для серверной части
  - Безопасная генерация токенов для аутентификации пользователей
  - Поддержка как простого, так и защищённого режимов SSO
  - Подписание токенов на основе HMAC-SHA256

- **Core Types** - Общие определения типов и утилиты
  - Модели комментариев и структуры метаданных
  - Настройки пользователей и тенантов
  - Вспомогательные функции для распространённых операций
