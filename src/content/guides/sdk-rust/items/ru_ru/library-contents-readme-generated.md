FastComments Rust SDK состоит из нескольких модулей:

- **Client Module** - API‑клиент для FastComments REST API
  - Полные определения типов для всех моделей API
  - Три API‑клиента, охватывающих все методы FastComments:
    - `default_api` (**DefaultApi**) — методы, аутентифицируемые с помощью API‑ключа, для серверного использования
    - `public_api` (**PublicApi**) — публичные методы без API‑ключа, безопасные для вызова из браузеров и мобильных приложений
    - `moderation_api` (**ModerationApi**) — обширный набор живых и быстрых API модерации. Каждый метод Moderation принимает параметр `sso` и может аутентифицироваться через SSO или cookie‑сессию FastComments.com.
  - Полная поддержка async/await с tokio
  - Смотрите [client/README.md](https://github.com/FastComments/fastcomments-rust/blob/main/client/README.md) для подробной документации API

- **SSO Module** — средства одноразового входа (SSO) на стороне сервера
  - Безопасная генерация токенов для аутентификации пользователей
  - Поддержка простых и защищенных режимов SSO
  - Подпись токенов на основе HMAC‑SHA256

- **Core Types** — общие определения типов и утилиты
  - Модели комментариев и структуры метаданных
  - Конфигурации пользователей и арендаторов
  - Вспомогательные функции для общих операций
