Το FastComments Rust SDK αποτελείται από αρκετές μονάδες:

- **Client Module** - Πελάτης API για τα FastComments REST APIs
  - Πλήρεις ορισμοί τύπων για όλα τα μοντέλα του API
  - Τρεις πελάτες API που καλύπτουν όλες τις μεθόδους FastComments:
    - `default_api` (**DefaultApi**) - μέθοδοι που αυθεντικοποιούνται με API key για χρήση από διακομιστή (server-side)
    - `public_api` (**PublicApi**) - δημόσιες μέθοδοι, χωρίς API key, που είναι ασφαλές να κληθούν από προγράμματα περιήγησης και κινητές εφαρμογές
    - `moderation_api` (**ModerationApi**) - μέθοδοι που υποστηρίζουν τον πίνακα ελέγχου του moderator, συμπεριλαμβανομένης της διαχείρισης σχολίων (λίστα, πλήθος, αναζήτηση, αρχεία καταγραφής, εξαγωγή), ενεργειών μετριασμού (αφαίρεση/επαναφορά, σημαία, ορισμός κατάστασης για ανασκόπηση/spam/έγκριση, ψήφοι, ξανανοίγω/κλείσιμο νήματος), απαγορεύσεων (ban από σχόλιο, αναιρέση, προ-προεπισκοπήσεις ban, κατάσταση/προτιμήσεις ban, μετρήσεις αποκλεισμένων χρηστών), και διακριτικών & εμπιστοσύνης (απονομή/αφαίρεση διακριτικών, χειροκίνητα διακριτικά, λήψη/ορισμός trust factor, εσωτερικό προφίλ χρήστη). Κάθε μέθοδος Moderation δέχεται παράμετρο `sso` ώστε η κλήση να μπορεί να γίνει εκ μέρους ενός moderator που έχει αυθεντικοποιηθεί με SSO.
  - Πλήρης υποστήριξη async/await με tokio
  - Δείτε το [client/README.md](https://github.com/FastComments/fastcomments-rust/blob/main/client/README.md) για αναλυτική τεκμηρίωση του API

- **SSO Module** - Εργαλεία Single Sign-On για τον server
  - Ασφαλής δημιουργία token για την αυθεντικοποίηση χρηστών
  - Υποστήριξη τόσο του απλού όσο και του ασφαλούς SSO
  - Υπογραφή token βάσει HMAC-SHA256

- **Core Types** - Κοινά ορισμοί τύπων και βοηθητικά εργαλεία
  - Μοντέλα σχολίων και δομές μεταδεδομένων
  - Διαμορφώσεις χρήστη και tenant
  - Βοηθητικές συναρτήσεις για κοινές λειτουργίες
