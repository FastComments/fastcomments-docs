The FastComments Rust SDK состоит из нескольких модулей:

- **Client Module** - Клиент для REST API FastComments
  - Полные определения типов для всех моделей API
  - Три клиента API, охватывающие все методы FastComments:
    - `default_api` (**DefaultApi**) - методы, аутентифицированные с помощью API-ключа, для использования на стороне сервера
    - `public_api` (**PublicApi**) - публичные методы без API-ключа, безопасные для вызова из браузеров и мобильных приложений
    - `moderation_api` (**ModerationApi**) - методы, используемые в панели модератора, включая модерирование комментариев (список, подсчёт, поиск, логи, экспорт), действия модерирования (удаление/восстановление, жалоба, установка статуса на проверку/спам/одобрение, голоса, повторное открытие/закрытие ветки), баны (бан по комментарию, отмена, предварительные сводки по банам, статус/настройки бана, количество забаненных пользователей) и бейджи и доверие (присвоение/удаление бейджей, ручные бейджи, получение/установка фактора доверия, внутренний профиль пользователя). Каждый метод Moderation принимает параметр `sso`, чтобы вызов можно было выполнить от имени модератора, аутентифицированного через SSO.
  - Полная поддержка async/await с tokio
  - См. [client/README.md](https://github.com/FastComments/fastcomments-rust/blob/main/client/README.md) для подробной документации по API

- **SSO Module** - Серверные утилиты единого входа (Single Sign-On)
  - Безопасная генерация токенов для аутентификации пользователей
  - Поддержка как простого, так и защищённого режимов SSO
  - Подписание токенов с использованием HMAC-SHA256

- **Core Types** - Общие определения типов и утилиты
  - Модели комментариев и структуры метаданных
  - Конфигурации пользователей и тенантов
  - Вспомогательные функции для общих операций
