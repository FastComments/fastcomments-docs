The FastComments Rust SDK се састоји из неколико модула:

- **Модул клијента** - API клијент за FastComments REST API-је
  - Комплетне дефиниције типова за све API моделе
  - Три API клијента који покривају све FastComments методе:
    - `default_api` (**DefaultApi**) - методе аутентификоване помоћу API кључа за коришћење на серверској страни
    - `public_api` (**PublicApi**) - јaвне, методе без API кључа које је сигурно позивати из прегледача и мобилних апликација
    - `moderation_api` (**ModerationApi**) - методе које покрећу модераторски интерфејс, укључујући модерацију коментара (листа, број, претрага, логови, извоз), радње модерације (уклони/врати, означи, постави статус за преглед/спам/одобрење, гласови, поновно отварање/затварање теме), забране (забрана по коментару, поништавање, прегледи пре-забране, статус/подешавања забране, бројеви забрањених корисника), и значке и поверење (додавање/уклањање значки, ручне значке, добијање/постављање фактора поверења, кориснички унутрашњи профил). Свака метода модерације прихвата параметар `sso` тако да се позив може извршити у име модератора аутентификованог преко SSO.
  - Пуна подршка за async/await са tokio
  - Погледајте [client/README.md](https://github.com/FastComments/fastcomments-rust/blob/main/client/README.md) за детаљну документацију API-ја

- **SSO модул** - серверске Single Sign-On алатке
  - Сигурна генерација токена за аутентификацију корисника
  - Подршка за једноставни и сигурни SSO режими
  - Потписивање токена засновано на HMAC-SHA256

- **Основни типови** - Заједничке дефиниције типова и алатке
  - Модели коментара и структуре метаподатака
  - Конфигурације корисника и тенанта
  - Помоћне функције за уобичајене операције
