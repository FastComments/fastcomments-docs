The FastComments Rust SDK се састоји од неколико модула:

- **Client Module** - API клијент за FastComments REST API-је
  - Потпуна дефиниција типова за све API моделе
  - Три API клијента који покривају све FastComments методе:
    - `default_api` (**DefaultApi**) - методе аутентификоване API кључем за серверско коришћење
    - `public_api` (**PublicApi**) - јавни, методе без API кључа које је безбедно позивати из прегледача и мобилних апликација
    - `moderation_api` (**ModerationApi**) - обимни набор живих и брзих API-ја за модерацију. Сваки модерацијски метод прихвата параметар `sso` и може се аутентиковати преко SSO или FastComments.com колачића сесије.
  - Пуна подршка за async/await са Tokio-ом
  - Погледајте [client/README.md](https://github.com/FastComments/fastcomments-rust/blob/main/client/README.md) за детаљну API документацију

- **SSO Module** - серверски алати за Single Sign-On
  - Безбедно генерисање токена за аутентификацију корисника
  - Подршка за оба режима, једноставни и безбедни SSO
  - Потписивање токена засновано на HMAC-SHA256

- **Core Types** - заједничке дефиниције типова и алати
  - Модели коментара и структуре метаподаци
  - Конфигурације корисника и закупника (тенанти)
  - Помоћне функције за уобичајене операције
