The FastComments Rust SDK се състои от няколко модула:

- **Client Module** - API клиент за FastComments REST API-та
  - Пълни дефиниции на типове за всички API модели
  - Три API клиента, обхващащи всички методи на FastComments:
    - `default_api` (**DefaultApi**) - методи, удостоверени с API ключ, за сървърна употреба
    - `public_api` (**PublicApi**) - публични методи, без API ключ, които са безопасни за извикване от браузъри и мобилни приложения
    - `moderation_api` (**ModerationApi**) - обширен набор от живи и бързи API-та за модерация. Всеки метод за модерация приема параметър `sso` и може да се удостоверява чрез SSO или чрез сесийна бисквитка на FastComments.com.
  - Пълна поддръжка на async/await с tokio
  - Вижте [client/README.md](https://github.com/FastComments/fastcomments-rust/blob/main/client/README.md) за подробна документация на API

- **SSO Module** - Инструменти за сървърно Single Sign-On
  - Сигурно генериране на токени за удостоверяване на потребители
  - Поддръжка както на прости, така и на сигурни SSO режимове
  - Подписване на токени, базирано на HMAC‑SHA256

- **Core Types** - Споделени дефиниции на типове и инструменти
  - Модели за коментари и структури на метаданни
  - Конфигурации за потребители и тенанти
  - Помощни функции за чести операции
