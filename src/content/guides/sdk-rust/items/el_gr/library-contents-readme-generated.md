Το FastComments Rust SDK αποτελείται από αρκετές μονάδες:

- **Μονάδα Πελάτη** - Πελάτης API για τα REST API του FastComments
  - Πλήρεις ορισμοί τύπων για όλα τα μοντέλα API
  - Τρεις πελάτες API που καλύπτουν όλες τις μεθόδους του FastComments:
    - `default_api` (**DefaultApi**) - μέθοδοι που αυθεντικοποιούνται με API key για χρήση από τον διακομιστή
    - `public_api` (**PublicApi**) - δημόσιες μέθοδοι χωρίς API key που είναι ασφαλές να κληθούν από προγράμματα περιήγησης και εφαρμογές κινητών
    - `moderation_api` (**ModerationApi**) - μέθοδοι που υποστηρίζουν τον πίνακα ελέγχου των συντονιστών, συμπεριλαμβανομένης της διαχείρισης σχολίων (λίστα, πλήθος, αναζήτηση, αρχεία καταγραφής, εξαγωγή), ενεργειών μέτριας (αφαίρεση/επανάκτηση, flag, ορισμός κατάστασης για ανασκόπηση/spam/έγκριση, ψήφοι, επανα-άνοιγμα/κλείσιμο θέματος), απαγορεύσεων (απαγόρευση από ένα σχόλιο, αναίρεση, προ-συνοψίσεις απαγόρευσης, κατάσταση/προτιμήσεις απαγόρευσης, αριθμοί απαγορευμένων χρηστών), και διακριτικών & εμπιστοσύνης (απονομή/αφαίρεση διακριτικών, χειροκίνητα διακριτικά, λήψη/ορισμός παράγοντα εμπιστοσύνης, εσωτερικό προφίλ χρήστη). Κάθε μέθοδος Moderation δέχεται μια παράμετρο `sso` ώστε η κλήση να μπορεί να γίνει εκ μέρους ενός συντονιστή που έχει αυθεντικοποιηθεί μέσω SSO.
  - Πλήρης υποστήριξη async/await με tokio
  - Δείτε το [client/README.md](https://github.com/FastComments/fastcomments-rust/blob/main/client/README.md) για λεπτομερή τεκμηρίωση API

- **Μονάδα SSO** - Εργαλεία Single Sign-On για την πλευρά του διακομιστή
  - Δημιουργία ασφαλών token για την αυθεντικοποίηση χρηστών
  - Υποστήριξη τόσο απλού όσο και ασφαλούς τρόπου λειτουργίας SSO
  - Υπογραφή token με βάση HMAC-SHA256

- **Βασικοί Τύποι** - Κοινές δηλώσεις τύπων και βοηθητικά εργαλεία
  - Μοντέλα σχολίων και δομές μεταδεδομένων
  - Ρυθμίσεις χρηστών και tenants
  - Βοηθητικές συναρτήσεις για κοινές λειτουργίες
