### Помилки 401 Unauthorized

Якщо ви отримуєте помилки 401 під час використання автентифікованого API:

1. **Перевірте ваш API-ключ**: Переконайтеся, що ви використовуєте правильний API-ключ із вашої панелі керування FastComments
2. **Перевірте tenant ID**: Переконайтеся, що tenant ID відповідає вашому обліковому запису
3. **Формат API-ключа**: API-ключ має бути переданий у Configuration:

```rust
let mut config = Configuration::new();
config.api_key = Some(ApiKey {
    prefix: None,
    key: "YOUR_API_KEY".to_string(),
});
```

### Проблеми з SSO-токенами

Якщо SSO-токени не працюють:

1. **Використовуйте захищений режим для продакшну**: Завжди використовуйте `FastCommentsSSO::new_secure()` з вашим API-ключем для продакшну
2. **Тільки на сервері**: Генеруйте SSO-токени на вашому сервері, ніколи не розкривайте ваш API-ключ клієнтам
3. **Перевірте дані користувача**: Переконайтеся, що всі обов'язкові поля (id, email, username) заповнені

### Помилки асинхронного рантайму

SDK використовує tokio для асинхронних операцій. Переконайтеся, що:

1. Додайте tokio до ваших залежностей:
```toml
[dependencies]
tokio = { version = "1", features = ["full"] }
```

2. Використовуйте рантайм tokio:
```rust
#[tokio::main]
async fn main() {
    // Ваш асинхронний код тут
}
```
