Følg de samme trin for `localhost`, som du ville for produktion. Sørg for, at du har produktionsdomæner og API Secrets konfigureret.

Først skal du navigere til [Webhooks-administration](https://fastcomments.com/auth/my-account/manage-data/webhooks). Dette er tilgængeligt via Administrer data -> Webhooks.

Konfigurationssiden vises som følger:

[app-screenshot-start url='/auth/my-account/manage-data/webhooks'; selector = '.content'; title='Webhooks Configuration' app-screenshot-end]

På denne side kan du angive endepunkter for hver type af kommentarhændelse.

For hver type hændelse skal du sørge for at klikke på Send Test Payload for at sikre, at du har opsat din integration korrekt. Se næste afsnit, "Testing", for detaljer.
