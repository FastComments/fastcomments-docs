Wykonaj te same kroki dla `localhost`, co dla środowiska produkcyjnego. Upewnij się, że masz skonfigurowane domeny produkcyjne i API Secrets.

Najpierw przejdź do [Webhooks admin](https://fastcomments.com/auth/my-account/manage-data/webhooks). Można to znaleźć w Manage Data -> Webhooks.

The configuration page appears as follows:

[app-screenshot-start url='/auth/my-account/manage-data/webhooks'; selector = '.content'; title='Webhooks Configuration' app-screenshot-end]

Na tej stronie możesz określić punkty końcowe dla każdego typu zdarzenia komentarza.

Dla każdego typu zdarzenia pamiętaj, aby kliknąć Send Test Payload, aby upewnić się, że integracja została poprawnie skonfigurowana. Zobacz następną sekcję, "Testowanie", aby uzyskać szczegóły.
