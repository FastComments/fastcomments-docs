Ακολούθησε τα ίδια βήματα για το `localhost` όπως θα έκανες για την παραγωγή. Βεβαιώσου ότι έχεις ρυθμίσει τα domains παραγωγής και τα API Secrets.

First, navigate to the [Webhooks admin](https://fastcomments.com/auth/my-account/manage-data/webhooks). This is accessible via Διαχείριση δεδομένων -> Webhooks.

The configuration page appears as follows:

[app-screenshot-start url='/auth/my-account/manage-data/webhooks'; selector = '.content'; title='Webhooks Configuration' app-screenshot-end]

Σε αυτή τη σελίδα μπορείς να καθορίσεις endpoints για κάθε τύπο συμβάντος σχολίου.

Για κάθε τύπο συμβάντος, βεβαιώσου ότι κάνεις κλικ στο Send Test Payload για να διασφαλίσεις ότι έχεις ρυθμίσει σωστά την ενσωμάτωσή σου. Δες την επόμενη ενότητα, "Δοκιμές", για λεπτομέρειες.
