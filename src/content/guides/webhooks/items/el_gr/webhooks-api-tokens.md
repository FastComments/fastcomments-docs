Στην κεφαλίδα του αιτήματος θα περάσουμε το API Secret σας στην παράμετρο που ονομάζεται "token".

Εάν δεν ελέγξετε σωστά αυτήν την παράμετρο, η ενσωμάτωσή σας δεν θα επισημανθεί ως Verified.
Αυτό είναι ένα μέτρο ασφαλείας για να διασφαλιστεί ότι οποιεσδήποτε ενσωματώσεις με το FastComments είναι ασφαλείς.
