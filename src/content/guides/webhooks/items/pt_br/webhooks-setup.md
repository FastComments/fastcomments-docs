Siga os mesmos passos para `localhost` como faria em produção. Garanta que você tenha domínios de produção e API Secrets configurados.

First, navigate to the [Webhooks admin](https://fastcomments.com/auth/my-account/manage-data/webhooks). This is accessible via Manage Data -> Webhooks.

A página de configuração aparece da seguinte forma:

[app-screenshot-start url='/auth/my-account/manage-data/webhooks'; selector = '.content'; title='Configuração de Webhooks' app-screenshot-end]

Nesta página você pode especificar endpoints para cada tipo de evento de comentário.

Para cada tipo de evento, certifique-se de clicar em Enviar payload de teste para garantir que você configurou sua integração corretamente. Veja a próxima seção, "Testes", para mais detalhes.
