FastComments WordPress додатак ће заменити подразумевани WordPress механизам за бројање системом који обезбеђује FastComments. Послаћемо један асинхрони захтев
по учитавању странице када постоји листа постова којима треба приказати број коментара. Ако корисник има искључен JavaScript, вратиће се на подразумеване WordPress бројаче како не би дошло до губитка функционалности.
