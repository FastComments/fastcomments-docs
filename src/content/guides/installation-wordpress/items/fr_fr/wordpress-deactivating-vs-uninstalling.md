La désactivation du plugin supprimera votre jeton de votre installation WordPress. Vos données restent sur les serveurs de FastComments jusqu'à ce que vous supprimiez votre compte. Si vous réactivez le plugin, vous devrez
effectuer de nouveau la synchronisation initiale.

La désinstallation du plugin supprime toutes les données liées au plugin, mais vous devrez toujours supprimer votre compte FastComments pour effacer les données qui y résident.
