Vous pouvez rencontrer une erreur d'autorisation, comme la suivante :

<div class="screenshot white-bg">
    <div class="title">Configuration du domaine manquante</div>
    <img class="screenshot-image" src="/images/installation-guides/installation-guide-webflow-step-5.png" alt="Configuration du domaine manquante" />
</div>

Cela arrive parce que FastComments ne sait pas qu'il doit autoriser l'utilisation de votre compte sur ce domaine.

Dans ce cas, la solution est simple. Vous devez simplement ajouter votre site à votre compte FastComments.com.

[Cliquez ici pour ajouter votre site à votre compte.](https://fastcomments.com/auth/my-account/configure-domains)

Les domaines sont ajoutés automatiquement à votre compte pendant la période d'essai, mais cela peut se produire lorsque vous ajoutez de nouveaux sites à votre compte FastComments.
