Disattivare il plugin rimuoverà il tuo token dall'installazione di WordPress. I tuoi dati rimangono sui server di FastComments fino a quando non elimini il tuo account. Se riattivi il plugin, allora dovrai
eseguire nuovamente la sincronizzazione iniziale.

Disinstallare il plugin rimuove tutti i dati relativi al plugin, ma dovrai comunque eliminare il tuo account FastComments per cancellare i dati che risiedono lì.
