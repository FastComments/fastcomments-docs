התוסף FastComments עבור WordPress יחליף את מנגנון הספירה המוגדר כברירת מחדל של WordPress במערכת המסופקת על ידי FastComments. נבצע בקשה אסינכרונית אחת
בכל טעינת דף כאשר יש רשימת פוסטים עם ספירות תגובות להצגה. אם למשתמש JavaScript מושבת, המערכת תחזור לספירות המוגדרות של WordPress כך
שלא תהיה ירידה בפונקציונליות.
