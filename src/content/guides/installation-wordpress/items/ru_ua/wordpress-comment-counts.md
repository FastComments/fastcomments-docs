Плагин FastComments для WordPress заменит стандартный механизм подсчёта WordPress системой, предоставляемой FastComments. Мы будем выполнять один асинхронный запрос
при загрузке страницы, когда есть список записей с отображаемыми счётчиками комментариев. Если у пользователя отключён JavaScript, будет выполнён откат к стандартным счётчикам WordPress, чтобы
не произошло потери функциональности.
