Das Deaktivieren des Plugins entfernt Ihren Token aus Ihrer WordPress-Installation. Ihre Daten verbleiben auf den FastComments-Servern, bis Sie Ihr Konto löschen. Wenn Sie das Plugin erneut aktivieren, dann müssen Sie
die Erstsynchronisierung erneut durchführen.

Die Deinstallation des Plugins entfernt alle mit dem Plugin verbundenen Daten, aber Sie müssten trotzdem Ihr FastComments-Konto löschen, um die dort gespeicherten Daten zu entfernen.
