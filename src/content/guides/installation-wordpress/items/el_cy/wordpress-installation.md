Η εγκατάσταση του FastComments WordPress plugin έχει σχεδιαστεί ώστε να είναι εύκολη και να περιορίζει οποιαδήποτε διακοπή της διαθεσιμότητας των σχολίων στον ιστότοπό σας. Το plugin δεν θα πραγματοποιήσει την αλλαγή από το WordPress plugin μέχρι να ολοκληρωθεί ο συγχρονισμός.

Πρώτα, θα θελήσετε να θέσετε προσδοκίες με τους moderators σας σύμφωνα με τις λεπτομέρειες στην παρακάτω ενότητα `Moderation`. Ενημερώστε τους ότι σύντομα θα διαχειρίζονται τα σχόλια από το FastComments, και όχι από το WordPress dashboard.

Στη συνέχεια, εγκαταστήστε το plugin. Μπορείτε να το κάνετε πηγαίνοντας στο WordPress plugins dashboard, κάνοντας κλικ στο `Add Plugin`, και αναζητώντας το `FastComments`. Το plugin είναι επίσης διαθέσιμο για λήψη [εδώ](https://wordpress.org/plugins/fastcomments/#installation).

Η ρύθμιση του plugin θα σας ρωτήσει αν βρίσκεστε στην ΕΕ. Εάν ισχύει αυτό, ο λογαριασμός σας θα βρίσκεται στην ΕΕ στο `eu.fastcomments.com`, διαφορετικά θα είναι στο `fastcomments.com`.

Επόμενο, έχετε την επιλογή να επιλέξετε αν έχετε ήδη λογαριασμό FastComments. Ακολουθήστε τα βήματα και τα σχόλιά σας μπορούν προαιρετικά να ανεβούν κατά τη διάρκεια της διαδικασίας εγκατάστασης.

<sup>Σημείωση!</sup> Μην κλείσετε τη σελίδα του WordPress plugin ενώ η ρύθμιση και ο συγχρονισμός βρίσκονται σε εξέλιξη.

Τα σχόλια ανεβαίνουν σε παρτίδες και ταξινομούνται χρησιμοποιώντας υπάρχοντα ευρετήρια στη βάση δεδομένων WordPress σας για να περιοριστεί το φορτίο στον ιστότοπό σας.

Μόλις ολοκληρωθεί ο συγχρονισμός, το προεπιλεγμένο σύστημα σχολιασμού WordPress θα αντικατασταθεί από το FastComments. Ελπίζουμε να σας αρέσει η πλατφόρμα μας, αλλά αν όχι [επικοινωνήστε μαζί μας](https://fastcomments.com/auth/my-account/help).
