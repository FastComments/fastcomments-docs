Το FastComments και το αντίστοιχο πρόσθετο για WordPress είναι έτοιμα ώστε οι συντονιστές και οι διαχειριστές σας να συνεχίσουν να εργάζονται.

Πιθανότατα θα θέλετε να προσθέσετε τους διαχειριστές σας στην [Χρήστες & Διαχειριστές](https://fastcomments.com/auth/my-account/users) και τους συντονιστές σας στους [Συντονιστές Σχολίων](https://fastcomments.com/auth/my-account/moderate-comments/moderators).

Αυτό απαιτείται εάν δεν χρησιμοποιείτε SSO.
