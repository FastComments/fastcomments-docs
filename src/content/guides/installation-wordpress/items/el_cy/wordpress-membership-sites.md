FastComments λειτουργεί με ιστότοπους μόνο για μέλη χρησιμοποιώντας κάτι που ονομάζεται SSO, δηλαδή single-sign-on. Τα μέλη σας συνδέονται στον ιστότοπό σας στο WordPress, αλλά δεν χρειάζεται να ανησυχούν για τη δημιουργία λογαριασμού στο FastComments ή για σύνδεση με κοινωνικά δίκτυα για να σχολιάσουν. Εάν τα μέλη σας (συμπεριλαμβανομένων των διαχειριστών) είναι συνδεδεμένα στον ιστότοπό σας στο WordPress, θα μπορούν να σχολιάζουν. Οι διαχειριστές και οι συντονιστές σας θα μπορούν επίσης να διαχειρίζονται σχόλια απευθείας από τις δημοσιεύσεις του ιστολογίου σας στο WordPress.

<sup>(Προαιρετικό)</sup> Θυμηθείτε επίσης να προσθέσετε τους διαχειριστές σας στο [User & Administrators](https://fastcomments.com/auth/my-account/users) και τους συντονιστές στο [Comment Moderators](https://fastcomments.com/auth/my-account/moderate-comments/moderators) για να βελτιώσετε την εμπειρία τους και να ενεργοποιήσετε την παρακολούθηση στατιστικών για τους συντονιστές.

Το SSO μπορεί να ενεργοποιηθεί πηγαίνοντας στον πίνακα του πρόσθετου και κάνοντας κλικ στο "SSO Settings".

Πρώτα θα πρέπει να ενεργοποιήσετε τη λειτουργία "Anyone can Register" του ιστότοπού σας.

Όλες οι πληροφορίες χρήστη μεταφέρονται αβίαστα και με ασφάλεια στο FastComments κάθε φορά που ένας χρήστης προβάλλει ένα νήμα σχολίων μέσω του [HMAC](https://en.wikipedia.org/wiki/HMAC).

Δεν απαιτείται αρχική ή συνεχής συγχρονισμός για να αντιγραφούν τα μέλη σας στους servers του FastComments. Αυτό γίνεται αυτόματα όταν προβάλλουν νήματα σχολίων ανοίγοντας μια ανάρτηση ιστολογίου.

## Ονόματα και Εικόνες προφίλ

Το πρόσθετο θα ενημερώνει αυτόματα το εμφανιζόμενο όνομα του χρήστη και την εικόνα προφίλ σε όλα τα σχόλιά του στο FastComments όταν προβάλει οποιοδήποτε νήμα σχολίων. Οι εικόνες προφίλ υποστηρίζονται μέσω του Gravatar ή οποιουδήποτε πρόσθετου διαχείρισης avatar στο WordPress, καθώς το πρόσθετο θα καλεί τη συνάρτηση `get_avatar_url`.
