Η απενεργοποίηση του προσθέτου θα αφαιρέσει το token από την εγκατάσταση του WordPress σας. Τα δεδομένα σας παραμένουν στους διακομιστές FastComments μέχρι να διαγράψετε τον λογαριασμό σας. Εάν επανενεργοποιήσετε το πρόσθετο, τότε θα
πρέπει να εκτελέσετε ξανά τον αρχικό συγχρονισμό.

Η απεγκατάσταση του προσθέτου αφαιρεί όλα τα δεδομένα που σχετίζονται με το πρόσθετο, αλλά παρ' όλα αυτά θα πρέπει να διαγράψετε τον λογαριασμό σας στο FastComments για να διαγράψετε τα δεδομένα που φιλοξενούνται εκεί.
