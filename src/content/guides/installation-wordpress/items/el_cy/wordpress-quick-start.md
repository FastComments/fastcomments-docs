Αυτή είναι η σύντομη (TL;DR) έκδοση των οδηγιών για το WordPress.

1. Μεταβείτε στη σελίδα Προσθέτων (Plugins) του πίνακα ελέγχου του WordPress σας. Κάντε κλικ στο `Add Plugin`.
2. Αναζητήστε το `FastComments`.
3. Εγκαταστήστε και ενεργοποιήστε το πρόσθετό μας.
4. Κάντε κλικ στο `FastComments` στο αριστερό μενού πλοήγησης.
5. Ακολουθήστε τις οδηγίες. Αυτό συνήθως διαρκεί μερικά λεπτά, όχι περισσότερο.
   1. Εάν έχετε δεκάδες χιλιάδες ή εκατομμύρια σχόλια, ο συγχρονισμός μπορεί να χρειαστεί κάποιο χρόνο.
