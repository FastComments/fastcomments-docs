Σε αυτή την περίπτωση, η επιδιόρθωση είναι απλή. Απλώς πρέπει να προσθέσετε τον ιστότοπό σας στον λογαριασμό σας στο FastComments.com.

[Πατήστε εδώ για να προσθέσετε τον ιστότοπό σας στον λογαριασμό σας.](https://fastcomments.com/auth/my-account/configure-domains)

Οι τομείς προστίθενται αυτόματα στον λογαριασμό σας κατά τη διάρκεια της περιόδου δοκιμής, αλλά αυτό μπορεί να συμβεί όταν προσθέτετε νέους ιστότοπους στον λογαριασμό σας στο FastComments.
