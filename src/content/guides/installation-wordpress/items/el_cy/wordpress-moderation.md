FastComments αντικαθιστά τα προεπιλεγμένα εργαλεία διαχείρισης του WordPress με τα δικά του. [Μπορείτε να αποκτήσετε πρόσβαση στη Διαχείριση Σχολίων εδώ](https://fastcomments.com/auth/my-account/moderate-comments).

Τα εργαλεία διαχείρισης έχουν σχεδιαστεί για να είναι γρήγορα και εύχρηστα. Όλες οι ενέργειες διαχείρισης γίνονται σε πραγματικό χρόνο.

Μάθετε περισσότερα για τη ροή εργασίας διαχείρισης, τα στατιστικά των συντονιστών, τη ρύθμιση και άλλα [εδώ](/guide-moderation.html).
