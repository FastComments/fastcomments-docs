Από προεπιλογή, το FastComments συγχρονίζει πίσω στον ιστότοπό σας WordPress καθημερινά. Αυτό γίνεται αποκλειστικά για σκοπούς αντιγράφου ασφαλείας ώστε να εξακολουθείτε να κατέχετε ένα αντίγραφο των δεδομένων, και για πρόσθετα που ενδέχεται να εξαρτώνται από αυτό.

Αυτό δεν συμβαίνει αμέσως με κάθε αποθηκευμένο σχόλιο λόγω του ότι κάποιοι ιστότοποι μπορούν να χειρίζονται υψηλή αναγνωστική κίνηση, αλλά οι εγκαταστάσεις της βάσης δεδομένων τους δεν είναι πάντα σε θέση να χειριστούν τη μεγάλη εγγραφική κίνηση (γι' αυτό και η εργασία αυτή ανατίθεται στο FastComments).

Το πρόγραμμα συγχρονισμού προς το WordPress μπορεί να προσαρμοστεί εγκαθιστώντας ένα πρόσθετο. Σας προτείνουμε το [WP Crontrol](https://wordpress.org/plugins/wp-crontrol/#description).

Βήματα:

1. Εγκαταστήστε το WP Crontrol
2. Μεταβείτε στο `Settings -> Cron Schedules`.
3. Μεταβείτε στην καρτέλα `Cron Events`.
4. Αναζητήστε το `fastcomments_cron_hook`.
5. Επεξεργαστείτε το γεγονός. Μπορείτε να ρυθμίσετε το hook ώστε να εκτελείται κάθε ώρα, δύο φορές την ημέρα, καθημερινά (προεπιλογή) ή μία φορά την εβδομάδα.

Ο συγχρονισμός προς το WordPress μπορεί επίσης να γίνει οποιαδήποτε στιγμή χειροκίνητα πηγαίνοντας στον πίνακα ελέγχου του πρόσθετου FastComments και επιλέγοντας `Manually Sync`. Θα έχετε την επιλογή να συγχρονίσετε πίσω στην εγκατάσταση του WP σας, ή να ανεβάσετε ξανά τα σχόλια του WP στους διακομιστές του FastComments.
