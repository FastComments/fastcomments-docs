FastComments WordPress додатак ће заменити подразумевани WordPress механизам за бројање системом који пружа FastComments. Направићемо један асинхрони захтев
по учитавању странице када постоји листа постова са бројевима коментара за приказ. Ако корисник има онемогућен JavaScript, он ће се вратити на подразумеване WordPress бројаче тако
да не долази до губитка функционалности.
