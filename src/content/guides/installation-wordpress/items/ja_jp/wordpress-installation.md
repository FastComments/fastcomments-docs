FastComments の WordPress プラグインのインストールは簡単で、サイトのコメント機能のダウンタイムを最小限に抑えるように設計されています。同期が完了するまでは、プラグインは WordPress のプラグインから切り替わりません。

まず、以下のセクション `Moderation` の詳細に従って、モデレーターに期待値を伝えてください。まもなく彼らは WordPress のダッシュボードではなく FastComments からモデレーションを行うことになると知らせておいてください。

次に、プラグインをインストールします。WordPress のプラグインダッシュボードへ行き、`Add Plugin` をクリックして `FastComments` を検索するとインストールできます。プラグインは [こちら](https://wordpress.org/plugins/fastcomments/#installation) からもダウンロードできます。

プラグインのセットアップでは、EU にいるかどうかを尋ねられます。EU にいる場合はアカウントは EU の `eu.fastcomments.com` に保存され、それ以外の場合は `fastcomments.com` に保存されます。

次に、すでに FastComments のアカウントをお持ちかどうかを選択できます。その後の手順に従うと、インストール中にコメントを任意でアップロードすることができます。

<sup>注意！</sup> セットアップと同期が進行中は、WordPress のプラグインページを閉じないでください。

コメントはバッチ単位でアップロードされ、サイトへの負荷を抑えるために既存の WordPress データベースのインデックスを使用してソートされます。

同期が完了すると、デフォルトの WordPress コメントシステムは FastComments に置き換えられます。私たちのプラットフォームを気に入っていただけることを願っていますが、もしそうでなければ [お問い合わせください](https://fastcomments.com/auth/my-account/help)。
