FastComments WordPress додатак ће заменити подразумевани WordPress механизам бројања системом који пружа FastComments. Направићемо један асинхрони захтев
по учитавању странице када постоји листа постова са бројем коментара за приказ. Ако корисник има онемогућен JavaScript, вратиће се на подразумеване WordPress бројеве тако да не
дође до губитка функционалности.
