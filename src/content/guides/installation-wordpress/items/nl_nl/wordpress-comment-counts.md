---
De FastComments WordPress plugin zal het standaard WordPress-telmechanisme vervangen door een systeem geleverd door FastComments. We zullen één asynchrone aanvraag
per paginalaadbeurt doen wanneer er een lijst met berichten is met het aantal reacties om te tonen. Als de gebruiker JavaScript heeft uitgeschakeld, valt het terug op de standaard WordPress-tellingen zodat er
geen verlies aan functionaliteit is.
