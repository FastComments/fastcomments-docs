Η εγκατάσταση του FastComments plugin για WordPress έχει σχεδιαστεί ώστε να είναι εύκολη και να περιορίζει οποιαδήποτε διακοπή στη διαθεσιμότητα των σχολίων στον ιστότοπό σας. Το plugin δεν θα αντικαταστήσει το σύστημα σχολιασμού του WordPress μέχρι να ολοκληρωθεί ο συγχρονισμός.

Πρώτα, θα θελήσετε να θέσετε τις προσδοκίες με τους συντονιστές σας σύμφωνα με τις λεπτομέρειες στην παρακάτω ενότητα `Moderation`. Ενημερώστε τους ότι σύντομα θα κάνουν moderation από το FastComments, και όχι από τον πίνακα ελέγχου του WordPress.

Στη συνέχεια, εγκαταστήστε το plugin. Μπορείτε να το κάνετε πηγαίνοντας στον πίνακα πρόσθετων (plugins) του WordPress, κάνοντας κλικ στο `Add Plugin`, και αναζητώντας το `FastComments`. Το plugin είναι επίσης διαθέσιμο για λήψη [εδώ](https://wordpress.org/plugins/fastcomments/#installation).

Το πρόγραμμα εγκατάστασης του plugin θα σας ρωτήσει αν βρίσκεστε στην ΕΕ. Εάν ισχύει αυτό, ο λογαριασμός σας θα φιλοξενηθεί στην ΕΕ στο `eu.fastcomments.com`, αλλιώς θα βρίσκεται στο `fastcomments.com`.

Έπειτα, έχετε την επιλογή να επιλέξετε αν έχετε ήδη έναν λογαριασμό FastComments. Ακολουθήστε τα βήματα στη συνέχεια και τα σχόλιά σας μπορούν προαιρετικά να ανεβούν κατά τη διάρκεια της διαδικασίας εγκατάστασης.

<sup>Σημείωση!</sup> Μην κλείσετε τη σελίδα του plugin του WordPress ενώ η ρύθμιση και ο συγχρονισμός βρίσκονται σε εξέλιξη.

Τα σχόλια ανεβαίνουν σε παρτίδες και ταξινομούνται χρησιμοποιώντας τους υπάρχοντες δείκτες στη βάση δεδομένων του WordPress για να περιοριστεί το φορτίο στον ιστότοπό σας.

Μόλις ολοκληρωθεί ο συγχρονισμός, το προεπιλεγμένο σύστημα σχολιασμού του WordPress θα αντικατασταθεί από το FastComments. Ελπίζουμε να αγαπήσετε την πλατφόρμα μας, αλλά αν όχι [επικοινωνήστε μαζί μας](https://fastcomments.com/auth/my-account/help).
