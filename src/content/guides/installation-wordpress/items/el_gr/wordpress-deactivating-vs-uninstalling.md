Η απενεργοποίηση του πρόσθετου θα αφαιρέσει το token σας από την εγκατάσταση WordPress. Τα δεδομένα σας παραμένουν στους διακομιστές του FastComments μέχρι να διαγράψετε τον λογαριασμό σας. Εάν επανενεργοποιήσετε το πρόσθετο, τότε θα
πρέπει να εκτελέσετε ξανά τον αρχικό συγχρονισμό。

Η απεγκατάσταση του πρόσθετου αφαιρεί όλα τα δεδομένα που σχετίζονται με το πρόσθετο, αλλά θα πρέπει να διαγράψετε τον λογαριασμό FastComments για να διαγράψετε τα δεδομένα που βρίσκονται εκεί.
