Αυτή είναι η "too long did not read" έκδοση των οδηγιών για το WordPress.

1. Μεταβείτε στη σελίδα Πρόσθετα του πίνακα ελέγχου του WordPress. Κάντε κλικ στο `Add Plugin`.
2. Αναζητήστε το `FastComments`.
3. Εγκαταστήστε και ενεργοποιήστε το πρόσθετό μας.
4. Κάντε κλικ στο `FastComments` στην αριστερή πλοήγηση.
5. Ακολουθήστε τις οδηγίες. Αυτό συνήθως διαρκεί λίγα λεπτά, αλλά όχι περισσότερο.
   1. Εάν έχετε δεκάδες χιλιάδες ή εκατομμύρια σχόλια, μπορεί να χρειαστεί κάποιος χρόνος για συγχρονισμό.
