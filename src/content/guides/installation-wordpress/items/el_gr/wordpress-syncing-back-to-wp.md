Από προεπιλογή, το FastComments συγχρονίζει πίσω στον ιστότοπό σας WordPress καθημερινά. Αυτό γίνεται αποκλειστικά για λόγους δημιουργίας αντιγράφων ασφαλείας ώστε να συνεχίσετε να κατέχετε ένα αντίγραφο των δεδομένων, και για πρόσθετα
που μπορεί να εξαρτώνται από αυτό.

Αυτό δεν συμβαίνει άμεσα με κάθε αποθηκευμένο σχόλιο εξαιτίας της φύσης ορισμένων ιστότοπων που μπορούν να χειρίζονται μεγάλο φόρτο ανάγνωσης, οι αναπτύξεις της βάσης δεδομένων τους δεν είναι πάντα σε θέση να χειριστούν τον μεγάλο φόρτο εγγραφής (εξ ου και η ανάθεση αυτής της εργασίας στο FastComments).

Το πρόγραμμα συγχρονισμού πίσω στο WordPress μπορεί να προσαρμοστεί εγκαθιστώντας ένα πρόσθετο. Συστήνουμε το [WP Crontrol](https://wordpress.org/plugins/wp-crontrol/#description).

Steps:

1. Εγκαταστήστε το WP Crontrol
2. Μεταβείτε στο `Settings -> Cron Schedules`.
3. Μεταβείτε στην καρτέλα `Cron Events`.
4. Αναζητήστε το `fastcomments_cron_hook`.
5. Επεξεργαστείτε το συμβάν. Μπορείτε να διαμορφώσετε το hook ώστε να εκτελείται ανά ώρα, δύο φορές την ημέρα, καθημερινά (προεπιλογή), ή μία φορά την εβδομάδα.

Ο συγχρονισμός πίσω στο WordPress μπορεί επίσης να εκτελεστεί οποιαδήποτε στιγμή χειροκίνητα μεταβαίνοντας στον πίνακα ελέγχου του πρόσθετου FastComments και επιλέγοντας `Manually Sync`. Θα έχετε
την επιλογή να συγχρονίσετε πίσω στην εγκατάσταση WP σας, ή να επαναφορτώσετε τα σχόλια WP σας στους διακομιστές του FastComments.
