Το πρόσθετο FastComments για WordPress θα αντικαταστήσει τον προεπιλεγμένο μηχανισμό καταμέτρησης του WordPress με ένα σύστημα που παρέχεται από το FastComments. Θα κάνουμε ένα ασύγχρονο αίτημα
ανά φόρτωση σελίδας όταν υπάρχει μια λίστα δημοσιεύσεων με αριθμούς σχολίων για εμφάνιση. Εάν ο χρήστης έχει απενεργοποιημένο το JavaScript, θα επιστρέψει στους προεπιλεγμένους μετρητές του WordPress έτσι
να μην υπάρξει απώλεια λειτουργικότητας.
