FastComments λειτουργεί με ιστότοπους μόνο για μέλη χρησιμοποιώντας κάτι που ονομάζεται SSO, ή single-sign-on. Τα μέλη σας συνδέονται στον ιστότοπό σας WordPress, αλλά
δεν χρειάζεται να ανησυχούν για τη δημιουργία λογαριασμού στο FastComments ή για σύνδεση μέσω κοινωνικών δικτύων, για να σχολιάσουν. Αν τα μέλη σας (συμπεριλαμβανομένων των διαχειριστών) είναι συνδεδεμένα
στον ιστότοπό σας WordPress, θα μπορούν να σχολιάζουν. Οι διαχειριστές και οι συντονιστές σας θα μπορούν επίσης να διαχειρίζονται σχόλια απευθείας από τις δημοσιεύσεις του ιστολογίου σας στο WordPress.

<sup>(Προαιρετικό)</sup> Θυμηθείτε επίσης να προσθέσετε τους διαχειριστές σας στους [User & Administrators](https://fastcomments.com/auth/my-account/users) και τους συντονιστές στους [Comment Moderators](https://fastcomments.com/auth/my-account/moderate-comments/moderators)
για να βελτιώσετε την εμπειρία τους και να ενεργοποιήσετε την παρακολούθηση στατιστικών για τους συντονιστές.

Το SSO μπορεί να ενεργοποιηθεί πηγαίνοντας στον πίνακα ελέγχου του plugin και κάνοντας κλικ στο "SSO Settings".

Πρώτα θα πρέπει να ενεργοποιήσετε τη δυνατότητα "Anyone can Register" του ιστότοπού σας.

Όλες οι πληροφορίες χρήστη μεταφέρονται απρόσκοπτα και με ασφάλεια στο FastComments κάθε φορά που ένας χρήστης προβάλλει ένα νήμα σχολίων μέσω του [HMAC](https://en.wikipedia.org/wiki/HMAC).

Δεν απαιτείται αρχική ή συνεχής συγχρονισμός για να αντιγραφούν τα μέλη σας στους servers του FastComments. Αυτό γίνεται αυτόματα όταν προβάλλουν νήματα σχολίων ανοίγοντας μια δημοσίευση στο ιστολόγιο.

## Ονόματα και Avatar

Το plugin θα ενημερώνει αυτόματα το εμφανιζόμενο όνομα του χρήστη και το avatar σε όλα τα σχόλιά τους στο FastComments όταν προβάλλουν
οποιοδήποτε νήμα σχολίων. Τα avatars υποστηρίζονται μέσω Gravatar ή οποιουδήποτε plugin διαχείρισης avatar στο WordPress καθώς το plugin θα καλεί `get_avatar_url`.
