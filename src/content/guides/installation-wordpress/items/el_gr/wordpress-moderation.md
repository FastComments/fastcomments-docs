Το FastComments αντικαθιστά τα προεπιλεγμένα εργαλεία εποπτείας του WordPress με τα δικά του. [Μπορείτε να αποκτήσετε πρόσβαση στην Εποπτεία Σχολίων εδώ](https://fastcomments.com/auth/my-account/moderate-comments).

Τα εργαλεία εποπτείας μας έχουν σχεδιαστεί για να είναι γρήγορα και εύκολα στη χρήση. Όλες οι ενέργειες εποπτείας γίνονται σε πραγματικό χρόνο.

Μάθετε περισσότερα για τη ροή εργασιών εποπτείας, τα στατιστικά των συντονιστών, τη διαμόρφωση και άλλα [εδώ](/guide-moderation.html).
