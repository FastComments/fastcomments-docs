Desativar o plugin removerá seu token da sua instalação do WordPress. Seus dados permanecem nos servidores do FastComments até que você exclua sua conta. Se você reativar o plugin, então você terá
que realizar a sincronização inicial novamente.

Desinstalar o plugin remove todos os dados relacionados ao plugin, mas você ainda terá que excluir sua conta do FastComments para apagar os dados que residem lá.
