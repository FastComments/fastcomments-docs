Плагін FastComments для WordPress замінить стандартний механізм підрахунку WordPress на систему, яку надає FastComments. Ми здійснюватимемо один асинхронний запит
на завантаження сторінки, коли є список дописів із кількістю коментарів для відображення. Якщо у користувача вимкнено JavaScript, воно повернеться до стандартного підрахунку WordPress, щоб
не сталося втрати функціональності.
