Плагин FastComments для WordPress заменит стандартный механизм подсчёта WordPress на систему, предоставляемую FastComments. Мы будем выполнять один асинхронный запрос
на загрузку страницы, когда есть список записей с количеством комментариев для отображения. Если у пользователя отключён JavaScript, будет выполнен возврат к стандартным подсчётам WordPress, так что
функциональность не терялась.
