Désactiver le plugin supprimera votre jeton de votre installation WordPress. Vos données restent sur les serveurs de FastComments jusqu'à ce que vous supprimiez votre compte. Si vous réactivez le plugin, alors vous
devrez effectuer à nouveau la synchronisation initiale.

Désinstaller le plugin supprime toutes les données liées au plugin, mais vous devrez toujours supprimer votre compte FastComments pour effacer les données qui y résident.
