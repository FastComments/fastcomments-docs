Voici la version "trop longue; pas lue" des instructions WordPress.

1. Accédez à la page Plugins de votre tableau de bord WordPress. Cliquez sur `Add Plugin`.
2. Recherchez `FastComments`.
3. Installez et activez notre plugin.
4. Cliquez sur `FastComments` dans la navigation de gauche.
5. Suivez les instructions. Cela prend généralement quelques minutes, pas plus.
   1. Si vous avez des dizaines de milliers, ou des millions de commentaires, la synchronisation peut prendre un certain temps.
