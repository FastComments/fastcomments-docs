L'installation du plugin WordPress FastComments est conçue pour être simple et limiter toute interruption de la disponibilité des commentaires sur votre site. Le plugin ne basculera pas depuis le plugin WordPress tant que la synchronisation n'est pas terminée.

Tout d'abord, vous voudrez préparer vos modérateurs en fonction des détails de la section ci‑dessous `Moderation`. Informez‑les qu'ils
modéreront bientôt depuis FastComments, et non depuis le tableau de bord WordPress.

Ensuite, installez le plugin. Vous pouvez le faire en vous rendant dans le tableau de bord des plugins WordPress, en cliquant sur `Add Plugin`, et en recherchant `FastComments`. Le plugin est également disponible en téléchargement [ici](https://wordpress.org/plugins/fastcomments/#installation).

Le processus d'installation du plugin vous demandera si vous êtes dans l'UE. Si c'est le cas, votre compte résidera dans l'UE sur `eu.fastcomments.com`, sinon il sera sur `fastcomments.com`.

Ensuite, vous aurez l'option d'indiquer si vous avez déjà un compte FastComments. Suivez ensuite les étapes et vos commentaires pourront éventuellement être importés lors du processus d'installation.

<sup>Remarque!</sup> Ne fermez pas la page du plugin WordPress pendant que la configuration et la synchronisation sont en cours.

Les commentaires sont importés par lots et triés en utilisant les index existants de votre base de données WordPress afin de limiter la charge sur votre site.

Une fois la synchronisation terminée, le système de commentaires par défaut de WordPress sera remplacé par FastComments. Nous espérons que notre plateforme vous plaira, mais si ce n'est pas le cas, [contactez‑nous](https://fastcomments.com/auth/my-account/help).
