Ceci est dû au fait que FastComments ne sait pas qu'il doit autoriser l'utilisation de votre compte sur ce domaine.

Dans ce cas, la solution est simple. Vous devez simplement ajouter votre site à votre compte FastComments.com.

[Rendez-vous ici pour ajouter votre site à votre compte.](https://fastcomments.com/auth/my-account/configure-domains)

Les domaines sont ajoutés automatiquement à votre compte pendant la période d'essai, mais cela peut aussi se produire lorsque vous ajoutez de nouveaux sites à votre compte FastComments.
