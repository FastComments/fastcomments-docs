### מזהי שידור

תראה שעליך להעביר `broadcast_id` בחלק מבקשות ה-API. כאשר תקבל אירועים, תקבל בחזרה את המזהה הזה, כך תדע להתעלם מהאירוע אם בכוונתך להחיל שינויים באופן אופטימי בצד הלקוח (סביר שתרצה לעשות זאת שכן זה מספק את חוויית המשתמש הטובה ביותר). יש להעביר כאן UUID. המזהה צריך להיות ייחודי מספיק כדי שלא יופיע פעמיים במהלך סשן בדפדפן.
