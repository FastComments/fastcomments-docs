### Αναγνωριστικά Broadcast

Θα δείτε ότι πρέπει να περάσετε ένα `broadcast_id` σε ορισμένες κλήσεις API. Όταν λαμβάνετε συμβάντα, θα λάβετε αυτό το ID πίσω, οπότε θα ξέρετε να αγνοήσετε το συμβάν εάν σκοπεύετε να εφαρμόσετε αλλαγές οπτιμιστικά στην πλευρά του πελάτη (κάτι που μάλλον θα θέλετε να κάνετε καθώς προσφέρει την καλύτερη εμπειρία). Περάστε εδώ ένα UUID. Το ID πρέπει να είναι αρκετά μοναδικό ώστε να μην εμφανιστεί δύο φορές σε μία συνεδρία προγράμματος περιήγησης.
