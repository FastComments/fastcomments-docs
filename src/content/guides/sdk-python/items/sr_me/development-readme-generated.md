### Покретање тестова

```bash
# Поставите променљиве окружења
export FASTCOMMENTS_API_KEY="your-api-key"
export FASTCOMMENTS_TENANT_ID="your-tenant-id"

# Покрените тестове
pytest tests/
```

### Поновно генерисање клијента

Да бисте поново генерисали API клијента из најновије OpenAPI спецификације:

```bash
./update.sh
```

Ово ће:
1. Преузети најновију OpenAPI спецификацију са покренутог FastComments сервера (или користити локални openapi.yaml)
2. Генерисати Python клиентски код
3. Упростити структуру директоријума
4. Уклонити сувишне конфигурационе датотеке
