### Вміст бібліотеки

Ця бібліотека містить два модулі: згенерований API-клієнт та базову Python-бібліотеку, яка містить вручну написані утиліти для спрощення роботи з API, зокрема підтримку SSO.

- [Документація бібліотеки клієнта API](https://github.com/FastComments/fastcomments-python/blob/main/client/README.md)
- [Документація ядра бібліотеки, включно з прикладами SSO](https://github.com/FastComments/fastcomments-python/blob/main/sso/README.md)

### Публічні та захищені API

У API-клієнті є три класи, `DefaultApi`, `PublicApi` та `ModerationApi`. `DefaultApi` містить методи, що потребують вашого API-ключа, а `PublicApi` містить методи, які можна викликати безпосередньо з браузера/мобільного пристрою тощо без аутентифікації. `ModerationApi` керує панеллю модератора і містить методи для модерації коментарів (перелік, підрахунок, пошук, журнали, експорт), дій модерації (видалення/відновлення, позначення, встановлення статусу перегляду/спаму/підтвердження, голоси, повторне відкриття/закриття теми), банів (заборона коментування, скасування, попередні зведення перед баном, статус/повідомлення про бан/налаштування, кількість забанених користувачів) та значків і довіри (призначення/зняття значка, ручні значки, отримання/встановлення коефіцієнта довіри, внутрішній профіль користувача). Кожен метод `ModerationApi` приймає параметр `sso`, щоб його можна було викликати від імені модератора, автентифікованого через SSO.
