### Αναγνωριστικά Broadcast

Θα δείτε ότι πρέπει να περάσετε ένα `broadcast_id` σε κάποιες κλήσεις API. Όταν λαμβάνετε συμβάντα, θα λάβετε αυτό το ID πίσω, ώστε να ξέρετε να αγνοήσετε το συμβάν αν σκοπεύετε να εφαρμόσετε τις αλλαγές οπτιμιστικά στην πλευρά του πελάτη (κάτι που πιθανότατα θα θέλετε να κάνετε καθώς προσφέρει την καλύτερη εμπειρία). Περάστε ένα UUID εδώ. Το ID θα πρέπει να είναι αρκετά μοναδικό ώστε να μην εμφανιστεί δύο φορές σε μια συνεδρία του προγράμματος περιήγησης.
