### Запуск тестов

```bash
# Установите переменные окружения
export FASTCOMMENTS_API_KEY="your-api-key"
export FASTCOMMENTS_TENANT_ID="your-tenant-id"

# Запустите тесты
pytest tests/
```

### Перегенерация клиента

Чтобы перегенерировать клиент API из последней спецификации OpenAPI:

```bash
./update.sh
```

Это выполнит:
1. Скачает последнюю спецификацию OpenAPI с запущенного сервера FastComments (или использует локальный openapi.yaml)
2. Сгенерирует клиентский код на Python
3. Выровняет структуру каталогов
4. Очистит лишние файлы конфигурации
