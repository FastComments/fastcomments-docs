### Стартиране на тестове

```bash
# Настройте променливите на средата
export FASTCOMMENTS_API_KEY="your-api-key"
export FASTCOMMENTS_TENANT_ID="your-tenant-id"

# Стартирайте тестовете
pytest tests/
```

### Прегенериране на клиента

За да презгенерирате API клиента от последната OpenAPI спецификация:

```bash
./update.sh
```

Това ще:
1. Изтегли най-новата OpenAPI спецификация от работещ FastComments сървър (или използвайте локален openapi.yaml)
2. Генерира Python клиентски код
3. Опростява структурата на директориите
4. Премахва излишните конфигурационни файлове
