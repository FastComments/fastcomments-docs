Te komponenty są nakładkami na nasze podstawowe komponenty VanillaJS. Możemy automatycznie aktualizować te komponenty (poprawiać błędy, dodawać funkcje) bez publikowania tej biblioteki, więc choć może nie być ona opublikowana przez jakiś czas, nie oznacza to, że FastComments nie jest aktywnie rozwijany! Zapraszamy do odwiedzenia [naszego bloga](https://blog.fastcomments.com/) po aktualizacje. Zmiany łamiące kompatybilność API lub nowe funkcje nigdy nie zostaną wprowadzone do podstawowej biblioteki bez podniesienia wersji w tej bibliotece.
