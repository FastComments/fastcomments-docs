Deze componenten zijn wrappers rond onze core VanillaJS-componenten. We kunnen deze componenten automatisch bijwerken (bugs oplossen, functies toevoegen) zonder deze bibliotheek te publiceren, dus ook al wordt deze misschien enige tijd niet gepubliceerd, betekent dat niet dat FastComments niet actief in ontwikkeling is! Bekijk gerust [onze blog](https://blog.fastcomments.com/) voor updates. Compatibiliteitsbrekende API-wijzigingen of functies zullen nooit naar de onderliggende core-bibliotheek worden doorgevoerd zonder een versieverhoging in deze bibliotheek.
