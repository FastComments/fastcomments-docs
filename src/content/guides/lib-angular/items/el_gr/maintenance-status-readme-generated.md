Αυτά τα components είναι wrappers γύρω από τα core VanillaJS components μας. Μπορούμε να ενημερώνουμε αυτόματα αυτά τα components (να διορθώνουμε σφάλματα, να προσθέτουμε λειτουργίες) χωρίς να δημοσιεύουμε αυτή τη βιβλιοθήκη, οπότε παρόλο που μπορεί να μην δημοσιευτεί για κάποιο διάστημα, αυτό δεν σημαίνει ότι το FastComments δεν βρίσκεται σε ενεργή ανάπτυξη! Μη διστάσετε να ελέγξετε [το blog μας](https://blog.fastcomments.com/) για ενημερώσεις. Αλλαγές που σπάνε το API ή νέες λειτουργίες δεν θα αποσταλούν ποτέ στη βασική βιβλιοθήκη χωρίς αύξηση έκδοσης σε αυτή τη βιβλιοθήκη.
