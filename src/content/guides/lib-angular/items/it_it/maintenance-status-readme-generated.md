Questi componenti sono wrapper attorno ai nostri componenti core di VanillaJS. Possiamo aggiornare automaticamente questi componenti (correggere bug, aggiungere funzionalità) senza pubblicare questa libreria, quindi il fatto che possa non essere pubblicata per un po' non significa che FastComments non sia in sviluppo attivo! Sentiti libero di consultare [il nostro blog](https://blog.fastcomments.com/) per aggiornamenti. Le modifiche o le funzionalità che rompono l'API non saranno mai inserite nella libreria core sottostante senza un aumento di versione in questa libreria.
