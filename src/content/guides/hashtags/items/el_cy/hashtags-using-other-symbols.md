Κατόπιν αιτήματος, μπορούν να χρησιμοποιηθούν σύμβολα διαφορετικά από το `#`.

Διαφορετικά σύμβολα ετικετών μπορούν να χρησιμοποιηθούν για να συνδεθούν με διαφορετικά κομμάτια περιεχομένου. Για παράδειγμα, το API θα μπορούσε να χρησιμοποιηθεί για να ορίσει hashtags με το σύμβολο `#`
ως στοιχεία θεμάτων, αλλά επίσης να ορίσει καταχωρίσεις όπως `~document_one` και `~document_two`. Σε αυτό το σενάριο, οι χρήστες θα μπορούσαν να χρησιμοποιούν τόσο hashtags όσο και προσαρμοσμένες ετικέτες
για να παραπέμψουν σε άλλα έγγραφα.
