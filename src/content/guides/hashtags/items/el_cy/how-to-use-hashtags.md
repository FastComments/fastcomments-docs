#### Πλήρωση της Αυτόματης Συμπλήρωσης

HashTags μπορούν να συμπληρωθούν με δύο τρόπους.

Πρώτον, από προεπιλογή οι χρήστες μπορούν να πληκτρολογήσουν το σύμβολο `#` και κάποιο κείμενο - και εάν υπάρχουν υπάρχοντα hashtags θα εμφανιστεί μια διεπαφή αυτόματης συμπλήρωσης.

Εάν ένας χρήστης εισάγει νέο hashtag, [εκτός αν απενεργοποιηθεί](/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation), το hashtag
θα δημιουργηθεί αυτόματα και θα επισημανθεί.

Ο δεύτερος τρόπος είναι να ορίσετε χειροκίνητα κάθε hashtag. Αυτό καλύπτεται σε αυτό το έγγραφο.

#### Περιπτώσεις Χρήσης

HashTags είναι χρήσιμα για τα μέλη της κοινότητας για να πλοηγηθούν και να βρουν σχετικές συζητήσεις. Είναι επίσης χρήσιμα για να επισημάνουν σχετικά κομμάτια πληροφορίας, ως συνδέσμους, που μπορούν να οριστούν μέσω του API μας. Για παράδειγμα, τα hashtags θα μπορούσαν να χρησιμοποιηθούν για να επισημάνουν:

- Tickets σε σύστημα καταγραφής αιτημάτων.
- Έγγραφα ή Σελίδες στο CRM.
- Αρχεία.
