Κατά προεπιλογή, κάνοντας κλικ σε ένα hashtag ανοίγει μια προβολή με μια λίστα σχολίων που περιέχουν το hashtag. Ωστόσο, μπορούμε να αλλάξουμε αυτό ώστε να συνδέεται με ένα
εξωτερικό περιεχόμενο για κάθε hashtag.

Αυτό μπορεί να γίνει όταν προσθέτουμε hashtags [μέσω του API](/guide-api.html#hash-tag-structure). Όταν προσθέτουμε ένα hashtag μέσω του API είναι δυνατό
να ορίσουμε ένα URL για κάθε ετικέτα.
