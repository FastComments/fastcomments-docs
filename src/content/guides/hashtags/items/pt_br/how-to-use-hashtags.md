#### Populando o Autocompletar

HashTags podem ser preenchidas de duas maneiras.

Primeiro, por padrão, os usuários podem digitar o símbolo `#` e algum texto - e se houver hashtags existentes uma interface de autocompletar será exibida.

Se um usuário inserir uma nova hashtag, [a não ser que esteja desativado](/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation), a hashtag
será criada automaticamente e destacada.

A segunda maneira é definir manualmente cada hashtag. Isto é coberto neste documento.

#### Casos de Uso

HashTags são úteis para membros da comunidade navegarem para encontrar discussões relevantes. Elas também são úteis para marcar partes relevantes
de informação, como links, que podem ser definidos via nossa API. Por exemplo, hashtags poderiam ser usadas para marcar:

- Tickets em um sistema de atendimento.
- Documentos ou Páginas no CRM.
- Arquivos.
