Mediante solicitação, símbolos diferentes de `#` podem ser usados.

Diferentes símbolos de tag podem ser usados para vincular a diferentes partes do conteúdo. Por exemplo, a API poderia ser usada para definir hashtags com o símbolo `#`
como itens de tópico, mas também definir entradas como `~document_one` e `~document_two`. Nesse cenário, os usuários poderiam usar hashtags assim como tags personalizadas
para apontar para outros documentos.
