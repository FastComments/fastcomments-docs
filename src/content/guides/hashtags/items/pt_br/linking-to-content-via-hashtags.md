Por padrão, ao clicar em uma hashtag, abre-se uma visualização com uma lista de comentários com a hashtag. No entanto, podemos alterar isso para vincular a um
conteúdo externo para cada hashtag.

Isso pode ser feito ao adicionar hashtags [via a API](/guide-api.html#hash-tag-structure). Quando adicionamos uma hashtag via a API, é possível
definir uma URL para cada tag.
