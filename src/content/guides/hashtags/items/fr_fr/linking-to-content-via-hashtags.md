Par défaut, cliquer sur un hashtag ouvre une vue avec une liste de commentaires avec le hashtag. Cependant, nous pouvons modifier cela pour le faire
pointer vers un contenu externe pour chaque hashtag.

Cela peut être fait lors de l'ajout de hashtags [via l'API](/guide-api.html#hash-tag-structure). Lorsque nous ajoutons un hashtag via l'API il est possible
de définir une URL pour chaque tag.
