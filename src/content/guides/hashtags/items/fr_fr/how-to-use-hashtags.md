#### Remplissage de l'autocomplétion

HashTags peuvent être renseignés de deux manières.

First, by default users can type the `#` symbol and some text - and if there are existing hashtags an autocomplete UI will be shown.

Si un utilisateur saisit un nouveau hashtag, [sauf si désactivée](/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation), le hashtag
sera automatiquement créé et mis en évidence.

The second way is to manually define each hashtag. This is covered in this document.

#### Cas d'utilisation

HashTags sont utiles pour les membres de la communauté afin de naviguer et de trouver des discussions pertinentes. Ils sont également utiles pour taguer des éléments d'information pertinents, sous forme de liens, qui peuvent être définis via notre API. Par exemple, les hashtags pourraient être utilisés pour taguer :

- Tickets dans un système de ticketing.
- Documents ou pages dans le CRM.
- Fichiers.
