Auf Anfrage können andere Symbole als `#` verwendet werden.

Verschiedene Tag-Symbole können verwendet werden, um auf unterschiedliche Inhalte zu verlinken. Beispielsweise könnte die API verwendet werden, um Hashtags mit dem Symbol `#`
als Themenelemente zu definieren, aber auch Einträge wie `~document_one` und `~document_two`. In diesem Szenario könnten Benutzer Hashtags sowie benutzerdefinierte Tags
verwenden, um auf andere Dokumente zu verweisen.
