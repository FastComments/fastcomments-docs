#### Autocomplete befüllen

Hashtags können auf zwei Arten befüllt werden.

First, by default users can type the `#` symbol and some text - and if there are existing hashtags an autocomplete UI will be shown.

Wenn ein Benutzer ein neues Hashtag eingibt, [sofern nicht deaktiviert](/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation), das Hashtag
wird automatisch erstellt und hervorgehoben.

Die zweite Möglichkeit besteht darin, jedes Hashtag manuell zu definieren. Dies wird in diesem Dokument behandelt.

#### Anwendungsfälle

Hashtags sind nützlich für Mitglieder der Community, um relevante Diskussionen zu finden. Sie sind auch nützlich, um relevante
Informationsstücke als Links zu kennzeichnen, die über unsere API definiert werden können. Zum Beispiel könnten Hashtags verwendet werden, um zu kennzeichnen:

- Tickets in einem Ticketsystem.
- Dokumente oder Seiten im CRM.
- Dateien.
