Standardmäßig öffnet ein Klick auf ein Hashtag eine Ansicht mit einer Liste von Kommentaren mit dem Hashtag. Wir können dies jedoch ändern, sodass für jedes Hashtag auf ein
externes Inhaltsstück verlinkt.

Dies kann beim Hinzufügen von Hashtags [über die API](/guide-api.html#hash-tag-structure) durchgeführt werden. Wenn wir ein Hashtag über die API hinzufügen, ist es möglich,
für jeden Tag eine URL zu definieren.
