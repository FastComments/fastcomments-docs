Κατόπιν αιτήματος, μπορούν να χρησιμοποιηθούν σύμβολα διαφορετικά από το `#`.

Διάφορα σύμβολα ετικετών μπορούν να χρησιμοποιηθούν για να συνδέσουν διαφορετικά κομμάτια περιεχομένου. Για παράδειγμα, το API θα μπορούσε να χρησιμοποιηθεί για να ορίσει hashtags με το σύμβολο `#`
ως στοιχεία θέματος, αλλά και να ορίσει καταχωρήσεις όπως `~document_one` και `~document_two`. Σε αυτό το σενάριο, οι χρήστες θα μπορούσαν να χρησιμοποιούν hashtags καθώς και προσαρμοσμένες ετικέτες
για να δείξουν σε άλλα έγγραφα.
