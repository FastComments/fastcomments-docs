#### Συμπλήρωση Αυτόματης Συμπλήρωσης

HashTags μπορούν να συμπληρωθούν με δύο τρόπους.

Πρώτον, εξ ορισμού οι χρήστες μπορούν να πληκτρολογήσουν το `#` symbol και κάποιο κείμενο - και αν υπάρχουν υπάρχοντα hashtags θα εμφανιστεί μια διεπαφή αυτόματης συμπλήρωσης.

Εάν ένας χρήστης εισάγει νέο hashtag, [εκτός αν έχει απενεργοποιηθεί](/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation), το hashtag
θα δημιουργηθεί αυτόματα και θα επισημανθεί.

Ο δεύτερος τρόπος είναι να οριστεί κάθε hashtag χειροκίνητα. Αυτό καλύπτεται σε αυτό το έγγραφο.

#### Περίπτωσεις Χρήσης

HashTags είναι χρήσιμα για τα μέλη της κοινότητας για να πλοηγηθούν και να βρουν σχετικές συζητήσεις. Είναι επίσης χρήσιμα για την επισήμανση σχετικών
κομματιών πληροφοριών, ως συνδέσμους, οι οποίοι μπορούν να οριστούν μέσω του API μας. Για παράδειγμα, τα hashtags θα μπορούσαν να χρησιμοποιηθούν για να επισημάνουν:

- Εισιτήρια σε σύστημα διαχείρισης αιτημάτων.
- Έγγραφα ή Σελίδες σε CRM.
- Αρχεία.
