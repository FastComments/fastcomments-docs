A petición, se pueden usar símbolos distintos de `#`.

Se pueden usar diferentes símbolos de etiqueta para enlazar a distintas piezas de contenido. Por ejemplo, la API podría usarse para definir hashtags con el símbolo `#`
como elementos de tema, pero también definir entradas como `~document_one` y `~document_two`. En este escenario, los usuarios podrían usar hashtags así como etiquetas personalizadas
para señalar otros documentos.
