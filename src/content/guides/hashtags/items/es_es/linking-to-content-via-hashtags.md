Por defecto, al hacer clic en un hashtag se abre una vista con una lista de comentarios que contienen ese hashtag. Sin embargo, podemos cambiar esto para que enlace a un contenido externo para cada hashtag.

Esto se puede hacer al agregar hashtags [mediante la API](/guide-api.html#hash-tag-structure). Cuando agregamos un hashtag mediante la API, es posible definir una URL para cada hashtag.
