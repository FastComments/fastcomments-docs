Op verzoek kunnen symbolen anders dan `#` worden gebruikt.

Verschillende tagsymbolen kunnen worden gebruikt om naar verschillende inhoudsitems te linken. Bijvoorbeeld, de API zou gebruikt kunnen worden om hashtags met het `#`-symbool
als onderwerpitems te definiëren, maar ook items zoals `~document_one` en `~document_two` te definiëren. In dit scenario kunnen gebruikers zowel hashtags als aangepaste tags
gebruiken om naar andere documenten te verwijzen.
