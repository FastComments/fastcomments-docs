Par défaut, cliquer sur un hashtag ouvre une vue avec une liste de commentaires contenant le hashtag. Toutefois, nous pouvons modifier cela pour que chaque hashtag pointe vers un
contenu externe.

Cela peut être fait lors de l'ajout de hashtags [via l'API](/guide-api.html#hash-tag-structure). Lorsqu'on ajoute un hashtag via l'API, il est possible de définir une URL pour chaque tag.
