По умолчанию при нажатии на хештег открывается представление со списком комментариев с этим хештегом. Однако мы можем изменить это, чтобы ссылка вела на
внешний ресурс для каждого хештега.

Это можно сделать при добавлении хештегов [через API](/guide-api.html#hash-tag-structure). При добавлении хештега через API возможно задать URL для каждого тега.
