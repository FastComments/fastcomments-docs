По запросу можно использовать символы, отличные от `#`.

Для связи с разными фрагментами содержимого можно использовать различные символы тегов. Например, API можно использовать для определения хэштегов с символом `#`
в качестве элементов тем, но также определять записи, такие как `~document_one` и `~document_two`. В этом сценарии пользователи могут использовать как хэштеги, так и пользовательские теги
для ссылок на другие документы.
