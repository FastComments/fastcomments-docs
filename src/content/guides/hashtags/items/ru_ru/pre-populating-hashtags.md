По умолчанию FastComments позволяет вашим пользователям создавать хэштеги, формируя список хэштегов и добавляя их в автозаполнение со временем.

Заблокированные пользователи не могут добавлять новые хэштеги, даже если автоматическое создание хэштегов включено.

Однако возможно предварительно задать набор хэштегов [через API](/guide-api.html#hash-tag-structure). Обратите внимание, что при добавлении большого числа хэштегов доступно API для массового добавления.

Предварительно заполненные и автоматически создаваемые хэштеги формируют единый набор данных и оба в итоге будут присутствовать в автозаполнении хэштегов.
