#### Заполнение автодополнения

Хэштеги можно заполнить двумя способами.

First, by default users can type the `#` symbol and some text - and if there are existing hashtags an autocomplete UI will be shown.

Если пользователь вводит новый хэштег, [если не отключено](/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation), хэштег
будет автоматически создан и выделен.

The second way is to manually define each hashtag. This is covered in this document.

#### Сценарии использования

Хэштеги полезны участникам сообщества для навигации и поиска релевантных обсуждений. Они также полезны для пометки соответствующих
фрагментов информации в виде ссылок, которые можно задавать через наш API. Например, хэштеги можно использовать для пометки:

- Тикеты в системе тикетов.
- Документы или страницы в CRM.
- Файлы.
