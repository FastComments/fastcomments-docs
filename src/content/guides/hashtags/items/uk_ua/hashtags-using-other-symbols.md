За запитом можна використовувати інші символи, окрім `#`.

Різні символи тегів можуть використовуватися для посилання на різні частини вмісту. Наприклад, API можна використовувати для визначення хештегів із символом `#`
як елементів тем, але також визначати записи, такі як `~document_one` і `~document_two`. У цьому сценарії користувачі могли б використовувати хештеги, а також користувацькі теги
щоб вказувати на інші документи.
