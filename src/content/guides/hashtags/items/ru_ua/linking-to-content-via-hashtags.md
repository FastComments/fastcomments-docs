По умолчанию при нажатии на хештег открывается представление со списком комментариев с этим хештегом. Однако мы можем изменить это так, чтобы он ссылался на
внешний материал для каждого хештега.

Это можно сделать при добавлении хештегов [через API](/guide-api.html#hash-tag-structure). Когда мы добавляем хештег через API, возможно
задать URL для каждого тега.
