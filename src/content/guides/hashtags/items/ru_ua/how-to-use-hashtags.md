#### Заполнение автозаполнения

Хэштеги можно заполнить двумя способами.

Во-первых, по умолчанию пользователи могут ввести символ `#` и некоторый текст - и если существуют хэштеги, будет показан интерфейс автозаполнения.

Если пользователь вводит новый хэштег, [если не отключено](/guide-customizations-and-configuration.html#disable-automatic-hashtag-creation), хэштег
будет автоматически создан и выделен.

Второй способ — вручную определить каждый хэштег. Это описано в данном документе.

#### Сценарии использования

Хэштеги полезны для участников сообщества для навигации и поиска релевантных обсуждений. Они также полезны для пометки соответствующих
фрагментов информации в виде ссылок, которые можно определить через наш API. Например, хэштеги можно использовать для пометки:

- Тикетов в системе тикетов.
- Документов или страниц в CRM.
- Файлов.
