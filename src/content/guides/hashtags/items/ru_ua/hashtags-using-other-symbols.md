По запросу можно использовать символы, отличные от `#`.

Разные символы тегов можно использовать для ссылки на разные части контента. Например, API может использоваться для определения хештегов с помощью символа `#`
в качестве элементов темы, но также определять записи, такие как `~document_one` и `~document_two`. В этом сценарии пользователи могут использовать хештеги, а также пользовательские теги
чтобы ссылаться на другие документы.
