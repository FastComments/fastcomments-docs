Na życzenie można używać symboli innych niż `#`.

Do łączenia z różnymi fragmentami treści można stosować różne symbole tagów. Na przykład API można użyć do zdefiniowania hashtagów za pomocą symbolu `#`
jako elementy tematyczne, ale także zdefiniować wpisy takie jak `~document_one` i `~document_two`. W takim przypadku użytkownicy mogliby używać hashtagów, jak również niestandardowych tagów
aby wskazywać na inne dokumenty.
