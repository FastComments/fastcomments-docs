Gdy hashtagi są dodawane przez API lub gdy użytkownicy wprowadzają nowe hashtagi, są one zapisywane w centralnym magazynie danych.

Po wpisaniu słowa wyzwalającego (`#` jest domyślne) i znaku, system zacznie wyszukiwać odpowiednie hashtagi
które zaczynają się od tego samego tekstu i wyświetli je jako sugestie. Wyniki są sortowane alfabetycznie.

Podobnie w przypadku `@mentions`, można użyć klawiatury, aby wybrać pozycję z menu autouzupełniania.
