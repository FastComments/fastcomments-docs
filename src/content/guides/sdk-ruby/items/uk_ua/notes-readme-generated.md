### Ідентифікатори трансляцій

Ви побачите, що у деяких викликах API потрібно передати `broadcastId`. Коли ви отримуєте події, цей ідентифікатор повертається, тож ви знатимете ігнорувати подію, якщо плануєте оптимістично застосовувати зміни на клієнті
(що, ймовірно, ви й захочете зробити, оскільки це забезпечує найкращий досвід). Передайте тут UUID. Ідентифікатор повинен бути достатньо унікальним, щоб не повторюватися двічі протягом сесії браузера.
