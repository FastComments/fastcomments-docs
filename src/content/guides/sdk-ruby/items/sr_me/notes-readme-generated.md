### ИД-ови емитовања

Видећете да треба да проследите `broadcastId` у неким API позивима. Када примите догађаје, добићете овај ID назад, тако да ћете знати да игноришете догађај ако планирате да оптимистички примените измене на клијенту
(што ћете вероватно желети да урадите јер пружа најбоље корисничко искуство). Овде проследите UUID. ID треба да буде довољно јединствен да се не појави два пута током једне сесије прегледача.
