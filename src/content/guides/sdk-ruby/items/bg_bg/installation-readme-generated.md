Добавете този ред във вашия Gemfile на приложението:

```ruby
gem 'fastcomments'
```

След това изпълнете:

```bash
bundle install
```

Или го инсталирайте сами така:

```bash
gem install fastcomments
```

### Library Contents

Тази библиотека съдържа генерирания API клиент и SSO помощни средства, които улесняват работата с API-то.

- [API Client Library Docs](https://github.com/FastComments/fastcomments-ruby/blob/master/client/README.md)

### Public vs Secured APIs

За API клиента има три класа, `DefaultApi`, `PublicApi` и `ModerationApi`. Класът `DefaultApi` съдържа методи, които изискват вашия API ключ, а `PublicApi` съдържа API повиквания, които могат да се извършват директно от браузър/мобилно устройство/и т.н. без автентикация. `ModerationApi` съдържа методите, които задвижват таблото на модератора.

`ModerationApi` обхваща модериране на коментари (изброяване, брой, търсене, логове, експортиране), действия по модериране (премахване/възстановяване, маркиране, задаване на статус преглед/спам/одобрение, гласове, повторно отваряне/затваряне на нишка),
забрани (бан от коментар, отмяна, предварителни обобщения за бан, статус/настройки на бана, брой забранени потребители), и значки и доверие (присъждане/премахване на значка, ръчни значки, получаване/задаване на коефициент на доверие, вътрешен профил на потребителя).
Всеки метод на `ModerationApi` приема параметър `sso`, така че заявката да може да бъде направена от името на модератор, автентикиран чрез SSO.
