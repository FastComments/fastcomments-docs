### Αναγνωριστικά Broadcast

Θα δείτε ότι πρέπει να περάσετε ένα `broadcastId` σε ορισμένες κλήσεις API. Όταν λαμβάνετε συμβάντα, θα λάβετε αυτό το ID πίσω, οπότε θα γνωρίζετε να αγνοήσετε το συμβάν αν σκοπεύετε να εφαρμόσετε αλλαγές "optimistically" στον πελάτη (κάτι που πιθανότατα θα θέλετε να κάνετε, καθώς προσφέρει την καλύτερη εμπειρία). Περάστε ένα UUID εδώ. Το ID θα πρέπει να είναι αρκετά μοναδικό ώστε να μην εμφανιστεί δύο φορές σε μια συνεδρία προγράμματος περιήγησης.
