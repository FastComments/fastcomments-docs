Moodle 4.0+ prend en charge l'enregistrement dynamique LTI 1.3 via le plugin External Tool.

#### Ouvrir l'écran de gestion des outils

1. Connectez-vous à Moodle en tant qu'administrateur du site.
2. Accédez à **Site administration** > **Plugins** > **Activity modules** > **External tool** > **Manage tools**.

#### Coller l'URL

Vous verrez une carte intitulée **Tool URL**. Collez l'URL d'enregistrement de FastComments dans le champ de texte et cliquez sur **Add LTI Advantage**.

Moodle ouvre un écran d'enregistrement affichant l'identité de l'outil et les autorisations demandées. Vérifiez-les puis cliquez sur **Activate** (ou **Register**, selon la version de Moodle).

La fenêtre contextuelle se ferme lorsque l'enregistrement est terminé ; le nouvel outil FastComments apparaît dans la liste **Tools** avec le statut **Active**.

#### Rendre disponible

Par défaut, Moodle ajoute les nouveaux outils à la liste "Course tools" mais ne les affiche pas dans le sélecteur d'activités. Pour rendre FastComments disponible dans tout le cours :

1. Cliquez sur l'icône d'engrenage sur la vignette FastComments.
2. Sous **Tool configuration usage**, choisissez **Show in activity chooser and as a preconfigured tool**.
3. Enregistrer.

Les enseignants peuvent désormais ajouter FastComments à n'importe quel cours via **Add an activity or resource** > **FastComments**.
