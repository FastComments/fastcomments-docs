Moodle 4.0+ prend en charge l'enregistrement dynamique LTI 1.3 via le plugin External Tool.

#### Open the Tool Management Screen

1. Connectez-vous à Moodle en tant qu'administrateur du site.
2. Naviguez vers **Administration du site** > **Extensions** > **Modules d'activité** > **Outil externe** > **Gérer les outils**.

#### Paste the URL

Vous verrez une carte intitulée **Tool URL**. Collez l'URL d'enregistrement FastComments dans le champ de texte et cliquez sur **Add LTI Advantage**.

Moodle ouvre un écran d'enregistrement affichant l'identité de l'outil et les autorisations demandées. Vérifiez-les et cliquez sur **Activate** (ou **Register**, selon la version de Moodle).

La fenêtre contextuelle se ferme à la fin de l'enregistrement ; le nouvel outil FastComments apparaît dans la liste **Tools** avec le statut **Active**.

#### Make It Available

Par défaut, Moodle ajoute les nouveaux outils à la liste « Course tools » mais ne les affiche pas dans le sélecteur d'activités. Pour rendre FastComments accessible dans tout le cours :

1. Cliquez sur l'icône d'engrenage sur la tuile FastComments.
2. Sous **Tool configuration usage**, choisissez **Show in activity chooser and as a preconfigured tool**.
3. Enregistrez.

Les enseignants peuvent maintenant ajouter FastComments à n'importe quel cours via **Add an activity or resource** > **FastComments**.
