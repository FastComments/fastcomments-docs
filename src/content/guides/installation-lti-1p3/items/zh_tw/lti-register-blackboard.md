Blackboard Learn SaaS 與 Ultra 支援 LTI 1.3 動態註冊。

#### 開啟工具提供者畫面

1. 以系統管理員身份登入 Blackboard。
2. 前往 **管理員面板** > **整合** > **LTI 工具提供者**。
3. 點選 **註冊 LTI 1.3 / LTI Advantage 工具**。

若您只看到「Register LTI 1.1 Provider」，表示您的 Blackboard 版本尚不支援 LTI 1.3 — 請升級或聯絡 Blackboard 支援。

#### 貼上 URL

將 FastComments 註冊 URL 貼到 **Client ID** / **Registration URL** 欄位（Blackboard 的標示依版本而異）。提交。

Blackboard 會與 FastComments 執行註冊握手，並顯示確認畫面。

#### 核准並啟用

Blackboard 預設會將新註冊的工具標示為 **已核准但排除**：

1. 在工具提供者清單中找到 FastComments 項目。
2. 打開選單並選擇 **Edit**。
3. 將 **Tool Status** 設為 **Approved**。
4. 在 **Institution Policies** 下，檢查會傳送的使用者資料（姓名、電子郵件、角色）。儲存。

當講師在課程中新增內容時，即可使用該工具。
