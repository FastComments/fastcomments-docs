FastComments LTI 1.3 已針對下列平台進行測試：

- **D2L Brightspace** (任何版本支援 LTI Advantage)
- **Moodle 4.0+** (External Tool plugin, LTI Advantage)
- **Blackboard Learn** (SaaS and Ultra)
- **Sakai** (支援 LTI Advantage 的版本)
- **Schoology** (Enterprise, Dynamic Registration 在可用時)

任何其他實作 IMS LTI 1.3 Advantage 規範並支援 Dynamic Registration 的 LMS 也應該可用 - 在產生您的註冊 URL 時，從平台下拉選單選擇 **Other LTI 1.3 platform**。

Canvas LMS 使用不同的設定流程（使用 per-developer-key configuration JSON 而非 Dynamic Registration）。請參閱專門的 Canvas LMS 安裝指南。
