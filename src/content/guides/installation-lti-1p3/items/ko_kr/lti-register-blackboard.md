Blackboard Learn SaaS 및 Ultra는 LTI 1.3 Dynamic Registration을 지원합니다.

#### 도구 제공자 화면 열기

1. 시스템 관리자 계정으로 Blackboard에 로그인합니다.
2. **Administrator Panel** > **Integrations** > **LTI Tool Providers**로 이동합니다.
3. **Register LTI 1.3 / LTI Advantage Tool**를 클릭합니다.

If you only see "Register LTI 1.1 Provider", your Blackboard version doesn't support LTI 1.3 yet - upgrade or contact Blackboard support.

#### URL 붙여넣기

FastComments 등록 URL을 **Client ID** / **Registration URL** 필드에 붙여넣고 제출합니다(Blackboard의 표기는 버전마다 다릅니다).

Blackboard가 FastComments와 등록 핸드셰이크를 수행하고 확인 화면을 표시합니다.

#### 승인 및 활성화

Blackboard는 새로 등록된 도구를 기본적으로 **Approved but excluded** 상태로 표시합니다:

1. 도구 제공자 목록에서 FastComments 항목을 찾습니다.
2. 메뉴를 열고 **Edit**를 선택합니다.
3. **Tool Status**를 **Approved**로 설정합니다.
4. **Institution Policies**에서 전송되는 사용자 데이터(이름, 이메일, 역할)를 검토합니다. 저장합니다.

이제 강사가 코스에 콘텐츠를 추가할 때 해당 도구를 사용할 수 있습니다.
