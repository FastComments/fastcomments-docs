1. FastComments에 로그인하고 <a href="https://fastcomments.com/auth/my-account/lti-config" target="_blank">귀하의 LTI 1.3 구성 페이지</a>로 이동합니다.
2. (선택 사항) 연결 중인 플랫폼을 **플랫폼** 드롭다운에서 선택하세요 - 표시 레이블을 설정하지만, 자동 감지도 잘 작동합니다.
3. **URL 생성**을 클릭하세요. 일회용 **등록 URL**이 표시됩니다 (30분 동안 유효, 한 번만 사용 가능).
4. 귀하의 LMS에서 LTI 1.3 동적 등록 화면을 열고 URL을 **도구 시작 등록 엔드포인트** (또는 해당) 필드에 붙여넣습니다. 제출하세요.
5. 귀하의 LMS가 FastComments에 콜백을 보내 키를 교환하고 통합을 생성합니다. 완료되면 팝업이 자동으로 닫힙니다.
6. FastComments로 돌아가면 새 구성이 **기존 구성** 표에 나타납니다. 이 도구를 이제 LMS 코스 내에서 사용할 수 있습니다.
