FastComments LTI 1.3 は次のプラットフォームでテストされています：

- **D2L Brightspace** (LTI Advantage を備えた任意のバージョン)
- **Moodle 4.0+** (External Tool プラグイン、LTI Advantage)
- **Blackboard Learn** (SaaS および Ultra)
- **Sakai** (LTI Advantage をサポートするバージョン)
- **Schoology** (Enterprise、利用可能な場合は Dynamic Registration)

IMS LTI 1.3 Advantage 仕様を Dynamic Registration で実装している他の LMS でも動作するはずです - 登録用 URL を生成するときはプラットフォームのドロップダウンから **Other LTI 1.3 platform** を選択してください。

Canvas LMS は別のセットアップフローを使用します（Dynamic Registration の代わりに per-developer-key configuration JSON を使用します）。専用の Canvas LMS インストールガイドを参照してください。
