FastComments LTI 1.3 已在以下平台上进行过测试：

- **D2L Brightspace**（任何包含 LTI Advantage 的版本）
- **Moodle 4.0+**（External Tool 插件，LTI Advantage）
- **Blackboard Learn**（SaaS 和 Ultra）
- **Sakai**（支持 LTI Advantage 的版本）
- **Schoology**（Enterprise，在可用时支持 Dynamic Registration）

任何其他实现 IMS LTI 1.3 Advantage 规范并支持 Dynamic Registration 的 LMS 也应该可以工作 — 在生成您的 registration URL 时，请从平台下拉菜单中选择 **Other LTI 1.3 platform**。

Canvas LMS 使用不同的设置流程（使用 per-developer-key 配置 JSON 而非 Dynamic Registration）。请参阅专门的 Canvas LMS 安装指南。
