FastComments LTI 1.3 wurde getestet gegen:

- **D2L Brightspace** (jede Version mit LTI Advantage)
- **Moodle 4.0+** (External Tool-Plugin, LTI Advantage)
- **Blackboard Learn** (SaaS und Ultra)
- **Sakai** (Versionen mit LTI Advantage-Unterstützung)
- **Schoology** (Enterprise, Dynamic Registration, falls verfügbar)

Any other LMS that implements the IMS LTI 1.3 Advantage spec with Dynamic Registration should also work - select **Other LTI 1.3 platform** from the platform dropdown when generating your registration URL.

Canvas LMS verwendet einen anderen Einrichtungsablauf (per-developer-key configuration JSON anstelle von Dynamic Registration). Siehe den dedizierten Canvas LMS Installationsleitfaden.
