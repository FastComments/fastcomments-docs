По умолчанию FastComments требует адрес электронной почты для комментирования. Он не обязательно должен быть действительным, однако до тех пор, пока пользователь не перейдёт по ссылке, отправленной ему,
его комментарий будет отображаться с меткой "Непроверенный комментарий".

Однако мы можем убрать требование электронной почты. Поле ввода электронной почты по-прежнему будет отображаться, но больше не будет обязательным.

Это можно настроить через интерфейс настройки виджета:
