По умолчанию FastComments разрешает загрузку изображений. Это можно отключить, установив флаг noImageUploads в значение true.

[code-example-start config = {noImageUploads: true}; linesToHighlight = [6]; title = 'Отключение загрузки изображений'; code-example-end]

Это можно настроить без кода, на странице настройки виджета:

[app-screenshot-start url='/auth/my-account/customize-widget/new'; clickSelector = '.disable-image-uploads'; selector = '.disable-image-uploads'; title='Отключение загрузки изображений' app-screenshot-end]
