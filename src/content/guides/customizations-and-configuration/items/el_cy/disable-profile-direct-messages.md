Ωστόσο, μπορούμε να απενεργοποιήσουμε αυτήν την καρτέλα:

[code-example-start config = {disableProfileDirectMessages: true}; linesToHighlight = [6]; title = 'Disable Profile Direct Messages'; code-example-end]

Αυτό μπορεί επίσης να γίνει χωρίς κώδικα. Στη σελίδα προσαρμογής του widget, δείτε την ενότητα "Απενεργοποίηση Άμεσων Μηνυμάτων".
