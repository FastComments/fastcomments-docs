За замовчуванням FastComments відображає посилання так: [https://exmaple.com](https://exmaple.com) - де URL посилання стає клікабельним
HTML-тегом.

Деякі сайти можуть захотіти вимкнути це, наприклад, щоб відлякати шахраїв. Ми надаємо цю можливість, встановивши параметр `Comment HTML Rendering Option` на `Links as Text`.

Це можна налаштувати без коду, на сторінці налаштування віджета, для цілого домену або сторінки:

[app-screenshot-start url='/auth/my-account/customize-widget/new'; clickSelectors = ['.show-advanced-option']; selector = '.comment-html-rendering-mode'; title='Render Links as Text' app-screenshot-end]
