Однак цю вкладку можна вимкнути:

[code-example-start config = {disableProfileComments: true}; linesToHighlight = [6]; title = 'Disable Profile Comments'; code-example-end]

Це також можна зробити без коду. На сторінці налаштування віджета див. розділ "Disable Profile Comments".
