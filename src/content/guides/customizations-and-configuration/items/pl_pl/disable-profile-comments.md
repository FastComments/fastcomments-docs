[related-parameter-start name = 'disableProfileComments'; type = 'boolean'; related-parameter-end]

Domyślnie FastComments wyświetla kartę "Profile Comments" na profilach użytkowników, umożliwiając odwiedzającym zamieszczanie komentarzy na czyimś profilu.

Można jednak wyłączyć tę kartę:

[code-example-start config = {disableProfileComments: true}; linesToHighlight = [6]; title = 'Disable Profile Comments'; code-example-end]

Można to również zrobić bez kodu. Na stronie dostosowywania widżetu zobacz sekcję "Wyłącz komentarze profilu".
