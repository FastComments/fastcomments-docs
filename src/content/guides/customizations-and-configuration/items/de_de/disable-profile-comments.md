[related-parameter-start name = 'disableProfileComments'; type = 'boolean'; related-parameter-end]

Standardmäßig zeigt FastComments im Benutzerprofil eine Registerkarte "Profilkommentare" an, die Besuchern erlaubt, Kommentare auf dem Profil einer Person zu hinterlassen.

Diese Registerkarte kann jedoch deaktiviert werden:

[code-example-start config = {disableProfileComments: true}; linesToHighlight = [6]; title = 'Disable Profile Comments'; code-example-end]

Das kann auch ohne Code gemacht werden. Auf der Seite zur Anpassung des Widgets finden Sie den Abschnitt "Profilkommentare deaktivieren".
