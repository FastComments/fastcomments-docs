[related-parameter-start name = 'disableToolbar'; type = 'boolean'; related-parameter-end]

Standardmäßig zeigt FastComments beim Schreiben eines Kommentars eine Symbolleiste an, die Schnellzugriffe zum Formatieren von Text und zum Hochladen
von Bildern bietet.

Diese Symbolleiste kann im Code oder über die Anpassungs-UI deaktiviert werden.

[code-example-start config = {disableToolbar: true}; linesToHighlight = [6]; title = 'Disabling The Toolbar'; code-example-end]

Dies kann auch ohne Code erfolgen. Auf der Seite zur Widget-Anpassung finden Sie die Option "Antwort-Symbolleiste deaktivieren".
