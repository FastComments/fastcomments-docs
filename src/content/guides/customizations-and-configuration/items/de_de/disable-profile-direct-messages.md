[related-parameter-start name = 'disableProfileDirectMessages'; type = 'boolean'; related-parameter-end]

Standardmäßig zeigt FastComments auf Benutzerprofilen einen Tab "Direktnachrichten" an, der Besuchern erlaubt, einem Benutzer Direktnachrichten zu senden.

Wir können diesen Tab jedoch deaktivieren:

[code-example-start config = {disableProfileDirectMessages: true}; linesToHighlight = [6]; title = 'Disable Profile Direct Messages'; code-example-end]

Das lässt sich auch ohne Code erledigen. Auf der Seite zur Anpassung des Widgets finden Sie den Abschnitt "Direktnachrichten deaktivieren".
