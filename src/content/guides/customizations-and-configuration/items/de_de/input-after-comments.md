Standardmäßig befindet sich der Kommentar-Eingabebereich **vor** dem Kommentar-Thread. Allerdings können wir durch Setzen dieses Konfigurationsparameters
auf true ihn **nach** verschieben.

[code-example-start config = {inputAfterComments: true}; linesToHighlight = [6]; title = 'Moving The Reply Box to The Bottom'; code-example-end]

Dies kann ohne Code auf der Seite zur Widget-Anpassung angepasst werden:
