Από προεπιλογή, το FastComments θα εμφανίζει μια καρτέλα "Σχόλια Προφίλ" στα προφίλ χρηστών, επιτρέποντας στους επισκέπτες να αφήνουν σχόλια στο προφίλ κάποιου.

Ωστόσο, μπορούμε να απενεργοποιήσουμε αυτή την καρτέλα:

[code-example-start config = {disableProfileComments: true}; linesToHighlight = [6]; title = 'Disable Profile Comments'; code-example-end]

Αυτό μπορεί επίσης να γίνει χωρίς κώδικα. Στη σελίδα προσαρμογής του widget, δείτε την ενότητα "Απενεργοποίηση Σχολίων Προφίλ".
