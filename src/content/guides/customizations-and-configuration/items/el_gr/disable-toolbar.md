Από προεπιλογή, το FastComments θα εμφανίζει μια γραμμή εργαλείων όταν γράφετε ένα σχόλιο για να παρέχει συντομεύσεις για τη διακόσμηση κειμένου και τη μεταφόρτωση
εικόνων.

Αυτή η γραμμή εργαλείων μπορεί να απενεργοποιηθεί στον κώδικα ή μέσω της διεπαφής προσαρμογής.

[code-example-start config = {disableToolbar: true}; linesToHighlight = [6]; title = 'Disabling The Toolbar'; code-example-end]

Αυτό μπορεί επίσης να γίνει χωρίς κώδικα. Στη σελίδα προσαρμογής του widget, δείτε την επιλογή "Απενεργοποίηση της γραμμής εργαλείων απάντησης".
