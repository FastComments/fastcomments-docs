[related-parameter-start name = 'disableProfileComments'; type = 'boolean'; related-parameter-end]

Per impostazione predefinita, FastComments mostrerà una scheda "Commenti sul profilo" nei profili utente, consentendo ai visitatori di lasciare commenti sul profilo di qualcuno.

Tuttavia, possiamo disattivare questa scheda:

[code-example-start config = {disableProfileComments: true}; linesToHighlight = [6]; title = 'Disable Profile Comments'; code-example-end]

Questo può anche essere fatto senza codice. Nella pagina di personalizzazione del widget, consulta la sezione "Disabilita Commenti sul Profilo".
