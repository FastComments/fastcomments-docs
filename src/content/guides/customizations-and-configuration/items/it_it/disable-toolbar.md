[related-parameter-start name = 'disableToolbar'; type = 'boolean'; related-parameter-end]

Per impostazione predefinita, FastComments mostra una barra degli strumenti quando si scrive un commento per fornire scorciatoie per decorare il testo e caricare
immagini.

Questa barra degli strumenti può essere disabilitata via codice o tramite l'interfaccia di personalizzazione.

[code-example-start config = {disableToolbar: true}; linesToHighlight = [6]; title = 'Disabling The Toolbar'; code-example-end]

Questo può essere fatto anche senza codice. Nella pagina di personalizzazione del widget, consulta l'opzione "Disabilita la barra delle risposte".
