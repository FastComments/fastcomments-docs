當使用者輸入標籤，形式為 `#someexampletag`，FastComments 會自動建立該 `HashTag` 並  
在他們的留言中將其突顯。

在某些情況下，可能希望停用此功能，或透過 API 控制可使用的標籤。

要做到這點，只需在 Widget 自訂介面中啟用 `Disable Automatic #hashtag Creation`。
