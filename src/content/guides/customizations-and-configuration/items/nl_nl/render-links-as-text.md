Standaard rendert FastComments links als volgt: [https://exmaple.com](https://exmaple.com) - waarbij de link-URL een klikbare
HTML anchor-tag wordt.

Sommige sites willen dit mogelijk uitschakelen, bijvoorbeeld om oplichters af te schrikken. Dit kan worden ingesteld door de `Comment HTML Rendering Option` op `Links as Text` te zetten.

Dit kan zonder code worden aangepast, op de pagina voor het aanpassen van de widget, voor een heel domein of een pagina:

[app-screenshot-start url='/auth/my-account/customize-widget/new'; clickSelectors = ['.show-advanced-option']; selector = '.comment-html-rendering-mode'; title='Render Links as Text' app-screenshot-end]
