Standaard toont FastComments een tabblad "Profielreacties" op gebruikersprofielen, waarmee bezoekers opmerkingen op iemands profiel kunnen achterlaten.

We kunnen dit tabblad echter uitschakelen:

[code-example-start config = {disableProfileComments: true}; linesToHighlight = [6]; title = 'Disable Profile Comments'; code-example-end]

Dit kan ook zonder code worden gedaan. Op de pagina voor het aanpassen van de widget ziet u de sectie "Profielreacties uitschakelen".
