Standaard toont FastComments een "Direct Messages" tabblad op gebruikersprofielen, zodat bezoekers directe berichten naar een gebruiker kunnen sturen.

We kunnen dit tabblad echter uitschakelen:

[code-example-start config = {disableProfileDirectMessages: true}; linesToHighlight = [6]; title = 'Disable Profile Direct Messages'; code-example-end]

Dit kan ook zonder code worden gedaan. Op de pagina voor het aanpassen van de widget, zie de sectie "Directe berichten uitschakelen".
