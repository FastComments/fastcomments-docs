Par défaut, FastComments affichera les liens comme ceci : [https://exmaple.com](https://exmaple.com) - où l'URL du lien devient cliquable
balise d'ancre HTML.

Certains sites peuvent vouloir désactiver cela, par exemple pour décourager les escrocs. Nous fournissons cela en réglant l'option `Comment HTML Rendering Option` sur `Links as Text`.

Cela peut être personnalisé sans code, sur la page de personnalisation du widget, pour un domaine entier, ou une page :

[app-screenshot-start url='/auth/my-account/customize-widget/new'; clickSelectors = ['.show-advanced-option']; selector = '.comment-html-rendering-mode'; title='Render Links as Text' app-screenshot-end]
