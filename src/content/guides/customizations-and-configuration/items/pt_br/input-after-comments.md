[related-parameter-start name = 'inputAfterComments'; type = 'boolean'; related-parameter-end]

Por padrão a área de entrada de comentários fica **antes** do encadeamento de comentários. No entanto, ao definir este parâmetro de configuração
para true podemos movê-la para **depois**.

[code-example-start config = {inputAfterComments: true}; linesToHighlight = [6]; title = 'Moving The Reply Box to The Bottom'; code-example-end]

Isso pode ser personalizado sem código, na página de personalização do widget:
