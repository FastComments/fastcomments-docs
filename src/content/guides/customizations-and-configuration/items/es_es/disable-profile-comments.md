[related-parameter-start name = 'disableProfileComments'; type = 'boolean'; related-parameter-end]

Por defecto, FastComments mostrará una pestaña "Comentarios de perfil" en los perfiles de usuario, permitiendo que los visitantes dejen comentarios en el perfil de alguien.

Sin embargo, podemos desactivar esta pestaña:

[code-example-start config = {disableProfileComments: true}; linesToHighlight = [6]; title = 'Disable Profile Comments'; code-example-end]

Esto también se puede hacer sin código. En la página de personalización del widget, consulte la sección "Desactivar comentarios de perfil".
