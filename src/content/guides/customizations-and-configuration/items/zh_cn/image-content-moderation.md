[related-parameter-start name = 'imageContentProfanityLevel'; type = 'string'; related-parameter-end]

默认情况下，FastComments 评论小部件的图像内容审核设置为 `off`。

可用选项包括 `off`、`low`、`medium` 和 `high`。

在小部件自定义的 UI 中，只要未勾选 `Disable Image Uploads?`，即可在 `Image Content Moderation Level` 下找到此设置。
