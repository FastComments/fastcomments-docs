默认情况下，FastComments 会在用户资料上显示一个“Profile Comments”选项卡，允许访问者在某人的资料上发表评论。

但是，我们可以禁用此选项卡：

[code-example-start config = {disableProfileComments: true}; linesToHighlight = [6]; title = 'Disable Profile Comments'; code-example-end]

这也可以在无需编写代码的情况下完成。在小部件自定义页面中，参见“Disable Profile Comments”部分。
