Το πρόσθετο προσθέτει μια ενότητα FastComments στις προτιμήσεις προφίλ κάθε χρήστη στο Moodle. Οι χρήστες μπορούν να διαχειρίζονται τις δικές τους ρυθμίσεις ειδοποιήσεων χωρίς να επισκέπτονται τον ιστότοπο FastComments.

#### Διαθέσιμες Προτιμήσεις

- **Ειδοποιήσεις Απαντήσεων** - λαμβάνουν ειδοποιήσεις όταν κάποιος απαντά στα σχόλιά τους.
- **Ειδοποιήσεις Συνδρομής** - λαμβάνουν ειδοποιήσεις για νήματα στα οποία είναι συνδρομητές.

Οι χρήστες μπορούν να έχουν πρόσβαση σε αυτές τις ρυθμίσεις από το προφίλ τους στο Moodle, στην ενότητα FastComments.
