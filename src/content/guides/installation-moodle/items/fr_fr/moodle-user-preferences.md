---
Le plugin ajoute une section FastComments aux préférences de profil Moodle de chaque utilisateur. Les utilisateurs peuvent gérer leurs propres paramètres de notification sans visiter le site FastComments.

#### Préférences disponibles

- **Notifications de réponse** - recevoir des notifications lorsque quelqu'un répond à leurs commentaires.
- **Notifications d'abonnement** - recevoir des notifications pour les fils de discussion auxquels ils sont abonnés.

Les utilisateurs peuvent accéder à ces paramètres depuis leur profil Moodle, dans la section FastComments.
