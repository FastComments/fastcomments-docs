Le plugin prend en charge trois styles de commentaire. Vous pouvez configurer cela dans les paramètres du plugin.

#### Commentaires

Le widget FastComments standard apparaît sous le contenu de la page. Les utilisateurs peuvent publier des commentaires en fils de discussion, répondre, voter et utiliser toutes les fonctionnalités standard.

#### Collab Chat

Avec Collab Chat, les utilisateurs sélectionnent du texte sur la page pour démarrer des discussions inline. Une barre supérieure affiche les utilisateurs en ligne et le nombre de discussions actives.

#### Collab Chat + Commentaires

Les deux modes activés simultanément. Le widget de commentaires apparaît sous le contenu de la page, et les utilisateurs peuvent également sélectionner du texte pour démarrer des discussions inline.
