Страница настроек плагина находится в **Site Administration > Plugins > Local plugins > FastComments**. Доступные параметры:

#### ID арендатора

Ваш Tenant ID FastComments. Найдите это в <a href="https://fastcomments.com/auth/my-account" target="_blank">панели FastComments</a> в настройках вашей учётной записи.

#### Секрет API

Ваш API Secret ключ, необходимый для режима Secure SSO. Найдите это в <a href="https://fastcomments.com/auth/my-account/api-secret" target="_blank">Мой аккаунт > Секрет API</a>.

#### Режим SSO

Выберите, как будут аутентифицироваться пользователи. Подробности о каждом варианте см. в разделе [SSO Modes](#moodle-sso-modes).

- **Secure** (рекомендуется) — аутентификация на стороне сервера с подписью HMAC-SHA256
- **Simple** — данные пользователя на стороне клиента без подписи
- **None** — анонимные комментарии, без интеграции входа Moodle

#### Контекст страниц

Управляет тем, где появляются комментарии:

- **Страницы курса** — комментарии на главных страницах курса
- **Страницы модуля/активности** — комментарии к отдельным активностям и ресурсам
- **Оба** — комментарии на всех типах страниц

#### Стиль комментирования

Выберите опыт комментирования. Скриншоты каждого режима см. в разделе [Commenting Styles](#moodle-commenting-styles).

- **Comments** — стандартный древовидный виджет комментариев под содержимым страницы
- **Collab Chat** — обсуждения по выделению текста с индикаторами присутствия
- **Оба** — комментарии и Collab Chat активны одновременно

#### URL CDN

URL CDN FastComments. По умолчанию `https://cdn.fastcomments.com`. Измените это на URL CDN ЕС, если ваши данные размещены в регионе ЕС.
