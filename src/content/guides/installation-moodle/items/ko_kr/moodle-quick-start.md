1. 최신 릴리스 ZIP을 <a href="https://github.com/FastComments/fastcomments-moodle/" target="_blank">GitHub</a>에서 다운로드합니다.
2. `<moodle-root>/local/fastcomments`에 압축을 풉니다.
3. 사이트 관리자 계정으로 로그인한 후 **Site Administration > Notifications**로 이동합니다. Moodle이 플러그인을 감지하고 설치를 실행합니다.
4. 구성하려면 **Site Administration > Plugins > Local plugins > FastComments**로 이동합니다.
5. <a href="https://fastcomments.com/auth/my-account/api-secret" target="_blank">FastComments dashboard</a>에서 **Tenant ID**와 **API Secret**을 입력합니다.

그게 다입니다! 댓글은 코스 및 활동 페이지에 표시됩니다.
