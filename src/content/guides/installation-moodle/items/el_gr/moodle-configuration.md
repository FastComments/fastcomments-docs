Η σελίδα ρυθμίσεων του προσθέτου βρίσκεται στο **Διαχείριση ιστότοπου > Πρόσθετα > Τοπικά πρόσθετα > FastComments**. Οι διαθέσιμες επιλογές είναι:

#### Αναγνωριστικό Tenant

Το Tenant ID του FastComments σας. Βρείτε αυτό στο <a href="https://fastcomments.com/auth/my-account" target="_blank">ταμπλό του FastComments</a> στις ρυθμίσεις του λογαριασμού σας.

#### Μυστικό API

Το API Secret κλειδί σας, απαιτούμενο για τη λειτουργία Secure SSO. Βρείτε αυτό στο <a href="https://fastcomments.com/auth/my-account/api-secret" target="_blank">Ο λογαριασμός μου > API Secret</a>.

#### Λειτουργία SSO

Επιλέξτε πώς γίνονται οι πιστοποιήσεις των χρηστών. Δείτε την ενότητα [Λειτουργίες SSO](#moodle-sso-modes) για λεπτομέρειες σχετικά με κάθε επιλογή.

- **Secure** (συνιστάται) - υπογραφή ελέγχου ταυτότητας HMAC-SHA256 από την πλευρά του διακομιστή
- **Simple** - δεδομένα χρήστη από την πλευρά του πελάτη χωρίς υπογραφή
- **None** - ανώνυμος σχολιασμός, χωρίς ενσωμάτωση σύνδεσης Moodle

#### Πλαίσια σελίδας

Ελέγχει πού εμφανίζονται τα σχόλια:

- **Course pages** - σχόλια στις κύριες σελίδες των μαθημάτων
- **Module/activity pages** - σχόλια σε μεμονωμένες δραστηριότητες και πόρους
- **Both** - σχόλια σε όλους τους τύπους σελίδων

#### Στυλ σχολιασμού

Επιλέξτε την εμπειρία σχολιασμού. Δείτε τα [Στυλ σχολιασμού](#moodle-commenting-styles) για στιγμιότυπα οθόνης κάθε λειτουργίας.

- **Comments** - τυπικό νηματικό widget σχολίων κάτω από το περιεχόμενο της σελίδας
- **Collab Chat** - ενσωματωμένες συζητήσεις μέσω επιλογής κειμένου με δείκτες παρουσίας
- **Both** - ενεργά και τα σχόλια και το Collab Chat ταυτόχρονα

#### CDN URL

Το URL του CDN του FastComments. Προεπιλογή είναι `https://cdn.fastcomments.com`. Αλλάξτε αυτό στο URL του CDN για την ΕΕ αν τα δεδομένα σας φιλοξενούνται στην περιοχή της ΕΕ.
