Η σελίδα ρυθμίσεων του πρόσθετου βρίσκεται στο **Διαχείριση ιστότοπου > Πρόσθετα > Τοπικά πρόσθετα > FastComments**. Οι διαθέσιμες επιλογές είναι:

#### Tenant ID

Το Tenant ID του FastComments σας. Βρείτε το στον <a href="https://fastcomments.com/auth/my-account" target="_blank">πίνακα ελέγχου FastComments</a> κάτω από τις ρυθμίσεις του λογαριασμού σας.

#### API Secret

Το API Secret κλειδί σας, απαραίτητο για τη λειτουργία Secure SSO. Βρείτε το στο <a href="https://fastcomments.com/auth/my-account/api-secret" target="_blank">Ο λογαριασμός μου > API Secret</a>.

#### SSO Mode

Επιλέξτε πώς αυθεντικοποιούνται οι χρήστες. Δείτε την ενότητα [SSO Modes](#moodle-sso-modes) για λεπτομέρειες σχετικά με κάθε επιλογή.

- **Secure** (προτείνεται) - υπογεγραμμένη αυθεντικοποίηση HMAC-SHA256 στην πλευρά του διακομιστή
- **Simple** - δεδομένα χρήστη στην πλευρά του πελάτη χωρίς υπογραφή
- **None** - ανώνυμα σχόλια, χωρίς ενσωμάτωση σύνδεσης Moodle

#### Page Contexts

Ελέγχει πού εμφανίζονται τα σχόλια:

- **Course pages** - σχόλια στις κύριες σελίδες του μαθήματος
- **Module/activity pages** - σχόλια σε μεμονωμένες δραστηριότητες και πόρους
- **Both** - σχόλια σε όλους τους τύπους σελίδων

#### Commenting Style

Επιλέξτε την εμπειρία σχολιασμού. Δείτε τις [Commenting Styles](#moodle-commenting-styles) για στιγμιότυπα οθόνης κάθε λειτουργίας.

- **Comments** - τυπικό νήμα σχολίων (widget) κάτω από το περιεχόμενο της σελίδας
- **Collab Chat** - συζητήσεις επιλεγμένου κειμένου εντός της σελίδας με ενδείξεις παρουσίας
- **Both** - σχόλια και Collab Chat ενεργά ταυτόχρονα

#### CDN URL

Το CDN URL του FastComments. Από προεπιλογή είναι `https://cdn.fastcomments.com`. Αλλάξτε το σε URL CDN για την ΕΕ αν τα δεδομένα σας φιλοξενούνται στην περιοχή της ΕΕ.
