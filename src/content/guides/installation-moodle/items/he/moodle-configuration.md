דף הגדרות התוסף נמצא ב־**ניהול האתר > תוספים > תוספים מקומיים > FastComments**. האפשרויות הזמינות הן:

#### Tenant ID

קוד ה‑Tenant של FastComments שלך. מצא זאת ב־<a href="https://fastcomments.com/auth/my-account" target="_blank">לוח הבקרה של FastComments</a> תחת הגדרות החשבון שלך.

#### API Secret

מפתח ה‑API Secret שלך, נדרש למצב SSO מאובטח. מצא זאת ב־<a href="https://fastcomments.com/auth/my-account/api-secret" target="_blank">חשבון שלי > API Secret</a>.

#### SSO Mode

בחר כיצד המשתמשים מאומתים. ראה את הסעיף [מצבי SSO](#items-moodle-sso-modes) לפרטים על כל אפשרות.

- **מאובטח** (מומלץ) - אימות בצד השרת חתום ב‑HMAC-SHA256
- **פשוט** - נתוני משתמש בצד הלקוח ללא חתימה
- **ללא** - אפשרות תגובה אנונימית, ללא אינטגרציה עם כניסה ל‑Moodle

#### Page Contexts

שולט היכן תוצג מערכת התגובות:

- **דפי קורס** - תגובות בדפי הקורס הראשיים
- **דפי מודול/פעילות** - תגובות על פעילויות ומשאבים בודדים
- **שניהם** - תגובות בכל סוגי הדפים

#### Commenting Style

בחר את חוויית התגובות. ראה את [סגנונות תגובה](#items-moodle-commenting-styles) לצילומי מסך של כל מצב.

- **תגובות** - ווידג'ט תגובות מקוננות סטנדרטי מתחת לתוכן הדף
- **צ'אט שיתופי** - דיונים בתוך טקסט הנבחר עם מדדי נוכחות
- **שני האפשרויות** - תגובות וצ'אט שיתופי פעילים יחד

#### CDN URL

כתובת ה‑CDN של FastComments. כברירת מחדל `https://cdn.fastcomments.com`. החלף זאת לכתובת CDN של האיחוד האירופי אם הנתונים שלך מאוחסנים באזור האיחוד האירופי.
