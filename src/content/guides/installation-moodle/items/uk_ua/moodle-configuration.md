Сторінка налаштувань плагіна знаходиться в **Адміністрування сайту > Плагіни > Локальні плагіни > FastComments**. Доступні опції:

#### Ідентифікатор орендаря (Tenant ID)

Ваш Tenant ID FastComments. Знайдіть його в <a href="https://fastcomments.com/auth/my-account" target="_blank">панелі керування FastComments</a> в налаштуваннях облікового запису.

#### Секрет API (API Secret)

Ваш ключ API Secret, необхідний для захищеного режиму SSO. Знайдіть його в <a href="https://fastcomments.com/auth/my-account/api-secret" target="_blank">Мій обліковий запис > API Secret</a>.

#### Режим SSO

Виберіть, як проводиться автентифікація користувачів. Див. розділ [Режими SSO](#moodle-sso-modes) для подробиць щодо кожного варіанта.

- **Безпечний** (рекомендовано) - серверна автентифікація з підписом HMAC-SHA256
- **Простий** - дані користувача на боці клієнта без підпису
- **Немає** - анонімні коментарі, без інтеграції входу Moodle

#### Контексти сторінок

Керує тим, де з'являються коментарі:

- **Сторінки курсу** - коментарі на головних сторінках курсу
- **Сторінки модулів/активностей** - коментарі на окремих активностях та ресурсах
- **Обидва** - коментарі на всіх типах сторінок

#### Стиль коментування

Виберіть тип інтерфейсу коментування. Див. [Стилі коментування](#moodle-commenting-styles) для скріншотів кожного режиму.

- **Коментарі** - стандартний вкладений віджет коментарів під вмістом сторінки
- **Collab Chat** - inline-обговорення шляхом виділення тексту з індикаторами присутності
- **Обидва** - коментарі та Collab Chat активні одночасно

#### URL CDN

URL CDN FastComments. За замовчуванням `https://cdn.fastcomments.com`. Змініть це на EU CDN URL, якщо ваші дані розміщені в регіоні ЄС.
