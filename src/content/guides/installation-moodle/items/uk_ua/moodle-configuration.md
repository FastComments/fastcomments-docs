Сторінка налаштувань плагіна знаходиться в **Site Administration > Plugins > Local plugins > FastComments**. Доступні параметри:

#### Ідентифікатор орендаря (Tenant ID)

Ваш Tenant ID у FastComments. Знайдіть його на <a href="https://fastcomments.com/auth/my-account" target="_blank">панелі керування FastComments</a> в налаштуваннях облікового запису.

#### API Secret (секретний ключ API)

Ваш секретний ключ API, необхідний для режиму Secure SSO. Знайдіть його на <a href="https://fastcomments.com/auth/my-account/api-secret" target="_blank">My Account > API Secret</a>.

#### Режим SSO

Виберіть спосіб автентифікації користувачів. Деталі кожного варіанту див. у розділі [Режими SSO](#moodle-sso-modes).

- **Secure** (recommended) - серверна автентифікація з підписом HMAC-SHA256
- **Simple** - дані користувача на боці клієнта без підпису
- **None** - анонімні коментарі, інтеграція входу Moodle відключена

#### Контексти сторінок

Керує тим, де з’являються коментарі:

- **Course pages** - коментарі на головних сторінках курсу
- **Module/activity pages** - коментарі на окремих сторінках активностей і ресурсів
- **Both** - коментарі на всіх типах сторінок

#### Стиль коментування

Виберіть досвід коментування. Перегляньте скріншоти кожного режиму в розділі [Стилі коментування](#moodle-commenting-styles).

- **Comments** - стандартний віджет з ниткоподібними коментарями під вмістом сторінки
- **Collab Chat** - дискусії за виділенням тексту всередині сторінки з індикаторами присутності
- **Both** - одночасно активні коментарі та collab chat

#### CDN URL

URL CDN FastComments. За замовчуванням `https://cdn.fastcomments.com`. Змініть це на URL CDN для ЄС, якщо ваші дані розміщені в регіоні ЄС.
