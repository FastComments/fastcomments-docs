Le plugin prend en charge trois styles de commentaires. Vous pouvez configurer cela dans les paramètres du plugin.

#### Commentaires

Le widget FastComments standard apparaît sous le contenu de la page. Les utilisateurs peuvent publier des commentaires en fil de discussion, répondre, voter et utiliser toutes les fonctionnalités standard.

#### Collab Chat

Avec Collab Chat, les utilisateurs sélectionnent du texte sur la page pour lancer des discussions en ligne (inline). Une barre supérieure affiche les utilisateurs en ligne et le nombre de discussions actives.

</div>

#### Collab Chat + Commentaires

Les deux modes activés simultanément. Le widget de commentaires apparaît sous le contenu de la page, et les utilisateurs peuvent également sélectionner du texte pour lancer des discussions en ligne (inline).
