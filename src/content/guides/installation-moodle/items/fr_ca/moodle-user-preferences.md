Le plugin ajoute une section FastComments aux préférences du profil Moodle de chaque utilisateur. Les utilisateurs peuvent gérer leurs propres paramètres de notification sans visiter le site FastComments.

#### Préférences disponibles

- **Reply Notifications** - recevoir des notifications lorsqu'une personne répond à leurs commentaires.
- **Subscription Notifications** - recevoir des notifications pour les fils auxquels ils sont abonnés.

Les utilisateurs peuvent accéder à ces paramètres depuis leur profil Moodle dans la section FastComments.
