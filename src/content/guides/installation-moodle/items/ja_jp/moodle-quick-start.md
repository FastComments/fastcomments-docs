1. 最新のリリース ZIP を <a href="https://github.com/FastComments/fastcomments-moodle/" target="_blank">GitHub</a> からダウンロードします。
2. それを `<moodle-root>/local/fastcomments` に解凍します。
3. サイト管理者としてログインし、**サイト管理 > 通知** にアクセスします。Moodle がプラグインを検出してインストールを実行します。
4. 設定するには **サイト管理 > プラグイン > ローカルプラグイン > FastComments** に移動します。
5. <a href="https://fastcomments.com/auth/my-account/api-secret" target="_blank">FastComments ダッシュボード</a> の **Tenant ID** と **API Secret** を入力します。

以上です！コメントはコースやアクティビティのページに表示されます。
