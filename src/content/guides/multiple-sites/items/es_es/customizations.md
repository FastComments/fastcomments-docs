El widget de comentarios puede <a href="/guide-customizations-and-configuration.html#custom-css" target="_blank">personalizarse</a> de forma diferente para cada sitio.

En la página de personalización del widget, encontrarás un desplegable etiquetado "domain". Seleccionar `All` en este campo aplicará la regla a todos
los sitios de tu cuenta, puedes introducir un dominio específico que restringirá los ajustes o el CSS personalizado que desees aplicar solo a ese dominio.

Esto te permite que FastComments tenga un aspecto o funcione de forma completamente diferente para cada sitio.

[app-screenshot-start url='/auth/my-account/customize-widget/new'; selector = 'label.domain'; title='Customization Rule Domain' app-screenshot-end]

Un sitio podría usar texto en negrita y permitir comentarios anónimos, y otro sitio podría
tener un estilo diferente y exigir al usuario que deje su correo electrónico cuando comente.
