Vous pouvez utiliser le même compte FastComments avec plusieurs déploiements WordPress. Lorsque vous
configurez le plugin, un jeton d'intégration distinct est ajouté à chaque fois. [Vous pouvez voir vos intégrations actives ici.](https://fastcomments.com/auth/my-account/manage-data/integrations) 

Vous pouvez également utiliser le même compte FastComments avec une installation WordPress qui s'étend sur plusieurs domaines et
contient le même contenu sur ces domaines.
