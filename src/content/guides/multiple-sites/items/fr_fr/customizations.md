Le widget de commentaires peut être <a href="/guide-customizations-and-configuration.html#custom-css" target="_blank">personnalisé</a> différemment pour chaque site.

Sur la page de personnalisation du widget, vous trouverez un menu déroulant intitulé "domaine". La sélection de `All` dans ce champ appliquera la règle à tous
les sites de votre compte, vous pouvez entrer un domaine spécifique qui limitera les paramètres ou le CSS personnalisé que vous souhaitez appliquer uniquement à ce domaine.

Cela vous permet de faire en sorte que FastComments ait une apparence ou un fonctionnement complètement différent pour chaque site.

[app-screenshot-start url='/auth/my-account/customize-widget/new'; selector = 'label.domain'; title='Customization Rule Domain' app-screenshot-end]

Sur un site, vous pourriez utiliser du texte en gras et autoriser les commentaires anonymes, et un autre site pourrait
avoir un style différent et exiger que l'utilisateur laisse son e-mail lorsqu'il commente.
