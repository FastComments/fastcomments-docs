評論小工具可以為每個網站進行不一樣的<a href="/guide-customizations-and-configuration.html#custom-css" target="_blank">自訂</a>。

在小工具自訂頁面，你會看到一個標記為「domain」的下拉選單。 在此欄位選擇 `All` 會將規則套用到所有
你帳戶中的網站，你可以輸入特定的網域，將你想套用的設定或自訂 CSS 限制只套用到該網域。

這讓你可以讓 FastComments 在每個網站上呈現或運作得完全不同。

[app-screenshot-start url='/auth/my-account/customize-widget/new'; selector = 'label.domain'; title='Customization Rule Domain' app-screenshot-end]

一個網站可能會使用粗體文字並允許匿名留言，而另一個網站可能
會有不同的樣式並要求使用者在留言時留下電子郵件。
