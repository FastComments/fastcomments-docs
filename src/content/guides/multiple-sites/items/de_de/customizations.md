Das Kommentar-Widget kann für jede Website unterschiedlich <a href="/guide-customizations-and-configuration.html#custom-css" target="_blank">angepasst</a> werden.

Auf der Seite zur Widget-Anpassung finden Sie ein Dropdown-Menü mit der Bezeichnung "Domain". Wenn Sie in diesem Feld `All` auswählen, wird die Regel auf alle
Websites in Ihrem Konto angewendet, Sie können eine bestimmte Domain eingeben, wodurch die Einstellungen oder benutzerdefiniertes CSS nur auf diese Domain beschränkt werden.

Dies ermöglicht es Ihnen, dass FastComments für jede Website optisch oder funktional völlig unterschiedlich wirkt.

[app-screenshot-start url='/auth/my-account/customize-widget/new'; selector = 'label.domain'; title='Customization Rule Domain' app-screenshot-end]

Auf einer Website könnten Sie beispielsweise fette Schrift verwenden und anonyme Kommentare erlauben, und eine andere Website könnte
ein anderes Styling haben und verlangen, dass der Benutzer seine E-Mail angibt, wenn er kommentiert.
