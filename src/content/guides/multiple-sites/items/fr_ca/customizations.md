Le widget de commentaires peut être <a href="/guide-customizations-and-configuration.html#custom-css" target="_blank">personnalisé</a> différemment pour chaque site.

Sur la page de personnalisation du widget, vous trouverez un menu déroulant étiqueté "domaine". En sélectionnant `All` dans ce champ, la règle s'appliquera à tous
les sites de votre compte, vous pouvez saisir un domaine spécifique qui limitera les paramètres ou le CSS personnalisé que vous souhaitez appliquer uniquement à ce domaine.

Cela vous permet de faire en sorte que FastComments ait une apparence ou un fonctionnement complètement différent pour chaque site.

[app-screenshot-start url='/auth/my-account/customize-widget/new'; selector = 'label.domain'; title='Customization Rule Domain' app-screenshot-end]

Un site peut par exemple utiliser du texte en gras et autoriser les commentaires anonymes, tandis qu'un autre site peut avoir un style différent et exiger que l'utilisateur laisse son courriel lorsqu'il commente.
