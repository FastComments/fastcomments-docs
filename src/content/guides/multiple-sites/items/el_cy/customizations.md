Το widget σχολίων μπορεί να <a href="/guide-customizations-and-configuration.html#custom-css" target="_blank">προσαρμοστεί</a> διαφορετικά για κάθε ιστότοπο.

Στη σελίδα προσαρμογής του widget, θα βρείτε ένα αναπτυσσόμενο μενού με ετικέτα "domain". Επιλέγοντας `All` σε αυτό το πεδίο, ο κανόνας θα εφαρμοστεί σε όλους
τους ιστότοπους στον λογαριασμό σας, μπορείτε να εισάγετε έναν συγκεκριμένο domain ο οποίος θα περιορίσει τις ρυθμίσεις ή το προσαρμοσμένο CSS που θέλετε να εφαρμόσετε μόνο σε εκείνον.

Αυτό σας επιτρέπει να κάνετε το FastComments να φαίνεται ή να λειτουργεί εντελώς διαφορετικά για κάθε ιστότοπο.

[app-screenshot-start url='/auth/my-account/customize-widget/new'; selector = 'label.domain'; title='Customization Rule Domain' app-screenshot-end]

Σε έναν ιστότοπο μπορεί να χρησιμοποιείται έντονη γραφή και να επιτρέπονται ανώνυμα σχόλια, και σε έναν άλλο ιστότοπο μπορεί
να υπάρχει διαφορετική μορφοποίηση και να απαιτείται από τον χρήστη να αφήσει το email του όταν σχολιάζει.
