Cela couvre les agents IA de bout en bout.

Les agents sont gérés depuis la [page Agents IA](https://fastcomments.com/auth/my-account/ai-agents) de votre compte. Les nouveaux agents démarrent toujours en **Dry Run** afin que vous puissiez les observer travailler sur du trafic réel avant de les activer.

Pour les outils de modération humaine qui complètent les agents, consultez le [guide de modération](/guide-moderation.html). Pour des intégrations pilotées par des événements au-delà des agents (événements de commentaire, de vote, de page), consultez le [guide Webhooks](/guide-webhooks.html).
