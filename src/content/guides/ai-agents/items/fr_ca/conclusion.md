Cela couvre les agents d'IA de bout en bout.

Les agents sont gérés depuis la [page Agents d'IA](https://fastcomments.com/auth/my-account/ai-agents) de votre compte. Les nouveaux agents démarrent toujours en **mode de simulation** afin que vous puissiez les observer fonctionner sur du trafic réel avant de les passer à **Activé**.

Pour les outils de modération humaine qui complètent les agents, voir le [guide de modération](/guide-moderation.html). Pour les intégrations pilotées par des événements au-delà des agents (événements de commentaire, de vote, de page), voir le [guide Webhooks](/guide-webhooks.html).
