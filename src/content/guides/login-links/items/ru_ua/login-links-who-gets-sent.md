Все пользователи FastComments могут получить ссылки для входа. Они аутентифицируют только в области
к которой у пользователя есть доступ. Если пользователь является комментатором, ему никогда не будут предоставлены права администратора через ссылку для входа.

Кроме того, администраторы могут отправить любому пользователю тенанта новую ссылку для входа с [страницы пользователей](https://fastcomments.com/auth/my-account/users).
