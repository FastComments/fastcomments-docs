Все пользователи FastComments могут получить ссылки для входа. Эти ссылки аутентифицируют только в пределах области
к которой у пользователя есть доступ. Если пользователь является комментатором, при входе по ссылке ему никогда не будут предоставлены права администратора.

Кроме того, администраторы могут отправить любому tenant user новую ссылку для входа со [страницы пользователей](https://fastcomments.com/auth/my-account/users).
