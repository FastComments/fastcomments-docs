Dies ist das offizielle Django‑Paket für FastComments.

Eine Django‑App, die die FastComments‑Widgets über Template‑Tags hinzufügt, Secure SSO serverseitig signiert und vollständigen API‑Zugriff über das FastComments Python SDK bietet.
