Lorsqu'on ajoute le widget de commentaires à une page spécifique, nous avons un peu plus de contrôle.

Nous pouvons placer le widget exactement où nous le souhaitons, ce que nous ferons dans ce tutoriel en ajoutant un bloc de code personnalisé.

Tout d'abord, ouvrez la page sur laquelle vous souhaitez ajouter des commentaires, puis cliquez sur Éditer en haut à gauche.

Dans cet exemple, nous allons ajouter une section spécialement pour les commentaires, mais vous pouvez les ajouter à une section existante
de la page.
