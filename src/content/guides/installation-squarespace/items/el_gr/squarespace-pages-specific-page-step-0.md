Σε αυτή την ενότητα θα μάθουμε πώς να προσθέσουμε ζωντανά σχόλια σε μια συγκεκριμένη σελίδα του ιστότοπού σας Squarespace.

Με αυτή την προσέγγιση θα πρέπει να προσθέσετε τον κώδικα σε κάθε σελίδα στην οποία θέλετε να προσθέσετε το widget σχολίων.

Τα νήματα σχολίων θα είναι μοναδικά για κάθε σελίδα.
