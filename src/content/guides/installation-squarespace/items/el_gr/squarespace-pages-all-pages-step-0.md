Σε αυτή την ενότητα θα μάθουμε πώς να προσθέσουμε ζωντανό σχολιασμό σε όλες τις σελίδες του ιστότοπού σας Squarespace.

Θα χρειαστεί να προσθέσετε τον κώδικα μόνο μία φορά και το widget θα προστεθεί αυτόματα σε οποιεσδήποτε υπάρχουσες σελίδες, και σε οποιεσδήποτε
σελίδες που δημιουργήσετε στο μέλλον.

Τα νήματα σχολίων θα είναι μοναδικά για κάθε σελίδα.
