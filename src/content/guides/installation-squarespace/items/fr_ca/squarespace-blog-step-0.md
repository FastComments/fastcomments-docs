Dans cette section, nous apprendrons comment ajouter des commentaires en temps réel à votre blogue Squarespace.

Vous n'aurez à ajouter le code qu'une seule fois et le widget sera automatiquement ajouté à tous les billets de blogue existants, et à tous les
billets de blogue que vous créerez à l'avenir.

Les fils de commentaires seront uniques pour chaque billet de blogue.
