Lors de l'ajout du widget de commentaires à une page spécifique, nous avons un peu plus de contrôle.

Nous pouvons ajouter le widget exactement où nous le voulons, ce que nous ferons dans ce tutoriel en ajoutant un bloc de code personnalisé.

D'abord, ouvrez la page à laquelle vous voulez ajouter des commentaires, puis cliquez sur Modifier en haut à gauche.

Dans cet exemple, nous allons ajouter une section spécifiquement pour les commentaires, mais vous pourriez les ajouter à une section existante
de la page.

Ajoutons notre section :

    <img class="screenshot-image" src="/images/installation-guides/squarespace-pages-specific-page-step-1-1-add-section.png" alt="Ajouter une section" />
</div>

Nous allons ajouter une section vide :

<div class="screenshot white-bg">
    <div class="title">Ajouter une section vide</div>
    <img class="screenshot-image" src="/images/installation-guides/squarespace-pages-specific-page-step-1-2-add-blank-section.png" alt="Ajouter une section vide" />
</div>

Vous verrez une grande zone vide ajoutée. Cliquez sur `ADD BLOCK` en haut à gauche :

<div class="screenshot white-bg">
    <div class="title">Ajouter un bloc</div>
    <img class="screenshot-image" src="/images/installation-guides/squarespace-pages-specific-page-step-1-3-add-block.png" alt="Ajouter un bloc" />
</div>

Faites défiler vers le bas jusqu'à ce que vous trouviez l'option `Code` et cliquez dessus :

<div class="screenshot white-bg">
    <div class="title">Sélectionner le widget Code</div>
    <img class="screenshot-image" src="/images/installation-guides/squarespace-pages-specific-page-step-1-4-add-code-widget.png" alt="Sélectionner le widget Code" />
</div>

Positionnez votre bloc de code là où vous souhaitez que le widget de commentaires soit placé :

    <img class="screenshot-image" src="/images/installation-guides/squarespace-pages-specific-page-step-1-5-position.png" alt="Position" />
