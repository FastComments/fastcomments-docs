Σε αυτή την ενότητα θα μάθουμε πώς να προσθέσετε τη δυνατότητα ζωντανών σχολίων σε μια συγκεκριμένη σελίδα του ιστότοπού σας στο Squarespace.

Με αυτή την προσέγγιση θα χρειαστεί να προσθέσετε τον κώδικα σε κάθε σελίδα στην οποία θέλετε να εμφανίζεται το widget σχολίων.

Τα νήματα σχολίων θα είναι μοναδικά για κάθε σελίδα.
