Οι διαχειριζόμενοι ενοικιαστές μπορούν να διαχειριστούν μέσω του πίνακα διαχείρισης μεταβαίνοντας στο [Ο λογαριασμός μου -> Ενοικιαστές](https://fastcomments.com/auth/my-account/tenants).

Όλες οι ενέργειες που εκτελούνται σε έναν διαχειριζόμενο ενοικιαστή διενεργούνται με το να μεταβείτε σε αυτόν τον ενοικιαστή. Μεταβείτε σε έναν ενοικιαστή για να προσθέσετε χρήστες, συντονιστές, να μετριάσετε σχόλια,
να λάβετε κλειδιά API και άλλα. Μπορείτε επίσης να μιμηθείτε χρήστες ενός ενοικιαστή για να δείτε τα πράγματα όπως θα τα έβλεπαν εκείνοι.
