כדי להשתמש ב-API כדי ליצור טננט ממותג לבן, עלינו לבצע את הפעולות הבאות:

1. בצע קריאה ל-[API טננטים](/guide-api.html#tenant-structure) כדי ליצור את הטננט.
2. בצע קריאה ל-[API חבילות טננט](/guide-api.html#tenant-package-structure) כדי ליצור חבילה אחת או יותר עבור הטננט.
3. בצע קריאה ל-[API טננטים](/guide-api.html#tenant-patch) כדי להגדיר איזו חבילה פעילה עבור הטננט.
4. בצע קריאה ל-[API משתמשי טננט](/guide-api.html#tenant-user-structure) כדי להוסיף משתמשים מנהלים לטננט.
5. בצע קריאה ל-[API המודרטורים](/guide-api.html#moderator-structure) כדי להוסיף ולהזמין מודרטורים לטננט.
6. באופן אופציונלי, [הגדר SSO](/guide-customizations-and-configuration.html#sso).

כל זאת ניתן לבצע במהלך תקופת הניסיון.
