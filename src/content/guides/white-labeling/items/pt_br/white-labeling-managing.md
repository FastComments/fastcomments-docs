Locatários gerenciados podem ser administrados pelo painel de administração acessando [Minha Conta -> Locatários](https://fastcomments.com/auth/my-account/tenants).

Todas as ações realizadas em um locatário gerenciado são feitas alternando para esse locatário. Alterne para um locatário para adicionar usuários, moderadores, moderar comentários,
obter chaves de API e muito mais. Você também pode se passar por usuários de um locatário para ver as coisas como eles veriam.
