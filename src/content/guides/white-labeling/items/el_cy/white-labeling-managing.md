Οι διαχειριζόμενοι tenants μπορούν να διαχειριστούν μέσω του admin dashboard πηγαίνοντας στο [My Account -> Tenants](https://fastcomments.com/auth/my-account/tenants).

Όλες οι ενέργειες που εκτελούνται σε έναν διαχειριζόμενο tenant γίνονται μεταβαίνοντας σε αυτόν τον tenant. Μεταβείτε σε έναν tenant για να προσθέσετε users, moderators, να διαχειριστείτε σχόλια,
να αποκτήσετε API keys και άλλα. Μπορείτε επίσης να impersonate χρήστες ενός tenant για να δείτε τα πράγματα όπως τα βλέπουν.
