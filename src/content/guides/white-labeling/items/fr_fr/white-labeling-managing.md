Les locataires gérés peuvent être administrés via le tableau de bord administrateur en allant dans [Mon compte -> Locataires](https://fastcomments.com/auth/my-account/tenants).

Toutes les actions effectuées sur un locataire géré se font en basculant vers ce locataire. Basculez sur un locataire pour ajouter des utilisateurs, des modérateurs, modérer des commentaires,
obtenir des clés API et plus encore. Vous pouvez également vous faire passer pour des utilisateurs d'un locataire pour voir les choses comme ils les verraient.
