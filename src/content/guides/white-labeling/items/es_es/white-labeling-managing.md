Los tenants gestionados se pueden administrar desde el panel de administración accediendo a [Mi cuenta -> Inquilinos](https://fastcomments.com/auth/my-account/tenants).

Todas las acciones realizadas sobre un tenant gestionado se hacen cambiando a ese tenant. Cambia a un tenant para añadir usuarios, moderadores, moderar comentarios,
obtener claves de la API y más. También puedes suplantar a usuarios de un tenant para ver las cosas como ellos las verían.
