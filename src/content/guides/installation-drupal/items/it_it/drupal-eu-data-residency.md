Se il tuo account FastComments è ospitato nell'UE, aggiorna due impostazioni in `Administration > Configuration > Content > FastComments`:

- **URL CDN** - `https://cdn-eu.fastcomments.com`
- **URL del sito** - `https://eu.fastcomments.com`

Dovrai inoltre prelevare il tuo Tenant ID e il tuo API Secret dalla dashboard UE su [eu.fastcomments.com/auth/my-account/api](https://eu.fastcomments.com/auth/my-account/api) invece della dashboard predefinita degli Stati Uniti. Il resto del modulo funziona allo stesso modo.
