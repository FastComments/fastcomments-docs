Если ваша учётная запись FastComments размещена в ЕС, обновите два параметра в `Administration > Configuration > Content > FastComments`:

- **CDN URL** - `https://cdn-eu.fastcomments.com`
- **Site URL** - `https://eu.fastcomments.com`

Вам также следует получить ваш Tenant ID и API Secret из панели управления ЕС по адресу [eu.fastcomments.com/auth/my-account/api](https://eu.fastcomments.com/auth/my-account/api) вместо панели управления США по умолчанию. Всё остальное в модуле работает так же.
