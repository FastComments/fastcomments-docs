Якщо ваш обліковий запис FastComments розміщено в ЄС, оновіть два налаштування в `Administration > Configuration > Content > FastComments`:

- **CDN URL** - `https://cdn-eu.fastcomments.com`
- **Site URL** - `https://eu.fastcomments.com`

Також вам слід отримати Tenant ID та API Secret з панелі керування ЄС за адресою [eu.fastcomments.com/auth/my-account/api](https://eu.fastcomments.com/auth/my-account/api) замість стандартної панелі керування США. Усе інше в модулі працює так само.
