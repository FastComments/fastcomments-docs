Εάν ο λογαριασμός FastComments σας φιλοξενείται στην ΕΕ, ενημερώστε δύο ρυθμίσεις στο `Administration > Configuration > Content > FastComments`:

- **CDN URL** - `https://cdn-eu.fastcomments.com`
- **Site URL** - `https://eu.fastcomments.com`

Επίσης, θα πρέπει να ανακτήσετε το Tenant ID και το API Secret από τον πίνακα ελέγχου της ΕΕ στη διεύθυνση [eu.fastcomments.com/auth/my-account/api](https://eu.fastcomments.com/auth/my-account/api) αντί για τον προεπιλεγμένο πίνακα ελέγχου των ΗΠΑ. Όλα τα υπόλοιπα στη μονάδα λειτουργούν με τον ίδιο τρόπο.
