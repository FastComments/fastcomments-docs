Wenn Ihr FastComments-Konto in der EU gehostet wird, aktualisieren Sie zwei Einstellungen unter `Administration > Configuration > Content > FastComments`:

- **CDN-URL** - `https://cdn-eu.fastcomments.com`
- **Site-URL** - `https://eu.fastcomments.com`

Außerdem sollten Sie Ihre Tenant ID und Ihr API Secret vom EU-Dashboard unter [eu.fastcomments.com/auth/my-account/api](https://eu.fastcomments.com/auth/my-account/api) abrufen, anstatt vom standardmäßigen US-Dashboard. Alles andere im Modul funktioniert gleich.
