Als uw FastComments-account in de EU wordt gehost, werk dan twee instellingen bij in `Administration > Configuration > Content > FastComments`:

- **CDN URL** - `https://cdn-eu.fastcomments.com`
- **Site URL** - `https://eu.fastcomments.com`

U wilt ook uw Tenant ID en API Secret ophalen uit het EU-dashboard op [eu.fastcomments.com/auth/my-account/api](https://eu.fastcomments.com/auth/my-account/api) in plaats van het standaard Amerikaanse dashboard. Verder werkt de module hetzelfde.
