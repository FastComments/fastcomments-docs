Если ваш аккаунт FastComments размещён в ЕС, обновите два параметра в `Administration > Configuration > Content > FastComments`:

- **CDN URL** - `https://cdn-eu.fastcomments.com`
- **Site URL** - `https://eu.fastcomments.com`

Также вам следует получить Tenant ID и API Secret из панели управления ЕС по адресу [eu.fastcomments.com/auth/my-account/api](https://eu.fastcomments.com/auth/my-account/api) вместо используемой по умолчанию панели управления США. Всё остальное в модуле работает так же.
