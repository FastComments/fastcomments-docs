1. Поставете този модул в директорията `modules/custom/fastcomments/` на вашия Drupal сайт (или инсталирайте чрез Composer: `composer require drupal/fcom`).
2. Активирайте модула:
   ```bash
   drush en fastcomments
   ```
   Или го активирайте чрез административния интерфейс в **Extend** (`/admin/modules`).
