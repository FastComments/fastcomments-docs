---
1. Posiziona questo modulo nella directory del tuo sito Drupal `modules/custom/fastcomments/` (o installalo tramite Composer: `composer require drupal/fcom`).
2. Abilita il modulo:
   ```bash
   drush en fastcomments
   ```
   Oppure abilitalo tramite l'interfaccia di amministrazione in **Extend** (`/admin/modules`).
