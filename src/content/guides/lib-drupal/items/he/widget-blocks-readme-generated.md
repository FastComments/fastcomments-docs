מספר בלוקים זמינים תחת **מבנה > פריסת בלוקים** (`/admin/structure/block`):

- **FastComments Widget** - הווידג'ט הראשי של תגובות. מזהה אוטומטית את הישות הנוכחית. מדלג על ישויות שכבר כוללות את שדה FastComments (כדי למנוע שכפולים).
- **FastComments Live Chat** - צ'אט זורם בזמן אמת. ניתן למקם לצד שדה התגובות באותו עמוד.
- **FastComments Collab Chat** - הערות ודיון המבוססים על בחירת טקסט.
- **FastComments Image Chat** - הערות מבוססות קואורדינטות על תמונות.
- **FastComments Recent Comments** - מציג תגובות אחרונות ברחבי האתר שלך. ניתן להגדיר את מספר התגובות.
- **FastComments Top Pages** - מציג דפים עם הכי הרבה תגובות.

הבלוקים המתמקדים בתוכן (Live Chat, Collab Chat, Image Chat) מזהים אוטומטית את הישות הנוכחית, ובדפים שאינם ישויות משתמשים במקום זאת בזיהוי מבוסס-נתיב.
