1. Τοποθετήστε αυτό το module στον φάκελο `modules/custom/fastcomments/` του ιστότοπου Drupal σας (ή εγκαταστήστε μέσω Composer: `composer require drupal/fcom`).
2. Ενεργοποιήστε το module:
   ```bash
   drush en fastcomments
   ```
   Ή ενεργοποιήστε μέσω του διαχειριστικού περιβάλλοντος (UI) στην επιλογή **Extend** (`/admin/modules`).
