1. Bu modülü Drupal sitenizin `modules/custom/fastcomments/` dizinine yerleştirin (veya Composer ile yükleyin: `composer require drupal/fcom`).
2. Modülü etkinleştirin:
   ```bash
   drush en fastcomments
   ```
   Veya yönetici arayüzünden **Genişlet** (`/admin/modules`) üzerinden etkinleştirin.
