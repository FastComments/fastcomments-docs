Bir kullanıcı FastComments alanı etkin olan bir varlığı ziyaret ettiğinde:

1. FastComments JavaScript widget'ı CDN'den yüklenir.
2. SSO yapılandırıldıysa, kullanıcının Drupal kimliği FastComments'a iletilir.
3. Bir `<noscript>` yedeği, JavaScript olmayan kullanıcılar için sunucuda oluşturulmuş yorumlar sağlar (sadece Live Comments ve Streaming Chat modlarında).
