1. Поставите овај модул у директоријум вашег Drupal сајта `modules/custom/fastcomments/` (или инсталирајте преко Composer-а: `composer require drupal/fcom`).
2. Омогућите модул:
   ```bash
   drush en fastcomments
   ```
   Или га омогућите преко административног интерфејса на **Extend** (`/admin/modules`).
