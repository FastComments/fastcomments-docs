1. Postavite ovaj modul u direktorij vaše Drupal stranice `modules/custom/fastcomments/` (ili instalirajte putem Composera: `composer require drupal/fcom`).
2. Omogućite modul:
   ```bash
   drush en fastcomments
   ```
   Ili omogućite putem administratorskog sučelja na **Extend** (`/admin/modules`).
