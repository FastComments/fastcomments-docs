Διατίθενται αρκετά μπλοκ μέσω του **Structure > Block layout** (`/admin/structure/block`):

- **FastComments Widget** - Το κύριο widget σχολιασμού. Εντοπίζει αυτόματα την τρέχουσα οντότητα. Παραλείπει οντότητες που ήδη έχουν το πεδίο FastComments (για να αποφεύγονται διπλότυπα).
- **FastComments Live Chat** - Συνομιλία ροής σε πραγματικό χρόνο. Μπορεί να τοποθετηθεί δίπλα στο πεδίο σχολίων στην ίδια σελίδα.
- **FastComments Collab Chat** - Σχολιασμός και συζήτηση με επιλογή κειμένου.
- **FastComments Image Chat** - Σχολιασμός σε εικόνες με βάση συντεταγμένες.
- **FastComments Recent Comments** - Εμφανίζει πρόσφατα σχόλια σε ολόκληρο τον ιστότοπό σας. Ρυθμιζόμενος αριθμός σχολίων.
- **FastComments Top Pages** - Εμφανίζει σελίδες με τα περισσότερα σχόλια.

Τα μπλοκ που επικεντρώνονται στο περιεχόμενο (Live Chat, Collab Chat, Image Chat) εντοπίζουν αυτόματα την τρέχουσα οντότητα και ως εφεδρικό χρησιμοποιούν ένα αναγνωριστικό βασισμένο στη διαδρομή σε σελίδες χωρίς οντότητα.
