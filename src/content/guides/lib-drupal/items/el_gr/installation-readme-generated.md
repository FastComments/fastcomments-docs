1. Τοποθετήστε αυτή τη μονάδα στον κατάλογο του ιστότοπού σας Drupal `modules/custom/fastcomments/` (ή εγκαταστήστε μέσω Composer: `composer require drupal/fcom`).
2. Ενεργοποιήστε τη μονάδα:
   ```bash
   drush en fastcomments
   ```
   Ή ενεργοποιήστε μέσω της διεπαφής διαχείρισης στο **Extend** (`/admin/modules`).
