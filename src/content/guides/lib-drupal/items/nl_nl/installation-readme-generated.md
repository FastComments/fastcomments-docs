---
1. Plaats deze module in de map van uw Drupal-site `modules/custom/fastcomments/` (of installeer via Composer: `composer require drupal/fcom`).
2. Activeer de module:
   ```bash
   drush en fastcomments
   ```
   Of activeer via de beheerdersinterface onder **Uitbreiden** (`/admin/modules`).
