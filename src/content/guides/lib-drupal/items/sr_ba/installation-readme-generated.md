---
1. Ставите овај модул у директоријум вашег Drupal сајта `modules/custom/fastcomments/` (или инсталирајте преко Composer-а: `composer require drupal/fcom`).
2. Омогућите модул:
   ```bash
   drush en fastcomments
   ```
   Или омогућите преко административног интерфејса на страници **Проширење** (`/admin/modules`).
