다음 블록들은 **Structure > Block layout** (`/admin/structure/block`)에서 사용할 수 있습니다:

- **FastComments Widget** - 주요 댓글 위젯입니다. 현재 엔티티를 자동으로 감지합니다. FastComments 필드가 이미 있는 엔티티는 건너뜁니다(중복 방지).
- **FastComments Live Chat** - 실시간 스트리밍 채팅입니다. 같은 페이지의 댓글 필드 옆에 배치할 수 있습니다.
- **FastComments Collab Chat** - 텍스트 선택을 통한 주석 및 토론입니다.
- **FastComments Image Chat** - 이미지에서의 좌표 기반 주석입니다.
- **FastComments Recent Comments** - 사이트 전체의 최근 댓글을 표시합니다. 표시할 댓글 수를 구성할 수 있습니다.
- **FastComments Top Pages** - 댓글이 가장 많은 페이지를 표시합니다.

콘텐츠 중심 블록 (Live Chat, Collab Chat, Image Chat)은 현재 엔티티를 자동으로 감지하며, 엔티티가 아닌 페이지에서는 경로 기반 식별자로 대체됩니다.
