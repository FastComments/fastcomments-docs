Ці компоненти є оболонками навколо наших основних компонентів VanillaJS. Ми можемо автоматично оновлювати ці компоненти (виправляти помилки, додавати функції) без публікації цієї бібліотеки, тож хоча вона може деякий час не бути опублікованою, це не означає, що FastComments не перебуває в активній розробці! Переглядайте [наш блог](https://blog.fastcomments.com/) для оновлень. Зміни API, що порушують сумісність, або нові функції ніколи не будуть додані до основної бібліотеки без підвищення версії в цій бібліотеці.
