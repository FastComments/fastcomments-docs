Ове компоненте су омоти око наших core VanillaJS компоненти. Можемо аутоматски ажурирати ове компоненте (поправљати грешке, додавати нове могућности) без објављивања ове библиотеке, па иако можда неће бити објављена неко време, то не значи да FastComments није у активној развоју! Слободно погледајте [наш блог](https://blog.fastcomments.com/) за ажурирања. Промене API-ја које нарушавају уназадну компатибилност или нове могућности никада неће бити испоручене у основну core библиотеку без повећања верзије ове библиотеке.
