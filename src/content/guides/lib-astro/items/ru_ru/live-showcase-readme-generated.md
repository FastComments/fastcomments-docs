Чтобы увидеть все виджеты и потоки, работающие локально с публичным арендатором `demo`, клонируйте репозиторий и выполните:

```bash
cd example
npm install
npm run dev
```

Каждый компонент имеет свою страницу в каталоге `example/src/pages/`, которую вы можете скопировать прямо в свой сайт на Astro.
