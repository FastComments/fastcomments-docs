Эти компоненты являются оболочками над нашими базовыми компонентами на VanillaJS. Мы можем автоматически обновлять эти компоненты (исправлять ошибки, добавлять функции) без публикации этой библиотеки, так что хотя она может некоторое время не публиковаться, это не означает, что FastComments не находится в активной разработке! Не стесняйтесь заглядывать в [наш блог](https://blog.fastcomments.com/) для получения обновлений. Нарушающие совместимость изменения API или новые возможности никогда не будут включены в базовую библиотеку без увеличения версии в этой библиотеке.
