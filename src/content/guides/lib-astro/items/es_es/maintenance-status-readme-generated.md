Estos componentes son envoltorios alrededor de nuestros componentes centrales VanillaJS. Podemos actualizar automáticamente estos componentes (corregir errores, añadir funciones) sin publicar esta biblioteca, así que, aunque puede que no se publique durante un tiempo, eso no significa que FastComments no esté en desarrollo activo. No dudes en consultar [nuestro blog](https://blog.fastcomments.com/) para actualizaciones. Los cambios o características que rompan la API nunca se incorporarán a la biblioteca central subyacente sin un incremento de versión en esta biblioteca.
