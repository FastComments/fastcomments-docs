Για να δείτε κάθε widget και flow να τρέχει τοπικά ενάντια στον δημόσιο tenant `demo`, κλωνοποιήστε το αποθετήριο και εκτελέστε:

```bash
cd example
npm install
npm run dev
```

Κάθε component έχει τη δική του σελίδα στο `example/src/pages/` που μπορείτε να αντιγράψετε απευθείας στον δικό σας ιστότοπο Astro.
