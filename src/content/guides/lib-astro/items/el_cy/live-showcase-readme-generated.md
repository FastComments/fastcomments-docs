Για να δείτε κάθε widget και flow να τρέχουν τοπικά ενάντια στον δημόσιο ενοικιαστή `demo`, κλωνοποιήστε το αποθετήριο και εκτελέστε:

```bash
cd example
npm install
npm run dev
```

Κάθε component έχει τη δική του σελίδα στον φάκελο `example/src/pages/` την οποία μπορείτε να αντιγράψετε απευθείας στον δικό σας ιστότοπο Astro.
