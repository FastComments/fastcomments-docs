Ови компоненти су омотачи око наших основних VanillaJS компоненти. Можемо аутоматски ажурирати ове компоненте (поправљати грешке, додавати функције) без објављивања ове библиотеке, тако да иако можда неће бити објављена неко вријеме, то не значи да FastComments није у активној развоју! Слободно погледајте [наш блог](https://blog.fastcomments.com/) за ажурирања. Промјене које нарушавају API или функције никада неће бити испоручене у основну библиотеку без повећања верзије ове библиотеке.
