Эти компоненты являются обёртками над нашими основными компонентами VanillaJS. Мы можем автоматически обновлять эти компоненты (исправлять ошибки, добавлять новые возможности) без публикации этой библиотеки, так что даже если она какое‑то время не будет опубликована, это не означает, что FastComments не находится в активной разработке! Не стесняйтесь заглядывать в [наш блог](https://blog.fastcomments.com/) для получения обновлений. Несовместимые изменения API или новые функции никогда не будут добавлены в основную (базовую) библиотеку без увеличения версии в этой библиотеке.
