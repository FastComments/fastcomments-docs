Чтобы увидеть все виджеты и потоки, запущенные локально против публичного тенанта `demo`, клонируйте репозиторий и выполните:

```bash
cd example
npm install
npm run dev
```

Каждый компонент имеет свою страницу в каталоге `example/src/pages/`, которую вы можете напрямую скопировать в свой собственный сайт на Astro.
