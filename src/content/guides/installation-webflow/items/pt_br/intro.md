Com o FastComments, podemos adicionar facilmente comentários ao vivo a qualquer site do Webflow.

As instruções abaixo são para um site estático simples, mas também funcionariam para um site dinâmico via Webflow Collections.

Observe que este tutorial requer uma conta FastComments. Recomendamos que você crie sua conta primeiro e depois volte aqui. [Você pode criar uma conta aqui.](https://fastcomments.com/auth/tenant-signup?packageId=flex)
