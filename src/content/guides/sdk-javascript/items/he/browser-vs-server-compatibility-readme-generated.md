---
ה-SDK הזה משתמש ב-**שתי נקודות כניסה** כדי להבטיח תאימות מיטבית ולמנוע שגיאות בזמן ריצה:

- **`fastcomments-sdk/browser`** - גרסה בטוחה לדפדפן עם `fetch` מובנה
- **`fastcomments-sdk/server`** - גרסה מלאה ל-Node.js עם תמיכה ב-SSO
- **`fastcomments-sdk`** (ברירת מחדל) - טיפוסים בלבד, בטוח לייבא בכל מקום
