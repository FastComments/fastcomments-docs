За програмере које можда нећете желети да буду `Administrators`, размислите о креирању `Administrator` корисника са следећим дозволама:

1. Analytics Admin
2. Customizations Admin
3. Data Management Admin
4. Comment Moderation Admin
5. API/SSO Admin

Овај скуп дозвола даће програмеру све што му је потребно да подеси FastComments, као и видљивост у систему потребну да се увери да функционише.

Разлог за ове дозволе је следећи:

1. **Analytics Admin**: Ово се може користити за отклањање проблема у коришћењу FastComments.
2. **Customizations Admin**: Биће потребно за примену прилагођених стилова на видгету за коментаре.
3. **Data Management Admin**: Биће потребно за управљање увозима и извозима, и подешавање webhooks.
4. **Comment Moderation Admin**: Биће потребно да би прегледали податке о коментарима, бар током подешавања.
5. **API/SSO Admin**: Ово ће им омогућити да директно преузму API keys са наше платформе. Сматрамо да је ово сигурније него да `Administrator` то копира за њих и пошаље API Secret путем е-поште, што можда није превише сигурно.
