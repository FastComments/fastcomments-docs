За програмере које можда не желите да буду `Administrators`, размислите о креирању `Administrator` корисника
са сљедећим дозволама:

1. Analytics Admin
2. Customizations Admin
3. Data Management Admin
4. Comment Moderation Admin
5. API/SSO Admin

Овај скуп дозвола ће програмеру дати све што му треба за подешавање FastComments-а као и увид у систем потребан да се осигура да он ради.

Разлог за ове дозволе је сљедећи:

1. **Analytics Admin**: Ово може бити кориштено за отклањање проблема у коришћењу FastComments-а.
2. **Customizations Admin**: Ово ће бити потребно за примјену прилагођеног стила на видгет за коментаре.
3. **Data Management Admin**: Ово ће бити потребно за управљање увозима и извозима, и подешавање webhooks.
4. **Comment Moderation Admin**: Ово ће бити потребно да се виде подаци о коментарима, барем током подешавања.
5. **API/SSO Admin**: Ово ће им омогућити да директно преузму API keys са наше платформе. Сматрамо да је ово сигурније него да `Administrator` то копира за њих и пошаље API Secret путем е-поште, што можда није баш сигурно.
