За програмере за које можда не желите да буду `Administrators`, размислите о креирању `Administrator` корисника
са следећим дозволама:

1. Analytics Admin
2. Customizations Admin
3. Data Management Admin
4. Comment Moderation Admin
5. API/SSO Admin

Овај скуп дозвола даће програмеру све што му треба да подеси FastComments, као и увид у систем
потребан да би се уверио да све функционише.

Образложење за ове дозволе је следеће:

1. **Analytics Admin**: Ово се може користити за отклањање проблема у коришћењу FastComments.
2. **Customizations Admin**: Ово ће бити потребно да бисте применили прилагођени стил на видгет за коментаре.
3. **Data Management Admin**: Ово ће бити потребно за управљање увозима и извозима, и за подешавање webhooks.
4. **Comment Moderation Admin**: Ово ће бити потребно да бисте видели податке о коментарима, барем током подешавања.
5. **API/SSO Admin**: Ово ће им омогућити да директно са наше платформе преузму API keys. Сматрамо
да је ово сигурније од тога да им `Administrator` то копира и пошаље API Secret путем е-поште, што
можда није баш безбедно.
