Με το FastComments, το `username` πρέπει να είναι μοναδικό σε όλες τις ιστοσελίδες και κοινότητες. Αυτό οφείλεται στο ότι χρησιμοποιούνταν για να σας συνδέει στο fastcomments.com.

Η εξαίρεση είναι το SSO - αφού οι χρήστες SSO συνδέονται με έναν συγκεκριμένο FastComments.com Tenant - μπορούν να έχουν οποιοδήποτε `username` εντός αυτής της κοινότητας.

#### Display Names

Το FastComments υποστηρίζει επίσης το `Display Name`. Τα display names δεν χρειάζεται να είναι μοναδικά, και μπορεί να περιέχουν emojis. Μπορείτε να ορίσετε το display name σας στη σελίδα [Στοιχεία λογαριασμού](https://fastcomments.com/auth/my-account/edit-details).

#### Μοναδικότητα του Display Name

Εάν δύο χρήστες του FastComments.com, σε δύο πολύ διαφορετικές μεγάλες κοινότητες, έχουν παρόμοια ονόματα, θα θέλαμε απλώς να μπορούν να ορίσουν το προτιμώμενο display name τους. Ωστόσο, δεν μπορούμε να περιορίσουμε πού σχολιάζει κάποιος με τον παγκόσμιο λογαριασμό FastComments, οπότε μερικές φορές αυτό μπορεί να προκαλέσει σύγχυση.

Το να έχετε παρόμοιο όνομα με κάποιον άλλο χρήστη ή με έναν διαχειριστή, από μόνο του, δεν αποτελεί περίπτωση κατάχρησης. Ωστόσο, εάν καταχραστείτε αυτό το χαρακτηριστικό με το να μιμηθείτε σκόπιμα άλλους χρήστες, ενδέχεται να απενεργοποιήσουμε αυτή τη δυνατότητα στον λογαριασμό σας, και θα περιοριστείτε στη χρήση μόνο του `username`.
