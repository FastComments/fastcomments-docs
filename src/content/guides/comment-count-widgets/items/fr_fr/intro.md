FastComments fournit des widgets de comptage de commentaires légers et autonomes qui peuvent afficher le nombre de commentaires pour des pages ou des articles spécifiques. Ces widgets sont séparés du widget de commentaires principal et sont conçus pour être utilisés partout où vous souhaitez afficher le nombre de commentaires sans l'interface complète de commentaires.

There are two types of comment count widgets available:

1. **Comment Count Widget** - Pour afficher le nombre de commentaires d'une seule page
2. **Bulk Comment Count Widget** - Pour afficher efficacement les totaux de commentaires pour plusieurs pages sur une même page

This article covers the VanillaJS widgets. Nos bibliothèques React/Angular/etc. disposent également de leurs propres widgets.
