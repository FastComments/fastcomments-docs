FastComments παρέχει ελαφριά, αυτόνομα widgets μέτρησης σχολίων που μπορούν να εμφανίζουν τον αριθμό σχολίων για συγκεκριμένες σελίδες ή δημοσιεύσεις. Αυτά τα widgets είναι ξεχωριστά από το κύριο widget σχολιασμού και έχουν σχεδιαστεί για χρήση οπουδήποτε θέλετε να εμφανίσετε αριθμούς σχολίων χωρίς ολόκληρο το περιβάλλον σχολιασμού.

Υπάρχουν δύο τύποι widgets μέτρησης σχολίων διαθέσιμοι:

1. **Comment Count Widget** - Για την εμφάνιση του αριθμού σχολίων μιας μεμονωμένης σελίδας
2. **Bulk Comment Count Widget** - Για την αποδοτική εμφάνιση των αριθμών σχολίων σε πολλές σελίδες στην ίδια σελίδα

Αυτό το άρθρο καλύπτει τα widgets VanillaJS. Οι βιβλιοθήκες μας για React/Angular/κ.λπ. έχουν επίσης τα δικά τους widgets.
