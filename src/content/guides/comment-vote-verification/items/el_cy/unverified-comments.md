Τα μη επαληθευμένα σχόλια θα εμφανίζονται με μια μικρή ετικέτα «Μη επαληθευμένο σχόλιο:» και θα φαίνονται ελαφρώς
ξεθωριασμένα.

Αυτά τα πάντα μπορούν να προσαρμοστούν, δείτε τον [οδηγό προσαρμογής](/guide-customizations-and-configuration.html#custom-css).

Τα μη επαληθευμένα σχόλια μπορούν να ρυθμιστούν να λήγουν, δείτε την τεκμηρίωση [Ρυθμίσεις μετριασμού](/guide-moderation.html#moderation-settings).

Τα ανώνυμα σχόλια είναι πάντα μη επαληθευμένα - αφού ο χρήστης δεν άφησε διεύθυνση email που θα μπορούσε
να χρησιμοποιηθεί για επαλήθευση.

Η ενεργοποίηση των Ανώνυμων Σχολίων δεν αφαιρεί την ετικέτα Μη επαληθευμένο, ωστόσο αυτή μπορεί να αφαιρεθεί με μορφοποίηση ή [διαμόρφωση](/guide-customizations-and-configuration.html#disable-unverified-label).
