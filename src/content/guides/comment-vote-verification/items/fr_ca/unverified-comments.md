---
Les commentaires non vérifiés s'afficheront avec une petite "Commentaire non vérifié:" étiquette, et apparaîtront légèrement
estompés.

Tout cela peut être personnalisé, consultez notre [guide de personnalisation](/guide-customizations-and-configuration.html#custom-css).

Les commentaires non vérifiés peuvent être configurés pour expirer, consultez notre documentation [Paramètres de modération](/guide-moderation.html#moderation-settings).

Les commentaires anonymes sont toujours non vérifiés - puisque l'utilisateur n'a pas laissé d'adresse courriel qui pourrait
être utilisée pour la vérification.

Autoriser les commentaires anonymes n'enlève pas l'étiquette « Non vérifié », cependant celle-ci peut être supprimée avec du style ou la [configuration](/guide-customizations-and-configuration.html#disable-unverified-label).
