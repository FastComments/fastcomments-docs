FastComments는 사용자 확인을 위한 고유한 솔루션을 구현하여 사용자에게 발송되는 이메일 수를 줄입니다
이를 우리는 Verification Sessions라고 부릅니다.

새 사용자가 동일한 브라우저 창에서 같은 페이지에 댓글을 열 번 남기고 투표를 두 번 한다면, 그들에게
열두 통의 확인 이메일을 보낼 필요는 없습니다.

대신, 이 경우 FastComments는 확인 이메일을 한 통만 보냅니다.

이것은 쿠키를 필요로 하지 않습니다.

브라우저 창이 닫히거나 사용자가 다른 페이지로 이동하면, 이를 검증 세션의 종료로 간주합니다.
