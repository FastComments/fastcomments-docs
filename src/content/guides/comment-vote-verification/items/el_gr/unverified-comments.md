Τα μη επαληθευμένα σχόλια θα εμφανίζονται με μια μικρή ετικέτα «Μη επαληθευμένο σχόλιο:», και θα φαίνονται ελαφρώς
ξεθωριασμένα

Αυτά όλα μπορούν να προσαρμοστούν, δείτε τον [οδηγό προσαρμογών](/guide-customizations-and-configuration.html#custom-css).

Τα μη επαληθευμένα σχόλια μπορούν να ρυθμιστούν ώστε να λήγουν, δείτε την τεκμηρίωση [Ρυθμίσεις Εποπτείας](/guide-moderation.html#moderation-settings).

Τα ανώνυμα σχόλια είναι πάντα μη επαληθευμένα - καθώς ο χρήστης δεν άφησε ένα email που θα μπορούσε
να χρησιμοποιηθεί για επαλήθευση.

Η ενεργοποίηση ανώνυμων σχολίων δεν αφαιρεί την ετικέτα «Μη επαληθευμένο», ωστόσο αυτό μπορεί να αφαιρεθεί με στυλ ή [διαμόρφωση](/guide-customizations-and-configuration.html#disable-unverified-label).
