FastComments реализует уникальное решение для проверки пользователей, чтобы сократить количество писем
отправляемых вашим пользователям — мы называем это Сессии верификации.

Если новый пользователь оставляет десять комментариев и дважды голосует на той же странице, в том же окне браузера, нет
необходимости отправлять ему дюжину писем с подтверждением.

Вместо этого FastComments отправит в этом случае только одно письмо с подтверждением.

Для этого не требуются cookies.

Если окно браузера закрывается, или пользователь уходит со страницы, это считается окончанием сеанса верификации.
