Непроверенные комментарии будут отображаться с маленькой меткой «Unverified comment:», и выглядеть слегка
приглушёнными.

Всё это можно настроить, см. наше [руководство по настройке](/guide-customizations-and-configuration.html#custom-css).

Можно настроить истечение срока действия непроверенных комментариев, см. нашу документацию [Настройки модерации](/guide-moderation.html#moderation-settings).

Анонимные комментарии всегда считаются непроверенными - поскольку пользователь не оставил адрес электронной почты, который мог
быть использован для верификации.

Разрешение анонимных комментариев не убирает метку Unverified, однако её можно удалить с помощью стилей или [конфигурации](/guide-customizations-and-configuration.html#disable-unverified-label).
