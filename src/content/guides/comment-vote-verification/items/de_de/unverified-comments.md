Unbestätigte Kommentare werden mit einem kleinen Label „Unbestätigter Kommentar:“ angezeigt und wirken leicht
ausgegraut.

All dies kann angepasst werden, siehe unseren [Anpassungsleitfaden](/guide-customizations-and-configuration.html#custom-css).

Unbestätigte Kommentare können so konfiguriert werden, dass sie ablaufen, siehe unsere [Moderationseinstellungen](/guide-moderation.html#moderation-settings)-Dokumentation.

Anonyme Kommentare sind immer unbestätigt - da der Benutzer keine E-Mail-Adresse hinterlassen hat, die zur
Verifizierung verwendet werden könnte.

Das Zulassen anonymer Kommentare entfernt nicht das Label „Unbestätigt“, dies kann jedoch mit Styling oder [Konfiguration](/guide-customizations-and-configuration.html#disable-unverified-label) entfernt werden.
