Цитата с https://oembed.com:

> oEmbed — это формат, позволяющий встраивать представление URL на сторонних сайтах. Простой API позволяет веб-сайту отображать встроенный контент (например, фотографии или видео), когда пользователь публикует ссылку на этот ресурс, без необходимости напрямую анализировать ресурс.

FastComments поддерживает формат oEmbed и доступен через продукты, использующие https://embed.ly.
