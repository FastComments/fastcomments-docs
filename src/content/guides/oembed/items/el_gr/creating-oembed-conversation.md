Με το FastComments, μια συνομιλία πρέπει πρώτα να δημιουργηθεί πριν μπορέσει να ενσωματωθεί.

Για να το κάνετε αυτό, απλώς επισκεφθείτε το [fastcomments.com/create](https://fastcomments.com/create).

[app-screenshot-start url='/create'; selector = '.content'; title='The Create a Conversation Form' app-screenshot-end]

Μόλις επιλέξετε ένα όνομα, θα δημιουργηθεί ένας ασφαλής σύνδεσμος.

Αυτός ο σύνδεσμος μπορεί να επισκεφθεί και να χρησιμοποιηθεί ως δικό του chat. Ή, μπορεί να παρασχεθεί απευθείας σε μια εφαρμογή που υποστηρίζει oEmbed, ή https://embed.ly, για να εισαχθεί η συνομιλία.

Οι ενσωματωμένες συνομιλίες μπορούν να προσαρμοστούν όπως κάθε άλλη εγκατάσταση FastComments χρησιμοποιώντας τη διεπαφή προσαρμογής. Διαβάστε για την προσαρμογή του widget σχολίων [εδώ](/guide-customizations-and-configuration.html).
