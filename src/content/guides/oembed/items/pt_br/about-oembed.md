Citação de https://oembed.com:

> oEmbed é um formato que permite uma representação incorporada de uma URL em sites de terceiros. A API simples permite que um site exiba conteúdo incorporado (como fotos ou vídeos) quando um usuário publica um link para esse recurso, sem ter que analisar o recurso diretamente.

FastComments suporta o formato oEmbed e está disponível através de produtos que usam https://embed.ly.
