Avec FastComments, une conversation doit d'abord être créée avant de pouvoir être intégrée.

Pour ce faire, visitez simplement [fastcomments.com/create](https://fastcomments.com/create).

[app-screenshot-start url='/create'; selector = '.content'; title='The Create a Conversation Form' app-screenshot-end]

Une fois le nom choisi, un lien sécurisé sera créé.

Ce lien peut être visité et utilisé comme son propre chat. Ou il peut être fourni directement à une application qui prend en charge oEmbed, ou https://embed.ly, pour insérer la conversation.

Les conversations intégrées peuvent être personnalisées comme toute autre installation FastComments en utilisant l'interface de personnalisation. Lisez comment personnaliser le widget de commentaires [ici](/guide-customizations-and-configuration.html).
