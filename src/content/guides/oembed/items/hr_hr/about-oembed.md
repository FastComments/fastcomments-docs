Citat s https://oembed.com:

> oEmbed je format koji omogućuje ugrađeni prikaz URL-a na web stranicama trećih strana. Jednostavan API omogućuje web stranici prikazivanje ugrađenog sadržaja (poput fotografija ili videa) kada korisnik objavi poveznicu na taj resurs, bez potrebe za izravnim analiziranjem resursa.

FastComments podržava oEmbed format i dostupan je kroz proizvode koji koriste https://embed.ly.
