Con FastComments, una conversazione deve essere prima creata prima di poter essere incorporata.

Per farlo, visita semplicemente [fastcomments.com/create](https://fastcomments.com/create).

[app-screenshot-start url='/create'; selector = '.content'; title='The Create a Conversation Form' app-screenshot-end]

Una volta scelto un nome, verrà creato un link sicuro.

Questo link può essere visitato e utilizzato come chat a sé stante. Oppure può essere fornito direttamente a un'applicazione che supporta oEmbed, o https://embed.ly, per inserire la conversazione.

Le conversazioni incorporate possono essere personalizzate come qualsiasi altra installazione di FastComments utilizzando l'interfaccia di personalizzazione. Leggi come personalizzare il widget dei commenti [qui](/guide-customizations-and-configuration.html).
