Citazione da https://oembed.com:

> oEmbed è un formato che consente una rappresentazione incorporata di un URL su siti di terze parti. La semplice API consente a un sito web di visualizzare contenuti incorporati (come foto o video) quando un utente pubblica un link a quella risorsa, senza dover analizzare direttamente la risorsa.

FastComments supporta il formato oEmbed ed è disponibile attraverso prodotti che utilizzano https://embed.ly.
