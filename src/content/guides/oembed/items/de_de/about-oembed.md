Zitat von https://oembed.com:

> oEmbed ist ein Format, das eine eingebettete Darstellung einer URL auf Websites Dritter ermöglicht. Die einfache API ermöglicht es einer Website, eingebettete Inhalte (wie Fotos oder Videos) anzuzeigen, wenn ein Benutzer einen Link zu dieser Ressource postet, ohne die Ressource direkt analysieren zu müssen.

FastComments unterstützt das oEmbed-Format und ist über Produkte verfügbar, die https://embed.ly verwenden.
