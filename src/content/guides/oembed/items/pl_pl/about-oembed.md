> oEmbed to format umożliwiający osadzoną reprezentację adresu URL na stronach trzecich. Proste API pozwala stronie internetowej wyświetlać osadzoną zawartość (taką jak zdjęcia lub filmy), gdy użytkownik opublikuje link do tego zasobu, bez konieczności bezpośredniego analizowania zasobu.
