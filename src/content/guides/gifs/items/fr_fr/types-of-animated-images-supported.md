Les images animées, ou parfois simplement appelées "gifs", peuvent être téléchargées manuellement via le bouton de téléchargement d'images
lors des commentaires, ou sélectionnées via le Gif Picker.

La désactivation des téléchargements d'images désactive entièrement les images dans les commentaires - à la fois via les téléchargements et le Gif Picker.
