Αυτά τα components είναι wrappers γύρω από τα βασικά μας components του VanillaJS. Μπορούμε να ενημερώνουμε αυτόματα αυτά τα components (να διορθώνουμε σφάλματα, να προσθέτουμε δυνατότητες) χωρίς να δημοσιεύουμε αυτή τη βιβλιοθήκη, οπότε παρόλο που μπορεί να μην δημοσιευτεί για κάποιο διάστημα, αυτό δεν σημαίνει ότι το FastComments δεν βρίσκεται υπό ενεργή ανάπτυξη! Μη διστάσετε να δείτε [το ιστολόγιό μας](https://blog.fastcomments.com/) για ενημερώσεις. Σοβαρές αλλαγές στο API ή νέες λειτουργίες δεν θα αποσταλούν ποτέ στην υποκείμενη βασική βιβλιοθήκη χωρίς αύξηση έκδοσης σε αυτή τη βιβλιοθήκη.
