サイトの `Gemfile` の `:jekyll_plugins` グループにこの gem を追加してください:

```ruby
group :jekyll_plugins do
  gem "fastcomments-jekyll"
end
```

次に:

```bash
bundle install
```

(Jekyll 3.7+ および 4.x と互換性があります。)
