Ове компоненте су омотачи око наших основних VanillaJS компоненти. Можемо аутоматски ажурирати те компоненте (поправљати грешке, додавати функције) без објављивања ове библиотеке, тако да, иако можда неће бити објављена неко време, то не значи да FastComments није активно у развоју! Слободно погледајте [наш блог](https://blog.fastcomments.com/) за ажурирања. Промене API-ја које нису уназад компатибилне или нове функције никад неће бити додате у основну библиотеку без повећања верзије у овој библиотеци.
