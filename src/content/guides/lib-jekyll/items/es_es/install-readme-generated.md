[![Gema](https://img.shields.io/gem/v/fastcomments-jekyll?logo=rubygems&label=gem&color=e9573f)](https://rubygems.org/gems/fastcomments-jekyll)

Añada la gema al grupo `:jekyll_plugins` en el `Gemfile` de su sitio:

```ruby
group :jekyll_plugins do
  gem "fastcomments-jekyll"
end
```

Luego:

```bash
bundle install
```

(Compatible con Jekyll 3.7+ y 4.x.)
