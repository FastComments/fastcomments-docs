Te komponenty są nakładkami na nasze podstawowe komponenty VanillaJS. Możemy automatycznie aktualizować te komponenty (naprawiać błędy, dodawać funkcje) bez publikowania tej biblioteki, więc chociaż może nie być opublikowana przez jakiś czas, nie oznacza to, że FastComments nie jest aktywnie rozwijany! Zachęcamy do sprawdzenia [naszego bloga](https://blog.fastcomments.com/) w poszukiwaniu aktualizacji. Zmiany łamiące API lub nowe funkcje nigdy nie zostaną wprowadzone do bazowej biblioteki bez zmiany wersji tej biblioteki.
