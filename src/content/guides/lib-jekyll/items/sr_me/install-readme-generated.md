[![Гем](https://img.shields.io/gem/v/fastcomments-jekyll?logo=rubygems&label=gem&color=e9573f)](https://rubygems.org/gems/fastcomments-jekyll)

Додајте гем у групу `:jekyll_plugins` у `Gemfile` вашег сајта:

```ruby
group :jekyll_plugins do
  gem "fastcomments-jekyll"
end
```

Затим:

```bash
bundle install
```

(Компатибилно са Jekyll 3.7+ и 4.x.)
