Pour ajouter des commentaires à votre site WordPress, vous pouvez trouver notre plugin WordPress <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">ici</a>.

Un guide d'installation complet et la documentation du plugin [se trouvent ici](/guide-installation-wordpress.html).

Ce plugin prend en charge les commentaires en direct, le SSO et l'installation sans code. Suivez simplement le guide d'installation dans la page d'administration
après avoir installé le plugin. Il vous guidera pour connecter votre installation WordPress à votre compte.

Tous les commentaires laissés via FastComments avec notre plugin WordPress peuvent être automatiquement synchronisés vers votre WordPress
installation afin que vous conserviez le contrôle de vos données. Cela peut être désactivé pour limiter la taille de votre base de données WordPress.
