Хотите добавить комментарии на страницы Notion, Google Sites или Nimbus Notes? Мы вам поможем.

### Notion

Чтобы добавить комментарии в документы Notion, [смотрите поясняющее видео здесь](https://blog.fastcomments.com/(10-05-2021)-adding-commenting-to-notion-pages.html).

### Google Sites

Google Sites также поддерживается, и процесс аналогичен.

1. [Создайте обсуждение для встраивания.](https://fastcomments.com/create)
2. При редактировании вашего Google Site перейдите в `Insert` -> `Embed`. Выберите `By URL`.
3. Вставьте URL из шага 1.
4. Нажмите `Insert`. Готово!

### Nimbus Note

Добавление живых комментариев в Nimbus Note также поддерживается, и процесс схож с Notion.

1. [Создайте обсуждение для встраивания.](https://fastcomments.com/create)
2. При редактировании заметки добавьте блок встраивания.
3. Используйте URL из шага 1 в качестве URL для встраивания.
4. Готово!
