Хотите добавить комментарии на ваши страницы Notion? Мы поможем.

### Notion

Для добавления комментариев в документы Notion [смотрите видео с объяснением здесь](https://blog.fastcomments.com/(10-05-2021)-adding-commenting-to-notion-pages.html).

### Google Sites

Google Sites также поддерживается, и процесс аналогичен.

1. [Создайте беседу для встраивания.](https://fastcomments.com/create)
2. При редактировании вашего Google Site перейдите в `Вставка` -> `Встроить`. Выберите `По URL`.
3. Вставьте URL из Шага 1.
4. Нажмите `Вставить`. Готово!

### Nimbus Note

Добавление комментариев в реальном времени в Nimbus Note также поддерживается, и процесс аналогичен Notion.

1. [Создайте беседу для встраивания.](https://fastcomments.com/create)
2. При редактировании вашей заметки добавьте блок встраивания.
3. Используйте URL из Шага 1 в качестве URL для встраивания.
4. Готово!
