VanillaJS 버전의 위젯은 설치가 매우 쉬우며 빌드 시스템이나 서버 측 코드가 필요하지 않습니다.

모든 페이지에 다음 코드 스니펫을 추가하기만 하면 됩니다:

[code-example-start config = {}; title = '간단한 코드 스니펫'; code-example-end]

동일한 코드 스니펫을 여러 페이지에서 사용할 수 있습니다. 페이지별로 자동으로 별도의 스레드가 생성됩니다.

많은 애플리케이션에는 "HTML 임베드 코드" 옵션이 있습니다. 해당 옵션을 선택하고 위의 코드 스니펫을 붙여넣으세요.

*체험하는 데 계정이 필요하지 않습니다!* 로그인하지 않은 경우 위 스니펫에 "tenantId: demo"가 표시될 수 있습니다. 이 경우
데모 계정이 사용됩니다.

위젯 구성에 대한 문서는 <a href="/guide-customizations-and-configuration.html" target="_blank">여기</a>에서 찾을 수 있습니다.

FastComments 위젯의 모든 버전은 핵심 VanillaJS 라이브러리의 래퍼입니다. 이를 통해 한 곳에서 기능을 추가하고
문제를 수정할 수 있으며, 변경 사항은 자동으로 댓글 위젯의 다른 변형에 전파됩니다.
