Щоб додати коментарі на ваш сайт WordPress, ви можете знайти наш плагін WordPress <a href="https://wordpress.org/plugins/fastcomments/" target="_blank">тут</a>.

Повний посібник зі встановлення та документація по плагіну [доступні тут](/guide-installation-wordpress.html).

Цей плагін підтримує живі коментарі, SSO та встановлення без коду. Просто виконайте інструкцію зі встановлення на сторінці адміністратора
після встановлення плагіну. Вона проведе вас через підключення вашої установки WordPress до вашого облікового запису.

Усі коментарі, залишені через плагін FastComments для WordPress, можуть автоматично синхронізуватися назад у вашу установку WordPress
щоб ви зберігали контроль над своїми даними. Це можна вимкнути, щоб обмежити розмір вашої бази даних WordPress.
