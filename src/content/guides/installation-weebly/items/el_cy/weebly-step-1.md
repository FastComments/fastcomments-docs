Πρώτα, βεβαιωθείτε ότι έχετε προσθέσει μερικές δημοσιεύσεις ιστολογίου στον ιστότοπό σας για να δείτε τα αποτελέσματα αυτού του οδηγού.

Μόλις είστε έτοιμοι, μπορούμε να προχωρήσουμε και να ανοίξουμε τον επεξεργαστή Weebly, και να κάνουμε κλικ στο `Settings` στο επάνω μέρος.

**Σημείωση** εάν θέλετε να μεταφέρετε υπάρχοντα σχόλια από τον ιστότοπό σας στο Weebly προς το FastComments, ο εκπρόσωπος υποστήριξής σας μπορεί να βοηθήσει με αυτή τη μετανάστευση.
