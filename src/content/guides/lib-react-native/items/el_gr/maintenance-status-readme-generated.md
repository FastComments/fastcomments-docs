Αυτά τα components είναι περιβλήματα γύρω από τα βασικά μας components VanillaJS. Μπορούμε να ενημερώσουμε αυτόματα αυτά τα components (διόρθωση σφαλμάτων, προσθήκη δυνατοτήτων) χωρίς να δημοσιεύσουμε αυτή τη βιβλιοθήκη, οπότε παρόλο που μπορεί να μην δημοσιευτεί για κάποιο διάστημα, αυτό δεν σημαίνει ότι FastComments δεν βρίσκεται σε ενεργή ανάπτυξη! Μη διστάσετε να επισκεφθείτε το [our blog](https://blog.fastcomments.com/) για ενημερώσεις. Αλλαγές που σπάνε το API ή δυνατότητες δεν θα αποσταλούν ποτέ στην υποκείμενη βασική βιβλιοθήκη χωρίς αύξηση έκδοσης σε αυτή τη βιβλιοθήκη.
