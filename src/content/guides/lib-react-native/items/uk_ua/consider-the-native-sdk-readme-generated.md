Ця бібліотека використовує webview для обгортання бібліотеки VanillaJS.

FastComments тепер підтримує повністю нативну реалізацію нашого клієнта в складі [fastcomments-react-native-sdk](https://github.com/FastComments/fastcomments-react-native-sdk).
Якщо вам потрібна повністю нативна реалізація, можливо, варто подивитися туди.
