Ці компоненти є обгортками навколо наших основних VanillaJS-компонентів. Ми можемо автоматично оновлювати ці компоненти (виправляти помилки, додавати можливості) без публікації цієї бібліотеки, тож навіть якщо вона деякий час не буде опублікована, це не означає, що FastComments не перебуває в активній розробці! Не соромтеся переглядати [наш блог](https://blog.fastcomments.com/) за оновленнями. Зміни API, що ламають сумісність, або нові функції ніколи не будуть додані до підлеглої основної бібліотеки без підвищення версії в цій бібліотеці.
