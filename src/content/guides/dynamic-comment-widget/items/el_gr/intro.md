Με το FastComments, μπορούμε να έχουμε όσα widget σχολίων θέλουμε στην ίδια σελίδα. Μπορούμε να τα ορίσουμε όλα στο ίδιο
νήμα, και θα παραμένουν συγχρονισμένα, ή μπορούμε να τα έχουμε όλα σε διαφορετικά νήματα ταυτόχρονα. Μπορούμε να προσθέτουμε, να αφαιρούμε,
και να αλλάζουμε τα νήματα σχολίων κατ' απαίτηση.
