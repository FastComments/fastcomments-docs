Για να ξεκινήσετε, πρέπει να δημιουργήσετε έναν λογαριασμό FastComments στο FastComments.com.

Αν θέλετε επίσης να φιλοξενήσετε το FastComments στους δικούς σας ιστότοπους, θα πρέπει να [εγγραφείτε εδώ](https://fastcomments.com/traffic-pricing).

Διαφορετικά, μπορείτε να δημιουργήσετε έναν κανονικό λογαριασμό "σχολιαστή". Μπορείτε να το κάνετε αυτό αφήνοντας το επιθυμητό όνομα χρήστη και email στη [σελίδα σύνδεσης](https://fastcomments.com/auth/login).

Αυτό θα δημιουργήσει έναν λογαριασμό σχολιαστή για εσάς. Μπορείτε να χρησιμοποιήσετε αυτόν τον λογαριασμό για να σχολιάζετε με το FastComments και επίσης να ρυθμίσετε το πρόγραμμα Συνεργατών μέσω του στοιχείου πλοήγησης Συνεργάτες.
