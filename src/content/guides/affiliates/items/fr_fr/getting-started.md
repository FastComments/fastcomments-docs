Pour commencer, vous devez créer un compte FastComments sur FastComments.com.

Si vous souhaitez également héberger FastComments sur vos propres sites, vous devriez [vous inscrire ici](https://fastcomments.com/traffic-pricing).

Sinon, vous pouvez créer un compte de « commentateur » régulier. Vous pouvez le faire en laissant votre nom d'utilisateur et e-mail souhaités sur la [page de connexion](https://fastcomments.com/auth/login).

Cela créera un compte de commentateur pour vous. Vous pouvez utiliser ce compte pour commenter avec FastComments et également configurer le programme d'affiliation via l'élément de navigation Affiliés.
