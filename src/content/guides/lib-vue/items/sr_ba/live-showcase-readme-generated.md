Да бисте видели сваки видџет и флов који се покрећу локално против јавног `demo` тенанта, клонирајте репозиторијум и покрените:

```bash
npm install
npm run serve
```

Пojedиначни демои такође имају своје улазне тачке (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) унутар `dev/`.
