За да видите всеки widget и flow, работещ локално срещу публичния `demo` tenant, клонирайте репото и стартирайте:

```bash
npm install
npm run serve
```

Отделните демота също имат своите собствени входни точки (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) в `dev/`.
