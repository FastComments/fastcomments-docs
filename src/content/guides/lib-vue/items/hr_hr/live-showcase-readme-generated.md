Da biste vidjeli svaki widget i flow koji se pokreće lokalno protiv javnog `demo` tenanta, klonirajte repo i pokrenite:

```bash
npm install
npm run serve
```

Pojedinačni demo primjeri također imaju svoje ulazne točke (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) pod `dev/`.
