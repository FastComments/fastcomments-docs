Για να δείτε κάθε widget και ροή να τρέχουν τοπικά ενάντια στον δημόσιο tenant `demo`, κλωνοποιήστε το αποθετήριο και τρέξτε:

```bash
npm install
npm run serve
```

Τα μεμονωμένα demos έχουν επίσης τα δικά τους σημεία εισόδου (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) στον φάκελο `dev/`.
