Te komponenty są wrapperami wokół naszych podstawowych komponentów VanillaJS. Możemy automatycznie aktualizować te komponenty (naprawiać błędy, dodawać funkcje) bez publikowania tej biblioteki, więc choć może ona nie być publikowana przez pewien czas, nie oznacza to, że FastComments nie jest aktywnie rozwijany! Zapraszamy do sprawdzania [naszego bloga](https://blog.fastcomments.com/) w celu uzyskania aktualizacji. Zmiany łamiące API lub nowe funkcje nigdy nie zostaną wprowadzone do bazowej biblioteki bez podbicia wersji w tej bibliotece.
