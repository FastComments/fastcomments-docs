---
כדי לראות את כל הווידג'טים והזרימות שמריצים באופן מקומי מול ה-tenant הציבורי `demo`, שכפל את המאגר והרץ:

```bash
npm install
npm run serve
```

לדמוים בודדים יש גם נקודות כניסה משלהם (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) תחת `dev/`.
