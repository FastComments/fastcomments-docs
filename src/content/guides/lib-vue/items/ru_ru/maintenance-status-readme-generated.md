Эти компоненты являются оболочками вокруг наших основных компонентов VanillaJS. Мы можем автоматически обновлять эти компоненты (исправлять ошибки, добавлять функции) без публикации этой библиотеки, так что, хотя она может некоторое время не публиковаться, это не означает, что FastComments не находится в активной разработке! За обновлениями заглядывайте в [наш блог](https://blog.fastcomments.com/). Ломающие изменения API или новые функции никогда не будут внесены в лежащую в основе core-библиотеку без повышения версии этой библиотеки.
