Ови компоненти су омотачи око наших основних VanillaJS компонената. Можемо аутоматски ажурирати ове компоненте (исправљати грешке, додавати функције) без објављивања ове библиотеке, па иако она можда неће бити објављена неко време, то не значи да FastComments није у активном развоју! Слободно погледајте [наш блог](https://blog.fastcomments.com/) за ажурирања. Промене API-ја које нарушавају уназадну компатибилност или нове функције никада неће бити испоручене у основну библиотеку без повећања верзије у овој библиотеци.
