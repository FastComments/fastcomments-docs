Če želite videti vse pripomočke in tokove, ki se lokalno izvajajo proti javnemu najemniku `demo`, klonirajte repozitorij in zaženite:

```bash
npm install
npm run serve
```

Posamezne demonstracije imajo tudi svoje vstopne točke (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) v mapi `dev/`.
