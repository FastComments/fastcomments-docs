Da biste videli svaki widget i flow koji rade lokalno protiv javnog `demo` tenanta, klonirajte repozitorijum i pokrenite:

```bash
npm install
npm run serve
```

Pojedinačni demo primeri takođe imaju svoje ulazne tačke (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) u `dev/`.
