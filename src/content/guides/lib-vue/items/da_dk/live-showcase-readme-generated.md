For at se alle widgets og flows, der kører lokalt mod den offentlige `demo`-tenant, klon repoen og kør:

```bash
npm install
npm run serve
```

Individuelle demoer har også deres egne startpunkter (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) under `dev/`.
