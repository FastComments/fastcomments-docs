Για να δείτε κάθε widget και flow να τρέχει τοπικά ενάντια στον δημόσιο tenant `demo`, κλωνοποιήστε το repo και τρέξτε:

```bash
npm install
npm run serve
```

Τα μεμονωμένα demos έχουν επίσης τα δικά τους σημεία εισόδου (`npm run serve-collab-chat`, `serve-live-chat`, `serve-image-chat`, `serve-pagination`, `serve-toggle-dark-mode`) κάτω από `dev/`.
