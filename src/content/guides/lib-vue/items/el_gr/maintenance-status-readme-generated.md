Αυτά τα components είναι wrappers γύρω από τα βασικά components του VanillaJS. Μπορούμε να ενημερώνουμε αυτόματα αυτά τα components (επιδιόρθωση σφαλμάτων, προσθήκη λειτουργιών) χωρίς να δημοσιεύουμε αυτή τη βιβλιοθήκη, οπότε το γεγονός ότι ίσως να μην έχει δημοσιευτεί για λίγο δεν σημαίνει ότι το FastComments δεν βρίσκεται σε ενεργή ανάπτυξη! Μη διστάσετε να επισκεφθείτε [το ιστολόγιό μας](https://blog.fastcomments.com/) για ενημερώσεις. Αλλαγές που σπάνε το API ή νέες λειτουργίες δεν θα προωθούνται ποτέ στην υποκείμενη βασική βιβλιοθήκη χωρίς αύξηση έκδοσης σε αυτή τη βιβλιοθήκη.
